View Linked E-mails as `Tasks’

If you have configured Everest MailBridge to link Outlooke-mails as ‘tasks’ in Everest, all linked e-mails are created as ‘tasks’ and displayed in the My Tasks/All Tasks browser as well as in the EverestE-Mail window (path: CRM > E-Mail).

To view e-mails that are linked as ‘tasks’, do the following:

  • From the Everest main window, select Contact Manager > My Tasks/All Tasks. You will see the My Tasks/All Tasks browser. The linked e-mails are displayed as ‘tasks’.
  • From the My Tasks/All Tasks browser, select Options > Edit > In New Window to view the e-mail.

When you receive e-mails from a customer that have been linked as ‘tasks’, you can view the From, To, CC and BCCdetails by clicking the icon in the Task profile (path: Contact Manager > All Tasks > Options > Edit > In New Window > Task tab). You will see the Notes window. The name and e-mail ID of the sender (person who sent the e-mail) is displayed in this window. Click Options > Exit to close the window.

When you send e-mails that have been linked as ‘tasks’, they are automatically marked as ‘Complete’.
