Select the bin type to be used when calculating required quantity. You can select one of the following values:

  • Normal Type Bins Only – This selection excludes repair, in process, scrap, exchange, write off, and defective bins. Select this option if you do not want to include all bin types in the Required quantity on the Item Replenishment browser.
  • All Bins – This selection includes bins of every type. Select this option if you want to include bins of every type in the Required quantity on the Item Replenishment browser.
