Payment Information

Terms
Specify the payment term allowed by the vendor. By default this field is populated with the payment terms defined for the vendor in the Vendor’s profile. You can select a different payment term, if required.

Due date
Specify the date when you are expected to pay the vendor.

Override
Select this checkbox to ignore the payment term’s due date specified for the document and use the Due Date field to specify the last date of payment. If this checkbox is selected the Terms field is disabled. If this checkbox is not selected the Due Date option is disabled.
