Entry for Inventoried Items - No Items Received

When you create a purchase invoice and no quantity is received on the invoice, the following entries are made in a system Purchase Journal:

Account Path Debit/


Credit		 Amount
Accounts Payable Account The account specified in the Accounts
Payable field (path: File > Setup > Accounting > Posting Group profile) of Vendor Posting Group type.


Defaults - Accounting		 Credit Total amount of the document (including recoverable and non-recoverable portion of tax)
Accrued Purchases Account The account specified in the Accrued
Purchases field (path: File > Setup > Accounting > Posting Group profile) of Vendor Posting Group type.


Defaults - Accounting		 Debit Invoiced amount (including non-recoverable portion of tax)
Tax Component Account (the asset account for Value Added Tax recoverable in tax component in the purchase tax code) The account selected for Tax Account Code in the Tax Components profile (path: File > Setup > Accounting > Taxes > Tax
Components > Options > Edit > In New Window > Purchase tab).		 Debit Amount of recoverable tax

If a negative quantity is included in the Invoiced column, an accounting entry will be created for the net amount.
