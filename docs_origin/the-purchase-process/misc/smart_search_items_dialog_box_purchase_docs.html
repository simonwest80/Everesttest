Smart Search Items Dialog Box

Search for
Enter the search string.
Get
This option allows you to retrieve records in the Search dialog box. Click the drop-down button and select All to retrieve all the records at one time. Select More to retrieve records in batches. Enter the number of records to be retrieved at one time, in the space provided.
Details Grid
The details grid displays the items that are retrieved based on the criteria entered in the Search for field. Each row displays information about the item specified in that row.
  • Type - This column displays whether the match was with a serial number, item code, model number, item alias or manufacturer’s serial number.
  • Matched - This column displays the code in the search parameter that the item is matched with.
If you enter 101, the code that it matched with could be the item 101FGTIPS, serial number 1010 or serial number 101.
The other columns in the details grid display information on the item code and description, the category the item belongs to, the item type, item cost, stock quantities, custom fields, etc.
Item Code
Select this check box if you want the search to display all item codes that start with the characters you entered in the Search for field.

If you enter “101”, the search lists all items that start with “101”.

Manufacturer’s Serial #
Select this check box if you want the search to display all manufacturer’s serial numbers that start with the characters you entered in the Search for field.

If you enter “101”, the search will list all manufacturer’s serial numbers that start with “101”.

Incremental Search
If you select this check box, Everest will display matching codes for the item as you start typing the characters in the Search for field.

If you enter “1”, the search will display codes that start with “1”. If you enter “0” after this, it will automatically list all matching codes that start with “10”, and so on.

Model
Select this check box if you want the search to display all models that start with the characters you entered in the Search for field.

If you enter “123”, the search will list all models that start with “123”.

Item Alias
Select this check box if you want the search to display all item alias codes that start with the characters you entered in the Search for field.

If you enter “Vendor”, the search will list all item aliases that start with “Vendor”.

Serial #
Select this check box if you want the search to display all serial numbers that start with the characters you entered in the Search for field.

If you enter “101”, the search will list all serial numbers that start with “101”.

Vendor Part #
Select this option to search the item based on the vendor’s part number. By default, this option will be selected.

You need adequate security rights in a purchase document to select items that are not associated with a vendor.
