View Options

The options in the View menu allow you to view a list of system generated, manual, historical and Payroll type of purchase receipt documents. You can choose the type of purchase receipt documents you want to view.

Detailed
Select this option to view a list of all the system generated purchase receipts in the Purchase Receipts browser. This option is selected by default. You can choose to deselect this option.
Detailed
Historical
Select this option to view a list of all vendor documents that are imported using the Data Import Wizard (path: Utility > Import > Data > Historical Vendor Documents).
Everest prefixes the document number you enter with ‘H’, that denotes a historical document.
