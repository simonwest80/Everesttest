Exchange Info

Click the Exchange Info button to view relevant information. The information displayed in this option has been divided into the following sections:

  • Document Currency
  • Bank Currency
  • Pay In
  • Base Currency
  • Amount

Whether the buy or sell rate is displayed in the Exchange Rate fields depends on the setting you make in the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting > Multi Currency > Exchange Rates tab > Rate for payments field).
