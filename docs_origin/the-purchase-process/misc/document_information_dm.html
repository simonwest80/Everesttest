Document Information

The following information are displayed about the document:

Document #
The document number is automatically assigned based on the sequence number specified for debit orders in the logged in location/sub-location.
The document number is displayed for a new debit order after the document details are saved.
Document Number
Document Alias
The default document alias specified for the debit order is displayed in this field. You can change this alias.
Document Alias
Document Notes
Click the icon beside the Document # field to specify document notes.
Document Notes
Document Date
This field beside the Document# field, displays the login date. You cannot edit this field.
Document Dates
Document Policy
Click the icon on the status bar to view the document policy. You can edit this policy.
Document Policy
Document Status
This section displays the status of the document.
Document Status
