Pre-requisites for E-Mailing Documents

Sales and purchase documents can be sent by e-mail to customers/vendors/users by using the Everest E-Mail, Outlook and Outlook Express mail clients. Click the button in the Document E-mail Preferences dialog box to send the document as an e-mail attachment.

If you set up an e-mail account using Everest E-Mail AND ** select the **Send e-mail from browsers and profiles using Everest e-mail option (path: CRM > E-Mail > Tools > Options), the Everest e-mail client opens.

To ensure that the other mail clients (Outlook or Outlook Express) open, do the following:

  • Clear the Send e-mail from browsers and profiles using Everest e-mail option (path: CRM > E-Mail > Tools > Options).
  • Specify a default printer for your system.
  • Specify the export file format as word or pdf (path: File > Setup > Preferences > Workstation Settings > General > General Settings tab > Export File Format field).
  • Specify the Export File Path in the Workstation Settings (path: Workstation Settings dialog box > General > General Settings tab > Export File Path (High Speed) field).
