Paths to the Accounts for Accounting Entries

Accounts Payable Account Specified in the Accounts Payable field (path: File > Setup > Accounting > Posting Group profile) of Vendor Posting Group type.
Accrued Purchases Account Specified in the Accrued Purchases field (path: File > Setup > Accounting > Posting Group profile) of Vendor Posting Group type.
Accrued Purchase (Adjustment) Account Specified in the Accrued Purchases field (path: File > Setup > Accounting > Posting Group profile) of Vendor Posting Group type.
Expense/Liability Account Specified in the Posting Group profile (path: File > Setup > Accounting > Posting Groups) of the Inventory Posting Group type.
Expense/Asset


Account		 Specified in the Posting Group profile (path: File > Setup > Accounting > Posting Groups) of the Inventory Posting Group type.
Gain on Purchase Returns Specified in the Gain on Purchase Returns field (path: File > Setup > Accounting > Posting Groups) of the Inventory Posting Group type.
Inventory Control Account Specified in the Accounts Payable field (path: File > Setup > Accounting > Posting Group profile) of Inventory Posting Group type.
Loss on Purchase Returns Specified in the Loss on Purchase Returns field (path: File > Setup > Accounting > Posting Groups) of the Inventory Posting Group type.
Purchase Price Variance Account Specified in the Purchase Price Variance Account field (path: File > Setup > Accounting > Posting
Group profile) of Vendor Posting Group type.
Recoverable Tax Liability Specified for Tax Account Code in the Tax Components profile (path: File > Setup > Accounting > Taxes > Tax Components > Options > Edit > In New Window > Purchase tab).
