The Debit Quotes Browser

The Debit Quotes browser (path: Purchasing > Debit Quotes) lists all the debit quotes that are not yet converted to debit orders.

Filter records based on date

The Dates field in Debit Quotes browser enables you to filter debit quotes based on their creation date. By default, the browser displays all documents created during the period specified in the Select the default date option based on which the records will be displayed in the purchase browser option (path: File > Setup > Preferences > Flow Control > Documents > Prompts > Purchase Document tab).

 Filter Purchase Documents based on Dates

Options common to all purchase return document browsers are available in this browser. This browser has the additional option Purge (path: Debit Quotes browser > Options > Purge).
