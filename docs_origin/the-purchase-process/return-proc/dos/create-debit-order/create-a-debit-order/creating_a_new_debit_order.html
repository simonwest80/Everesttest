Create a Debit Order

To create a debit order from the debit order browser, do the following:

  • From the main menu bar, select Purchasing > Debit Order. You will see the Debit Orders browser.
  • From the Debit Orders browser, select Options > New > In New Window. You will see the Debit Order profile.

You can also create a debit order from an existing Debit Order profile by selecting Options > New > In New Window.
