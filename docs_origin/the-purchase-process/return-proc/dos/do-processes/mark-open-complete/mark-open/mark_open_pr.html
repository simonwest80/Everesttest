Mark Open

Mark a closed debit order ‘Open’ (path DebitOrder profile > Process > Mark Close/Open) by adding to the quantity requested or adding new items on the order.

Before you mark an order open, you must do the following:

  • Open the closed order. To view closed orders, select the ShippedOrders option (path: DebitOrders browser > Options > Shipped DebitOrders).
  • Add the item being returned, if the item does not exist on the order.
  • Increase the quantity requested in case of an existing item.

When an order is marked open and saved, the order status changes to ‘Open’.
