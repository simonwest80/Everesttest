Extended Notes

Select the Extended Notes option to view the Extended Notes dialog box. This dialog box displays the notes as entered in the Notesand Specifications field (path: Inventory > Items > Options > Edit > In New Window > Notes tab).

To view the extended notes for an item, do the following:

  • From the purchase return document profile, select Options > Item > ExtendedNotes. You will see the Extended Notes dialog box.

You can only view the information in the ExtendedNotes dialog box.
