Retrieve an Item Matrix

To retrieve an item matrix into a purchase return document, do the following:

  • From the purchase return document profile, select Options > Retrieve > Item Matrix. You will see the Items browser. The Items browser displays all the item matrix groups that have been created.
  • Double-click on an item matrix group to select it. You will see the Matrix ItemsView dialog box.

You can select any number of matrix items from the browser. A Matrix Items View dialog box will be displayed once for every matrix that has been selected.
