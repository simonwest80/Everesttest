Hide

The Hide column (check box) allows you to hide lines in the Item Details Grid.

To hide a line in the item details grid, do the following:

  • From the Purchase Return Document profile highlight the item.
  • Scroll to the Hide column.
  • Check the box.
  • Click on the Hide button displayed in the purchase return document status bar. The selected line is hidden and will not be displayed in the Item Details Grid.
  • Click on the Show button displayed in the purchase return document status bar to view the hidden line. The selected line is displayed in the Item Details Grid.
