Document Date

The date displayed in a purchase return document:

  • Is by default is the login date. You cannot change the document date once the document has been created and saved. To create documents with different dates, you must login with a different date.
  • Is used to compute the due date for the document if the payment terms are of the On Account.

Processed Documents

If a quote is converted to an order, or an order is converted to a memo, the date displayed on the order or the memo will be the login date on which the document was converted and saved.

Document Date for a Debit Memo

If the document is a debit memo, the document date determines the accounting period in which the accounting entries for purchase return must be created.
