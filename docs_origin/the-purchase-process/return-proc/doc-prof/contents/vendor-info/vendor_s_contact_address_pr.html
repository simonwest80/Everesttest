Vendor’s Contact Address

The Contact field displays the address of the person who is to be contacted for matters relating to the purchase return document.

Document Type Contact Address
Debit Quote The name and address of the person with whom negotiations are conducted for a particular transaction.
Debit Orders and Invoices The name and address of the person responsible for the fulfillment of the return transaction.

Everest also allows you to select any billing, contact, and shipping or return address of any other vendor or customer using the Addresses browser. This will help you to select contact addresses for group companies of the vendor or a customer’s contact address if the vendor is also your customer.
