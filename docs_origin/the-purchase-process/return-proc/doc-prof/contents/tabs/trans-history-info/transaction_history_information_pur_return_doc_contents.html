Transaction History Information

The History tab in a purchase return document records the entire history of a transaction from the inception (debit quote) to the payment on the final document.

This tab is displayed only when you save the document.

The History tab displays the following information:

  • Details of debit quote creation such as date and time of creation, amount of document and the user who created the quote.
  • Details of debit order creation.
  • Details of debit memo creation.
  • Details of payments or deposits made and refunded on the document such as the payment type and reference number. Details of payments that are applied to the document are also displayed. Credit card numbers are masked using the format XXXXXXXXXXXX when the **Enable Extended Mode** check box is cleared (path: **File > Setup > Preferences > Workstation Settings > Company Settings** tab).
  • Details of modifications made to documents such as changing the status of the debit order, changes in the vendor contact and shipping addresses, document alias, pay terms, delivery method, location/sub-location, and jurisdiction.

To maintain transaction history for documents, you must select the option Maintain Transaction Historyfor Documents (path: File > Setup > Preferences > FlowControl > Documents > Miscellaneous > Others > Others1 tab).

To ensure that the transaction history attached to a quote or order is copied to the new document when it is converted to order or memo, you must select the Copy transaction history attached to an order to its corresponding final document (when converting from an order to a final document) check box (path: File > Setup > Preferences >Flow Control > Documents > Miscellaneous > Others > Others 1 tab).

Documents Dialog Box

To open a document displayed in the browser, select the OpenDocument option from the right-click menu. The Transaction History browser has the standard options common to all browsers.
