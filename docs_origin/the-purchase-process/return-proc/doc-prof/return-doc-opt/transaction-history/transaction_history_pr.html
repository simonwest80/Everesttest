Transaction History

Everest records the entire history of a transaction beginning from the inception (debit quote), order creation, conversion to debit memo, refunds received, closing of the order, opening of the closed order, payments applied, and modifications to the document.

You can view the TransactionHistory browser from any of the following locations:

  • From the Purchase Return Documents browser by selecting Options > Transaction History.
  • From the Purchase Return Document profile by selecting Options > Transaction History.
  • From the Purchase Return Document profile by selecting the History tab.

The Transaction History browser displays the following:

  • Details of debit quote/order/memo creation such as date and time of creation, amount of document and the user who created the document.
  • Details of payments or deposits made and refunded on the document such as the payment type and reference number. Details of payments that are applied to the document are also displayed. Credit card numbers are masked using the format XXXXXXXXXXXX when the **Enable Extended Mode** check box is cleared (path: **File > Setup > Preferences > Workstation Settings > Company Settings** tab).
  • Details of modifications made to documents such as changing the status of the debit order, changes in the vendor contact and shipping addresses, change in the document alias, pay terms, delivery method, location/sub-location and jurisdiction.
  • Displays whether the transaction status is Normal, Void, Reversed, or Reversal.

To open the documents displayed in the browser, select Open Document from the right-click menu. The Transaction History browser has standard options common to all browsers.

To maintain transaction history for documents, you must select the option Maintain Transaction Historyfor Documents (path: File > Setup > Preferences > FlowControl > Documents > Miscellaneous > Others > Others 1 tab).

To copy the transaction history attached to an order, you must select the option Copy the transaction history attached to an order to its corresponding final document (when converting from an order to a final document) (path: File > Setup > Preferences > Flow Control > Documents > Miscellaneous > Others > Others 1 tab).
