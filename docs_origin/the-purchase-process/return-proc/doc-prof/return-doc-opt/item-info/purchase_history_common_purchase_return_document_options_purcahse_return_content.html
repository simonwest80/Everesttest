Purchase History

The Purchase History option (path: Purchase ReturnDocument profile > Options > Item > PurchaseHistory) allows you to view the purchase history of an item for a selected document type.

The item’s purchase history browser for the selected document type displays a list of all documents in which the item has been selected and the quantity, price, discount, tax code and value for each document.

To view documents of another type, change the document type from the View Documents option.

To view the purchase history for a document, do the following:

  • From the Purchase ReturnDocument profile, select Options > Item > Purchase History. You will see a list of the associated document types.
  • Select the relevant document type. You will see the ItemPurchase History browser.
