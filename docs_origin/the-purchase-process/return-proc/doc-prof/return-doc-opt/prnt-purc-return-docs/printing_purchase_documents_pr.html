Print Purchase Return Documents

Purchase return documents can be printed from within a purchase return document or a purchase return document browser.

A debit order can be printed from the following locations:

  • From the Debit Order, select Options > Print, or
  • From the Debit Order Documents browser. Highlight the document and select Options > Print Document.

If the document is already printed, you cannot access this option unless you are assigned the security right to print more than one copy of the same document.

To print a purchase return document, do the following:

  • From the main menu bar, select Purchasing > Debit Quote/Order/Memo. You will see the purchase return document’s browser.
  • From the browser, highlight the relevant document and double-click. You will see the document profile.
  • Select Options > Print. You will see the Document PrintingPreferences dialog box.

To print multiple purchase return documents, do the following:

  • From the main menu bar, select Purchasing > Debit Quote/Order/Memo. You will see the purchase document’s browser.
  • Press the Shift key to select multiple documents from the document browser.
  • Select Options > Print Documents. You will see the Document PrintingPreferences dialog box.
