Modify a Purchase Order

You can modify a purchase order from any of the following locations:

  • Purchase Orders browser - Highlight the record and select Options > Edit > In New Window.
  • Purchase Order profile - Specify the document number in the Document # field.

You can modify all information except the following on a purchase order that does not have a payment history:

  • Document Number
  • Expiry Date
  • Vendor Code
  • Purchase Rep
  • Payment Terms

In a purchase order with payment history, you can modify all fields except the following:

  • Vendor code
  • Currency
  • Exchange Rate
  • Weight
