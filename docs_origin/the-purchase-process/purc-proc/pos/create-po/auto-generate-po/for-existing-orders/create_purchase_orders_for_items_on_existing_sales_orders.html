Create Automated Purchase Orders for Items on Existing Sales Orders

To create purchase orders for items on existing sales orders, do the following:

  • From the main menu bar, select Inventory > Automated Purchases > For Existing Orders. You will see the Automated Purchases - Existing Orders browser.
  • Specify in the Dates field, the delivery date range of sales orders for which you want to create purchase orders.
  • Deselect any item if you do not want to create a purchase order for it.
  • Select Options > Generate PO Without Links/Generate Linked POs. You will be asked to confirm your selection.
  • Click Yes. You will see the Purchase Order Status dialog box, which displays a list of purchase orders created.
  • Double-click on a document to view it.
