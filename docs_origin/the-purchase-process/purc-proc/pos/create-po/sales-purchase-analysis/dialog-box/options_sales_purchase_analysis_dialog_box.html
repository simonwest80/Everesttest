Options

Sales
Check this box, to view the sales of all the items during the selected period.
Sales by Customer
Check this box, to view the sales details along with details of the customers to whom sales were made.

If you select this option, you can view only the analysis of sales; the purchase analysis option will be disabled.

Purchases
Check this box, to view the purchases of all items during the selected period.
Purchases by Vendor
Check this box, to view the purchase details along with details of the vendors from whom the purchases were made.

If you select this option, you can view only the analysis of purchases; the Sales option will be disabled. This option is disabled if you select the Sales option.

Show Details
Check this box, to view the document details (such as the document number) in the Sales/Purchase Analysis browser. The Sales Measure field shows the measure used in the sales document. The Purchase Measure field shows the measure used in the purchase document. The quantity is shown as in the sales or purchase document.
Items Filter
Use the Item Filter button to filter the items to be displayed in the Sales/Purchase Analysis browser.

You can choose to display an analysis for only serialized items by category or by stock areas.
