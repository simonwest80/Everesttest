The Purchase Invoices Browser

The Purchase Invoices browser (path: Purchasing > Purchase Invoices) lists all the purchase invoices that have been created.

This browser contains the options common to all purchase document browsers.
Standard Browser Options

Filter records based on date

The Dates field in the Purchase Invoices browser enables you to filter purchase invoices based on their creation date. By default, the browser displays all documents created during the period specified in the Select the default date option based on which the records will be displayed in the purchase browser option (path: File > Setup > Preferences > Flow Control > Documents > Prompts > Purchase Document tab).
