Purchase Receipt Profile Options

Payment History
The Payment History browser (path: Options > Payment > Payment History) displays payment/refund details for the document. This option is enabled only if a payment is made on the purchase receipt.
Show Serial Number
The Show Serial Number option (path: Options > Item > Show Serial #) displays all the serial numbers that are attached to a selected item.
Serial Number History
This option (path: Options > Serial Number History) displays a history of movement of a serial number between different stock areas and the documents in which the movement was recorded. It displays the date on which a serial number was generated, the date it entered stock, the purchase order or purchase receipt on which it was generated, the stock area to which it was received, the sales order to which it is attached and the area it was moved to when it was attached etc.
Serial Number History Browser
Price Summary
This option brings up the Purchase Receipt Doc # window, which displays the original values that were saved after creating the purchase receipt as well as the values that were saved after the last modification. This option will be enabled only when the columns like Item Cost, Discount Value, Discount %, Tax Code, Job Code, and Add-on Cost of Purchase receipt are modified.
Show All Serial #
This option brings up the Serial # for All Items browser. All serial numbers attached to the document are displayed in this browser.
Show/Attach area-wise stock
This option (path: Purchase Receipt profile > Attach/Retrieve Serialnumber > Show/Attach area-wise stock) is available only for non-serialized items. Select this option to view the stock areas to which received quantities of the selected item are assigned.
Transfer/Assign new stock
Select this option to assign quantities in the purchase document. Everest displays the Assign Quantity dialog box.
Detach Serial #
This option allows you to detach the serial number attached to an item in the purchase receipt before it is saved. Enter the serial number in the Detach serial # field.

The other options available in the Purchase Receipt profile are:

  • View Tax Breakdown
  • Fill Sales Order
