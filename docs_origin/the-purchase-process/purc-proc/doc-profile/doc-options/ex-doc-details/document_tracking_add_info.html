Document Tracking

The Document Tracking option (path: Purchase Document profile > Options > Document > DocumentTracking) allows you to tag the document with the appropriate tracking selections for the tracking groups available for the document type. You cannot modify the selections if the tracking group is in the ‘View Only’ mode for that document type.

Document Tracking Groups

You can set document tracking preferences to ensure that the Document Tracking Groups browser automatically pops up when you create, edit or save a document, or only when you create a document (path: File > Setup > Preferences > FlowControl > DocumentTracking).
