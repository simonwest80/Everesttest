Document Printing Preferences Dialog Box

Use the Document Printing Preferences dialog box to define printing preferences for individual purchase/purchase return documents. By default, it displays the preferences set in the Reporting Preferences dialog box (path: File > Setup > Preferences > Defaults > Reporting Preferences > Print Settings tab > Documents tab). It contains the following fields:

  • Report Type
  • Company Information
  • Billing Address
  • Printer
  • Copies
  • Document Details
