Inline Notes

The Inline Notes option allows you to:

  • Enter/Edit the information in the Description column. The notes entered in such a manner will not wrap to any other line.

Edit an Item Description

  • Display each line entered in the Description dialog box, as a separate line in the purchase document.

You can specify packing instructions for an item.

To edit an item’s description, do the following:

  • From the purchase document profile, highlight the line item.
  • Select Options > Inline Notes. You will see the Description dialog box.
  • In the Description dialog box, enter/modify the item’s description.
  • Select File > Save and Close.
  • The Description/Note column displays the updated information.

To display each line in the description dialog box as a separate line in the document, do the following:

  • From the purchase document profile, select Options > Inline Notes.
  • You will see the Description dialog box.
  • Type in the description or note for the item.
  • Select File > Expand.
  • Each line in the Description dialog box is displayed in a separate line.

If you enter a note with 4 lines and extract it to a document, 4 more lines are added.

Timer

A timer can be invoked in the Description/Notes dialog box to record time spent on that item.

In case of service calls that you pay by the hour, you can select an item that represents the service cost. When you begin to speak to the vendor’s customer service, you can activate the timer.
