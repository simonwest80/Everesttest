Compute Freight Using the Everest Freight Estimator

To include freight on a purchase document, do the following:

  • From the purchase document profile select Options > Utility > Everest Freight Estimator. You will see the Everest Freight Estimator dialog box **

The freight is computed and automatically displayed in this dialog box if a destination zone is defined for origin and destination zip codes/addresses and a freight rate is set up for the combination of destination zone and delivery method (as selected on the document profile).

  • Click Update Document to apply the freight to the document.

The freight is added as a new line item to the document. The item specified in the Default item used for shipping charges option (path: File > Setup > Preferences > Defaults > Items) is used to account for the shipping charges.
