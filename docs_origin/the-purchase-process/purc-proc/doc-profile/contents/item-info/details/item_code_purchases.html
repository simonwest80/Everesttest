Item Code

The item code identifies an item in the purchase document. When an Item Code is selected:

  • Item information such as description, type, quantity, tax, unit of measure, price, discount value, discount%, tax amount, tax code, expense account, location/sub-location, job code, weight and amount are retrieved from the Item profile and displayed in the respective columns.
  • Upsell information entered in the Upsell information field (path: Inventory > Items > Options > Edit > In New Window > Notes tab) is displayed. You cannot edit the information in this window.

Item Notes

If you create a new item from the Item Code field (Click the drop-down button > right-click in the drop-down list > select New from the right-click menu options > mention details and save), this field defaults to the newly created item’s code.

The item selected could be of type Kit or Matrix. Items codes in a document are either selected or retrieved.
