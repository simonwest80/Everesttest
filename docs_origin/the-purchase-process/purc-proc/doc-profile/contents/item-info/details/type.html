Type

When you select the item code, the Type column is automatically updated with the icon assigned to the item. Item ** types are listed below:

  • - Non - inventoried
  • - Serialized
  • - Auto - serialized
  • - Inventoried (Non - serialized)
  • - Kit
  • - Matrix Group
  • - Matrix Item
  • (an ‘X’ on the top left corner) - Discontinued Item
