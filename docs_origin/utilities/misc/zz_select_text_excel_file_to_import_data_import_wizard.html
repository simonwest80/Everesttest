If you select Text File or Excel Spreadsheet file options in the Import File Type field, specify the name of the Text/Excel file that has to be imported. Click the browse button to specify the path and the file name.

The Update Information for existing items check box is displayed only while importing item data into Everest. Select this check box to update the relevant fields when importing items that already exist in your database. When you select this option, the Field Mapping section in the Data Import Wizard displays the Update column. You can update the existing item details in your database by selecting the required fields.
