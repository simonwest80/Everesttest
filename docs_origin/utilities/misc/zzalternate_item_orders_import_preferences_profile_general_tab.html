Specify an alternate item code. This is useful in cases where the item imported from your online store does not have an identical item code match in the Everest database. This is a mandatory field.

You can import a sales order for the item ‘ Books’ from your online store. This item may not have been created as an Item profile (path: Inventory > Items > from the Items browser, select Options > Edit > InNew Window > Item profile) in the Everest database. In such cases use the Alternate Item field to specify an alternate Item profile to import the item data.
