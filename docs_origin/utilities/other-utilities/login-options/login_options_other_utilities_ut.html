Login Options

The login options available through the status bar are also available through the Utility menu option. This enables Everest Developers to create programs which access these functionalities using the Everest Add-Ins architecture.

Active Users
Use this option to access the list of the users who have logged into Everest (path: Utility > Login Options > Active Users). You can also access this list by clicking on the User Count icon on the status bar. For multiple companies, this list shows the logged in users of the logged in company only.

List of Logged in Users

Change User
Use this option to access the Login Information dialog box (path: Utility > Login Options > Change User). Use the Login Information dialog box to change the logged-in user or to change a password. You can also view the currently logged in user.

The Login Information Dialog Box

Change Login Date
Use this option to modify the Everest login date (path: Utility > Login Options > ChangeLogin Date). You will see the Login Date dialog box.

Figure: Login Date Dialog Box
Select the default date for all documents and accounting entries and then click OK.
Session Lock/Unlock
This option toggles the global password feature on and off (path: Utility > Login Options > Session Lock/Unlock). Once the global password is enabled, the system requests a user name and password when you try to access any feature.
Reminders On/Off
This option toggles the display of reminder pop-ups on and off (path: Utility > Login Options > Reminders On/Off).
