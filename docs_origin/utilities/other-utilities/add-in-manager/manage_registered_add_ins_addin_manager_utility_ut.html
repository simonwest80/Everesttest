Manage Registered Add-Ins

You can manage the list of registered Everest add-ins on a particular workstation. The add-in list shows the current status of each add-in; descriptions of selected add-ins display in the Details section of the Add-Ins dialog box.

To access the Add-In Manager utility for a workstation, do the following:

  • From the main menu bar of the appropriate workstation, select Utility > Add-In Manager. You will see the Add-In Manager dialog box.
