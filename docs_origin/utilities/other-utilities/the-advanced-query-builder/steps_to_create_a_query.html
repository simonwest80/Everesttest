Steps to Create a Query

To create a query, do the following:

  • From the main menu bar, select Utility > Advanced Query Builder. You will see the Advanced Query Builder dialog box.
  • Enter a name for the query in the Filter Name field, so that the query can be saved and used at a later time.
  • Add table(s) from the Everest Schema Information (Tables) section and select the fields based on which you want to retrieve records.
  • Create joins or establish relationships between the fields, if necessary.
  • Specify the conditions for the query in the Filters tab.
  • Specify a primary key in the Browser Settings tab.
  • Select the searchable fields in the Field Settings tab, if you want to enable search tabs on these fields.
  • Select any actions that you would like to happen when you click on the field in the output filter in the Action column. Select any actions parameters required in the Action Parameter 1 and Action Parameter 2 columns.
  • Use the Parameter Information tab if you want to refine an interactive query.
  • Click Save to save your filter.

After creating the query, click the Find button to view the output. The result of the query will be displayed in the conventional Everest browser.
