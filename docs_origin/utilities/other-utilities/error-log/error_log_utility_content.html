Error Log

The Error Log utility (Utility > ErrorLog) displays a list of all the errors that have occurred in Everest. If your system has Everest installed as a server installation, errors occurred in your system as well as client systems connected to you are displayed. If your system has Everest installed as a client installation then only the errors occurred in your system are displayed.
