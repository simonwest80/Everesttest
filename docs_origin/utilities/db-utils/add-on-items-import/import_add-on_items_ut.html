Import Add-on Items

Use the Everest Add-on Items Import Wizard to select your source excel file in which you have stored data for add-on items.

To import Add-on items, do the following:

  • From the main menu bar, select Utility > Import > Add-on Items. You will see the Welcome screen of the Everest Add-on Items Import Wizard.
  • Click the Next button. You will see the Select Source File Type screen of the Wizard.
