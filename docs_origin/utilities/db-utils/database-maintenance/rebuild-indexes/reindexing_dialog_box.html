Reindexing Dialog Box

Log File Path
By default, the log file path is displayed in this field. You cannot edit the file path. After re-indexing, the log file is stored in a temporary folder. The log file displays the tables that are re-indexed in your database, and the start and completion time of the process.

Options

Run in Foreground
Select this option to display and view the progress of re-indexing.
Run in Background
Select this option to hide the progress of re-indexing.
