The Purge Browser

The Purge <Record name> browser displays a list of the records that can be purged.

Purge [Sales Quote] browser (path: Utility > Purge > Quote).

You cannot view the quote profiles from this browser.

Dates
Specify the period during which the records were created.

The From and To values are populated by default, based on the value specified in the Dates field. If you select the Custom option, you have to specify the date range.
