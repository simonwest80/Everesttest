Zip/FIPS Codes Import Dialog Box

Use the Zip/FIPS Codes Import dialog box to import zip codes. This dialog box displays the number of records you have successfully imported and also the number of records not imported due to errors.

City State file
Specify the name and path of the file you want to import. Use the icon to browse.
Number of Records Imported
This field displays the number of records that were imported.
Erroneous Records
This field displays the number of records that were not imported due to errors.
View Log
Click this button to open the log file containing details of the import process. This button is displayed only when the import process is completed.
