Import Custom Tracking

Use the Everest Custom Tracking Import Wizard to select your source excel file in which you have stored data for Custom Tracking.

It is mandatory that the customer/ vendor / item should exist in Everest.

To import Custom Tracking, do the following:

  • From the main menu bar, select Utility > Import > Custom Tracking. You will see the Welcome screen of the Everest Custom Tracking Import Wizard.
  • Click the Next button. You will see the Select Source File Type screen of the Wizard.
