Everest Custom Tracking Import Wizard

Everest allows you to import Custom Tracking details for Customers, Vendors and Items using the Everest Custom Tracking Import Wizard. This feature is especially useful to import a large number of custom tracking details at a time by importing an excel (.xls/.csv) file that lists all the custom tracking details.

You have 100 items and for each item you have defined cusatom tracking details. You can easily use the Everest Custom Tracking Import Wizard to import all the data.

Custom Tracking Import

Access the Welcome screen of the Everest Custom Tracking Import Wizard by selecting Utility > Import > Custom Tracking from the main menu bar.

Select Data to Import

Create a .xls (.csv) file with each custom tracking detail in a row and map the detail to the respective customer/vendor/item to the various fields in separate columns. Refer to the c:\Program Files\Icode\Everest\Client\Utilities\ImportTemplates folder to create the import file.

It is important that your import file is an exact copy of the sample sheet named “sample” in the template.

Clean up your excel source files before importing them. Remove rows that contain only empty cells by selecting the entire row and deleting it.

Everest allows you to import data from excel files of the following file formats:

  • .xls
  • .xlsx
  • .xlsm
  • .xlsb
