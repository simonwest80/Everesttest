Everest Related Items Import Wizard

The **Everest Related Items
Import Wizard** helps you import related items to create related items in Everest. Related items for kits and matrix groups can also be imported. Access the Welcome screen of the Everest Related Items Import Wizard by selecting Utility > Import > **Related
Items** from the main menu bar.
Related Items Import
The Item Profile - Related Items

Select Data to Import

Create a .xls (.csv) file with each related item in a row and map the related item to the various fields in separate columns.

Refer to the RelatedItem.xls file in c:\Program Files\Icode\Everest\Client\Utilities\ImportTemplates folder to create the import file.

It is important that your import file is an exact copy of the sample sheet named “sample” in the template.

You have five related items (Printer, Scanner, DVD Writer, CD Writer, and Web Camera) for the parent item computer of a particular model and you wish to import the related items. There are 25 models of the camera and each model has five related items. Create an excel with the data and import the related items with the Everest Related Items Import Wizard.

Clean up your excel source files before importing them. Remove rows that

contain only empty cells by selecting the entire row and deleting it.

Everest allows you to import data from excel files of the following file formats:

  • .xls
  • .xlsx
  • .xlsm
  • .xlsb
