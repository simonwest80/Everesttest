Delete Transactions - Manual Purchase Journals

Transactions can be deleted from the Purchase Journal profile only if the purchase journal has NOT been posted. You cannot void a purchase journal that accounts for manual purchase invoices/debit memos against which a payment has been made/received.

A journal is deemed to be voided if all transactions in the purchase journal are deleted. On voiding a journal, it’s status changes to Voided Journal.

System purchase transactions cannot be deleted.

You can delete a purchase transaction from any of the following locations:

  • The Purchase Journal profile
  • The Purchase Journal Transactions browser **
