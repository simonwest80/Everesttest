Delete Transactions - Purchase Journal Profile

To delete a manual purchase transaction from the journal profile, do the following:

  • From the main menu bar, select Accounting > Account Payable > Purchase Journal. You will see the Unposted Purchase Journals browser.
  • From the Unposted Purchase Journals browser, highlight the required journal and select Options > List Transaction. You will see the Purchase Journal Transactions browser.
  • From the Purchase Journal Transactions browser, highlight the required transaction and select Options > Edit > In New Window. You will see the Purchase Journal profile.
  • From the Purchase Journal profile, select the required transaction and select Options > Delete transaction. You will see the Confirm window.
  • Click OK. Everest informs you that the transaction has been deleted.
  • Click OK to return to the Purchase Journal profile.

Only manual purchase transactions can be deleted.
