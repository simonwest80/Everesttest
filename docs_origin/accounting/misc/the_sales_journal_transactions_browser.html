The Sales Journal Transactions Browser

This browser lists all the transactions in the journal and displays details such as the customer code, document type and amounts debited or credited.

The Sales Journal Transactions browser has the following additional options:

  • Split Journal - If an unposted journal contains too many transactions, you may split the journal. Splitting the journal moves the selected transaction(s) into a new journal.

Split Sales Journals

 The split journal option is not available for Posted Journals.

  • Delete transaction - You can delete a sales transaction by highlighting the required transaction and selecting this option.

Delete Transactions

 The delete transaction option is not available for Posted Journals.

  • Select / Unselect All - Use this option to select or de-select all the records in the browser. This is a toggle option.
  • List entry - The list entry option allows you to view the entire accounting entry for a selected transaction.

The options common to all browsers are also available in this browser.
