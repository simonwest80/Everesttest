The Purchase Journal Transactions Browser

The Purchase Journal Transactions browser is displayed when Options > List Transaction is selected from the Unposted Purchase Journals browser. This browser has the following options:

  • Split Journal - If a journal that is not posted contains too many transactions, you may split the journal. Splitting the journal moves the selected transaction(s) into a new journal.

: Split Purchase Journals

  • Delete transaction - You can delete a transaction by highlighting the required transaction and selecting this option.

Delete Transactions

  • Select/Unselect All - Use this option to select or de-select all the records in the browser. This is a toggle option.
  • List Entry - The list entry option allows you to view the entire accounting entry for a selected transaction.

Options common to all browsers are also available in this browser.
