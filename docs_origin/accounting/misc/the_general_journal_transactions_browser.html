The General Journal Transactions Browser

The General Journals Transactions browser is displayed when the Transactions option (path: Unposted General Journals Browser > Options > List Transaction) is selected. This browser lists all the transactions in the journal with the account code and amounts debited or credited.

This browser has the following options:

  • Select/Unselect All - Use this option to select or de-select all the records in the browser. This is a toggle option.
  • Split Journal - If a journal that is not posted contains too many transactions, you may split the journal. Splitting the journal moves the selected transaction(s) into a new journal.

: Split General Journals

  • Purge Transaction - You can delete a transaction from the General Journal Transactions browser.

Purge Transactions - Manual General Journal

  • List Entry - The list entry option allows you to view the entire accounting entry for a selected transaction.

The options common to all browsers in Everest are also available in this browser.
