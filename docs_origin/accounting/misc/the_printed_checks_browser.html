The Printed Checks Browser

The Printed Checks browser is displayed when checks have been printed. A check that is marked as "printed" cannot be printed again. Thus use this browser to identify the checks that have printed correctly. The checks that are not selected in this browser can be printed again.

The

Reference Number field in the 'Check Display Section' of the Payment/Receipt Journal profile is updated with check numbers of the checks that have been selected in this browser.
