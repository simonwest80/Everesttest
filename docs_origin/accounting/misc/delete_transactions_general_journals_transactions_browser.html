Purge Transactions - General Journal Transactions Browser

To purge a manual general journal transaction from general journal transactions browser, do the following:

  • From the main menu bar, select Accounting > General Journal. You will see the Unposted General Journals browser.
  • From the Unposted General Journals browser, highlight the required journal and select Options > List Transaction. You will see the General Journal Transactions browser.
  • From the General Journal Transactions browser, highlight the required transaction and select Options > Purge transaction. You will see the Confirm window.
  • Click OK. Everest informs you that the transaction has been deleted.
  • Click OK to return to the General Journal Transactions browser.
