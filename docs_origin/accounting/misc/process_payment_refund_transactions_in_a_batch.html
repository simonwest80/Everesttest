Process Payment/Refund Transactions in a Batch

To process the payment/refund transactions, do the following:

  • From the main menu bar, select Accounting > Accounts Receivable > Batch Payment Processing > New Sale or Refund/New Authorization/Use Prior Authorization. You will see the Batch CreditCard Process browser.
  • From the Batch CreditCard Process browser, select the relevant records.

Batch Credit Card Process Browser - Options

  • Select Options > Process.

Once the transaction has been processed successfully, the records are not displayed in the BatchCredit Card process browser anymore.
