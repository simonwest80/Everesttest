In Everest, all vendors do not have an individual accounts payable account. Instead, all payables are clubbed together under the Accounts Payable Control Account. Specify the control account to be used while accounting for vendor related transactions. The Accounts Payable control account is a liability account.

You can only select an account that has been set up in the base currency. You can modify this account only before it is used in a manual/system accounting entry.
