Check Formats

Everest provides the following check formats:

For Vendor Checks

  • Vendor Checks
  • Vendor Dot Matrix (348)
  • Vendor Dot Matrix (501)
  • Vendor Laser Check (052L)
  • Vendor Laser Check (800L)

For Customer Checks

  • Customer Checks
  • Customer Dot Matrix (348)
  • Customer Dot Matrix (501)
  • Customer Laser Check (052L)
  • Customer Laser Check (800L)

The default check format for vendor and customer checks has to be selected in Document Defaults (path: File > Setup > Preferences > Defaults > Documents > Internal Reports > POS/Bank Reports tab).
