The Posted Sales Journal Transactions Browser

This browser is displayed when the General Ledger (Detail) option is selected (path: Posted Sales Journals browser > Options > Transactions > General Ledger (Detail)).

The Posted Sales Journal Transactions browser lists all the transactions in the selected journal with details such as the document number, reference data, payment terms and amounts debited or credited.

The options common to all browsers are also available in this browser.
