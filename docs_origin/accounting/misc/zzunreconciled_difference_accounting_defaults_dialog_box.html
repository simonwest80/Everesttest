Specify the account that should be used as the clearing account while you set up the company in Everest initially. The balance of this account displays the value by which your initial import of accounts receivable, accounts payable and inventory is out of balance. The balance in this account is normally zeroed out when you complete setting up the company. The Unreconciled Difference account is an expense account.

 You can only select an account that has been set up in the base currency.
