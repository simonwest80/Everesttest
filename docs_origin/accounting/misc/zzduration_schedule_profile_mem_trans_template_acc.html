Select the length of time that the reminders should be sent in the Type field. Valid values include:

  • Date Range – Select this option to choose start and ending dates for the transaction reminders.
  • Indefinitely from Start Date – Select this option if you do not want an end date for the transaction reminders.
  • Maximum Occurrences – Select this option if you only want to be reminded a specified number of times.

You can select 12 for Maximum Occurrences if you want to pay monthly for only one year.
