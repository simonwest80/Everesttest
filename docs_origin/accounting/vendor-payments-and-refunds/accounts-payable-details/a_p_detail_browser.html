A/P Detail Browser

The A/P Detail Browser displays:

  • Purchase and purchase return documents that have outstanding balances. System and manual documents are included in this browser.
  • The total amount of the document and the balance that is payable/receivable on each document.
  • The vendor’s telephone number as defined in the Vendor profile.

  • Whether the vendor has put you on credit hold.

  • The number of days in time blocks (such as 0-30 days and 30-60 days) by which the documents are due.
  • Payroll transactions for which payroll journals are created.

Transactions pertaining to voided journals are not displayed in this browser.

This browser is displayed when the A/P Detail option is selected (path: Accounting > Account Payable > A/P Detail). The A/P Detail Browser has the following options:

Information Available

  • Open Current Document
  • Account Profile
  • Journal Transactions
  • Transaction History
  • Payment History
  • Serial number history

Other Options

  • Document Tracking
  • Multimedia Browser
  • Print Document

Options common to all browsers are also available in this browser.
