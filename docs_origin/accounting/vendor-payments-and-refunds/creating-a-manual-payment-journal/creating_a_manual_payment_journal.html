Create a Manual Payment Journal

To create a manual payment journal, do the following:

  • From the main menu bar, select Accounting > Account Payable > Payment Journal. You will see the Unposted Payment Journals browser.
  • Select Options > New > In New Window. You will see the Payment Journal profile.
  • Select the required vendor and enter the amount of payment/refund in the Amount field of the ‘Check Display Section’.
  • Select the documents on which the payment is made or refunded.
  • Save the transaction.

You can also create a payment journal from the Payment Journal profile by selecting Options > New Journal.
