Create a Manual Sales Journal

To create a manual sales journal, do the following:

  • From the main menu bar, select Accounting > Account Receivable > Sales Journal. You will see the Unposted Sales Journals browser.
  • Select Options > New > In New Window. You will see the Sales Journal profile.

You can also create a manual sales journal from the Sales Journal profile by selecting Options > New Journal.
