Add Transactions to a Manual Sales Journal

Entries can be added to an existing manual sales journal as required.

To add entries to an existing manual sales journal, do the following:

  • From the main menu bar, select Accounting > Account Receivable > Sales Journal. You will see the Unposted Sales Journals browser.
  • From the Unposted Sales Journals browser, highlight the required journal and select Options > List Transaction. You will see the Sales Journal Transactions browser.
  • From the Sales Journal Transactions browser, select Options > New > In New Window. You will see the Sales Journal profile.

A new transaction is automatically displayed for entry when you select this option. The next transaction number is automatically generated.
