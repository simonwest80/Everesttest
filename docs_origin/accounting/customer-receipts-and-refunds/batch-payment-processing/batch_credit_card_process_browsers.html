Batch Payment Process Browsers

The Batch PaymentProcess browser displays a list of pending payment/receipt transactions that must be processed. This browser is displayed when Accounting > Accounts Receivable > Batch Payment Processing > New Sale or Refund/New Authorization/Use Prior Authorization/Other is selected. When you select any of these options, the respective browser with records that apply only to that particular option is displayed. All these browsers display the same details.
