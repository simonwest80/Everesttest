The Multiple Receipts Wizard

The Multiple Receipts Wizard contains the following sections:

  • Welcome
  • Receipt Journal - Information
  • Receipt Information - Payment Method
  • Document Criteria - Payment Terms
  • Document Criteria - Document Aliases
  • Document Criteria - Document Dates
  • Conclusion

 You can navigate within the wizard by clicking the Next button or double-clicking a section in the left panel.

When you access the Multiple Receipts Wizard from the Customers browser, only the Receipt Information – Payment Method and Document Criteria – Document Dates sections display.

 We recommend you follow the steps in the wizard sequentially. This ensures that all the required data for processing multiple receipts is specified.
