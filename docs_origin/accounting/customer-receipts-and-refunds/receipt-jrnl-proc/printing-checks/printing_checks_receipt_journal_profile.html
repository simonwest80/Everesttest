Print Checks - Receipt Journal Profile

To print computerized payment checks from the receipt journal profile, do the following:

  • From the main menu, select Accounting > Account Receivable > Receipt Journal. You will see the Unposted Receipt Journals browser.
  • From the Unposted Receipt Journals browser, highlight the required journal and select Options > List Transaction. You will see the Receipt Journal Transactions browser.
  • From the Receipt Journal Transactions browser, highlight the required transaction and select Options > Edit > In New Window. You will see the Receipt Journal profile.
  • Select Options > Print Check(s). You will see the Check Printing dialog box.
  • After specifying the details, click Preview to view the check report from where you can print the checks by clicking the ** icon. Click **Print to directly print the checks without previewing them. The Printed Checks Browser will be displayed.
  • Select the checks that have printed correctly and click OK.

The transactions for which the checks have been printed are marked ‘printed’.
