Add E-mail Accounts

E-mail accounts are required in order to send e-mails to your customers, vendors or other external entities.

To add e-mail accounts, do the following:

  • From the main menu, select CRM > CRM Studio > Administrator. You will see the Everest CRM Studio Administrator dialog box.
  • From the dialog box, select Software Setup > Email Accounts option in the tree view.
  • Select File > New > Email Account. You will see the Everest CRM Studio - Email Accounts dialog box.
  • Add new email account.
