Modify a Process

The Process profile can be modified from any of the following locations:

  • The Process/Activities browser - Highlight the record and select Options > Edit > Process > In New Window.
  • The Process profile - Specify the process to be modified in the Process ID field.

All fields in the profile can be modified except the Process ID. You can also modify or change the sequence of activities from this profile.
