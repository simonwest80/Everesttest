Run a Report Batch

You can run the report batch from the Reports Manager window.

To run the report batch, do the following:

  • From the main menu bar, select Reports > Reports Manager. You will see the Reports Manager window.
  • From the Reports Manager window, select a report batch under the folder and choose File > Report Batch > Run. You will see the Report Batch Wizard – Parameter Assignment section.
  • Click Run to run the report batch.
