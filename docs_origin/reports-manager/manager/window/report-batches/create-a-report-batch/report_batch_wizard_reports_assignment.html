Report Batch Wizard - Reports Assignment

This section displays a list of all reports available for selection while creating a batch.

You can select Balance Sheet, Income Statement and Trial Balance from ‘Financial Reports’ and Activity from ‘Accounting Reports’ to create a batch called ‘Month End Financial’.

  • To add reports - Select the reports in the Available Reports panel and use , buttons to add reports to the Added Reports panel.
  • To remove reports - Select the reports in the Added Reports panel and use , buttons to move reports back to the Available Reports panel.
