Create a Third Party Catalog

To create a third party catalog, do the following:

  • From the main menu bar, select Inventory > Third Party Catalogs. You will see the Third Party Catalogs browser.
  • From the Third Party Catalogs browser, select Options > New > In New Window. You will see the Third Party profile.

You can create a third party catalog from any of the following locations:

  • The Third Party Catalogs browser - Select Options > New > In New Window.
  • An existing Third Party profile - Select Options > New.
  • Any profile/dialog box with the Third Party field - Select New from the right-click menu option available from the drop-down list of the field.
