Set up Related Items

To set up related item(s), do the following:

  • From the main menu bar, select Inventory > Items. You will see the Items browser.
  • From the Items browser, highlight the item for which you want to specify related item(s), and select Options > Related Items. You will see the Related Items browser.
  • From the Related Items browser, select Options > New > In New Window. You will see the Related Items profile.

You can also set up related items from the Item profile > Related Items tab.
