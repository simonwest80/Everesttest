The Brands Browser

The Brands browser (path: Inventory > Brands) displays a list of brands created for your company. The following option (path: Brands browser > Options) is available from the Brands browser:

E-Commerce
Use this option to access the E-Commerce setup - Brands dialog box wherein you can web-enable the items. From the E-Commerce setup - Brands dialog box you can:
  • Attach audio and video files.
  • Specify the images and banners to be displayed.
  • Specify other sales information for the items on the online shop.
  • Enter a detailed description for the item.
  • Specify hyperlinks for banners.
