Modify a Promotional Pricing Scheme

You can access the Promotional Pricing dialog box you want to modify from the following locations:

  • The Promotional Pricings browser - Select Options > Edit > InNew Window.
  • An existing Promotional Pricing dialog box - Specify the Discount code of the record you want to modify.

You can modify the following fields:

  • Start date
  • Start time
  • End Date
  • End time
  • All the enabled fields in the Select the pricing method section.
