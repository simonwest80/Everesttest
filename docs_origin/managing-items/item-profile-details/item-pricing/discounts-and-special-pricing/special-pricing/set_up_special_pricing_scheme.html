Set up Special Pricing Scheme

To set up a special pricing scheme, do the following:

  • From the main menu bar, select Inventory > Discounts > Special Pricing. You will see the Special Pricing browser.
  • From the Special Pricing browser, select Options > In New Window. You will see the Special Pricing dialog box.
  • Make your special pricing selections and click the Save button.

You can also set up a special pricing scheme from:

  • An existing Special Pricing dialog box - Select Options > New.
  • An existing Item profile - Select Sales Tab > Special Pricing button. ** On the **Special Pricing tab, select Options > New > In New Window.
