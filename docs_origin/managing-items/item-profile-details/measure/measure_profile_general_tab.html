The Measure Profile

Use the Measure profile (Path: **Inventory > Measure > Options >
New > In New Window) to define the parameters of a measure and specify the conversion factor between measures. Define the quantity of a measure using one of two Quantity Entry Methods. Select **Decimal if you want your quantity to be defined with decimal accuracy and select Fractional if you want your quantity to be defined in fractions.
