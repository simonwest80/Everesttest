Default Vendor

Use this option to find items by their default vendor.

If you purchase various computer assembly parts from a vendor and want a list of all the items purchased from that particular vendor, you can specify the vendor’s name for your search.

Click the browse button to view the Addresses browser from where you can select single or multiple vendors. The vendors you select from the Addresses browser are displayed in the Selection dialog box. If you select multiple vendors, the word ‘Multiple’ is displayed in the field in the form of a URL link.

Multiple Selection in Find Fields

Click the drop-down button to view the history of prior search parameter values.
