Manufacturers

The Manufacturers option allows you to search for an item by the manufacturer’s name that is entered in the Item profile. Click the browse button to view the Manufacturers browser from where you can select a single or multiple manufacturer(s).

If you select multiple manufacturers, the word ‘Multiple’ is displayed in the field in the form of a URL link by default. Click this link to view the selected manufacturers.
