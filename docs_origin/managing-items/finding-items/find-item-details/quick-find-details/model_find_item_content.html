Model

Search for items by the model specified in the Item profile.

The Item Profile - General Tab

To search for the model Nike Air, you can type the model name to search for matching items.

You can also search using partial values in the Model field. The search displays the list of items whose description begins with the partial value(s) specified.

If you enter the value ‘N’ in this parameter field, the search will yield all items with a Model description beginning with ‘N’.

Click the drop-down button to view the history of prior search parameter values.
