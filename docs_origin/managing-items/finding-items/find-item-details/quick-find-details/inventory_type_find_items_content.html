Inventory Type

The parameter values in the Inventory Type section are used to search for items by the type of inventory defined in the Item profile. You can further narrow down the criteria for an item type by selecting an inventory type.

If you select **Regular in the Item type section and Serialized in the Inventory type section, the search will yield all serialized items. Thus you are narrowing the search for regular items to regular - serialized items.**
