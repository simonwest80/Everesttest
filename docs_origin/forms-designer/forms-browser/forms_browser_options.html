This option enables you to quickly access a selected form in design mode to make any necessary changes.

Users with the appropriate security rights can rename a version of a form in the Forms Browser.

Access to Customized Profiles from a Central Location

To rename a version of a form, do the following:

  • Right click the version of a form to be renamed in the Forms browser and select Rename. The Rename Form dialog box is displayed.
  • In the Form Name field, enter the new name for the form.
  • Click OK to rename the form.

The Rename option is always disabled for standard versions.

Only users with the appropriate security rights can purge or remove forms from the Forms browser.

Access to Customized Profiles from a Central Location

The Purge option is always disabled for standard versions.

To purge one or more form versions, do the following:

  • Highlight one or more rows containing form versions to be purged.
  • Right click and select Purge. A confirmation dialog box displays a purge confirmation message.
  • Click Yes to purge the selected forms.

The Assign Forms option is used to assign form versions to users and groups.

The Export Forms option is used to export a customized form version to an external source.

The Import forms option is used to import  customized form versions that were created using Export Forms option.

Forms Browser Options

Four options are available from the Forms browser:

  • Design - Opens the selected profile in design mode
  • Rename - Opens a dialog box for renaming the version
  • Purge - Deletes the highlighted version or versions
  • Assign Forms - Enables user and group assignment for the selected profile form
  • Export Forms - Exports a customized form version
  • Import Forms - Imports customized form version
  • : When the standard versions of the profiles are the only versions available, the Rename, Purge, and Assign menu items are disabled.
  • : When a standard version is selected in the browser, Rename and Purge are disabled.
