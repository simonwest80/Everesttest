Add a New Tab to the Custom Fields Profile

To add a new tab to the Custom Fields profile, do the following:

  • Click inside the Characters tab. You should see eight black dots as shown below which indicate that the Tab Control is selected.

Figure: Tab Showing Tab Control Selected

  • For our example, right click the Characters tab and select New Page to create a new tab.

Figure: Select New Page to Create a New Tab

Figure: New Tab Created
