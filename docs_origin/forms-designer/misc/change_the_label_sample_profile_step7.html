Change the Label

To change the label on the new tab, do the following:

  • On the target form, click on the new tab page. Notice that the properties of this object are displayed in the Object Inspector dialog box.
  • In the Object Inspector dialog box - Caption field, type Items. The new tab is renamed to “Items”.

You decide to remove the Perishable field from the “Items” tab, remove all tabs except the Items tab, and add all mandatory fields to the target form.
