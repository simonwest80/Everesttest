Add Group Boxes in the Custom Fields Profile

To add group boxes in the Custom Fields profile, do the following:

  • Click the Group Box icon and click the location on the tab where you want it to appear as shown below. In this example, we are adding two group boxes, one for Dealerships and one for Distributors.

Figure: Group Boxes Added

  • To rename a Group box, click a group box and change the caption name in the Object Inspector to the Left. In our example we will rename both group boxes.

Figure: Rename Group Box (Dealerships)

Figure: Rename Group Box (Distributors)
