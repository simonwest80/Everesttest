Delete a Tab in the Custom Fields Profile

To delete a tab in the Custom Fields profile, do the following:

Click the Characters tab to select it. Notice that the page control is selected.

Figure: Characters Tab with Page Control Select

  • Click in the center of the Characters tab to select the Tab Control. Notice that this does not include the tab itself. We are going to delete a tab and do not want to delete the entire page.

Figure: Characters Tab with Tab Control Selected.

  • Press the Delete key to delete the tab.

Figure: Characters Tab Deleted
