First Name

This field allows you to search for customers using the first name as per the Customer profile. You can enter a partial value. The search results display a list of customers whose first name begins with the partial value specified.

 Type ‘Ar’ in this field and click the Find button. A list of all customers whose first names begin with ‘Ar’ is displayed.

Multiple selections are not allowed in this search field. Click the drop down button to view the history of prior search parameter values.
