Find Customers - Query

The SQL (Structured Query Language) tab displays queries. Queries are automatically generated when you specify search parameters in the Quick Find/More Choices/Custom Fields/Advanced Find tabs of the FindCustomers dialog box.

 The parameters in the Query tab will be populated automatically when you select an existing filter (that contains the relevant Query tab parameters).

You cannot edit or add new SQL queries from the Query tab. This is a display tab.
