The Address Profile - General (Customers)

The General tab allows you to specify address information for a customer.

General Information

  • Residential
  • Company
  • Title
  • First name and Last name
  • Middle name

Address Information

  • Address, City, State/Province
  • Zip/Postal code
  • Country

Contact Information

  • Telephone code
  • Abbreviation
  • Telephone - 1, Extension
  • Telephone - 2, Extension
  • Fax, Extension
  • Mobile Cell
  • Pager
  • E-mail ID
  • Web URL
  • Destination zone
  • Apply Masks
