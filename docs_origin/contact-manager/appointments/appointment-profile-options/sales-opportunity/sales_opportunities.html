Sales Opportunity

The Sales Opportunity option allows you to enter the details of the sales opportunity. This option is enabled only when you select a customer in the Account Type and Account Code fields for the appointment.

To create a sales opportunity in the appointment profile, do the following:

  • From the main menu bar, select Contact Manager > New Appointment. You will see the Appointment profile.
  • From the Appointment profile, select Options > Sales Opportunity. You will see the Sales Opportunity profile.

You can also create an appointment from the Calendar View profile (path: Contact Manager > Calendar View) by selecting Options > New > Appointment.
