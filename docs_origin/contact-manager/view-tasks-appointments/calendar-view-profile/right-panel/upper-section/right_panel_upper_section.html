Right Panel - Upper Section

The Upper Section of the Right Panel displays options with respect to the icons selected on the Left Panel. Tasks/appointments are displayed for each icon such as the User icon.

When you click the User icon on the Left Panel, the Upper Section of the Right Panel displays the User Code and User fields. In addition, if you click the Weekly tab, the tasks created for the user for the week are displayed.
