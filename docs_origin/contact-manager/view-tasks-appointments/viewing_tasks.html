Viewing Tasks and Appointments

Tasks can be viewed for the currently logged in user in the My Tasks browser.

To view a task in the My Tasks browser, do the following:

  • From the main menu bar, select Contact Manager > My Tasks. You will see the My Tasks browser.

Tasks for all users can also be viewed depending on the security rights that have been granted in the All Tasks browser.

Similarly, appointments for the logged-in user can be viewed in the My Appointments browser and appointments for all users can be viewed in the All Appointments browser.
