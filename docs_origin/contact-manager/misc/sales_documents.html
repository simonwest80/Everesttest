Sales Documents

Use this option to specify the sequence number of the sales document associated with the task/appointments in your search.

If you have a task associated with the sales invoice # 1459, you can view all tasks associated with this document.

To specify the sequence number of the sales document, do the following:

  • Click the browse button . You will see the Sales Document browser.
  • From the Sales Document browser, select the document(s) that you want to include in your search criteria.

If you do not select any sales document type, all the sales documents are listed in the Sales Document browser.

If you select ‘Sales Order’ and ‘Sales Invoice’ documents, all sales orders and sales invoices are listed in the Sales Document browser.

The drop-down button shows a history of earlier searches.
