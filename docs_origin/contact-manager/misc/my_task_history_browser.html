My Task History Browser

The My Task History browser displays a list of tasks for the logged in user, created for the selected document/customer/vendor/work order depending on the security rights that have been granted for the user.

Tasks with reminders set for the logged in user will have the icon displayed beside them. If a task is marked as Complete and Followup , the reminder is carried forward to the new task.

The browser options are given below:

  • Options
  • View
