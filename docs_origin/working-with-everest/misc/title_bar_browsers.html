Title Bar - Browsers

The Title Bar displays the name of the browser. Click on the browser icon in the Title Bar or right-click anywhere in the Title Bar to access the Title Bar menu options. Changes to the browser settings such as changing the size of columns, changing the display, etc., can be saved by selecting the Save Now or Save On Exit options in the Title Bar menu.

You can open more than one instance of the same browser. The title bar of an additional browser that you open, will display ‘+ 1’, beside the name of the browser. The following instance that you open, will display ‘+ 2’, and so on, for every additional instance you open.
