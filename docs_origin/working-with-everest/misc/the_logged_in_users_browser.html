The Logged-in Users Browser

The Logged-in Users browser is displayed when you click in the User Count icon on the Status Bar.

 Active and Inactive Users

The Logged-in Users browser has the following menu options:

  • Events
  • Standard Browser Options
  • List Groups
  • List Semaphore Locks
  • Clear all inactive users
  • Clear inactive user
  • Edit clear Lock Job Schedule
