List Groups

The List Groups option (path: Logged-in Users browser > Options > List Groups) displays the Groups browser. This browser lists all the groups that the user selected in the Logged-in Users browser belongs to.

To view the groups to which the user belongs, do the following:

  • From the Logged-in Users browser, highlight the user and select Options > List Group. You will see the Groups browser listing all the groups to which the user belongs.
