Filters - Top (Display) Panel

This panel allows you to select a filter related to the selected browser. The name and ID of the filter are displayed, along with the code of the user who created the filter. You can either modify or run the selected filter. Click a column header to change the parameter of the Search by <parameter> field. The options available from beside the Top (Display) Panel are:

Add
Click the drop-down arrow beside this button to display the types of filters that can be created. The options available are:
  • Classic Style: Select this option to create a Classic Style filter, by constructing a condition statement. This option displays the Specify Filter dialog box.
  • Expert Style: Select this option to create an Expert Style filter, by using SQL queries. This option displays the Expression Expert dialog box.
Delete
Click this button to delete the selected filter.
