Everest Windows Toolbar

The Everest Windows toolbar contains the following options:

Icon Name Description
Large icons The options are displayed as large icons in the right panel of Everest Windows.
Small icons The options are displayed as small icons in the right panel of Everest Windows.
List The options are listed vertically in the right panel of Everest Windows.
Details Details of each option are listed along with the name in the right panel of Everest Windows.
