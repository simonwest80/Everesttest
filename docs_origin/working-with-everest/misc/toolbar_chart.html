‘Title’ Dialog Box - Toolbar Options

The options contained in the chart toolbar are detailed below:

Open Chart Select this option to open an existing chart.
Save Chart Select this option to save the chart.
Copy to clipboard Select this option to copy the chart to the clipboard.
Gallery Select a chart type for the selected series or the entire chart. If a particular series is selected, a multi-type chart will be displayed in the chart area.


You cannot combine all chart types.
Color Click the drop-down button and select the required color. Drag the icon and click the required area to change its color.
Vertical grid Click this button to show the vertical grid on the current graph. This option is not available if you are currently viewing the ‘Pie’, ‘Doughnut’ and the ‘Pyramid’ chart. To switch the vertical grid off, click the button once again.
Horizontal grid Click this button to show the horizontal grid on the current graph. This option is not available if you are currently viewing the ‘Pie’, ‘Doughnut’ and the ‘Pyramid’ chart. To switch the horizontal grid off, click the button once again.
Legend Box Click this button to show or hide the legend. The legend is only printed if you have chosen to display the same on the window.


A legend explains what the colors/ designs used in the chart indicate.
Data Editor Click this button to show or hide the data based on which the chart is generated.
Properties Click this button to view and specify the chart properties. When you select this option you will see the Chart FX
Properties dialog box.
3D/2D Click this button to view the chart in 3D or 2D mode.
Rotate This option allows you to activate the 3-D rotated view for a 3-D chart. When you select this option you will see the Chart FX
Properties dialog box.
Z-clustered This option lets you display a series in different rows along the Z-axis. In this case, you will view a 3-series chart. Each data series will occupy one row of data and there will be 3 rows (Z-axis clusters) in the chart. AREA charts are always clustered, as there is no way to paint different series side by side. On the other hand, if you have a BAR chart, it is possible to paint bars side-by-side and not clustered in the Z-axis.
Zoom This option allows you to magnify a section of the chart. Click this button and then select the area to be magnified.
Print preview Click this button to view the chart. You will see the Page Setup dialog box. You can specify additional print settings here.
Print Prints the current graph. Click on this option to view the Print dialog box where the printer and the print settings have to be defined. The graph title, the graph and the legend (if you have the Series or Values Legend option on) are printed.
Tools Click this button to access the ‘Tools’ available with the Graph editor.


Values Legend - Choose this option to display the legend for the X-axis.


Series Legend - Choose this option to display the legend for the Z-axis.


Data Editor - Choose this option to view the data editor. The data editor displays the data based on which the chart has been generated.


Palette Bar - Choose this option to change the colors in the graph.


Pattern Bar - Choose this option to change the patterns of the bars in the graph.
