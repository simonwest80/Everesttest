Text Field - Custom Field

This specifies the field defined in any profile under Custom Fields - Character tab.

You can define the ‘Country of Manufacture’ as a text custom field in the Serial Number profile. To view all serial numbers originating from countries containing ‘United’ in the country’s name, select Contains in the Is field, and enter United in the Start Range field.

You can define ‘Task to be Reviewed by’ as a text custom field in the Task profile. To view all tasks that have to be reviewed by you, select Is (exactly) in the Is field, and enter your name in the Start Range field.
