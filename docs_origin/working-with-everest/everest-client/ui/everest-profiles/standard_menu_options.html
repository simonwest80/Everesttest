Standard Menu Options in Profiles

Common options available in a profile and their functionality are briefly explained below:

Button Name Description
New Use this option to create a new profile.
Save Use this option to save the changes made to the profile and continue working with it.
Save and close Use this option to save the changes made and close the profile.
Cancel Use this option to cancel the changes made and revert to the originally saved profile.
Help Use this option to get context-sensitive help for the specific profile.
