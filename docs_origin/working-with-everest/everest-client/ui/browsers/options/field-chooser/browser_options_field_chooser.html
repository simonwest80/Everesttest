Browser Options - Field Chooser

The icon allows you to hide/display columns in the browser. When this option is selected, the Customize window is displayed. Information specified in the relevant profile is displayed in this window.

Custom fields can also be displayed in the browser. You can then group or sort the browser based on these custom fields. Thus you can define the information to be displayed in browser.

 Custom Fields

In some browsers - for example, the Customers, Vendors and Document browsers - certain additional fields are displayed in the Customize window. Everest automatically computes the data displayed in these columns. Click here for details on these columns.
