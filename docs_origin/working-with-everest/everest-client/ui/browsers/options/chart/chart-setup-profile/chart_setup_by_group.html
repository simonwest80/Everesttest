Chart Setup - By Group

This tab allows you to specify the parameters for a 3D chart. A 3D graph or chart has an added third dimension called the Z-axis using which the user can view graphs that are three-dimensional and based on group totals.

A graph of yearly sales by sales rep. shows the value of sales in the Y-axis, the years in the X-axis and sales for each sales rep. for the year in the Z-axis.

Title
Enter the title of the graph in this field. By default the name of the browser is displayed in this field.
Y-axis
Specify whether the Y-axis should display the value of the records or the number of records in the selected criteria. If Value is selected, choose a numeric field for the field in the Y-axis column. If Record Count is selected, you cannot select a field for the Y-axis since the number of records for the criteria specified in the X- and Z-axis is displayed here.

Say you want to find out the number of sales invoices created by each sales representative in the fiscal year 2001. Select record count in the Y-axis. Enter the title for Y-axis in Enter title for Y-axis field. Select ‘Year’ as the field in the X-axis column and ‘Rep.’ as the field in the Z-axiscolumn. Click OK to view the report.

Y-axis column
Specify the field that is to be displayed in the Y-axis. If you have selected Record Count in the Y-axis field, enter the title of the Y-axis in this field.
X-axis column
Specify the field that is to be displayed in the X-axis.
Z-axis column
Specify the field that is to be displayed in the Z-axis.

**Selection criteria**

Selected records
Select this option to include only the selected records in the report. This option is enabled only if you have selected records in the browser before choosing the Chart Setup icon from the browser.
All records
Select this option to include all the records from the browser in the report.

**Other criteria**

Show total for Y-axis
Select this check box to display the total value of the field or parameter selected in the Y-axis.
Include zero values/counts for X-axis
Select this check box when you want to include X-axis fields that have a zero value or record count in the report.
Include zero values/counts for Z-axis
Select this check box when you want to include Z-axis fields that have a zero value or record count in the report.
