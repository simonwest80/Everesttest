The Chart FX Properties Dialog Box - Axes

Axis Selector
Select the axis to be formatted.
Visible
This option allows you to show or hide the axis in the chart area. When hidden, the axis will not display gridlines, check marks or labels.
Major Unit
Enter a value in this field to specify the interval of major check marks and grid lines (if they are displayed) in the selected axis. Labels on the selected axis will also be displayed according to the value specified here.
Show Gridlines
Select this check box to display grid lines at major intervals on the selected axis. If this check box is not selected, the grid lines are hidden.

Tick mark type

Specify the shape that marks the major grid lines (cross, inside, outside, none) for the selected axis.

Minor Unit
Enter a value in the minor unit box to specify the intervals for the minor check marks and grid lines (if they are displayed) for the selected axis.
Show Gridlines
Select this check box to display grid lines at minor intervals on the selected axis. If this check box is not selected the grid lines are hidden.

Tick mark type

Specify the shape that marks the minor grid lines (cross, inside, outside, none) for the selected axis.
