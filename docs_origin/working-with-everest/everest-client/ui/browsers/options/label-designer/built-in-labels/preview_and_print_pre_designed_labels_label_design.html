Preview and Print Pre-designed Labels

Everest provides you pre-designed labels which saves you the time and effort involved in creating new labels. You can preview and print pre-designed labels, using the Label Designer option.

To preview and print a pre-designed label, do the following:

  • From the required browser, click the icon and select Label Preview from the drop-down list. You will see a list of pre-designed labels available from the browser.

User-designed labels are also displayed in the list of labels available from the browser.

  • Select the required label. You will see the label displayed in the Print Preview dialog box.
  • To print the label, select Label Print from the drop-down list under the icon. From the list of labels, select the required label for printing.
