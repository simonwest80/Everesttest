Animate the Display of Submenus

You can change the way menus are displayed when you click on a command.

The Slide animation option causes the menus to be displayed with a sliding motion.

The following options are available:

  • None
  • Random
  • Unfold
  • Slide

To animate the way menus are displayed, do the following:

  • In the Options tab of the Customize dialog box, select the type of animation in the Menu Animations field.
