Customize the Display of the Main Menu Bar

You can modify or change the appearance of the main menu bar to suit your requirements.

You can customize your menu bar to display only those menus that you use normally, so that you can find and use them quickly.

By customizing, you can:

  • Add menus to and remove menus from the main menu bar
  • Add/remove toolbars
  • Add options to and remove options from the toolbars
  • Move the toolbars to a new location
  • Rename menus
  • Animate the manner in which the sub-menus are displayed
  • Display Large Menu Icons on the Toolbar
  • Customize Display of Icons and Text
  • Resize icons
  • Display only the recently used commands
  • Display the infrequently used commands after a short delay
  • Display tool tips for icons
  • Display shortcuts in the tool tips
  • Create or Change Shortcuts for Toolbars and Menus
  • Reset toolbars to the default setting
  • Group related buttons
