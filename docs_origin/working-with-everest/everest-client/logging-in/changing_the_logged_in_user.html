Change the Logged-in User

Everest allows you to change the logged-in user at any time. You can change the logged-in user from the Status Bar or by exiting the application and logging in once again.

To change the logged-in user from the Status Bar, do the following:

  • Click the Logged-in User icon on the Status Bar. You will see the Login Information dialog box.
  • Click the Change User button. The Change User dialog box is displayed.
  • Specify a user code and password.
