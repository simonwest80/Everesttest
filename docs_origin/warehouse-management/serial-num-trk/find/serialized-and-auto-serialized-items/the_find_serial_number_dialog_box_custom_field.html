Custom Field

The Custom Field tab is used to facilitate a search using the custom fields defined in the respective profiles of items, serial numbers, customers, vendors, documents and tasks and appointments.

If a serialized item’s expiry date is set in the custom field Date, you can search for the item using the Expiry Date field in the Custom Field tab.

Field Name/Is/Start Range/End Range
Details of the custom fields you specified will be listed in this display field with Field Name and Start Range and End Range dates respectively. The details of the custom field will be displayed only when you click Add to list.
Fields
Click the Fields button, and select the custom field type you want from the drop-down menu.
Add to list
Click Add to list to get details of the custom field displayed in the field provided with Field Name/Is/Start Range/End Range.
Remove
This option allows you to cancel the custom field specified for your search. Highlight the custom field you want to cancel and click Remove.
Field Name
Name of the custom field you selected is displayed in this field.
Is
This field explains the condition defined to choose the field range. Select the specification you want from the drop-down list.
Start Range
Specify start range, if the custom field you selected comes under the types Text, Numeric and Date.
End Range
Specify end range, if the custom field you selected comes under the types Numeric and Date.
