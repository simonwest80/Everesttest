Serial Number History

View History
The View history option allows you to view the serial number history of a selected serial number. Select Options > View history. You will see the Serial Number History browser.

The Serial Number History browser provides the history of stock movement for the serial number you selected. The history provides information on the date, the document number, the user who created the document and the stock area to which it was moved. You can view this document by selecting Options > Show Document.
