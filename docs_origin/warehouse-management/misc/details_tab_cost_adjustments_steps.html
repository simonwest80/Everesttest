Details Tab

User & Document Information

Created By
This field displays the name of the user who created the document.
Loc/Sub-Loc
The login location/sub-location is displayed by default when you open the Cost Adjustment profile. You can change the location/sub-location if required. This is a mandatory field.

Results

Total
The total value of all the cost adjustment amounts is displayed in this field. You cannot edit this field.
General Journal #
The General Journal number created to account for the inventory adjustment journal and the transaction number in the journal are displayed when an inventory adjustment document is saved. The transaction number appears next to General Journal number.
Purchase Receipt #
This field displays the link of the corresponding Purchase Receipt. You can click on the link to view the respective Purchase Receipt. This field will appear only if there is any Purchase Receipt reference.
Purchase Invoice #
This field displays the link of the corresponding Purchase Invoice You can click on the link to view the respective Purchase Invoice. This field will appear only if there is any Purchase Invoice reference.
