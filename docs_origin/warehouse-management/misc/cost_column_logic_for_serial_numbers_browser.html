Cost Column Logic for Serial Numbers Browser

The costs displayed in the Serial Numbers browser depends on whether the Extended Mode is enabled or not.

Extended Mode Enabled

If the Extended Mode (path: File > Setup > Preferences > Work Station Settings dialog box > Company Settings tab > check Enable extended mode) is enabled, the Serial Numbers browser displays both average and last cost of items without factoring.

Extended Mode Disabled

If the Extended Mode (path: File > Setup > Preferences > Work Station Settings dialog box > Company Settings tab > uncheckEnable Extended Mode) is disabled, ** the costs displayed in the **Serial Numbers browser depend on the flow control settings. These settings are defined in the Other Flow Control dialog box (path: File > Set up > Preferences > Flow Control > Other Flow Control dialog box > Item Cost/Sales Summary tab > Type of cost to be displayed in Inventory Browsers and Pricing Analysis field).
