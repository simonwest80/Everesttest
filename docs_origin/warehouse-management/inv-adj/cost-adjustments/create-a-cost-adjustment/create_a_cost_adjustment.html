Create a Cost Adjustment

To make cost adjustments, do the following:

  • From the main menu bar, select Inventory > Inventory Adjustment > Cost Adjustment. You will see the Cost Adjustment browser.
  • From the Cost Adjustment browser, select Options > New > In New Window. You will see the Cost Adjustment profile.

Cost adjustments can also be made from an existing Cost Adjustment profile by selecting Options > New.
