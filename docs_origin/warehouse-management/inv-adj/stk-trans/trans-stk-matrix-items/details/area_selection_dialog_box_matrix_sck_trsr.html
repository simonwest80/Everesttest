Bin Selection Dialog Box

Bin From
Specify the stock bin from which the item matrix is transferred.
Bin To
Specify the stock bin to which the item matrix is transferred.

If the quantity specified is more than one, the manufacturer’s serial numbers are automatically generated and assigned to the quantity specified.

Everest allows you to select only those stock bins for the logged in location, depending upon your flow control preferences (path: File > Set up > Preferences > Flow Control > Other Flow Control > General tab > Allow Users to access stock bins of option).

Lot No.
If the matrix item is associated with a lot, specify the lot number in this field.
