Purchase Invoice Dialog Box

Select the following parameters to view the Purchase Invoice report.

Vendor Code
Specify the vendor code whose purchase invoices/debit memos are to be included in the report.
Drop-Down List - Right-Click Menu Options
Purchase Rep
Enter the name of the user, who created the purchase invoice/debit memo. This helps in analyzing the employee’s performance.
Drop-Down List - Right-Click Menu Options

If the Purchase Rep field is left blank, all users (purchase representatives) are included in the report.

Terms to Include
Specify the payment terms, that the report should be based on. If the Terms to Include field is left blank, all payment terms are included.
Drop-Down List - Right-Click Menu Options
Terms to Exclude
Specify the payment terms that you want to exclude in the report. If you leave the Terms to Include field blank and specify a payment term to exclude, the report includes all the payment terms except the one specified in this field.
Drop-Down List - Right-Click Menu Options
Location/Sub-Location
Specify the location or sub-location to be included from the report. If this field is left blank, by default the payment terms for all locations or sub-locations will be included in the report.
Drop-Down List - Right-Click Menu Options
Date From
Specify the start date of the purchase invoices/debit memos to be included in the report. The date indicates the date of creation of the document.
Date To
Specify the end date of the purchase invoices/debit memos to be included in the report. The date indicates the date of creation of the document.

If the Date From and Date To fields are left blank, the current system date will be included in the report by default.

Document Type
Specify the type of document (purchase invoice and debit memo) to be included in the report.
Print in Base Currency
Select this check box to display the amounts relating to foreign currency purchases from vendors in your base currency.
