Specify the type of inventoried items that must be selected and displayed in the report.

Option Description
None Select this option if you do not want to specify
a stock type.
Normal Select this option to view the items normally used
in your business for trading or assembly.
In Process Select this option to view items that are in the
process of being converted to finished goods.
Repair Select this option to view items that are under repair
or are to be repaired.
Scrap Select this option to view items that are the leftovers
of a process that has been carried out and have no resale value.
Exchange Select this option to view items that have been received
for exchange.
Write off Select this option to view items that are written
off.
