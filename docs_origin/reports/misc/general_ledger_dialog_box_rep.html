General Ledger Dialog Box

Select the following parameters to view the General Ledger report.

Account Selection Method
Select the report filter method to filter the report.  You can choose either Range (a range of accounts) or Smart Filter. The Smart Filter allows you to use wildcards to select accounts based on your segments.
Account From
Specify the starting codes for the range of accounts to be included in the report.
Drop-Down List - Right-Click Menu Options
Account To
Specify the ending codes for the range of accounts to be included in the report.
If the Account From and Account To fields are left blank, all the account codes are included in the report.
Drop-Down List - Right-Click Menu Options
Smart Filter
Use the Smart Filter to select accounts based on your segments. Wildcards can be used in this field and up to 250 characters can be entered. Segment separators must be entered between segments. Enter a semi-colon (;) to separate accounts in the Smart Filter.
Smart Filter Wildcard Meaning
? One Character
* Multiple Characters
: Smart Filter
Segmented_Chart_of_Accounts
Location/Sub-Location From
Specify the starting location/sub-location code for the range of locations or sub-locations to be included in the report.
Drop-Down List - Right-Click Menu Options
Location/Sub-Location To
Specify the starting location/sub-location code for the range of locations or sub-locations to be included in the report.
All documents created for a location/sub-location that fall in the specified range will be listed in the report.
Drop-Down List - Right-Click Menu Options
Fiscal Year From
Specify the starting fiscal year for the range of fiscal years to be included in the report.
Drop-Down List - Right-Click Menu Options
Fiscal Year To
Specify the ending fiscal year for the range of fiscal years to be included in the report.
Drop-Down List - Right-Click Menu Options
Start Period
Select the starting accounting period for the range of accounting periods to be included in the report. The summary will include only amounts of those transactions belonging to the specified periods.
Drop-Down List - Right-Click Menu Options
End Period
Select the ending accounting period for the range of accounting periods to be included in the report. The summary will include only amounts of those transactions belonging to the specified periods.
If a financial year is divided into more than one period, the general ledger can be printed for different periods.
Drop-Down List - Right-Click Menu Options
Reporting Currency
Select the currency in which debit, credit and balance amounts are to be displayed.
Account Currency Select this option to display amounts in the currency specified for the account in the COA.
Base Currency Select this option to display amounts in the base currency.
Both Select this option to display amounts in the account and base currencies. The exchange rate between the two currencies is also displayed.
