Specify the document status for which you want to generate the report.

  • All - To include all document types in the report.
  • Closed - To display only closed purchase documents.
  • Open - To display open purchase documents.
  • BackOrders - To display back orders.
  • FullyOpen - To display only those purchase documents on which no payment is made.

This field is enabled only when you select Purchase Order or Debit Order in the Document Type field.
