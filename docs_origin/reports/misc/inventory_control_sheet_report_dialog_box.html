Inventory Control Sheet Report Dialog box

Select the following parameters to view the Inventory Control Sheet report:

Item From
Specify the starting code for the range of items to be included in the report.
Drop-Down List - Right-Click Menu Options
Item To
Specify the ending code for the range of items to be included in the report.
If the Item Code From and the Item Code To fields are left blank, all items are included in the report.
Drop-Down List - Right-Click Menu Options
Location From
Specify the starting code for the range of locations, to be included in the report.
Drop-Down List - Right-Click Menu Options
Location To
Specify the ending code for the range of locations, to be included in the report.
All documents created for a location/sub-location in the range specified are listed in the report.
Drop-Down List - Right-Click Menu Options
Stock Area From
Specify the starting code for the range of stock areas, to be included in the report. Only stock type of areas and stock bin type of areas are considered.
Drop-Down List - Right-Click Menu Options
Stock Area To
Specify the ending code for the range of stock areas, to be included in the report. Only stock type of areas and stock bin type of areas are considered.

If the Stock Area From and Stock Area To fields are left blank, all stock areas are included in the report.

Order By
Specify in which order you want the report to be displayed. The data is grouped by Category, Item, Brand or Stock area based on the option selected.
  • Category, Item, Stock area
  • Brand, Item, Stock area
  • Stock area, Category, Item
  • Inventory reports show values in the stock measure. Sub-totals do not display if the quantities are not in the same measure.
  • If the Stock Bin Code exceeds ten characters, the Stock Bin Code font is reduced to show all of the characters.
