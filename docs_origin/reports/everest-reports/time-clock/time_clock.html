Time Clock

The Time Clock report provides the attendance records for each user. This report is important for tracking the attendance history and calculating the regular and overtime pay for individuals. The report also includes the date, the name of the user, and the time-in and time-out details of the user.

To generate the Time Clock report, do the following:

  • From the main menu bar, select Reports > Time Clock > Time Clock. You will see the Time Clock dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
