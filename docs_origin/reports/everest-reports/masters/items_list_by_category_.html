Items List (By Category)

The Items List (By Category) report lists items grouped under their respective categories. The report includes item category, item codes belonging to that category, model number, complete description of the item, last cost and the calculated average cost of the item, purchase measure, sales measure and selling price of each item.

Item Categories

To generate the Items List (By Category) report, do the following:

  • From the main menu bar, select Reports > Masters > Items List (by Category). You will see the Items List (by Category) dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
