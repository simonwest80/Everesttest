Mailing List Export (With Filters)

The Mailing List Export (With Filters) report can be used for promotional campaigns and mail marketing. You can filter the report based on the parameters selected.

The report displays:

  • Account name of the customer.
  • First and last name for the customer.
  • Billing/Shipping/Contact addresses as specified in the Customer profile.

The report displays only those shipping and contact addresses which are specified as default addresses in the Addresses tab of the Customer profile.

To generate the Mailing List Export (With Filters) report, do the following:

  • From the main menu bar, select Reports > Masters > Mailing List Export (With Filters). You will see the Mailing List Export (With Filters) dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
