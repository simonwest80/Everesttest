Price Sheets By Category

Select the following parameters to view the Price Sheets By Category report:

Category From
Specify the starting code for the range of categories to be included in the report.
Drop-Down List - Right-Click Menu Options
Category To
Specify the ending code for the range of categories to be included in the report.
Drop-Down List - Right-Click Menu Options

: If both the Category Code From and the Category Code To fields are left blank, by default all the item categories are included in the report.

Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box, before printing
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
