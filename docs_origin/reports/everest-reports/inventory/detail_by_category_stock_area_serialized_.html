Detail By Category - Stock Bin

The Detail By Category - Stock Bin report displays both inventoried and serialized items available in your stock that were received during the period specified in the report parameters. The report is sorted based on stock bin, item category and the items in the category.

  • Some of the column details in the report are applicable only to serialized items.
  • Items that do not have stock in the specified stock bins do not feature in this report.

To generate the Detail by Category - Stock Bin report, do the following:

  • From the main menu bar, select Reports > Inventory > Detail By Category - Stock Bin. You will see the Detail By Category - Stock Bin dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
