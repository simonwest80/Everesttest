Vendor Tracking (Selection)

The Vendor Tracking (Selection) report is similar to the Vendor Tracking (Group) report except that it groups the vendors based on the tracking selection that belongs to a tracking group.

To generate the Vendor Tracking (Selection) report, do the following:

  • From the main menu bar, select Reports > Purchasing > Vendor Tracking (Selection). You will see the Vendor Tracking (Selection) dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
