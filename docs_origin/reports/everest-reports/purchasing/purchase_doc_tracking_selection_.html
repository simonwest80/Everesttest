Purchase Document Tracking (Selection)

This report is similar to the Purchase Document Tracking (Group Report) except that it displays the documents by each tracking selection that belongs to a group.

To generate the Purchase Document Tracking (Selection), do the following:

  • From the main menu bar, select Reports > Purchasing > Purchase Document Tracking (Selection). You will see the Purchase Document Tracking (Selection) dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
