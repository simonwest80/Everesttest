Sales Invoices

The Sales Invoices report lists sales invoices that are created during a specified period. The report also displays the following details:

  • Item details such as the code, description from the Item profile, and the quantity shipped.
  • The balance due and the number of days by which the amount is out standing.
  • The location/sub-location  to which the documents belong.
  • The profit/loss on each item and document.

: Sales Invoice

To generate the Sales Invoices report, do the following:

  • From the main menu bar, select Reports > Invoicing > Sales Invoice. You will see the Sales Invoices Report dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
