Sales Tax Details

The Sales Tax Details report displays the value of sales and returns and their associated taxes for both system and manual documents appropriate for the search parameters. This report also displays the taxable and exempt amounts on the sales invoices and credit memos.

Document Totals

If a Detailed report is generated, the item-wise breakup for each sales invoice and credit memo is also displayed.

This report displays document details for journals that are posted.

All relevant sales journals should be posted before running the report and filing the required returns with the Tax Authorities.

To generate the Sales Tax Details report, do the following:

  • From the main menu bar, select Reports > Invoicing > Sales Tax Details. You will see the Sales Tax Details dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
