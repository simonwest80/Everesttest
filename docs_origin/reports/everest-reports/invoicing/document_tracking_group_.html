Document Tracking (Group)

The Document Tracking (Group) report tracks sales documents based on the tracking groups defined for sales documents. This report is similar to the Purchase Document Tracking (Group) report.

Document Tracking

This report provides details on the status of sales documents. The report includes the document number, the date of the document, the customer against whom the document was raised, the sales representative that created the document, the status of the document, the amount of the document and the balance due by the customer.

To generate the Document Tracking (group) report, do the following:

  • From the main menu bar, select Reports > Invoicing > Document Tracking (Group). You will see the Document Tracking (Group) dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
