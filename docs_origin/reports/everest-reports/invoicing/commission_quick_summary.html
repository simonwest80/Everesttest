Commission (Quick Summary)

The Commission (Quick Summary) report lists all documents (sales invoices and credit memos) that qualify for commission. The amounts are grouped based on the sales representatives. Credit memo amounts are set off against the sales invoice amounts.

To generate the Commission (Quick Summary) report, do the following:

  • From the main menu bar, select Reports > Invoicing > Commission (Quick Summary). You will see the Commission (Quick Summary) Report dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
