Document Tracking (Selection)

The Document Tracking (Selection) tracks sales documents based on the tracking selections to which they are tagged. This report is similar to the Purchase Document Tracking (Selection) report.

Tracking documents

To generate the Document Tracking (Selection) report, do the following:

  • From the main menu bar, select Reports > Invoicing > Document Tracking (Selection). You will see the Document Tracking (Selection) dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
