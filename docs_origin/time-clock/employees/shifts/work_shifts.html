Shifts

Work shifts are the work timings assigned to employees. Shifts can be assigned to employees from any of the following:

  • The User profile (path: File > Setup > Users > Users).
  • The Other Defaults dialog box (path: File > Setup > Preferences > Defaults > Other Defaults).

An employee may be assigned to work on a day shift, a night shift or a noon shift.

If a work shift is not assigned in the User profile, the default work shift (if defined) in the Other Defaults dialog box is automatically assigned to the employee.
