Discontinue a Holiday

A holiday that is discontinued is not permanently ** deleted, but only deactivated. It can be activated when required.

To discontinue a holiday, do the following:

  • From the main menu bar, select Time Clock > Setup > Holidays. You will see the Holidays browser.
  • From the Holidays browser, highlight the record to be discontinued.
  • Select Options > Discontinue. You will see the Confirm window.
  • Click OK.
  • The selected record is discontinued and will not be displayed in the Holidays browser.
