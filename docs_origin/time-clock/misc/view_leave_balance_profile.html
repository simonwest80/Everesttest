View Leave Balance Profile

To define the Leaves Balance for an employee, do the following:

  • From the main menu bar, select File > Setup > Users > Users. You will see the Users browser.
  • Select the user from the Users browser, click on Options > Leave Balance to view the Leave Balance of <User Name> browser.
  • From the Leave Balance of , select **Options** > **Edit** > **In New Window**. You will see the [**Leave Balance** Profile](/versata-everest-docs/time-clock/employees/leave-management/leave-balances/the_leave_balance_profile.html)
