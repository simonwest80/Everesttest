Create an Everest Advanced Dashboard Sub-Site

Create an Everest Advanced Dashboard Sub-Site to display restricted information. Add only the users to this sub-site that should get access. For more information, see the Additional Information on Microsoft Windows® SharePoint® Services section.

Additional Information on Microsoft Windows® SharePoint® Services

To create an Everest Advanced Dashboard Sub-site, do the following:

  • From the Everest Advanced Dashboard menu bar, select Create. The Everest Dashboard Create Page is displayed.
  • In the Web Pages section, click the Sitesand Workspaces link.  The Everest Dashboard New SharePoint Site Page is displayed.

Everest Dashboard New SharePoint Site Page

  • In the Title field, enter a unique name for your sub-site.
  • In the Description field, enter a description for your sub-site. This field is optional.
  • In the URL Name field, enter the last part of the URL address for your sub-site, such as the sub-site name.
  • Click Create. The Template Selection page for your sub-site displays.

Template Selection Page

• In the Template list box, select Everest.

• Click OK. Your newly created sub-site displays.
