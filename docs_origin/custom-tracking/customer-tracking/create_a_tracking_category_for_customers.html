Create a Tracking Category for Customers

To create a tracking category for customers, do the following:

  • From the main menu bar, select Invoicing > Custom Tracking > Customer. You will see the Custom Tracking Setup - Customer browser.
  • From the Custom TrackingSetup - Customer browser, select Options > New > Category > In NewWindow. You will see the Tracking Category profile.

 You can also create a tracking category for customers from an existing Tracking Category (Customer) profile by selecting Options > New.
