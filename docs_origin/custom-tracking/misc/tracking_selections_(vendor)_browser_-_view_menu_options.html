Tracking Selections (Vendor) Browser - View Menu Options

Unique Match

Select this option to view only unique records that match the selection criteria.

All Matches

Select this option to view all the records that match the selection criteria.

Vendor Ace Watch is tracked to the Vendor Tracking Group Vendor Classification and tagged to the choices – Inventory/Product Suppliers and Maintenance. When you specify this tracking group and choices in the Selection Criteria for Vendor List dialog box, the All Matches option displays two records specific to each of the choices the vendor is tagged to; the Unique Match option brings up only one record of the tracking group unique to the vendor.
