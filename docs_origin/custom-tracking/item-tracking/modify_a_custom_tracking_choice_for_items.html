Modify a Custom Tracking Choice for Items

Everest allows you to modify tracking choices created for items. You can modify all fields except Code.

You can access the Custom Tracking Choice (Item) profile you want to edit from the following locations:

  • The Custom Tracking Setup - Inventory browser - Highlight the tracking group and select the choice. Then, select Options > Edit > Choice > In New Window.
  • An existing Custom Tracking Choice (Item) profile - Specify the Code of the record you want to modify.
