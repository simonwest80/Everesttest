The Custom Tracking - Inventory Browser

The Custom Tracking - Inventory browser (path: Inventory > Items > Options > Item Tracking) displays all the custom tracking groups and choices created for items. The options (path: Custom Tracking - Inventory browser > Options) available in this browser are:

  • Tree view
  • Notes Preview
  • Switch Pane
  • Prompt for Save
  • Custom Fields
