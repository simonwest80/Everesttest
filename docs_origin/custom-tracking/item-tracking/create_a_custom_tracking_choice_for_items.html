Create a Tracking Choice for Items

To create a tracking choice for items, do the following:

  • From the main menu bar, select Inventory > Custom Tracking > Inventory. You will see the Custom Tracking Setup - Inventory browser.
  • From the Custom Tracking Setup - Inventory browser, select a tracking group for which you want to generate choices. Select Options > New > Choice > In New Window. You will see the Custom Tracking Choice under Group profile.

 You can also create a custom tracking choice for items from an existing Custom Tracking Choice (Item) profile by selecting Options > New.
