Create a Tracking Category for Sales Documents

To create a tracking category for sales documents, do the following:

  • From the main menu bar, select Invoicing > Custom Tracking > Documents. You will see the Custom Tracking Setup - Document browser.
  • From the Custom Tracking Setup - Document browser, select Options > New > Category > In New Window. You will see the Tracking Category profile.

 You can also create a tracking category for sales documents from an existing Tracking Category profile by selecting Options > New.
