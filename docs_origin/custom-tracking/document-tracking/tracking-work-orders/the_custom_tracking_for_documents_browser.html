The Custom Tracking - Document Browser

The Custom Tracking - Document browser (path: Work Order profile > Options > Document > Document Tracking) displays all the custom tracking groups and choices created for the documents. The options (path: Custom Tracking - Documents browser > Options) available in this browser are:

  • Tree view
  • Notes Preview
  • Switch Pane
  • Prompt for Save
  • Custom Fields
