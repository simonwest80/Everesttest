Create a Custom Tracking Group for Work Orders

To create a custom tracking group for work orders, do the following:

  • From the main menu bar, select Inventory > Custom Tracking > Documents. You will see the Custom Tracking Setup - Documents browser.
  • Highlight the category, and from the Custom Tracking Setup - Documents browser, select Options > New > Group > In New Window. You will see the Document - Custom Tracking Group under category profile.

 You can also create a custom tracking group for work orders from an existing Document - Custom Tracking Group profile - Select Options > New.
