Create a Custom Tracking Choice for Work Orders

To create a custom tracking choice for work orders, do the following:

  • From the main menu bar, select Inventory > Custom Tracking > Document. You will see the Custom Tracking Setup - Document browser.
  • From the Custom Tracking Setup - Document browser, select a tracking group for which you want to generate choices. Select Options > New > Choice > In New Window. You will see the Custom Tracking Choice profile.

 You can also create a custom tracking choice for work orders from an existing Custom Tracking Choice profile - Select Options > New.
