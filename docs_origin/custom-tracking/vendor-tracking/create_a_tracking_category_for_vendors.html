Create a Tracking Category for Vendors

To create a tracking category for vendors, do the following:

  • From the main menu bar, select Purchasing > Custom Tracking > Vendors. You will see the Custom Tracking Setup - Vendor browser.
  • From the Custom Tracking Setup - Vendor browser, select Options > New > Category > In New Window. You will see the Tracking Category profile.

 You can also create a tracking category for vendors from an existing Tracking Category (Vendor) profile by selecting Options > New.
