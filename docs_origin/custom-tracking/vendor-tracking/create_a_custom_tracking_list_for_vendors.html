Create a Custom Tracking List for Vendors

The Custom Tracking List dialog box allows you to set parameters for generating the vendor list.

To create a custom tracking list for vendors, do the following:

  • From the main menu bar, select Purchasing > Custom Tracking > Vendor Selection. You will see the Custom Tracking List (Vendors) dialog box.
  • After specifying the selections, click OK. You will see the Selection criteria for vendor list dialog box.
  • After specifying the details, click OK. You will see the Tracking Selections (Vendor) browser displaying the list of vendors generated by Everest.
