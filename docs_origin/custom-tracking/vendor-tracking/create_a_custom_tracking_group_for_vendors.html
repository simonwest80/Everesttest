Create a Custom Tracking Group for Vendors

To create a custom tracking group for vendors, do the following:

  • From the main menu bar, select Purchasing > Custom Tracking > Vendors. You will see the Custom Tracking Setup - Vendor browser.
  • From the Custom Tracking Setup - Vendor browser, select Options > New > Group > In New Window. You will see the Custom Tracking Group under Category < > Vendors profile.

 You can also create a custom tracking group for vendors from an existing Custom Tracking Group - Vendor profile by selecting Options > New.
