Tracking Category Details for Vendors

The Tracking Category (Vendor) profile (path: Purchasing > Custom Tracking > Vendors > Options > New > Category > In New Window) allows you to create categories and sub-categories for classifying custom tracking groups.

Code
Code is the unique number assigned to the tracking category. It is automatically generated when you save a new tracking category.
Description
The description identifies the name of the tracking category.
Parent Category
Parent category identifies the category to which a sub-category belongs.
 Everest creates a default tracking category called ‘Default-Vendors’. All tracking groups and tracking choices, by default, is organized under this category, unless you create a new one to reorganize the tracking groups.
