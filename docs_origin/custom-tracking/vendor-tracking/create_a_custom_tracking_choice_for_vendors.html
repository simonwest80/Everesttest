Create a Custom Tracking Choice for Vendors

To create a custom tracking choice for vendors, do the following:

  • From the main menu bar, select Purchasing > Custom Tracking > Vendors. You will see the Custom Tracking Setup - Vendor browser.
  • From the Custom Tracking Setup - Vendor browser, select a tracking group for which you want to generate choices. Select Options > New > Choice > In New Window. You will see the Custom Tracking Choice under group profile.

 You can also create a custom tracking choice for vendors from an existing Custom Tracking Choice (Vendor) profile by selecting Options > New.
