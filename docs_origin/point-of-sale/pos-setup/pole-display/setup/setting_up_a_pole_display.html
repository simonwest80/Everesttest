Set Up a Pole Display

To set up a pole display, do the following:

  • From the main menu bar, select File > Setup > Miscellaneous > Pole Display. You will see the PoleDisplay browser.
  • From the Pole Display browser, select Options > New > In New Window/In Current Window. You will see the Pole Display Properties profile.

Pole Displays ** can also be set up from an existing **Pole Display Properties profile by clicking Options > New.
