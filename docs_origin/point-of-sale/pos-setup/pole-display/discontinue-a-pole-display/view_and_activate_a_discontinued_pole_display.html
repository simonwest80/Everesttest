View and Activate a Discontinued Pole Display

To view and activate a discontinued pole display, do the following:

  • From the main menu bar, select File > Setup > Miscellaneous > Pole Display. You will see the PoleDisplay browser.
  • From the Pole Display browser, select Options > View Discontinued. You will see the discontinued record in the Discontinued Pole Display browser.
  • From the Discontinued Pole Display browser, highlight the record.
  • Select Options > Activate. You will see the Confirm window.
  • Click OK. The activated record will be displayed in the Pole Display browser.
