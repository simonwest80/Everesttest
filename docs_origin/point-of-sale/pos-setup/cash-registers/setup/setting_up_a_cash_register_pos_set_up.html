Set Up a Cash Register

To set up a cash register, do the following:

  • From the main menu bar, select File > Setup > Miscellaneous > Cash Register. You will see the CashRegister browser.
  • From the Cash Register browser, select Options > New > In New Window/In Current Window. You will see the Cash Register profile.

Cash registers can also be created:

From an existing CashRegister profile by clicking Options > New.
