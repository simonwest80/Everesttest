View a Point of Sale (Sales Invoice) Profile

You can view the Point of Sale(sales invoice) profile from one of the following locations:

  • The Sales Invoices browser - Highlight the record and select Options > Edit > In Current/In New Window.

You can view only Point of Sale (sales invoice) documents from the Sales Invoice browser by selecting View > Point of Sale. All the other options under the View menu should be deselected.

  • The Sales Invoice profile - Specify the document number of the required invoice in the Document # field.
