Flow Control Preferences for Force Sale

Set the preference Force Sale in the When out of stock non-serialized items are inserted in a POS Document field of the Documents dialog box (path: File > Setup > Preferences > Flow Control > Documents > Miscellaneous > Point of Sale tab).

When the quantity scanned into the Point of Sale document exceeds the quantity available, either of the following happens based on the availability of stock:

  • The icon appears in the Type column, indicating that it is a force sale,

OR

  • You will see the Force Sale Options dialog box.
