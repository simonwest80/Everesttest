Print Pick Location Report

You can set preferences to print pick location of all items including uncommitted items or only committed items (path: File > Setup > Preferences > Defaults > All > Others tab > PickLocation drop-down). You can also use the pick location report as a packing slip.

To print a pick location report, do the following:

  • Select Options > Print Pick Location from the POS profile before saving the invoice. The Pick Location report that details the bin to pick the stock displays.
