Point of Sale - Action Menu

Access the Action menu options from the main menu of the Point of Sale (sales invoice) profile.

Figure: Point of Sale Document – Action menu

Quantity

Select Action > Quantity for the Enter Quantity dialog box, in which you can enter the quantity for the selected line item.

Return

Select Action > Return to enter all quantities as negative quantities in the document, signifying that these are returns.

Gift

Select Action > Gift to add a gift receipt to the next line item. The type of gift receipt is displayed as an icon in the status bar of the document.

Change Gift Receipt Printing Type

New Order After Save

This option is applicable only when you are in the Sales Order mode in the POS profile. By default this option is checked.

  • New Order After Save option is checked - When you save a sales order, the system defaults to the next new sales order profile.
  • New Order After Save option is not checked – When you save the current sales order, you have to select Options > New to open the next sales order.
