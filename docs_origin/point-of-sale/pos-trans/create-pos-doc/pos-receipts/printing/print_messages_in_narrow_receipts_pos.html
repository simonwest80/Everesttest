Print Messages in Narrow Receipts

Everest allows you to specify messages/tags in the following profiles:

  • The Message at end of receipt field of the Cash Register profile (path: File > Setup > Miscellaneous > Cash Register> Options > Edit > In New Window > Point of Sale tab).
  • The Narrow Receipt Message field of the Payment Method profile (path: File > Setup > Accounting > PaymentMethods > Options > Edit > In New Window > Miscellaneous tab).

If you specify messages/tags in the Payment Method dialog box AND in the Cash Register profile, BOTH the messages/tags appear in the narrow receipt that you print from a Point of Sale terminal.

The messages/tags are printed in the narrow receipt only if an internal report is used. To ensure that internal reports are used, select the Use Internal Report check box in the Cash Register profile (path: File > Setup > Miscellaneous > Cash Register > Options > Edit > In New Window > Forms tab).
