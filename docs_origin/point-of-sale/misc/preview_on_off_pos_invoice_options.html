Preview On/Off

The Preview On/Off option (path: Point of Sale (sales invoice) profile > Options > Preview On/Off) allows you to view the extended notes for a line item. When an inline note extends to more than one line, it appears in the line below the line item.

You can choose to view the inline notes or hide the inline notes. Click the button to view only the first line of the inline notes or the button to view the full content.
