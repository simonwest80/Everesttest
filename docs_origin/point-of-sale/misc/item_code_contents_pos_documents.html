Item Code and Description

When you select an item in the

Point of Sale (sales invoice) profile, item information such as description, type, price and UM are retrieved from the Item’s profile and displayed in the respective columns in the Item Details Grid.

Items Profile

You can select the item code manually or scan the item code into the system using a bar code scanner. If the item’s serial number is scanned, Everest automatically fills in the item code.

When an item is retrieved into a document

the up-sell Information (path: Inventory > Items > Options > Edit > In New Window > Notes tab) attached to the item is displayed.

The up-sell information is not printed along with the Point of Sale (sales invoice) profile. You cannot edit the information displayed in this window.
