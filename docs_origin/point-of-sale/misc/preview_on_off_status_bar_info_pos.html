You can enter inline notes for a line item in the Description/Notes column of the Item Details Grid. Click the button to view the full content of the inline notes for the line item. When the inline notes extend to more than one line, the inline note appears in the line below the line item. You can choose to view the inline notes or hide the inline notes. Click the button ** to view only the first line of the inline notes for the line item.
