Attach a Note to a Point of Sale Document

To attach a note to the Point of Sale document, do the following:

  • From the Point of Sale (sales invoice) profile click on the Note icon displayed beside the < New Sale > icon. You will see the Notes window.
  • Enter any information about the document.
  • Click Save.

You can view notes only on a saved Point of Sale (sales invoice) invoice. You can also edit the note attached to the Point of Sale document.
