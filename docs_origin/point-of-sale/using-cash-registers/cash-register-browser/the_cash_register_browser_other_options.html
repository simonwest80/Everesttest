The Cash Register Browser - Other Options

Lock/Unlock
If you are temporarily leaving the workstation, you can lock the cash drawer and unlock it later. Locking ensures that no unauthorized person accesses the cash drawer. Select the option Lock/unlock to lock and unlock the cash drawer.

To lock a cash drawer, do the following:

  • From the main menu bar, select Invoicing > Point of Sale/ File > Setup > Miscellaneous > Cash Register. You will see the POS Cash Registers browser/Cash Register browser.
  • From the POS Cash Registers browser/Cash Register browser, highlight the required cash register and select Options > Lock/Unlock.

A cash drawer that is locked displays the status ‘Locked’ in the Status column of the POS Cash Registers browser/Cash Register browser. You cannot create transactions from a cash register that has been locked. To create more transactions, unlock the register by selecting the Lock/Unlock option.

Declare
This option allows you to enter the collection values for the selected cash register at the end of a day. These values are then used to calculate discrepancies in collection and physically counted values.
