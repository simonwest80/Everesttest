Company Information - Address

Use this section to specify the address details of your company. The address specified in this section may be the address in which the company is incorporated or the address of the head office.

You can also set up the address information for your company from the **Company profile (path: File > Setup > Company > General tab).**

You cannot modify the company address in the **Company Creation Wizard. You have to open Jurisdiction profile (path: File >Setup > Accounting > Taxes > Jurisdictions), to modify the address.**

Address
Enter the primary address of the company being defined. This is a mandatory field.
City
Enter the name of the city that the company is located in using 1 to 40 characters. This is a mandatory field. The name of the city is automatically updated if a zip code is selected in the Zip field.
State
Enter the name of the state in which your company is located using 1 to 40 characters. This is a mandatory field. If your company is located in the United States, enter the 2-letter state abbreviation. The name of the state is automatically updated if a zip code is selected in the Zip field.
Zip
Specify the zip or postal code of the city your company is located in. When a zip code is selected, the City, State and Country fields are automatically updated. This is a mandatory field.
Country
Select the country that your company is located in. This is a mandatory field. The name of the country is automatically updated if a zip code is selected in the Zip field.
Telephone
Enter the telephone number of your company using 1 to 40 characters of the supported type.
Fax
Enter the fax number of your company using 1 to 40 characters of the supported type.

Click Next. You will see the Company - Logo section.
