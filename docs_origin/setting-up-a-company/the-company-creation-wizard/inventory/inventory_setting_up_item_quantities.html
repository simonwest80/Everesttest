Inventory - Setting up Item Quantities

Setting up item quantities comprises:

  • Indicating the default stock area for the initial item quantities.
  • Entering the initial inventory.
Default stock area for initial quantities
Before you specify the opening balance of items, select the appropriate stock area from the drop-down list. This default stock area is used unless the stock area is changed for individual items in the Initial Items Quantity dialog box.

To set up the quantities for inventoried items that are available in your stock, click the option Click here to set up item quantities. You will see the Initial Item Quantities dialog box.
