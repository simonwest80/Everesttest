The Location Profile - General

Use the General tab to specify the basic information about a location such as the name address and telephone number.

Code
Enter a unique code for the location using 1 to 15 characters. This is a mandatory field. Once saved, this field cannot be modified.
Code
Description
Enter an appropriate alphanumeric description for the location using 1 to 40 characters. This is a mandatory field.
Description

Apply mask

Use this option to apply the telephone, fax and mobile number formats that are specified for the country.

Telephone
Select this check box if the location’s telephone number format must correspond to the format as defined in the Country profile.
Mobile/Cell
Select this check box if the location’s mobile/cell number format must correspond to the format as defined in the Country profile.
Zip/Postal Code
Select this check box if the location’s zip/postal code format must correspond to the format defined in the Country profile.

Abbreviation
The International Organization for Standards has published a standard for abbreviations of countries. This field is automatically updated with the abbreviation of the country selected as per the ISO 3166 standard. This is a display field and cannot be edited.
