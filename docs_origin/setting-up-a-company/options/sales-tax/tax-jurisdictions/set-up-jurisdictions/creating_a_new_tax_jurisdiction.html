Create a Tax Jurisdiction

To create a tax jurisdiction, do the following:

  • From the main menu bar, select File > Setup > Accounting > Taxes > Jurisdictions. You will see the Jurisdictions browser.
  • From the Jurisdictions browser, select Options > New. You will see the Jurisdiction profile.

You can also create a Tax Jurisdiction from the following:

  • An existing Jurisdiction profile by clicking Options > New.
  • Any profile/dialog box that contains the Jurisdiction field by selecting New from the right-click menu options.

You cannot modify the company address in the Company Creation Wizard. You have to open Jurisdiction profile (path: File >Setup > Accounting > Taxes > Jurisdictions), to modify the address.
