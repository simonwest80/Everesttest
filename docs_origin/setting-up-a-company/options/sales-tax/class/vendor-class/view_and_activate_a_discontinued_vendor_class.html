View and Activate a Discontinued Vendor Class

To view and activate a discontinued vendor class, do the following:

  • From the main menu bar, select File > Setup > Accounting > Class > Vendor Class. You will see the Vendor Classes browser.
  • From the Vendor Classes browser, select Options > View Discontinued. You will see the discontinued record in the DiscontinuedCustomer Classes browser.
  • From the Discontinued CustomerClasses browser, highlight the record.
  • Select Options > Activate. You will be prompted for confirmation.
  • Click OK. The activated record will be displayed in the Vendor Classes browser.
