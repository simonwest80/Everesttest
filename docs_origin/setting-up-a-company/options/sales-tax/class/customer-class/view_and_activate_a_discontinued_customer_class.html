View and Activate a Discontinued Customer Class

To view and activate a discontinued customer class, do the following:

  • From the main menu bar, select File > Setup > Accounting > Class > Customer Class. You will see the Customer Classes browser.
  • From the Customer Classes browser, select Options > View Discontinued. You will see the discontinued record in the DiscontinuedCustomer Classes browser.
  • From the Discontinued CustomerClasses browser, highlight the record.
  • Select Options > Activate. You will be prompted for confirmation.
  • Click OK. The activated record will be displayed in the Customer Classes browser.
