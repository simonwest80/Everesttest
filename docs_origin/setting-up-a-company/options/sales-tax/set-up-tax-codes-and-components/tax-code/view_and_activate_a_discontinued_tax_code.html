View and Activate a Discontinued Tax Code

To view and activate a discontinued tax code, do the following:

  • From the main menu bar, select File > Setup > Accounting > Taxes > Tax Code. You will see the Tax Codes browser.
  • From the Tax Codes browser, select Options > View Discontinued. You will see the discontinued record in the Discontinued Tax Codes browser.
  • From the Discontinued Tax Codes browser, highlight the record.
  • Select

    Options > Activate. You will be prompted for confirmation.

  • Click

    OK. The activated record will be displayed in the Tax Codes browser.
