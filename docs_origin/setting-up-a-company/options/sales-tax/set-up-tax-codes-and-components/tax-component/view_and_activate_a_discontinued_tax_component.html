View and Activate a Discontinued Tax Component

To view and activate a discontinued tax component, do the following:

  • From the main menu bar, select File > Setup > Accounting > Taxes > Tax Components. You will see the Tax Components browser.
  • From the Tax Components browser, select Options > View Discontinued. You will see the discontinued record in the Discontinued Tax Components browser.
  • From the Discontinued Tax Components browser, highlight the record.
  • Select

    Options > Activate. You will be prompted for confirmation.

  • Click

    OK. The activated record will be displayed in the Tax Components browser.
