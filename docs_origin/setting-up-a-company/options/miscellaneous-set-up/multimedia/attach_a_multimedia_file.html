Attach a Multimedia File

To attach a multimedia file, do the following:

  • From the Multimedia Setup browser, select Options > New. A new record will be added to the browser.
  • Specify the required file by clicking the Multimedia File column and then the lookup button . The other columns in the browser are automatically updated.
  • Save the details.

You can drag and drop multimedia files on to the multimedia browsers in the Customer, Vendor, User, Item, Task, and Appointment profiles.
