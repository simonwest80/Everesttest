Custom Fields Browser

To access the Custom Fields browser, do the following:

  • For customers, vendors, items, tasks, documents, users, and serial numbers, select the Custom Fields option from the respective browsers. You will see the Custom Fields dialog box.
  • From the Custom Fields dialog box, select Options > Custom Field > List. You will see the Custom Fields browser.

This browser displays all the custom fields defined, the custom field title and properties. It also indicates whether the custom field is mandatory and if a default value is specified for it. If you access the Custom Field option from any browser/profile, the custom fields pertaining to that browser/profile are displayed in this browser.

If you access the Custom Field option from the Users browser/profile, all the custom fields defined for users are displayed in the Custom Fields browser.
