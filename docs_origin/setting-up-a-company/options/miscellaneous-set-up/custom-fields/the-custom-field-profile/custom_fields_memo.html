Custom Fields - Memo

Use the Memo tab of the Custom Fields dialog box to specify any notes, comments or other information for the record.

The field name in a sales invoice may be ‘Despatch Instructions’ and the field data may contain comments that there will be no despatch of goods on weekends.

You can define any number of memo fields in this tab.
