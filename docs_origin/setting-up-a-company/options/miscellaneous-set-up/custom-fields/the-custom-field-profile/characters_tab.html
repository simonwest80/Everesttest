Custom Fields - Characters

Use the Characters tab of the Custom Fields dialog box to enter alphanumeric information.

The field name in a **User profile may be ‘Emergency Contact’ and the field data ‘John Doe (912) 555-1212’.**

You can define any number of character fields in this tab. By default, only four character fields are displayed.

Create a Custom Field

You can also define selections that must be available for each character custom field.
