The Job Costing Profile - Status

Use the Status tab to specify the estimated income and revenue from the job. You can also specify whether the job is open for transactions or is closed.

Status
Specify whether the job has an ‘Open’ or ‘Close’ status.
Status Information
Estimated Revenue
Enter the approximate revenue expected from the project being defined.

The number of decimals displayed in the Estimated Revenue and Estimated Cost fields depends on the following specifications:

  • Currency profile - File > Setup > Accounting > Multi-Currency > Currency.
  • **Regional
    Settings Properties** dialog box – **Operating
    Systems > Regional Options > Currency** tab.
