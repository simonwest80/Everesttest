Create a User

To create a user, do the following:

  • From the main menu bar, select File > Setup > Users > Users. You will see the Users browser.
  • From the Users browser, select Options> New > In New Window. You will see the User profile.

You can also create a User from the following:

  • An existing User profile by clicking Options > New.
  • Any profile/dialog box that contains the User field by selecting New from the right-click menu options.
