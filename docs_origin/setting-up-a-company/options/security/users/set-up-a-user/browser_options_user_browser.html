The Users Browser

You can access the Users browser from File > Setup > Users > Users. The options in the Users browser are categorized under the following heads:

  • Information Available
  • Other Options

Information Available

  • Show Treeview
  • View Password
  • Employee History
  • Leave Balance

Other Options

  • Custom Fields
  • Multimedia
  • Rights
  • Rights - Users
  • Rights - Groups
  • Rights - All Rights
  • Events - History
  • E-mail

Options common to all browsers in Everest are also available in this browser.
