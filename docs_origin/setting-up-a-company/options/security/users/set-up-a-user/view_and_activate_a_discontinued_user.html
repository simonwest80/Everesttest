View and Activate a Discontinued User

To view and activate a discontinued user, do the following:

  • From the main menu bar, select File > Setup > Users > Users. You will see the Users browser.
  • From the Users browser, select Options > View Discontinued. You will see the discontinued user in the Discontinued Users browser.
  • From the Discontinued Users browser, highlight the user.
  • Select

    Options > Activate. You will be prompted for confirmation.

  • Click

    OK. The activated user will be displayed in the Users browser.
