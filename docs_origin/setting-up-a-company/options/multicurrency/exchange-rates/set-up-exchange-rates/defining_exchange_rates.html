Set up Exchange Rates

Exchange rates can be set up from the Exchange Rates browser.

To set up the exchange rates for a currency, do the following:

  • From the main menu bar, select File > Setup > Accounting > Multi-currency > Exchange Rates. You will see the Exchange Rates browser.
  • From the Exchange Rates browser, select Options > New > In New Window/In Current Window. You will see the Exchange Rates profile.
