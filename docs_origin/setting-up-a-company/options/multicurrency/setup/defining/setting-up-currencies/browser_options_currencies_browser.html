The Currencies Browser

The Currencies browser (path: File > Setup > Accounting > Multi - Currency > Currency) lists all the currencies that you have defined in the database.

The browser options are detailed below:

Exchange Rates
Use this option to view all the exchange rates that are defined for the selected currency. Selecting Options > Exchange Rates displays the Exchange Rates browser.

Options common to all browsers in Everest are also available in this browser.
