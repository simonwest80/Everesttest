Set up a Currency

To define a currency, do the following:

  • From the main menu bar, select File > Setup > Accounting > Multicurrency > Currency. You will see the Currencies browser.
  • From the Currencies browser, select Options > New > In New Window. You will see the Currency profile.

Currencies may also be set up from any profile/dialog box that contains the Currency field by selecting New from the right-click menu options.
