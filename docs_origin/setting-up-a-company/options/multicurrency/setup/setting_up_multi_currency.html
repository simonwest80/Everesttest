Set up Multicurrency

Setting up multicurrency in Everest involves:

  • Defining the base currency and other currencies
  • Setting up accounts that are to be maintained in foreign currency

A bank account in Paris maintained in EURO, is set up as a foreign currency account.

  • Setting up vendors and customers in foreign currency
  • Identifying multicurrency preferences for postings and rates to be used in transactions
  • Identifying the exchange rates from the base to other currencies or vice versa
