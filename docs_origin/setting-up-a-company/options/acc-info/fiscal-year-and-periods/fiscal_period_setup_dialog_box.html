Fiscal Period Setup

Use the Fiscal Period Setup dialog box to specify the details required to create a fiscal year.

Fiscal Year
Enter a unique alphanumeric code to identify the fiscal year. Use 1 to 15 characters to define the Fiscal Year code. This is a mandatory field. Once saved, this field cannot be modified.
Fiscal Period Setup Details
Description
Enter a description for the fiscal year, up to 40 characters. This is a mandatory field.
Fiscal Period Setup Details
Start Date
Specify the starting date for the fiscal year. This field is enabled only when you are setting up the fiscal year for the first time. The Start Date for the subsequent years is computed automatically based on the End Date of ** the last defined year. This is a mandatory field.
Fiscal Period Setup Details
End Date
Specify the ending date of the fiscal year. This is a mandatory field. Once saved, this field cannot be modified.
Fiscal Period Setup Details
No of Periods
Enter the number of accounting periods that make up the fiscal year. The number of periods should be such that, the start and end dates of all the periods differ at least by one day.
If the number of periods specified is 12, the start date and end date should span a minimum of 12 days.
Fiscal Period Setup Details
Status
This field displays the status of the fiscal year. By default all fiscal years are created with the status Open.
Fiscal Period Setup Details

Click Next to continue. Based on the information specified in the previous window, the periods in the fiscal year are calculated and displayed.
