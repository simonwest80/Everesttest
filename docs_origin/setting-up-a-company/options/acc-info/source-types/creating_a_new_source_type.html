Create a Source Type

To create a source type, do the following:

  • From the main menu bar, select File > Setup > Accounting > Source Types. You will see the Source Types browser.
  • From the Source Types browser, select Options > New > In New Window/In Current Window. You will see the Source Type profile.

You can also create Source Types from the following:

  • An existing Source Type profile by clicking Options > New.
  • Any profile/dialog box that contains the Source Type field by selecting New from the right-click menu option available from the drop-down list of the field.
