View and Activate a Discontinued Source Type

To view and activate a discontinued source type, do the following:

  • From the main menu bar, select File > Setup > Accounting > Source Types. You will see the Source Types browser.
  • From the Source Types browser, select Options > View Discontinued. You will see the discontinued record in the Discontinued Source Types browser.
  • From the Discontinued Source Types browser, highlight the record.
  • Select Options > Activate. You will be prompted for confirmation.
  • Click OK. The activated record will be displayed in the Source Types browser.
