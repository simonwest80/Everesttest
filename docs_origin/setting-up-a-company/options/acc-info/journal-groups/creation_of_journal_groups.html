Create a Journal Group

To create a journal group, do the following:

  • From the main menu bar, select File > Setup > Accounting > Journal Groups. You will see the Journal Groups browser.
  • From the Journal Groups browser, select Options > New > In New Window. You will see the Journal Group profile.

You can also create a Journal Group from the following:

  • An existing Journal Group profile by clicking Options > New.
  • Any profile/dialog box that contains the Journal Group field by selecting New from the right-click menu options.
