View and Activate a Discontinued Zip Code

To view and activate a discontinued zip code, do the following:

  • From the main menu bar, select File > Setup > Miscellaneous > Zip Codes. You will see the Zip Codes browser.
  • From the Zip Codes browser, select Options > View Discontinued. You will see the discontinued record in the Discontinued Zip Codes browser.
  • From the Discontinued Zip Codes browser, highlight the record.
  • Select

    Options > Activate. You will be prompted for confirmation.

  • Click

    OK. The activated record will be displayed in the Zip Codes browser.
