Set up a Processor

Set up a processor for each merchant account that you will use to process credit cards.

A business has two merchant accounts with their merchant bank, one for retail and one for online sales. They set up a processor in Everest for both merchant accounts.

If you are using multiple processors for ICVerify, assign a sequence number for each processor.

To set up a processor, do the following:

  • From the main menu bar, select File > Setup > Accounting > Processors. You will see the Processors browser.
  • From the Processors browser select Options > New > In New Window. You will see the Processors profile.

You can also create a processor in any of the following ways:

  • From an existing Processors profile by clicking Options > New.
  • From any profile/dialog box that contains the Processor field by selecting New from the right-click menu options.
