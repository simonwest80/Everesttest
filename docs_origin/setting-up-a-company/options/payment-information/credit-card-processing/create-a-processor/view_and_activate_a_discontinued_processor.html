View and Activate a Discontinued Processor

To view and activate a discontinued processor, do the following:

  • From the main menu bar, select File > Setup > Accounting > Processors. You will see the Processors browser.
  • From the Processors browser, select Options > View Discontinued. You will see the discontinued records in the Discontinued Processors browser.
  • From the Discontinued Processors browser, highlight the record.
  • Select

    Options > Activate. You will be prompted for confirmation.

  • Click

    OK. The activated record will be displayed in the Processors browser.
