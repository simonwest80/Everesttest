Group Rights

This option is used to view the security rights assigned to the groups to which the selected user belongs. Select this option and you will see the Group Rights dialog box.

The Group Rights dialog box contains the following sections:

  • Left Panel - The left panel displays a list of all the functions/user interfaces based on which the security rights have been grouped.
  • Assigned column - This column displays all the rights that have been assigned to the group.
  • Unassigned column - This column displays the rights that are not assigned to the selected group.
