Currency Details

Use the Currency profile (path: File > Setup > Accounting > Multicurrency > Currency > Options > Edit > In New Window) to define the general information for the currency such as the currency code and description, the format for the currency and the display information.

The information in the Currency profile has been classified under the following:

  • General Information
  • Format Specification
  • Quotation Information
  • Display and Printing Information
