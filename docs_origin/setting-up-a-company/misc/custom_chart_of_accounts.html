Custom Chart of Accounts

Everest also allows you to define your own chart of accounts. To do this, select Creating your own COA in the **Accounting

  • Chart of Accounts** section.

If you choose to create your own COA, you will have to manually select all the default accounts in the **Accounting

  • Defaults** section. If you select a predefined COA, Everest automatically populates the default accounts.

To create your own chart of accounts, do the following:

  • From the Accounting - Chart of Accounts section, select Creating your own COA.
  • Click Next. You will see the Confirmation dialog box.
  • Click Yes to proceed. You will see the Chart of Accounts - User-defined section.
  • Click Define COA. You will see an empty Chart of Accounts browser.
  • Create the required accounts.

Create an Account

If you opted to create your own Chart of Accounts while setting up your company, first you must create the parent accounts for the financial categories: Asset, Liability, Capital, Revenue and Expenditure.

 You must set up the first set of parent accounts in the following order only:

  1. Asset

  2. Liability

  3. Capital

  4. Revenue

  5. Expenditure

When setting up these accounts, the account Type is always General, the Parent account code is left blank, and the Level is 1.
