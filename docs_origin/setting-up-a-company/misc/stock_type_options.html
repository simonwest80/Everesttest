Stock Type Options

The Stock Type contains the following options:

Stock Type Description
Normal Select this option, if the items are normally used in your business for trading or assembly.
Repair Select this option, if the items are under repair or are to be repaired. You can repair these items in-house or send them to a third party.
In Process Select this option, if the items are in the process of being converted to finished goods.
Scrap Select this option, if the items are the leftovers of a process, and have no resale value. Scrap items usually have a negligible inventory value.
Exchange Select this option, if the items are received for the purpose of exchange with other items.
Write Off Select this option, if the stock bin is used to move items that are written off to a specified bin. For example, items that are obsolete, and are no longer of any use to the company may be written off.
Defective Select this option, if the stock bin is used to move defective items.
