Multimedia Setup Browser

You can view the multimedia files attached to items, customers, vendors, users, documents or tasks from the Multimedia Setup browser (path: Options > Multimedia from the respective browser).

The columns of the Multimedia Setup browser are explained in this section.

Extension
This column displays the file icons and the type of multimedia file (such as BMP, JPG, etc.).
Multimedia File
This column displays the multimedia file name.
Notes
You can specify special notes for the files in this column.

To specify notes, do the following:

  • Save the Multimedia file by selecting Options > Save.
  • Click the notes icon.
  • Click the drop-down list to see the notes dialog box.
  • Enter the notes and click the Save icon to save the changes.
File Size
This column displays the size of the selected file.
File Path
This column specifies the path where the file is located. A path is only stored if the file is not in the multimedia repository.
Web Enable
The Web Enable column specifies whether the image file can be viewed on the web store.
Sequence
The Sequence column specifies the order of appearance of multiple images in the Image Gallery for items on the web store. The sequence number will be visible if you access the Multimedia Setup browser from the Item profile – E-Commerce tab – Images tab to add your images manually. This is based on in the E-Commerce preferences (path: File > Setup > E-Commerce > Preferences > E-Commerce Preferences profile > Display tab > ItemSettings tab > Clear the Search images folder for additional items check box).
Copy to Multimedia Repository
The multimedia repository is a directory in which the multimedia files are located. The check box is enabled only when the Multimedia Repository is specified in the Workstation Settings dialog box (path: File > Setup > Preferences > Workstation Settings > Company Settings tab) or the Global Multimedia Repository is specified in the default preferences (path: File > Setup > Preferences > Defaults > Other Defaults).
Users with the appropriate security rights can change the Copy to Multimedia Repository setting. If the Copy to Multimedia Repository field is checked then the multimedia file is stored in the multimedia repository and the File Path column will not display a path for the multimedia file. If you change the Multimedia Repository path, all files in the Multimedia Repository can be accessed from the new location provided that they are manually moved to the new location. If the Copy to Multimedia Repository field is cleared, then the multimedia file is not stored in the Multimedia Repository and the File Path column displays a path for the multimedia file.
