Stock Transfer - Assembly 1

Use this tab to specify the default stock area into which item components and finished goods are transferred/received, during the manufacture of an assembly item.

Select stock area to issue inventoried components of assembly on a work order

Work Order
Select the default stock area from where inventoried components are issued to assemble an item. This is usually a warehouse or a store on the shop floor.

Select stock area to receive back inventoried components of assembly previously issued on a work order

Assembly Information

In Process

Assembly Information

Work Order
