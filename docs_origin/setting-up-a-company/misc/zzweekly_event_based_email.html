Select the Weekly option to set a weekly schedule for sending e-mails. When Weekly is selected the Every <number of > Week(s) field and the week day check boxes are enabled. Select the required fields based on your schedule.

If you want e-mails to be sent every alternate Friday, enter 2 in the Every <number of > Week(s) field and select the Friday check box. The maximum number of weeks that can be specified is 100.
