Enter the name with which the company has been incorporated. This name will appear on all statutory forms and returns, and on all reports and other forms generated from Everest. Use 1 to 50 characters to define the legal name. This is a mandatory field.

If this name is used for an existing company, enter a different name in this field.

If at a later date, you change the **Legal Name of the company and click Done, you will have to obtain a new RegistrationCode from Everest Software.**
