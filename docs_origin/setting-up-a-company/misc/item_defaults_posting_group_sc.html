Item Defaults - Posting Group

Use the Item defaults-Posting Group tab to configure the default settings for the accounting information related to the item i.e., Posting group field. This field is optional.

Accounting information

Posting Group
Specify the posting group to be displayed by default in the Posting Group field, when you create a new item. The Posting Group options available here are derived from Inventory Posting Group of Posting Groups browser.
