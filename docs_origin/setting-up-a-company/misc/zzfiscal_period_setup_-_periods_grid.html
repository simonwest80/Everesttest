Based on the information specified in the previous window, the periods in the fiscal year are calculated and displayed.

The Description column displays the name of the period. You can modify this column, as required, using 1 to 40 characters.

The Start Date for the first period cannot be modified. You can change the end dates for all the periods except the last one. When you change the end date of a period, the start date of the subsequent period is automatically computed.

You can also change the status ** of a period. Click in the **Status column to view a drop-down menu. Select any of the three statuses from the list: Open, Active or Lock.

Click Apply to save the changes.
