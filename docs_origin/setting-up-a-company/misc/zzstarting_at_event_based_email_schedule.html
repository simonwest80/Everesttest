You can also set the duration during which the mail server must verify the mails in queue. Enter the start time when the mail server should start scanning the mails in queue. This option is enabled only if the Occurs every option is selected in the Daily frequency section.

In the above instance, if you select the start time as 10.00 a.m. and end time as 6.00 p.m., the mail server will trigger the mails once in 4 hours on every third day, i.e., once at 2 p.m. and again at 6.00 p.m.
