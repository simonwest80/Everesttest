Leave Balance

Select this option to open the Leave Balance of <User Name> browser where you can view the leave available and the leave balance for a user.

The Opening Leaves column displays the number of leaves carried over from the previous year(s).

The Leaves Availed column displays the number of leaves availed by the user till date for the current year.

The Leaves Credited column displays the number of leaves that have been credited to the user’s account.

Two days of vacation leave may be credited to a user’s account at the end of every month.

Leave Management

Each employee should have an opening balance of leaves, for leaves to be carried forward. You can create an opening balance for an employee from the Leaves Balance profile. (path: File > Setup > Users > Users > Options > Leave Balance > Options > New > In New Window.)
