Create a Stock Bin

To create a stock bin, do the following:

  • From the main menu bar, select Inventory > Stock Areas. You will see the Stock Areas browser.
  • From the Stock Areas browser, highlight a row where the Stock Level is “Bin” and select Options > New > In New Window. You will see the Stock Bin profile.

You can also create Stock Bins from the following:

  • An existing Stock Bin profile by clicking Options > New.
