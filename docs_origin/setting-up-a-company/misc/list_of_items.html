List of Serialized Items

This option provides a list of all the serialized inventoried items contained in the selected stock area. Selecting this option displays the Stock Area Items browser. This browser provides details of the item’s manufacturer number and purchase details such as the cost, purchase receipt number and entry date.

This browser contains the following option:

Lots
If the item has any lot numbers assigned to it, the Item Lot Stock Status browser is displayed.
