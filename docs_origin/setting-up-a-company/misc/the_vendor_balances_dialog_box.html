The Vendor Balances Dialog Box

The Vendor balances dialog box lists all the vendors who are created in your database. You can enter the opening balances for vendors from this dialog box.

To enter the opening balance for a vendor, do the following:

  • From the Vendor Balances dialog box, select the required vendor.
  • Select Open Documents from the right-click menu. You will see the Open Documents dialog box.
  • Enter the details, and click OK.

When you click OK in the Open Documents dialog box, an accounting entry is automatically created where the ‘Accounts Payable Control account’ is ‘credited’ and the ‘Unreconciled Difference account’ is ‘debited’ with the value of the payable. For further details on account opening balances click here.

  • Continue the process of entering the open documents for the vendors. Click OK to return to the [**Vendor
