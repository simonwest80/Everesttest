Group Rights

The Group Rights dialog box displays all the rights that are/are not assigned to the selected group. For your convenience, the security rights are grouped based on the various functions in Everest such as “Accounting” and “Invoicing”, which are displayed in the left panel.

The Group Rights dialog box contains the following sections:

  • Left Panel
  • Rights assigned to group column
  • Rights not assigned column
