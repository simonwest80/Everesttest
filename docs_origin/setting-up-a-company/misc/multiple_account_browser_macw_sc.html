Multiple Account Browser

Once you have entered the necessary information to set up your accounts in the Multiple Account Creation Wizard, do the following:

Multiple Account Creation Wizard - Segment Selection Range

  • Click Preview to preview the accounts to be created. The Multiple Account browser is displayed.
  • In the Description fields, change the description, if necessary.
  • Select the check box of each of the accounts you want to create or use the Select All Accounts option.

: To deselect accounts, uncheck the box for each account you no longer want to create or uncheck the Select All Accounts option.

  • After reviewing the accounts to be created, click Save. The Confirmation dialog box is displayed.
  • Click Yes to confirm the number of accounts to be created. The selected accounts are created.
  • Click Back to continue creating accounts.
  • Click Close when you are finished creating multiple accounts.
  • If you accessed the Multiple Account Creation Wizard from the Company Setup Wizard, you will see the Chart of Accounts - User Defined screen. Click Next. You will see the Chart of Accounts - Default screen.
