Comparing System Database with Everest Defaults

This section of the wizard compares the latest EVEREST_SYSTEM _DEFAULT database supplied by Everest Software with the existing EVEREST_SYSTEM database.

You can see the status of the comparison. A red check mark appears against each section when the comparison of that section is complete. Click Cancel to stop the process.

Click Next. You will see the System Database Update section of the wizard.
