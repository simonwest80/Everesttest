Eproxy Utility

The eProxy.dat file (path: Icode >Everest >Client folder >eProxy.dat) on all remote client systems which connect to the Everest application server, must contain a valid domain user name and password, in addition to the IP address and domain name.

To create a new eProxy.dat file, or modify the existing file for a client system, do the following:

  • From the Everest Client folder > Utilities, open EproxyUtility. You will see the EProxy Utility dialog box.
