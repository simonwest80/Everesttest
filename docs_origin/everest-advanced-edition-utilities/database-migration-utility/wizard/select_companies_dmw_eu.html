Select Companies

Use the Select Companies dialog box to display the list of companies available from the selected Database. You can select multiple companies you want to migrate, by checking the boxes on the first column. You will not see the Select Companies dialog box, if the source selected is Database File.

: You cannot select the Companies that are shown in red with an icon instead of a checkbox.

  • Click Next. You will see the Progress Indicator which indicates the Database is being migrated.

On completion of Database Migration, you will see the Conclusion dialog box.
