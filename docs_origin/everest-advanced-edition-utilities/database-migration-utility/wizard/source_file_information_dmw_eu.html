Source File Information

Use the Source File Information dialog box to select the Source Database File.

: Everest Database Migration supports .mdf files from SQL Server, back up Everest files or any detached database for migration.

Everest Database File
Click the button to the right of the field to select the Source Database File to be migrated.
Everest Database Log File
This is optional. You can select the log file associated with the Everest database file.
**Company Code**
Enter the Company Code for the company in the Source Database File.
  • Click Next. You will see the Destination Database Information dialog box.
