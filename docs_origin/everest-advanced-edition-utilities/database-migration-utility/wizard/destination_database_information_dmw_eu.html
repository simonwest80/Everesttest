Destination Database Information

Use the Destination Database Information dialog box to select the destination database.

Database Server Name
Select the Name of the Destination Server from the drop-down list.

: The Source Information Server and Destination Information Server cannot be the same if you are migrating a database server.

User Name
This defaults to the administrator’s user name.
Password
Enter the system administrator password.
  • Click Next. If the source to be migrated is Database file, you will see the Progress Indicator which indicates the Database is being migrated.

On completion of Database Migration, you will see the Conclusion dialog box.

: If the source to be migrated is a Database Server, You will see the Select Companies dialog box.
