Accounting Entries - Abnormal Loss/Gain

The accounting entry for abnormal loss or gain is created along with the entry for the assembled item. This journal entry occurs as part of the journal entry created to account for the assembled item.

The accounting entry created for an abnormal loss/gain is:

Account Description Path Debit/


Credit		 Amount
Abnormal Loss account The account selected in the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting > Accounts tab > Abnormal
Loss field).		 Debit  
Work in Progress account The account selected in the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting > Accounts tab > Work
In Progress Account field).		 Credit Value of work-in-process
Abnormal Gain account The account selected in the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting > Accounts tab > Abnormal
Gain field)		 Credit
