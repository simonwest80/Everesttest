Work Order Status

The status of a work order is displayed beside the Document # field. A work order can have any of the following statuses:

Status Assigned to the Work
Order
Not Started When a work order is created and saved.


Create a Work Order
In Process When production is in process.


Work-In-Process
Build When the item is assembled/manufactured.


Build Work Order
Cancel When the work order is cancelled. Work orders can be cancelled only in the ‘In Process’ stage.


Cancel Work Order
Void When the work order is voided. Work orders can be voided in the ‘Not Started’ and ‘In Process’ stages.


Void Work Order
