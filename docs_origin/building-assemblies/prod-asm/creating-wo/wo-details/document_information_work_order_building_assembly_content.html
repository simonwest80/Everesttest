Document Information

Document information for a work order are listed below:

Document Alias
The Document Alias button allows you to set a document alias for the work order. You can define different aliases for different types of jobs. You can assign a different document alias to a work order, until you select the In Process option.
Assign another Document Alias
Document Alias
Document Notes
Click the Notes button to enter notes for the work order in order to track important messages or information pertaining to the work order. This is not printed with the work order. You can enter notes for work orders until you select the Build option.

View and Edit Document Notes

Work Order Status
The status of a work order is displayed beside the Document # field.
Work Order Status
Document Number
This field displays the document number assigned automatically to the work order when it is saved. You cannot edit this field.
The next number in sequence assigned for work orders in the logged in location’s profile (path: File > Setup > Accounting > Location/ Sub-Location > Options > Edit > In New Window > Inventory tab > Work Order # field) or the location of the logged in sub-location is assigned to the work order.
If sequence numbers are not specified here, Everest assigns the sequence numbers specified for the default location (path: File > Setup > Preferences > Defaults > Accounting > Miscellaneous tab > Default Location field).

The document number will be assigned to the work order only after the work order is saved.
