Custom Field Details

The Custom Fields tab allows you to define additional information for any record as character strings, or date, logical, memo or numeric fields. This ** tab is displayed only when you save the profile.

 To compute the machining hours for work orders, you can create a numeric custom field called ‘Number of Machining Hours’. You can update this field with the actual number of hours during the work order processing stage.
