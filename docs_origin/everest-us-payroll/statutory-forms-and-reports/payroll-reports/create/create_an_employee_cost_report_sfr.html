Employee Cost Report

This report shows the total cost of the company towards employees for a specified period. The employee cost comprises payments made to employees in the form of salary, regular pay, overtime pay, accruals for sick and vacation leave credited to the employee’s leave balances, additions to the employee’s paycheck, company contributions and employer taxes.

The employee cost report displays the following:

  • Employee code/Employee ID
  • Pay period
  • Total hours worked
  • Salary particulars
  • Additions to paycheck
  • Total Employer associated expenses

To create an employee cost report, do the following:

  • From the main menu bar, select Reports > Payroll > Employee Cost. You will see the Employee Cost Report dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box, before printing
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats.
