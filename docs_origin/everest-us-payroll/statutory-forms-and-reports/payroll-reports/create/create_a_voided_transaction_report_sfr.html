Voided Transactions Report

This report lists all the payroll transactions are voided during the specified period. The voided transactions report displays the following:

  • Employee code
  • Employee name
  • Transaction number
  • Transaction type
  • Start date
  • End date
  • Take-home pay

To create a voided transactions report, do the following:

  • From the main menu bar, select Reports > Payroll > Voided Transactions. You will see the Voided Transactions dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box, before printing
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats.
