Create a Form W2 Report

You have to create a Form W2 report for each employee every year.

To create a Form W2 report, do the following:

  • From the main menu bar, select Accounting > US Payroll > Statutory Forms > W2. You will see the [Form **W2
  • Employee List** dialog box](/versata-everest-docs/everest-us-payroll/misc/form_w2_employee_list_dialog_box_sfr.html), listing all the employees created in Everest US Payroll.
