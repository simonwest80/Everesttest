Employee Form W3

a) Control Number
This is an optional box that you may use for numbering the whole transmittal.
Enter the control number of the W3 form generated for a year. Make certain that entries do not cross over into the form identification number box (33333).
b) Kind of Payer
Check the appropriate box that applies to your company. Check only one box unless the second, checked box is “Third-party sick pay”. If you have more than one type of Form W2, send each type (except Third-party sick pay) with a separate Form W3.
The options available are listed below:
  • 941
  • Military
  • 943
  • 944
  • CT-1
  • Household Employer
  • Medicare Government Employee
  • Third-party sick pay
c) Total Number of Forms W-2
Total number of completed Form W2s for the selected year is displayed by default. Voided Forms W2 are not included in this list. You cannot modify this field.
d) Establishment Number
Enter your establishment number, which is used to identify separate establishments of your business. You may file a separate Form W3 with Form W2s for each establishment even if they have the same EIN number, or you may file a single Form W3 for all Form W2s of the same type.
e) Employer Identification Number
The Federal ID number entered in the Payroll Company Setup profile is displayed by default. Verify that the Employer Identification Number is correct. This number should be the same as the one shown on your Form 941, Form 943 and Form CT-1 in the following format: 00-0000000.
f) Employer’s Name
The company name specified in the Payroll Company Setup profile is displayed by default. This name should be the same as the one shown on your Form 941, Form 943 and Form CT-1.
g) Employer’s Address and ZIP Code
Employer’s address and the zip code specified in the Payroll Company Setup profile are displayed by default below the Employer’s name field.
h) Other EIN used this year
If you have used an Employer Identification Number (EIN) (including a prior owner’s EIN) on forms submitted to the IRS that is different from the EIN entered in Box e, enter that number used.
