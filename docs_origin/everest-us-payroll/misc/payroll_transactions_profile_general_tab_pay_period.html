Pay period

The pay period you specified in the Payroll Transaction Processing section of the wizard is displayed in this field. You cannot modify this field.

Pay Date
The pay date you specified in the Payroll Transaction Processing section of the wizard is displayed in this field. You can change this as per the requirements.
Start Date
The start date that you specified for the transactions in the Payroll Transaction Processing section of the wizard is displayed here. You cannot modify this field.
End Date
The end date that you specified for the transactions in the Payroll Transaction Processing section of the wizard is displayed here. You cannot modify this field.
