Payroll Transactions Profile Options

Following are the options (path: The Payroll Transactions profile > Options) ** available in the **Payroll Transactions profile:

Previous Transaction
This option brings up the preceding employee for payroll transaction in the Payroll Transaction Processing section of the wizard.
Next Transaction
This option brings up the next employee for payroll transaction in the Payroll Transaction Processing section of the wizard.
