Create a Supplemental Payroll Transaction

You can process a supplemental payroll transaction from the following location:

  • The Payroll Transactions browser - Select Options > New > In Current Window.

To create a supplemental payroll transaction, do the following:

  • From the main menu bar, select Accounting > US Payroll > Processing > Payroll Transactions. You will see the Payroll Transactions browser.
  • From the Payroll Transactions browser, select Options > New > In Current Window. You will see the Welcome window of the Payroll Transaction Processing wizard.
  • Click Next. You will see the Payroll Transaction Processing section of the wizard.
