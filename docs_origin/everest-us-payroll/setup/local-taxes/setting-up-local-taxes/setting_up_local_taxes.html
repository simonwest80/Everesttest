Set up a Local Tax

You can access the Local Tax dialog box from the following locations:

  • The Local Taxes browser - Select Options > New > In New Window.
  • An existing Local Tax dialog box - Select Options > New.

To set up a local tax, do the following:

  • From the main menu bar, select Payroll > Local Taxes. ** You will see the **Local Taxes browser.
  • From the Local Taxes browser, select Options > New > In New Window. You will see the Local Tax dialog box.
