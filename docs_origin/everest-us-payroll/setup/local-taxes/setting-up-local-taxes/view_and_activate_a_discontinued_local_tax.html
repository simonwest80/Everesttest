View and Activate a Discontinued Local Tax

You can make a discontinued local tax available for selection by activating it again.

To view and activate a discontinued local tax, do the following:

  • From the main menu bar, select Payroll > Local Taxes. You will see the Local Taxes browser.
  • From the Local Taxes browser, select Options > View Discontinued. You will see the Discontinued Local Taxes browser listing all the discontinued records.
  • Highlight the record you want to activate.
  • From the Discontinued Local Taxes browser, select Options > Activate. You will see the Confirm window.
  • Click OK to activate. The activated record will be displayed in the Local Taxes browser.
