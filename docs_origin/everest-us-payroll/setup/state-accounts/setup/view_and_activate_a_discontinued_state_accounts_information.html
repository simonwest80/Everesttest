View and Activate Discontinued State Accounts

You can make a discontinued state accounts record available for selection by activating it again.

To view and activate discontinued state accounts, do the following:

  • From the main menu bar, select Payroll > State Taxes > Accounts. You will see the State Accounts browser.
  • From the State Accounts browser, select Options > View Discontinued. You will see the Discontinued State Accounts browser listing all the discontinued records.
  • Highlight the record you want to activate.
  • From the Discontinued State Accounts browser, select Options > Activate. You will see the Confirm window.
  • Click OK to activate. The activated record will be displayed in the State Accounts browser.
