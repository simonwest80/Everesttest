Set up a Payroll Item

You can access the Payroll Items dialog box from the following locations:

  • The Payroll Items browser - Select Options > New > In New Window.
  • An existing Payroll Items dialog box - Select Options > New.

To set up a payroll item, do the following:

  • From the main menu bar, select Payroll > Payroll Items. You will see the Payroll Items browser.
  • From the Payroll Items browser, select Options > New > In New Window. You will see the Payroll Items dialog box.
