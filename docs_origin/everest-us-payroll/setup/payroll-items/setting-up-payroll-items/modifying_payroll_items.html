Modify a Payroll Item

You can access the Payroll Items dialog box ** you want to modify from the following locations:

  • Payroll Items browser - Highlight the record you want to edit and select Options > Edit >  In New Window.
  • Payroll Items profile - Specify the Code of the record you want to modify.

The following fields cannot be modified:

  • Code
  • Type
  • Pay Type
