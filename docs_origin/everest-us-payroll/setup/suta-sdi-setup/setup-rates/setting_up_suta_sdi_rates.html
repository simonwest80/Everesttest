Set up SUTA and SDI Rates

You can access the SUTA/SDI Setup dialog box from the following locations:

  • The SUTA/SDI browser - Select Options > New > In New Window.
  • An existing SUTA/SDI Setup dialog box - Select Options > New.

To set up SUTA/SDI rates, do the following:

  • From the main menu bar, select Payroll > State Taxes > SUTA/SDI. ** You will see the **SUTA/SDI browser.
  • From the SUTA/SDI browser, select Options > New > In New Window. You will see the SUTA/SDI Setup dialog box.
