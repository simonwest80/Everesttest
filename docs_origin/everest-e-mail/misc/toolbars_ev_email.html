Toolbars

Toolbars provide quick access to commonly used menu options. They allow you to organize options in Everest E-Mail as per your requirements, so that you can find and use them quickly. You can add or remove menus and buttons, create your own custom toolbars, hide or display toolbars, and move toolbars.

Toolbars in Everest E-Mail:

Button Description
One level Up - ** Allows you to ** move to different upper levels of the folder list.
Folder List - Folder list displays the folders available in your mailbox. In the list, click the folder you want to view. To view a subfolder, click the (+) sign.
Preview Pane - Preview pane lets you view the contents of the message selected in the table view pane.
Print Preview - Gives you a preview of the message to be printed.
Current View - Gives you options to view information in a folder as per your requirement. Each folder in Everest E-Mail has a set of pre-defined views. You can change the way you look at items or files by changing the standard views for each folder (path: From the main menu bar, View > Current View).
Group Box - Using the option Group box you can group similar items together.


Group e-mail messages from the same sender.
Field Chooser - This option enables you to hide/ display columns in
the table view.
Auto Preview - This option displays or hides the first line of items in a table.
More Buttons - This option lets you add or remove buttons to/from your toolbar.
