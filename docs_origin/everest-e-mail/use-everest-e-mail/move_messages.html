Move Messages

To move a message to a folder, do the following:

  • From the Folder List in the Everest E-Mail main window, select the folder from which you want to move the messages.
  • Select the messages you want to move.
  • From the Everest E-Mail menu, select Edit > Move to Folder. You will see the Move Messages dialog box.
  • Select the location where you want to move the messages.
  • Click OK. The messages are moved to the selected folder.

If you want to create a new folder and move the message into that folder, click New Folder. You will see the New Folder dialog box. Enter the name of the new folder, select the location where you want to create the folder and click OK. The new folder is created.
