Pull Sheet Wizard - Stock Bin Selection

This section allows you to define the stock bins from which the items should be picked.

Include Stock Bin
Only items contained in the bins specified in this column can be attached to the pick ticket.
If you set a preference by selecting the option Allow users to access stock bins of - Logged in Location (path: File > Setup > Preferences > Flow Control > Other Flow Control > General tab), all bins set up for the logged in location or sub-location are displayed by default in this column.
Stock Bins Profile
To exclude a bin, highlight it and click the button. Clicking the button moves all the bins to the Exclude column.
It is mandatory to include at least one stock bin.

You need adequate security rights to include the bins that are not part of the logged in location.

Exclude Stock Bin
Items contained in the bins specified in this column cannot be attached to the pick ticket. All bins that are set up for locations other than the logged in location are displayed by default in this column.
You can include bins by highlighting the required bin in the Exclude column and clicking the . button. To include all the bins click the button.

Click Next. You will see the Selected Stock Bins section of the Pull Sheet Wizard.
