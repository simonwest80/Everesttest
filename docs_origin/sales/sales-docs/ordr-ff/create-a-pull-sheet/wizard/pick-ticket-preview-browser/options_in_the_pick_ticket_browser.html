Options in the Pick Ticket Browser

The following options are available in the Pick Ticket Preview Browser (path: Pick Ticket Browser > Options):

  • Select All - Use this option to select all the items for picking.
  • Unselect All - Use this option to deselect the items that have been selected for picking.
  • Select Highlighted - Use this option to select the highlighted item for picking. The highlighted item can also be selected by clicking in the check box beside the Item Code column. When an item is selected, the check box beside the Item Code column is checked.
  • Unselect Highlighted - Use this option to deselect the highlighted item. You should highlight the required item before selecting this option. When an item is deselected, the checkbox beside the Item Code column is NOT checked. The highlighted item can also be deselected by clicking in the check box beside the Item Code column.

Options common to all browsers are also available in this browser.
