Common Sales Document Browser Options

The options (path: Sales Document browser > Options) common to all sales documents browsers are listed below:

  • Transaction History
  • View
  • Document Tracking
  • Multimedia
  • Print Document
  • Print Shipping Label
  • Search by Customer
  • Contact Manager
  • Sales Funnel
  • Sales Opportunity
  • Export Ship Address
  • E-mail Documents

The following information can be accessed from the View menu of a sales/sales return browser:

  • My Documents
  • All Documents
