Export Ship Address

The Export Ship Address option allows you to export the shipping address of the selected sales document into an MS Excel or text file.

When you select this option the Save As dialog box is displayed. Specify the directory in which you want to save the file.

You can export Shipping addresses pertaining to more than one document by highlighting all the required records before selecting the Export Ship Address option.

If the selected sales document does not have a shipping address defined, the document’s billing address is exported.
