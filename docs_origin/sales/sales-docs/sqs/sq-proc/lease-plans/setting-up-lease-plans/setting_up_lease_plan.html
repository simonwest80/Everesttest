Setting up a Lease Plan

To create a new lease plan, do the following:

  • From the main menu bar, select File > Setup > Miscellaneous > Lease Plan. You will see the Lease Plan Browser.
  • From the Lease Plan Browser, select Options > New > In Current Window/ In New Window. You will see the Lease Plan profile.

Lease plans can also be created:

From any profile that contains the Lease Plan field.

From an existing Lease Plan profile by selecting Options > New.
