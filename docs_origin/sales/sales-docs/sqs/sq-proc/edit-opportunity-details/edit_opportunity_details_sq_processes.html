Modify Opportunity Details

Everest allows you to edit the details in the Sales Opportunity profile from within the Sales Quote profile. Select Edit Opportunity (path: Sales Quote profile > Options > Edit Opportunity) to view the Sales Opportunity profile. You can edit the details in the Sales Opportunity profile as required.

You cannot modify the following fields:

  1. Opportunity ID
    • Opening Date
    • Document Alias
    • Customer Code
    • Sales Rep
    • Value
    • Currency
