Generate PO - Drop Ship Items Browser Options

The options available in this browser are listed below:

  • Edit > Line Items
  • Select/Deselect All Items
  • Generate POs

Options common to all browsers are also available in this browser.

Standard Browser Options

Edit > Line Items

This option (path: Options > Edit Line Items) allows you to change the vendor, cost and item quantity for the selected item.

When you select this option, the Profile - Line Item(s) dialog box is displayed.

Select/Deselect All Items

The Select/Deselect All Items toggle option (path: Options > Select/ Unselect All Items) allows you to select all the records displayed in the browser or deselect all the selected records.

Generate POs

Use the Generate POs option (path: Options > Create PO) to create purchase orders for the items selected in the Generate PO - Drop Ship Items browser. You will see the Purchase Order Status browser.
