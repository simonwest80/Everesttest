Accounting Entries - Processing Finance Charges

When finance charges are processed, the following accounting entry is created in the sales journal:

  • The Accounts Receivable Control Account specified in the Finance Charge field of the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting > Revenue Accounts tab) is debited.
  • The Finance Charges account specified in the Accounting dialog box (Revenue Accounts tab) is credited.
