The Finance Charges Setup Browser

The Finance Charges Setup browser is displayed when the parameters are specified in the Finance Charges profile (path: Accounting > Account Receivable > Finance Charges) and the OK button is clicked.

The browser contains two sections:

  • The Finance Charges section and
  • The Customer Document section

Select the required documents/customers and select Options > Save from the Finance Charges Setup browser to create a sales journal accounting for the finance charges. These charges appear on the customer statements and are receivable from the customer.
