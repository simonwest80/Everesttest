Job Code

The Job Code column displays the job to which the revenues and expenses for the line item in a sales document must be allocated. All open jobs are available for selection. If required, you can assign different job codes to different line items in a sales document.

To assign a job code to a line item, do the following:

  • Place your cursor in the Job Code column in the Item Details Grid ** of a sales document and click **Enter. You will see a drop-down arrow.
  • Click on the drop-down arrow, you will see a list of job codes. Select a record and double-click.

Everest validates the job code before assigning it to a line item. The following validations are carried out:

  • Verification of the job code for the selected customer
  • Verification of expired/closed job codes
  • Verification of job codes when a new line item is added to an existing sales document
