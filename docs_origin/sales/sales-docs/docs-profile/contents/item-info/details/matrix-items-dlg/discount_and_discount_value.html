Discount % and Discount Value

The discount % and discount value for the matrix item is calculated in the same manner as regular items.

Quantity and other discounts apply to individual matrix items, and not to the sum of quantities of items in the matrix.

If a matrix group consists of 25 items, a discount of 5% would be applicable to each of the items and not to the matrix group.

You can change the discount value or the discount %.

You can enter either the Discount% or the DiscountValue. If you specify the Discount % the Discount Value is automatically calculated and vice-versa.
