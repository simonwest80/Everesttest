Vendor

The Vendor column displays the default primary vendor for the selected item. If required, you can change the vendor. It also identifies the vendor on whom a drop-ship purchase order can be created for this item. You can view the Replenishment Details browser displaying all the vendors.

View the Replenishment Details Browser

To view the Replenishment Details browser, do the following:

  • Click the drop-down  arrow in the Vendor field and press F2.

OR

  • Click the drop-down  arrow, right-click, and select the Browser option from the drop down list.

The Replenishment Details browser appears.

  • Select the vendor and click OK to get the vendor’s name displayed in the SO document.
