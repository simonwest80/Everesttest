Model

You may have two different models of an item manufactured by the same company

Printers HP Color Laserjet 5M and HP Color Laserjet 5/ 5M PS are 2 models manufactured by HP in the Laserjet range.

If you have specified the model of the item in the Model field of the Item profile (path: Inventory > Items > Options > Edit > In New Window > General tab), the model of the item is displayed in the Model column. You cannot edit this column.
