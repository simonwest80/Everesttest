Document Alias

The document alias distinguishes a sales document from other sales documents of the same type.

Document Name (For Products) Document Alias (For Services)
Sales Quote Service Quote
Sales Order Service Order
Sales Invoice Service Bills

You could create an alias called Contract Proposal to record sales quotations given for service contracts and an alias called Repair and Service Orders for repair jobs.

Document Aliases

When you create a sales document from the sales quote, order or invoice browser, the default alias specified for each type of document is displayed in the document. You can also select a different alias for a document.

You can also create a document with a specific alias from the related document alias browser (path: Invoicing > Other Documents).

Document Aliases Defaults

When you convert a sales quote or a sales order with a specified document alias, the alias specified for the converted document (order or invoice) is used. If you do not define an alias for the document, the default alias for the respective document defined in the Document Alias dialog box (path: File > Setup > Preferences > Defaults > Document Alias) is used.

When you convert a sales quote with the alias Service Quote to an order, the document takes the alias Service Order specified for the sales order.
