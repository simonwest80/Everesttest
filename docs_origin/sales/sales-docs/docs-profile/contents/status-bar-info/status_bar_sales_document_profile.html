Status Bar - Sales Document Profile

The Status Bar is present at the bottom of all document profiles in Everest. The options available in the status bar of sales document profiles are:

  • Preview Off/On
  • Show/Hide
  • Zoom Out/In
  • Document Policy
  • Unposted/Posted
  • CC Authorization
  • Document Profitability
  • Line Item Profitability
  • Customer Status
  • Item Notes
  • Customer Balance
  • Stock Status

In a sales order, the status bar also displays a message if the order is under picking.
