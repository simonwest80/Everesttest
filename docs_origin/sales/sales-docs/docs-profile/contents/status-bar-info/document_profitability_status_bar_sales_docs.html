Document Profitability

This option displays the profitability of the sales document. Click this icon and you will see the Pricing Analysis dialog box for the entire sales document. The profitability of the sales document is encrypted in the icon in the format ‘Dyynnnnnn-ppppp-mmmm-yy’ where ‘yy’ is a random number, ‘nnnnnn’ is the cost, ‘pppp’ is the profit amount, and ‘mmm’ is the margin/markup. Profit and cost are rounded off to nearest whole number. The first alphabet, the first two numbers and the last two numbers do not represent any information and are random.

The encoded document profitability icon follows the Cost-Profit-Mark up %/Margin format. In this icon, the total cost of the items in the document is 2470. The profit (selling price - cost) of the item is 990 and the mark up % is 40.

Based on the preference set in the Analysis By field of the Pricing Analysis dialog box (path: File > Setup > Preferences > Defaults > Pricing Analysis), the document profitability status indicator displays the pricing analysis either by margin or markup.

You will see the Document Profitability indicator in the status bar of a sales document only if you have set a flow control preference by selecting the Show document line item profitability in encoded fashion check box in the Documents dialog box (path: File > Setup > Preferences > Flow Control > Documents > Miscellaneous > Others > Others1 tab).
