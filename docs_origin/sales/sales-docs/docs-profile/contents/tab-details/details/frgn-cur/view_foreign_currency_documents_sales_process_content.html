Foreign to Base

Where the quote type is Foreign to Base, the translation formula used is:

Foreign Currency Amount x (Exchange Rate/Quote Amount)

Base to Foreign

Where the quote type is Base to Foreign, the translation formula is:

Foreign Currency Amount x (Quote Amount /Exchange Rate)

If your base currency is the US Dollar, and the document currency is Great Britain Pounds (GBP), and the rate is expressed as 1 GBP = 1.42064. Using the formula for quote type foreignto base, 1000 GBP = 1000 x (1.42064/1) = 1420.64.

If the rate is expressed as 1 USD = 0.703973 GBP, then using the formula for quote type baseto foreign, 1000 GBP = 1000 x (1/0.703973) = 1420.64.
