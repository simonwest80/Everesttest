Third Party Catalog - Import Item

Using the Import Item (path: Sales Document profile > Options > Third-Party Catalog > Import Item) option, you can import an item present in a third-party catalog into the Everest database.

To import an item present on a third-party catalog into the Everest database, do the following:

  • From the Sales Document profile, select an item that exists in a third-party catalog and choose Options > Third-Party Catalog > Import Item. You will see the Item profile.
  • Enter the required details and select Options > Save. The item is imported into the Everest database.
