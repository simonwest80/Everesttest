Add-Ons

The Add-Onsoption (path: Sales Document profile > Options > Item > Add-Ons) allows you to add accessories for the item in the sales document. Accessories refer to those items that can be bought or sold along with a particular item. This option is disabled for the kit code in sales documents, since add-ons cannot be defined for a Kit.

Add-Ons

To add accessories to the items contained in a sales document, do the following:

  • From the Sales Document profile, highlight the required item and select Options > Item > Add-Ons. You will see the Add-On Items browser.

If you have not set up add-on items for a particular item, you will be prompted with a message that no add-ons exist for the selected item. Click OK to return to the sales document.

  • From the Add-On Items browser, select the item that is defined as an add-on.
  • Click OK. The item is added to the sales document.

The selling price of the add-on item is the price that would be displayed if the item was retrieved from the Items browser (path: Inventory > Items).

Item Pricing

You can set a preference in the Documents dialog box by selecting the Display list of add-ons when adding items to sales documents check box (path: File > Setup > Preferences > Flow Control > Documents > Prompts > Sales Document > Sales Document 2 tab).

Add-ons are defined in the Add-Ons tab of the Item profile (path: Inventory > Items > Options > Edit > In New Window).
