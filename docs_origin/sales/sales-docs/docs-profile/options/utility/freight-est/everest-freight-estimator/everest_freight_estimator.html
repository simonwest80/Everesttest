Everest Freight Estimator

The Everest Freight Estimator option allows you to compute freight based on destination zones for all countries.

To enable the **Everest Freight Estimator option to compute freight, the name of the carrier has to be identified in the DeliveryMethod profile. Freight is calculated only when the delivery method is company delivery and not a customer pickup.**

Delivery Methods

Freight is calculated based on the:
  • Originating zip code (of the document’s parent  location).
  • Destination zip code (of the ship to address in the document).
  • Ship-via agent (the delivery method selected).
  • Rate Definition.
The freight charges include transportation cost, COD fee, residential charges and insurance cost wherever applicable.
