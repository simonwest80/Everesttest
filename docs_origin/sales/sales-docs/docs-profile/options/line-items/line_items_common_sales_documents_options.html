Line Items

The Line Items option (path: Sales Document profile > Options > Line Items) allows you to do the following:

Options Description
Insert Inserts a blank line in the Item Details Grid.
Upsell Information Displays information entered in the Upsell
Information field of the Item profile (path: Inventory
> Items > Options > Edit > In New Window > Notes tab).
Line Profile Displays information about the line highlighted in the Item Details Grid in the Sales
Document Lines dialog box.
Move up Moves the selected line item one line up.
Move down Moves the selected line item one line down.
Find When you have a number of items in a sales/sales return document, use this option to find and edit the specific line item. When you select this option, you will see the Find Line Items browser.
