E-mail Information

To
This field displays the e-mail ID entered while specifying the default contact address of the customer. The default contact address is specified in the Contact Address field of the Customer profile (path: Invoicing > Customers > Options > Edit > In New Window > Addresses tab). If a default contact address has not been specified, this field defaults to the e-mail ID as specified in the billing address.
Click the button and you will see the Address Book dialog box. You can add additional e-mail addresses from this dialog box.
CC
This field displays the default e-mail ID entered while specifying the default billing address of the customer. The default billing address is specified in the Customer profile (path: Invoicing > Customers > Options > Edit > In New Window > General tab).
If you have already specified the e-mail ID specified as per the billing address of the customer in the To field, then this field will remain blank.
Click the button and you will see the Address Book dialog box. You can add additional e-mail addresses from this dialog box.
Subject
This field displays the document alias defined for the document/s in the Document profile. The document number/s is/are also displayed in this field.
