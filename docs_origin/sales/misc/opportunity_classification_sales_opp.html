Opportunity Classification

Stage
Every sales opportunity progresses through several stages before it can be converted to a sale. This field identifies the sales stage for the opportunity. This field displays the default sales stage. You can define a default sales stage by selecting the Default check box on the Sales Stage dialog box (path: CRM > Opportunity Management > Setup > Sales Stage > Options > Edit > In New Window).
The first stage may be the Initial Contact phase and the second stage may be the Demonstration phase.
Sales Stages
Modified
This field displays the date on which you assign the sales stage for the opportunity. If you modify the stage at a later date, this field displays the altered date.
Source
The source identifies the origin of the opportunity.
In case of a new customer, the source could be a referral or magazine advertisement.
Identifying the source helps in analyzing the effectiveness of advertising. By default this field displays the Source specified for a customer (path: Invoicing > Customers > Options > Edit > In New Window > E-Commerce tab).

Modified
This field displays the date on which the source is identified in the Source field. If you modify the source at a later date, this field displays the altered date.
Grade
Use this field to assign a rating to the opportunity.
Opportunity Grade
