Document Information

Document #
The number assigned to the document is displayed in this field. The document number is automatically assigned based on the sequence number specified for credit memos in the logged in location or sub-location. If no sequence numbers have been assigned to the location, the sequence numbers specified for the default location is used. The document number is displayed for a new credit memo only when the document details are saved.
Document Number
Loc/Sub-Loc
The logged in location or sub-location is displayed by default in this field. The location/sub-location can be changed as required.
Loc/Sub-Loc
Document Alias
The default document alias specified for credit memos is displayed in this field. The document alias can be changed by clicking the icon.
Document Alias
Document Notes
Click this icon beside the Document # field to specify document notes.
Document Notes
Document Date
The document creation date is displayed in this field. This field cannot be edited.
Document Date
Document Policy
Click this icon to view the document policy. This policy can be edited.
Document Policy
Document Status
The status of the credit memo - whether ‘open’ or ‘paid’ is displayed in this section. The credit memo number and creation date is displayed in this section. If the credit memo is processed from a credit order, this section also displays the credit order number and date.
Document Status
