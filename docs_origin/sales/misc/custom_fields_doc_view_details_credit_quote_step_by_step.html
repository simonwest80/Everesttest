Custom Fields

The Custom Fields tab allows you to define additional information for any record as per your requirement. Custom fields can be defined as character strings, or date, logical, memo or numeric fields.

This tab is displayed only when you save the credit quote document.

If a credit quote has an expiry date that is set in the “Date” custom field, you can search for the credit quote using the Expiry Date field in the Custom Field tab.
