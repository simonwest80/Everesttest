E-mail Information

To
This field displays the e-mail ID entered while specifying the default contact address of the customer. The default contact address is specified in the Contact Address field of the Customer profile (path: Invoicing > Customers > Options > Edit > In New Window > Addresses tab). If a default contact address is not specified, this field defaults to the e-mail ID specified in the billing address.
Click the button and add the desired addresses from the Address Book dialog box.
CC
This field displays the default e-mail ID specified in the billing address in the Customer profile (path: Invoicing > Customers > Options > Edit > In New Window > General tab).
If you have already specified the e-mail ID as per the billing address of the customer in the To field, this field remains blank.
Click the button and add the desired addresses from the Address Book dialog box.
Subject
This field displays the document alias defined for the document/s in the Document profile. The document number/s is/are also displayed in this field.
