Document Information

Document #
The number assigned to the document is displayed in this field. The document number is automatically assigned based on the sequence number specified for sales quotes in the logged in location or sub-location. If no sequence numbers have been assigned to the location, the sequence numbers specified for the default location is used. The document number is displayed for a new sales quote when the document details are saved.
Document Number
Loc/Sub-Loc
The logged in location or sub-location is displayed by default in this field. The location/sub-location can be changed as required.
Loc/Sub-Loc
Document Alias
The default document alias specified for sales quotes is displayed in this field. The document alias can be changed by clicking the icon.
Document Alias
Document Notes
Click this icon to specify document notes.
Document Notes
Document Date
The document creation date is displayed in this field. This field cannot be edited.
Document Date
Document Policy
Click this icon in the status bar of the sales quote to view the document policy. This policy can be edited.
Document Policy
Document Status
The sales quote number and creation date is displayed in this section. The details in this section are displayed only after the sales quote details have been saved.
Document Status
