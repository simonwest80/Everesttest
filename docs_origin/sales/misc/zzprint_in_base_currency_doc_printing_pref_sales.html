This drop-down list allows you to specify the currency in which to print the document. It has the following options:

  • Print in Base Currency - Select this option to print the document in the base currency, irrespective of the currency defined for the customer. The Currency field defaults to this option.
  • Print in Document Currency - Select this option to print the document in the currency defined for the customer selected in the document.
  • Print in both Base and Document Currency - Select this option to print the document in the currency defined for the customer selected in the document. However, the document and the tax totals are displayed additionally in the base currency at the bottom right hand side of the document.
