Exchange Rate Details

This section allows you to specify the exchange rates for foreign currency refunds. The exchange rate details have to be specified for the following:

  • Document Currency
  • Bank Currency
  • Payment Currency
  • Base Currency

The settings in the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting > Multi Currency > Exchange Rates tab > Rate for receipts field) determine whether the buy or sell rate is displayed in the exchange rate fields.
