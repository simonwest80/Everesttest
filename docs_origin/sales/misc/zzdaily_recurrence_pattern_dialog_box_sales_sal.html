Select the Daily option to create a sales order:

  • On each day of the selected period.

: Enter 1 in this field to create a sales order every day during the period specified in the Duration section.

  • On every ‘nth’ day of the selected period.

: Specify 3 in this field to create sales orders every third day during the period specified in the Duration section.

When Daily is selected, the Every <number of > Day(s) field is enabled. Specify the frequency of the schedule in this field.

The maximum number of days that can be specified is 100.
