Notes

This field allows you to search for sales orders by using the Document Notes as the parameter value. Document Notes are used to record any information related to the document. You can enter the text of the document note in this field as your search criteria. You can also search using partial values in this field.

If you enter the value ‘This Sales Invoice..’ in this field, the search will yield all sales invoices with a document note beginning with ‘This Sales Invoice..’.

Click the drop-down button to view the history of parameter values used.
