Display Browsers/Dialog Boxes

Press Key(s) To
Ctrl+Alt+E Display the Export
Dialog box.
Ctrl+F Display the Custom
Fields dialog box for the selected sales invoice.
Ctrl+F11 Display the Shipment
Tracking Information dialog box.
Ctrl+P Display the Browser
Property Editor dialog box.
Ctrl+T Display the TransactionHistory browser for the selected sales invoice.
Ctrl+U Display the Journal
Transactions dialog box for the selected sales invoice.
F8 Display the Payment
History dialog box for selected sales invoice.
Shift+F2 Display the MultimediaSetup browser for the selected sales invoice.
Shift+F5 Display the Custom
Tracking browser for the selected sales invoice.
Ctrl+E Display the Document
E-mail Preferences dialog box.
Shift+Ctrl+K Display the Task profile.
Shift+Ctrl+I Display the Incoming
Call profile.
Shift+Ctrl+O Display the Outgoing
Call profile.
Shift+Ctrl+N Display the General
Notes profile.
Shift+Ctrl+H Display the Task History browser.
Shift+Ctrl+M Display the My Tasks browser.
Shift+Ctrl+A Display the Appointment profile.
Shift+Ctrl+L Display the Appointment
History profile.
Shift+Ctrl+C Display the Calendar
View profile.
