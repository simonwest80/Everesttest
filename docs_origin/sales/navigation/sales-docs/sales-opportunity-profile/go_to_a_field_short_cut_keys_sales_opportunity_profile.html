Press Key(s) To
Alt+A Go to the EstimatedClose Date field.
Alt+I Go to the Opportunity
ID field on an opportunity that has been saved.

Alt+S Go to the Stage field.
Alt+T Go to the Document
Alias field.
Alt+U Go to the CustomerCode field.
Alt+V Go to the Value field.
Alt+Y Go to the Probability field.
