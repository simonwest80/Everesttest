Other Shortcut Keys

Press Key(s) To
Ctrl+Alt+H Import items from the third party catalog.
Ctrl+Alt+Space Preview on/off of the Item Details Grid.
Ctrl+Down arrow Move the selected item down in the Item Details Grid.
Ctrl+F4 Close the sales quote.
Ctrl+N Create a new sales quote from a quote that has been saved.
Ctrl+P Print the sales quote from a quote that has been saved.
Ctrl+S Save the sales quote.
Ctrl+Space Zoom in/out of the Item Details Grid.
Ctrl+Up arrow Move the selected item up in the Item Details Grid.
F3 Process the quote to an order.
F6 Toggle between document currency and the base document.
Shift+Ctrl+Z Cancel the changes made to an existing sales quote.
Shift+F4 Apply discount applicable to the customer.
Shift+F10 Displays the Stock
Status Indicators for all line items in a sales document. This is only enabled if the “Automatically display stock status indicators” preference in the Items dialog box – Item Details tab (path: File > Setup > Preferences > Flow
Control > Items) is NOT selected (disabled). We recommend that you disable this option to improve performance when adding line items to sales documents. In this situation, the Stock Status Indicators do not automatically display for the line items.
