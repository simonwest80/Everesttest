Display Browsers/Dialog Boxes

Press Key(s) To
` Display the RetrieveSerial Numbers dialog box.
Alt+F8 Display the Description dialog box for the selected line item.
Ctrl+A Display the CustomerAging Information browser.
Ctrl+Alt+C Display the CustomFields dialog box from a saved sales order.


Display Find Customers dialog box on a sales order that has not been saved.
Ctrl+Alt+F8 Display the Detachserial # dialog box.
Ctrl+Alt+L Display the Item Stock
Status dialog box for the selected line item.
Ctrl+Alt+S Display the Quantitystock areawise browser for the selected item. This option is enabled in saved sales orders for non-serialized items.
Ctrl+Alt+V Display the Serial
# for all items browser.
Ctrl+Alt+W Display the Serial
# browser for the selected serialized item.
Ctrl+D Display the PricingAnalysis dialog box for all the items on the order.
Ctrl+F11 Display the Shipment
Tracking Information browser.
Ctrl+G Display the Custom
Tracking dialog box for the customer.
Ctrl+H Display the AttachQuantity dialog box only when inventoried, non - serialized items are attached or negative quantities are attached to the sales order.
Ctrl+I Display the Items browser for items selection.
Ctrl+K Display the RetrieveKits dialog box.
Ctrl+L Display the Item Stock
Status dialog box for all items on the document.
Ctrl+M Display the Matrix
Items View dialog box if a matrix item is selected.
Ctrl+O Display the Item profile for the selected line item.
Ctrl+Q Display the RetrieveSerial Numbers (range) dialog box.
Ctrl+R Display the Payment dialog box for the selected sales order.
Ctrl+Right Arrow Display the Sales Document
Lines dialog box for the selected line item.
Ctrl+T Display the TransactionHistory browser from a saved sales order.
Ctrl+U Display the Customer’s profile.
Ctrl+W Display the MultimediaSetup browser from a saved sales order.
Ctrl+Y Display the Item Sales
History browser for the selected line item.
F10 Displays the Stock
Status Indicator for the selected line item in a sales document. This is only enabled if the “Automatically display stock status indicators” preference in the Items dialog box – Item Details tab (path: File > Setup > Preferences > Flow
Control > Items) is NOT selected (disabled). We recommend that you disable this option to improve performance when adding line items to sales documents. In this situation, the Stock Status Indicators do not automatically display for the line items.
F11 Displays a list of Add-on items associated with the selected line item.
F12 Displays a list of Related
Items associated with the selected line item.
F8 Display the PaymentHistory browser for the sales order.
Shift+` Display the Retrieve
Mfg Serial
Numbers dialog box.
Shift+F5 Display the Custom
Tracking dialog box for the document from a saved sales order.
Shift+F6 Display the DocumentNotes dialog box.
Shift+F7 Display the Customers browser for customer selection.
Shift+F8 Display Items (Kits) browser for kit selection.
Shift+F9 View and edit the Sales
Opportunity profile for a saved sales order.
Ctrl+E Display the Document
E-mail Preferences dialog box.
Shift+Ctrl+K Display the Task profile.
Shift+Ctrl+I Display the Incoming
Call profile.
Shift+Ctrl+O Display the Outgoing
Call profile.
Shift+Ctrl+N Display the General
Notes profile.
Shift+Ctrl+H Display the Task History browser.
Shift+Ctrl+M Display the My Tasks browser.
Shift+Ctrl+A Display the Appointment profile.
Shift+Ctrl+L Display the Appointment
History profile.
Shift+Ctrl+C Display the Calendar
View profile.
Ctrl+F12 Display the Find Line
Items browser.
Ctrl+Alt+T Display the items from the default Third
Party Catalog.
