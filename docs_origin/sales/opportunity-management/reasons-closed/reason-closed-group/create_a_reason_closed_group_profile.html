Create a Reason Closed Group Profile

To create a reason closed group, do the following:

  • From the main menu bar, select CRM > Opportunity Management > Setup > Reason Closed. You will see the Reason Closed Setup browser.
  • From the Reason Closed Setup browser, select New > Reason Group. You will see the Reason Closed Group profile.

You can also create a reason closed group from an existing Reason Closed Group profile by selecting Options > New.
