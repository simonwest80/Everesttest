Stage

You can specify the sales stage to find the sales documents.

You can search for sales documents in the ‘Draft proposal’ stage.

If you create opportunities, specify the stage assigned in the Sales Opportunity profile (path: Invoicing > Sales Opportunities > Options > Edit > In New Window).

If you create sales documents directly without creating a sales opportunity, Stage 1- Initial Contact is assigned by default to the opportunity.
