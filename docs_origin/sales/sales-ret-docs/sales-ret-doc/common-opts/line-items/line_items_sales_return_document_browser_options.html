Line Items

The Line Items option (path: Sales ReturnDocument profile > Options > Line Items) allows you to do the following:

Options Description
Insert Inserts a blank line in the Item Details Grid.
Upsell Information Displays information entered in the Upsellinformation field of the Item profile (path: Inventory
> Items > Options > Edit > In New Window > Notes tab).
Line Profile Displays brief information about the line highlighted in the Item Details Grid. When you select this option the Sales Document Lines dialog box ** is displayed.
Move up Moves the selected line item one line up.
Move down Moves the selected line item one line down.
Find Locates a line item in the sales return document. When you have a number of items in a sales return document and you want to edit a particular line item, use this option to find the specific line item. When you select this option you will see the Find Line Items browser.
