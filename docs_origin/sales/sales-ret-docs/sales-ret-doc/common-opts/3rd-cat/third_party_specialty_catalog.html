Third Party Catalog

Third party databases such as vendor catalogs can be imported and browsed for quick reference.

 Import Third Party Catalogs

The Third Party Catalog option in sales return documents (path: Sales Return Document profile > Options > Third Party Catalog) allows you to verify if the item returned by the customer belongs to a third party catalog. You can also import the returned item into the database. Third party catalog information that can be accessed from within a document are listed below:

  • All Catalogs
  • Default Catalog
  • Import Item
