Third Party Catalog - Default Catalog

Using this option, you can view items that belong to the default third party catalog. This option is useful when:

  • You want to compare prices of items that exist in your database with items in the default third party catalog.

A third party catalog can be set as a default by choosing a record from the Third Party Catalogs browser and selecting Options > Set as Default Catalog.

To view the default third party catalog, do the following:

  • From the Sales Return Document profile, select Options > Third Party Catalog > Default Catalog. You will see the Items browser listing all the items contained in the third party catalog.

If you have not set a default third party catalog, you will see the Third Party Catalogs browser.

  • If you want to add an item to the sales return document, select the item and click OK.
