Status Bar - Sales Return Document Profile

The Status Bar is present at the bottom of all document profiles in Everest. The options available in the status bar are:

  • Preview Off/On
  • Show/Hide
  • Zoom Out/In
  • Document Policy
  • Unposted/Posted
  • Document Profitability
  • Line Item Profitability
  • Item Notes
  • Customer Status
  • Customer Balance
  • Stock Status
