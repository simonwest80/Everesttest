Document Alias

The document alias distinguishes a sales return document from other sales return documents of the same type.

You could create two aliases Customer RMA - Quotes (for items returned) and customer return quotes - service (for unsatisfactory service) for the document type ‘Credit Quote’.

Document Aliases

When you create a new sales return document from the Credit Quote, Order or Memo browser, the default alias specified for each type of document in the default preferences (path: File > Setup > Preferences > Defaults > Document Alias) is displayed in the document. A different alias can be selected for a document, if required.

You can also create a new document with a specific alias from the browser (path: Invoicing > Other Documents).

When you convert a credit quote or a credit order with a specified document alias, the alias specified for the order or the memo in the profile of the quote or order (which is converted) is used. If the alias for the order is blank, then the default alias for the order or memo is used.

Document Aliases Defaults

You convert a credit quote with the alias customer RMA quote to an order. The alias for credit orders in the customer RMA quote's profile is RMA order. RMA order will be the alias used for the converted order.

If the alias for the order is blank, and the default credit order alias defined in the

Document Alias dialog box (path: File > Setup > Preferences > Defaults > Document Alias) is credit order, then the customer RMA quote is converted to a credit order.
