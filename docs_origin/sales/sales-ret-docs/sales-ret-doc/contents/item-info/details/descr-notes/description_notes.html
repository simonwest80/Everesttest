Description/Notes

The

Description/Note column displays the description of an item as entered in the Item profile. To edit the description, click on the Description/Note column for the item, and type the required description.

You may want to use the item’s description that appears in the customer’s catalog.

Extended Description

The extended description of the item as described in the Items profile is displayed below the item description. Each line in the extended description is displayed on a separate line.
