Brand

A brand is a trademark or distinctive name identifying a product. If you have specified a brand to which the item belongs in the Brand field of the Item profile (path: Inventory > Items > Options > Edit > In New Window > General tab), the brand code is displayed in the Brand column. You can set up brands in the Brand profile (path: Inventory > Brands > Options > Edit > In New Window). You cannot edit this column.
