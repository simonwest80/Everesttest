View and Edit Notes Attached to a Customer

To view and edit notes attached to a customer, do the following:

  • From the sales return profile, click the Notes icon beside the CustomerName field. You will see the Customer Notes dialog box.

You can view the Customer Notes dialog box by clicking on the Notes icon if notes are entered in Comments section of the Customer profile (path: Invoicing > Customers > Options > Edit > In New Window > Miscellaneous tab).
