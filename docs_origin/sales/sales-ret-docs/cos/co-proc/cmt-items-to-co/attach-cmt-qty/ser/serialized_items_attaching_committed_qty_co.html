Serialized Items

Serial numbers of the item to be attached to the credit order or memo may be retrieved in any of the following ways:

  • One by one
  • As a range
  • By manufacturer’s serial number

You can select serial numbers from stock areas for the logged in location if you select the Allow Users to access only those stock areas in the logged in location check box (path: File > Set up > Preferences > Flow Control > Other Flow Control > General tab).

You will be prompted with the message Item does not exist on order. Serialize? when you attach a serial number of an item not present on the credit order or memo. Click Yes to attach the item on the order or memo.

This prompt is received if the Prompt when trying to retrieve and serialize an item that is not already present on a document option (path: File > Setup > Preferences > Flow Control > Documents > Miscellaneous > Others > Others 2 tab) is selected.

Adding to Items Committed on Credit Orders

After committing items to a Credit Order, you can commit additional items by attaching the serial numbers of the additional quantity on the Credit Order profile.
