The Sales Return Document Browser

The Sales Return Document browser lists all the documents of the selected sales return document type.

To view the credit quote/order/memo browser, do the following:

  • From the main menu, select Invoicing > Credit Quotes/Orders/Memos.
  • You will see the relevant document browser.

To view documents with a specific alias, do the following:

  • From the main menu, select Invoicing > Other Documents. You will see a submenu that lists the sales return documents.
  • From the submenu, select the specific document type. You will see another submenu that lists the aliases for the document.
  • Select the alias to view its document browser.

The Other Documents option displays credit quotes, credit orders and credit memos along with their document aliases. These are the system and user-defined aliases.

If you create a credit quote without selecting an alias for it, the document comes up in the Credit Quotes browser from the Other Documents option.

The Dates field in a sales document’s browser enables you to filter sales documents based on their creation date.
