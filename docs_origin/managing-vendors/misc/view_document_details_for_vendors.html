View Document Details for Vendors

To view the document details belonging to a vendor, do the following:

  • From the main menu bar, select Purchasing > Vendors. You will see the Vendors browser.
  • From the Vendors browser, highlight the vendor.
  • Select Options > Documents. You will see a submenu that lists all the purchase, purchase return, system and manual documents.
  • From the submenu, select the document type for which you want to view details. You will see a browser displaying all documents (of the selected type) that have been created for the selected vendor.
  • You can create a document of the related type from the browser displayed.

 Select Purchase Order (path: Vendors browser > Options > Documents > Purchase Order), to view the Purchase Orders browser and create a purchase order.

From the related document type browser, when you click New > In New Window to create a document, the vendor’s code and other information is automatically retrieved into the new document.

If you select the All System Documents option, you can create documents of all the types that are listed in the browser. The document created depends on the record you highlight before selecting Options > New > In New Window.

 If the selected record was a purchase receipt, a new purchase receipt profile is displayed.
