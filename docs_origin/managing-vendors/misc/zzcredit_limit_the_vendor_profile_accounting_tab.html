Enter the credit limit allowed by the vendor.

The Credit Limit field cannot have a negative value and the last decimal is rounded off. The number of decimals displayed depends on the number of decimal digits specified in your settings:

  • Currency profile - path: File > Setup > Accounting > Multi-Currency > Currency.
  • Regional Settings Properties dialog box - From the Currency tab in the regional settings. The regional settings can be modified from the Control Panel of your operating system.
