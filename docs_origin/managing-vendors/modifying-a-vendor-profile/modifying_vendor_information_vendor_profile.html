Modify a Vendor Profile

You can modify the Vendor profile from the following locations:

  • The Vendors browser - Highlight the record you want to modify. Select Options > Edit > In New Window.
  • The Vendor profile - Specify the vendor’s Account Code.

You cannot modify the Account Code field in the Vendor profile, once the profile is saved. You can modify the Currency Code field until the vendor is selected on a document or journal.

To modify the First Name, Middle Name, Last Name, Address, City, State, Zip and Country fields for a vendor whose billing address is used on a final document, select Options > Modify Address from the Vendor profile. To modify the State/Province, you must select a different Zip/Postal Code.
