The Vendor Profile - Options

The options available from the Options menu of the Vendor profile (path: Purchasing > Vendors > Options > New > In New Window > Options) have been grouped under the following headings:

Information Available

  • Documents
  • Quick Aging
  • History

Other Profile Options

  • View Formatted Address
  • Modify Address
  • Vendor Tracking
  • Send e-mail
  • Go to Website
  • Contact Manager
  • Events History

The Vendor profile contains standard menu options common to all profiles in Everest.
