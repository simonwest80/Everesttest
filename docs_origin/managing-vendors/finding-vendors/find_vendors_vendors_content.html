Find Vendors

The **Find Vendors function facilitates a search for vendors as per specific criteria that you define.**

 If you have a large vendor database and require a list of vendors who belong to the manufacturing department, use the Find Vendors function.

You can select the search parameters individually or in combination with any of the other parameters to narrow down the search.

The search results help you perform various functions such as:

  • Quick aging
  • View documents associated with vendors
  • View the transaction history
  • Send e-mails and faxes
