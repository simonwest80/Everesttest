# Adjustments to the Values in Form W2


You can make adjustments to the values specified in **Form W2** and also keep track of them.


{:.note}
You can make adjustments to all the values  in the boxes 1 to 14 except for the local taxes and state taxes.


{:.steps}
To make adjustments, do the following:

- From the main  menu bar, select **Accounting** > **US** **Payroll**  > **Statutory Forms &gt; W2**.  You will see the **Form W2 -** **Employee List** dialog box, listing all  the employees on your payroll.
- Select the  employee for whom you want to create a Form W2 report and then click **W2**. You will see the **Employee 
 Form W2** created for the selected employee.
- Click on the  box for which you want to make adjustments. You will see the [**Form W2** **Details**  dialog box]({{site.prl_baseurl}}/misc/form_w2_details_dialog_box_sfr.html).



{:.example}
Clicking the **Social Security Wages**  field takes you to **Form W2 Details - 
 Box3**.
