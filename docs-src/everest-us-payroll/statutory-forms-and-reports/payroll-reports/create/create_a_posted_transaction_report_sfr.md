# Posted Transactions Report


This report lists all the payroll transactions created and posted during  the specified period. The posted transactions report displays the following:

- Employee code
- Employee name
- Transaction number
- Transaction type
- Start date
- End date
- Take-home pay



{:.steps}
To create a posted transactions report, do  the following:

- From the main  menu bar, select **Reports &gt; Payroll 
 &gt; Posted Transactions**. You will see the [**Posted Transactions** dialog box]({{site.prl_baseurl}}/misc/posted_transactions_dialog_box_sfr.html).
- Specify the  report parameters and click:

 - **Preview**,  to preview the report in the **Report Preview**  dialog box, before printing
 - **Print**,  to print the report
 - **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats.
