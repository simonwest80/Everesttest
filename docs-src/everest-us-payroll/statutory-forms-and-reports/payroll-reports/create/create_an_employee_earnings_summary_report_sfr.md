# Employee Earnings (Summary) Report


This report shows the summary of earnings of all the employees for the  specified period. It does not generate a payroll item-wise breakup of  employeesâ€™ earnings under each category. It displays the following:

- Employee code/Employee  ID
- Pay period
- Total hours worked
- Salary particulars
- Standard earnings
- Take-home pay



{:.steps}
To create an employee earnings (summary)  report, do the following:

- From the main  menu bar, select **Reports &gt; Payroll 
 &gt; Employee Earnings** (**Summary**).  You will see the [**Employee Earnings Summary** dialog box]({{site.prl_baseurl}}/misc/employee_earnings_summary_dialog_box_sfr.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview**  dialog box, before printing
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats.
