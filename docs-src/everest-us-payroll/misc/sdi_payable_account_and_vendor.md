# SDI Payable Account and Vendor


**SDI Payable Account**
: This is the liability account to which the SDI taxes  withheld from employees have to be credited.


**SDI Payable Vendor**
: This is the vendor to whom the SDI taxes withheld  have to be paid. You will have to identify the vendor code for the State  Government or its Revenue Department.
