# SUTA Accounts


**SUTA Expense Account and Location/Sub-Location**
: This is the expense account to which the SUTA taxes  payable as an employer are debited. The **Location/Sub-Location** is the location or sub-location to which this expense is allocated.


![example.gif]({{site.prl_baseurl}}/img/example.gif) If you have defined a  cost center called Payroll Tax Expenses - CA as location/sub-location,  select it here.
: ![Lens.gif]({{site.prl_baseurl}}/img/lens.gif) [Locations  and Sub-Locations]({{site.sc_chm}}/setting_up_a_company.html)


**SUTA Payable Account**
: This is the liability account to which the SUTA  taxes payable as an employer have to be credited.


**SUTA Payable Vendor**
: This is the vendor to whom the employer's SUTA taxes  have to be paid. You will have to identify the vendor code for the State  Government or its Revenue Department.
