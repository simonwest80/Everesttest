# Payroll Transactions Browser Options


Except **Purge** and **Employee Data** (path: The **Payroll Transactions** browser > **Options**) all the other options available with the **Payroll Transactions** browser are standard options common to all browsers in **Everest**.


**Purge**
: Use this option to delete payroll transaction records permanently from the database.


![Lens.gif]({{site.prl_baseurl}}/img/lens.gif)[Purge a Payroll Transaction]({{site.prl_baseurl}}/payroll-process/purge/deleting_payroll_transactions.html)


**Employee Data**
: Use this option to view the employee information such as payroll items specified for a particular employee in the **Employee** profile.
