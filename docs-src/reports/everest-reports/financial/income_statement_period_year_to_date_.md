# Income Statement (Period Year to Date)


The **Income Statement (Period Year to 
 Date)** report compares the income of the specified period of a specified  year with the cumulative income up to the specified period. The report  includes the account description and the income details in the specified  period and the year to the current date in the fiscal year for the specified  period. The income statement is prepared with the posted balances by default.  You can also include provisionally posted or unposted amounts, and hide  accounts with zero balance by selecting the required parameters.


{:.steps}
To generate the income statement (period year to date) report,  do the following:

- From the main  menu bar, select **Reports &gt; Financial 
 &gt; Income Statement (Period Year to Date)**. You will see the [**Income Statement (Period Year to Date)**  dialog box]({{site.rpt_baseurl}}/misc/income_statements_period_year_to_date_dialog_box_rep.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
