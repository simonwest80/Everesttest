# Job Costing


The **Job Costing** report is used  to analyze whether a specific project or job is profitable. This report  displays the following details of the documents that are assigned to each  job code. This report displays details of the documents that are assigned  to each job code and revenue and expenditure accounted on a job via general  journals. All general journal entries (posted and unposted) are included.  This report displays the following details:

- The document/general  journal (GJ)/transaction  number and date.
- The item/account  code and item description/account description.
- The total revenues  and expenses incurred on the job and the net margin (profit/loss) on the  job.



{:.steps}
To generate the job costing report, do the following:

- From the main  menu bar, select **Reports** >  **Financial** > **Job 
 Costing**. You will see the [**Job Costing** dialog box]({{site.rpt_baseurl}}/misc/job_costing_dialog_box.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
