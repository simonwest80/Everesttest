# Income Statement Yearly


The **Income Statement Yearly**  report displays a detailed or summarized statement of the income and expenses  for the selected fiscal year. The detailed report gives a break up of  the revenues and expenses by locations/sub-locations. The income statement  is prepared with the posted balances. You can also include provisionally  posted or unposted amounts, and hide accounts with zero balance by selecting  the required parameters.


{:.steps}
To generate the income statement (yearly)  report, do the following:

- From the main  menu bar, select **Reports &gt; Financial 
 &gt; Income Statement Yearly**. You will see the [**Income Statement (Yearly)** dialog box]({{site.rpt_baseurl}}/misc/income_statement_yearly_dialog_box_rep.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
