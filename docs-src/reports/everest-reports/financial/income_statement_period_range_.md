# Income Statement (Period Range)


The **Income Statement (Period Range)**  report, generates a report of the income and expenses for a range of accounting  periods in the selected fiscal year. If the company is classified by location,  you can generate the income statement for each location/sub-location separately.  The income statement is prepared with posted balances by default.


{:.steps}
To generate the income statement (period range) report, do  the following:

- From the main  menu bar, select **Reports &gt; Financial 
 &gt; Income Statement (Period Range)**. You will see the [**Income Statement (Period Range)** dialog  box]({{site.rpt_baseurl}}/misc/income_statement_period_range_dialog_box_rep.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
