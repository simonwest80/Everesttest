# Purchase Document Tracking (Group)


The **Purchase Document Tracking (Group)** report allows you to view purchase documents based on the custom  tracking groups. This report includes the document number and date, purchase  representative, vendor name, scheduled date for the document, the amount  mentioned on the document and the balance of the document.


![]({{site.rpt_baseurl}}/img/lens.gif) [Track  a Purchase Document]({{site.ct_chm}}/document-tracking/tracking-purchase-documents/track_a_purchase_document.html)

{:.steps}
To generate the Purchase Document Tracking (Group) report,  do the following:

- From the main  menu bar, select **Reports &gt; Purchasing 
 &gt; Purchase Document Tracking (Group Report)**. You will see the  [**Purchase 
 Document Tracking (Group)** dialog box]({{site.rpt_baseurl}}/misc/purchase_document_tracking_group_dialog_box_rep.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box, before printing
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
