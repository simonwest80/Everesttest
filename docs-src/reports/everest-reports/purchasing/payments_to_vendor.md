# Payments to Vendors


The **Payments** **to 
 Vendors** report lists all payments made to selected vendors including  refunds during a time period. The report lists payments for only those  transactions which are posted and displays the following details:

- Code and name of  the vendor for whom payments or refunds are made
- Date of payment  and the payment method
- Reference number,  if any
- Code and description  of the account accounting for the payments or refunds
- Amount of payment  or refunds made (negative amounts signify refunds from vendors and positive  amounts denote payments made to vendors for goods and services received)
- Total amount of  net payments made to each vendor



![]({{site.rpt_baseurl}}/img/lens.gif) [Type  of Payment]({{site.sc_chm}}/options/payment-information/payment-methods/payment-method-details/type_of_payment.html)


[The Posting  Process]({{site.acc_chm}}/accounting-structure-in-everest/posting/posting.html)


{:.note}
Employees created as vendors are not included  in this report.


{:.steps}
To generate the Payments to Vendors report, do the following:

- From the main  menu bar, select **Reports** >  **Purchasing** > **Payments****to** **Vendors**.  You will see the [**Payments 
 to Vendors** dialog box]({{site.rpt_baseurl}}/misc/payments_to_vendor_dialog_box.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
