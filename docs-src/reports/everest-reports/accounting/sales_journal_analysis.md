# Sales Journal Analysis


The **Sales Journal Analysis** report  displays details of the sales journals that are not posted.


![]({{site.rpt_baseurl}}/img/lens.gif) [The Posting Process]({{site.acc_chm}}/accounting-structure-in-everest/posting/posting.html)


The report details the commissionable and taxable amounts on each sales  invoice that comprises the sales journal. The **Sales 
 Journal Analysis** report also displays details of the selected sales  journals:

- **Document 
 details** - The date of creation, the representative who created  it, the total taxable and exempt amounts on the document, and the tax  amount on the document.
- **Account 
 details** - The accounts debited/credited, and the total value of  debits and credits for all the documents contained in the journal.



{:.steps}
To generate this report, do the following:

- From the main  menu bar, select **Reports** >  **Accounting** > **Sales 
 Journal Analysis**. You will see the [**Sales Journal Analysis** dialog box]({{site.rpt_baseurl}}/misc/sales_journal_analysis_dialog_box.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
