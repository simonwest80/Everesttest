# General Ledger


The **General Ledger** report displays  a summary of the aggregate of debits and credits to an account (including  discontinued accounts) for the selected range of fiscal years. This report  is based on posted transactions. You may specify additional parameters  to filter the report based on transactions allocated to specific locations/sub-locations.  The report displays amounts either in the account currency or the base  currency or both, depending on the parameters selected.


{:.steps}
To generate the General Ledger report, do the following:

- From the main  menu bar, select **Reports** >  **Accounting** > **General 
 Ledger**. You will see the [**General Ledger** dialog box]({{site.rpt_baseurl}}/misc/general_ledger_dialog_box_rep.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
