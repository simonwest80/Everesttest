# Customer Tracking (Selection)


The **Customer Tracking** (**Selection**) report is similar to the  [Customer Tracking (Group)]({{site.rpt_baseurl}}/everest-reports/invoicing/customer_tracking_group_report_.html) except that  it groups the customers based on the **Tracking 
 Selection** that belongs to a **Tracking 
 Group**.


You might have classified customers by the tracking selections ‘Magazine  Advertisements’, ‘Trade shows’, ‘Newspaper’ etc. belonging to the tracking  group ‘How did you hear about us’.


![]({{site.rpt_baseurl}}/img/lens.gif) [Customer  Tracking]({{site.ct_chm}}/customer-tracking/customer_tracking.html)


{:.steps}
To generate the Customer Tracking (Selection) report, do  the following:

- From the main  menu bar, select **Reports &gt; Invoicing 
 &gt; Customer Tracking (Selection)**. You will see the [**Customer Tracking (Selection)** dialog  box]({{site.rpt_baseurl}}/misc/customer_tracking_selection_dialog_box_rep.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box, before printing
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
