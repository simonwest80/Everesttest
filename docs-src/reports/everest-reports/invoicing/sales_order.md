# Sales Orders


The **Sales Orders** report **** lists sales orders created during a  specified time period. **** This report  gives the order details such as the item code, the item description from  the Item profile, and the amount of the **Sales 
 Order**. ****


![]({{site.rpt_baseurl}}/img/lens.gif)[Sales  Order]({{site.bp_chm}}/docs/sys/sales-documents/sales_order_businesss_process_in_everest_content.html)


{:.steps}
To generate the Sales Orders report, do the following:

- From the main  menu bar, select **Reports** >  **Invoicing** > **Sales 
 Order**. You will see the [**Sales Orders Report** dialog box]({{site.rpt_baseurl}}/misc/sales_order_report_dialog_box.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
