# Commission (Quick Summary) 


The **Commission (Quick Summary)**  report lists all documents (sales invoices and credit memos) that qualify  for commission. The amounts are grouped based on the sales representatives.  Credit memo amounts are<font style="color: #ff0000;" color="#FF0000"> </font>set off against the  sales invoice amounts.


{:.steps}
To generate the Commission (Quick Summary) report, do the  following:

- From the main  menu bar, select **Reports &gt; Invoicing 
 &gt; Commission (Quick Summary)**. You will see the [**Commission (Quick Summary) Report** dialog  box]({{site.rpt_baseurl}}/misc/commission_quick_summary_report_dialog_box.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
