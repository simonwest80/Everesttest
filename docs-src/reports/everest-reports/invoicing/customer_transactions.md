# Customer Transactions


The **Customer Transactions** report  lists all business transactions made with customers during a specified  time period. The history of all the transactions can be analyzed to determine  customers who are prompt in their payments and with whom business is profitable.


{:.note}
This report displays the customer balances for all sales and receipt  journals irrespective of whether they are posted or not.


The report displays the following:

- Account code  and name of the customer
- Address details  of the customer
- Whether customer  is on credit hold or not
- Current net  balance of the customer
- Payment terms  as specified in the Payment Information tab of the Customer profile.
- Sales and sales  return documents created for the customer, along with document date, number  and amount.



{:.steps}
To generate the Customer Transaction report, do the following:

- From the main  menu bar, select **Reports** >  **Invoicing** > **Customer 
 Transactions**. You will see the [**Customer Transactions** dialog box]({{site.rpt_baseurl}}/misc/customer_transactions_dialog_box_rep.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
