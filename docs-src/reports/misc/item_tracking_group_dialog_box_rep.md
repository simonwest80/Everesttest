# Item Tracking Group Dialog Box


Select the following parameters to view the **Item 
 Tracking** (**Group**) report.


**Date From**
: Specify the start date for the tracking selections  that were tagged to a group, to be included in the report.


**Date To**
: Specify the end date for the tracking selections  that were tagged to a group, to be included in the report.


{:.note}
If the **Date 
 From** and **Date To** fields  are left blank, by default the current system date is included in the  report.


**Print Memos**
: Select this check box to include notes attached  to the tracking selection in the report.


**Sort By**
: Select the **Sort 
 By** option to sort the report by **Item** or **Status**.


**Group**
: Specify the code of the tracking group to be included  in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**<font style="color: #000000;" color="#000000">Sales Rep</font>**
: You can filter the report based on the sales representative  who tagged the item to the tracking selection.
: Select a sales representative in this field. If  this field is left blank, all users are included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)
