# Commission (Quick Summary) Report Dialog Box


Select the following parameters to view the **Commission 
 (Quick Summary)** report.


**Sort By**
: Select this option to sort the report by:

- Sales Rep,  Document Date, Document
- Sales Rep,  Document, Document Date
- Sales Rep,  Customer, Document, Document Date



**Date From**
: Specify the start date from which the report is  generated.


**Date To**
: Specify the end date till which the report is generated.


{:.note}
If the **Date From**  and the **Date To** fields are left  blank, the current system date appears in these fields.


**Sales Representative From**
: Specify a starting code for the range of sales representatives  to be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Sales Representative To**
: Specify an ending code for the range of sales representatives  to be included in the report.


{:.note}
If the **Sales Representative 
 From** and **Sales Representative 
 To** fields are left blank, all the sales representatives are included  in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Customer From**
: Specify the starting code for the range of customers  to be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Customer To**
: Specify the ending code for the range of customers  to be included in the report.


{:.note}
If the **Customer From**  and the **Customer To** fields are  left blank, all the customers are included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Location/Sub-Location**
: Specify the location/sub-location code based on  which the report is to be generated. If this field is left blank, the  item categories for all locations/sub-locations are included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)
