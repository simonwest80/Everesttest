# Attendance Dialog Box


**User From**
: Specify the starting user code to be included in  the report.


**User To**
: Specify the ending user code to be included in the  report.


{:.note}
If the **User 
 From** and the **User To** fields  are left blank, all users are included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Shift Code**
: Select the shift code for which you want to generate  the report.


{:.note}
If the **Shift 
 Code** field is left blank, the report is generated for all shift  codes.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Ascending Desc**
: Select the alphabetical order, in which you want  user names to be displayed in the report.


**Date From**
: Specify the start date of the period for which you  want to generate the report.


**Date To**
: Specify the end date of the period for which you  want to generate the report.


{:.note}
If the **Date 
 From** and **Date To** fields  are left blank, the current system date is included in the report by default.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)
