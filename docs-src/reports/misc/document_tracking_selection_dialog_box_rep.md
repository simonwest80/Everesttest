# Document Tracking (Selection) Dialog Box


Select the following parameters to view the **Document Tracking (Selection)** report.


**Group**
: Select the code of the tracking group to be included  in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Tracking Selection**
: Select the code of the tracking selection to be  included in the report.
: Only those documents that are assigned to the selected  tracking selection under the specified group are included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Document Type**
: Specify the types of sales documents to be included  in the report.

- A - Sales Quote
- J  - Sales Order
- I - Sales Invoices
- S - Credit  Quote
- R - Credit  Order
- C - Credit  Memo



**Document Status**
: Specify the document status to be included in the  report.

- **All**  - To include all document types in the report.
- **Closed**  - To display only closed sales documents.
- **Open**  - To display open sales documents.
- **Back****Orders** - To display back orders.
- **Fully****Open** - To display only those sales  documents on which no payment is made.

: Selecting **All**  includes all document types in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Document  Status](the-sales-process.chm::/document_status_document_information_sales_return_document_content.htm)


**Location/Sub-Location**
: Select the code of the location/sub-location to  be included in the report.


{:.note}
If this field is left blank,the document tracking  details for all locations/sub-locations are included in the report.


**Document Date From**
: Specify the start date for the documents, to be  included in the report.


**Document Date To**
: Specify the end date for the documents, to be included  in the report.


{:.note}
If the **Document 
 Date From** and **Document Date To**  fields are left blank, the current system date are included in the report.


**Tag Date From**
: Specify the start date, as to when the tracking  selections were tagged to a group.


**Tag Date To**
: Specify the end date, as to when the tracking selections  were tagged to a group.


{:.note}
If the **Tag 
 Date From** and **Tag Date To** fields  are left blank, by default the current system date are included in the  report.
