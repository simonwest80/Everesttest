# Customer Activity Report Dialog Box


Select the following parameters to view the **Customer 
 Activity Report**.


**Customer From**
: Specify the starting customer code for the range  of customers to be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Customer To**
: Specify the ending customer code for the range of  customers to be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Date From**
: Specify the start date for the activity, to be included  in the report.


**Date To**
: Specify the end date for the activity, to be included  in the report.


{:.note}
By default, the current system date will be  displayed in the **Date From** and  **Date To** fields.


**Sales Rep From**
: You can filter the report to see only those customers  assigned to a specific sales representative.
: Specify the starting sales representative code for  the range of sales representatives to whom customers are assigned to be  included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Sales Rep To**
: You can filter the report to see only those customers  assigned to a specific sales representative.
: Specify the ending sales representative code for  the range of sales representatives to whom customers are assigned to be  included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


{:.note}
If the customer code is selected previously,  and **Sales Rep From** and **Sales 
 Rep To** fields are left blank, by default all sales representatives  for that customer are included in the report.


**Report Type**
: Specify whether you want to print either the **Last Activity** report or **Without 
 Activity** report.
