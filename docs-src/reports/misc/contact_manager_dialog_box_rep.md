# Contact Manager Dialog Box


Select the following parameters to view the **Contact 
 Manager** report:


**Sort By**
: Select this option to sort the report by:

- A - User, Creation  date
- B - User, Start  date
- C - User, Due  date
- D - User, Remind  date
- E - User, Customer,  Creation date
- F - User, Vendor,  Creation date
- G - User, Document,  Creation date



**User From**
: Specify the starting code for the range of users  who have created the tasks, to be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**User To**
: Specify the ending code for the range of users who  have created the tasks, to be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Create Date From**
: Specify the start date for the range of creation  dates of the task, to be included in the report.


**Create Date To**
: Specify the end date for the range of creation dates  of the task, to be included in the report.


{:.note}
If the **Create 
 Date From** and **Create Date To**  fields are left blank, the current system date is included in the report.


**Start Date From**
: Specify the start date for the range of tasks, to  be included in the report.


**Start Date To**
: Specify the end date for the task, to be included  in the report


{:.note}
If the **Start 
 Date From** and **Start Date To**  fields are left blank, the current system date is included in the report.


**Due Date From**
: Specify the start date for the range of due dates  of the task, to be included in the report.


**Due Date To**
: Specify the end date for the range of due dates  of the task, to be included in the report.


{:.note}
If the **Due 
 Date From** and **Due Date To**  fields are left blank, the current system date is included in the report.


**Remind Date From**
: Specify the start date for the range of remind dates  of the task, to be included in the report.


**Remind Date To**
: Specify the end date for the range of remind dates  of the task, to be included in the report.


{:.note}
If the **Remind 
 Date From** and **Remind Date To**  fields are left blank, the current system date is included in the report.


**Account Type**
: You can further filter the report to include tasks  pertaining to customers only, vendors only, or both.


**Document**
: If **Customers Only** is selected in the **Account Type** field, all customer related documents are displayed in the report.  If you select **Vendors Only** in  the **Account Type** field, all vendor  related documents are displayed in the report. If **All 
 Customers and Vendors** is selected in the **Account 
 Type** field, all the document types are displayed in the report.


**Customer From**
: Specify the starting code for the range of customers  to be included in the report if you select **Customers 
 Only** in the **Account** **Type** field.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Customer To**
: Specify the ending code for the range of customers  to be included in the report if you select **Customers 
 Only** in the **Account Type**  field.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Vendor From**
: Specify the starting code for the range of vendors  to be included in the report if you select **Vendors 
 Only** in the **Account Type**  field.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Vendor To**
: Specify the ending code for the range of vendors  to be included in the report if you select **Vendors 
 Only** in the **Account Type**  field.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Document From**
: Specify the starting number of the document for  the range of selected documents, to be included in the report. All tasks  associated with documents in the specified range are listed.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Document To**
: Specify the ending number of the document for the  range of selected documents, to be included in the report. All tasks associated  with documents in the specified range are listed.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Display Address**
: Select this check box to include the customer's  or vendor's address in the report.


**Pending**
: Select this check box to include the pending tasks  of the user in the report.


**Display Notes**
: Select this check box to include notes for the tasks  to be displayed in the report.


**Completed**
: Select this check box to include the completed tasks  of the user in the report.
