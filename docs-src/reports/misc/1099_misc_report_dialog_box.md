# 1099-Misc Report Dialog Box


**Vendor From**
: Specify the starting code for the range of vendors  to be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Vendor To**
: Specify the ending code for the range of vendors  to be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Date From**
: Specify the start date of the report period. This  is generally the first day of a calendar year.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Date To**
: Specify the end date of the report period. This  is generally the last day of a calendar year.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Threshold Amount**
: This field defaults to 600 (dollars), but you can  change this if necessary. For example, the IRS currently requires issuing  a 1099-Misc only to those vendors you paid $600 or more during the entire  calendar year; if the IRS requirement changes you may change the amount  in this field.


**Include 1099 Vendors Only**
: This is selected by default. Uncheck this to view  all vendors which meet the above search criteria.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [1099  Vendor]({{site.mv_chm}}/vendor-details/accounting-information/federal_id_number_and_1099_vendor_accounting_information.html)


**Mark Corrected 1099 Box on Form**
: Select this option to mark the correction code box  with a cross mark in the IRS 1099-Misc form.
