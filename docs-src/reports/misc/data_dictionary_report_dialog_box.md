# Data Dictionary Report Dialog Box


**Table From**
: Specify the starting name for the range of tables  to be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Table To**
: Specify the ending name for the range of tables  to be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Report Type**
: Specify the type of report you want to generate.


| **Summary** | To view a summary of the data tables in **Everest**. |
| **Detail** | To view a detailed report of the data tables in **Everest**. A detailed report also contains  a short description of the fields in the table. |

: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)
