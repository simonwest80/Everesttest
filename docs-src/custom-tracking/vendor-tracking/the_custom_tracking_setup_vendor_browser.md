# The Custom Tracking Setup - Vendor Browser


The **Custom Tracking Setup - Vendor**  browser (path: **Purchasing &gt; Custom 
 Tracking &gt; Vendors**) displays the tracking categories, tracking  groups and tracking choices created for vendors. The groups and choices  are ranked according to the order of their creation. You can modify the  rank.


The options (path: **Custom Tracking 
 Setup - Vendor** browser > **Options**)  available in this browser are:

- [Custom  Fields for Groups]({{site.ct_baseurl}}/misc/custom_fields_for_group.html)
- [Show  Details]({{site.ct_baseurl}}/misc/show_details_ct_setup_vendor_browser_options.html)
- [Selection  Break-up]({{site.ct_baseurl}}/misc/selection_break_up_ct_setup_vendor_browser_options.html)
- [Tree  View]({{site.ct_baseurl}}/misc/tree_view_ct_setup_vendor_browser_options.html)
- [Switch  Pane]({{site.ct_baseurl}}/misc/switch_pane_ct_setup_vendor_browser_options.html)
- [Standard  Browser Options]({{site.wwe_chm}}/everest-client/ui/browsers/standard_browser_options.html)
