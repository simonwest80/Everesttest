# The Users Browser 


You can access the **Users** browser  from **File &gt; Setup &gt; Users &gt; 
 Users**. The options in the **Users** browser are categorized under the following heads:

- Information  Available
- Other Options



**Information Available**

- [Show  Treeview]({{site.sc_baseurl}}/misc/show_treeview_user_profile_browser_option.html)
- [View  Password]({{site.sc_baseurl}}/misc/view_password.html)
- [Employee  History]({{site.sc_baseurl}}/misc/employee_history_users_browser_option.html)
- [Leave  Balance]({{site.sc_baseurl}}/misc/leave_balance.html)



**Other Options**

- [Custom  Fields]({{site.sc_baseurl}}/misc/custom_fields_user_profile.html)
- [Multimedia]({{site.sc_baseurl}}/misc/multimedia_user_profile.html)
- Rights
- [Rights  - Users]({{site.sc_baseurl}}/misc/rights_users.html)
- [Rights  - Groups]({{site.sc_baseurl}}/misc/rights_groups.html)
- [Rights  - All Rights]({{site.sc_baseurl}}/misc/all_rights_user_browser_option.html)
- [Events  - History]({{site.sc_baseurl}}/misc/events_history_user_browser_option.html)
- [E-mail]({{site.sc_baseurl}}/misc/e_mail_user_browser_option.html)



Options common to all browsers in **Everest**  are also available in this browser.
