# The Delivery Methods Browser 


The **Delivery Methods** browser  (path: **File &gt; Setup &gt; Freight &gt; 
 Delivery Methods**) **** displays  a list of all the delivery methods that are defined.


This browser has the options that are common to all browsers in **Everest**.
