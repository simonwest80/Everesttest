# The Jurisdictions Browser


The **Jurisdictions** browser (path:  **File &gt; Setup &gt; Accounting &gt; 
 Taxes &gt; Jurisdictions**) displays all the tax jurisdictions that  are set up.


Options common to all browsers in **Everest**  are available in this browser.
