# The Tax Components Browser


The **Tax Components** browser (path:  **File &gt; Setup &gt; Accounting &gt; 
 Taxes &gt; Tax Components**) displays a list of tax components that  are defined.


Options common to all browsers in **Everest**  are available in this browser.
