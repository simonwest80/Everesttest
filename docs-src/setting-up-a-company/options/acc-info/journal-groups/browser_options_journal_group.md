# The Journal Groups Browser 


The **Journal Groups** browser (path:  **File &gt; Setup** **&gt; 
 Accounting &gt; Journal Groups**) lists all the journal groups that  are created.


Options common to all browsers in **Everest**  are also available in this browser.
