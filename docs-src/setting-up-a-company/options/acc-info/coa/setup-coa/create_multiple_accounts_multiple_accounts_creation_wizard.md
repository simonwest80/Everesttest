Select a control **Account Type**  from the drop-down list.


If you select **Bank Account** from  the drop-down list, the **Last Check Number**  field must be populated. This is a mandatory field.

Select the currency code to be used for the account type you are creating.  First and second level accounts must be in the base currency. Third level  accounts can be any currency. Accounts below the third level must match  the immediate parent's currency. This is a mandatory field.

Select Re-measurement Required if the accounts you are creating are  in a foreign currency.

# Create Multiple Accounts


The **Multiple Account Creation Wizard**  can be used to easily mass create your accounts using your configured  segment format.


{:.steps}
To access the Multiple Account Creation Wizard,  do one of the following:

- From the Company Setup Wizard, access the Multiple  Account Creation Wizard from the **Chart 
 of Accounts - User-Defined** screen.
- From the main menu bar, select **Setup**  > **Accounting** > **Create 
 Multiple Accounts**.



The **Welcome** screen is displayed.


{:.steps}
To create multiple accounts, do the following:

- Click **Next**.  You will see the **Account Posting Level**  screen.
- Click **General**  or **Detail** depending upon the account  type you are creating. To create a Detail account associated with a General  account, the General account must first be created.
- Click **Next**.  You will see the **Account Details**  screen. Make selections for the following fields. Click on the links for  field details.
