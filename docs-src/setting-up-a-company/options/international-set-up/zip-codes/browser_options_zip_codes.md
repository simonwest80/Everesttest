# The Zip Codes Browser


The **Zip Codes** browser (path:  **File &gt; Setup &gt; Miscellaneous &gt; 
 Zip Codes**) lists all the zip codes that are defined.


This browser contains the options that are common to all browsers in  **Everest**.
