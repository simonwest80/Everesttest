# The Countries Browser


The **Countries** browser (path:  **File &gt; Setup &gt; Miscellaneous &gt; 
 Country**) displays a list of all countries.


You can access the [**Restore Default Formats** option]({{site.sc_baseurl}}/options/international-set-up/country/profile/menu-option/restore_default_formats_country_profile_menu_option.html) from  the **Countries** browser.


Standard browser options common to all browsers in **Everest**  are available in this browser.
