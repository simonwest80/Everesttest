# View and Activate Discontinued Currencies


{:.steps}
To view and activate a discontinued currency, do the following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Accounting &gt; Multicurrency &gt; Currency**. You will see the **Currencies** browser.
- From the **Currencies** browser, select **Options** > **View Discontinued**.  You will see the discontinued record in the **Discontinued 
 Currencies** browser.
- F<font color="#000000" class="hcp2">rom the </font>**Discontinued**<font color="#000000" class="hcp2"> </font>**Currencies** <font color="#000000" class="hcp2">browser, highlight the record.</font>
- <font color="#000000" class="hcp2">Select </font>**Options &gt; Activate**<font color="#000000" class="hcp2">. 
 You will be prompted for confirmation.</font>
- <font color="#000000" class="hcp2">Click </font>**OK**<font color="#000000" class="hcp2">. The activated record 
 will be displayed in the </font>**Currencies** <font color="#000000" class="hcp2">browser. 
 </font>
