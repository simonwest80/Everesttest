# The Currencies Browser


The **Currencies** browser (path:  **File &gt; Setup &gt; Accounting &gt; 
 Multi - Currency &gt; Currency**) lists all the currencies that you  have defined in the database.


The browser options are detailed below:


**Exchange Rates**
: Use this option to view all the exchange rates that  are defined for the selected currency. Selecting **Options 
 &gt; Exchange Rates** displays the [**Exchange Rates**]({{site.sc_baseurl}}/options/multicurrency/exchange-rates/set-up-exchange-rates/browser_options_exchange_rates.html) browser.


Options common to all browsers in **Everest**  are also available in this browser.
