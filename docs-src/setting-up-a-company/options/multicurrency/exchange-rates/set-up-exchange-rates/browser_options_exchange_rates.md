# The Exchange Rates Browser 


The **Exchange Rates** browser (path:  **File &gt; Setup &gt; Accounting &gt; 
 Multi-currency &gt; Exchange Rates**) displays the exchange rates  that are defined for all currencies.


The following option is available in the **Exchange 
 Rates** browser:


**Multiple Rate Entry**
: The **Multiple****Rate** **Entry**  option enables you to enter the exchange rates for a number of currencies  at one stretch. This option brings up the **Multiple** **Exchange****Rates** dialog box


**Double-click the **Buy/Sell**  column of the required currency and enter the rate.**


**<font style="color: #000000;" color="#000000">You can also mention 
 the date on which the exchange rates are quoted.</font>**


Options common to all browsers in **Everest**  are also available in this browser.
