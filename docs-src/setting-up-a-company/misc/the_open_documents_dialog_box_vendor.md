# The Open Documents - Vendors Dialog Box


Use the **Open Documents** dialog  box to enter the open documents for each vendor. It is displayed when  you select the **Open Documents** option  from the right-click menu in the **Vendor 
 Balances** dialog box. The columns contained in the **Open 
 Documents** dialog box are explained below:


**Document Type**
: A vendor can have only the final documents as open  documents: **Debit Memo** and **Purchase Invoice**. Click the column to  view a drop-down list of the available document types. Select the required  document type.


**Document #**
: Enter the document number in this column.


**Amount in Base Currency**
: Enter the amount of the document in this column.  The amount entered should be in the [base currency](javascript:BSSCPopup('{{site.sc_baseurl}}/misc/base_currency_pop_up.html');){:id="A1"}<script type="text/javascript" language="JavaScript1.2"><!--
kadovFilePopupInit('A1');
//--></script>.


**Rate**
: If the document currency is not the base currency,  enter the conversion rate of the foreign currency to the base currency  in this column.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Setting  up Exchange Rates]({{site.sc_baseurl}}/options/multicurrency/exchange-rates/set-up-exchange-rates/setting_up_exchange_rates.html)


**Amount in Document Currency**
: Specify an amount in this column, only if the document  currency is a foreign currency.


**Terms**
: Click the row below the column header to view a  drop-down list of the available payment terms created by default. Select  the required payment term.


**Date**
: Click the column to view the calendar. Select the  required date.


Click **OK** when you have entered  all the open documents for the selected vendor. The [**Vendor Balances** dialog box]({{site.sc_baseurl}}/misc/the_vendor_balances_dialog_box.html) is displayed.  Repeat the process till all the open vendor documents are specified.


{:.note}
**Click **OK**  in the **Vendor** **balances**  dialog box to save the data in the database.**


Click **Next**. You will see the  [**Other 
 Preferences** section]({{site.sc_baseurl}}/the-company-creation-wizard/other-preferences/other_preferences.html).
