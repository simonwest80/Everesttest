# Defaults Dialog Box


**Everest** allows you to define  defaults for documents, customers, vendors, items and accounts to facilitate  quicker processing of transactions and automatically populate fields in  documents. The tabs displayed in this dialog box are dependent on the  icons you select in the left panel. The left panel has the following icons:

- [Documents]({{site.bp_baseurl}}/flow-ctrl/defs/rpt/documents_dialog_box.html)
- [Items]({{site.mi_chm}}/item-preferences/item-defaults/items_dialog_box.html)
- [Printing  Preferences]({{site.bp_baseurl}}/rpt-prt/set-prt/the_printing_preferences_dialog_box_step_by_step_bp.html)
- [Pricing  Analysis]({{site.sp_chm}}/sales-docs/sqs/sq-proc/pricing-analysis/pricing-analysis-defaults/pricing_analysis_dialog_box.html)
- [Accounting]({{site.acc_chm}}/accounting-flow-control-and-defaults/accounting-defaults/the_accounting_dialog_box.html)
- [Document  Alias]({{site.bp_baseurl}}/flow-ctrl/defs/alias-defs/document_alias_dialog_box.html)
- [Other  Defaults]({{site.bp_baseurl}}/other-defaults/the_other_defaults_dialog_box.html)
