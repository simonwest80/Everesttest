# The Document Policies Browser


The **Document Policies** browser  (path: **File &gt; Setup &gt; Miscellaneous 
 &gt; Document Policy**) displays a list of the document policies  and other related information. To view or edit a document policy from  the browser, highlight the record and double-click it. Options common  to all browsers are also available in this browser.
