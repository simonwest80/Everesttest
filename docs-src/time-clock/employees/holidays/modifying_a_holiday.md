# Modify a Holiday


You can modify the **Holiday** profile  from the following locations:

- The **Holidays**  browser (path: **Time Clock** >  **Setup** > **Holidays**)  - Highlight the record to be modified and select **Options**  > **Edit** > **In 
 New Window**.
- The **Holiday**  profile.
