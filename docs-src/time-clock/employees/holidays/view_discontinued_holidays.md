# View Discontinued Holidays


When you discontinue a holiday, it is temporarily unavailable for selection.  The record is not deleted from the database; it is merely deactivated.  A discontinued record can be made available for selection by activating  it again.


{:.steps}
To view discontinued holidays, do the following:

- From the main  menu bar, select **Time Clock** >  **Setup** > **Holidays**.  You will see the **Holidays** browser.
- From the **Holidays** browser, select **Options**  > **View Discontinued**. You will  see the **Discontinued Holidays**  browser.
