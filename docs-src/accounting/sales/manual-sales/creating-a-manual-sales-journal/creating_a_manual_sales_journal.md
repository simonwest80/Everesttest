# Create a Manual Sales Journal


{:.steps}
To create a manual sales journal, do the  following:

- From the main  menu bar, select **Accounting &gt; Account 
 Receivable &gt; Sales Journal**. You will see the **Unposted 
 Sales Journals** browser.
- Select **Options** > **New**  > **In New Window**. You will see  the [**Sales 
 Journal** profile]({{site.acc_baseurl}}/sales/manual-sales/creating-a-manual-sales-journal/the_sales_journal_profile.html).



{:.note}
You can also create a manual sales journal  from the **Sales Journal** profile  by selecting **Options** > **New Journal**.
