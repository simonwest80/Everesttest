---
title: Transaction Amount
---


This field allows you to search by the account activity by specifying  the transaction amount.


![]({{site.acc_baseurl}}/img/example.gif)  If  you select $ 1000 in this field, all entries with an amount of $1000 will  be displayed in the account activity browser.


![]({{site.acc_baseurl}}/img/note.gif)  The  amount in this field is denoted in the Base Currency. Therefore, the search  is carried out only in base currency.
