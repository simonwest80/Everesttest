---
title: Journal Group
---


Specify the journal group to which the transaction belongs. This field  defaults to the journal group specified in the **Workstation****Settings** dialog box (path: **File**  > **Setup** > **Preferences**  > **Workstation Settings** >  **Company Settings** tab > **Journal** **Group**  field) of the user creating the entry.
