# General Ledger (Detail)


Select this option (path: **P****osted Payment Journals** browser **&gt; Options &gt; Transactions &gt; General Ledger (Detail)**) to view details of the purchase documents that generated the payment entry. When this option is selected, you will see the [**Posted Payment Journal Transactions** browser]({{site.acc_baseurl}}/misc/the_posted_payment_journal_transactions_browser.html).
