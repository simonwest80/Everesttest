# General Ledger (Summary)


The **Posted Receipt Journal Summary** browser is displayed when **Options** > **General** **Ledger** (**Summary**) is selected in the **Posted** **Receipt** **Journals** browser.


This browser lists all the accounts debited or credited in the journal and the net amount of debit and credit for the account.


Options common to all browsers are available in this browser.
