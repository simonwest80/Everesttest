# The Posted General Journal Transactions Browser


This browser is displayed when the **Transactions** option (path: **Posted** **General Journals** browser **&gt; Options &gt; Transactions**) is selected. The **Posted General Journal Transactions** browser allows you to view details of the accounts debited or credited in the journal.


It also displays other details such as the amounts debited/credited, document details (for system generated general journals) and reference details.


The options common to all browsers in **Everest** are also available in this browser.
