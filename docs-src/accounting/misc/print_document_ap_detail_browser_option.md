# Print Document


This option allows you to print a document. When you select this option, you will see the [Document Printing Preferences dialog box]({{site.pp_chm}}/purc-proc/doc-profile/doc-options/prnt-purc-docs/the_document_printing_preferences_step_by_step.html), which allows you to view the document or print it.
