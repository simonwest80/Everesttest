# The Receipt Journal Summary Browser


This browser is displayed when the **General Ledger (Summary)** option (path: **Unposted Receipt Journals** browser > **Options &gt; Transactions &gt; General Ledger (Summary)**) is selected. The **Receipt Journal Summary** browser lists all the accounts debited or credited in the journal, and the net amount of debit and credit for the accounts.


Options common to all browsers are available in this browser.
