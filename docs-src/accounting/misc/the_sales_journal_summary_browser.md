# The Sales Journal Summary Browser


This browser is displayed when **Options** > **General** **Ledger** (**Summary**) is selected in the **Unposted** **Sales** **Journals** browser.


The **Sales Journals Summary** browser lists all the accounts debited or credited in the journal, and the net amount of debit and credit for the account.


The options common to all browsers are available in this browser.
