# Payment on Invoice


Select this option if you are receiving part payment or complete settlement  on an existing sales invoice.


When this entry type is selected:

- The **Accounts 
 receivable** account specified in the **Accounting**  dialog box (path: **File** > **Setup** > **Preferences** > **Defaults** > **Accounting**) can only be credited.
- The **Sales 
 Invoice** number must be specified in the **Document 
 #** column of the **Transaction Entries**  section.
- The **Credit**  column automatically displays the amount outstanding on the selected document.
