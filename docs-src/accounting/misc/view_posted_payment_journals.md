# View Posted Payment Journals


{:.steps}
To view the posted payment journals, do the  following:

- From the main  menu, select **Accounting &gt; Account 
 Payable &gt; Payment Journal**. You will see the **Unposted 
 Payment Journals** browser.
- From the **Unposted Payment Journals** browser, select  **Options &gt; Posted Journals**.  You will see the [**Posted Payment Journals** browser]({{site.acc_baseurl}}/misc/the_posted_payment_journals_browser.html).
