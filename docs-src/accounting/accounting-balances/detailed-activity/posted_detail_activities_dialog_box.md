# Posted Detail Activities Dialog Box


The **Posted Detail Activities**  dialog box displays a detailed activity report for the selected account  during a specified period. You can also view details of journals that  have NOT been posted.


**General Ledger Account**
: Select the account for which the detailed activity  is to be displayed. The account has to be a detailed account. The account's  description is displayed beside the account code.


**Account Currency**
: This field displays the currency in which the selected  account has been set up.
: ![]({{site.acc_baseurl}}/img/lens.gif) [Set  up Accounts in Foreign Currency]({{site.sc_chm}}/options/multicurrency/setup/set-up-accounts-in-foreign-currency/setup_accounts_in_foreign_currency.html)
: All amounts in the **Posted 
 Detail Activities** dialog box are by default, displayed in this  currency.


**Account type**
: This field identifies the account class; whether  the account is an asset, liability, capital, revenue or expense.


**Include Unposted**
: Select this check box if you also want to view the  details of journals that have NOT been posted. Provisionally posted journals  will also be displayed when you select this option.
: ![]({{site.acc_baseurl}}/img/lens.gif) [Provisional  Posting]({{site.acc_baseurl}}/provisionally-posting-journals/provisional_posting.html)


Base Currency


This section displays amounts in your base currency.


**Posted Balance**
: The total amount in base currency, posted to the  selected account is displayed here.


**Current Balance**
: The current balance (sum of the posted and unposted  balances) in base currency, of the account selected in the **General 
 Ledger Account** field is displayed here.


Account Currency


This section displays amounts in the currency in which the selected  account has been set up.


**Posted Balance**
: The total amount in account currency, posted to  the selected account is displayed here.


**Current Balance**
: The current balance (sum of the posted and unposted  balances) in account currency, of the account selected in the **General 
 Ledger Account** field is displayed here.


**Dates**
: Specify the transaction date in this field. All  transactions whose transaction date lies within the selected period are  included.
: Select **Custom** to  enter a specific date range.


**From**
: Specify the start date of transactions to be included  in the summary. This field is enabled only if you select **Custom** in the **Dates** field.


**To**
: Specify the end date of transactions to be included  in the summary. This field is enabled only if you select **Custom** in the **Dates** field.


**Apply**
: When you specify a date in the **From**  and **To** fields, click **Apply**  to view the transactions that meet the specified criteria.


Detailed Activity Transactions Grid
: The **Detailed Activity 
 Transactions** grid is displayed in the lower half of the **Posted 
 Detail Activities** dialog box. The details of all transactions matching  the specified criteria are displayed in this section.
: The **Reference #**  column may display a check number, credit card number or any other reference  that you enter. Credit card numbers are masked using the format XXXXXXXXXXXX<last  four digits of the card> when the **Enable 
 Extended Mode** check box is cleared (path: **File 
 &gt; Setup &gt; Preferences &gt; Workstation Settings &gt; Company Settings**  tab).


**Date**
: The date field in the **Detailed 
 Activity Transactions** grid allows you to further filter the transactions  that are displayed. All transactions whose transaction date is subsequent  to the date specified in this field are displayed in the detailed activity  transactions section.
