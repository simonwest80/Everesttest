# Journal Information


Journal information relates to the number, description and fiscal period  for the transaction. It is common to all transactions in the journal.


**Journal #**
: This field displays the journal number and is automatically  generated. For a new journal, the journal number is generated only when  the transaction details are saved. You can view other journals by selecting  journal numbers from the drop-down list.
: When a new purchase journal is created, the number  assigned is the following number in sequence specified for purchase journals.  This sequence number is specified in the **Location**  profile (path: **File** > **Setup** > **Accounting**  > **Location**/**Sub-Location** > **Options** > **Edit** > **In 
 New Window** > **Accounting**  tab > **Purchase journal #** field)  of the login location or the parent location of the login sub-location.
: If sequence numbers are not defined for the login  location or the parent location of the login sub-location, the sequence  number assigned to the default location defined in the **Accounting** dialog box (path: **File &gt; Setup 
 &gt; Preferences &gt; Defaults &gt; Accounting &gt; Miscellaneous tab 
 &gt; Default Location** field) is assigned to the document.
: ![]({{site.acc_baseurl}}/img/lens.gif) [Journals]({{site.acc_baseurl}}/accounting-structure-in-everest/journals/journals.html)
: [Default  Location]({{site.sc_chm}}/misc/default_location.html)


**Fiscal Year and Period**
: The fiscal year and period for the journal is displayed.  The transaction date of all accounting entries in the journal must fall  within this period. These fields cannot be modified once the journal is  saved.
: ![]({{site.acc_baseurl}}/img/lens.gif) [Fiscal  Year and Periods]({{site.sc_chm}}/options/acc-info/fiscal-year-and-periods/fiscal_year_and_periods_1.html)
: [Period  Status]({{site.sc_chm}}/options/acc-info/fiscal-year-and-periods/periods_status.html)


**Journal Group**
: This field identifies the group to which the journal  belongs. You can specify only one journal group for all the transactions  in a journal. This field may be left blank.
: ![]({{site.acc_baseurl}}/img/lens.gif) [Journal  Groups]({{site.sc_chm}}/options/acc-info/journal-groups/journal_groups.html)


**Description**
: The journal description helps the user to identify  the nature of transactions in the journal. This description is printed  on all applicable reports.
: ![]({{site.acc_baseurl}}/img/example.gif)  When  you are accounting for purchase of office supplies, you can enter the  description 'Purchases – Office Supplies'.


**Additional Information**
: Use the **Narration** tab to specify additional information about the journal. You can  enter only one narration for all the transactions in the journal.
: ![]({{site.acc_baseurl}}/img/example.gif)  You  may append notes on the journal such as the reason for creating the journal  or the nature of transactions recorded.
