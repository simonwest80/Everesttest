# View Posted Purchase Journals


{:.steps}
To view the posted purchase journals, do  the following:

- From the main  menu, select **Accounting &gt; Account 
 Payable &gt; Purchase Journal**. You will see the **Unposted 
 Purchase Journals** browser.
- From the **Unposted Purchase Journals** browser,  select **Options &gt; Posted Journals**.  You will see the [**Posted Purchase Journals** browser]({{site.acc_baseurl}}/misc/the_posted_purchase_journals_browser.html).
