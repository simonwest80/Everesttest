# Find Account Activity - More Choices


**Transaction amount**
: Specify the transaction amount based on which search  has to be performed.
: ![]({{site.acc_baseurl}}/img/lens.gif)[Transaction  Amount]({{site.acc_baseurl}}/find-account-activity/find-account-activity-details/more-choices/transaction_amount.html)


**Foreign currency amount**
: Specify the foreign currency amount based on which  search has to be performed.
: ![]({{site.acc_baseurl}}/img/lens.gif)[Foreign  currency amount]({{site.acc_baseurl}}/find-account-activity/find-account-activity-details/more-choices/foreign_currency_amount.html)


**Location/Sub-Location**
: Specify the location/sub-location based on which  search has to be performed.
: ![]({{site.acc_baseurl}}/img/lens.gif)[Location/Sub-Location]({{site.acc_baseurl}}/find-account-activity/find-account-activity-details/more-choices/department.html)


**Currency**
: Specify the currency based on which search has to  be performed.
: ![]({{site.acc_baseurl}}/img/lens.gif)[Currency]({{site.acc_baseurl}}/find-account-activity/find-account-activity-details/more-choices/currency_find_account_activity.html)


**Journal Group**
: Specify the journal group to base your search on.


**Smart Filter**
: Use the **Smart Filter**  to select accounts based on your segments. Wildcards can be used in this  field and up to 250 characters can be entered. **Segment 
 separators** must be entered between segments because accounts can  contain fewer characters than the segment length. Enter a semi-colon (;)  to separate accounts in the **Smart Filter**.


| Smart Filter Wildcard | Meaning |
| ? | One Character |
| \* | Multiple Characters |

: ![]({{site.acc_baseurl}}/img/lens.gif)[Smart  Filter]({{site.acc_baseurl}}/find-account-activity/find-account-activity-details/more-choices/smart_filter_accounting.html)
: ![]({{site.acc_baseurl}}/img/lens.gif)[Segmented\_Chart\_of\_Accounts]({{site.sc_chm}}/options/acc-info/coa/chart-of-accounts-details/segmented_chart_of_accounts.html)


**Account Type**
: Specify whether customer or vendor related transactions  have to be searched.
: ![]({{site.acc_baseurl}}/img/lens.gif) [Account  Type]({{site.acc_baseurl}}/find-account-activity/find-account-activity-details/more-choices/account_type_more_choices.html)
