# The Payment Register Wizard


The **Payment Register Wizard**  contains the following sections:

- [Welcome]({{site.acc_baseurl}}/payment-register/wizard/welcome_payment_register.html)
- [Bank  account Selection]({{site.acc_baseurl}}/payment-register/wizard/bank_account_selection_payment_register.html)
- [Vendor/Customer  Selection]({{site.acc_baseurl}}/payment-register/wizard/vendor_customer_selection_payment_register.html)
- [Payment  Method Selection]({{site.acc_baseurl}}/payment-register/wizard/payment_method_selection_payment_register.html)



**&nbsp;**You  can navigate within the wizard by clicking the **Next** button or double-clicking a section in the left panel.
