# Item Code and Description


<font color="#000000" class="hcp1">When you select an item in the </font>**Point 
 of Sale (sales invoice)**<font color="#000000" class="hcp1"> profile, item information 
 such as description, type, price and UM 
 are retrieved from the <span class="hcp2">Item</span>’s 
 profile and displayed in the respective columns in the <span class="hcp2">Item 
 Details Grid</span>.</font>


<font color="#800080" class="hcp3"><img src="{{site.pos_baseurl}}/img/lens.gif" x-maintain-ratio="TRUE" style="width:26px; height:27px;" width="26" height="27" border="0" class="hcp4"> </font>[**Items** Profile]({{site.mi_chm}}/create-regular-items-kits-and-assemblies/creating-an-item/setting_up_an_item.html)


You can select the item code manually or scan the item code into the  system using a bar code scanner. If the item’s serial number is scanned,  **Everest** automatically fills in  the item code.


{:.note}
<font color="#000000" class="hcp1">When an item is retrieved into a 
 document</font> the up-sell Information (path: **Inventory**  > **Items** > **Options 
 &gt; Edit &gt; In New Window** > **Notes**  tab) attached to the item is displayed<font color="#000000" class="hcp1">.</font>


The up-sell information is not printed along with the  **Point of Sale (sales invoice)**  profile. You cannot edit the information displayed in this window.
