# The Point of Sale Line Profile - Custom Fields Tab


The **Custom Fields** tab allows  you to define additional information as character strings, or date, logical,  memo or numeric fields. This **** tab  is displayed only when you save the profile.


![]({{site.pos_baseurl}}/img/example.gif) If a  Point of Sale document has an expiry date set in the "Date"  custom field, you can search for the document using the **Expiry 
 Date** field.
