# Save as Sales Order


Select this option to save a POS document as a sales order.


{:.example}
A customer places a request for a special  item not available at the time (may be a particular size). He pays a deposit  against the order. You can convert your POS document into a sales order,  apply the deposit and later retrieve the document and convert to invoice.
