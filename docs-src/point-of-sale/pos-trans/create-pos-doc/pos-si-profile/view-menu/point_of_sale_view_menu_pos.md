# Point of Sale - View Menu


Access the **View** menu  options from the main menu of the Point of Sale (sales invoice) profile.


**![]({{site.pos_baseurl}}/img/pos_view_options_pos.gif)**


**Figure: View Menu Options – Point of Sale Profile**


Zoom In


Select **View** >  **Zoom In** to hide the Categories  tab of the POS profile. Select **View**  > **Zoom Out** to view the Categories  tab again.


{:.note}
You can also click the **Zoom 
 In** and **Zoom Out** buttons  on the Status Bar of the document for this option.


Preview On


Select **View** >  **Preview On** to enter inline notes  for a line item in the **Description/Notes**  column of the **Item Details Grid**.  The inline note extending to more than one line appears in the line below  the line item when **Preview On**  is selected.


Select **View** >  **Preview Off** to hide the inline  notes.


{:.note}
You can also click the **Preview 
 In** and **Preview Out** buttons  on the Status Bar of the POS profile to preview or hide the inline notes.


Group Panel


Select **View** >  **Group Panel** to view the group  panel to group items by column. Drag and drop any column on the group  panel to group by that column.


Screen Properties


Select **View** >  **Screen Properties** to open the  Browser Properties window.


![]({{site.pos_baseurl}}/img/lens.gif) [Change  the Browser View]({{site.wwe_chm}}/everest-client/ui/browsers/options/property-editor/changing_the_browser_view.html)


Subtotal / Tax


Select **View** >  **Subtotal /Tax** for a display of  the Subtotal and Tax amounts in the POS profile. The Subtotal amount gives  the amount of the items before tax. The Tax amount displays the total  tax applicable on the items.


Outlook Bar


Select **View** >  **Outlook Bar** to view the Outlook  bar on the left of the POS profile.


![]({{site.pos_baseurl}}/img/lens.gif) [The  Point-of-Sale Outlook Bar]({{site.pos_baseurl}}/pos-trans/create-pos-doc/pos-si-profile/details/pos-outlook-bar/the_point_of_sale_outlook_bar.html)


Barcode Search


Select **View** >  **Barcode Search** to enable the **Barcode Search** field in the POS profile.  Scan the item’s barcode to enter the line item or type the item code in  the **Barcode Search** field and press  **Enter**.


![]({{site.pos_baseurl}}/img/lens.gif)[Barcode  Search Field]({{site.pos_baseurl}}/pos-trans/create-pos-doc/pos-si-profile/view-menu/barcode_search_field_point_of_sale_profile_pos.html)


Font


Select **View** >  **Font** for the **Font**  dialog box in which you can set your options for the font for the POS  profile.
