# Action


The **Action** field allows you  to search for activities by the action associated with the activity. The  search result displays the list of activities associated with the specified  actions.


Click the drop-down button to view the history of prior search parameter  values. If you select multiple actions, the word 'multiple' is displayed  in this field in a URL link format.
