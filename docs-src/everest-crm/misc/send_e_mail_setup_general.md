# Send E-mail Setup - General


General


**This section displays the process and the  activity to which the action is associated.**


**Template Description**
: Select a template to be used for this e-mail, from  the drop-down list. This field is dependent on event group and the [type of template](javascript:BSSCPopup('{{site.crm_baseurl}}/misc/template_types.html');){:id="A1"}<script type="text/javascript" language="JavaScript1.2"><!--
kadovFilePopupInit('A1');
//--></script>.  In other words, the drop-down list will display only those templates that  are defined for a particular event group and of a certain type. Click  the Notes ![]({{site.crm_baseurl}}/img/crm_notes_icon.gif) icon next to the field to view the body of the  template.


**'From' E-mail Address**
: Select the e-mail address that should be used as  the **From** e-mail address when an  e-mail is sent. The host name is displayed next to the field.


Send E-mail to Group of
: You can send e-mails to a group of customers or  vendors. These options are filter based, i.e., you have to specify the  criteria and find customers/vendors to whom you have to send the mail.  When you check the **Customers** or  **Vendors** option, the Find buttons  get enabled. Click these buttons and set the criteria for the filter.  E-mail will be sent to those customers/vendors group who fall within the  filtered group.

: To send attachments along with the e-mail, select  the check box and specify the file you want to attach in the field below.
