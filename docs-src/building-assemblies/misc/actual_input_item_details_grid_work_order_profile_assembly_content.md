# Actual Input


This column displays the actual quantity of the item that has been attached  to the document. The quantity input is the gross quantity before process  loss. You can enter the actual quantity used for production in this column.  When the work order is completed, this column indicates the quantity utilized  in production.


![]({{site.ba_baseurl}}/img/example.gif) Â If  100 units are input out of which 8 units are lost or wasted in the process,  the net quantity that goes into the assembly is 92. The quantity entered  in the **Actual** **Input**  column is 100 and not 92. The difference between the standard and actual  input results in quantity variances.
