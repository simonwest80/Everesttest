# Inline Notes


You can specify additional information on the work order through the  **Inline Notes** option. This information  is added as a line in the Item Details Grid section if you click the **Expand to Document** button ![]({{site.ba_baseurl}}/img/ba_expand_to_document.gif) in the **Description**  dialog box.


{:.steps}
To add inline notes, do the following:

- From the **Work Order** profile, select **Options**  > **Inline Notes**. You will see  the **Description** dialog box.
- Enter the required  note and select **File** > **Save/Save and Close** to return to the  work order.
