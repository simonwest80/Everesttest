# View and Edit Document Notes


{:.steps}
To define, view and edit document notes,  do the following:

- From the **Work** **Order**  profile, click the **Notes** ![]({{site.ba_baseurl}}/img/ba_notes_icon.gif) button. You will see the **Document****Notes** window.
- Enter relevant  notes.
- Select **Options** > **Save**.
