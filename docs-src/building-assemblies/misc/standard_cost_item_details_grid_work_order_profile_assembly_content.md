# Standard Cost


This column displays the standard cost of the item specified in the  Bill of Material (path: **Inventory** > **Item** > **Options** > **Edit** > **In New Window** > **Assembly** > **Input** tab).  The standard cost is shown in the cost price decimal accuracy of the item.
