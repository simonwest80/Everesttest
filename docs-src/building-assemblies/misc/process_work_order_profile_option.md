# Process


The **Process** option allows you  to:

- Change the  status of a work order to "[In  Process]({{site.ba_baseurl}}/prod-asm/work-in-proc/process_work_orders_work_order_profile_assembly_content.html) ".
- Mark the work  order as [built]({{site.ba_baseurl}}/prod-asm/building-wo/building_work_orders_content_building_assembly.html)  once the assembly components have been processed to the finished goods.
- [Cancel]({{site.ba_baseurl}}/prod-asm/cancelling-wo/cancel_a_work_order_work_order_profile_options_process_cancel_assembly.html)  a work order before it is built.
- [Void]({{site.ba_baseurl}}/prod-asm/voiding-a-work-order/void_a_work_order.html)  a work order
