# Document Information


Document information for a work order are listed below:


**Document Alias**
: The **Document Alias** button ![]({{site.ba_baseurl}}/img/ba_document_alias_button.gif) allows you to set a document alias for the  work order. You can define different aliases for different types of jobs.  You can assign a different document alias to a work order, until you select  the [In Process](javascript:BSSCPopup('{{site.ba_baseurl}}/misc/in_process_wo_status_assembly_content.html');){:id="A1"}<script type="text/javascript" language="JavaScript1.2"><!--

: [Document  Alias]({{site.bp_chm}}/doc-aliases/document_aliases_businesss_process_in_everest_content.html)


**Document Notes**
: Click the **Notes**  button ![]({{site.ba_baseurl}}/img/ba_notes_icon.gif) to enter notes for the work order in order to track  important messages or information pertaining to the work order. This is  not printed with the work order. You can enter notes for work orders until  you select the [**Build**](javascript:BSSCPopup('{{site.ba_baseurl}}/misc/build_wo_status_assembly_content.html');){:id="A2"}<script type="text/javascript" language="JavaScript1.2"><!--

**Work Order Status**
: The status of a work order is displayed beside the  **Document #** field.
: ![]({{site.ba_baseurl}}/img/lens.gif) [Work  Order Status]({{site.ba_baseurl}}/prod-asm/creating-wo/wo-details/work_order_status_work_order_profile_assembly_content.html)


**Document Number**
: This field displays the document number assigned  automatically to the work order when it is saved. You cannot edit this  field.
: The next number in sequence assigned for work orders  in the logged in locationâ€™s profile (path: **File**  > **Setup** > **Accounting**  > **Location/ Sub-Location** >  **Options** > **Edit**  > **In New Window** > **Inventory** tab > **Work 
 Order #** field) or the location of the logged in sub-location is  assigned to the work order.
: If sequence numbers are not specified here, **Everest** assigns the sequence numbers  specified for the default location (path: **File** > **Setup** > **Preferences**  > **Defaults** > **Accounting**  > **Miscellaneous** tab > **Default Location** field).


{:.note}
The document number will be assigned to the  work order only after the work order is saved.
