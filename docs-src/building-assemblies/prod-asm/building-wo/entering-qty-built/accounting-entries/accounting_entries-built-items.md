# Accounting Entries - Built Items


The entries created to account for the assembled/built items and the  cost of production are:


| **Account** | **Path** | **Debit/**<br/><br/><br/>**Credit** | **Amount** |
| Inventory Control account | The account selected in the **Accounting**  dialog box (path: **File** > **Setup** > **Preferences**  > **Defaults** > **Accounting**  > **Accounts** tab > **Inventory <br/> control** field). | Debit | The total value of actual output and scrap. |
| Asset or Expense account | The account selected in the **Work Order** profile (path: **Inventory**  > **Bill of Material &gt; Work Order** > **Options** > **Edit** > **In <br/> New Window** > Item Details Grid > **Expense** **Account** column). | Credit | The actual value of non-inventoried components |
| Abnormal Loss account | The account selected in the **Accounting**  dialog box (path: **File** > **Setup** > **Preferences**  > **Defaults** > **Accounting**  > **Accounts** tab > **Abnormal <br/> Loss** field). | Debit | The value of abnormal loss. |
| Work in Progress account | The account selected in the **Accounting**  dialog box (path: **File** > **Setup** > **Preferences**  > **Defaults** > **Accounting**  > **Accounts** tab > **Work <br/> In Progress Account** field). | Credit | The value of work in progress |
| Abnormal Gain account | The account selected in the **Accounting**  dialog box (path: **File** > **Setup** > **Preferences**  > **Defaults** > **Accounting**  > **Accounts** tab > **Abnormal <br/> Gain** field). | Credit | The value of abnormal gain |
