# Extended Notes


This option allows you to view the **Extended Notes** window for the selected line item. This window displays the notes as entered in the **Notes and Specifications** field of the **Item** profile (path: **Inventory** > **Items** > **Options** > **Edit** > **In New Window** > **Notes** tab).
