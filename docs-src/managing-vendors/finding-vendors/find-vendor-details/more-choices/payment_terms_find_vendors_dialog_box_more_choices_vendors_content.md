# Payment Terms


**You can search for vendors by specifying  the payment term type defined in the **Vendor**  profile.**


![]({{site.mv_baseurl}}/img/example.gif) Â If  you choose the payment term 'CC', all the vendors with the payment term  'CC' are displayed.


{:.steps}
To select  a payment term of the vendor, do the following:

- Click the browse  ![]({{site.mv_baseurl}}/img/managing_vendors_selection_button.gif) icon beside the **Payment 
 Terms** field. You will see the **Selection**  dialog box.
- From the **Selection** dialog box, select the payment  term(s) of your choice.
- Click **OK**. The **Payment 
 Terms** field is populated with the relevant values.



**If  you select multiple payment terms the word 'multiple' is displayed in  this field in a URL link format. Click the drop down button to  view the history of prior search parameter values.**
