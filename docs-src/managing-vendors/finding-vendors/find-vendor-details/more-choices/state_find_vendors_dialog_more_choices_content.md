# State


You can search for vendors based on the state specified in the billing/  shipping/contact addresses.


If an address type has been selected in the **Address 
 Type** field of the **Quick Find**  tab, vendors whose address (of the selected type) contains the specified  state are displayed.


If an address type is NOT selected, vendors whose address (billing/contact/shipping)  contains the specified state are displayed.


Multiple selections are not allowed for this field. Click the drop-down  button to view the history of prior search parameter values.
