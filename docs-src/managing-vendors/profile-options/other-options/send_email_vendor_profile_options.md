# Send E-mail


Use this option to send an e-mail  to a vendor using **Everest E-mail**.  It displays the **Untitled-New Message** window, with the selected vendor's e-mail  address in the **To** field. You must  specify the e-mail  address of the vendor in the **General** tab of the **Vendor** profile.
