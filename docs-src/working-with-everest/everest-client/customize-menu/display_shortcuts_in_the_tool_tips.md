# Display Shortcuts in the ToolTips


Shortcuts are keyboard strokes for a command. Shortcuts can be displayed  in the ToolTips  by selecting the **Show shortcut keys in 
 ToolTips**check  box in the **Options** tab of the  [Customize dialog box]({{site.wwe_baseurl}}/misc/customize_dialog_box.html).
