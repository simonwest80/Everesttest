# Add/Remove Menus from the Main Menu Bar


**Adding Menus to the Main Menu Bar**


{:.steps}
To add a menu to the main menu bar, do the  following:

- Right-click  on any toolbar and select **Customize** from  the drop-down list. You will see the **Customize** dialog box.
- From the **Customize** dialog box, select the **Commands** tab.
- Select the  required category from the **Categories**  panel and then the required command from the **Commands**  panel. Drag and drop it on the main menu bar.



**Removing Menus from the Main Menu Bar**


Menus can be removed from the main menu bar in any of the following  ways:

- Dragging and  dropping the menu into the **Customize**  dialog box
- Using the **Delete** option



{:.steps}
To hide a menu in the main menu bar, do the  following:

- Right-click  on any toolbar and select **Customize** from  the drop-down list. You will see the **Customize** dialog box.
- Click on the  menu to be removed from the toolbar and drag and drop it into the **Customize** dialog box.



Notice that the menu will no longer be displayed in the main menu bar.


{:.steps}
To delete a menu from the main menu bar,  do the following:

- Right-click  on any toolbar and select **Customize** from  the drop-down list. You will see the **Customize** dialog box.
- Right-click  on the required menu in the **main menu 
 bar** and select **Delete** from  the drop-down menu.



{:.example}
To remove the **Utility**  menu from the **main menu bar**, right-click  on **Utility** and select **Delete**  from the drop-down menu.


{:.hint}
When all options in the main menu bar such  as **File**, **Inventory**,  and **Invoicing** are removed by customizing,  they can be restored by doing the following:

- Invoke the  **Customize** dialog box (right-click  on any toolbar and select **Customize**  from the drop-down menu).
- Select the  **Commands** tab. The **Commands**  panel displays all the menu options of the main menu bar.
- In the **Categories** panel, select the **Menus**  option.
- Drag the required  menus from the **Commands** panel  onto the main menu bar.
