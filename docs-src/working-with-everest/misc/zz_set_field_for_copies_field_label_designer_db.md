title: Set Field for Copies
---


Select a field, on which you want to base the quantity of bar code labels,  to be printed. The drop-down list displays the numeric fields from the  related browser.


{:.example}
If you need to print bar code labels for your  entire stock of a certain item, highlight the item and select **Label**  > **Design** from the **Items**  browser. In the **Label** **Designer**,  select **Total** **Stock**  in the **Set** **field****for** **Copies**  field.
