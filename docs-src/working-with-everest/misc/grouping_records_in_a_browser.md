# Grouping Records in a Browser


You can group records in a browser in any of the following ways:

- **Drag-and-drop** - Open the browser you want to group and point the cursor to the  required [column header](javascript:BSSCPopup('{{site.wwe_baseurl}}/misc/column_header_1.html');){:id="A1"}<script type="text/javascript" language="JavaScript1.2"><!--
kadovFilePopupInit('A1');
//--></script>.  Then drag and drop the column header onto the **Group 
 panel**.



{:.note}
Dragging and dropping column headers to group  the records is possible only if the **Group 
 Panel** is displayed. You should have the [Show  Group Panel]({{site.wwe_baseurl}}/everest-client/ui/browsers/options/property-editor/browser_property_editor_grid_tab.html) check box selected in the **Browser 
 Properties** to view this section of the browser.

- **Using 
 the Group by this column option** - To group the records of a browser  with this option, right-click on the required column header and select  **Group by this column**.



{:.example}
To review sales performance over time, you  can view the sales generated by all sales representatives per year and  further group the sales quarter-wise. Additionally, you can view the total  sales per quarter and year by specifying the **Summary 
 Type** as "sum" for the **Amount**  column. Click **Show Me** to view  a multimedia presentation that displays how this can be done.


![]({{site.wwe_baseurl}}/img/avi.gif) [Show Me]({{site.wwe_baseurl}}/misc/group_records.html){:target="_blank" }


The records in the browser may be grouped based on the column headers.  These groupings are indicated by the **Group 
 By Box**.
