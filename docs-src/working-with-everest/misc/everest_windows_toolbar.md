# Everest Windows Toolbar


The **Everest Windows** toolbar  contains the following options:


| Icon | Name | Description |
| ![]({{site.wwe_baseurl}}/img/wwe_windows_toolbar_options_large_icons.gif) | Large icons | The options are displayed as large icons in the right  panel of **Everest Windows**. |
| ![]({{site.wwe_baseurl}}/img/wwe_windows_toolbar_options_small_icons.gif) | Small icons | The options are displayed as small icons in the right  panel of **Everest Windows**. |
| ![]({{site.wwe_baseurl}}/img/wwe_windows_toolbar_options_list.gif) | List | The options are listed vertically in the right panel  of **Everest Windows**. |
| ![]({{site.wwe_baseurl}}/img/wwe_windows_toolbar_options_details.gif) | Details | Details of each option are listed along with the name  in the right panel of **Everest Windows**. |
