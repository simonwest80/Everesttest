# Modify a Classic Style Filter


{:.steps}
To modify a classic style filter, do the following:

- From the required  browserâ€™s toolbar menu, click the **Expert 
 Style Filter** ![]({{site.wwe_baseurl}}/img/wwe_standard_browser_options_expert_style_filter.gif) icon. You will see the **Filters**  dialog box.
- Select the  required **Classic Style** filter  from the **Top (Display) Panel**.
- Click the **Modify** button beside the **Filter 
 Details Panel**. You will see the **Specify 
 Filter** dialog box.
- Modify the  filter according to your requirements and click **OK**.
