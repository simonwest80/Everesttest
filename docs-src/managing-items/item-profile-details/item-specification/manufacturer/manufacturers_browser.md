# The Manufacturers Browser


The **Manufacturers** browser (path: **Inventory &gt; Manufacturers**) displays  a list of all manufacturers created in **Everest**.  The following option (path: **Manufacturers**  browser > **Options**) is available  from the **Manufacturers** browser:


**E-Commerce**
: Use this option to access the **E-Commerce 
 setup - Manufacturers** dialog box wherein you can web-enable the  items. From this dialog box you can:

- Attach audio  and video files.
- Specify the  images and banners to be displayed.
- Specify other  sales information for the items on the online shop.
- Enter a detailed  description for the item.
- Specify hyperlinks  for banners.

: ![Lens.gif]({{site.mi_baseurl}}/img/lens.gif) [Web  Enable a Manufacturer](building-an-online-store.chm::/web_enable_manufacturers.htm)


Options common to all browsers are also available in this browser.
