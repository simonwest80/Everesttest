# The Categories Browser


The **Categories** browser (path:  **Inventory &gt; Categories**) displays  a list of the item categories created in **Everest**.  The options (path: **Categories**  browser > **Options**) available  in the **Categories** browser are:


**List of Items**
: Use this option to view a list of the items that  belong to the selected category in the **Items**  browser
: ![Lens.gif]({{site.mi_baseurl}}/img/lens.gif) [The  Items Browser]({{site.mi_baseurl}}/the-items-browser/the_items_explorer.html)


**E-Commerce**
: Use this option to access the **E-Commerce 
 Setup - Category** dialog box wherein you can web-enable the items.  You can:

- Attach audio  and video files.
- Specify the  images and banners to be displayed.
- Enter a detailed  description for the category.
- Specify hyperlinks  for banners.

: ![Lens.gif]({{site.mi_baseurl}}/img/lens.gif) [Web  enable Category](building-an-online-store.chm::/web_enable_category.htm)


**Custom Fields**
: The **Custom Fields**  feature allows you to define additional information such as character  strings, date fields, logical fields or numeric fields.


![Lens.gif]({{site.mi_baseurl}}/img/lens.gif) [Custom  Fields]({{site.sc_chm}}/options/miscellaneous-set-up/custom-fields/custom_fields_setupco.html)


Options common to all browsers are also available in this browser.
