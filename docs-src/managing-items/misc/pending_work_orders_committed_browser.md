# Pending Work Orders Committed Browser


The **Pending Work Orders Committed**  browser displays a list of work orders that contain committed quantities  of the assembly item. The number and the date of the work orders can be  viewed in this browser. The **On Order 
 Committed** column displays the quantity that is committed on the  order.


Options common to all browsers are also available in this browser.
