# Related Items Browser


The **Related Items** browser displays  a list of related items that are defined for an item. All sales measures  listed are in the measure selected in the **Related 
 Items** profile. Options other than the standard options are detailed  below:


**All related items**
: Use this option to view all the items that are related  to the item selected.


**All related items with Available Stock > 0**
: Use this option to view the related items whose  available stock is greater than zero.


**All related items with <Free Stock> >  0**
: Use this option to view the related items whose  free stock is greater than zero.


{:.note}
The name <Free Stock> is derived from  the **Stock Quantity Name** field  that you mentioned in Items dialog box (path: **File**  > **Setup** > **Preferences**  > **Flow Control** > **Items**  > **Stock Quantities** tab).


**All related items with Net Free Stock > 0**
: Use this option to display the related items whose  net free stock is greater than zero.


**All related items with Total Stock > 0**
: Use this option to display the related items whose  total stock is greater than zero.


{:.note}
The name <Net Free Stock> is derived  from the **Stock Quantity Name** field  that you mentioned in Items dialog box (path: **File**  > **Setup** > **Preferences**  > **Flow Control** > **Items**  > **Stock Quantities** tab).


Standard options common to all browsers are also available in this browser.
