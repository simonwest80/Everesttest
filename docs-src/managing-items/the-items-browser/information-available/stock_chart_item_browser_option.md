# Stock Chart


Use this option (Path: **Item Browser 
 &gt; Options &gt; Stock Chart**) to view a graphical representation  of itemsâ€™ stock status. This chart details total stock, available stock,  reserved stock, stock on debit, stock on order and stock on credit.


All the standard chart toolbar options are available in the **Stock 
 Chart**. All quantities are shown in the stock measure.
