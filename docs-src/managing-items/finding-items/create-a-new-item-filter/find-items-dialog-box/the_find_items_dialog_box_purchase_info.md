# The Find Items Dialog Box - Purchase Info


Use this tab to search for items using their purchase information.


**Last Cost**
: Specify the last cost of the item.
: ![]({{site.mi_baseurl}}/img/lens.gif) [Last  Cost]({{site.mi_baseurl}}/finding-items/find-item-details/purchase-information/last_cost_find_items_content.html)


**Average Cost**
: Specify the average item cost.
: ![]({{site.mi_baseurl}}/img/lens.gif) Average Cost


**High Cost**
: Specify the item’s high cost.
: ![]({{site.mi_baseurl}}/img/lens.gif) [High  Cost]({{site.mi_baseurl}}/finding-items/find-item-details/purchase-information/high_cost_find_items_content.html)


**Low Cost**
: Specify the item’s low cost.
: ![]({{site.mi_baseurl}}/img/lens.gif) [Low  Cost]({{site.mi_baseurl}}/finding-items/find-item-details/purchase-information/low_cost_find_items_content.html)


**Default Measure**
: Specify the item's default purchase measure.
: ![]({{site.mi_baseurl}}/img/lens.gif) [Default  Measure]({{site.mi_baseurl}}/finding-items/find-item-details/purchase-information/measure_find_items_content.html)


**Put Away Bin**
: Specify the item’s put away bin.
: ![]({{site.mi_baseurl}}/img/lens.gif) [Stock  Area]({{site.mi_baseurl}}/finding-items/find-item-details/purchase-information/stock_area_find_items_content.html)


**Default Vendor**
: Specify the item’s default vendor.
: ![]({{site.mi_baseurl}}/img/lens.gif) [Default  Vendor]({{site.mi_baseurl}}/finding-items/find-item-details/purchase-information/default_vendor_find_items_content.html)


**Notes**
: Specify the item’s extended purchase description.
: ![]({{site.mi_baseurl}}/img/lens.gif) [Notes]({{site.mi_baseurl}}/finding-items/find-item-details/purchase-information/notes_find_items_content.html)
