# The Update Price Differentials for Kits Browser


The **Update Price Differentials for 
 Kits** browser (path: **Utility &gt; 
 Update Price Differentials &gt; Kits**) displays a list of all kit  updates created in **Everest**. The  options available in the **Update Price 
 Differentials for Kits** browser are detailed below.


**Process**
: Click the **Process**  button to update the kit price. The prices are updated as specified in  the **Related Items** profile (path:  **Inventory** > **Items**  > **Options** > **Related 
 Items**).


Options common to all browsers are also available in this browser.
