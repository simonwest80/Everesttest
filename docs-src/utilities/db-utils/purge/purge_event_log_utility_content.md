# Purge Event Log


All events created in **Everest**  are displayed in the **Events** browser.  You can use the **Purge** **Event****Log** **dialog**  box to permanently delete these records. Events relating to documents  will not be purged.


{:.steps}
To purge an event log, do the following:

- From the main  menu bar, select **Utility** >  **Purge** > **Event****History**. You will see the [**Purge Event Log** Dialog Box]({{site.utl_baseurl}}/misc/purge_event_log_dialog_box.html).
- Specify a date  in the **Remove the History Before**  field. All events up to and including the date specified here will be  purged.
- Click **Start Purge** button. The number of records  that are purged are displayed in the dialog box.



![]({{site.utl_baseurl}}/img/example.gif) Specify  the date as 10th October 2001 in the **Purge 
 Event** **Log** dialog box, all  event logs created before this date will be purged.
