# View the Matrix Import Items


{:.steps}
To view the matrix items that you imported,  do the following:

- Select **Inventory**  > **Item Matrix** > **Matrix 
 Groups**. You will see the **Item 
 Matrix Groups** browser.
- Click the Â **Refresh**  button. The matrix group that you created is listed in the browser.
- Select the matrix  group and then select **Options**  > **View Matrix**. The **Matrix 
 Items View** dialog box displays with the matrix items.
- Select **Options**  > **View Items**. The **Matrix 
 Items** browser displays with all the matrix items.
