---
title: Filter
---


The **Filter** button allows you to filter unwanted data in the source data file. You can filter for special characters such as commas, semi colon, colon or any user defined character from the source data to be able to import clean data.
: You can change the format of the source data characters by converting into:

- <font face='Symbol'> </font>Upper Case
- <font face='Symbol'> </font>Lower Case
- <font face='Symbol'> </font>Proper Case
- <font face='Symbol'> </font>Smart Case
