---
title: Item for negative tax
---


Specify the item code for adjusting the negative difference (the tax  calculated in **Everest** for the  transaction is more than the tax calculated in the online shop). This  is a mandatory field.


{:.note}
Only a non-inventoried item code must be selected  in this field.
