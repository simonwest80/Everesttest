title: Settings

The **Settings** button allows you to save the Map field settings to a .Map file for later use.
: ![]({{site.utl_baseurl}}/img/example.gif)  When you import item categories, you can save the field mappings in a file. When you import other categories into **Everest**, you can load the Map Field Settings from this file and the fields will be mapped automatically.
