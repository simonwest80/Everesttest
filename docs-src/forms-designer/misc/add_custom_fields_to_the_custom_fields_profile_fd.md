# Add Custom Fields to the Custom Fields Profile


{:.steps}
To add your custom fields to the Custom fields  profile, do the following:

- Click a custom  field to the right and place your cursor over the area where you would  like to place the custom field. You will see a target cursor.



![]({{site.fd_baseurl}}/img/field_chooser_add_custom_fields_stepr1_fd.gif)


**Figure: Target Form showing Target Cursor**

- Click the target  form in the Group Box to place the custom field on the form. Add the remaining  fields.



![]({{site.fd_baseurl}}/img/field_chooser_add_custom_fields_stepr2_fd.gif)


**Figure: New Custom Field Added to the Target  Form**


![]({{site.fd_baseurl}}/img/new_custom_field_profile_design_steps_fd.gif)


**Figure: New Custom Field Profile Design**
