# Purchase Document Lines Dialog Box - Custom Fields


Use the **Custom Fields** tab to  define additional information as character strings, or date, logical,  memo or numeric fields for each line item on the document. Custom fields  defined in the **Purchase Document Lines**  dialog box â€“ **Custom Fields** tab  apply to all purchase document line items.


{:.example}
If purchase document line items have different  priorities, create a **Priority**  character or numeric custom field to track this information.


Custom fields defined in the **Item**  profile do not carry forward to the document line items.


{:.example}
A **Color**  custom field is defined only in the **Item**  profile. When adding a line item to a purchase order, the **Color**  field does not display.
