**CC**
: Enter the email ID of a recipient who should receive  a copy of this mail. This field displays the default e-mail ID entered  in the vendor's default billing address (path: **Purchasing 
 &gt; Vendors &gt; Options &gt; Edit &gt; In New Window &gt; General** tab).


{:.note}
If you specify the e-mail ID as per the vendor's billing  address in the **To** field, this  field remains blank.
: Click the ![]({{site.pp_baseurl}}/img/pur_cc_button.gif) button and you will see  the [**Address 
 Book** dialog box]({{site.pp_baseurl}}/misc/address_book_dialog_box_email_option_pur_doc.html). You can specify additional addresses from  this dialog box.


**Subject**
: Enter an appropriate subject for the mail. This  field, by default, displays the document alias defined for the document(s)  in the document profile. The document number(s) is/are also displayed  in this field.
