# Multimedia Information


The **Multimedia** tab allows you  to attach images, sound, video, documents, spreadsheets, and other files  to a purchase document. You can store these files in the multimedia repository.


{:.note}
This tab is displayed only when you save the  purchase document.


{:.example}
You can attach a video of the product demo,  the vendor's price list for all items in this category, or the vendor's  product catalog.


![]({{site.pp_baseurl}}/img/lens.gif) [Multimedia  Repository]({{site.sc_chm}}/options/miscellaneous-set-up/multimedia/multimedia_repository.html)


![]({{site.pp_baseurl}}/img/steps.gif)To attach a multimedia file to a purchase  document, do the following:

- From the browser  in the **Multimedia** tab of the purchase  document **** profile, right-click  and select **New**. A new record is  added to the browser.
- Click the **Multimedia File** column of the new record  to enable the lookup icon ![]({{site.pp_baseurl}}/img/pur_look_up_button.gif).
- <font color="#000000" class="hcp6">Click the lookup button and select the required multimedia 
 file. The other columns in the browser are automatically updated. </font>
- <font color="#000000" class="hcp6">Save the details.</font>
