# Item Information


Items in a document are listed in the â€˜Item Details Grid'. The item  details grid displays the item(s) attached to a document in a tabular  format. Each row in the grid displays information about the item specified  in the **Item** **Code**  column. Items can be purchased in multiple units of measure.


{:.note}
- You may create  purchase quotes, debit quotes, purchase orders, and debit orders without  specifying item details; they can be entered later. However, you must  select at least one item in a purchase receipt, purchase invoice, or debit  memo.
- Adequate security  right is required when accessing items of a location/sub-location that  is different from logged in location/sub-location and when accessing items  that are not explicitly associated with the selected vendor.


**Entering Item Information**


You can enter item information in a purchase or purchase return document  in any of the following ways:

- **Retrieve from Document**  - Retrieve item information from another purchase quote, order, receipt  or invoice, sales quote order or receipt, or credit memo.
- **Select Items** - Select the items  from the **Items** browser (path:  purchase/purchase return document profile > **Options 
 &gt; Items &gt; Select Items**).
- **Bar 
 Code Scanner** - Scan the item code or serial number into the system  using a bar code scanner.
- **Items 
 Browser** - Drag and drop the items from the **Items**  browser (path: **Inventory &gt; Items**).
- Item  Details Grid
- Enter the item code  in the **Item Code** column.
- Type the first few  letters of the item code in the **Item 
 Code** column, and select the item from the **Smart Search**  dialog box.
