# Due Date


If the delivery date for a line item is different from the document  delivery date as specified in the [Delivery  Date]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/tabs/details/delivery-information/delivery_date.html) field of the **Details**  tab, enter a date in this column.


You can enter a date in this field either in a purchase quote or an  order.
