# Purchase Order Profile Options


The options available in a purchase order (path: **Purchase 
 Order** profile > **Options**)  are:


**Payment History**
: The [Payment History  browser]({{site.pp_baseurl}}/purc-proc/pos/po-processes/payments/payment-history/payment_history_pur.html) **** (path: **Options** > **Payment** > **Payment****History**) displays payment/refund  details for a purchase document.


{:.note}
This option is enabled only if a payment is made on  the purchase order.


****Show all Serial 
 #****
: The **Show****all** **serial****#** option (path: **Options**  > **Show** **all****serial** **#**)  displays all serial numbers attached to the document.


{:.note}
The **Serial # for all 
 Items** browser displays serial numbers only if they are attached  to the document.


**Show/Attach Area-wise Stock**
: The **Show/Attach****Area-wise** **stock**  option (path: **Options** > **Attach**/**Retrieve****Serial** **#**  > **Show/Attach** **Area-wise****stock**) allows you to allows you  to attach negative quantities of non-serialized items.


{:.note}
This option is available only for inventoried items  that are not serialized.
: ![]({{site.pp_baseurl}}/img/lens.gif) Â [Enter  Negative Quantities]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/item-info/details/add-retrieve-items/entering_negative_quantities_in_purchase_documents.html)


**Show serial  number**
: The **Show****Serial** **Number**  option (path: **Options** > **Item** > **Show****Serial** **#**)  displays all the serial numbers that are attached to a selected item.


**Serial number history**
: The **Serial****number** **history**  option (path: **Options** > **Serial** **History 
 History**)  displays a history of movement of a serial number between different stock  areas and the documents in which the movement was recorded. It displays  the date on which a serial number was generated, the date it entered stock,  the purchase order or purchase receipt on which it was generated, the  stock area to which it was received, the sales order to which it is attached  and the area it was moved to when it was attached etc.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Serial  Number History Browser]({{site.pp_baseurl}}/purc-proc/pos/po-processes/cmt-items-po/enter-cmt-qty/serial-number-history-browser/serial_number_history_browser_pur_con.html)


**Detach Serial #**
: This option allows you to detach the serial number  attached to an item in the purchase order. Enter the serial number of  the item in the **Detach serial #**  field.
