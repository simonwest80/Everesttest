# Brand


A brand is a trademark or distinctive name identifying a product. If  you have specified the brand of an item in the **Brand**  field of the **Item** profile (path:  **Inventory &gt; Items &gt; Options &gt; 
 Edit &gt; In New Window &gt; General** tab), the code for the same  is displayed in the **Brand** column.  You cannot edit this column.
