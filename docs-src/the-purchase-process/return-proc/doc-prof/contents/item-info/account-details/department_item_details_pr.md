# Loc/Sub-Loc


The login location or sub-location is the displayed by default in the  **Loc/Sub-Loc** column of the [line item](javascript:BSSCPopup('{{site.pp_baseurl}}/misc/line_item_1.html');){:id="A1"}<script type="text/javascript" language="JavaScript1.2"><!--
kadovFilePopupInit('A1');
//--></script>  for the item selected. This is the location to which the items are received.


Non - inventoried Items


Select another location or sub-location if the item is non-inventoried  and the expense has to be allocated to any other cost center or location/sub-location.


![]({{site.pp_baseurl}}/img/lens.gif) [Non-inventoried  Items]({{site.mi_chm}}/item-profile-details/inventory-type/non-inventoried/non_inventoried.html)


Change the Document’s Location/Sub-Location


If you change the document's location/sub-location you can either update  all line item’s locations/sub-locations with the <location/sub-location>  selected in the **Loc/Sub-Loc** field  or choose to retain the location/sub-location selected for the items in  the Item Details Grid. You should have adequate security rights to retain  the line item location/sub-location (that is different from the document’s  location/sub-location).


![]({{site.pp_baseurl}}/img/lens.gif) [Document's Location/Sub-Location]({{site.pp_baseurl}}/return-proc/doc-prof/contents/document-information/department_pr1.html)
