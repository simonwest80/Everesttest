# Multimedia Information


The **Multimedia** tab allows you  to attach images, sound, video, documents, spreadsheets, and other files  to a purchase return document. You can store these files in the multimedia  repository.


{:.note}
This tab is displayed only when you save the  purchase return document.
Â 


{:.example}
You can attach audio files of your communication  with the vendor, a document containing a detailed explanation of why you  returned the product, etc.


![]({{site.pp_baseurl}}/img/lens.gif) [Multimedia  Repository]({{site.sc_chm}}/options/miscellaneous-set-up/multimedia/multimedia_repository.html)


{:.steps}
To attach a multimedia file to a purchase  return document, do the following:

- From the browser  in the **Multimedia** tab of the purchase  return document **** profile, right-click  and select **New**. A new record will  be added to the browser.
- Click on the  **Multimedia File** column of the  new record to enable the lookup icon ![]({{site.pp_baseurl}}/img/pur_look_up_button.gif).
- <font color="#000000" class="hcp7">Select the required multimedia file by clicking on the 
 lookup icon. The other columns in the browser are automatically updated. 
 </font>
- <font color="#000000" class="hcp7">Save the details.</font>
