# Document Tracking Information


The **Document Tracking** tab allows  you to tag a purchase return document with the appropriate tracking selections  for the groups available.


{:.note}
This tab is displayed only when you save the  document.


You cannot modify the selections if the tracking group is in "view  only" mode for that document type.


![]({{site.pp_baseurl}}/img/lens.gif) [Track  a Purchase Document]({{site.ct_chm}}/document-tracking/tracking-purchase-documents/track_a_purchase_document.html)


You can set document tracking preferences to ensure that the tracking  groups automatically pop up either when you create, edit or save a document,  or only when you create a document (path: **File 
 &gt; Setup &gt; Preferences &gt; Flow Control &gt; Document Tracking**).


![]({{site.pp_baseurl}}/img/lens.gif) [Document  Tracking Preferences]({{site.bp_chm}}/document-tracking/set_document_tacking_preferences.html)
