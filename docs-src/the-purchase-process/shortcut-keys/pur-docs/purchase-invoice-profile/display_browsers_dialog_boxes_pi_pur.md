# Display Browsers/Dialog Boxes


| **Press Key(s)** | **To** |
| ~ | Display the **Retrieve** **Serial****Number** dialog box for a purchase  invoice that has not been saved. |
| Alt+F8 | Display the **Description** dialog  box for the selected line item. |
| Ctrl+A | Display the **Vendor** **Aging****Information** browser. |
| Ctrl+Alt+C | Display the **Custom** **Fields**  dialog box for a saved purchase invoice.<br/><br/><br/>Display the **Find** **Customers**  dialog box on a purchase invoice that has not been saved. |
| Ctrl+Alt+F8 | Display the **Detach** **Serial****#** dialog box. |
| Ctrl+Alt+I | Display the **Assign** **Quantity**  profile. This option will be enabled when the received quantity is equal  to zero. |
| Ctrl+Alt+L | Display the **Item** **Stock****Status** dialog box for the selected  item. |
| Ctrl+Alt+S | Display the **Quantity** **Stock****Area** **Wise**  browser. In a saved purchase invoice this option is enabled for non-serialized  items, if Received quantity is greater than zero. |
| Ctrl+Alt+V | Display all **Serial** **# <br/> for all items** browser. |
| Ctrl+G | Display the **Vendor** **Tracking**  browser. |
| Ctrl+H | Display the **Attach** **Quantity**  dialog box only in cases when negative quantities of inventoried, non  - serialized items are attached to the purchase invoice. |
| Ctrl+I | Display the **Items** browser for  item selection. |
| Ctrl+K | Display the **Retrieve** **Kits**  dialog box for a purchase invoice that has not been saved. |
| Ctrl+L | Display the **Item** **Stock****Status** dialog box for all items. |
| Ctrl+M | Edit the **Matrix** **Items <br/> View** where the item selected line item is a matrix on a purchase  invoice that has not been saved. |
| Ctrl+O | Display the **Item** profile for  the selected line item. |
| Ctrl+P | Display the **Document** **Printing****Preferences** dialog box. |
| Ctrl+Q | Display the **Retrieve** **Serial****Number** (Range) dialog box for  a purchase invoice that has not been saved. |
| Ctrl+R | Display the **Payment** dialog box  on a purchase invoice that has not been fully paid. |
| Ctrl+RIGHT Arrow | Display **Purchase** **Documents****Lines** dialog box for the selected  line item. |
| Ctrl+T | Display the **Transaction** **History** browser for a saved purchase  invoice. |
| Ctrl+U | Display the **Vendor** profile. |
| Ctrl+W | Display the **Multimedia** **Setup** browser. |
| F11 | Displays the **Tax** **Breakdown**  dialog box |
| F8 | Display the **Payment** **History**  browser on a purchase invoice on which payment has been made. |
| Shift+~ | Display the **Retrieve** **Manufactures****Serial** **Number**  dialog box for a purchase invoice that has not been saved. |
| Shift+F5 | Display the **Document** **Tracking**  browser for a saved document. |
| Shift+F6 | Display the **Document** **Notes**  dialog box. |
| Shift+F7 | Display the **Vendor** **Selection**  browser for a purchase invoice that has not been saved. |
| Shift+F8 | Display the **Items** (Kits) browser  for kit selection for a purchase invoice that has not been saved. |
| Ctrl+E | Display the **Document E-mail Preferences**  dialog box. |
| Shift+Ctrl+K | Display the **Task** profile. |
| Shift+Ctrl+I | Display the **Incoming Call** profile. |
| Shift+Ctrl+O | Display the **Outgoing Call** profile. |
| Shift+Ctrl+N | Display the **General Notes** profile. |
| Shift+Ctrl+H | Display the **Task History** browser. |
| Shift+Ctrl+M | Display the **My Tasks** browser. |
| Shift+Ctrl+A | Display the **Appointment** profile. |
| Shift+Ctrl+L | Display the **Appointment History**  profile. |
| Shift+Ctrl+C | Display the **Calendar View** profile. |
| Ctrl+F12 | Display the **Find Line Items**  browser. |
| Ctrl+Y | Display the **Item Purchase History**  browser. |
