# Status Bar Information


The **Status Bar** is displayed  at the bottom of all document profiles in **Everest**.  The options available in the status bar of stock transfer document profiles  are:


Zoom Out/In


click the **Zoom Out** ![]({{site.wm_baseurl}}/img/wh_zoom_out.gif) button to view only the Document Information, Item Grid information  and Status Bar Information. Click the **Zoom 
 In** button to see all the details for a document.


Item Notes


This option brings up the item notes for the selected line item, as  specified in the **Item** profile  (path: **Inventory** > **Items**  > **Options** > **Edit**  > **In New Window** > **Notes** Tab). You cannot edit the item  notes from here.


| Notes Available | Status Displayed |
| ![]({{site.wm_baseurl}}/img/wh_upsell_notes.gif) | Upsell  information, and notes and specifications |
| ![ebx_849669662.gif]({{site.wm_baseurl}}/img/wh_upsell.gif) | Only upsell  information |
| ![]({{site.wm_baseurl}}/img/wh_notes.gif) | Only notes and specifications |



![Lens.gif]({{site.wm_baseurl}}/img/lens.gif) [The  Items Profile - Notes]({{site.mi_chm}}/create-regular-items-kits-and-assemblies/creating-an-item/the_items_profile_-_notes.html)


Stock Status


This option displays the stock status for the items in the stock transfer  documents. The stock status indicator ![wh_stock_status.gif]({{site.wm_baseurl}}/img/wh_stock_status.gif) denotes that the  available stock for the item is greater than zero. click this icon to  view the **Item Stock Status** dialog  box that displays detailed information about the stock status of the items  in the stock transfer documents.


![Lens.gif]({{site.wm_baseurl}}/img/lens.gif) [Item  Stock Status]({{site.mi_chm}}/the-items-browser/information-available/item_stock_status_item_browser_option.html)


Posted/Not Posted


The posting status of the general journal, in which the accounting entry  is made, is displayed in this section. When you click in this section,  the **Journal Transactions** browser,  which contains the accounting entry pertaining to the stock transfer,  is displayed.

| Status Displayed | Description |
| ![wh_posted.gif]({{site.wm_baseurl}}/img/wh_posted.gif) | The journal has been posted to the accounts |
| ![wh_not_posted.gif]({{site.wm_baseurl}}/img/wh_not_posted.gif) | The journal has not been posted |
