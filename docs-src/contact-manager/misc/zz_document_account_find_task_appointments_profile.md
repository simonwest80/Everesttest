---
title: Document/Account
---


While creating a new task/appointment, the document type and number are associated to a task/appointment.


The **Document/Account** tab contains parameters that facilitate you to find tasks/appointments that are associated to the documents.


These parameters can be used individually or in combination with other parameters from **Quick 
Find** tab and **More Choices** tab to find tasks/appointments.
