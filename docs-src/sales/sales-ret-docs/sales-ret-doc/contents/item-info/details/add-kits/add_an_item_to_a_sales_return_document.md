# Add an Item to a Sales Return Document


You can add an item to a sales return document in the following ways:

- From the Sales Return Document profile by selecting **Options &gt; Item &gt; Select Items**.



![]({{site.sp_baseurl}}/img/lens.gif)[Add Items - Options Menu]({{site.sp_baseurl}}/misc/add_items_options_menu_sales_return_doc.html)

- By using the [Smart Search]({{site.sp_baseurl}}/misc/smart_search_items_dialog_box_sales_return_doc.html) option.
