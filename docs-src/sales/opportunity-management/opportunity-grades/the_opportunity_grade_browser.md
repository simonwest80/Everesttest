# The Opportunity Grade Browser


The **Opportunity Grade** browser  (path: **CRM 
 &gt; Opportunity Management &gt; Setup &gt; Grade**) displays all  the grades that are defined.


The browser displays the **Rank**  column. The rank identifies the hierarchy of the grades.


![]({{site.sp_baseurl}}/img/example.gif) If Excellent  is the best grade that can be given, you would defined its rank as 1.


Use the **Up** ![]({{site.sp_baseurl}}/img/sales_up_arrow_icon1.gif) and  **Down** ![]({{site.sp_baseurl}}/img/sales_down_arrow_icon1.gif) arrows to  change the rank of an opportunity grade.


Options common to all browsers are available in this browser.


![]({{site.sp_baseurl}}/img/lens.gif) [Standard  Browser Options]({{site.wwe_chm}}/everest-client/ui/browsers/standard_browser_options.html)
