# Processed


This option allows you to search for invoicing documents based on the  **Processed** check box in the documents.


![]({{site.sp_baseurl}}/img/lens.gif) [Processed  Check Box]({{site.sp_baseurl}}/sales-docs/docs-profile/contents/tab-details/details/other/processed_sales_document_content.html)


The options available are listed in the table below:


| Option | Search Results |
| All | Displays all documents irrespective of whether the document  is marked as processed or not. |
| Yes | Displays only documents marked as processed. |
| No | Displays  only unprocessed documents. |
