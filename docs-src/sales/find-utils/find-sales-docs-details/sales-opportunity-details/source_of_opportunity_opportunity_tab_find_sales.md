# Source of Opportunity


This field allows you to search for invoicing documents using the source  of opportunity.


{:.note}
If you create sales documents directly, a source of  opportunity is not assigned by default to the opportunity.


Unless you input the source, invoices, orders and quotes  do not appear in the search result.
