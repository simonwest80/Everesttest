# Stage


You can specify the sales stage to find the sales documents.


![]({{site.sp_baseurl}}/img/example.gif) You can  search for sales documents in the 'Draft proposal'  stage.


{:.note}
If you create opportunities, specify the stage assigned  in the **Sales Opportunity** profile  (path: **Invoicing &gt; Sales Opportunities 
 &gt; Options &gt; Edit &gt; In New Window**).


If you create sales documents directly without creating  a sales opportunity, Stage 1- Initial Contact is assigned by default to  the opportunity.
