# Change Pipe Color


Select the **Change Pipe Color** option  to change the colors of the rings in the [Customer Tracking Pipeline](javascript:BSSCPopup('{{site.sp_baseurl}}/misc/customer_tracking_pipeline_popup.html');){:id="A1"}<script type="text/javascript" language="JavaScript1.2"><!--
kadovFilePopupInit('A1');
//--></script>  that is displayed in the **Left Panel** of  the browser. You can specify different colors for the first eight rings  (each ring depicting a tracking choice). The rest of the rings, if any,  follow the same color scheme in the order specified for the first eight  rings.


{:.steps}
To change the colors of the customer tracking  pipeline, do the following:

- From the **Customer Tracking Pipeline** browser,  select **Options &gt; Change Pipe Color**.  You will see the **Setup Colors**  dialog box.
- To change the  color of a ring, select the required color from the drop down list of  colors.
- Select the  option **Restore Original Colors** to  reset the pipeline to the default colors that are defined for each ring.
- Click **OK** to save changes, if any, made to  the color settings.
