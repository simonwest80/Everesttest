# Opportunity Classification


**[]()Stage**
: Every sales opportunity progresses through several  stages before it can be converted to a sale. This field identifies the  sales stage for the opportunity. This field displays the default sales  stage. You can define a default sales stage by selecting the **Default**  check box on the **Sales Stage** dialog  box (path: **CRM 
 &gt; Opportunity Management &gt; Setup &gt;** **Sales 
 Stage &gt; Options** > **Edit**  > **In New Window**).
: ![]({{site.sp_baseurl}}/img/example.gif) The  first stage may be the Initial Contact phase and the second stage may  be the Demonstration phase.
: ![]({{site.sp_baseurl}}/img/lens.gif) [Sales Stages]({{site.sp_baseurl}}/opportunity-management/sales-stages/sales_stages.html)


**Modified**
: This field displays the date on which you assign  the sales stage for the opportunity. If you modify the stage at a later  date, this field displays the altered date.


**[]()Source**
: The source identifies the origin of the opportunity.
: ![]({{site.sp_baseurl}}/img/example.gif) In  case of a new customer, the source could be a referral or magazine advertisement.
: Identifying the source helps in analyzing the effectiveness  of advertising. By default this field displays the **Source**  specified for a customer (path: **Invoicing**  > **Customers** > **Options**  > **Edit** > **In 
 New Window** > **E-Commerce**  tab).


{:.hint}
To view the source of the sales opportunity in the **Sales Quotes** browser, drag and drop  the **Source** column from the **Customize** window to [the Column  Header section](javascript:BSSCPopup('{{site.sp_baseurl}}/misc/column_header_popup.html');){:id="a2"}<script type="text/javascript" language="JavaScript1.2"><!--

//--></script>.  The **Customize** window is displayed  when you select **Options** > **Field** **Chooser**  from the browser.
: ![]({{site.sp_baseurl}}/img/lens.gif) [Source  of Opportunity]({{site.sp_baseurl}}/opportunity-management/sources-of-opportunities/source_of_opportunity.html)


**Modified**
: This field displays the date on which the source  is identified in the **Source** field.  If you modify the source at a later date, this field displays the altered  date.


**[]()Grade**
: Use this field to assign a rating to the opportunity.
: ![]({{site.sp_baseurl}}/img/lens.gif) [Opportunity  Grade]({{site.sp_baseurl}}/opportunity-management/opportunity-grades/opportunity_grade.html)


**Modified**
: This field displays the date on which you assign  the grade to the opportunity. If you modify the grade at a later date,  this field displays the altered date.
