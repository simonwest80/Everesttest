Only items contained in the bins specified in this column can be attached  to the pick ticket. All bins that have been set up for the logged in location  or the location of the logged in department are displayed by default in  this column.

To exclude a bin, highlight it and click on the ![]({{site.sp_baseurl}}/img/sales_include_button.gif) button.  Clicking on the ![]({{site.sp_baseurl}}/img/sales_exclude_all_button.gif) button moves all the bins to the **Exclude** column.


It is mandatory to include at least one stock area.
