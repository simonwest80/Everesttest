# Item Alias


You can add an item to the document by selecting the item  alias of the item in the **Item Code**  field. When you select the alias of an item, the original item code and  its details are populated in the **Item 
 Code** field and the item alias appears in the **Item**  Alias field.


{:.example}
**Type C-1255, (the item alias for item 1255),  in the item code field. The item code field shows 1255 and the item alias  field shows C-1255.**
