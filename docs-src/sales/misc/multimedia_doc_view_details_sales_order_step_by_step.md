# <font style="color: #000000;" color="#000000">Multimedia</font>


The **Multimedia** tab allows you  to attach images, sound, video, documents, spreadsheets, and other files  to a sales order. You can store these files in the multimedia repository.


{:.note}
This **** tab is  displayed only when you save the sales order document.


![]({{site.sp_baseurl}}/img/example.gif) You  can attach audio files recording conversations with the customer related  to the current business or documents of communication with the customer.
