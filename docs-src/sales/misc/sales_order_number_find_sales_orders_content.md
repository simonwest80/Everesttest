# SO #


Choose a particular sales order by specifying its number in this field.


{:.note}
You cannot select multiple sales order numbers in this  field at a time.


When you perform a search using the sales order number, **Everest**  maintains a history of the document numbers. This facilitates subsequent  searches, where you can select a document number from an earlier search.
