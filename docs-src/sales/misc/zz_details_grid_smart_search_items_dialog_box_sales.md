---
title: Details Grid
---


The **Details Grid** displays the  items that are retrieved based on the search criteria entered in the **Search for** field. Each row in the **Details Grid** displays information about  the item specified in that row.


![]({{site.sp_baseurl}}/img/example.gif) If  the user enters 101, the code that it matched with could be the item 101FGTIPS,  serial number 1010 or serial number 101.
