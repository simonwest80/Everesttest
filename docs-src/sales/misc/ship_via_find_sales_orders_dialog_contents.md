# Ship Via


This field allows you to search for sales orders using the shipping  agent specified on the order. You can view sales orders pertaining to  different shipping agents by choosing multiple shipping agents.


![]({{site.sp_baseurl}}/img/example.gif) Enter  'UPC' in this  field to search for all sales orders shipped by UPC.
