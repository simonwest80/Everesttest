# Display Browsers/Dialog Boxes


| Press Key(s) | To |
| ` | Display the **Retrieve****Serial** **Numbers**  dialog box. |
| Alt+F8 | Display the **Description**  dialog box for the selected line item. |
| Ctrl+A | Display the **Customer****Aging** **Information**  browser. |
| Ctrl+Alt+C | Display the **Custom****Fields** dialog box from a saved  sales order.<br/><br/><br/>Display **Find Customers** dialog box on a sales order that has not been saved. |
| Ctrl+Alt+F8 | Display the **Detach****serial** **#**  dialog box. |
| Ctrl+Alt+L | Display the **Item Stock <br/> Status** dialog box for the selected line item. |
| Ctrl+Alt+S | Display the **Quantity****stock** **area****wise** browser for the selected  item. This option is enabled in saved sales orders for non-serialized  items. |
| Ctrl+Alt+V | Display the **Serial <br/> # for all items** browser. |
| Ctrl+Alt+W | Display the **Serial <br/> #** browser for the selected serialized item. |
| Ctrl+D | Display the **Pricing****Analysis** dialog box for all the  items on the order. |
| Ctrl+F11 | Display the **Shipment <br/> Tracking Information** browser. |
| Ctrl+G | Display the **Custom <br/> Tracking** dialog box for the customer. |
| Ctrl+H | Display the **Attach****Quantity** dialog box only when  inventoried, non - serialized items are attached or negative quantities  are attached to the sales order. |
| Ctrl+I | Display the **Items** browser  for items selection. |
| Ctrl+K | Display the **Retrieve****Kits** dialog box. |
| Ctrl+L | Display the **Item Stock <br/> Status** dialog box for all items on the document. |
| Ctrl+M | Display the **Matrix <br/> Items View** dialog box if a matrix item is selected. |
| Ctrl+O | Display the **Item**  profile for the selected line item. |
| Ctrl+Q | Display the **Retrieve****Serial** **Numbers**  (range) dialog box. |
| Ctrl+R | Display the **Payment** dialog box for the selected sales order. |
| Ctrl+Right  Arrow | Display the **Sales Document <br/> Lines** dialog box for the selected line item. |
| Ctrl+T | Display the **Transaction****History** browser from a saved sales  order. |
| Ctrl+U | Display the **Customer**'s  profile. |
| Ctrl+W | Display the **Multimedia****Setup** browser from a saved sales  order. |
| Ctrl+Y | Display the **Item Sales <br/> History** browser for the selected line item. |
| F10 | Displays the **Stock <br/> Status Indicator** for the selected line item in a sales document.  This is only enabled if the “Automatically display stock status indicators”  preference in the **Items** dialog  box – **Item** **Details**  tab (path: **File** > **Setup**  > **Preferences** > **Flow <br/> Control** > **Items**) is  NOT selected (disabled). We recommend that you disable this option to  improve performance when adding line items to sales documents. In this  situation, the Stock Status Indicators do not automatically display for  the line items. |
| F11 | Displays a list of **Add-on**  items associated with the selected line item. |
| F12 | Displays a list of **Related <br/> Items** associated with the selected line item. |
| F8 | Display the **Payment****History** browser for the sales  order. |
| Shift+` | Display the **Retrieve <br/> Mfg Serial <br/> Numbers** dialog box. |
| Shift+F5 | Display the **Custom <br/> Tracking** dialog box for the document from a saved sales order. |
| Shift+F6 | Display the **Document****Notes** dialog box. |
| Shift+F7 | Display the **Customers** browser for customer selection. |
| Shift+F8 | Display **Items**  (Kits) browser for kit selection. |
| Shift+F9 | View and edit the **Sales <br/> Opportunity** profile for a saved sales order. |
| Ctrl+E | Display the **Document <br/> E-mail Preferences** dialog box. |
| Shift+Ctrl+K | Display the **Task**  profile. |
| Shift+Ctrl+I | Display the **Incoming <br/> Call** profile. |
| Shift+Ctrl+O | Display the **Outgoing <br/> Call** profile. |
| Shift+Ctrl+N | Display the **General <br/> Notes** profile. |
| Shift+Ctrl+H | Display the **Task History**  browser. |
| Shift+Ctrl+M | Display the **My Tasks**  browser. |
| Shift+Ctrl+A | Display the **Appointment**  profile. |
| Shift+Ctrl+L | Display the **Appointment <br/> History** profile. |
| Shift+Ctrl+C | Display the **Calendar <br/> View** profile. |
| Ctrl+F12 | Display the **Find Line <br/> Items** browser. |
| Ctrl+Alt+T | Display the items from the default **Third <br/> Party Catalog**. |
