# Display Browsers/Dialog Boxes


| Press Key(s) | Description |
| Ctrl+A | Display the **Serial <br/> Number History** browser for the selected credit memo. |
| Ctrl+Alt+E | Display the **Export****Dialog** box. |
| Ctrl+F | Display the **Custom <br/> Fields** dialog box for the selected credit memo. |
| Ctrl+P | Display the **Browser****Property** **Editor**  dialog box. |
| Ctrl+T | Display the **Transaction****History** browser for the selected  credit memo. |
| Ctrl+U | Display the **Journal****Transactions** browser for the selected  credit memo. |
| F8 | Display the **Payment****History** browser for the selected  credit memo. |
| Shift+F2 | Display the **Multimedia <br/> Setup** browser for the selected credit memo. |
| Shift+F5 | Display the **Custom <br/> Tracking** browser for the selected memo. |
| Ctrl+E | Display the **Document <br/> E-mail Preferences** dialog box. |
| Shift+Ctrl+K | Display the **Task**  profile. |
| Shift+Ctrl+I | Display the **Incoming <br/> Call** profile. |
| Shift+Ctrl+O | Display the **Outgoing <br/> Call** profile. |
| Shift+Ctrl+N | Display the **General <br/> Notes** profile. |
| Shift+Ctrl+H | Display the **Task History**  browser. |
| Shift+Ctrl+M | Display the **My Tasks**  browser. |
| Shift+Ctrl+A | Display the **Appointment**  profile. |
| Shift+Ctrl+L | Display the **Appointment <br/> History** profile. |
| Shift+Ctrl+C | Display the **Calendar <br/> View** profile. |
