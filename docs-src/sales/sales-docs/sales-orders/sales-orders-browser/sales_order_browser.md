# The Sales Orders Browser


The **Sales Orders** browser (path:  **Invoicing &gt; Sales Orders**) lists  all the sales orders that have been created and are not yet converted  to sales invoices.


By default the **Sales Orders** browser  displays documents that have been created in the default period specified  for sales document browsers.


![]({{site.sp_baseurl}}/img/lens.gif) [Filter  Sales Documents based on Dates]({{site.sp_baseurl}}/sales-docs/browser/filtering_sales_documents_browsers_based_on_dates.html)


[Prompts  - Sales Documents Tab]({{site.bp_chm}}/flow-ctrl/ctrl/opt/prompts-tab/prompts_sales_document_tab_step_by_step_business_process_in_everest.html)


You can change the period by selecting the required option in the **Dates** field.


This browser contains the options common to all sales document browsers.


![]({{site.sp_baseurl}}/img/lens.gif) [Sales  Document Browser Options]({{site.sp_baseurl}}/sales-docs/browser/cmn-opts/sales_document_browser_options_contents.html)


In addition the browser has the following options:

- [Shipped/Open  Sales Order]({{site.sp_baseurl}}/misc/shipped_open_sales_orders_so_browser_options.html)
- [Payment  Information]({{site.sp_baseurl}}/misc/payment_information_so_browser_options.html)
- [Shipment  Tracking Information]({{site.sp_baseurl}}/misc/shipment_tracking_information_sales_invoice_browser_option.html)
- [Serial  Number History]({{site.sp_baseurl}}/sales-docs/sales-orders/so-proc/cmt-items/enter-cmt-qty/serial_number_history.html)
- [Print  Pick Location]({{site.sp_baseurl}}/misc/print_pick_location_so.html)
- [Recurring  Documents]({{site.sp_baseurl}}/sales-docs/sales-orders/so-proc/recurring_sales_documents_in_everest.html)
- [Commit  Inventory]({{site.sp_baseurl}}/misc/commit_inventory_sal.html)



In addition to the common **View**  menu options, the **Sales Orders**  browser has the following options:

- [Standard]({{site.sp_baseurl}}/misc/detailed_si_browser_view_option.html)
- [Accounting]({{site.sp_baseurl}}/misc/summary_si_invoice_browser_view_option.html)



The print status of sales orders and Pick Location reports can be displayed  from the **Sales Orders** browser.


![]({{site.sp_baseurl}}/img/lens.gif) [Display  Print Status in the Sales Orders Browser]({{site.sp_baseurl}}/sales-docs/sales-orders/sales-orders-browser/display_print_status_in_the_sales_orders_browser_sal.html)


Options common to all browsers are also available in this browser.
