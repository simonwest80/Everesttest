# Generate PO Browser


The **Generate PO** browser displays  a list<font style="color: #ff0000;" color="#FF0000"> </font><font style="color: #000000;" color="#000000">of all the items 
 in the sales order. You can generate purchase orders that are NOT linked 
 to sales orders, from this browser.</font>


{:.note}
If the item is marked for drop shipment, the **Drop 
 Ship** column in the **Generate PO**  browser displays **Yes**.


**![]({{site.sp_baseurl}}/img/lens.gif) [Drop Ship  Items]({{site.sp_baseurl}}/sales-docs/sales-orders/so-proc/gen-po-util/gen-dropship-po/generate_drop_ship_purchase_orders.html#drop_ship_items)**


{:.steps}
To generate a purchase order from a sales  order, do the following:

- From a **Sales Order** profile, select **Options 
 &gt; Generate Purchase Orders &gt; Generate PO**. You will see the  **Generate** **PO**  browser.



Generate PO Browser Options


The options available in this browser are listed below:

- Edit > Line  Items
- Select/Deselect  All Items
- Generate POs



![]({{site.sp_baseurl}}/img/lens.gif) [Generate  PO - Drop Ship Items Browser Options]({{site.sp_baseurl}}/sales-docs/sales-orders/so-proc/gen-po-util/gen-dropship-po/browser/generate_po_drop_ship_items_browser_options.html)


Options common to all browsers are also available in this browser.
