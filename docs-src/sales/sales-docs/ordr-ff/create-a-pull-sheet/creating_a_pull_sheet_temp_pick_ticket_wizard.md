# Creating a Pull Sheet Template


You can create your own pull sheet template for later use. This helps  you to avoid traversing through the **Pull 
 Sheet Wizard** to create a pull sheet. Simply select your predefined  template when you create a new pull sheet.


The **Find Sales Order** filter  criteria that are a part of the **Pull 
 Sheet Wizard** must be saved before saving a pull sheet template.  Otherwise, you cannot create a new pull sheet template.


![]({{site.sp_baseurl}}/img/lens.gif)[The Find Sales  Orders Dialog Box]({{site.sp_baseurl}}/sales-docs/ordr-ff/create-a-pull-sheet/wizard/so-selection/find-so-dlg/find-so-dlg/the_find_sales_orders_dialog_box.html)


{:.steps}
To create a pull sheet template, do the following:

- Use the Pull Sheet  Wizard to create a pull sheet and make your selections until you reach  the **Sales** **Order 
 Selection** screen.
- On the **Sales 
 Order Selection** screen, click **Select 
 Sales Orders** to view the **Find 
 Sales Order** filter.
- In the **Filter 
 Name** field, enter a unique name for the filter that relates to  your filter parameter selections, such as “Fully Paid Sales Orders” or  “Includes SOs with Accounts on Hold”.



![]({{site.sp_baseurl}}/img/find_sales_orders_includes_sos_with_accounts_on_hold_sal.gif)


**Figure: Find Sales Order Profile - Create Pull  Sheet Template Example**

- Click **Save**.  This saves the filter so that it can be used as a template.
- Click **Find**  to display the filter results.
- Click **OK**.
- Click **Yes**  to save the profile.
- Once you are finished  selecting the appropriate sales orders, click the **Save 
 Template** button in the **Pull Sheet 
 Wizard - Sales Order Selection** section. You will see the **Pull Sheet Template** dialog box.



![]({{site.sp_baseurl}}/img/pull_sheet_template_dialog_box_sal.gif)


**Figure: Pull Sheet Template Dialog Box**

- Leave the **Set as default for All users** field blank  unless you want to set your template as a common default template for  all users. This is optional.
- Leave the **Set as default for &lt;login user&gt;**  field blank unless you want to set the user defined template as a default  template for the logged-in user. This will override the common default,  if defined. This is optional.
- Enter the name  of the template in the **Template Name**  field. If you have defined the common default template already, this field  is disabled.
- Click the **Save** button to save the template. (You  can view pull sheets created with this template in the **Pull 
 Sheets Templates** browser. To create a pull sheet with your new  template, open the Pull Sheet Wizard, select your template on the **Welcome** screen, and click **Finish**.  This eliminates steps and saves time.)



![]({{site.sp_baseurl}}/img/lens.gif)[Template  Browser]({{site.sp_baseurl}}/misc/template_browser_pull_sheet_wizard.html)

- Click **Next**.  You will see the **Pick 
 Ticket (Preview) Browser**.
