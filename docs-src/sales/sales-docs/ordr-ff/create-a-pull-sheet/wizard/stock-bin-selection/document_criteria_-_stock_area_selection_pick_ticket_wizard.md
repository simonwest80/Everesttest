# Pull Sheet Wizard - Stock Bin Selection


This section allows you to define the stock bins from which the items  should be picked.


**Include Stock Bin**
: Only items contained in the bins specified in this  column can be attached to the pick ticket.
: If you set a preference by selecting the option  **Allow users to access stock bins of - 
 Logged in Location** (path: **File 
 &gt; Setup &gt; Preferences &gt; Flow Control &gt; Other Flow Control 
 &gt; General** tab), all bins set up for the logged in location or  sub-location are displayed by default in this column.
: ![]({{site.sp_baseurl}}/img/lens.gif) [Stock  Bins Profile]({{site.sc_chm}}/options/miscellaneous-set-up/stock-areas/stock-bins/the_stock_bin_profile.html)
: To exclude a bin, highlight it and click the ![]({{site.sp_baseurl}}/img/sales_include_button.gif) button. Clicking the ![]({{site.sp_baseurl}}/img/sales_exclude_all_button.gif) button moves all the  bins to the **Exclude** column.
: It is mandatory to include at least one stock bin.


{:.note}
You  need adequate security rights to include the bins that are not part of  the logged in location.


**Exclude Stock Bin**
: Items contained in the bins specified in this column  cannot be attached to the pick ticket. All bins that are set up for locations  other than the logged in location are displayed by default in this column.
: You can include bins by highlighting the required  bin in the **Exclude** column and  clicking the ![]({{site.sp_baseurl}}/img/sales_exclude_button.gif). button. To include all the bins click the  ![]({{site.sp_baseurl}}/img/sales_include_all_button.gif) button.


Click **Next**. You will see the  [**Selected 
 Stock Bins**]({{site.sp_baseurl}}/sales-docs/ordr-ff/create-a-pull-sheet/wizard/selected-stock-bins/selected_stock_areas_pick_ticket_wizard.html) section of the **Pull 
 Sheet Wizard**.
