# Other Details


The **Others** tab provides you  with search parameters that use information from other fields in sales  orders.


![]({{site.sp_baseurl}}/img/example.gif) You can  use the document alias or terms or jurisdiction to find the sales orders.


This tab comprises the following fields:

- [Terms]({{site.sp_baseurl}}/misc/terms_find_sales_orders_dialog_contents.html)
- [Jurisdiction]({{site.sp_baseurl}}/misc/jurisdiction_find_sales_orders_dialog_contents.html)
- [Loc/Sub-Loc]({{site.sp_baseurl}}/misc/department_find_sales_orders_dialog_contents.html)

- [PO  number]({{site.sp_baseurl}}/misc/po_number_find_sales_orders_dialog_contents.html)
