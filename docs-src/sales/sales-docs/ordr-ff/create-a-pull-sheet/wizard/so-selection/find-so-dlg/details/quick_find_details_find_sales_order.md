# Quick Find Details


The **Quick Find** tab lists the  most common search parameters used to find sales orders. The **Quick 
 Find** tab consists of the following fields:

- [SO#]({{site.sp_baseurl}}/misc/sales_order_number_find_sales_orders_content.html)
- [Order  Status]({{site.sp_baseurl}}/misc/order_status_find_sales_orders_content.html)
- [Sales  Representative]({{site.sp_baseurl}}/sales-docs/docs-profile/contents/tab-details/details/other/sales_representative_sales_document_content.html)
- [Picked  Status]({{site.sp_baseurl}}/misc/picked_status_find_sales_order_-_pick_ticket.html)
- [Account  Name]({{site.sp_baseurl}}/misc/account_name_find_sales_orders_content.html)
- [SO  Date]({{site.sp_baseurl}}/misc/document_date_find_sales_orders_dialog_contents.html)
- [Number  of Records to Retrieve]({{site.wwe_chm}}/advanced-options/find-function/number_of_records_to_retrieve_common_find_wwe.html)



You can use these parameters individually or in combination with other  parameters to narrow down your search. The options selected in this tab  also appear in the **Advanced Find** tab.
