<font color="#000000" class="hcp1">The </font>**Description**<font color="#000000" class="hcp1">/</font>**Note**<font color="#000000" class="hcp1"> column displays the description of an item as entered 
 in the </font>**Item**<font color="#000000" class="hcp1"> profile. To edit the description, click the </font>**Description/Note**<font color="#000000" class="hcp1"> column for the item, and type the required description.
