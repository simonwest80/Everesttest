# Retrieve Matrix Items


<font color="#000000" class="hcp1">A </font>[Matrix  Item]({{site.mi_chm}}/creating-matrix-group-and-matrix-items/item_matrix.html)<font color="#000000" class="hcp1"> can be retrieved by selecting the </font>master  item<font color="#000000" class="hcp1"> for the matrix from the </font>**Items** browser<font color="#000000" class="hcp1">, or by selecting the <span class="hcp2">Item 
 Matrix</span> option (path: <span class="hcp2">Sales Document 
 </span>profile &gt; <span class="hcp2">Options</span> &gt; 
 <span class="hcp2">Retrieve</span> &gt; <span class="hcp2">Item 
 Matrix</span>)</font>.


When you select the master item for the matrix, the quantity specified  for the item of all combinations of the matrix is retrieved into the document.


Advantages of Retrieving a Matrix Item:

1. You can enter the sale details of  more than one combination in the Item Details Grid simultaneously.
- You can view the stock status and  availability of each combination.
