# Weight


The weight of the item is computed using the quantity and the weight  per sales unit defined in the **Weight****M****easure**  and **Weight** fields of the **Item** profile (path: **Inventory 
 &gt; Items &gt; Options** > **Edit** > **In New Window** >  **Freight** > **Sales**  tab). If you use **Everest** Freight  Estimator, the computed weight is used to calculate freight charges. The  weight specified in the Item Details Grid is the total weight of the quantity  specified for the line item. This is a display field and cannot be edited.


![]({{site.sp_baseurl}}/img/example.gif) The  weight measure is defined as Kilograms and weight unit assigned is 10  in the **Item** profile (path: **Inventory** > **Items**  > **Options** > **Edit**  > **In New Window** > **Freight** > **Sales**  tab). When you select a quantity of 100 units of the item in the sales  document, the weight column displays 10 (100 units by 10 kilograms).
