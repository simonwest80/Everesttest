# Line


**This column displays the line number of  the Item Details Grid. All lines in the Item Detail Grid are printed in  order of their line numbers.**


{:.note}
You  can move the line up or down. Use the **Move 
 up**/**Move down** option (path:  Sales Document **** profile > **Options** > **Line 
 Item** > **Move up**/**Move down**) to rearrange the order of  the items in the Item Detail Grid.
