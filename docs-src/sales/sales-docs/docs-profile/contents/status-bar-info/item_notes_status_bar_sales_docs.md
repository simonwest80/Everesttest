# Item Notes


This option brings up the item notes for the selected line item, as  specified in the **Item** profile  (path: **Inventory** > **Items**  > **Options** > **Edit**  > **In New Window** > **Notes** tab).


| Status Displayed | Notes Available |
| ![]({{site.sp_baseurl}}/img/sales_upsell_info_notes_specifications.gif) | Upsell  information, and notes and specifications. |
| ![]({{site.sp_baseurl}}/img/sales_item_notes_upsell_informtion_icon.gif) | Only upsell  information. |
| ![]({{site.sp_baseurl}}/img/sales_item_notes_and_specifications_icon.gif) | Only notes and specifications. |
