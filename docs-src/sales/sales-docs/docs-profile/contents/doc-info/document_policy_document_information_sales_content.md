# Document Policy


Document policies are custom messages/notes/information/conditions attached  to the system documents (and their alias) that print along with the document  (usually printed at the bottom of the document).


The policy that is applicable to the document can be viewed by clicking  the **Document Policy** ![]({{site.sp_baseurl}}/img/sales_document_policy_icon.gif) icon in the status bar of the document. You can edit this  policy.


{:.warn}
Try to restrict your document policy to a  maximum of ten lines as the rest would be truncated when you print the  document.


The document policy is determined using the following rules:

1. If the customer has  a specific document policy for that document type (sales quotes/orders/invoices),  the customer's document policy is automatically selected.



![]({{site.sp_baseurl}}/img/lens.gif)<font color="#800080" class="hcp10"> 
 </font>[Customers  - Document Policy]({{site.mc_chm}}/customer-details/document-policy/document_policy_content.html)

1. If the customer does  not have a policy associated with that document type, the policy associated  with the document alias is displayed.



![]({{site.sp_baseurl}}/img/lens.gif)<font color="#800080" class="hcp10"><a href="{{site.bp_chm}}/doc-aliases/info/document_type_document_alias_document_policy_web_enabled_fields_doc_alias_content_businesss_process_in_everest_content.html"> 
 </a></font>Document Alias - Document Policy

1. If both the customer  and the document alias do not have a policy associated, the company's  default policy for that document type is used (path: **File 
 &gt; Setup &gt; Preferences &gt; Defaults &gt; Documents &gt;** **Sales Policy** tab)



![]({{site.sp_baseurl}}/img/lens.gif)<font color="#800080" class="hcp10"> 
 </font>[Documents  Defaults]({{site.bp_chm}}/flow-ctrl/defs/document-policies-defaults/document_policies_defaults.html)


{:.note}
When a sales quote is converted to an order or when  a sales order is converted to a sales invoice, the default document policy  applicable to the order or invoice is determined and applied to the converted  document.
