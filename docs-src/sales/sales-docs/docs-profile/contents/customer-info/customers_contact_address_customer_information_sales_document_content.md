# Customer's Contact Address


The **Contact** field displays the  address of the person who is to be contacted for matters relating to the  sales document.


| **Document** | **Contact Refers To** |
| Sales Quote | The name and address of the person with  whom negotiations are conducted for a particular transaction. |
| Sales Orders and Invoices | The name and address of the person responsible  for the fulfillment of the sale transaction. |



When you select the customer code, the [default contact address](javascript:BSSCPopup('{{site.sp_baseurl}}/misc/default_contact_address.html');){:id="A1"}<script type="text/javascript" language="JavaScript1.2"><!--

 New Window** > **Addresses**  tab) is displayed in the **Contact**  field on the sales document.
