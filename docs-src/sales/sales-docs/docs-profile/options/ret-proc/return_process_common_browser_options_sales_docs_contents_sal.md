# Return Process


The **Return Process**  option (path: **Options** > **Return Process**) allows you to process  returns on the sales invoice/credit memo document and allows you two further  options:


1. [Return]({{site.sp_baseurl}}/sales-docs/docs-profile/options/ret-proc/return_process_return_option_sales_docs_sal.html)


2. [Return  with Gift Receipt]({{site.sp_baseurl}}/sales-docs/docs-profile/options/ret-proc/return_process_return_with_gift_receipt_sales_docs_sal.html)


The system supports returns and exchanges from the sales  invoice profile, only when the value of the items being returned is less  than the items being purchased.
