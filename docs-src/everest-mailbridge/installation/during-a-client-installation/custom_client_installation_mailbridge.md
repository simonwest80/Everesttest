# Custom

- Select **Custom** from the [**Setup Type** dialog]({{site.mb_baseurl}}/installation/during-a-client-installation/client_installation_mailbridge.html#setup_type_client_install) box to selectively  install the required **Everest** components.
- Click **Next**. You will see the **Custom 
 Setup** dialog box.



{:.note}
To view the description of a feature, click on the feature.  The relevant description is displayed in the **Feature 
 Description** section on the right side of the dialog box.

- Select **MailBridge**.



{:.note}
Tota  select/deselect  the components, click the ![]({{site.mb_baseurl}}/img/mailbridge_options_drop_down_button.gif) icon displayed beside each component.  You will see the **Options** menu.  Choose the **This feature will be installed 
 on local hard drive** option to install the selected feature on your  hard disk.

- Click **Next**. You will see the **Ready 
 to Install the Program** dialog box.
- Click **Install**. You will see the **Installing****Everest** **Advanced****Client Edition** dialog box.

: The Installation Wizard installs **Everest** on your computer.
: If any other programs are running on  the workstation where **Everest**  is being installed, you will see the [**Files in Use** dialog box]({{site.mb_baseurl}}/misc/files_in_use_dialog_box_mailbridge.html).

- Once installation  has been completed, you will see the **Installation****Wizard** **Completed**  dialog box.
- Click **Finish** to exit the wizard.
