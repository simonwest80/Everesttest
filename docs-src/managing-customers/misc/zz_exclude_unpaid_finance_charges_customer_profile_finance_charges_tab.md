---
title: Exclude Unpaid Finance Charges
---


Select this option, to impose finance charges only on the outstanding  amount. This field is enabled if you select "Percentage" in  the **Compute method** field.


![]({{site.mc_baseurl}}/img/note.gif) Â Do  not select this option, if you want to impose finance charges even on  the previous unpaid finance charges.
