# Goto Website


This option allows you to open the default web browser and access the  customer’s URL if it is specified in the **Web 
 URL** field in the **General** tab  of the **Customer** profile. If an  URL is not specified, the error message `URL is not defined for the selected  entries’ is  displayed. Click **OK** to continue.
