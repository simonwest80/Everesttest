# Send E-mail


This option enables you to send an e-mail to a customer using **Everest E-mail**. This option displays  the **Untitled-New Message** window,  with the selected customer's e-mail address in the **To**  field. The e-mail address of the customer must be specified in the **General** tab of the **Customer** profile.
