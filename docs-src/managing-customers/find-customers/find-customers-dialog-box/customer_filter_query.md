# Find Customers - Query


The **SQL** (Structured Query Language)  tab displays queries. Queries are automatically generated when you specify  search parameters in the **Quick Find/More 
 Choices/Custom Fields/Advanced Find** tabs of the **Find****Customers** dialog box.

You cannot edit or add  new SQL queries from the Query tab. This is a display tab.
