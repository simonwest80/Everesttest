---
title: Parameters
---


This section displays the list of reports added to the batch along with default parameters associated with the reports.


This section of the wizard contains three columns:


**Description**
: This column displays the report name that is chosen for batch processing.


**Parameters**
: This column allows you to set default parameter values for the batch reports. When you run the batch, the reports are executed based on the parameter values set here.


**Print Action**
: Select the default action to be performed when the batch is running by clicking the drop-down button. The options available are **Preview** or **Print**.
