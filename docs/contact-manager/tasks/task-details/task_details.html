Task Details Information

The Details tab allows you to associate a task with a specific customer, vendor, sales or purchase document.

Document Type
This field denotes the type of document; you want to associate with a task.
Document
This field allows you to associate a document number to a task.

Purchase Order 1003

Alias
This field ** displays the alternate used for the selected document type and cannot be edited.
Document Alias
Document Tracking
Use this option to track the status of the document selected in the Document field based on the tracking groups that you have set up for that document type. Click this icon to view the Custom Tracking browser.
Account Type
Use this field to specify whether a task has been created for a customer or vendor. When a task is being created for a particular document, the type of account is automatically updated. This field is enabled only if the Document Type selected is None.
Account Code
This field ** displays the code and address details for the customer or vendor account. You can however select the required code (customer/vendor) depending on whether you have selected customer/vendor in the **Account Type field. If a document number is selected in the Document field, this field is a display field.
Customer/Vendor Tracking
Use this option to track the status of the customer/vendor selected in the Account Type and AccountCode fields based on the tracking groups that you have set up for that document type. Click this icon to view the Customer/Vendor Tracking browser.

The Customer/Vendor Tracking icon is enabled only when you specify the AccountType and Account Code fields.

Sales Opportunity
Use this option to create a sales opportunity for the task. Click the icon to view the Sales Opportunity profile. In case there are any records already created for that customer, this option opens the Sales Opportunities browser.
The **Sales
Opportunity** icon is enabled only when you specify customer in the Account Type and Account Code fields.
Sales Opportunity
E-mail
This field displays the e-mail ID of the selected account (vendor or customer) if it is specified in the General tab of the Customer/Vendor profile (path: Invoicing > Customers > Options > New > In New Window or Purchasing > Vendors > Options > New > In New Window).
Web page address
This field displays the Internet address defined for the vendor or customer account/contact if it is specified in the General tab of the Customer/Vendor profile (path: Invoicing > Customers > Customer profile or Purchasing > Vendors > Vendor profile).

The e-mail and web page addresses are displayed for both the billing address and contact address of the customer/vendor.

Contact Code
Use this field to display the code and contact address details of the customer or vendor account. Selecting the required code (customer/vendor) depends on whether you have selected either a document in the Document Type and Document fields or customer/vendor in the Account Type and Account Code fields.
E-mail
This field displays the e-mail ID of the selected contact (customer or vendor) if it is specified in the Addresses tab of the Customer/Vendor profile (path: Invoicing > Customers > Options > New > In New Window or Purchasing > Vendors > Options > New > In New Window).
Web page address
This field displays the Internet address defined for the customer or vendor account/contact if it is specified in the Addresses tab of the Customer/Vendor profile (path: Invoicing > Customers > Options > New > In New Window or Purchasing > Vendors > Options > New > In New Window).
