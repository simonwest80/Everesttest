The Incoming Call Profile - Task

Use the Task tab to specify a description, assign users and set a reminder for the incoming call.

The Incoming Call profile is similar to the Task profile (path: Contact Manager > New Task). The options unavailable are Complete and Complete and Followup.

The Task Profile - Task

Timer
This field displays the duration of the incoming call. The Timer is invoked when you open the Incoming Call profile and stops once you save the profile.

In case of service calls that you pay by the hour, the timer gives the actual duration of telephone call.
