Specify the account code for the vendor or customer that you are associating with the appointment. This field is enabled only if an Account Type is selected.

If a customer account is selected, the Customer Tracking and Sales Opportunity buttons beside this field are enabled. If a vendor account is selected, only the Vendor Tracking button beside this field is enabled.

Click the Customer/VendorTracking icon to view the Customer/VendorTracking browser. Click the Sales Opportunity icon to view the Sales Opportunity profile. In case there are sales opportunities already created for the selected customer, this option opens the Sales Opportunities browser.

The Customer/VendorTracking and Sales Opportunity icons are enabled only when you select an AccountType and Account Code in the relevant fields.
