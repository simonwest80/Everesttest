# View


The menu options (path: **My Tasks**  browser > **View**) in this browser  are as follows:


**General Notes**
: Use this option to view the list of general notes  created and assigned to the logged-in user.


**Incoming Call**
: Use this option to view the list of incoming calls  created and assigned to the logged-in user.


**Outgoing Call**
: Use this option to view the list of outgoing calls  created and assigned to the logged-in user.


**Pending Tasks**
: Use this option to view the list of pending tasks  created and assigned to the logged-in user.


**Completed Tasks**
: Use this option to view the list of completed tasks  created and assigned to the logged-in user.


**Discontinue/Reactivated Items**
: Use this option to view all the tasks, incoming  calls, outgoing calls and general notes that have been discontinued by  the logged-in user.


**Auto Preview**
: This option displays notes for each task in the  browser.


**Preview Pane**


This option displays the task details such as associated account name  if any, task notes, custom fields and linked tasks.


Options common to all browsers are also available in this browser.
