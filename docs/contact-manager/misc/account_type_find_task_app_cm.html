Account Type

Use this field to search for all tasks/appointments that are associated with a customer/vendor. You can select the following options to find tasks/appointments based on the Account Type:

Option Description
Both Select this option and click the browse button in the Account field to view the Addresses browser, which displays all Vendors and Customers.
Vendor Select this option and click the browse button in the Account field to view the Addresses browser, which displays all Vendors.
Customer Select this option and click the browse button in the Account field to view the Addresses browser, which displays all Customers.
