Create an Appointment

To create an appointment, do the following:

  • From the main menu bar, select Contact Manager > New Appointment. You will see the Appointment profile.

You can also create appointments from the:

  • Appointment profile by selecting Options > New.
  • Calendar View profile (path: Contact Manager > Calendar View) by selecting Options > New > Appointment.
  • All Appointments/Active Appointments browser (View > All Appointments/Active Appointments) by clicking the Click here to add an appointment link under the Subject column, in the Calendar View profile.
