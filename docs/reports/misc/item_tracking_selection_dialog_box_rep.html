Item Tracking Selection Dialog Box

Select the following parameter to view the Item Tracking (Selection) report:

Group
Specify the code of the ** tracking group ** to be included in the report.
Drop-Down List - Right-Click Menu Options
Tracking Selection
Specify the tracking selection to be included in the report.
Drop-Down List - Right-Click Menu Options
Date From
Specify the start date for the tracking selections tagged to a group, to be included in the report.
Date To
Specify the end date for the tracking selections tagged to a group, to be included in the report.

If the Date From and Date To fields are left blank, by default the current system date is included in the report.

Print Memos
Select this check box to include notes attached to the tracking selection in the report.
