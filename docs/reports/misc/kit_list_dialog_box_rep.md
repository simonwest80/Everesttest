# Kit List Dialog Box


Select the following parameters to view the **Kit List** report.


**Kit Code From**
: Specify the starting code for the range of kits  to be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Kit Code To**
: Specify the ending code for the range of kits to  be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


{:.note}
If values are not specified in the **Kit 
 Code From** and **Kit Code To**  fields, by default all kits are included in the report.


**Status**
: Select the status for the kit to be included in  the report.


| Active | To include all kits that are in use. |
| Inactive | To include all kits that are discontinued. |



**Show Details**
: Select this check box to include the items, which  make a kit in the report. If you clear this box, a summarized list of  kits without the items is generated.


{:.note}
- If this option is selected, the **Kit 
 List (Detail)** report is displayed.
- In the **Kit List**  report, the purchase and sales quantities are shown with up to six decimal  places as entered in the **Kit** **Details** tab. The cost and price fields  are shown with two decimal places.
