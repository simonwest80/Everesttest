Options in Report Preview Dialog Box

Explained below are the options available in the Report Preview dialog box along with their functionality:

Button Name Description
Print This option brings up the Print dialog box, which allows you to specify page range and number of copies to be printed.


Printing Reports
Printer Setup This option brings up the Print Setup dialog box, which allows you to specify settings for printing reports.


Printing Reports
Refresh Data Use this option to refresh the dialog box to update the report with any changes in data.
Export This option brings up the Export dialog box, which enables you to export the report to a file format of your choice, for retrieval at a later date.


Exporting Reports
Search This option enables you to carry out a text search on the report output. Specify the text in the text box beside the icon.
