Detail By Category - Entry Date (Serialized) Dialog Box

Select the following parameters to view the Detail By Category - Entry Date (Serialized) report:

Category From
Specify the starting code for the range of categories to be included in the report. If you leave this field blank and select a category in the Category To field, the report generates all categories from the beginning to the category code specified.
Drop-Down List - Right-Click Menu Options
Category To
Specify the ending code for the range of categories to be included in the report.
If both the **Category
From** and the Category To fields are left blank, all the item categories are included in the report.
Drop-Down List - Right-Click Menu Options
Date From
Specify the start date for the category code to be included in the report.
Date To
Specify the end date for the category code to be included in the report.

If the Date From and Date To fields are left blank, by default the current system date is included in the report.
