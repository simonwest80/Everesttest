Vendor Statements Dialog Box

Select the following parameters to view the Vendor Statements report.

Vendor From
Specify the starting code for the range of vendors to be included in the report.
Drop-Down List - Right-Click Menu Options
Vendor To
Specify the ending code for the range of vendors to be included in the report.
Drop-Down List - Right-Click Menu Options

If the Vendor From and the Vendor To fields are left blank, all vendors are included in the report.

Statement Date
Select the date up to which you want to view your dues to the vendor.
Include Purchase Order
Select this check box to include deposits received on purchase orders in the report.
Include Debit Memos
Select this check box to include payments for the debit memos raised.
Include Only Outstanding Documents
Select this check box to include only those purchase documents that have outstanding balances .
Print in Base Currency
Select this check box to display the amounts relating to foreign currency vendors in your base currency.

The credit limit allowed by the vendor is always displayed in the vendor currency irrespective of the Print in Base Currency selection.
