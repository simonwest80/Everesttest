Use this field to group the report based on the customer name, code or balance. The options available in this field are as follows.

  • Account Name - Select this option to view the customers by their names as specified in the Account Name field of the customers’ profiles.
  • Account Code - Select this option to view the customers by their codes as displayed in the Account Code field of the customers’ profiles.
  • Account Balance - Select this option to view the customers by their balances.
