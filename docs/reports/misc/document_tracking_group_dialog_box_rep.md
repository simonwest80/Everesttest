# Document Tracking (Group) Dialog Box


Select the following parameters to view the **Document Tracking (Group)** report.


**Group**
: Enter the tracking group code to be included in  the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Sort By**
: Select this **** option  to sort the report by **Status** or  **Document**.


**Document Date From**
: Specify the start date for the documents, to be  included in the report.


**Document Date To**
: Specify the end date for the documents, to be included  in the report. ****


{:.note}
**I**f  the **Document Date From** and the  **Document Date To** fields are left  blank, the current system date is included in the report.


**Document Type**
: Specify the type of sales documents to be included  in the report.
: The options available are:

- Sales Quote
- Sales Order
- Sales Invoice
- Credit Quote
- Credit Order
- Credit Memo
- SI/CM  only



**Document Status**
: Specify the document status to be included in the  report.

- All
- Closed
- Open



Selecting **All** includes all document  types in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Document  Status]({{site.sp_chm}}/sales-docs/docs-profile/contents/tab-details/details/other/document_status_document_information_sales_content.html)


**Location/Sub-Location**
: Select the location/sub-location code based on which  the report is generated.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Sales/Purchase Rep**
: You can filter the report based on the sales or  purchase representative.


{:.note}
If this field is left blank, all the sales/purchase  representatives are included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Tag Date From**
: Enter the start date, as to when the tracking selections  were tagged to a group.


**Tag Date To**
: Enter the end date, as to when the tracking selections  were tagged to a group.


{:.note}
If the **Tag 
 Date From** and **Tag Date To**  fields are left blank, by default the current system date is included  in the report.


**Schedule Date From**
: Specify the start schedule date in this field. The  schedule date is the date when the sales document was initially tagged.  Subsequently, when the document was processed, the tag was transferred  across documents.


**Schedule Date To**
: Specify the end date for the documents, (that are  scheduled to be received) to be included in the report.


{:.note}
If the **Schedule 
 Date From** and **Schedule Date To**  fields are left blank, by default the current system date is included  in the report.


**Show Memos**
: Select this check box to include the notes attached  to the tracking selection in the report.
