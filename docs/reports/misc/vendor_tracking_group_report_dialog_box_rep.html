Vendor Tracking (Group) Dialog Box

Sort By
Specify whether you want to sort the report by Vendor or Status.
Group
Enter the tracking groups for which you want to generate the report. The report prints the status of all vendors pertaining to the selected group.
Drop-Down List - Right-Click Menu Options
Purchase Rep
Enter the name of the user, who tagged the vendor. (This helps in analyzing the employee’s performance).
Drop-Down List - Right-Click Menu Options
Date From
Specify the start tag date in this field. The tag date indicates the date of assigning the documents to a tracking selection.
Date To
Specify the end tag date in this field. The tag date indicates the date of assigning the documents to a tracking selection.

If the Date From and Date To fields are left blank, the current system date is included in the report by default.

Print Memos
Select this check box to include the notes attached to the tracking selections, in the report.
