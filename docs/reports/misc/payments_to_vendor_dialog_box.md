# Payments to Vendors Dialog Box


Select the following parameters to view the **Payments 
 to Vendors** report.


**Vendor From**
: Specify the starting code for the range of vendors  to be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Vendor To**
: Specify the ending code for the range of vendors  to be included in the report.


{:.note}
If the **Vendor 
 From** and the **Vendor To** fields  are left blank, payments to all vendors are included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Date From**
: Specify the start date of the report period. The  date you specify here is the date of the payment on purchase documents.  By default, the login date is displayed.


**Date To**
: Specify the end date of the report period. The date  you specify here is the date of the payment on purchase documents. By  default, the login date is displayed.


**1099 Only**
: Select this check box to display payments made to  1099 vendors in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [1099  Vendor]({{site.mv_chm}}/vendor-details/accounting-information/federal_id_number_and_1099_vendor_accounting_information.html)
