# Rights Dialog Box


**User From**
: Specify the starting code for the range of users  to be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**User To**
: Specify the ending code for the range of users to  be included in the report.


{:.note}
If  the **User From** and **User 
 To** fields are left blank, all users are included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)
