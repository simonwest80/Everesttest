Security Rights

The Security Rights report displays a list of group and user rights assigned to selected users. The rights for each user are displayed on a new page. This report lists details such as the user code and name, security right number and the right.

To generate the Security Rights report, do the following:

  • From the main menu bar, select Reports > Miscellaneous > SecurityRights. You will see the Rights dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
