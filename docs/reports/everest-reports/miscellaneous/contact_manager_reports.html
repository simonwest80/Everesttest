Contact Manager

Use the Contact Manager report for viewing the tasks created for various users. The report includes details such as the user name, subject, and the created date, start date, end date and remind date, the document number and status if attached to the task and the task status.

To generate the Contact Manager report, do the following:

  • From the main menu bar, select Reports > Miscellaneous > Contact Manager. You will see the Contact Manager dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
