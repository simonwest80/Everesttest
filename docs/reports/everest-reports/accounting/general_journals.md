# General Journals 


The **General Journals** report  displays a list of general journals **** created.  The report includes details such as the journal group, journal number,  the source type, transaction number, the date for each entry, the reference  number and date for each transaction, the account that is affected and  the debit/credit details for each transaction.


This report displays the general journals that are NOT posted.


![]({{site.rpt_baseurl}}/img/lens.gif) [General  Journals]({{site.acc_chm}}/general-journals/general_journal.html)


{:.steps}
To generate the General Journals report,  do the following:

- From the main  menu bar, select **Reports** >  **Accounting** > **General 
 Journal**. You will see the **General 
 Journal** dialog box.
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats



General Journals Dialog Box


**Journal Number From**
: Select the starting number for the range of journals  to be included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Journal Number To**
: Select the ending number for the range of journals  to be included in the report.


{:.note}
If  the **Journal Number From** and the  **Journal Number To** fields are left  blank, by default all the journal numbers are included in the report.
: ![]({{site.rpt_baseurl}}/img/lens.gif) [Drop-Down  List - Right-Click Menu Options]({{site.rpt_baseurl}}/common-report-options/drop_down_button_and_menu_options.html)


**Journal Group From**
: Select the starting group for the range of journal  groups to be included in the report.


**Journal Group To**
: Select the ending group for the range of journal  groups to be included in the report.


{:.note}
If you select journal groups in the **Journal Group From** and **Journal 
 Group To** fields and leave the **Journal 
 Number From** and **Journal Number 
 To** fields blank, the report displays journals only for the selected  journal groups. Further, journals which have not been assigned to any  journal groups in **Everest**, are  included in the report only when the **Journal 
 Group From** and **Journal Group To**  fields are left blank.
