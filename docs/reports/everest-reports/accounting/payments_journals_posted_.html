Payment Journal (Posted)

The Payment Journal (Posted) report is similar to the Payment Journal report except that it lists only the transactions that have been posted to the general ledger.

To generate the Payment Journal (Posted) report, do the following:

  • From the main menu bar, select Reports > Accounting > PaymentJournal (Posted). You will see the Payment Journal (Posted) dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box, before printing
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats

Payment Journal (Posted) Dialog Box

Select the following parameters to view the Payment Journal (Posted) report:

Journal Number From
Select the starting journal number for the range of journals to be included in the report.
Drop-Down List - Right-Click Menu Options
Journal Number To
Select the ending journal number for the range of journals to be included in the report.
If the **Journal
Number From** and the **Journal Number
To** fields are left blank, by default all the journal numbers will be included in the report.
Drop-Down List - Right-Click Menu Options
Journal Group From
Select the starting journal group for the range of journal groups to be included in the report.
Journal Group To
Select the ending journal group for the range of journal groups to be included in the report.

If you select Journal Groups in the Journal Group From and Journal Group To fields and leave the Journal Number From and Journal Number To fields blank, the report displays journals only for the selected journal groups. Further, journals which have not been assigned to any journal groups in Everest, will be included in the report only when the Journal Group From and the Journal Group To fields are left blank.
