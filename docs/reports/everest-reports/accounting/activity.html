Activity

The Activity report displays the transactions in a particular account (including discontinued accounts). The report includes the document affecting the account, the journal number, the customer/vendor involved in a transaction, the date of the transaction, whether the account was debited or credited and the balance in the account. The report displays amounts either in the account currency or the base currency or both, depending on the parameters selected.

The report displays transactions for those journals which are posted. You can also include unposted amounts in the report if you select the option in the parameters.

This report will display journals that have been posted to their respective general ledger accounts.

To generate the Activity report, do the following:

  • From the main menu bar, select Reports > Accounting > Activity. You will see the Activity Report dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
