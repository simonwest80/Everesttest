# Number of Orders Per Shopper


This report lists the shoppers who have purchased items from your online  store, based on the number of orders placed by them.


{:.note}
For every store you will see the shoppers name and the number of orders  placed.


{:.steps}
To generate a report for the number of orders per shopper,  do the following:

- From the main  menu, select **Reports** > **E-Commerce** > **Number 
 of Orders Per Shopper**. You will see the [**Number of Orders Per Shopper**]({{site.rpt_baseurl}}/misc/report_parameters_number_of_orders_per_shopper.html) dialog  box.
- Specify the  report parameters.
- Click **Preview** or **Print**  to view the report on screen or print it through a printer.
