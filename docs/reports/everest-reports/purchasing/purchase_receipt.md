# Purchase Invoice


The **Purchase Invoice** report  gives details of the purchase invoices and debit memos created during  a given time period. This is an analytical report that helps you manage  your accounts payable.


![]({{site.rpt_baseurl}}/img/lens.gif) [Purchase  and Sales analysis]({{site.pp_chm}}/purc-proc/pos/create-po/sales-purchase-analysis/sales_purchase_analysis.html)


{:.note}
Employees created as vendors are not included  in this report.


{:.steps}
To generate the Purchase Invoice report, do the following:

- From the main  menu bar, select **Reports &gt; Purchasing 
 &gt; Purchase Invoice**. You will see the **Purchase Invoice Dialog Box**.
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
