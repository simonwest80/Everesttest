Customer List

The Customer List report displays details of all active and/or inactive customers created within a date range. It is useful for viewing customer details, including details of those customers who are on credit hold. You can also sort the report by sales representatives. The report displays the following information:

  • Customer account code and account name
  • Billing address with the City, State and Zip code
  • Telephone number
  • Sales representative assigned to the customer
  • Date of creation of the customer in Everest
  • Location/Sub-Location the customer is assigned to
  • Customer level

In addition, the detailed output also displays the following details:

  • Fax number
  • Credit limit
  • Whether the customer is on credit hold or not

The Customer List report does NOT display Point of Sale (POS) customers.

To generate the Customer List report, do the following:

  • From the main menu bar, select Reports > Masters > Customer List. You will see the Customer List dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
