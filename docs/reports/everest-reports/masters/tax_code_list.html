Tax Code List

The Tax Code List report displays a list of tax codes and tax components set up for your company. The report displays:

  • Tax code and description.
  • Individual tax components that make up the tax code and the description.
  • Sales tax and purchase tax rates applicable.
  • The order in which tax components are associated with the tax code to determine the sequence in which taxes will be calculated.
  • Whether the tax components are individually taxable (T) or not (F). The Tax on Tax column displays whether the tax has been calculated on the tax that is computed based on the tax component, in the order specified.

To generate the Tax Code List report, do the following:

  • From the main menu bar, select Reports > Masters > Tax Code List. You will see the Tax Code List dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats

Tax Code List Dialog Box

Select the following parameters to view the Tax Code List report.

Tax Code From
Specify the starting code for the range of tax codes to be included in the report.
Drop-Down List - Right-Click Menu Options
Tax Code To
Specify the ending code for the range of tax codes to be included in the report.
Drop-Down List - Right-Click Menu Options

If the Tax Code From and Tax Code To fields are left blank, all tax codes are included in the report.
