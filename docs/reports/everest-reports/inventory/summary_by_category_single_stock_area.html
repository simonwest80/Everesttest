Summary by Category - Single Stock Bin

The Summary by Category - Single Stock Bin report gives the stock status of items in a category in the specified stock bin. The report gives the available stock in the stock bin, the total stock that is on sales order and debit order for the item, the quantity committed from the selected stock bins towards sales order and debit order and the value of stock in that stock bin.

The report also gives the average cost and last cost of the item. The report does NOT use the factored costs.

To generate the Summary by Category - Single Stock Bin report, do the following:

  • From the main menu bar, select Reports > Inventory > Summary by Category - Single Stock Bin. You will see the Summary By Category - Single Stock Bin dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
