Detail By Item - Entry Date (Serialized)

The Detail By Item - Entry Date (Serialized) report details serialized items available in your stock that were received during the period specified in the report parameters. The report is sorted based on item code. This report includes information on each serial number such as its entry date, age in number of days based on entry date, current stock bin code and actual cost.

To generate the Detail by Item - Entry Date (Serialized) report, do the following:

  • From the main menu bar, select Reports > Inventory > Detail By Item - Entry Date (Serialized). You will see the Detail By Item - Entry Date (Serialized) dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
