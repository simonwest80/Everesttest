# Item Tracking (Group)


The **Item Tracking** **(****Group)** report provides information that  helps in categorizing items on the basis of status tags. This report displays  the items that have been tagged to a particular tracking group. The report  includes the item code, the item, the tag date, the user, and the status/memo  description.


{:.example}
‘Featured items on the Web’  is a tracking group you have created. To generate a report for all the  items that are featured on the Web, generate the report for the tracking  group ‘Featured items on Web’.


{:.steps}
To  generate the Item Tracking (Group) report, do the following:

- From the main  menu bar, select **Reports** >  **Inventory** > **Item 
 Tracking** (**Group**). You will  see the [**Item 
 Tracking Group** dialog box]({{site.rpt_baseurl}}/misc/item_tracking_group_dialog_box_rep.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
