# Summary by Item Code - Stock Bin


The **Summary by Item Code** - **Stock Bin** report displays the stock  status of the selected items in the specified stock bin. The report displays  the available stock in the stock bin, the total stock that is on sales  order and debit order for the item, the quantity committed from the selected  stock bin towards sales order and debit order and the value of stock in  that stock bin.


{:.steps}
To generate the Summary by Item Code - Stock Bin report,  do the following:

- From the main  menu bar, select **Reports &gt; Inventory 
 &gt; Summary by Item Code - Stock Bin**. You will see the [**Summary By Item Code - Stock Bin** dialog  box]({{site.rpt_baseurl}}/misc/summary_by_item_code_stock_area_dialog_box_rep.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
