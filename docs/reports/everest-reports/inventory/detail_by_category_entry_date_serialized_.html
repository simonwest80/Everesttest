Detail By Category - Entry Date (Serialized)

The Detail By Category - Entry Date (Serialized) report gives the details of the serial numbers of items belonging to the selected categories. The serial numbers of an item are listed in the order of the date of entry into stock. This report gives the details of serial numbers in both stock and out areas.

To generate the Detail by Category - Entry Date (Serialized) report, do the following:

  • From the main menu bar, select Reports > Inventory > Detail By Category - Entry Date (Serialized). You will see the Detail By Category - Entry Date (Serialized) dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
