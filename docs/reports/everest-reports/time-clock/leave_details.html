Leave Request Details

The Leave Request Details report displays specific leave details for each employee. It shows employee leave applied, rejected, approved and cancelled. It lists the leave type used, the applied and approved dates, the start and end dates for the leave, and to whom the specified leave was applied for approval. This report is useful for senior management during audits, as they can view the status and time when the leave was used by subordinates and by whom they were approved.

To generate the Leave Request Details report, do the following:

  • From the main menu bar, select Reports > Time Clock > Leave Request Details. You will see the Leave Request Details dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
