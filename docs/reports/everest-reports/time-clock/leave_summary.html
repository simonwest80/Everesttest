Leave Summary

The Leave Summary report gives a summarized output of leave taken by your employees for a specified period, including the name of the employee who has availed the leave and the type of leave availed.

To generate the Leave Summary report, do the following:

  • From the main menu bar, select Reports > Time Clock > Leave Summary. You will see the Leave Summary dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
