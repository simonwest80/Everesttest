# Document Tracking (Selection)


The **Document Tracking** (**Selection**) tracks sales documents based  on the tracking selections to which they are tagged. This report is similar  to the [Purchase Document  Tracking (Selection)]({{site.rpt_baseurl}}/everest-reports/purchasing/purchase_doc_tracking_selection_.html) report.


![]({{site.rpt_baseurl}}/img/lens.gif) [Tracking  documents]({{site.ct_chm}}/document-tracking/tracking-sales-documents/track_a_sales_document.html)


{:.steps}
To  generate the Document Tracking (Selection) report, do the following:

- From the main  menu bar, select **Reports &gt; Invoicing 
 &gt; Document Tracking (Selection)**. You will see the [**Document Tracking (Selection)** dialog  box]({{site.rpt_baseurl}}/misc/document_tracking_selection_dialog_box_rep.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
