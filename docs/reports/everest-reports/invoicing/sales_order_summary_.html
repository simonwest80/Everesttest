Sales Order (Summary)

The Sales Order (Summary) report displays a summary of the sales orders that were created during a specified time period. It does not include manual documents. This report provides summary information such as:

  • Amount of the sales order
  • Number of days since the document date
  • Status of the sales order

To generate the Sales Order (Summary) report, do the following:

  • From the main menu bar, select Reports > Invoicing > Sales Order (Summary). You will see the Sales Order Report (Summary) dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
