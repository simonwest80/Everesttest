# Promotional Campaign Detail


The **Promotional Campaign Detail** report  displays the revenue earned through promotion campaigns. You can also  keep track of revenues and expenditures on those campaigns.


{:.steps}
To generate the Promotional Campaign Detail  report, do the following:

- From the main  menu bar, select **Reports &gt; Invoicing 
 &gt; Promotional Campaign Detail**. You will see the [**Promotional Campaign Detail**]({{site.rpt_baseurl}}/misc/promotional_campaign_detail_dialog_box.html) dialog  box.
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
