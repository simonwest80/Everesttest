Customer Statements

The Customer Statements report displays a summary of Customer transactions. It is in the form of a statement, which is sent as a mailer to customers, reminding them of their outstanding payments. You can use this statement as audit evidence of the balance due from your customer. You also receive information about all the transactions pertaining to the customer. You can include complete customer balances for both posted and unposted journal transactions.

The Customer Statements report output contains the following information:

  • Company Address
  • Statement Date
  • Customer’s Billing Address
  • Customer Code
  • Credit Limit
  • Sales Representative
  • Page Number
  • Date Printed On
  • Document Date
  • Document Number
  • Reference
  • Description
  • Due
  • Charge - Amount
  • Credit - Amount
  • Balance - Amount
  • Aging of Balances

To generate the Customer Statements report, do the following:

  • From the main menu bar, select Reports > Invoicing > Customer Statements. You will see the Customer Statements dialog box.

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
