# Accounts Receivable Aging


Use the **Accounts Receivable Aging**  report to view an itemized list of your outstanding receivables along  with payment details for each outstanding document, and to get accurate  data for collection decisions and decisions for a proper receivables turn  around. This report helps you determine your cash flow and losses that  could result from accounts that are not collected.


The report displays the following:

- Customer Account  Name, Code and Customer Balance (in base or customer currency).
- Details for  posted or both posted and unposted documents, filtered for all or a specific  customer and sales representative, depending on the parameter preferences.
- Details of  all document types outstanding such as sales orders, sales invoices, credit  orders and credit memos for both system and manual documents depending  on the parameter preferences.
- Details of  customers who have outstanding balances.
- Details of  each receipt including the Payment Date, Payment Method and Reference  # if any, depending on the parameter preferences.
- Displays Credit  Limit, Invoice/PO #, Rep, and On Hold columns. If the customer is marked  as **On Hold** in the **Customer**  profile, this will be indicated with an asterisk mark.
- Advances against  sales orders/ credit orders depending on the parameter preferences. Sales  orders are displayed in the report only if any advances have been received  against them.
- Amount received,  Paid and the Outstanding Balance.
- Aging details  using either the due date or the document date. For multiple receipts  against the same sales order, all amounts falling in the same time bucket  is summed and displayed in the respective time bucket as a negative amount.
- Total number  of days outstanding (Delq.)  based on the document date or the due date.



{:.steps}
To generate the Accounts Receivable Aging report, do the  following:

- From the main  menu bar, select **Reports** >  **Invoicing** > **Account****Receivable Aging**. You will see  the **Account 
 Receivable** **Aging** dialog  box.
- Specify the  report parameters and click:
- **Preview**,  to preview the report in the **Report Preview**  dialog box, before printing
- **Print**,  to print the report
- **Export**,  to save the report in text, XML, PDF,  Excel, HTML or Crystal Report file formats
