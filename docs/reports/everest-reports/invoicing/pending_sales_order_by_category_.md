# Pending Sales Orders


Use the **Pending Sales Orders**  report to list all sales orders that are not fulfilled. This report provides  a detailed list of all sales orders pertaining to the selected category,  brand, or manufacturer. It contains information such as the document number,  the customer name, the number of items ordered and shipped, and the price  of each line item.


{:.steps}
To generate the Pending Sales Orders report, do the following:

- From the main  menu bar, select **Reports &gt; Invoicing 
 &gt; Pending Sales Orders** report. You will see the [**Pending Sales Orders Report** dialog box]({{site.rpt_baseurl}}/misc/pending_sales_order_report_by_category_dialog_box_rep.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview** dialog box
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats
