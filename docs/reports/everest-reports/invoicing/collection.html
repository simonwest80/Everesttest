Sales Rep Collection

The Sales Rep Collection report displays collections from, and refunds to customers, depending on whether the receipt journal recording the payment/refund transaction has been posted. This report also displays:

  • Document number and date of creation of the sales or return document
  • Sales representative as displayed in the document
  • Taxable value of items on the document
  • Tax amount for the document
  • Document total, including the tax amount
  • Amount paid or refunded on the document
  • Details of each receipt or refund transaction executed on the document, such as the check number, date when the payment or refund was made, whether the payment or refund against the document is applied from other documents, and the sales, tax and total collections for a particular sales representative.

This report is useful in the following ways:

  • Assists you in identifying the total sales and collections for a sales representative over a period of time.
  • Displays a list of all sales and sales return documents where payments or refunds are made, with payment details, over a period of time.

To generate the Sales Rep Collection report, do the following:

  • From the main menu bar, select Reports > Invoicing > Sales Rep Collection. You will see the Sales Rep Collection Report Dialog Box

  • Specify the report parameters and click:

  • Preview, to preview the report in the Report Preview dialog box
  • Print, to print the report
  • Export, to save the report in text, Excel, HTML or Crystal Report file formats
