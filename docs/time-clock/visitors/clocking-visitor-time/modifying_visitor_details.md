# Modify Visitor Details


You can modify a visitor log record, if you have been assigned the necessary  **security rights**. You can modify  the a **Visitor Log** profile from  the following locations:

- The<font style="color: #ff0000;" color="#FF0000"> </font>**Visitor Logs**  browser (path: **Time Clock** >  **Visitor Log** > **Options**  > **Edit** > **In 
 New Window**).
- <font color="#000000" class="hcp2">The </font>**Visitor**<font color="#000000" class="hcp2"> profile - Enter 
 the visitor code for the record that has to be modified.</font>
