Modify Visitor Details

You can modify a visitor log record, if you have been assigned the necessary security rights. You can modify the a Visitor Log profile from the following locations:

  • The Visitor Logs browser (path: Time Clock > Visitor Log > Options > Edit > In New Window).
  • The

    Visitor profile - Enter the visitor code for the record that has to be modified.
