# Delete a Visitor Log Record


You can delete a record from the following location, if you have been  assigned the necessary **security rights**.


{:.steps}
To delete a visitor log record, do the following:

- From the main  menu bar, select **Time Clock** >  **Visitor Logs**. You will see the  **Visitors Log** browser.
- From the **Visitor Logs** browser, highlight the  record and select **Options** >  **Delete**. ****



{:.note}
You cannot retrieve a deleted record.
