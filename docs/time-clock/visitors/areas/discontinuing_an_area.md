# Discontinue an Area


An area that is discontinued is not permanently deleted; it is merely  deactivated. It can be activated when required.


You cannot discontinue an area if it is assigned in the **Purpose 
 of Visit** profile.


To discontinue an assigned area, you must change/reassign the Area in  the **Purpose of Visit** profile.


{:.steps}
To discontinue an area, do the following:

- From the main  menu bar, select **Time Clock** >  **Setup** > **Areas**.  You will see the **Areas** browser.
- From the **Areas** browser, highlight the record  that has to be discontinued.
- Select **Options** > **Discontinue**.  You will see the **Confirm** window.
- Click **OK**.
- The record  is discontinued and will not be displayed in the **Areas**  browser.
