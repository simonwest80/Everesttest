# The Areas Browser


The **Areas** browser displays a  list of all the areas you have defined and other related information.  Browser columns can be re-sized or moved as needed. You can add, edit  or discontinue areas from the browser. To view an **Area**  profile, from the browser, highlight a record and double click.
