# Activate a Discontinued Area


You can make a discontinued area  available for selection by activating it again.


{:.steps}
To activate a discontinued area, do the following:

- From the main  menu bar, select **Time Clock** >  **Setup** > **Areas**.  You will see the **Areas** browser.
- From the **Areas** browser, select **Options**  > **View Discontinued**. You will  see the **Discontinued Areas** browser.
- From the **Discontinued** **Areas**  browser, highlight the record that has to be activated.
- Select **Options** > **Activate**.  You will see the **Confirm** window.
- Click **OK**.
- The record  is activated and will be displayed in the **Areas**  browser.
