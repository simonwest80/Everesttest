# Define an Area


{:.steps}
To define an area, do the following:

- From the main  menu bar, select **Time Clock** >  **Setup** > **Areas**.  You will see the **Areas** browser.
- From the **Areas** browser, select **Options**  > **New &gt; In New Window**. You  will see the **Area** profile.
