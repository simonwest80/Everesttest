# Is Carry Forward
: This column displays ‘Yes’ if the leaves can be carried forward. If you want the leaves to be carried forward then check the option "Is Carry Forward" in the [**Leave** **Types**]({{site.tc_baseurl}}/employees/leave-management/leave-types/the_leave_types_profile.html) profile.
