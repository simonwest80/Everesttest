Check the option ‘Is Carry Forward’ check box

To carry forward leave (for each type of leave), you have to check the option “ls Carry Forward” in the Leave Type profile.

To check the option ‘is carry forward’, do the following:

  • From the main menu bar, select Time Clock > Setup > Leave Types. You will see the Leaves Types browser.
  • Select the leave type from the Leaves Types browser, Options > Edit > In New Window. You will see the Leave Type profile.
  • Check the box “Is Carry Forward” option from the Leave Type profile.
