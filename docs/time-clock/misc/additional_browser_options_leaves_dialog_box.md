# Browser Options 


The **Leaves** browser consists  of the following additional options:


**Approval by Applied To**
: The **Approval by 
 Applied To** option allows the user to approve leave if necessary  security rights have been assigned.


**Approval by CC To**
: The **Approval by 
 CC To** **** option  allows the user (to whom a copy of the leave application has been sent)  to approve leave if necessary security rights have been assigned.


**View All**
: The **View All**  option allows the user to view leave details for the logged in user. If  the necessary security rights have been assigned, then the logged in user  can view leave details for all employees.
