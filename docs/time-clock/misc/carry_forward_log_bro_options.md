# View Carry Forward Log


{:.steps}
To view the leave carry forward log browser,  do the following:

- From the main  menu bar, select **Time Clock** >  **Leave Carry Forward**. You will  see the **Leave Carry Forward** browser.
- From the **Leave Carry Forward** browser, select  **Options** > **Carry 
 Forward Log**. You will see the **Leave Carry Forward Log**  browser.
