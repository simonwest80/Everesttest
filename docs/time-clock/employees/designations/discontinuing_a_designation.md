# Discontinue a Designation


A designation that is discontinued is not permanently deleted, but only  deactivated. It can be activated when required.


You cannot discontinue a designation if it is assigned in the **User** profile (path: **File**  > **Setup** > **Users**  > **Users**).


To discontinue an assigned designation, you must change/reassign the  designation in the **User** profile.


![]({{site.tc_baseurl}}/img/lens.gif) [The  User Profile]({{site.sc_chm}}/options/security/users/set-up-a-user/the_user_profile.html)


{:.steps}
To discontinue a designation, do the following:

- From the main  menu bar, select **Time Clock** >  **Setup** > **Designation**.  You will see the **Designations**  browser.
- From the **Designations** browser, highlight the  record that has to be discontinued.
- Select **Options** > **Discontinue**.  You will see the **Confirm** window.
- Click **OK**.
- The record  is deactivated and will not be displayed in the browser.
