# Record Employee Clock In/Clock Out Time


{:.steps}
To record  employee clock in/out time, do the following:

- From the main  menu bar, select **Time Clock** >  **Clock In/Out**. You will see the  **Clock In/Out** dialog box.
- Specify the  employee’s name. The system automatically registers the start time and  date an employee reported to work. The system uses the time from the client  workstation’s operating system settings.
- When the employee’s  name is re-entered, the system automatically clocks the end time for the  employee.
