# The Leave Types Profile


The **Leave Types** profile allows  you to enter the code and description of a leave, and the frequency with  which the leave type is credited to an employee's account.


**Code**
: Enter a unique code that identifies the leave type  using from 1 to 15 characters of the supported type. You cannot edit this  field once the profile is saved.


**Description**
: Enter a suitable description for the leave type  using 1 to 40 characters of any type.


**Is Special Leave**
: Check this option if the employee is granted leave  for a special reason.


{:.note}
For a special leave, frequency of leaves is  not applicable. Hence, the **Frequency**  and **Is Carry Forward** fields are  disabled if you check this option.
: Select the frequency with which leaves are credited  to an employee’s leave account in a year. Different types of leaves may  be credited at different points of time in a year.


![]({{site.tc_baseurl}}/img/example.gif) Medical  leave can be credited every three months; casual leave, every month; and  sick leave, every two months.


**Is Carry Forward**
: Check this field if the employee’s leave balance  of the selected type must be carried forward to the following period.


**Is Intervene off Leave**
: Check this field if regular holidays that occur  in between the period of the employee’s leave are counted as leave and  not as regular offs.


![]({{site.tc_baseurl}}/img/example.gif) An employee may take  leave from Friday to Monday. Though Saturday and Sunday are regular offs,  these days are counted as leave.
