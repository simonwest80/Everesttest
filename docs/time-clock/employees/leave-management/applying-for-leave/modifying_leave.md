# Modify Leave


You can modify the Leave from the following locations:

- The **Leave 
 Requests** browser (path: **Time Clock** > **Leave Requests**) -  Highlight the record to be modified, select **Options**  > **Edit** > **In Current Window**/**In New Window**.
- The **Leave 
 Request** profile - Specify the leave types code in the code field.
