# Apply for Leave


All employees who require leave must apply for the same.


{:.steps}
To apply for leave, do the following:

- From the main  menu bar, select **Time Clock** >  **Leave Requests**. You will see the  **Leave** **Requests**  browser.
- From the **Leave Requests** browser, select **Options** > **New** > **In New Window**. You  will see the **Leave Request** profile.
