# View Leave Credit


{:.steps}
To run the leave credit process, do the following:

- From the main  menu bar, select **Time Clock** >  **Leave Credit**. You will see the  **Leave Credit** browser.
- From the **Leave Credit** browser, select **Options**  > **New** > **In 
 Current Window**. You will see the **Leave 
 Credit** profile.
