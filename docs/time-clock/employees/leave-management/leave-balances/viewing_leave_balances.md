# View Leave Balances


The leave balance for a user can be viewed from the **User**  profile.


{:.steps}
To view the leave balance for a user, do  the following:

- From the main  menu, select **File &gt; Setup &gt; Users 
 &gt; Users**. You will see the **Users**  browser.
- Highlight the  user for whom you want to view the leave balance and select **Leave 
 Balance** from the browser options. The **Leave 
 Balance** browser that appears displays the leave types that the  user is entitled, the opening leave balance as well as the number of leaves  availed.
