Leaves Balance

Each employee should have an opening balance of leaves, for the leaves to be carried forward. You can assign opening leave balance for each type of leave, for employees, from the Leaves Balance profile.You will have to assign the leave type and the corresponding opening balance if any for each leave type separately.
The leave types for an employee can include sick leave, casual leave or paid leave.

You can assign leaves only if have security rights assigned to you.
