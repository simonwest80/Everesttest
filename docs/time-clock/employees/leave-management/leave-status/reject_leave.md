# Reject Leave


A superior to whom the leave application is submitted, can reject the  leave.


{:.steps}
To reject leave, do the following:

- From the main  menu bar, select **Time Clock** >  **Leave Requests**. You will see the  **Leave Requests** browser.
- From the **Leave** **Requests**  browser, select the user, for whom you want to reject leave. Click on  **Options** > **Reject**.
