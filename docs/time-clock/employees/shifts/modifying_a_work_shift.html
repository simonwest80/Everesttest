Modify a Shift

You can modify the Shift profile from the following locations:

  • The Shifts browser (path: Time Clock > Setup > Shifts) - Highlight the record to be modified and select Options > Edit > In New Window.
  • The Shift profile - Specify the shift code in the code field.

You cannot modify the Code field.

You can only edit the time in the Shift profile if there are no pending leave requests. Use the View Pending Leave Requests option in the Shifts browser to view any pending leave requests.
