# Delete an Employee Log Record


{:.steps}
To delete a record from the employeeâ€™s log,  do the following:

- From the main  menu bar, select **Time Clock** >  **Employee Logs**. You will see the  **Employee Log** browser.
- From the **Employee Log** browser, highlight the  record and select **Options** >  **Delete**. The record is deleted.



{:.note}
You cannot retrieve a deleted record. Therefore,  delete records only after you have transferred the leave balances, and  the payroll reports have been generated.
