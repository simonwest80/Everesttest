# Modify an Employee Log Record


You can modify an Employee Log record from the following location:

- The **Employee 
 Log** browser (path: **Time Clock**  > **Employee Logs &gt; Options**  > **Edit** > **In 
 New Window**).
- The **Employee 
 Log** profile - Enter the employee code for the record to be modified.
