# The Holidays Browser


The **Holidays** browser displays  a list of all the holidays/weekly offs and other related information.  Browser columns can be re-sized or moved as needed. You can add, edit  or discontinue holidays/weekly offs from the browser. To view the **Holiday/Weekly off** profile, select the  record form the browser and double click.
