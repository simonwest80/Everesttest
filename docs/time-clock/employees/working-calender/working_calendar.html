Working Calendar

The organization’s working calendar is created based on the information set up in the following profiles:

  • Shifts
  • Holidays
  • Weekly offs

Working calendars can be set up for each shift or operation.

You have a retail outlet in Canada and a factory in the United States. The retail outlet is open on weekends while the factory is closed. You can create separate working calendars for each of these operations.
