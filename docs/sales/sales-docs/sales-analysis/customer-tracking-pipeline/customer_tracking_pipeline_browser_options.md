# Customer Tracking Pipeline Browser Options


The **Customer Tracking Pipeline** browser  (path: **Invoicing** > **Sales Analysis** > **Customer Tracking 
 Pipeline**) displays all sales documents that are tagged with a custom  tracking selection.


The additional options (path: **Customer 
 Tracking Pipeline** browser > **Options**)  available in this browser are:

- **Change 
 Pipe Color** - Select this option to change the colors in the [Customer Tracking Pipeline](javascript:BSSCPopup('{{site.sp_baseurl}}/misc/customer_tracking_pipeline_popup.html');){:id="A1"}<script type="text/javascript" language="JavaScript1.2"><!--
kadovFilePopupInit('A1');
//--></script>.![]({{site.sp_baseurl}}/img/lens.gif) [Change  Pipe Color]({{site.sp_baseurl}}/misc/change_pipe_color_cust_tracking_pipeline.html)


- **Toggle 
 Pipeline Display** - This is a toggle option that hides/displays  the diagrammatic representation of the customer tracking pipeline.
- **Save** - Select this option to save the customer tracking pipeline as  a **.bmp**file. You will see the **Save As** dialog box. Specify the name and location where you want to save  the file.



Options common to all browsers in **Everest** are also available in this browser.
