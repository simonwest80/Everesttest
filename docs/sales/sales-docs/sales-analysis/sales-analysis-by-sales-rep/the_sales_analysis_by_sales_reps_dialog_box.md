# The Sales Analysis by Sales Rep Dialog Box


The **Sales Analysis by Sales Rep**  dialog box comprises the following sections:

- [Period]({{site.sp_baseurl}}/misc/period_sales_analysis_by_sales_reps_dialog_box.html)
- [Include  periodic columns]({{site.sp_baseurl}}/misc/include_periodic_columns_sales_analysis_by_sales_reps_dialog_box.html)



After specifying the details in this dialog box and selecting the **Show Document Details** check box, click  **OK**. The [**Sales Document Breakup** browser]({{site.sp_baseurl}}/sales-docs/sales-analysis/sales-analysis-by-sales-rep/the_sales_document_breakup_browser.html) is  displayed. This browser displays all the documents that meet the criteria  specified in the **Sales Analysis by Sales 
 Rep** dialog box.
