# When a Deposit is Received in Advance


If the entire amount of payment due for all recurring sales orders in  the series is collected in advance as a credit card receipt, **Everest** automatically applies the payment to the invoice when each order  is processed.


**![]({{site.sp_baseurl}}/img/example.gif) If an  amount of 12000 is received in advance towards rent for the whole year,  1000 is automatically applied to the rent every month when the order is  processed to an invoice.**


{:.steps}
To receive advance payment on recurring sales  orders, do the following:

- From the **Recurrence Pattern** dialog box (path:  **Invoicing** > **Sales Orders** browser > **Options**  > **Recurring Documents** > **Options** > **New** > **In New Window** > **Sales Order** profile > **Options**  > **Recurrence Pattern**), select  the **Credit Card Payment** check  box. Select a **Payment Method** and  select the **Full Credit Card Payment** check  box.
- Click **OK**. The [**Payment on Recurring Document** dialog  box]({{site.sp_baseurl}}/misc/payment_on_recurring_document_dialog_box.html) is displayed.
- Enter the details  and click **OK**.
- You will see  the [**Card 
 Verification** dialog box]({{site.sp_baseurl}}/sales-docs/sales-orders/so-proc/pmts-refunds/cc-pmts/cc-verify-dlg/the_card_verification_dialog_box.html).
