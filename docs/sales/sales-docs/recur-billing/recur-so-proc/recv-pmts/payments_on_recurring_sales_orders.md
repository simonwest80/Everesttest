# Payments on Recurring Sales Orders


You can receive payments on recurring sales orders in any of the following  ways:

- In advance  using a credit card type of payment method.



**![]({{site.sp_baseurl}}/img/lens.gif) [When  a Deposit is Received in Advance]({{site.sp_baseurl}}/sales-docs/recur-billing/recur-so-proc/recv-pmts/when_deposit_is_received_in_advance.html)**

- As a deposit  when each sales order becomes due.



**![]({{site.sp_baseurl}}/img/lens.gif) [Payments  on Sales Orders]({{site.sp_baseurl}}/sales-docs/sales-orders/so-proc/pmts-refunds/pmts-on-so/payments_on_sales_orders.html)**

- When the recurring  sales order is converted to an invoice.



**![]({{site.sp_baseurl}}/img/lens.gif) [Receiving  Payments on Sales Invoices]({{site.sp_baseurl}}/sales-docs/sis/sales-invoice-processes/receive-payments-make-refunds/receiving_payments_on_sales_invoices.html)**
