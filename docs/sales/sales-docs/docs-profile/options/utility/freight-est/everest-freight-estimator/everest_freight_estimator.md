# Everest Freight Estimator


The **Everest Freight Estimator** option  allows you to compute freight based on destination zones for all countries.


**To enable the **Everest 
 Freight Estimator** option to compute freight, the name of the carrier  has to be identified in the **Delivery****Method** profile. Freight is calculated  only when the delivery method is company delivery and not a customer pickup.**


**![]({{site.sp_baseurl}}/img/lens.gif) [Delivery  Methods]({{site.sc_chm}}/options/everest-freight-estimator/delivery-methods/delivery_methods_sales4.html)**


<font face="Verdana" class="hcp4">Freight is calculated 
 based on the:</font>

- Originating  zip code (of the document's parent Â location).
- Destination  zip code (of the ship to address in the document).
- Ship-via agent  (the delivery method selected).
- Rate Definition.



<font face="Verdana" class="hcp4">The freight charges 
 include transportation cost, COD fee, residential charges and insurance 
 cost wherever applicable. </font>
