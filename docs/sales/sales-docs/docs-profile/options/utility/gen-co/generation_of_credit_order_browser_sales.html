Generation of Credit Order From Sales Invoice Browser

The Generation of Credit Order from Sales Invoice browser allows you to create a credit order for the selected item (path: Sales Document profile > Options > Utility > Generate Credit Order).

Generation of Credit Order Browser Options

The following are the browser options available in the Generation of Credit Order from Sales Invoice browser:

Select/Unselect All Items

The Select/Unselect All Items toggle option (path: Options > Select/ Unselect All Items) allows you to select all the records displayed in the browser or deselect all the records that have been selected.

Create CO

The Create CO option allows you to create a credit order for the line items you have selected in the Generation of Credit Order from Sales Invoice browser.

Options common to all browsers are also available in this browser.
