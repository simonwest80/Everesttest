Get Latest Exchange Rate

The Get Latest Exchange Rate option (path: Sales Document profile > Options > Get Latest Exchange Rate) allows you to retrieve the latest exchange rate on a sales document, as on the login date. You can edit the exchange rate. The Exchange Rate option is enabled only for foreign currency customers.

The exchange rate is retrieved from the Buy Rate/Sell Rate fields in the Exchange Rates profile (path: File > Setup > Accounting > Multicurrency > Exchange Rates > Options > Edit > In New Window).

The rate displayed in the Exchange Rate field (whether buy rate or sell rate), in turn, depends on the option selected in the Rate for sales documents field in the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting > Multicurrency > Exchange Rates tab).
