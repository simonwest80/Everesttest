# Edit Matrix Group


{:.steps}
To modify an item matrix, do the following:

- From the sales  document highlight the matrix item.
- Select **Options &gt; Item &gt; Edit Matrix Group**.  You will see the<font style="color: #ff0000;" color="#FF0000"> </font>[**Matrix Items View** dialog box]({{site.sp_baseurl}}/sales-docs/docs-profile/contents/item-info/details/matrix-items-dlg/the_matrix_items_view_dialog_box.html).
