# Show Lots


This option allows you to view all Lot related details of the line item  in a single browser. The lot details for this browser have been captured  and stored at the time of attaching the quantities. A relevant security  right is needed to view this browser. You can access this option either  through the **Options** menu or pressing  Ctrl + Alt + Y. You can sort the records by any of the columns.
