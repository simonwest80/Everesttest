# Line Items


The **Line** **Items**  option (path: **Sales** **Document**  profile > **Options** > **Line** **Items**)  allows you to do the following:


| Options | Description |
| Insert | Inserts a blank line in the Item Details Grid. |
| Upsell  Information | Displays information entered in the **Upsell <br/> Information** field of the [**Item** profile]({{site.mi_chm}}/create-regular-items-kits-and-assemblies/creating-an-item/the_items_profile_-_notes.html) (path: **Inventory <br/> &gt; Items &gt; Options &gt; Edit &gt; In New Window &gt; Notes**  tab). |
| Line Profile | Displays information about the line highlighted in the Item Details  Grid in the [**Sales <br/> Document Lines** dialog box]({{site.sp_baseurl}}/sales-docs/docs-profile/options/line-items/the_sales_document_lines_dialog_box.html). |
| Move up | Moves the selected line item one line up. |
| Move down | Moves the selected line item one line down. |
| Find | When you have a number of items in a sales/sales return document, use  this option to find and edit the specific line item. When you select this  option, you will see the [**Find Line Items** browser]({{site.sp_baseurl}}/sales-docs/docs-profile/options/line-items/the_find_line_items_browser_sales.html). |
