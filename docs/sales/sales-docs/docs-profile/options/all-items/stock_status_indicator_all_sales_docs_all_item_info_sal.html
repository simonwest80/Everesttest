Stock Status Indicator (All)

The Stock Status Indicator (All) option (Shift + F10) is available if the “Automatically display stock status indicators” preference in the Items dialog box – Item Details tab (path: File > Setup > Preferences > Flow Control > Items) is NOT selected (disabled). We recommend that you disable this option to improve performance when adding line items to sales documents. In this situation, the Stock Status Indicators do not automatically display for the line items.

To view the stock status for all of the line items in the sales document, press Shift + F10 (path: From a sales document, select Options > All Items > Stock Status Indicator (All). You will see the Stock column populated for all stocked line items.

Likewise, to view the stock status for only one line item, you can select the line item and press F10 (path: From a sales document, select Options > Item > Stock Status Indicator). You will see the Stock column indicating stock status color and quantity for the selected line item.

If you select “Automatically display stock status indicators”, the stock status automatically displays on the sales documents and the Stock Status Indicator (All) option is disabled.
