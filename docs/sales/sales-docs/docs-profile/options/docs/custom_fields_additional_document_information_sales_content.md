# Custom Fields


The **Custom Fields** option (path:  Sales Document **** profile > **Options** > **Document**  > **Custom Fields**) allows you  to track records based on user-defined fields. Custom fields can be defined  as character strings, date fields, logical, memo or numeric fields. **Everest** provides the facility of using  unlimited custom fields, which can be displayed in the associated browser.  When a sales quote or sales order is converted, the data in the custom  fields are copied to the converted document.


{:.note}
Custom fields are available for sales and sales return  documents in common. You cannot create separate custom fields for a specific  document type.
