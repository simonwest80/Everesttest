Document Tracking

The Document Tracking option (path: Sales Document ** profile > **Options > Document > Document Tracking) allows you to tag the document with tracking selections for tracking groups available for the document type. You can only view the selections and cannot change selections if the tracking group is in Viewonly mode for that document type. To tag a selection, highlight the tracking selection and choose the option Select/Unselect from the right-click menu.

Tracking Sales Documents

You can set Document Tracking preferences (path: File > Setup > Preferences> Flow Control > Document Tracking) to ensure that the Document Tracking Groups automatically pop up whenever you create, edit and save a document or only when you create a document.
