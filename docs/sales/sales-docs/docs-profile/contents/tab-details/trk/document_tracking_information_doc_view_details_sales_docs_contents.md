# <font style="color: #000000;" color="#000000">Document Tracking Information</font>


The **Document Tracking** tab allows  you to tag the document with the appropriate tracking selections for the  groups available.


{:.note}
This tab is displayed only when you save the sales document.


You cannot modify the selections if the tracking group is in "view  only" mode for that document type. Â To  tag a selection to the document, right-click on the **Selection**  section and choose the option 'Select'.


You can set document tracking preferences to ensure that the tracking  groups automatically pop up either when you create, edit or save a document,  or only when you create a document (path: **File 
 &gt; Setup &gt; Preferences &gt; Flow Control &gt; Document Tracking**).
