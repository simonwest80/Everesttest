# Tax Calculation Type


You can modify the tax amount in a sales document based on the type of tax calculation method you select. The **Tax Calculation Type** field contains three options:

- [Automatic]({{site.sp_baseurl}}/misc/automatic_tax_calculation_type_pyt_infon.html)
- [Special tax rule override]({{site.sp_baseurl}}/misc/special_tax_rule_override_tax_calcuation_type_pyt_infon.html)
- [Manual override]({{site.sp_baseurl}}/misc/manual_override_tax_calculation_type_pyt_infon.html)
