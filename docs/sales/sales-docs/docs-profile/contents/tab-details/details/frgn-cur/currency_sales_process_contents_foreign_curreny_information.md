# Currency


The **Currency** field displays  the currency defined for the customer in the **Currency****Code** field of **Customer**  profile (path: **Invoicing** >  **Customers** > **Options**  > **Edit** > **In 
 New Window** > **Accounting**  tab). A sales document cannot be created in a currency other than the  customer's currency. You cannot edit this field.
