Custom Fields Information

Use the Custom Fields tab to define additional information for any record as characters, date, logical, memo or numeric fields.

If a sales document has a last date for receiving returns that is set in the Date custom field, you can search for the sales document using this parameter in the Custom Field tab.

Custom fields defined on the Custom Fields tab of the sales document apply to the overall sales document. They do not apply to individual line items. Line item custom fields are defined separately. Use the Sales Document Lines dialog box – Custom Fields tab (path: sales/sales return document profile > Options > Line Items > Line Profile or press Ctrl + Right Arrow in any document) to create line item custom fields for sales documents.
