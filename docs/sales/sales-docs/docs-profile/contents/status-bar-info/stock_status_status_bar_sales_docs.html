Stock Status

This option displays the stock status for the items in the sales document. The stock status indicators are as follows:

Status Displayed Icon Description
Enough stock


(color: green)		 The available stock for the item is greater than zero. Click this icon to view the Item Stock
Status dialog box that displays detailed information about the stock status of the items in the sales document.
Enough stock but not enough available


(color: yellow)		 The total stock is greater than zero, but the available stock is zero or has a negative value. Click this icon to view the Item Stock Status dialog box that displays detailed information about the stock status of the items in the sales document.
Not enough stock


(color: red)		 The total stock is zero and the available stock is negative. Click this icon to view the Item Stock
Status dialog box that displays detailed information about the stock status of the items in the sales document.

Item Stock Status Dialog Box

  • Even if one item in the document has no stock, the Not enough stock icon is displayed in the status bar of the document.
  • The Stock Area field displays in the status bar when you need to select a stock area. Select a stock area and press the Enter key or ESC key to exit this field and continue.
