# Show/Hide


Click the **Hide** ![]({{site.sp_baseurl}}/img/sales_hide_button.gif) button to hide lines that are marked hidden. Click the **Show** ![]({{site.sp_baseurl}}/img/sales_show_button.gif) button to display  all lines including hidden lines. By default all hidden lines are displayed  in sales documents.
