# Order Under Picking


The left corner of the status bar displays the message 'Order under  picking' if  the sales order is selected for picking in the **Pull 
 Sheet Wizard**.


The sales order becomes an order under picking when the pick ticket  number is generated.


The status "Order under picking" is displayed till the pull  sheet created for the sales order is manually marked as closed or is automatically  closed by committing all the required items on the pull sheet.
