# Unposted/Posted


The posting status of the sales journal, in which the accounting entry  for the sales invoice is made, is displayed in this section. This section  of the status bar is displayed only for sales invoices.


| **Status Displayed** | **Description** |
| ![]({{site.sp_baseurl}}/img/sales_posted_icon_status_bar.gif) | The sales journal has been posted to the general ledger. |
| ![]({{site.sp_baseurl}}/img/sales_not_posted_icon_status_bar.gif) | The sales journal has not been posted. |



Click in this section to view the [**Journal Transactions** browser]({{site.sp_baseurl}}/misc/journal_transactions_browser.html)<font style="color: #000000;" color="#000000">.</font>
