# Stock


The **Stock** column displays the  stock status for the line item in the sales document. The stock status  indicators are as follows:


| Status Displayed | Icon | Description |
| Enough stock | ![]({{site.sp_baseurl}}/img/green enough stock.jpg) | Displays this color when the Available Stock for the  item is greater than zero. Click the ![]({{site.sp_baseurl}}/img/sales_enough_stock_icon.gif) icon on the Status  bar to view the **Item Stock Status**  dialog box that displays detailed information about the stock status of  the items in the sales document. |
| Enough stock but not enough available | ![]({{site.sp_baseurl}}/img/yellow not enough available.jpg) | Displays this color when the total stock is greater  than zero, but the available stock is zero or has a negative value. Click  the ![]({{site.sp_baseurl}}/img/sales_st_enough_but_not_available_icon.gif) icon on the Status bar to view the **Item <br/> Stock Status** dialog box that displays detailed information about  the stock status of the items in the sales document. |
| Not enough stock | ![]({{site.sp_baseurl}}/img/red no stock color.jpg) | Displays this color when the total stock is less than  zero. Click the ![]({{site.sp_baseurl}}/img/sales_not_enough_stock_icon.gif) icon on the Status bar to view the **Item Stock Status** dialog box that displays  detailed information about the stock status of the items in the sales  document |



The **Stock**  column displays for the line item depending upon the preference set in  the **Items** dialog box - **Item 
 Details** tab (path: **File**  > **Setup** > **Preferences**  > **Flow Control** > **Items**).  You can set the Stock Status Indicator to automatically display or to  display only when requested to improve performance. If the **Stock**  column is set to not automatically display on the sales documents, you  can select a single line item and press **F10**  (or press **Shift + F10** for all  line items) to view the Stock Status Indicators. In this situation, the  stock status indicator on the status bar only appears after you manually  select the stock status indicators for the line items.
