Retrieve Items from Documents

Items in a sales document can be specified or retrieved from another document. You can also modify the retrieved information. You can retrieve item information from the following documents:

  • Sales Quote
  • Sales Order
  • Sales Invoice
  • Credit Memo

To retrieve items from a document, do the following:

  • From a sales document, select Options > Retrieve > Copy of. You will see a list of document types in Everest.
  • Click the document type that contains the document from which items have to be extracted to the current document profile. You will see the document browser.
  • From the document browser double-click on the relevant records. The items in the selected document are copied to the current document.

When you retrieve items that belong to a different customer from a credit memo, Everest prompts you of the same. You are also asked whether you want to retain the current price used in the retrieved document. Select your option and click OK.

Information pertaining to the vendor or customer of the original document is not retrieved into the new document.

In case of serialized items, the quantity fields in the Item Details Grid are not populated with values from the original item.

If the document that you retrieve contains discontinued item(s), you will receive a message stating that the sales document contains discontinued items. You can choose to:

  • Retrieve active items - Select this option to add only active items to the sales document.
  • Retrieve active and discontinued items - Select this option to add both active and discontinued items to the sales document.
