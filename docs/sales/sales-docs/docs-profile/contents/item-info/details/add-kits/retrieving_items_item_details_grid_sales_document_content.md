# Add an Item to a Sales Document


You can add an item to a sales document in the following ways:

- From the Sales  Document profile by selecting **Options 
 &gt; Item &gt; Select Items**.



![]({{site.sp_baseurl}}/img/lens.gif) [Add  Items - Options Menu ]({{site.sp_baseurl}}/misc/add_items_-_options_menu_sales_doc.html)

- By using the  [Smart Search]({{site.sp_baseurl}}/misc/smart_search_items_dialog_box_sales_doc.html)  option.
