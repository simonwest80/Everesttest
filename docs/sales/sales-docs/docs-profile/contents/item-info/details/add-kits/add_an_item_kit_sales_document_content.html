Add an Item Kit to a Sales Document

To add an item kit to the sales document, do the following:

  • From the Sales Document profile, select Options > Item > Select Kits. You will see the Items browser.
  • From the Items browser, double-click on the required item kit. You will see the Retrieve Kits dialog box.
  • Enter the required quantity and click OK.
  • All items that belong to the item kit are added to the sales document.

The measure of kit components in documents can be changed.
