# Inline Notes


The **Inline Note** can be added  as a line in the Item Details Grid. The inline note can also be entered  from the **Description** column in  the Item Details Grid.


{:.steps}
To add inline notes, do the following:

- From a sales  document profile, select **Options**  > **Inline Notes**. You will see  the **Description** dialog box.
- Enter the required  note and select **File &gt; Save/Save and 
 Close** to return to the document.
