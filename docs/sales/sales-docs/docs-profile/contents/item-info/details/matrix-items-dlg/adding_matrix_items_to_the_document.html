Add Matrix Items to the Document

The item codes, for all combinations with ordered quantity greater than zero, are added to the document. The selling price, quote cost, discount and ordered quantity columns in the sales document are populated with the corresponding values in the MatrixItems View dialog box.

If a matrix item for T-shirts is selected, combinations of size and color are displayed. If you enter the quantity of sales for size - large and color - white, and size - medium and color - red, the item codes for these combinations are added to the grid.

Only the ordered quantity can be entered from the Matrix Items View dialog box. Quantities attached can only be specified in the sales document.
