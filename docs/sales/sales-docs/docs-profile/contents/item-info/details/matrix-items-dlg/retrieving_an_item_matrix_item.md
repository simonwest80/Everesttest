# Retrieve a Matrix Item


{:.steps}
To retrieve a matrix item, do the following:

- From the Sales  Document profile, select **Options**  > **Retrieve** > **Item 
 Matrix**. You will see the **Items**  browser. This browser displays only items of the type 'matrix group'.
- From **** the **Items** browser, double-click  the required item. The [**Matrix Items View** dialog box]({{site.sp_baseurl}}/sales-docs/docs-profile/contents/item-info/details/matrix-items-dlg/the_matrix_items_view_dialog_box.html) or  the [Matrix  Items browser]({{site.mi_chm}}/creating-matrix-group-and-matrix-items/the_matrix_items_browser.html) is displayed depending on whether you select **Matrix**  or **Items** in the **Matrix 
 Group Display Preference Type** field (path: **Inventory 
 &gt; Items &gt; Options &gt; Edit &gt; In New Window &gt; Item** profile **- General** tab).



{:.note}
You can select any number of matrix items from the browser.  The **Matrix Items View** dialog box  is displayed once for every matrix that is selected.


If there is a time lag before the **Matrix 
 Items View** dialog box is displayed, you can view the progress in  the **Retrieving Matrix** window.  Click **View List** in this window  to view the **Matrix Items** browser
