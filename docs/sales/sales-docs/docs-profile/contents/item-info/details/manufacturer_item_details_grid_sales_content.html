Manufacturer

A manufacturer can be a person, an enterprise or an entity that produces the item that you sell. If you have specified the manufacturer of the item in the Manufacturer field of the Item profile (path: Inventory > Manufacturers > Options > Edit > In New Window), the manufacturer’s code is displayed in the Manufacturer column. You can set up manufacturers in the Manufacturer profile (path: Inventory > Manufacturers > Options > Edit > In New Window). You cannot edit this column.
