# Hide


The **Hide** column (check box)  allows you to hide lines in the Item Details Grid.


{:.steps}
To hide a line, do the following:

- From a sales  document profile, highlight the item.
- Scroll to the  **Hide** column.
- Check the box  by pressing the **Shift** key.



{:.note}
You can only hide those items for which no  price has been specified.

- Click the **Hide** button ![]({{site.sp_baseurl}}/img/sales_hide_button.gif) or the **Show** button ![]({{site.sp_baseurl}}/img/sales_show_button.gif) displayed  in the sales document status bar to hide or view the selected line.
