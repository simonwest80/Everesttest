Drop Ship

A check mark appears in the Drop Ship column of the Item Details Grid for items that are marked for drop shipment in the Item profile (path: Inventory > Items > Options > Edit > In New Window > Purchases tab). A check mark also appears in the Drop Ship column if you have created a drop ship purchase order for the line item.

This option is available only in sales orders. You can choose to deselect the check box in the Drop Ship column if you do not want to create a drop ship purchase order for the item.
