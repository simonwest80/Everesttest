# Brand


A brand is a trademark or distinctive name identifying a product. If  you have specified a brand to which the item belongs in the **Brand**  field of the [**Item** profile]({{site.mi_chm}}/create-regular-items-kits-and-assemblies/creating-an-item/the_item_profile_-_general.html) (path: **Inventory 
 &gt; Items &gt; Options &gt; Edit &gt; In New Window &gt; General** tab),  the brand code is displayed in the **Brand**  column. You can set up brands in the **Brand** profile  (path: **Inventory &gt; Brands &gt; Options 
 &gt; Edit &gt; In New Window**). You cannot edit this column.
