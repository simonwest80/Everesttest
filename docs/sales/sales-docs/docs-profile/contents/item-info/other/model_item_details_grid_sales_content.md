# Model


You may have two different models of an item manufactured by the same  company


![]({{site.sp_baseurl}}/img/example.gif) Printers  HP Color Laserjet  5M and HP Color Laserjet  5/ 5M PS are 2 models manufactured by HP in the Laserjet  range.


If you have specified the model of the item in the **Model**  field of the [**Item** profile]({{site.mi_chm}}/create-regular-items-kits-and-assemblies/creating-an-item/the_item_profile_-_general.html) (path: **Inventory 
 &gt; Items &gt; Options &gt; Edit &gt; In New Window &gt; General** tab),  the model of the item is displayed in the **Model**  column. You cannot edit this column.
