Kit Code

This column displays the code of the kit to which the item belongs, thus enabling you to easily identify line items that are part of a kit.

Items with the same kit code as selected in the Related Items profile are displayed when you view the related items (path: Document profile > Options > Item > Related Items) for a line item that is part of a kit.
