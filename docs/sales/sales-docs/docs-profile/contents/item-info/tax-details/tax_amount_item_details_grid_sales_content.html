Tax Amount

Tax is computed on the item’s net sale value (selling price less discount). It is the total of the tax amounts calculated on each tax component of the tax code. The tax amount has a decimal accuracy of six.

A tax code consists of a set of individual tax components that are applied together.

A local tax component and a state tax component might be combined into a tax code.

For a foreign currency document, the tax amount in the document currency mode is expressed in terms of the foreign currency.

Sales Tax Calculations

Click the Tax icon beside the Tax field to obtain a detailed breakup of the tax charged. You will see the Tax Breakdown window. This window displays the description of the items, the taxes charged for each item and the tax amount for each item.
