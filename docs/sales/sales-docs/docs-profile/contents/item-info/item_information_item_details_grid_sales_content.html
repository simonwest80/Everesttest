Item Information

Items in a sales document are displayed in the **Item Details Grid in a tabular format. Each row in the Item Details Grid displays information about the item specified in that row.**

A sales quote or a sales order can be created without specifying item details; the item details can be entered later.

  • You must select at least one item in a Sales Invoice.
  • Adequate security right is required when accessing items of a location/sub-location that is different from logged in location/sub-location.

Item information in the **Item Details Grid has been grouped under the following sections:**

  • Item Details
  • Discounts
  • Tax Details
  • Account Details
  • Other Item Details
