Ship to Address

The **Ship To field specifies the customer’s warehouse or office address to which the items sold must be delivered. The freight on the shipment and sales tax is calculated based on the shipping address.**

When you select a customer code, the default shipping address specified in the Shipping Address field of the Customer profile (path: Invoicing > Customers > Options > Edit > In New Window > Addresses tab) is displayed.

Customer Profile - Addresses

You can specify a different **Ship To address or create a new one.**
