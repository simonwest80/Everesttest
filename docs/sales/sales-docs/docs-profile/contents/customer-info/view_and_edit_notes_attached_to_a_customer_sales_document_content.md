# View Notes Attached to a Customer


{:.steps}
To view notes attached to a customer, do the following:

- From the **Sales** **Document**  profile, click the **Notes** icon  ![]({{site.sp_baseurl}}/img/sales_notes_button.gif) beside the customer name. You will see the **Customer 
 Notes** dialog box.



{:.note}
You can view the **Customer 
 Notes** dialog box by clicking the **Notes**  icon if notes are entered in the **Comments**  section of the **Customer** profile  (path: **Invoicing &gt; Customers &gt; 
 Options** > **Edit** >  **In New Window &gt; Miscellaneous** tab).
