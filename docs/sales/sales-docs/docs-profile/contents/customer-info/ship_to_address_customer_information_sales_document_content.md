# Ship to Address


**The **Ship 
 To** field specifies the customer's warehouse or office address to  which the items sold must be delivered. The freight on the shipment and  sales tax is calculated based on the shipping address.**


When you select a customer code, the [default shipping address](javascript:BSSCPopup('{{site.sp_baseurl}}/misc/default_shipping_address.html');){:id="A1"}<script type="text/javascript" language="JavaScript1.2"><!--

//--></script>  specified in the **Shipping** A**ddress** field of the **Customer**  profile (path: **Invoicing** >  **Customers** > **Options**  > **Edit** > **In 
 New Window** > **Addresses**  tab) is displayed.


![]({{site.sp_baseurl}}/img/lens.gif) [Customer  Profile - Addresses]({{site.mc_chm}}/creating-a-customer/the-customer-profile-addresses/the_customer_profile_addresses.html)


**You can specify a different **Ship 
 To** address or create a new one.**
