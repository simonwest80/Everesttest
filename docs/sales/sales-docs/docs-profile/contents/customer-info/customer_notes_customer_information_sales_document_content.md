# Customer Notes


You can attach notes to a **Customer**  profile to keep track of information pertaining to the customer.


You can also set up the following preferences:

- **Pop 
 up comments on document** (path: **Invoicing 
 &gt; Customers &gt;** **Options**  > **Edit** > **In 
 New Window** **** > **Miscellaneous**  tab).



![]({{site.sp_baseurl}}/img/lens.gif) [Customer  Profile]({{site.mc_chm}}/creating-a-customer/creating_a_customer.html)

- **Popup 
 and first show notes attached to the respective customer or vendor associated 
 with the document prior to opening it** (path: **File**  > **Setup** > **Preferences**  > **Flow Control** > **Documents** > **Popups**tab).



![]({{site.sp_baseurl}}/img/lens.gif) [Documents  - Flow Control - Popups  Tab]({{site.bp_chm}}/flow-ctrl/ctrl/opt/popups-tab/flow_control_setup_dialog_box_popups_tab_steps.html)
