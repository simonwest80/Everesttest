# Document Alias


The document alias distinguishes a sales document from other sales documents  of the same type.


**![]({{site.sp_baseurl}}/img/example.gif)<font color="#000000" class="hcp3"> </font>**


| Document Name (For Products) | Document Alias (For Services) |
| <font color="#000000" class="hcp3">Sales Quote</font> | <font color="#000000" class="hcp3">Service Quote</font> |
| <font color="#000000" class="hcp3">Sales Order</font> | <font color="#000000" class="hcp3">Service Order</font> |
| <font color="#000000" class="hcp3">Sales Invoice</font> | <font color="#000000" class="hcp3">Service Bills</font> |



You could create  an alias called **Contract Proposal**  to record sales quotations given for service contracts and an alias called  **Repair and Service Orders** for  repair jobs.


![]({{site.sp_baseurl}}/img/lens.gif)<font color="#800080" class="hcp7"> </font>[Document  Aliases]({{site.bp_chm}}/doc-aliases/document_aliases_businesss_process_in_everest_content.html)


When you create a sales document from the sales quote, order or invoice  browser, the default alias specified for each type of document is displayed  in the document.<font color="#000000" class="hcp3"> You can also select a different 
 alias for a document.</font>


![]({{site.sp_baseurl}}/img/note.gif)<font color="#000000" class="hcp3"> You 
 can also create a document with a specific alias from the related document 
 alias browser (path: </font>**Invoicing**<font color="#000000" class="hcp3"> &gt; </font>**Other Documents**<font color="#000000" class="hcp3">). </font>


![]({{site.sp_baseurl}}/img/lens.gif)<font color="#800080" class="hcp7"> </font>[Document  Aliases Defaults]({{site.bp_chm}}/flow-ctrl/defs/alias-defs/document_aliases_defaults.html)


When you convert a sales quote or a sales order with a specified document  alias, the alias specified for the converted document (order or invoice)  is used. If you do not define an alias for the document, the default alias  for the respective document defined in the **Document 
 Alias** dialog box (path: **File &gt; 
 Setup &gt; Preferences &gt; Defaults &gt; Document Alias**) is used.


**![]({{site.sp_baseurl}}/img/example.gif)<font color="#000000" class="hcp3"> </font>When you convert a sales quote with the alias  Service Quote to an order, the document takes the alias Service Order  specified for the sales order.**
