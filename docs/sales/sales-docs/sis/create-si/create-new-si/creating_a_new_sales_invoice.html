Create a Sales Invoice

To create a sales invoice, do the following:

  • From the main menu bar, select Invoicing > Sales Invoices. You will see the Sales Invoice browser.
  • From the Sales Invoice browser, select Options > New > In Current Window/ In New Window. You will see the Sales Invoice profile.

Sales invoices can also be created the following:

  • The Sales Invoice browser by selecting Options > New.
  • An existing Sales Invoice profile by selecting Options > New.
