Convert an Existing Order to an Invoice

An existing sales order can be converted to an invoice. All the details of the order are copied to the new Sales Invoice profile.

You cannot convert a sales order to an invoice for a Point of Sale customer from the Sales

Order profile. Such an order is a held document for a POS_CUST and can be invoiced only from the respective Cash Register profile.

To convert a sales order into a sales invoice, do the following:

  • From the main menu bar, select Invoicing > Sales Orders. You will see the Sales Orders browser.
  • From the Sales Orders browser, select the order that has to be converted to a sales invoice.
  • Select Options > Edit > In New Window. You will see the Sales Order profile.
  • From the Sales Order profile, select Options > Process > Invoice. The information from the sales order is copied onto the sales invoice.
  • Save the sales invoice after checking that the details are correct.
