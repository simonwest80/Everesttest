# Create a Sales Order


{:.steps}
To create a sales order, do the following:

- From the main  menu bar, select **Invoicing &gt; Sales 
 Order**. The **Sales Orders** browser  is displayed.
- From the **Sales Orders** browser, select **Options 
 &gt; New &gt; In Current Window/ In New Window**. You will see the  [**Sales 
 Order** profile]({{site.sp_baseurl}}/sales-docs/sales-orders/create-a-sales-order/create-a-new-sales-order/the_sales_order_profile.html).



{:.note}
Sales orders can also be created from the following:

- The **Sales 
 Order** browser by selecting **Options 
 &gt; New**.
- An existing **Sales Order** profile by selecting **Options &gt; New**.
