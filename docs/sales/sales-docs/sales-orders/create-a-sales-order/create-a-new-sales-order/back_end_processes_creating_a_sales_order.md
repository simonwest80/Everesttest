# Back End Processes - Creating a Sales Order


The following back-end processes occur when a sales order is created:

- The sequence  number of the sales order is generated automatically by the system.



**![]({{site.sp_baseurl}}/img/lens.gif) [Document  Information - Document Number]({{site.sp_baseurl}}/sales-docs/docs-profile/contents/doc-info/document_number_document_information_sales_content.html)**

- The total quantity  'on reserve' for each item increases by the quantity of the item ordered.
- Available stock  reduces.



**![]({{site.sp_baseurl}}/img/lens.gif) [Stock  Status]({{site.sp_baseurl}}/sales-docs/docs-profile/options/items/stock_status_items_option_sales_contents.html)**

- A sales quote  number is assigned to the sales order.



**![]({{site.sp_baseurl}}/img/lens.gif) [Canceling  Sales Orders]({{site.sp_baseurl}}/sales-docs/sales-orders/so-proc/cancel-a-sales-order/canceling_a_sales_order.html)**

- The sales order  is initially assigned the status 'Open order'.



**![]({{site.sp_baseurl}}/img/lens.gif) [Document  Status]({{site.sp_baseurl}}/sales-docs/docs-profile/contents/tab-details/details/other/document_status_document_information_sales_content.html)**
