Create a Sales Order

To create a sales order, do the following:

  • From the main menu bar, select Invoicing > Sales Order. The Sales Orders browser is displayed.
  • From the Sales Orders browser, select Options > New > In Current Window/ In New Window. You will see the Sales Order profile.

Sales orders can also be created from the following:

  • The Sales Order browser by selecting Options > New.
  • An existing Sales Order profile by selecting Options > New.
