Credit Card Processing with Payflow Pro

When the payment method is attached to a processor with PayflowPro as the service selected, you can process the following types of transactions and requests:

  • Address verification.
  • Sale type of transactions where the payment is authorized and funds transferred to your bank account in the next settlement period.
  • Pre-authorization requests for authorization. Authorizations include requests to book funds in delayed settlement transactions such as mail order shipments.
  • Prior authorization requests for Delayed Capture to capture funds where the authorization has been obtained earlier. The authorizations include voice authorizations and mail order shipments for which prior authorization was obtained.
