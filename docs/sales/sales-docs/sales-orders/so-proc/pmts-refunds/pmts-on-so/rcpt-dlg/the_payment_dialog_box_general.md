# The Receipt Dialog Box - General


The **General** tab of the **Receipt** dialog box displays the details  of payment such as the payment method and amount.


The information in this tab is classified under the following heads:

- [Payment  Reference Details]({{site.sp_baseurl}}/misc/payment_reference_details_payment_dialog.html)
- [Payment  Amount Details]({{site.sp_baseurl}}/misc/payment_amount_details_payment_dialog.html)
- [Exchange  Rate Details]({{site.sp_baseurl}}/misc/exchange_rate_details_payment_dialog_box.html)
