# The Generate Purchase Orders Utility


The **Generate Purchase Orders**  utility (path: **Sales Order** profile  > **Options** > **Generate****Purchase** **Orders**)  is available only from sales order documents.


This utility is disabled in the following conditions:

- When the sales  order is not saved.
- When the sales  order is marked 'Closed'. To enable the utility, mark the sales order  open (path: **Sales Order &gt; Process 
 &gt; Mark Close/Open**).
- In sales invoices  and sales quotes.



**Using the**  Generate Purchase Orders **utility you 
 can do the following**:

- [Generate  a Drop Ship PO]({{site.sp_baseurl}}/sales-docs/sales-orders/so-proc/gen-po-util/gen-dropship-po/generate_drop_ship_purchase_orders.html)
- [Generate  PO with links]({{site.sp_baseurl}}/sales-docs/sales-orders/so-proc/gen-po-util/gen-po-links/generate_a_purchase_order_with_links_sales.html)
- [Generate  PO]({{site.sp_baseurl}}/sales-docs/sales-orders/so-proc/gen-po-util/gen-po/generate_purchase_orders_sales.html)
- [Break  Link]({{site.sp_baseurl}}/sales-docs/sales-orders/so-proc/gen-po-util/break-link/break_link_option_sales_content.html)
