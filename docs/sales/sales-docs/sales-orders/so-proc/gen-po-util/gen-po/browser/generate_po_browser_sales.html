Generate PO Browser

The Generate PO browser displays a list of all the items in the sales order. You can generate purchase orders that are NOT linked to sales orders, from this browser.

If the item is marked for drop shipment, the Drop Ship column in the Generate PO browser displays Yes.

Drop Ship Items

To generate a purchase order from a sales order, do the following:

  • From a Sales Order profile, select Options > Generate Purchase Orders > Generate PO. You will see the Generate PO browser.

Generate PO Browser Options

The options available in this browser are listed below:

  • Edit > Line Items
  • Select/Deselect All Items
  • Generate POs

Generate PO - Drop Ship Items Browser Options

Options common to all browsers are also available in this browser.
