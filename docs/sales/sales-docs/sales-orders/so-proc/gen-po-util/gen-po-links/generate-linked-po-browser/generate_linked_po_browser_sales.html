Generate Linked PO Browser

The Generate Linked PO browser displays a list of all the items in the sales order. If an item is marked for drop shipment, the ‘Drop ship’ column displays Yes in the browser.

To generate a purchase order that is linked to a sales order, do the following:

  • From a Sales Order profile, select Options> Generate Purchase Orders > Generate PO with Links. You will see the Generate Linked PO browser.

Generate Linked PO Browser Options

The options available in this browser are listed below:

  • Edit > Line Items
  • Select/Deselect All Items
  • Generate Linked POs

Generate PO - Drop Ship Items Browser Options

Options common to all browsers are also available in this browser.
