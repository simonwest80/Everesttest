# Generate Linked PO Browser


The **Generate Linked PO** browser  displays a list<font style="color: #ff0000;" color="#FF0000"> </font>of all the items in the sales  order. If an item is marked for drop shipment, the 'Drop ship' column  displays **Yes** in the browser.


{:.steps}
To generate a purchase order that is linked  to a sales order, do the following:

- From a **Sales Order** profile, select **Options****&gt; Generate Purchase Orders**  > **Generate PO with Links**. You  will see the **Generate Linked PO**  browser.



Generate Linked PO Browser Options


The options available in this browser are listed below:

- Edit > Line  Items
- Select/Deselect  All Items
- Generate Linked  POs
