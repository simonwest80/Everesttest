The Generate Purchase Orders Utility

The Generate Purchase Orders utility (path: Sales Order profile > Options > GeneratePurchase Orders) is available only from sales order documents.

This utility is disabled in the following conditions:

  • When the sales order is not saved.
  • When the sales order is marked ‘Closed’. To enable the utility, mark the sales order open (path: Sales Order > Process > Mark Close/Open).
  • In sales invoices and sales quotes.

Using the Generate Purchase Orders utility you can do the following:

  • Generate a Drop Ship PO
  • Generate PO with links
  • Generate PO
  • Break Link
