Generate PO - Drop Ship Items Browser

The Generate PO - Drop Ship Items browser displays a list of all the items in the sales order. By default, all the drop-ship items for which no drop-ship purchase order is created are selected in this browser. All the items are selected by default. You can deselect the items for which you do not want to create a purchase order.

To access the Generate PO - Drop Ship Items browser, do the following:

  • From a Sales Order profile, select Options> Generate Purchase Order > Generate Drop Ship PO. You will see the Generate PO - Drop Ship Items browser.
