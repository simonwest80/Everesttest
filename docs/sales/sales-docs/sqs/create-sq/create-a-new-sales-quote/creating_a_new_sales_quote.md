# Create a New Sales Quote


{:.steps}
To create a sales quote from the sales quote  browser, do the following:

- From the main  menu bar, select **Invoicing &gt; Sales 
 Quotes**. You will see the **Sales 
 Quotes** browser.
- From the **Sales Quotes** browser, select **Options 
 &gt; New &gt; In Current Window/In New Window**. You will see the  [**Sales 
 Quote** profile]({{site.sp_baseurl}}/sales-docs/sqs/create-sq/create-a-new-sales-quote/the_sales_quote_profile.html).



{:.note}
Sales quotes can also be created:


From the **Sales Quotes**  browser by selecting **Options &gt; New**  > **In New Window/In Current Window**.


From an existing **Sales 
 Quote** profile by selecting **Options 
 &gt; New**.
