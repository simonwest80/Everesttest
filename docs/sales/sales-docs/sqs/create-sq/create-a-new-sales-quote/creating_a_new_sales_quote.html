Create a New Sales Quote

To create a sales quote from the sales quote browser, do the following:

  • From the main menu bar, select Invoicing > Sales Quotes. You will see the Sales Quotes browser.
  • From the Sales Quotes browser, select Options > New > In Current Window/In New Window. You will see the Sales Quote profile.

Sales quotes can also be created:

From the Sales Quotes browser by selecting Options > New > In New Window/In Current Window.

From an existing Sales Quote profile by selecting Options > New.
