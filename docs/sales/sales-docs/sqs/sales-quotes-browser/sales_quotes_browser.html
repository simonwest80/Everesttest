Sales Quotes Browser

The **Sales Quotes browser (path: Invoicing > Sales Quotes) lists all the sales quotes that are created but not yet converted to sales orders. It displays documents that have been created in the default period specified in the Flow Control Preferences (path: File > Setup > Preferences > Flow Control > Documents > Prompts > Sales Document > Sales Document 1).**

Filter Sales Documents based on Dates

Prompts - Sales Documents Tab

You can change the period by selecting the required option in the Dates field.

Options common to all sales document browsers are available in this browser.

Sales Document Browser Options
