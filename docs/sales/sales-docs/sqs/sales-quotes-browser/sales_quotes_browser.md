# Sales Quotes Browser


**The **Sales 
 Quotes** browser (path: **Invoicing 
 &gt; Sales Quotes**) lists all the sales quotes that are created  but not yet converted to sales orders. It displays documents that have  been created in the default period specified in the **Flow 
 Control Preferences** (path: **File 
 &gt; Setup &gt; Preferences &gt; Flow Control &gt; Documents &gt; Prompts 
 &gt; Sales Document &gt; Sales Document 1**).**


![]({{site.sp_baseurl}}/img/lens.gif) [Filter  Sales Documents based on Dates]({{site.sp_baseurl}}/sales-docs/browser/filtering_sales_documents_browsers_based_on_dates.html)


[Prompts  - Sales Documents Tab]({{site.bp_chm}}/misc/select_the_default_date_option_based_on_which_the_records_will_be_displayed_in_the_sales_browser.html)


You can change the period by selecting the required option in the **Dates** field.


**Options common to all sales document browsers  are available in this browser.**


**![]({{site.sp_baseurl}}/img/lens.gif) [Sales  Document Browser Options]({{site.sp_baseurl}}/sales-docs/browser/cmn-opts/sales_document_browser_options_contents.html)**


This browser has [**Purge**]({{site.sp_baseurl}}/sales-docs/sqs/sales-quotes-browser/purging-a-sales-quote/purging_a_sales_quote.html)  (path: **Sales Quotes** browser >  **Options**) as an additional option.


**Options common to all browsers are also  available in this browser.**


**![]({{site.sp_baseurl}}/img/lens.gif) [Standard  Browser Options]({{site.wwe_chm}}/everest-client/ui/browsers/standard_browser_options.html)**
