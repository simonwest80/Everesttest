# Pricing Analysis


**Everest**'s elaborate pricing  analysis feature (path: **Sales****Invoice** profile > **Options**  > **Pricing** **Analysis**)  provides the sales department with instant and accurate profit margin  calculations for a line item or for the entire document. You can view  the sales price of an item at progressively increasing rates of margin  or mark-up.


**![]({{site.sp_baseurl}}/img/example.gif) You  can see what the selling price must be for every 2% increase in margin  rates.**


**The pricing analysis  feature helps to compare or match a quote with a competitor.**


![]({{site.sp_baseurl}}/img/example.gif) If  a competitor quotes a certain price, you can use this feature to identify  the lowest price that you can bid without making a loss on the sale.


The default parameters for pricing analysis are defined in the **Pricing Analysis** dialog box (path: **File &gt; Setup &gt; Preferences &gt; Defaults 
 &gt; Pricing Analysis**).


![]({{site.sp_baseurl}}/img/lens.gif) [Pricing Analysis  Dialog Box]({{site.sp_baseurl}}/sales-docs/sqs/sq-proc/pricing-analysis/pricing-analysis-defaults/pricing_analysis_dialog_box.html)


You can select the price at a required margin and apply it to the document  to change selling prices instantaneously.


**View Pricing Analysis on a Sales Document**


{:.steps}
To view the pricing analysis for a document,  do the following:

- From the document  profile, select **Pricing Analysis**  (path: Sales Document profile > **Options 
 &gt; Pricing Analysis**). You will see the [**Pricing Analysis** browser]({{site.sp_baseurl}}/sales-docs/sqs/sq-proc/pricing-analysis/the_pricing_analysis_browser.html).
- From the **Pricing Analysis** browser, select a row  and click **Apply**; **Everest**  will re-calculate all selling prices using the mark-up/margin specified  for all the items in the document.



Selling prices selected must fulfill the minimum/critical markup/margins  specified.
