# Pricing Analysis Defaults


The critical and minimum mark up/margin is specified in the **Pricing 
 Analysis** dialog box. This feature allows you to ensure that the  selling price of the item does not fall below this level.


{:.steps}
To view the pricing analysis dialog box,  do the following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Preferences &gt; Defaults &gt; Pricing Analysis**. You will see the  [**Pricing 
 Analysis** dialog box]({{site.sp_baseurl}}/sales-docs/sqs/sq-proc/pricing-analysis/pricing-analysis-defaults/pricing_analysis_dialog_box.html).
