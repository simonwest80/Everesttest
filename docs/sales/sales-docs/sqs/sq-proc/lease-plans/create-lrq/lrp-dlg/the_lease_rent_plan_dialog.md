# The Lease Rent Plan Dialog Box


When **Options &gt; Lease** is selected in a sales document, the **Lease Rent Plan Dialog** box is displayed.


This dialog box has the following tabs:

- [Lease Plan]({{site.sp_baseurl}}/sales-docs/sqs/sq-proc/lease-plans/create-lrq/lrp-dlg/the_lease_rent_plan_dialog_lease_plan.html)
- [Rental Plan]({{site.sp_baseurl}}/sales-docs/sqs/sq-proc/lease-plans/create-lrq/lrp-dlg/the_lease_rent_plan_dialog_rental_plan.html)
