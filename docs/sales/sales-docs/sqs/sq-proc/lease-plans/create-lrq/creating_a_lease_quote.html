Create a Lease Quote

A sales quote to which a lease plan is attached is called a ‘Lease Quote’. To distinguish a lease quote from an ordinary quote, create a document alias for the document, called ‘Lease Quote’.

Document Alias

To create a lease quote, do the following:

  • From the main menu bar, select Invoicing > Sales Quotes. You will see the Sales Quotes browser.
  • Create a new quote or open an existing quote for editing.
  • Select Options > Lease from the Sales Quote profile. You will see the Lease Rent Plan Dialog box.
  • Specify a Lease ID ** in the **Lease Plan field of the Lease Plan tab.
  • Click OK to save the quote as a Lease Quote.
