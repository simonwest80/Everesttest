Create a Rental Quote

A sales quote to which a rental plan is attached is called a ‘Rental Quote’. To distinguish a rental quote from an ordinary quote, create a document alias for the document, called ‘Rental Quote’.

Document Alias

To create a rental quote, do the following:

  • From the main menu bar, select Invoicing > Sales Quotes. You will see the Sales Quotes browser.
  • Create a new quote or open an existing quote for editing.
  • Select Options > Lease from the Sales Quote profile. You will see the Lease Rent Plan Dialog box.
  • Specify a Rental ID in the Rental Plan tab.
  • Click OK to save the quote as a ‘Rental Quote’.
