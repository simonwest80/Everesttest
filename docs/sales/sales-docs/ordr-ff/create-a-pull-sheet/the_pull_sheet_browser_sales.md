# The Pull Sheet Browser


The **Pull Sheet** browser (path:  **Inventory** > **Pull 
 Sheet** browser) displays all the pull sheets. By default only open  pull sheets will appear. Â You  can create new pull sheets based on the template you defined in the **Template** browser . You can access the  Pick Tickets browser and view the pick tickets of an open pull sheet by  selecting **Options** > **Pick 
 Tickets** from this browser. Alternatively, you can open multiple  pick tickets of different pull sheets from the Pick Tickets browser (path:  **Inventory** > **Pick 
 Tickets**). This browser also displays the user code, date of creation,  pull sheet status, and the template used to create the pull sheet. You  can view each pull sheet's details by clicking the respective rows.

{:.note}
If no template has been used to create a pull  sheet the **Template** column will  display 'Custom' against that pull sheet.
