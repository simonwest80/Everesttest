Serialized Items

Serial numbers of the item to be picked may be retrieved in any of the following ways:

  • One by one
  • In a range

The serial numbers can also be scanned into the system using your bar code scanner or scan / enter the serial number using the Commit Inventory option. You have to select the serial # from the bin that has been selected on Stock Bin Selection page of the wizard.

You cannot pick serial numbers for items that do not exist on the Pull Sheet.
