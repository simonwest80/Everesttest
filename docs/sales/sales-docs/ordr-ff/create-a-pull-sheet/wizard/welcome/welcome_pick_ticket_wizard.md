# Pull Sheet Wizard - Welcome


When you access the **Pull Sheet Wizard**,  the **Welcome** section is displayed.  This section provides you with information on how the wizard is arranged.


This page also allows you to create pull sheets using either the wizard  or existing template.


To create a pull sheet using the wizard, click **Next**  and traverse the wizard.


{:.steps}
To create a pull sheet using the existing  template, do the following:

- Select **Use Template** optional button.
- Select the  template from **Pull Sheet Template**  drop down field.



{:.note}
If there is no template the **Select 
 Pull Sheet Template** drop down will be disabled.


You can view the template details in a **Template**  browser by either pressing **F2**  while the cursor is on **Pull Sheet Template**  drop down field or selecting **Browser**  from right click menu.

- Click **Finish** button. The **Pick 
 Ticket** browser with the template's settings will appear.



{:.note}
If you click Next button after selecting Use  Template optional button you will still be able to traverse the wizard.


To modify an existing template's settings, select the template from  **Pull Sheet Template** drop down  field and traverse the wizard.


Click **Next**. You will see the  **Pick 
 Ticket Option** section of the **Pull 
 Sheet Wizard**.
