The Pull Sheet Wizard

The Pull Sheet Wizard guides you through the process of picking items from various bins. This wizard is especially useful when you want to pick items in an orderly manner from a large warehouse in the minimum time. The wizard comprises the following sections:

  • Welcome
  • Document Criteria - Pick Ticket Option
  • Priority
  • Line Items
  • Payment Terms
  • Stock Bin Selection
  • Selected Stock Bins
  • Quantity Allocation
  • Sales Order Selection
  • Pick Ticket Browser
  • Pull Sheet
  • Conclusion
