Other Details

The Others tab provides you with search parameters that use information from other fields in sales orders.

You can use the document alias or terms or jurisdiction to find the sales orders.

This tab comprises the following fields:

  • Document alias group
  • Document alias
  • Terms
  • Jurisdiction
  • Loc/Sub-Loc
  • Ship via
  • Currency
  • Account ship to
  • PO number
  • Reference number
  • Shipment tracking
  • FOB
  • Notes
  • Lease plan
  • Rent plan
  • Policy
  • Processed
  • Include Manual Documents
