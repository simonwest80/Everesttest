Common Amount Field Options

The options common to all the fields in the Amount tab of the Find Sales Orders dialog box are enumerated below:

Option Search Result
All All documents (ranging from the smallest to the largest amount).
Between Documents between the specified amount range.
Equals All documents where the selected amount equals the specified amount.
Not equal to Documents where the amount does not equal the specified amount.
At most Documents whose amount equals or is lesser than the specified amount.
At least Documents whose amount equals or is more than the specified amount.
More than Documents whose amount is more than the specified amount.
Less than Documents whose amount is lesser than the specified amount.
Is Empty Documents where the amount field is empty.
Is not Empty Documents where the amount field is NOT empty.
Ask This field allows you to make an amount field mandatory. If the filter parameters are saved, the saved filter cannot be run the next time, unless a value is specified for the field that has the Ask option selected.


You create a find filter for sales orders and select the Ask option in the Balance field. The next time, you will not be able to use the saved filter, unless you specify a value in the Balance field.
