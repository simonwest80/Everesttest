Quick Find Details

The Quick Find tab lists the most common search parameters used to find sales orders. The Quick Find tab consists of the following fields:

  • SO#
  • Order Status
  • Sales Representative
  • Picked Status
  • Account Name
  • SO Date
  • Number of Records to Retrieve

You can use these parameters individually or in combination with other parameters to narrow down your search. The options selected in this tab also appear in the Advanced Find tab.
