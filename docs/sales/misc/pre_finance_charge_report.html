Pre-finance Charge Report

The Pre-finance Charge report allows you to verify the finance charges that will be debited to the customers. This report can be viewed by selecting the Pre-finance Charge Report option from the Finance Charges Setup browser.

To generate the pre-finance charge report, do the following:

  • From the main menu bar, select Accounting > Accounts Receivable > Finance Charges. You will see the Finance Charges profile.
  • Enter the details and click OK. You will see the Finance Setup Charges browser.
  • From the Finance Setup Charges browser, select Options > Pre-Finance Charge Report. You will see the Print Pre-Finance Charge Report dialog box.
  • Select Print to print the report or Preview to view the report on screen.
