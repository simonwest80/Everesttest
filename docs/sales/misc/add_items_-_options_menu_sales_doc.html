Add Items - Options Menu

To add an item to the sales document, do the following:

  • From the Sales Document profile, select Options > Item > Select Items. You will see the Items browser. This browser displays only items of type ‘Kit’.
  • From the Items browser, double-click the required item. The item is added to the sales document.
The same item can be selected more than once in a document. This is useful in cases where customers are offered items at lower rates for additional quantities purchased.

You may select Item # 10578 in Line 1 and again in Line 5.

  • The item selected in the Item Details Grid can be a special type of item such as an Item Kit or an Matrix Item.
  • You can also create a new item in the Items browser that opens. Create an item, and refresh the browser to select the new item.

Add a Discontinued Item

You can add a discontinued item into the sales document.

To add a discontinued item to a sales document, do the following:

  • From the Sales Document profile, select Options > Item > Select Items. You will see the Items browser.
  • From the Items browser, select View > Discontinued. The discontinued items are also displayed in the Items browser.
  • From the Items browser, highlight the discontinued item and click OK.

An ‘X’ on the top left corner of the icon indicates that the item is discontinued.

Everest displays a message stating that the item is a discontinued one. You will be prompted to confirm whether you want to use the discontinued item. If you select Yes, the discontinued item is added to the sales document.

Add Multiple Items

If you add multiple items to a sales document of which one is a discontinued item, you will receive a message stating that discontinued items exist in the selection. You can choose to:

  • Retrieve active items - Select this option to add only active items to the sales document.
  • Retrieve active and discontinued items - Select this option to add both active and discontinued items to the sales document.
