---
title: SO Status
---


The status of the sales order - whether 'open', 'closed' or on 'backorder'  is displayed in this section. The sales quote and order number and creation  date is also displayed. The sales quote number is automatically assigned  to the sales quote based on the sequence number specified for sales quotes  in the logged in location or the location of the logged in department.  If no sequence numbers has been assigned to the location the sequence  numbers specified for the default location is used.
