Choose the currency the sales document has to appear in, while sending the document by e-mail to the customer. The options available are:

  • Print in Base Currency - Select this option if you want the document that you send by e-mail, to be in the base currency, irrespective of the currency defined for the customer. The Currency field defaults to this option.
  • Print in Document Currency - Select this option if you want the document that you send by e-mail, to be in the currency defined for the customer selected in the document.
  • Print in both Base and Document Currency - Select this option if you want the document that you send by e-mail, to be in the currency defined for the customer selected in the document. However, the document and the tax totals are displayed additionally in the base currency at the bottom right hand side of the document.
