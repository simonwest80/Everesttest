Shipment Destination Details

This field displays the address details of the default shipping address of the customer. This address is specified in the ShippingAddress section of the Customer profile (path: Invoicing > Customers > Options > Edit > In New Window > Addresses tab). You cannot modify this field.

For sales return documents, this field defaults to the document’s location/sub-location address and can be edited.
