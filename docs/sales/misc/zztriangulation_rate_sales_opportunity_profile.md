---
title: Triangulation Rate
---


This field is enabled if your base currency is a Euro  currency and the customer's currency is also a Euro  currency. This rate is automatically displayed from the **Accounting** dialog box (path: File >  Setup > Preferences > Defaults > Accounting > Multi Currency  > Accounting tab) and is editable.
