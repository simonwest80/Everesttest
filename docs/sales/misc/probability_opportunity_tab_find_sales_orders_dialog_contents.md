# Probability


The **Probability** field allows  you to search for sales orders based on the probability defined in the  sales opportunity associated with the order.


The options available for the **Probability** field are:


| Option | Search result will display a list of: |
| All | All documents (with all probabilities). |
| Between | Documents whose probability falls between the specified  probability range. |
| Equals | Documents where the probability equals the specified  percentage. |
| Not equal to | Documents whose probability does not equal the specified  percentage. |
| At most | Documents whose probability equals or is lesser than  the specified percentage. |
| At least | Documents whose probability equals or is more than the  specified percentage. |
| More than | Documents whose probability is more than the specified  percentage. |
| Less than | Documents whose probability is lesser than the specified  percentage. |
| Is empty | Documents where the probability field is empty. |
| Is not empty | Documents where the probability field is NOT empty. |
| Ask | This option allows you to make the probability parameter a mandatory  field.<br/><br/><br/>If you select the **Ask** option  in the **Probability** field and save  the filter, the next time you use the saved filter, it becomes mandatory  to specify a value on the **Probability**  field. You cannot use the filter until a value is specified in the **Probability** field. |
