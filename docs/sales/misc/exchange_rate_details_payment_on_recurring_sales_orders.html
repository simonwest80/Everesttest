Exchange Rate Details

This section allows you to specify the exchange rates for foreign currency receipts. Exchange rate details have to be defined for the following:

  • Document Currency
  • Bank Currency
  • Payment Currency
  • Base Currency

Everest determines whether the Buy Rate/Sell Rate should be used depending on the setting in the Rate for Receipts field (path: File > Setup > Preferences > Defaults > Accounting > Multicurrency > Exchange Rates).
