Custom Fields

The Custom Fields tab allows you to define additional information for any record as per your requirement. Custom fields can be defined as character strings, or date, logical, memo or numeric fields.

This tab is displayed only when you save the credit order document.

If a credit order has an expiry date that is set in the “Date” custom field, you can search for the credit order using the Expiry Date field in the Custom Field tab.
