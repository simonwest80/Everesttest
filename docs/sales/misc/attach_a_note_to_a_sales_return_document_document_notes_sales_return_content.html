Attach a Note to a Sales Return Document

To attach a note to the sales return document, do the following:

  • From the sales return document profile, click on the Note icon displayed beside the Document # field. You will see the Document Notes window.
  • Enter any information about the document.
  • Select Options > Spell Check or press the F7 key to spell check your notes.
  • Click on the Save button and exit from the window.

OR

  • From the sales return document profile, select Options > Document > Document Notes.
  • Enter any information about the document.
  • Click on the Save button and exit from the window.
