title: Cost
---


The cost of the item is displayed in this field. This field displays the average cost or the last cost, depending on the preferences set in the **Default value for the user defined cost column in a document** option of the **Other** **Flow Control** dialog box (path: File > Setup > Preferences > Flow Control > Other Flow Control > Item Cost/Sales Summary tab).
