---
title: Document Number
---


The number assigned to the document is displayed in this field. The  document number is automatically assigned based on the sequence number  specified for sales quotes in the logged in location or the location of  the logged in department. If no sequence numbers has been assigned to  the location the sequence numbers specified for the default location is  used. The document number is displayed for a new sales quote when the  document details are saved.
