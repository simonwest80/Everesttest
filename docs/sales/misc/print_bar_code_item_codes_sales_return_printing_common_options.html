Print Bar Code - Item Codes

The Bar Code - Item Codes allows you to print item codes in the document as bar codes. When this option is selected the BarCodes browser is displayed. This browser displays all the item codes that are attached to the document.

This option is NOT available for sales quotes and credit quotes.

By default the Quantity and the Committed columns display the item quantities as entered in the sales/sales return documents.

To print an item code as a bar code, do the following:

  • From the Sales/SalesReturn document, select Options > Print > Bar Code - Item Codes. You will see the Bar Codes browser.
  • From the Bar Codes browser, select the item codes that have to be printed as bar codes.
  • Click the button > LabelPrint > Item Code to print the item codes as bar codes.

You can also create a new label using the LabelDesigner option.
