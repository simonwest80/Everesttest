Third Party Catalogs Browser

The Third-Party Catalogs browser displays all the third-party catalogs created in Everest.

Third-Party Catalogs Browser Options

View Items

Use this option to view the Items browser that lists all items in the third-party catalog. You can import an item into the Everest database by selecting Options > Import Item.
