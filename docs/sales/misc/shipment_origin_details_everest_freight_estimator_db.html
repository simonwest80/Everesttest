Shipment Origin Details

This field displays the address from which the item is shipped. This field defaults to the address details of the document’s location/sub-location.

Click the button to modify details on the origin of the shipment. You will see the Shipment Origin dialog box.

For sales return documents, the Shipment Origin field defaults to the customer’s default shipping address or the billing address (if there is no default shipping address).
