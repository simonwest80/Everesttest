# Processed


This option allows you to search for sales orders based on the **Processed** check box.


![]({{site.sp_baseurl}}/img/lens.gif) [Processed  Check Box]({{site.sp_baseurl}}/sales-docs/docs-profile/contents/tab-details/details/other/processed_sales_document_content.html)


The following options are available:

- **All**  - All sales orders are searched, irrespective of whether the document  is marked as processed or not.
- **Yes**  - Only sales orders marked as processed are searched.
- **No**  - Sales orders that are not marked as processed are searched.
