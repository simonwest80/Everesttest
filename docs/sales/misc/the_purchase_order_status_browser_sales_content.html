The Purchase Order Status Browser

The Purchase Order Status browser displays the following:

  • Purchase Order number
  • Vendor
  • Item
  • Order Quantity
  • Cost

Highlight the required document and select Open Document from the right-click menu to view the Purchase Order profile.

When you close the Purchase Order Status browser, the sales order is saved. The PO # column in the sales order Item Details Grid is automatically updated with the purchase order number. You can double-click the PO # column to view the purchase order that is created.
