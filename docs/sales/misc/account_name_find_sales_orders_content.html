Account Name

This field allows you to search for sales orders raised on a particular customer. You can also select multiple customers in this field.

To select the names of customers, do the following:

  • Click the Browse icon beside the field. The Customers browser is displayed.
  • From the Customers browser, select the required customers and click OK. The Selection dialog box is displayed which contains a list of the selected customers.
  • You can deselect the included individual or multiple customers from the list by clearing the check box beside the customer Account Name column.

If this parameter is not used in combination with other parameters, all sales orders belonging to the selected customer(s) are displayed.
