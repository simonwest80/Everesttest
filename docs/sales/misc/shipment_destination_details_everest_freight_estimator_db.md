# Shipment Destination Details


This field displays the address details of the default shipping address  of the customer. This address is specified in the **Shipping****A****ddress**  section of the **Customer** profile  (path: **Invoicing &gt; Customers &gt; 
 Options &gt; Edit &gt; In New Window &gt; Addresses** tab). You cannot  modify this field.


{:.note}
For sales return documents, this field defaults  to the document's location/sub-location address and can be edited.
