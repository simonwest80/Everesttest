---
title: View
---


Displays **Specific Location** by  default if **Logged in Location**  option has been selected in **Allow users 
 to select stock bin of** (path: **File**  > **Setup** > **Preferences**  > **Flow Control** > **Other 
 Flow Control**) drop down field. Otherwise it displays **All 
 Locations** if **All Locations**  option has been selected in **Allow users 
 to select stock bin of** drop down field.
