title: Rate Shop
---


Click the ![]({{site.sp_baseurl}}/img/sales_rate_shop_button.gif) button if you want to change the delivery  method in this field. You will see the **Rate 
 Shop Browser** that displays a list of all the delivery methods and  the corresponding rates that have been set up. To change a delivery method,  select a record from the **Rate Shop****Browser** and click **OK**.
