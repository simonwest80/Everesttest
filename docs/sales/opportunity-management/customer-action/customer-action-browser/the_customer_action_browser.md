# The Customer Action Browser


The **Customer Action** browser  (path: **CRM 
 &gt; Opportunity Management &gt; Setup &gt; Customer Action**) displays  all the customer action groups, and the corresponding customer action  selections that have been defined.


Options common to all browsers are available in this browser.


![]({{site.sp_baseurl}}/img/lens.gif) [Standard  Browser Options]({{site.wwe_chm}}/everest-client/ui/browsers/standard_browser_options.html)


Use the **Up** ![]({{site.sp_baseurl}}/img/sales_up_arrow.gif) and  **Down** ![]({{site.sp_baseurl}}/img/sales_down_arrow_button.gif) arrows to  change the rank of the customer action groups and choices.
