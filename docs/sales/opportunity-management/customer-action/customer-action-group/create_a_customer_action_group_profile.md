# Create a Customer Action Group Profile


{:.steps}
To create a customer action group, do the  following:

- From the main  menu bar, select **CRM 
 &gt; Opportunity Management &gt; Setup** **&gt; Customer Action**. You will see the **Customer****Action** browser.
- From the **Customer Action** browser, select **New &gt; Customer Action Group**. You  will see the [**Customer 
 Action Group** profile]({{site.sp_baseurl}}/opportunity-management/customer-action/customer-action-group/the_customer_action_group_profile.html).



{:.note}
You can also create a customer action group from an  existing **Customer Action Group**  profile by clicking **Options &gt; New**.
