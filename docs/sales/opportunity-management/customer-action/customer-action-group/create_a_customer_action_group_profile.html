Create a Customer Action Group Profile

To create a customer action group, do the following:

  • From the main menu bar, select CRM > Opportunity Management > Setup > Customer Action. You will see the CustomerAction browser.
  • From the Customer Action browser, select New > Customer Action Group. You will see the Customer Action Group profile.

You can also create a customer action group from an existing Customer Action Group profile by clicking Options > New.
