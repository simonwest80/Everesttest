The Sales Stage Profile

Use this profile (path: CRM > Opportunity Management > Setup > Sales Stage > Options > Edit > In New Window) to set up the information required to create a sales stage.

Code
The code uniquely identifies the sales stage throughout Everest and is automatically generated when the profile is saved.
Description
Enter a suitable title for the sales stage using from 1 to 40 characters of any type.
Probability
Enter the probability (as a percentage) of an opportunity being converted to a sale at this stage.
When an opportunity is assigned to this stage, the Probability field in the Sales Opportunity profile displays this value, by default. This can be changed as required for individual opportunities.
Include in Sales Funnel
Select this check box to include this sales stage in the sales funnel.
Default
Select this check box to display this sales stage by default in the Stage field in a Sales Opportunity profile, when a new sales opportunity is created. You can select only one stage as the default sales stage.
