# The Opportunity Source Profile Option


Click the ![]({{site.sp_baseurl}}/img/sales_custom_fields_icon.gif) icon in the toolbar to define and keep track  of additional information pertaining to the opportunity source. Custom  fields can be defined as character strings, date fields, logical fields,  numeric fields or memo fields.
