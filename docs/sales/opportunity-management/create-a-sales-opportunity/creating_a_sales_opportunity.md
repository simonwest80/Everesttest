# Create a Sales Opportunity


{:.steps}
To create a sales opportunity, do the following:

- From the main  menu bar, select **Invoicing &gt; Sales 
 Opportunities**. You will see the **Sales 
 Opportunities** browser.
- From the **Sales Opportunities** browser, select  **Options &gt; New**. You will see  the [**Sales 
 Opportunity** profile]({{site.sp_baseurl}}/opportunity-management/create-a-sales-opportunity/the_sales_opportunity_profile.html).



{:.note}
Sales opportunities **** can also be created:

- From an existing  **Sales Opportunity** profile by selecting  **Options &gt; New**.
- If you create a  sales quote directly without first creating a sales opportunity, the **Sales** **Opportunity**  profile is populated with details from the sales quote. You can view it  Â from the  **Sales Quote** profile by selecting  **Options &gt; Edit Opportunity**.
