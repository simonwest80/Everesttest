# Create a Reason Closed Group Profile


{:.steps}
To create a reason closed group, do the following:

- From the main  menu bar, select **CRM 
 &gt; Opportunity Management &gt; Setup** **&gt; Reason Closed**. You will see the **Reason 
 Closed Setup** browser.
- From the **Reason Closed Setup** browser, select **New &gt; Reason Group**. You will see  the **Reason 
 Closed Group** profile.



{:.note}
You can also create a reason closed group from an existing  **Reason Closed Group** profile by  selecting **Options &gt; New**.
