Create a Reason Closed Choice Profile

To create a reason closed choice, do the following:

  • From the main menu bar, select CRM > Opportunity Management > Setup > Reason Closed. You will see the ReasonClosed Setup browser.
  • From the Reason ClosedSetup browser, select New > Reason Selection. You will see the Reason ClosedChoice profile.

You can also create a reason closed selection from an existing Reason Closed Choice profile by selecting Options > New.
