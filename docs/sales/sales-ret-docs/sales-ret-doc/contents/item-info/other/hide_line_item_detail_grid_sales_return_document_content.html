Hide Line

The Hide column (check box) allows you to hide lines in the Item Details Grid.

You cannot hide a line that has a price or value in it.

To hide a line, do the following:

  • From the sales return document profile, highlight the item.
  • Scroll to the Hide column.
  • Check the box.
  • Click on the Hide button displayed in the sales return document status bar. The selected line is hidden and will not be displayed in the Item Details Grid.
  • Click on the Show button displayed in the sales return document status bar to view the hidden line.
