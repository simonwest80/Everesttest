# Line Number


**The **Line****Number** column displays the line  number for the item. All lines in the Item Details Grid are printed in  order of their line numbers.**


{:.note}
You can move the line up or down. Use the **Move 
 up**/**Move down** option (path:  **Sales Return Document** profile  > **Options** > **Line 
 Items** > **Move up**/**Move down**) to rearrange the order of  the items in the Item Details Grid.
