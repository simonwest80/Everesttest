# Third Party Table


The **Third Party Table** column  displays the name of the third party catalog that has been defined in  the **Catalog Code** field of the  **Third Party** profile (path: **Inventory** > **Third****Party** **Catalogs**  > **Options** > **New**  > **In New Window**). The name  of the catalog is displayed in this column only if a third party item  is returned from your **Everest eStorefront**  store.
