Third Party Code

The Third Party Code column displays the code of the third party catalog that has been defined in the Catalog code field of the Third Party profile (path: Inventory > ThirdParty Catalogs > Options > New > In New Window).

The code is displayed only for items from third party catalogs that are returned from your Everest eStorefront store.

This column will not have any value for sales return documents that have not been raised from your Everest eStorefront store.
