Item Information

Items in a sales return document are listed in the Item Details Grid. The Item Details Grid displays the items attached to a document in a tabular format. Each row in the Item Details Grid displays information about the item specified in that row.

A credit quote or a credit order can be created without specifying item details. The item details can be entered later if required, by editing the document.

A credit memo must have at least one item selected.

Item information for the items (in the Item Details Grid) are listed below:

  • Item Details
  • Discounts
  • Tax Details
  • Account Details
  • Other Information
