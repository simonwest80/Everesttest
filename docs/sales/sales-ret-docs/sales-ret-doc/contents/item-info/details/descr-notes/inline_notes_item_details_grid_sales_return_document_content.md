# Inline Notes


**Inline** **Notes**  can be added as a line in the Item Details Grid. The inline note can also  be entered from the Description column in the Item Details Grid. The notes  entered using the Inline option will not wrap to any other line.


{:.steps}
To add inline notes, do the following:

- From the sales  return document profile, select **Options**  > **Inline Notes**. You will see  the **Description** dialog box.
- Enter the required  note and select **File** > **Save**/**Save**  and **Close** to return to the document.
