# Category


Item categories are groups under which related items are classified.  If you have specified the category to which an item belongs in the **Category** field of the [**Item** profile]({{site.mi_chm}}/create-regular-items-kits-and-assemblies/creating-an-item/the_item_profile_-_general.html) (path: **Inventory 
 &gt; Categories &gt; Options &gt; Edit &gt; In New Window**), the  category code is displayed in the **Category**  column. You can create categories in the **** [**Category** profile]({{site.mi_chm}}/item-profile-details/item-categories/the_category_profile.html) (path: **Inventory &gt; 
 Categories &gt; Options &gt; Edit &gt; In New Window**). You cannot  edit this column.
