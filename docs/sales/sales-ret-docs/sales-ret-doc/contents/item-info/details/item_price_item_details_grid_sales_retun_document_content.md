# Item Price


The **Item** **Price**  column displays the price at which items returned are accepted from the  customer.


![]({{site.sp_baseurl}}/img/lens.gif) [Item  Price]({{site.mi_chm}}/item-profile-details/item-pricing/item_pricing.html)


If a quote is converted to an order or an order is converted to a memo,  the price specified on the quote or the ordered price is displayed. The  price can be modified on a quote or an order at any time.


{:.note}
The item price on a credit memo cannot be modified once  it the credit memo has been saved.


If the document is a foreign currency document, the price in the document  currency mode is expressed in terms of the foreign currency.
