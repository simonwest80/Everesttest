Add an Item Kit to a Sales Return Document

To add an item kit to the sales return document, do the following:

  • From the sales return document profile, select Options > Item > Select Kits. You will see the Items browser.
  • From the Items browser, highlight the required kit and double-click. You will see the Retrieve Kits dialog box.
  • Enter the required quantity and click OK.
  • The item is added to the sales document.
