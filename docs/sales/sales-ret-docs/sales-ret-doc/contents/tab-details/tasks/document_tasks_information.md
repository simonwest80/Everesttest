# <font style="color: #000000;" color="#000000">Document Tasks Information</font>


The **Document Tasks** tab allows  you to:

- View existing  tasks, incoming and outgoing calls, and general notes that have been associated  to a sales return document.
- Associate new  tasks, incoming and outgoing calls, and general notes to a sales return  document.



{:.note}
This tab is displayed only when you save the sales return  document.


![]({{site.sp_baseurl}}/img/lens.gif) [Contact  Manager]({{site.cm_chm}}/contact_manager.html)


{:.steps}
To associate a new task to a sales return  document, do the following:

- From the browser  in the **Document Tasks** tab of the  Sales Return Document, right-click and select **New 
 &gt; Task**. You will see the **Task** profile.
- Enter and save  the required details.



{:.note}
Similarly,  you can associate an incoming or outgoing call and general notes to a  sales return document.
