# Zoom In/Out


Click the **Zoom In** ![]({{site.sp_baseurl}}/img/sales_zoom_in_icon_status _bar.gif) button to see all the details for a document.


Click on the **Zoom Out** ![]({{site.sp_baseurl}}/img/sales_zoom_out_icon_status _bar.gif) button to view only the Customer Information, Item Detail  Grid and Document Totals.


{:.note}
You can also access this option from the **Options** menu of the document profile (path: Sales Return Document **** profile > **Options** > **Zoom In**/**Out**).
