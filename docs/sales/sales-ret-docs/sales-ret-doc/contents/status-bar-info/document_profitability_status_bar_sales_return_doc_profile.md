# Document Profitability


This option displays the profitability of the line item. Click this  ![]({{site.sp_baseurl}}/img/sales_document_profitability_icon_status_bar.gif) icon and you will see the [**Pricing Analysis** dialog box]({{site.sp_baseurl}}/sales-docs/sqs/sq-proc/pricing-analysis/pricing-analysis-defaults/pricing_analysis_dialog_box.html) for  the entire sales return document. The profitability of the sales return  document is encrypted in the icon in the format 'Dyynnnnnn-ppppp-mmmm-yy'  where 'yy'  is a random number, 'nnnnnn'  is the cost, 'pppp'  is the profit amount, and mmm  is the margin/markup. Profit and cost is rounded off to nearest whole  number. The first alphabet, the first two numbers and the last two numbers  do not represent any information and are random.


![]({{site.sp_baseurl}}/img/example.gif) The  encoded document profitability icon follows the Cost-Profit-Mark up %/Margin  format. In this ![]({{site.sp_baseurl}}/img/sales_document_profitability_icon_status_bar.gif) icon, the total cost of the items in  the document is 2470. The profit (selling price - cost) of the item is  990 and the mark up % is 40.


Based on the preference set in the **Analysis 
 By** field of the [**Pricing Analysis** dialog box,]({{site.sp_baseurl}}/sales-docs/sqs/sq-proc/pricing-analysis/pricing-analysis-defaults/pricing_analysis_dialog_box.html) the  document profitability status indicator displays the pricing analysis  either by margin or markup (path: **File 
 &gt; Setup &gt; Preferences &gt; Defaults &gt; Pricing Analysis**).


{:.note}
You  will see the Document Profitability indicator in the status bar of a sales  return document only if you have set a flow control preference by selecting  the **Show document line item profitability 
 in encoded fashion** check box in the [**Documents** dialog box]({{site.bp_chm}}/flow-ctrl/ctrl/doc-frm/misc/flow_control_setup_dialog_box_miscellaneous_tab_con.html) (path: **File &gt; Setup &gt; Preferences &gt; Flow 
 Control &gt; Documents &gt; Miscellaneous &gt; Others &gt; Others 1**  tab).
