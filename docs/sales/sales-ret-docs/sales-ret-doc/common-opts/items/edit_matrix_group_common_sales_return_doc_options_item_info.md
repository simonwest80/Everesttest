# Edit Matrix Group


{:.steps}
To modify an item matrix, do the following:

- From the sales  return document highlight the matrix item.
- Select **Options &gt; Item &gt; Edit Matrix Grou**p.  You will see the **Matrix**<font color="#000000" class="hcp2"> </font>**Items**<font color="#000000" class="hcp2"> </font>**View** <font color="#000000" class="hcp2">dialog box.</font>
