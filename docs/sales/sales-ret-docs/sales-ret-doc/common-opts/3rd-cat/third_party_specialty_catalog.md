# Third Party Catalog


Third party databases such as vendor catalogs can be imported and browsed  for quick reference.


![]({{site.sp_baseurl}}/img/lens.gif) Â [Import  Third Party Catalogs]({{site.utl_chm}}/item-management/importing-3rd-catalogs/import_third_party_catalogs_into_everest.html)


The **Third Party Catalog** option  in sales return documents (path: Sales Return Document profile > **Options** > **Third 
 Party Catalog**) allows you to verify if the item returned by the  customer belongs to a third party catalog. You can also import the returned  item into the database. Third party catalog information that can be accessed  from within a document are listed below:

- [All  Catalogs]({{site.sp_baseurl}}/sales-ret-docs/sales-ret-doc/common-opts/3rd-cat/third_party_speciality_catalog_all_catalogs.html)
- [Default  Catalog]({{site.sp_baseurl}}/sales-ret-docs/sales-ret-doc/common-opts/3rd-cat/third_party_catalog_default_catalog_sales_return_docs.html)
- [Import  Item]({{site.sp_baseurl}}/sales-ret-docs/sales-ret-doc/common-opts/3rd-cat/third_party_specialty_catalog_item_import_common_sales_return_document_options.html)
