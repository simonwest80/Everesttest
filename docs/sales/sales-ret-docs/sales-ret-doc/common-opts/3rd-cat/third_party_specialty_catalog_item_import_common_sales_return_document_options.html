Third Party Catalog - Import Item

Using the Import Item (path: Sales Return Document profile > Options > Third Party Catalog > Import Item) option, you can import an item present in a third party catalog into the Everest database.

In sales return documents, if a customer returns an item purchased from a third party catalog, you can import the item into the database and record the return. You can then create a debit document and return the item to the third party vendor.

To import an item present on a third party catalog into the Everest database, do the following:

  • From the Sales Return Document profile, select an item that exists in a third party catalog and choose Options > Third Party Catalog > Import Item. You will see the Item profile.
  • Enter the required details and select Options > Save. The item is imported into the Everest database.

Once the item is imported into the database, you can create a debit memo document and return the item to the vendor.
