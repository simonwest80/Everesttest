# Browser Column Filters


You can filter for documents that match specified criteria for each  of the columns listed in the Customer Aging Information browser.


{:.steps}
To filter for documents, do the following:

- Click on the  **Down** ![]({{site.sp_baseurl}}/img/sales_down_arrow1.gif) arrow on  the column title. You will see a list with the options **All**,  **Custom** and a list of all data  for that column.



![]({{site.sp_baseurl}}/img/example.gif) If  you click on the column **Document #**,  you will see a list of the Document Numbers of all the documents in the  browser.

- Click on **Custom** to set filters for the selected  column.
