# Profile


The **Customer** profile option  (path: **Sales Document** profile  > **Options** > **Customer**  > **Profile**) allows you to access  a customer profile from any of the sales return documents. You can view  the profile of the customer selected on the document by default. You can  make any modifications required to the **Customer**  profile.


![]({{site.sp_baseurl}}/img/example.gif) You can view the **Customer**  profile to check the customer's balance account.
