# The Sales Return Function


The sales return cycle involves decisions and processes related to returns  of defective products sold to customers. The customer must be compensated  for unsatisfactory service.


The sales return function allows you to  execute the following functions:

- Creating a  credit quote for items that the customer intends to return.
- Creating a  credit order.
- Creating credit  memos for items that are returned.
- Managing item  returns for exchange and repair.
- Follow-up on  open credit orders on which items are yet to be returned.
- Refunds.
- Managing accounts  receivable.
