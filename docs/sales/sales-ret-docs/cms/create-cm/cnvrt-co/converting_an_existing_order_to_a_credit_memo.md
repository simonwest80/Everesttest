# Convert an Existing Order to a Credit Memo


A credit order may be converted to a Credit  Memo by selecting the **Options**  > **Process &gt; Memo** option  in the **Credit Order** profile. When  you select this option, all details of the order are copied to a new **Credit Memo** profile.


{:.steps}
To convert a credit order into a credit memo,  do the following:

- From the main  menu bar, select **Invoicing &gt; Credit 
 Order**. You will see the **Credit 
 Orders** browser.
- From the **Credit Orders** browser, double - click  the order that has to be converted to a credit memo. You will see the  **Credit Order** profile.
- From the **Credit Order** profile, select **Options**  > **Process &gt; Memo**. The information  from the credit order is copied onto the credit memo.
- Save the credit  memo after checking the details.
