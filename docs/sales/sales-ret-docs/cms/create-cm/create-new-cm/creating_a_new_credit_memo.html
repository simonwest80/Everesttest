Create a Credit Memo

To create a credit memo, do the following:

  • From the main menu bar, select Invoicing > Credit Memos. The Credit Memos browser is displayed.
  • From the Credit Memos browser, select Options > New > In New Window. You will see the Credit Memo profile.

Credit memo profiles can also be created:

From an existing Credit Memo profile by selecting Options > New.
