# Modify a Credit Memo


The **Credit 
 Memo** profile can be modified from any one of the following locations:

- The **Credit 
 Memos** browser - Highlight the record and select **Options 
 &gt; Edit &gt; In New Window**.
- The **Credit 
 Memo** profile - Specify the document number of the required invoice  in the **Document #** field.



The information that can be modified is different for credit memos that  have and have not been posted.


**Credit Memos that have not been posted**


Accounting entries for credit memos are recorded in system **Sales 
 Journals**. If the sales journal is not Â posted,  the following fields can be modified in the credit memo:

- Item Details  Grid (Add a non-inventoried item but not an inventoried item)
- Delivery method
- Delivery date
- Promotion Code
- FOB and reference
- Customer's  PO number and date
- Shipment tracking  number
- Location/Sub-Location
- Selling price/cost/discounts/accounts/tax  code/due date for all items
- Sales representative
- Processed



**Posted Credit Memos**


If the sales journal in which the credit memo is recorded is posted,  the following fields can be modified in the credit memo:

- Sales Rep
- F.O.B
- Reference
- Customer's  purchase order number and date
- Shipment tracking  number
- Processed
- Document Policy



{:.hint}
To differentiate the credit memos that are posted from  those that are not posted, add the field **Is 
 Posted** to the **Credit Memos** browser  from the **Field Chooser**.
