# Refund Dialog Box - General


The **General** tab of the **Refund** dialog box allows you to enter  the details of the refund such as the payment method and amount.


The information in this tab is classified under the following:

- [Payment  Reference Details]({{site.sp_baseurl}}/misc/payment_reference_details_cm.html)
- [Payment  Amount Details]({{site.sp_baseurl}}/misc/payment_amount_details_refund_cm.html)
- [Exchange  Rate Details]({{site.sp_baseurl}}/misc/exchange_rate_information_refund_dialog_box.html)
