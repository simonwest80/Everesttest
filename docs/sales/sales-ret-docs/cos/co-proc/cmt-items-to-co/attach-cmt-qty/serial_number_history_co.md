# Serial Number History


When a serial number is attached to a credit order or credit memo, an  entry is made in the history of the serial number which can be viewed  from the **Serial Number History** browser.  The serial number history provides details of the stock areas and the  documents to which a serial number is assigned.


The **Serial Number History** browser  can be viewed by selecting any of the following options:

- From the [**Credit 
 Order/Credit Memo** profile]({{site.sp_baseurl}}/misc/serial_number_history_browser.html) by selecting **Options 
 &gt; Serial Number History**.
- From the [**Credit 
 Orders/Credit Memo** browser]({{site.sp_baseurl}}/misc/serial_number_history_browser.html) by selecting **Options 
 &gt; Serial Number History**.
