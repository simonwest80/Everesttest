Create a Credit Order

To create a credit order, do the following:

  • From the main menu bar, select Invoicing > Credit Order. The Credit Orders browser is displayed.
  • From the Credit Orders browser, select Options > New > In New Window. You will see the Credit Order profile.

You can create Credit Order profiles from any of the following:

  • The Credit Order browser by selecting Options > New.
  • An existing Credit Order profile by selecting Options > New.
