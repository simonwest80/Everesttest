# Create a Credit Order


{:.steps}
To create a credit order, do the following:

- From the main  menu bar, select **Invoicing &gt; Credit 
 Order**. The **Credit Orders** browser  is displayed.
- From the **Credit Orders** browser, select **Options &gt; New &gt; In New Window**.  You will see the [**Credit 
 Order** profile]({{site.sp_baseurl}}/sales-ret-docs/cos/create-co/create-a-new-credit-order/the_credit_order_profile.html).



{:.note}
You can create Credit Order profiles from any of the  following:

- The **Credit 
 Order** browser by selecting **Options 
 &gt; New**.
- An existing **Credit Order** profile by selecting **Options &gt; New**.
