# Other Shortcut Keys


| Press Key(s) | To |
| Ctrl+F4 | Close the **Sales****Opportunity** profile. |
| Ctrl+N | Create a new sales opportunity from an opportunity that  has been saved. |
| Ctrl+F | Display the **Custom <br/> Fields** dialog box for the selected sales opportunity. |
| Ctrl+S | Save the sales opportunity. |
| Shift+Ctrl+Z | Cancel the changes made to the sales opportunity. |
