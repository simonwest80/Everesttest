# Go to a Field


| Press Key(s) | To |
| Alt+A | Go to the **Date**  field. |
| Alt+B | Go to the **F.O.B**  field. |
| Alt+C | Go to the **Contact Address**. |
| Alt+D | Go to the **Due Date**  field on an invoice that has not been saved. |
| Alt+E | Go to the **Customer**  field in an invoice that has NOT been saved. |
| AltF+F | Go to the **Reference**  field. |
| Alt+G | Go to the **Exchange****Rate** field on an invoice that  has not been saved. |
| Alt+J | Go to the **Jurisdiction**  field on an invoice that has not been saved. |
| Alt+K | Go to the **Shipment****Tracking** field. |
| Alt+L | Go to the **Loc/Sub-Loc** field. |
| Alt+N | Go to the **Customer's <br/> PO No.** field. |
| Alt+R | Go to the **Sales Rep** field. |
| Alt+S | Go to the **Ship to address**. |
| Alt+Shift+# | Go to the **Document <br/> Number** field. |
| Alt+T | Go to the **Terms**  field. |
| Alt+U | Go to the **Currency**  field on an invoice that has not been saved. |
| Alt+V | Go to the **Ship Via**. |
| Alt+Y | Go to the **Delivery <br/> Date** field on an invoice that has not been saved. |
