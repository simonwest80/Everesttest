# Other Shortcut Keys


| Press Key(s) | Description | Function |
| Alt+G | Get the credit orders. | Gets the credit orders into the browser. |
| Ctrl+F4 | Close the **Credit Orders** browser. | Closes the **Credit Orders**  browser. |
| Ctrl+Ins | Create a new credit order in the current window. | Creates a new credit order in the current window. |
| Enter | Open and modify the selected credit order. | Opens and modifies the selected credit order. |
| F3 | Print the selected credit order. | Prints the selected credit order. |
| F4 | Toggle between shipped credit orders and open credit  orders. | By default the **Credit <br/> Orders** browser displays only orders that have not been shipped.  To view shipped credit orders, press F4. To view open credit orders press  F4 again. |
| F5 | Refresh the **Credit <br/> Orders** browser. | Updates the **Credit <br/> Orders** browser with all current changes made. |
| Ins | Create a new credit order. | Creates a new credit order by opening a new window without  closing the existing open window. |
| Shift+Ctrl+C | Collapse grouping. | Hides the records under each group and display only  the main groups. |
| Shift+Ctrl+E | Expand grouping. | Displays all the records under each group. |
| Shift+F4 | Export the 'ship to address'. | Opens a **Save As** dialog  box where you can define whether the 'ship to address'  of the selected order has to be exported into excel sheet or a .txt  file. |
