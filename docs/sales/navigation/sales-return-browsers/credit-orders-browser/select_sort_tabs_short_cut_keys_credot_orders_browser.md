# Select Sort Tabs


| Press Key(s) | Description | Function |
| Alt+A | Select the **Date**  sort tab. | Sorts the records in the browser by credit order creation  date. |
| Alt+C | Select the **CO #**  (credit order number) sort tab. | Sorts the records in the browser by the credit order  number. |
| Alt+F | Select the **Reference <br/> #** sort tab. | Sorts the records in the browser by the document reference  number. |
| Alt+L | Select the **Loc/Sub-Loc** sort tab. | Sorts the records in the browser by the documents' locations/sub-locations. |
| Alt+N | Select the **Account****Name** sort tab. | Sorts the records in the browser by the customers' account  names. |
| Alt+P | Select the **PO #** (purchase  order number) sort tab. | Sorts the records in the browser by the customers' purchase  order numbers. |
| Alt+Q | Select the **CQ <br/> #** (credit quote number) sort tab. | Sorts the records in the browser by the credit quote  numbers. |
| Alt+R | Select the **Rep**  (sales representative) sort tab. | Sorts the records in the browser by the sales representatives  assigned to the orders. |
| Alt+U | Select the **Account**  sort tab. | Sorts the records in the browser by the customer account  numbers. |
