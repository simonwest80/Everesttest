Other Shortcut Keys

Press Key(s) Description Function
Alt+G Get the credit memos. Gets the credit memos into the browser.
Ctrl+F4 Close the Credit Memos browser. Closes the Credit Memos browser.
Ctrl+Ins Create a new credit memo in the current window. Creates a new credit memo in the current window.
Enter Open and modify the selected credit memo. Allows you to view and modify the selected credit memo.
F3 Print the selected credit memo. Prints the selected credit memo.
F5 Refresh the Credit
Memos browser.		 Updates the Credit
Memos browser with all current changes made.
Ins Create a new credit memo. Creates a new credit memo by opening a new window without closing the existing open window.
Shift+Ctrl+C Collapse grouping. Hides the records under each group and display only the main groups.
Shift+Ctrl+E Expand grouping. Displays all the records under each group.
Shift+F4 Export the ‘ship to address’. Opens a Save As dialog box where you can define whether the ‘ship to address’ of the selected credit memo has to be exported into an excel sheet or a .txt file.
