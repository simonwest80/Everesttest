# Other Shortcut Keys


| Press Key(s) | Description | Function |
| Alt+G | Get the items. | Gets the credit quote documents into the browser. |
| Ctrl+F4 | Close the **Credit Quotes** browser. | Closes the **Credit Quotes** browser. |
| Ctrl+Ins | Create a new credit quote **** in the current window. | Creates a new credit quote in the current window. |
| Enter | Open and modify the selected record. | Opens and allows you to modify the selected credit quote. |
| F3 | Print the selected credit quote. | Prints the selected credit quote. |
| F5 | Refresh the **Credit <br/> Quotes** browser. | Updates the **Credit <br/> Quotes** browser with all current changes made. |
| Ins | Create a new credit quote. | Creates a new credit quote by opening a new window without  closing the existing open window. |
| Shift+Ctrl+C | Collapse grouping. | Hides the records under each group and displays only  the main groups. |
| Shift+Ctrl+E | Expand grouping. | Displays all the records under each group. |
| Shift+F4 | Export the 'ship to address'  of the selected quote. | Opens a **Save As** dialog  box where you can define whether the 'ship to address'  of the selected quote has to be exported into excel sheet or a .txt  file. |
| Shift+F6 | Purge the selected credit quote from the **Credit <br/> Quotes** browser. | Purges the selected credit quote(s). |
