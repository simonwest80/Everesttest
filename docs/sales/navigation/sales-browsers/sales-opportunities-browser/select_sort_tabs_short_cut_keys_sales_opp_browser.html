Select Sort Tabs

Press Key(s) Description Function
Alt+A Select the Account sort tab. Sorts the records in the browser by the customer’s account numbers.
Alt+P Select the Opportunity
ID sort tab.		 Sorts the records in the browser by the opportunity ID.
Alt+S Select the SalesRep sort tab. Sorts the records in the browser by document’s sales representative.
