# Other Shortcut Keys


| Press Key(s) | Description | Function |
| Alt+G | Get the sales opportunities. | Gets the sales opportunities into the browser. |
| Ctrl+Alt+E | Display the **Export**  dialog box. | Displays the **Export**  dialog box. |
| Ctrl+F4 | Close the **Sales Opportunities** browser. | Closes the **Sales Opportunities** browser. |
| Ctrl+Ins | Create a new sales **o**pportunity in the current window. | Creates a new sales opportunity in the current window. |
| Ctrl+P | Display the **Browser <br/> Property Editor** dialog box. | Displays the **Browser <br/> Property Editor** dialog box. |
| Enter | Open and modifies the selected sales opportunity. | Allows you to view and modify the selected sales opportunity. |
| F5 | Refresh the **Sales Opportunities**<font color="#FF0000" class="hcp6"> </font>browser. | Updates the **Sales****Opportunities**<font color="#FF0000" class="hcp6"> <br/> </font>browser with all changes made. |
| Ins | Create a new sales opportunity. | Creates a new sales opportunity by opening a new window  without closing the existing open window. |
| Shift+Ctrl+C | Collapse grouping. | Hides the records under each group and display only  the main groups. |
| Shift+Ctrl+E | Expand grouping. | Displays all the records under each group. |
