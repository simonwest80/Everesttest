# Display Browsers/Dialog Boxes


| Press Key(s) | To |
| Ctrl+Alt+E | Display the **Export <br/> Dialog** box. |
| Ctrl+F | Display the **Custom <br/> Fields** dialog box for the selected sales invoice. |
| Ctrl+F11 | Display the **Shipment <br/> Tracking Information** dialog box. |
| Ctrl+P | Display the **Browser <br/> Property Editor** dialog box. |
| Ctrl+T | Display the **Transaction****History** browser for the selected  sales invoice. |
| Ctrl+U | Display the **Journal <br/> Transactions** dialog box for the selected sales invoice. |
| F8 | Display the **Payment <br/> History** dialog box for selected sales invoice. |
| Shift+F2 | Display the **Multimedia****Setup** browser for the selected  sales invoice. |
| Shift+F5 | Display the **Custom <br/> Tracking** browser for the selected sales invoice. |
| Ctrl+E | Display the **Document <br/> E-mail Preferences** dialog box. |
| Shift+Ctrl+K | Display the **Task**  profile. |
| Shift+Ctrl+I | Display the **Incoming <br/> Call** profile. |
| Shift+Ctrl+O | Display the **Outgoing <br/> Call** profile. |
| Shift+Ctrl+N | Display the **General <br/> Notes** profile. |
| Shift+Ctrl+H | Display the **Task History**  browser. |
| Shift+Ctrl+M | Display the **My Tasks**  browser. |
| Shift+Ctrl+A | Display the **Appointment**  profile. |
| Shift+Ctrl+L | Display the **Appointment <br/> History** profile. |
| Shift+Ctrl+C | Display the **Calendar <br/> View** profile. |
