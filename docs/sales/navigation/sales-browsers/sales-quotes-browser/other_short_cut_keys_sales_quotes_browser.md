# Other Shortcut Keys


| Press Key(s) | **To** | **Function** |
| Alt+G | Get all records. | Gets the sales quotes into the browser. |
| Ctrl+F4 | Close the **Sales Quotes** browser. | Closes the **Sales Quotes** browser. |
| Ctrl+Ins | Create a new sales quote in the current window. | Creates a new sales quote in the current window. |
| Enter | Open and modify the selected record. | Opens and allows you to modify the selected record. |
| F3 | Print the selected sales quote. | Prints the selected sales quote. |
| F5 | Refresh the **Sales Quotes** browser. | Updates the **Sales Quotes** browser with all current changes that have been made. |
| Ins | Create a new sales quote. | Creates a new sales quote by opening a new window without  closing the existing open window. |
| Shift+Ctrl+C | Collapse grouping. | Hides the records under each group and display only  the main groups. |
| Shift+Ctrl+E | Expand grouping. | Displays all the records under each group. |
| Shift+F4 | Export the **Ship To**  address of the selected quote. | Opens a **Save As** dialog  box where you can define whether the 'ship to address'  of the selected quote has to be exported into excel sheet or a .txt  file. |
| Shift+F6 | Purge the selected record from the **Sales <br/> Quotes** browser. | Purges the selected record from the **Sales <br/> Quotes** browser. |
