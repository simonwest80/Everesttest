# Display Browsers/Dialog Boxes


| Press Key(s) | To |
| ` | Display the **Retrieve****Serial** **Numbers**  dialog box. |
| Alt+F8 | Display the **Description**  dialog box for the selected line item. |
| Ctrl+A | Display the **Customer****Aging** **Information**  browser. |
| Ctrl+Alt+C | Display the **Custom <br/> Fields** dialog box from a saved credit order.<br/><br/><br/>Display the **Find Customers** dialog box on a credit order that has not been saved. |
| Ctrl+Alt+F8 | Display the **Detach****serial** **#**  dialog box. |
| Ctrl+Alt+L | Display the **Item Stock <br/> Status** dialog box for the selected line item. |
| Ctrl+Alt+S | Display the **Quantity****stock** **area****wise** browser for the selected  item. This option is enabled in saved credit orders only for non-serialized  items. |
| Ctrl+Alt+V | Display the **Serial <br/> # for all items** browser. |
| Ctrl+Alt+W | Display the **Serial <br/> #** browser for the selected serialized item. |
| Ctrl+D | View the **Pricing Analysis** dialog box for all items in the order. |
| Ctrl+G | Display the **Custom****Tracking** browser for the customer. |
| Ctrl+H | Display the **Attach****Quantity** dialog box only when  inventoried, non - serialized items are attached or negative quantities  are attached to the credit order. |
| Ctrl+I | Display the **Items**  browser for item selection. |
| Ctrl+K | Display the **Retrieve****Kits** dialog box. |
| Ctrl+L | Display the **Item Stock <br/> Status** dialog box for all items on the document. |
| Ctrl+M | Display the **Matrix <br/> Items View** dialog box if a matrix item is selected. |
| Ctrl+O | Display the **Item**  profile for the selected line item. |
| Ctrl+Q | Display the **Retrieve****Serial** **Numbers**  (range) dialog box. |
| Ctrl+Right  Arrow | Display the **Sales Document <br/> Lines** dialog box for the selected line item. |
| Ctrl+T | Display the **Transaction <br/> History** browser from a saved credit order. |
| Ctrl+U | Display the **Customer**  profile. |
| Ctrl+W | Display the **Multimedia <br/> Setup** browser from a saved credit order. |
| Ctrl+Y | Display the **Item Sales History** browser  for the selected line item. |
| F10 | Displays the **Stock Status Indicator**  for the selected line item in a sales document. This is only enabled if  the “Automatically display stock status indicators” preference in the  **Items** dialog box – **Item****Details** tab (path: **File**  > **Setup** > **Preferences**  > **Flow Control** > **Items**)  is NOT selected (disabled). We recommend that you disable this option  to improve performance when adding line items to sales documents. In this  situation, the Stock Status Indicators do not automatically display for  the line items. |
| F11 | Displays a list of **Add-on** items  associated with the selected line item. |
| F12 | Displays a list of **Related Items**  associated with the selected line item. |
| Shift+` | Display the **Retrieve Mfg <br/> Serial Numbers** dialog box. |
| Shift+F5 | Display the **Custom Tracking** browser  for the document from a saved credit order. |
| Shift+F6 | Display the **Document Notes** dialog  box. |
| Shift+F7 | Display the **Customers** browser  for customer selection. |
| Shift+F8 | Display the **Items** (Kits) browser  for kit selection. |
| Shift+F9 | View and edit the **Sales Opportunity** profile on a saved credit order. |
| Ctrl+E | Display the **Document <br/> E-mail Preferences** dialog box. |
| Shift+Ctrl+K | Display the **Task**  profile. |
| Shift+Ctrl+I | Display the **Incoming <br/> Call** profile. |
| Shift+Ctrl+O | Display the **Outgoing <br/> Call** profile. |
| Shift+Ctrl+N | Display the **General <br/> Notes** profile. |
| Shift+Ctrl+H | Display the **Task History**  browser. |
| Shift+Ctrl+M | Display the **My Tasks**  browser. |
| Shift+Ctrl+A | Display the **Appointment**  profile. |
| Shift+Ctrl+L | Display the **Appointment <br/> History** profile. |
| Shift+Ctrl+C | Display the **Calendar <br/> View** profile. |
| Ctrl+F12 | Display the **Find Line <br/> Items** browser. |
| Ctrl+Alt+T | Display the items from the default **Third <br/> Party Catalog**. |
