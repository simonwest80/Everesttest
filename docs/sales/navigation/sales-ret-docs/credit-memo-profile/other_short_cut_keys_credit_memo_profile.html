Other Shortcut Keys

Press Key(s) To
Ctrl+Alt+G Generate serial numbers for line items that do not have a negative quantity.
Ctrl+Alt+H Import items from third party catalogs.
Ctrl+Alt+Space Preview on/off of the items.
Ctrl+Down arrow Move the selected line item one line down in the Item Details Grid.
Ctrl+F4 Close the credit memo.
Ctrl+N Create a new credit memo from a saved document.
Ctrl+P Print the saved credit memo.
Ctrl+S Save the credit memo.
Ctrl+Space Zoom in/out of the item grid.
Ctrl+UP arrow Move the selected line item one line up in the Item Details Grid.
F6 Toggle between document currency and the base document.
Shift+Ctrl+Z Cancel the changes made to the credit memo.
Shift+F11 Automatically attach serial numbers to the selected item on a credit memo that has NOT been saved.
Shift+F12 Automatically attach serial numbers to all auto-serialized items on the memo.
Shift+F4 Apply the discount applicable to the customer.
Shift+F10 Displays the Stock
Status Indicators for all line items in a sales document. This is only enabled if the “Automatically display stock status indicators” preference in the Items dialog box – Item Details tab (path: File > Setup > Preferences > Flow
Control > Items) is NOT selected (disabled). We recommend that you disable this option to improve performance when adding line items to sales documents. In this situation, the Stock Status Indicators do not automatically display for the line items.
