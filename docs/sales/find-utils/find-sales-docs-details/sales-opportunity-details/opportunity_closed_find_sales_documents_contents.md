# Opportunity Closed


This option allows you to search for invoicing documents based on the  closed status of the sales opportunity associated with the document.


![]({{site.sp_baseurl}}/img/lens.gif) [Sales  Opportunity]({{site.sp_baseurl}}/opportunity-management/sales_opportunities_an_introduction.html)


The options available are:


| Option | Search Result Displays: |
| All | All documents whether the opportunity is closed or not |
| Yes | Only documents whose associated opportunity is closed |
| No | Only documents whose associated opportunity is open |
