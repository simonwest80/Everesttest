# Notes


This field allows you to search for documents using the **Document 
 Notes** as the parameter value. You can also enter a partial value  in this field.


![]({{site.sp_baseurl}}/img/example.gif) If  you input the value 'This Sales Invoice..' in this field, the search yields  all documents whose document note begins with the specified text.


Click the drop-down button to view the history of prior search parameter  values.
