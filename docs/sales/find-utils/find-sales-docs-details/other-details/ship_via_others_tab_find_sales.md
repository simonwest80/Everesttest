# Ship via


This field allows you to search for invoicing documents using the shipping  agent specified in the documents.


![]({{site.sp_baseurl}}/img/example.gif) If  you want to search for invoices of items shipped by UPS Ground, enter  'UPSG' in this field and 'Invoices' in **Document 
 Type** field of **Quick Find**  tab.
