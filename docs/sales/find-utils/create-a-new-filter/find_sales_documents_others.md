**Document alias**
: Specify the document alias on which to base your  search.

**Terms**
: Specify the payment terms on which to base your  search.
: ![]({{site.sp_baseurl}}/img/lens.gif) [Terms]({{site.sp_baseurl}}/find-utils/find-sales-docs-details/other-details/terms_others_tab_find_sales.html)


**Jurisdiction**
: Specify the jurisdiction on which to base your search.


**Loc/Sub-Loc**
: Specify the location/sub-location on which to base  your search.


**Ship via**
: Specify the shipping agent to base your search on.
: ![]({{site.sp_baseurl}}/img/lens.gif) [Ship  Via]({{site.sp_baseurl}}/find-utils/find-sales-docs-details/other-details/ship_via_others_tab_find_sales.html)


**Currency**
: Specify the document currency on which to base your  search.
: ![]({{site.sp_baseurl}}/img/lens.gif) [Currency]({{site.sp_baseurl}}/find-utils/find-sales-docs-details/other-details/currency_others_tab_find_sales.html)


**Account ship to**
: Click the browse button to view the **Addresses**  browser, which displays the shipping addresses of all customers. Select  the addresses from the browser.


**PO Number**
: Specify the purchase order number to base your search  on. Click the drop-down button to view a history of prior search parameter  values.


**Reference Number**
: Specify the reference number to base your search  on. Click the drop-down button to view a history of prior search parameter  values.


**Shipment tracking**
: Specify the shipment tracking number, to base your  search on. Normally, the shipment tracking reference number is entered  in sales invoices, as items are shipped only at this stage. Click the  drop-down button to view a history of prior search parameter values.


**FOB**
: Specify the FOB (Free on Board) carrier to base  your search on. Click the drop-down button to view a history of previous  search parameter values.


**Notes**
: Enter the text of the document note based on which  your search has to be performed.
: ![]({{site.sp_baseurl}}/img/lens.gif) [Notes]({{site.sp_baseurl}}/find-utils/find-sales-docs-details/other-details/notes_others_tab_find_sales.html)


**Policy**
: Enter the text of the document policy to base your  search on. You can also enter partial values. Click the drop-down button  to view a history of prior search parameter values.
: ![]({{site.sp_baseurl}}/img/lens.gif) [Document  Policy]({{site.sp_baseurl}}/sales-docs/docs-profile/contents/doc-info/document_policy_document_information_sales_content.html)


**Lease plan**
: Specify the lease plan assigned to the document  on which to base your search.


{:.note}
You can assign lease plans only to sales quotes and  sales orders.


**Rent plan**
: Specify the rent plan assigned to the document on  which to base your search.


{:.note}
You can assign rent plans only to sales quotes and sales  orders.


**Processed**
: Specify whether documents marked as processed must  be included in the search.
: ![]({{site.sp_baseurl}}/img/lens.gif) [Processed]({{site.sp_baseurl}}/find-utils/find-sales-docs-details/other-details/processed_others_tab_find_sales_documents_content.html)


**Include Manual Documents**
: Specify whether manual documents must be included  in the search results.
: ![]({{site.sp_baseurl}}/img/lens.gif) [Include  Manual Documents]({{site.sp_baseurl}}/find-utils/find-sales-docs-details/other-details/include_manual_documents_find_sales_documents_contents.html)
