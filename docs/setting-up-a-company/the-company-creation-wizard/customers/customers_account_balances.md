# Customers - Account Balances


Once customers are created or imported into the database, the amounts  due from them can either be imported into the database or specified manually  for each customer.


**Click here to import open documents**
: Click this option to import the customer account  balances. Ensure that the file containing the account balance details  meets the requirements specified. The **Source File Selection** section of the **Data 
 Import Wizard** is displayed.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Data  Import Wizard]({{site.utl_chm}}/db-utils/data-import/wizard/importing_data_data_import_wizard_utility_content.html)


{:.note}
<font color="#000000" class="hcp2">You can also access the import option by selecting 
 <span class="hcp1">Utility &gt; Import &gt; Data</span>.</font>


**Click here to enter document details of each  customer**
: Use this option to enter the customer balances manually  for each customer. You will see the [**Customer Balances** dialog box]({{site.sc_baseurl}}/misc/the_customer_balances_dialog_box.html).

- Click **Next**.  You will see the [Vendors - Vendor  Details]({{site.sc_baseurl}}/the-company-creation-wizard/vendors/vendors_vendor_details.html) section.
