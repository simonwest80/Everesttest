Customers - Account Balances

Once customers are created or imported into the database, the amounts due from them can either be imported into the database or specified manually for each customer.

Click here to import open documents
Click this option to import the customer account balances. Ensure that the file containing the account balance details meets the requirements specified. The Source File Selection section of the Data Import Wizard is displayed.
Data Import Wizard
You can also access the import option by selecting Utility > Import > Data.
Click here to enter document details of each customer
Use this option to enter the customer balances manually for each customer. You will see the Customer Balances dialog box.
