Flow Control

These options define how the flow of data and control must be handled in Everest.

You can configure **Everest so that the Vendors browser pops up whenever you create a Purchase Order.**

You can access these preferences from File > Setup > Preferences > Flow Control. Flow control preferences can be set for the following:

  • Documents
  • Document Tracking
  • Items
  • Other Flow Control
