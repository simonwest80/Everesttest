# Defaults


These options are used to define the default settings throughout **Everest**. Setting up default settings  helps in the following:

- To minimize  data entry.



{:.example}
**If the default accounts that are affected  by the sale or purchase of an item are specified, they need not be entered  when an item is created.**

- To identify  the control accounts for Accounts Receivable and Payable. Therefore, you  need not have individual accounts for each customer or vendor. You can  also identify the account in which the net income for the year should  be accounted, when the fiscal year is closed.



These preferences can be accessed from **File 
 &gt; Setup &gt; Preferences &gt; Defaults**. Default preferences  can be set for the following:

- [Documents]({{site.bp_chm}}/flow-ctrl/defs/rpt/documents_dialog_box.html)
- [Customers]({{site.mc_chm}}/misc/customers_dialog_box.html)

- [Items]({{site.mi_chm}}/item-preferences/item-defaults/item_defaults.html)
- [Printing  Preferences]({{site.bp_chm}}/rpt-prt/print_preferences_details_bp_contents.html)
- [Pricing  Analysis]({{site.sp_chm}}/sales-docs/sqs/sq-proc/pricing-analysis/pricing_analysis.html)
