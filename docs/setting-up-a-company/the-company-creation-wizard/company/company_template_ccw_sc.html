Company - Template

Use the Company – Template screen to select the existing company database file to use as a default database when you create a new company. Everest enables you to use the default values of Vendors, COA, Customers, Items, Document aliases etc from an existing company in the new company.

Set up company information using the wizard
Select this radio button if you want the Company Creation Wizard to guide you through the company setup process.
Copy setup information from an existing company
Select this radio button to specify the template database file to be used as a default database file for a new company and activate the Company Database File selection box.
Company Database File
Select the database file to be loaded to the new company by clicking the button.

: This field will be activated only if you selected the Copy Setup information from an existing company radio button.
