# Company Information - Name


Use this section to specify the code used to identify the company in  the database, the name of the company and the trade name, if any.


**Company Code**
: Enter a unique code to identify the company being  defined. You can use 1 to 8 characters of the supported type. Once the  code has been defined, you cannot change it. This is a mandatory field.


**Legal Name**
: Enter the name with which the company has been incorporated.  This name will appear on all statutory forms and returns, and on all reports  and other forms generated from **Everest**.  Use 1 to 50 characters to define the legal name. This is a mandatory field.
: If this name is used for an existing company, enter  a different name in this field.


{:.note}
: If at a later date, you change the **Legal 
 Name** of the company and click **Done**,  you will have to obtain a new **Registration****Code** from **Everest**  Software.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Everest  Registration]({{site.sc_baseurl}}/everest-registration/everest_registration.html)


**Trade Name**
: Enter the trade name of your company using 1 to  50 characters. This is a mandatory field.
: Once you specify the details, click **Create** button. **Everest** creates  a database for the new company. The company's database is created on the  database server that you have specified in the **Everest 
 Administrator Login**  dialog box.


{:.note}
- The database file  selected in the **Company - Template**  screen will be checked for version and validity. No template database  files can be imported unless it is originally exported from **Everest**  using **Export Company** option.

: ![]({{site.sc_baseurl}}/img/lens.gif) [Exporting  a Company]({{site.wwe_chm}}/advanced-options/exporting-a-company/exporting_a_company.html)

- If the selected  template database file was created with a different version, then you  have to upgrade the database file or version of **Everest**.

: ![]({{site.sc_baseurl}}/img/lens.gif) [Database  Update Utility](everest-advanced-edition---utilities.chm::/dbupdate_database_update_utility.htm)
: You will see a confirmation message that the company  was created successfully.

- Click **OK**  to log in to the company. You will see the [**Company Information - Address** section]({{site.sc_baseurl}}/the-company-creation-wizard/company/company_information_address.html).
