---
title: Groups Assigned
---


The **Groups Assigned** column displays  all the groups that have been assigned the selected right. To remove the  rights assigned to a group, highlight the specific group in this column  and select **Remove**  from the right-click menu or click the ![]({{site.sc_baseurl}}/img/setup_move_right.gif) icon.


To remove all the rights assigned to a group, select **Remove 
 All** from the right-click menu or click the ![]({{site.sc_baseurl}}/img/setup_moveall_right.gif) icon.
