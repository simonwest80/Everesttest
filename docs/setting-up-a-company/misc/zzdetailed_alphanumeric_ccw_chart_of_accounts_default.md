---
title: Detailed (Alphanumeric)
---


The **Detailed (alphanumeric)**  account template contains all of the accounts necessary to begin processing  transactions. The account codes created are alphanumeric. If you create  your chart of accounts using the **Detailed 
 (alphanumeric)** account template, the accounting defaults and default  payment methods are automatically populated with the appropriate accounts.  Four Company default posting groups are also created.
