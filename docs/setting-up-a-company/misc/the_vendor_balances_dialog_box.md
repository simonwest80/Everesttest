# The Vendor Balances Dialog Box


The **Vendor balances** dialog box  lists all the vendors who are created in your database. You can enter  the opening balances for vendors from this dialog box.


{:.steps}
To enter the opening balance for a vendor, do the following:

- From the **Vendor Balances** dialog box, select the  required vendor.
- Select **Open Documents** from the right-click  menu. You will see the [**Open Documents** dialog box]({{site.sc_baseurl}}/misc/the_open_documents_dialog_box.html).
- Enter the details,  and click **OK**.



{:.note}
When you click **OK**  in the **Open Documents** dialog box,  an accounting entry is automatically created where the 'Accounts Payable  Control account' is 'credited' and the 'Unreconciled Difference account'  is 'debited' with the value of the payable. For further details on account  opening balances click [here]({{site.sc_baseurl}}/the-company-creation-wizard/accounting/opening_account_balances.html).

- Continue the  process of entering the open documents for the vendors. Click **OK**  to return to the [**Vendor 
 - Account Balances** section]({{site.sc_baseurl}}/the-company-creation-wizard/vendors/vendors_account_balances.html) of the **Company 
 Creation Wizard**.
