# User Profile - General - General Information


**Code**
: Enter a unique alphanumeric code using 1 to 15 characters  of the supported type to identify the user. This is a mandatory field.


Apply Mask


Use this section to apply the formats for telephone numbers, mobile  number and zip code for the selected country.


**Telephone**
: Select this check box, to apply the telephone format  specified for the country to the telephone numbers.


**Mobile/Cell**
: Select this check box, to apply the mobile/cell  number formats specified for the country to the mobile number.


**Zip/Postal code**
: Select this check box, to apply the zip format specified  for the country to the zip/postal code.


**Join date**
: Specify the date on which the user joined the company,  in the mm/dd/yy  format.


**Title**
: Enter the user's title in this field using 1 to  8 characters of any type.


**First Name**
: Enter the user’s first name in this field.


**Last Name**
: Enter the user’s last name in this field.
