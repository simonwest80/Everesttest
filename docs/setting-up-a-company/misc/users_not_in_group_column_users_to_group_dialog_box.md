# Users Not in Group Column


The **Users not in group** column  displays all the users who are not assigned to the selected group. User(s)  may be added to a group by selecting the user(s) and choosing **Select** from the right-click menu or by clicking the ![]({{site.sc_baseurl}}/img/setup_move_left.gif) icon.


You can add all users to the group by choosing the **Select 
 All** option from the right-click menu or by clicking the ![]({{site.sc_baseurl}}/img/setup_moveall_left.gif) icon.
