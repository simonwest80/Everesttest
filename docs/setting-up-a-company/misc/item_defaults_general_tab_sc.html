Item Defaults - General

Use the Item Defaults – General tab to configure the default Item settings for Item type, Inventory type, Category, Manufacturer and Brand.

: All the fields are optional.

Item Type
Specify the type of the item to be displayed by default in the Item Type field, when you create a new item. You can select any type of the item from the options such as Regular, Kit and Assembly.
Inventory Type
Specify the type of inventory to be displayed by default in the Inventory Type field, when you create a new item. The Inventory Types associated to the Item Types are given in the table below:
Item Type Inventory Type
Regular Inventoried, Inventoried-Serialized, Inventoried-Auto Serialized and Non-Inventoried
Kit Non-Inventoried
Assembly Inventoried, Inventoried-Serialized and Inventoried-Auto Serialized
Category
Specify the category of the item to be displayed by default in the Category field, when you create a new item. The categories displayed here are derived from Categories browser (path: Inventory > Categories).
Manufacturer
Specify the Manufacturer of the item to be displayed by default in the Manufacturer field, when you create new items. The manufacturers displayed here are derived from Manufacturers browser (path: Inventory > Manufacturers).
Brand
Specify the brand of the item to be displayed by default in the Brand field, when you crate items. The brands displayed here are derived from Brands browser. (path: Inventory > Brands)
