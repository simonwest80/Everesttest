Currency Selection Dialog Box

The Currency Selection dialog box allows you to choose the currencies you want the processor to support.

To add a currency to the Supported Currencies section, highlight a currency in the Unsupported Currencies section and click the icon. Alternatively, you can click the icon to select all currencies.

To remove a currency from the Supported Currencies section, highlight a currency in the Supported Currencies section and click the icon. Alternatively, you can click the icon to remove all currencies.
