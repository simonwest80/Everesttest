# Payment Terms Details


Use the **Payment Terms** profile  (path: **File &gt; Setup &gt; Accounting 
 &gt; Payment Terms &gt;** **Options 
 &gt; Edit &gt; In New Window**) to specify the due date for payment  and the discount available for early payments or receipts. The details  to be specified in the **Payment Terms**  profile is categorized under the following heads:

- [General  Information]({{site.sc_baseurl}}/options/payment-information/payment-terms/payment-term-details/general_information_payment_terms.html)
- [Due  Date Information]({{site.sc_baseurl}}/options/payment-information/payment-terms/payment-term-details/due_date_information.html)
- [Discount  Information]({{site.sc_baseurl}}/options/payment-information/payment-terms/payment-term-details/discount_information.html)
