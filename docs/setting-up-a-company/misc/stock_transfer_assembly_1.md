# Stock Transfer - Assembly 1


Use this tab to specify the default stock area into which item components  and finished goods are transferred/received, during the manufacture of  an assembly item.


Select stock area to issue inventoried components of assembly  on a work order


**Work Order**
: Select the default stock area from where inventoried  components are issued to assemble an item. This is usually a warehouse  or a store on the shop floor.


**![]({{site.sc_baseurl}}/img/lens.gif) [Assembly  Information]({{site.sc_baseurl}}/misc/stock_transfer_information_assembly.html)**


Select stock area to receive back inventoried components  of assembly previously issued on a work order


**Work Order**
: Select the default stock area to receive inventoried  components that are detached from a work order.


**![]({{site.sc_baseurl}}/img/lens.gif) [Assembly  Information]({{site.sc_baseurl}}/misc/stock_transfer_information_assembly.html)**


Select stock area to transfer inventoried components of  assembly items in process


**In Process**
: Select the default stock area to transfer inventoried  components when the work order status changes to 'In Process'.


**![]({{site.sc_baseurl}}/img/lens.gif) [Assembly  Information]({{site.sc_baseurl}}/misc/stock_transfer_information_assembly.html)**


Select stock area to transfer inventoried components of  completed assembly items


**Work Order**
: Select the default stock area where the inventoried  components are transferred when the assembly item is built.


**![]({{site.sc_baseurl}}/img/lens.gif) [Assembly  Information]({{site.sc_baseurl}}/misc/stock_transfer_information_assembly.html)**


Select stock area to receive completed assembly items


**Work Order**
: Select the default stock area to receive assembly  items that are built.


![]({{site.sc_baseurl}}/img/lens.gif) [Assembly  Information]({{site.sc_baseurl}}/misc/stock_transfer_information_assembly.html)
