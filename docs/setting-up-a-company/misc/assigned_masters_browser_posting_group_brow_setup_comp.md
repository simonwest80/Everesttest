# Assigned Masters Browser


The **Assigned Masters** browser  displays list of masters to which each Posting Group is assigned. The  Posting Group assigned in the **Customer**  and **Vendor** tabs (path: **File**  > **Setup** > **Accounting**  > **Location/SubLocation**) will  be displayed under **New Customer Default 
 for Location** type in this browser. You can sort the types by clicking  **Type** button.
