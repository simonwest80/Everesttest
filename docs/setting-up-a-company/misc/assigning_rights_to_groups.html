Assigning Rights to Groups

To assign rights to a group, do the following:

  • From the main menu bar, select File > Setup > Users > Groups. You will see the Groups browser.
  • From the Groups browser, select the group to which you want to assign rights.
  • Select Options > Rights. You will see the Group Rights dialog box.
  • Select a function on the left panel.
  • Select the required right in the Rights not assigned column and click on the icon, OR choose All from the right-click menu. To assign all the rights to the group, select the icon from the right-click menu.
