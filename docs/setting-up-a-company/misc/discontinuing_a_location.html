Discontinue a Location/Sub-Location

To discontinue a location/sub-location, do the following:

  • From the main menu bar, select File > Setup > Accounting > Locations/Sub-Locations. You will see the Locations/Sub-Locations browser.
  • From the Location/Sub-Locations browser, highlight the required location/sub-location.
  • Select Options > Discontinue. You will be prompted for confirmation.
  • Click OK. The record that was selected is no longer visible in the Locations/Sub-Locations browser.
