Assign Users to a Group

You can assign users to a group when you know the composition of a group.

To assign a user to a group, do the following:

  • From the main menu bar, select File > Setup > Users > Users belonging to group OR from the main menu bar, select File > Setup > Users > Groups > Options > List of Users. You will see the Users belonging to group dialog box.
  • Select the required group in the left panel.
  • Select the user name in the Users not in group column and click the icon, OR choose Select from the right-click menu.

To select all users, click the icon or choose Select All from the right-click menu.

Similarly, users can be removed from a group.
