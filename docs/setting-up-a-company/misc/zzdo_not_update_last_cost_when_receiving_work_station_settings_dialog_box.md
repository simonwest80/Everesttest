---
title: Do Not Update Last Cost When Receiving
---


Select this check box if you do not want the last cost/last add-on cost  of the item to be updated when the item is purchased.


{:.example}
**Select this option if an item's selling price  is defined as a mark-up on its last cost and you do not want the last  cost/last add-on cost to fluctuate every time the item is purchased.**
