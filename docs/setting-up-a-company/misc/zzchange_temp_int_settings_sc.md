---
title: Change Template
---


Use this window to change the template of the selected user settings.  You can select the template to be applied in the **New 
 Template** drop-down box. You will get a confirmation message: ‘Any  customizations made by the user will be lost. Are you sure you want to  change the template?’ Click **Yes**  to change the template.


{:.note}
: If you change the template, the new template will  be applied to all screens for that user and any customizations made by  the user will be lost.
