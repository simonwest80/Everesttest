# Change Period Status - Close/Open Period Profile


Additionally, you can change the status of a period from **Open**  to **Closed** and vice versa from  the **Accounting** menu.


{:.steps}
To lock the period from the Accounting menu,  do the following:

- From the main  menu bar, select **Accounting &gt; Period 
 End &gt; Lock Period**. You will see the [**Lock Period** profile]({{site.sc_baseurl}}/misc/the_close_period_porfile.html).
- Select the  period to be closed and click **Lock Period**.  The selected period will be locked.



{:.steps}
To change the status of a period to open from the Accounting  menu, do the following:

- From the main  menu bar, select **Accounting &gt; Period 
 End &gt; Open Period**. You will see the [**Open Period** profile]({{site.sc_baseurl}}/misc/the_open_period_profile.html)**.**
- Select the  period to be opened and click **Open Period**.  The selected period will be marked as Open.
