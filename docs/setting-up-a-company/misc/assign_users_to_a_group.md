# Assign Users to a Group


You can assign users to a group when you know the composition of a group.


{:.steps}
To assign a user to a group, do the following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Users &gt; Users belonging to group** OR from the main menu bar,  select **File &gt; Setup &gt; Users &gt; 
 Groups &gt; Options &gt; List of Users**. You will see the [**Users belonging to group** dialog  box]({{site.sc_baseurl}}/misc/users_to_group_dialog_box.html).
- Select the  required group in the left panel.
- Select the  user name in the **Users not in group** column  and click the ![]({{site.sc_baseurl}}/img/setup_move_left.gif) icon, OR choose **Select**  from the right-click menu.



{:.note}
To select all users, click the ![]({{site.sc_baseurl}}/img/setup_moveall_left.gif) icon or choose **Select All** from the right-click menu.


Similarly, users can be removed from a group.
