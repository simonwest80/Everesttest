# Tax Component - General Information


**Component Code**
: The component code is a unique code that identifies  this particular tax component throughout **Everest**.  Once defined, it cannot be changed.


<font color="#000000" class="hcp1">We recommend that 
 you make the code informative, for easy identification. You can use letters 
 to describe the tax jurisdiction, combined with numbers to identify the 
 tax rate. The abbreviation scheme must be flexible to handle future changes 
 in the tax rates.</font>


{:.example}
<font color="#000000" class="hcp1">ST050 for state sales tax at 5% and LO15 for local 
 sales tax at 1.5%.</font>


**Description**
: Use this field to enter a description of the tax  component providing full details of the tax rate. Use 1 to 40 characters  to enter the description.


{:.example}
The tax code CA100, can be described as <font face="verdana" class="hcp7">'</font>California 1% Local Sales Tax<font face="verdana" class="hcp7">'</font>.
