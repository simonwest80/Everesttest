# Stock Transfer - Assembly 2


Use this tab to specify the default stock area into which defective  and scrap items are transferred after the manufacturing process.


Select stock area to receive completed defective assembly  items


**Work Order**
: Select the default stock area to receive the assembled  items that are defective.


Select stock area to receive scrap items


**Work Order**
: Select the default stock area to receive scrap items.
