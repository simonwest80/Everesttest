# Currency Selection Dialog Box


The **Currency Selection** dialog  box allows you to choose the currencies you want the processor to support.


To add a currency to the **Supported 
 Currencies** section, highlight a currency in the **Unsupported 
 Currencies** section and click the ![]({{site.sc_baseurl}}/img/setup_move_left.gif) icon. Alternatively,  you can click the ![]({{site.sc_baseurl}}/img/setup_moveall_left.gif) icon to select all currencies.


To remove a currency from the **Supported 
 Currencies** section, highlight a currency in the **Supported 
 Currencies** section and click the ![]({{site.sc_baseurl}}/img/setup_move_right.gif) icon. Alternatively,  you can click the ![]({{site.sc_baseurl}}/img/setup_moveall_right.gif) icon to remove all currencies.
