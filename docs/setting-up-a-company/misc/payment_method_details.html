Payment Method Details

Use the Payment Method profile (path: File > Setup > Accounting > Payment Methods > Options > Edit > In New Window) to set up the information required to create a payment method. The details to be set up in the Payment Method profile is classified under the following heads:

  • General Information
  • Type of Payment
  • Credit Card Information
  • Narrow Receipt Message
  • Miscellaneous Information
