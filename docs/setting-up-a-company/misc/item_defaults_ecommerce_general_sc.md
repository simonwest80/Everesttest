# Item Defaults - E-Commerce - General


Use the **E-Commerce - General**  tab to configure the default web enabling options to show the items on  your online shop.


**Web enabled**
: Check this box to ensure that the **Web 
 enabled** field is checked by default, when you create a new item.  This field is optional.
