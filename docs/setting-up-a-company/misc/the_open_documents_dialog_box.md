# The Open Documents - Customers Dialog Box


Use the **Open Documents** dialog  box to enter the open documents for each customer. This dialog box is  displayed when you select the **Open documents** option from the right-click menu in the **Customer 
 Balances** dialog box. The columns contained in the **Open 
 Documents** dialog box are explained below:


**Document Type**
: A customer can have only the final documents as  open documents: **Credit Memo** and  **Sales Invoice**. Click on the row  below the column header to view a drop-down list of the available document  types. Select the required document type. This is a mandatory field.


**Document #**
: The document number displays in this column. This  is an optional field, but you can update it if necessary.


**Amount in Base Currency**
: Enter the amount of the document in this column.  The amount entered should be in the [base currency](javascript:BSSCPopup('{{site.sc_baseurl}}/misc/base_currency_pop_up.html');){:id="A1"}<script type="text/javascript" language="JavaScript1.2"><!--
kadovFilePopupInit('A1');
//--></script>.  This is a mandatory field.


**Rate**
: If the document was created in a currency other  than the base currency, enter the conversion rate of the foreign currency  to the base currency in this column.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Setting  up Exchange Rates]({{site.sc_baseurl}}/options/multicurrency/exchange-rates/set-up-exchange-rates/setting_up_exchange_rates.html)


**Amount in Document Currency**
: Specify an amount in this column, only if the document  currency is a foreign currency.


**Terms**
: Click the column to view a drop-down list of the  available payment terms, created by default. Select the required payment  term. This is a mandatory field.


**Date**
: Specify the date of the document in this column.  This is a mandatory field.


Click **OK** when you have entered  all the open documents for the selected customer. You will see the [**Customer 
 Balances**]({{site.sc_baseurl}}/misc/the_customer_balances_dialog_box.html) dialog box. Repeat the process till all the open customer  documents have been specified.


{:.note}
**Click **OK**  in the **Customer** **Balances**  dialog box to save the data in the database.**


Click **Next**. You will see the  [**Vendors 
 - Vendor Details** section]({{site.sc_baseurl}}/the-company-creation-wizard/vendors/vendors_vendor_details.html).
