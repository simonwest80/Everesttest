# The Security Rights Browser - Groups Tab


The **Security Rights** browser  - **Groups** tab displays the groups  that are/are not assigned a particular security right.


The **Security Rights** browser  with the **Groups** tab selected consists  of the following sections:

- [Left  Panel]({{site.sc_baseurl}}/misc/left_panel_security_rights_groups.html)
- [Groups  Assigned Column]({{site.sc_baseurl}}/misc/groups_assigned_column_security_rights_groups.html)
- [Groups  Not Assigned Column]({{site.sc_baseurl}}/misc/groups_not_assigned_column_security_rights_groups.html)
