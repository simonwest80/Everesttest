# The Lock Period Profile


Use this profile to specify the period that must be marked as 'Lock'.


Period


**Select the Period**
: Select the period to be locked. The drop-down list displays all the periods whose status is not locked.


**Start Date - End Date**
: This field displays the start and end dates of the period selected in the **Select the Period** field.
