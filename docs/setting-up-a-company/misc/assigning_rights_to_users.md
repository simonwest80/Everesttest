# Assigning Rights to Users


{:.steps}
To assign rights to a user, do the following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Users &gt; Users**. You will see the **Users**  browser.
- From the **Users** browser, select the user to whom  you want to assign rights.
- Select **Options &gt; Rights &gt; User Rights**.  You will see the **User 
 Rights** dialog box.
- Select a function  in the left panel.
- Select the  required right in the **Rights not assigned**  column and click on the ![]({{site.sc_baseurl}}/img/setup_move_left.gif) icon OR choose **Add** from the right-click menu. Click on the ![]({{site.sc_baseurl}}/img/setup_moveall_left.gif) icon to add  all the rights displayed in the **Rights 
 not assigned** column, to the user.
