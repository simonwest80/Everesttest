# Assigning Rights - Security Rights Browser


{:.steps}
To assign rights to a user, do the following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Security Rights**. You will see the **Security 
 Rights** browser**.**
- Click the **Users** tab.



![]({{site.sc_baseurl}}/img/lens.gif)[Security  Rights Browser - Users Tab]({{site.sc_baseurl}}/misc/security_rights_users_dialog_box.html)

- Select the  required function on the left panel.
- Select the  required user in the **Users not assigned**  column and click the   ![]({{site.sc_baseurl}}/img/setup_move_left.gif) icon OR choose **Add**  from the right-click menu to assign the selected right to the user displayed  in the **Users not assigned** column.  To assign a right to all users, select the ![]({{site.sc_baseurl}}/img/setup_moveall_left.gif)  icon  or choose **Add All** from the right-click  menu.



{:.note}
An audit trail is created in the **Everest 
 Event History** browser every time a particular user's credentials  are overridden by a user with more access privileges. This information  can assist with determining additional rights needed by a user.
