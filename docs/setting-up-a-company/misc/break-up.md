# <font style="color: #000000;" color="#000000">Breakup</font>


Select this option to view a graphical representation of the breakup  for an account.


{:.note}
**The **Breakup** option is not available for ‘detailed’ accounts.**


This option displays the percentage breakup of all the detailed accounts  of the parent account selected.
