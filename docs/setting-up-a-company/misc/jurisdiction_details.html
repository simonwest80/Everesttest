Jurisdiction Details

Use the Jurisdiction profile (path: File > Setup > Accounting > Taxes > Jurisdictions > Options > Edit > In New Window) to associate a tax code to the jurisdiction, thereby specifying the tax to be charged on sales of items in the jurisdiction.

The information to be specified in the Jurisdiction profile has been grouped under the following heads:

  • Code and Name
  • Type of Jurisdiction
  • Location Information
  • Tax Code
  • FIPS codes
