---
title: Account Code
---


Enter a unique alphanumeric code to identify the account being defined.  This code cannot be modified once the account has been defined. This is  a mandatory field.


If you are using segmented accounts, enter the **Account 
 Code** without the segment separators. The segment separators are  added automatically.
