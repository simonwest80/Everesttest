# <font color="#000000" class="hcp1">Summary</font>


This option provides a summary of the transactions that are posted to  the general ledger for the selected detail account.


{:.example}
**If you choose to view the summary for the  Inventory Control account, all journal transactions that contain the account  are displayed.**


This option displays the **Posted Account Summary** dialog box.
