# Reconciliation Details:


The reconciliation details are displayed only for bank accounts. This  section displays information in relation to Bank Reconciliation.


**Reconciled On**
: Displays the date when the last reconciliation was  carried out for the bank. This field cannot be edited.


**Reconciled As Of**
: Displays the date as on which the balance was reconciled.  This field cannot be edited.


**Amount - Base Currency**
: Displays the reconciled balance for the last reconciliation  carried out for the bank account in base currency. This field cannot be  edited.


**Amount - Account Currency**
: Displays the reconciled balance for the last reconciliation  carried out for the bank account in account currency. This field is displayed  only for foreign currency accounts and cannot be edited.
