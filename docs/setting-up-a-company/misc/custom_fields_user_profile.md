# Custom Fields


Use the **Custom Fields** option to define additional information for any record as  per your requirement. Custom fields can be defined as character strings,  date fields, logical fields, numeric fields or memo fields.


{:.example}
**In a **User**  profile, you might want to maintain a record of the user's emergency contact  number.**
