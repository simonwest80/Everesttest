# Item Defaults - Inventory


Use the Â **Item 
 Defaults-Inventory** tab to configure the default settings for inventory  information such as Factor cost, Percent, Measure, Use lot tracking, Auto  generate Lot numbers, Critical and Bulk item, when you create a new item.


{:.note}
: All the fields are optional.


Cost  Information


**Factor cost**
: Check this box to ensure that the **Factor cost** field is checked by default, when you create a new  item.


**Percent**
: Enter the Percentage of factor cost of item to be  displayed by default in **Percent**  field, when you create a new item. You can specify a maximum value of  999.99 in this field.


{:.note}
: The **Percent**  field is activated only if you checked **Factor 
 cost** field.


Other  inventory information


**Stock measure**
: Specify the stock measure of the item to be displayed  by default in the **Stock measure**  field, when you create a new item. The **Stock 
 measure** options displayed here are derived from **Measures**  browser (path: **Inventory** >  **Measures**).


**Use lot tracking**
: Check this box to ensure that the **Use 
 lot tracking** field is checked by default, when you create a new  item. The **Use lot tracking** check  box is enabled according to the **Inventory 
 Type** options configured in the **Item 
 Defaults-General** tab.


| Inventory Type | Lot tracking |
| Inventoried | Enabled |
| Inventoried-Serialized | Enabled |
| Inventoried-Autoserialized | Enabled |
| Non-Inventoried | Disabled |



**Auto generate Lot Numbers**
: Check this box to ensure that the **Auto 
 generate Lot Numbers** field is checked by default, when you create  a new item.


{:.note}
: The Auto generate Lot Numbers field is activated  only if you checked the Use lot tracking field.


**Critical**
: Check this box to ensure that the **Critical 
 field** is checked by default, when you create a new item.


**Bulk Item**
: Check this box to ensure that the **Bulk 
 Item** field is checked by default, when you create a new item.
