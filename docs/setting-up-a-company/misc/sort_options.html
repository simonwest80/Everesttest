Sort Options

You can select the option by which data must be sorted in the User Rights dialog box. You can sort the security rights in the left panel of the User Rights dialog box based on:

  • Functional group or
  • User Interface group

If you sort the security rights by the “Functional” group, the security rights under each function will be sorted, by default,  based on the “User Interfaces” and vice versa.
