# WorldPay Settings


**Installation ID**
: Enter the installation ID provided by WorldPay service.  It is a unique ID that identifies the merchant using the service. This  information is passed to WorldPay during the validation and authorization  routine, so that WorldPay can correctly track which merchant the sales  should be credited to.


**Time-out (seconds)**
: Enter the number of seconds after which the request  for authorization is timed out.
