The Location Profile - Accounts - Fixed Asset

In the Fixed Asset tab, specify the accounts to use for the Fixed Asset control accounts for this location.

Original Asset
Select an asset account to use for the Fixed Asset control account for initial assets for this location.
Accumulated Depreciation
Select an asset account to use for the Fixed Asset control account for depreciation on original assets for this location.
Depreciation Expense
Select an asset account to use for the Fixed Asset control account for the depreciation expenses for this location.
