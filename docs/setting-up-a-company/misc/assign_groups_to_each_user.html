Assign Groups to Each User

To assign a group to a user, do the following:

  • From the main menu bar, select File > Setup > Users > Groups users belong to. You will see the Groups user belongs to dialog box.
  • Select the required user in the left panel of the dialog box.
  • Select the group in the Groups user not belonging to column and click the icon,

To select all users, click the icon or choose Select All from the right-click menu.

Similarly, users can be removed from a group.
