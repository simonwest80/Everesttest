# Currency Details


Use the **Currency** profile (path:  **File &gt; Setup &gt; Accounting &gt; 
 Multicurrency &gt; Currency &gt; Options &gt; Edit &gt; In New Window**)  to define the general information for the currency such as the currency  code and description, the format for the currency and the display information.


The information in the **Currency**  profile has been classified under the following:

- [General  Information]({{site.sc_baseurl}}/misc/general_information_currency_details.html)
- [Format  Specification]({{site.sc_baseurl}}/options/multicurrency/setup/defining/details/format-specification/format_specification_currency_details.html)
- [Quotation  Information]({{site.sc_baseurl}}/options/multicurrency/setup/defining/details/quotation-information/quotation_information_currency_details.html)
- [Display  and Printing Information]({{site.sc_baseurl}}/options/multicurrency/setup/defining/details/print-info/display_and_printing_information_currency_details.html)
