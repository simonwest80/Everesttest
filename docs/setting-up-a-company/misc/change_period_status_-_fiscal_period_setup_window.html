Change Period Status - Fiscal Period Setup

This topic details how to change the status of a period from the Fiscal Period Setup dialog box.

To change the status of a period from the Fiscal Period Setup dialog box, do the following:

  • From the main menu bar, select File > Setup > Accounting > Periods. You will see the Fiscal Period Setup dialog box.
  • Select the required fiscal year and click Next. All the periods defined for the selected year are displayed in this dialog box.
  • Click the status column of the required period. You will see a drop-down list displaying the three status types.
  • Select the required status and click Apply to save the changes.

You can also lock the period from the Accounting menu.
