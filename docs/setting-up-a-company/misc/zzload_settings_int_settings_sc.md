title: Load Settings
---


Use this option to load the settings currently selected in the browser  and edit them. These settings will remain loaded until you log out or  load different settings.


{:.note}
: Load Settings option is desensitized when any other  userâ€™s settings are selected in the browser.


The title bar will reflect the currently loaded settings, when the **Customize** or **Browser 
 Properties** screen is selected.


{:.hint}
: Use the **Enter**  key as a shortcut key to load the selected settings.
