Assigning Rights to Users

To assign rights to a user, do the following:

  • From the main menu bar, select File > Setup > Users > Users. You will see the Users browser.
  • From the Users browser, select the user to whom you want to assign rights.
  • Select Options > Rights > User Rights. You will see the User Rights dialog box.
  • Select a function in the left panel.
  • Select the required right in the Rights not assigned column and click on the icon OR choose Add from the right-click menu. Click on the icon to add all the rights displayed in the Rights not assigned column, to the user.
