# Groups User Not Belonging To Column


The **Groups user not belonging to** column  displays all the groups not assigned to the selected user.


To assign a group to a user, highlight the specific group and choose  **Select** from the right-click menu  or click the ![]({{site.sc_baseurl}}/img/setup_move_left.gif) icon.


To assign all groups to the user, choose the **Select 
 All** option from the right-click menu or click the ![]({{site.sc_baseurl}}/img/setup_moveall_left.gif) icon.
