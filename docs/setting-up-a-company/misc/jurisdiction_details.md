# Jurisdiction Details


Use the **Jurisdiction** profile  (path: **File &gt; Setup &gt; Accounting 
 &gt; Taxes &gt; Jurisdictions &gt;** **Options 
 &gt; Edit &gt; In New Window**) to associate a tax code to the jurisdiction,  thereby specifying the tax to be charged on sales of items in the jurisdiction.


The information to be specified in the Jurisdiction profile has been  grouped under the following heads:

- [Code  and Name]({{site.sc_baseurl}}/options/sales-tax/tax-jurisdictions/jurisdiction-details/code_and_name.html)
- [Type  of Jurisdiction]({{site.sc_baseurl}}/options/sales-tax/tax-jurisdictions/jurisdiction-details/type_of_jurisdiction.html)
- [Location  Information]({{site.sc_baseurl}}/options/sales-tax/tax-jurisdictions/jurisdiction-details/location_information.html)
- [Tax  Code]({{site.sc_baseurl}}/options/sales-tax/tax-jurisdictions/jurisdiction-details/tax_code.html)
- [FIPS  codes]({{site.sc_baseurl}}/options/sales-tax/tax-jurisdictions/jurisdiction-details/fips_codes.html)
