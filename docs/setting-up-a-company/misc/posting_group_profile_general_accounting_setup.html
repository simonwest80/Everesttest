Posting Group Profile - General

The General tab of Posting Group profile allows you to define the Posting Group ID, Posting Group Description and select the Posting Group Type.

Posting Group ID
Enter an identification code for the Posting Group using up to 10 alphanumeric characters. You can edit this ID from the option Edit Posting Group ID (path: Posting Groups browser > Options).
Description
Enter a description for the Posting Group being defined. Use up to 40 alphanumeric characters. You can edit this field any time.
Posting Group Type
Select a Posting Group Type from the drop-down menu. The following options are available:
  • Customers
  • Vendors
  • Inventory
  • Fixed Assets

Once you have defined the accounts for a Posting Group and saved the profile, you cannot change the Posting Group Type.
