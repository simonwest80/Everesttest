---
title: Apply Mask
---


Click the **Apply Mask** button  to apply the formats for telephone, zip code and mobile specified in the  profile of the country selected in the **Country**  field.


**Telephone**
: Select this check box if you want the telephone  format specified for the country to be applied to the telephone numbers.


**Mobile/Cell**
: Select this check box if you want the mobile/cell  number formats specified for the country to be applied to the mobile number.


**Zip/Postal code**
: Select this check box if you want the zip format  specified for the country to be applied to the zip/postal code.


{:.note}
For every country that is specified in any  address, you can specify the format for the telephone and mobile numbers  and zip codes.
