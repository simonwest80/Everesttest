System Source Types - Purchasing Transactions

The system source types for purchasing transactions are:

Source Type Description
DGV Deposit for Purchase Order
DGR Deposit Returned for Purchase Order
MRC Purchase Receipt
PAY Payment for Purchase
PRT Payment Refunded for Purchase
DRM Debit Memo
DAY Payment for Debit Memo
VAD Deposit/Debit Adjustment on Purchase Receipt
