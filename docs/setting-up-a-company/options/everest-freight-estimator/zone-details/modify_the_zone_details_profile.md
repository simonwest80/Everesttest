# Modify the Zone Details Profile


You can modify the **Zone Details**  profile from any of the following locations:

- The **Zone 
 Details** browser - Highlight the record and select **Options 
 &gt; Edit &gt; In New Window**.
- The **Zone 
 Details** profile - Specify the code of the required zone in the  **Code** field.



You cannot modify the **Zone** **Code**.
