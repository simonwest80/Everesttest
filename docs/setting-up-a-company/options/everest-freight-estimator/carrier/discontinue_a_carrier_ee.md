# Discontinue a Carrier


When you discontinue a carrier, it is not deleted from the database;  it is merely deactivated. A discontinued carrier can be made available  for selection by activating it.


{:.steps}
To discontinue a carrier, do the following:

- From the main  menu bar, select **File &gt; Setup &gt;****Freight** > **Carrier**.  You will see the **Carriers** browser.
- From the **Carriers** browser highlight the carrier  that has to be discontinued.
- Select **Options** > **Discontinue**.  You will be prompted for confirmation.
- Click **OK**. The selected record is no longer  visible in the **Carriers** browser.
