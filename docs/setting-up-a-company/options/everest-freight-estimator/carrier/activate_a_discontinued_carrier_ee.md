# Activate a Discontinued Carrier


You can make a discontinued carrier available for selection by activating  it again.


{:.steps}
To view and activate a discontinued carrier,  do the following:

- From the main  menu bar, select **File** > **Setup** > **Freight**  > **Carrier**. You will see the  **Carrier** browser.
- From the **Carrier** browser, select **Options**  > **View** **Discontinued**.  You will see the **Discontinued Carrier**  browser listing all the deactivated carriers.
- Highlight the  carrier you want to activate.
- From the **Discontinued Carrier** browser, select  **Options** > **Activate**.  You will be prompted for confirmation.
- Click **OK** to activate. The activated record  will be displayed in the **Carrier**  browser.
