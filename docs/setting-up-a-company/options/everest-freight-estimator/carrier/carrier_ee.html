Carrier

Carriers (UPS, Fedex) are professional delivery agents, who deliver goods to specific destinations using various delivery methods (Fedex Priority Overnight or Fedex 2 Day). You can define any number of carriers in Everest. You can allow your customers to view the delivery status of their orders from the carrier’s website (by using a shipment-tracking number assigned to the order).

To create a carrier, do the following:

  • From the main menu bar, select File > Setup > Freight > Carrier. You will see the Carrier browser.
  • From the Carrier browser, select Options > New > In New Window. You will see the Carrier profile.
