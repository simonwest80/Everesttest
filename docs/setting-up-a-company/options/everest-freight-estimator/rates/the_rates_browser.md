# The Rates Browser


The **Rates** browser (path: **File** > **Setup** > **Freight** > **Rates**) displays a list of the freight  rates and other related information.


You can add, edit or discontinue freight rates from the browser. Highlight  and double-click on a record to view or edit a **Rates** profile.


This browser contains options that are common to all browsers in **Everest**.
