The Zones Browser

The Zones browser (path: File > Setup > Freight > Zones) displays a list of the zones and other related information.

Zones can be added, edited or discontinued from the browser. Highlight and double-click on a record to view or edit a Zones profile.

Some additional options available in this browser are:

Zone Detail
This option brings up the Zone Details browser for the selected zone.
New Zone Detail
Use this option to create the details of a new zone.
