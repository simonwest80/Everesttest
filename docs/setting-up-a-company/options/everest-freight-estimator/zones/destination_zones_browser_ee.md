# The Zones Browser


The **Zones** browser (path: **File** > **Setup**  > **Freight** > **Zones**)  displays a list of the zones and other related information.


Zones can be added, edited or discontinued from the browser. Highlight  and double-click on a record to view or edit a [**Zones**]({{site.sc_baseurl}}/options/everest-freight-estimator/zones/destination_zones_profile.html) profile.


Some additional options available in this browser  are:


**Zone Detail**
: This option brings up the [**Zone Details** browser]({{site.sc_baseurl}}/options/everest-freight-estimator/zone-details/destination_zones_details_browser_ee.html) for the selected  zone.


**New Zone Detail**
: Use this option to create the details of a [new  zone]({{site.sc_baseurl}}/options/everest-freight-estimator/zone-details/destination_zone_details_profile.html).
