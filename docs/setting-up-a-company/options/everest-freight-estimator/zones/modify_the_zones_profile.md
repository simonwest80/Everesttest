# Modify the Zones Profile


You can modify the **Zones** profile  from any of the following locations:

- The **Zones**  browser - Highlight the record and select **Options 
 &gt; Edit &gt; In New Window**.
- The **Zones**  profile - Specify the code of the required zone in the **Code**  field.



You can modify the **Description**  and **Carrier** fields in this profile.
