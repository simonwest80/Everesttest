The Payment Method Profile

Use the following tabs to set up the information required to create a payment method:

  • General
  • Credit Card Processing
  • Gift Card Processing
  • Miscellaneous
**The **Credit
Card Processing** tab is enabled only if you select CreditCard in the Type field in the General tab. The Gift Card Processing tab is displayed only if you select Gift Card from the Type field in the General tab.**
