# Modify the Payment Method Profile


**<font color="#000000" class="hcp2">You can modify the 
 payment method profile from any of the following locations:</font>**

- The **Payment 
 Methods** browser - Highlight the record and select **Options 
 &gt; Edit &gt; In New Window**.
- The **Payment 
 Method** profile - Specify the code of the required payment method  in the **Code** field.



**<font color="#000000" class="hcp2">The following fields 
 cannot be modified in the </font>**Payment**<font color="#000000" class="hcp2"> </font>**Method**<font color="#000000" class="hcp2"> profile:</font>**

- Code
- Account
- Security Description
