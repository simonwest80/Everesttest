Create a Payment Method

To create a payment method, do the following:

  • From the main menu bar, select File > Setup > Accounting > Payment Methods. You will see the Payment Methods browser.
  • From the Payment Methods browser, select Options > New > In New Window. You will see the Payment Method profile.

You can also create a payment method from the following:

  • An existing Payment Method profile by clicking Options > New.
  • Any profile/dialog box that contains the Payment Method field by selecting New from the right-click menu options.
