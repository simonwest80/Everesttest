# The Payment Method Profile


Use the following tabs to set up the information required to create  a payment method:

- [General]({{site.sc_baseurl}}/options/payment-information/payment-methods/set-up-a-payment-method/payment_method_profile_general_tab.html)
- [Credit  Card Processing]({{site.sc_baseurl}}/options/payment-information/payment-methods/set-up-a-payment-method/payment_method_profile_cc_authorization_tab.html)
- [Gift  Card Processing]({{site.sc_baseurl}}/options/payment-information/payment-methods/set-up-a-payment-method/the_payment_method_profile_gift_card_processing_tab_sc.html)
- [Miscellaneous]({{site.sc_baseurl}}/options/payment-information/payment-methods/set-up-a-payment-method/payment_method_profile_miscellaneous_tab.html)



{:.note}
**The **Credit 
 Card Processing** tab is enabled only if you select **Credit****Card** in the **Type**  field in the **General** tab. The  **Gift Card Processing** tab is displayed  only if you select **Gift Card** from  the **Type** field in the **General**  tab.**
