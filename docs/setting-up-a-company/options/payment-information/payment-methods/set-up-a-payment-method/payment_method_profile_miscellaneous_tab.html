The Payment Method Profile - Miscellaneous

Use the Miscellaneous tab to specify the security information and the format for the Point of Sale receipts.

Security Description
Enter the description for the security right to be used to indicate if users have the right to use this payment method. This field is populated by default and can be changed if required. This is a mandatory field.
Miscellaneous information
Narrow Receipt Message
Enter text to be printed on point of sale invoices in this section.
Narrow Receipt Message
For Gift Cards, enter the message to be printed on the narrow receipt in the Message box. This field is not mandatory.

Once you have entered all of the information for the Payment Method profile, click the Save button to save the payment method profile.
