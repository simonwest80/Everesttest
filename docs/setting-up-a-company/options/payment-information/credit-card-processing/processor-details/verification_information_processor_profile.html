Verification Information

This section displays the information to be verified by the processor.

Service
Service identifies the payment service provider for the processor. Everest supports IC Verify, Payflow Pro and WorldPay.

Fraud Prevention

Use this section to specify if the processor supports fraud prevention, by verifying the address information or the security code of the credit card.

Settings in AVS .ini Files for Payflow Pro

If these options are not checked, you cannot verify the address or the security code number of the credit card. The options in the Payment Method profile > Credit Card Processing > Fraud Prevention section tab will be disabled.
