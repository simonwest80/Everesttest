# Processor Details


Use the **Processors** profile (path:  **File &gt; Setup &gt; Accounting &gt; 
 Processor &gt;** From the **Processor** browser, double-click the required record) to specify the information  required to set up a processor. The details to be set up in the **Processors** profile are categorized under  the following heads:

- [General  Information]({{site.sc_baseurl}}/options/payment-information/credit-card-processing/processor-details/general_information_processor_profile.html)
- [Contact  Information]({{site.sc_baseurl}}/options/payment-information/credit-card-processing/processor-details/contact_information_processor_profile.html)
- [Verification  Information]({{site.sc_baseurl}}/options/payment-information/credit-card-processing/processor-details/verification_information_processor_profile.html)
- [IC  Verify Specific Information]({{site.sc_baseurl}}/misc/ic_verify_specific_information_processor_profile.html)
- [Payflow  Pro Information]({{site.sc_baseurl}}/options/payment-information/credit-card-processing/processor-details/processor_profile_payflow_pro.html)
- [WorldPay  Information]({{site.sc_baseurl}}/misc/worldpay_settings_processor_profile_settings_tab.html)
- [Currency  Information]({{site.sc_baseurl}}/options/payment-information/credit-card-processing/processor-details/currency_information_processor.html)
