# Verification Information


This section displays the information to be verified by the processor.


**Service**
: Service identifies the payment service provider  for the processor. **Everest** supports  IC Verify, Payflow Pro and WorldPay.


Fraud Prevention


Use this section to specify if the processor supports fraud prevention,  by verifying the address information or the security code of the credit  card.


![]({{site.sc_baseurl}}/img/lens.gif)
: [Settings  in AVS .ini  Files for Payflow Pro]({{site.sc_baseurl}}/misc/settings_in_avs.ini_file.html)


{:.note}
If these options are not checked, you cannot  verify the address or the security code number of the credit card. The  options in the **Payment Method**  profile > **Credit Card Processing &gt; 
 Fraud Prevention** section tab will be disabled.
