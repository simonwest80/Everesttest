Processor Details

Use the Processors profile (path: File > Setup > Accounting > Processor > From the Processor browser, double-click the required record) to specify the information required to set up a processor. The details to be set up in the Processors profile are categorized under the following heads:

  • General Information
  • Contact Information
  • Verification Information
  • IC Verify Specific Information
  • Payflow Pro Information
  • WorldPay Information
  • Currency Information
