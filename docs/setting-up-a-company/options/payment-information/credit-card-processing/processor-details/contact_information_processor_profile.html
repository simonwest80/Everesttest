Contact Information

This section details the contact information for the processor.

Help Desk Telephone
Use this option to specify the telephone number for the help desk of the payment gateway. You can contact the help desk using this telephone number, if there is any problem during processing or data transmission.
Voice Authorization Telephone
There are certain occasions when the financial institution may require voice authorization. In such cases, you can contact the financial institution, using the telephone number specified in the VoiceAuthorization Telephone field, to obtain approval for the payment.
