# Modify the Processor Profile


**You can modify the **Processor**  profile from any of the following locations:**

- The **Processors** browser - Highlight the record and select **Options 
 &gt; Edit &gt; In New Window**.
- The **Processor** profile - Specify the code of the required processor in the **Code** field.



**<font color="#000000" class="hcp3">You cannot modify 
 the <span class="hcp2">Code </span>field in the </font>**Processor**<font color="#000000" class="hcp3"> profile.</font>**
