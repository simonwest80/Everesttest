# View and Activate a Discontinued Processor


{:.steps}
To view and activate a discontinued processor, do the following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Accounting &gt; Processors**. You will see the **Processors** browser.
- From the **Processors** browser, select **Options** > **View Discontinued**.  You will see the discontinued records in the **Discontinued 
 Processors** browser.
- F<font color="#000000" class="hcp2">rom the </font>**Discontinued**<font color="#000000" class="hcp2"> </font>**Processors**<font color="#000000" class="hcp2"> browser, 
 highlight the record.</font>
- <font color="#000000" class="hcp2">Select </font>**Options &gt; Activate**<font color="#000000" class="hcp2">. 
 You will be prompted for confirmation.</font>
- <font color="#000000" class="hcp2">Click </font>**OK**<font color="#000000" class="hcp2">. The activated record 
 will be displayed in the </font>**Processors**<font color="#000000" class="hcp2"> browser. </font>
