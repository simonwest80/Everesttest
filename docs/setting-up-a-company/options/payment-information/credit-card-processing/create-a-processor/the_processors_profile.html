The Processor Profile

The Processor profile contains information on the different credit card processing services that you use (path: File > Setup > Accounting > Processors > Options > Edit > In New Window). This information is automatically directed to the appropriate payment gateway.

The Processor profile contains the following tabs:

  • General
  • Settings
  • Currencies

The tabs are displayed based on the option you select in the Service field in the General tab.
