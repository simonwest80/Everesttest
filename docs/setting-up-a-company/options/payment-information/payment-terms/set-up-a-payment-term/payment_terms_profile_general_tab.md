# The Payment Terms Profile


Use the payment terms profile to specify the due date for payment and  the discount available for early payments or receipts. This is a mandatory  field.


**Code**
: Enter the code for the payment term you are defining,  using 1 to 15 characters. This code is unique and once saved, you cannot  modify this field.


**Description**
: Enter a suitable alphanumeric description for the  payment term, using 1 to 40 characters. This is a mandatory field.


**Payment Term Type**
: Specify the type of payment term in this field.  The options available are: **On Account**,  **COD** and **Prepay**.
: ![]({{site.sc_baseurl}}/img/lens.gif) [General  information]({{site.sc_baseurl}}/options/payment-information/payment-terms/payment-term-details/general_information_payment_terms.html)


**Web enabled**
: Select this option to make the payment term available  for online transactions on your **Everest 
 eStorefront** store.


Due Date Information


**Due Date Type**
: This field is used to specify how the due date of  the invoice is to be calculated.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Due  date information]({{site.sc_baseurl}}/options/payment-information/payment-terms/payment-term-details/due_date_information.html)


**Specified days from invoice/A specified day of  next month**
: According to the due date type option selected,  specify the number of days in this field. This field is not enabled for  the due date types <font face="Verdana" class="hcp3">'</font>End of next month<font face="Verdana" class="hcp3">'</font> and <font face="Verdana" class="hcp3">'</font>Same day next  month<font face="Verdana" class="hcp3">'</font>.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Due  date information]({{site.sc_baseurl}}/options/payment-information/payment-terms/payment-term-details/due_date_information.html)


**Discount (%)**
: Enter the discount percentage that is applicable,  if payment is made on or before the due date specified in the **Discount 
 type** field.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Discount  information]({{site.sc_baseurl}}/options/payment-information/payment-terms/payment-term-details/discount_information.html)


**Discount Type**
: This field is used to specify how the discount for  early payment is calculated.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Discount  information]({{site.sc_baseurl}}/options/payment-information/payment-terms/payment-term-details/discount_information.html)


**Specified days from invoice/A specified day of  next month**
: Enter the number of days from the invoice date or  the date up to which the discount is available.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Discount  information]({{site.sc_baseurl}}/options/payment-information/payment-terms/payment-term-details/discount_information.html)
