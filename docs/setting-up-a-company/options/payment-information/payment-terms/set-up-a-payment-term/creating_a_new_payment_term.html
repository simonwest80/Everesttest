Create a Payment Term

To create a payment term, do the following:

  • From the main menu bar, select File > Setup > Accounting > Payment Terms. You will see the Payment Terms browser.
  • From the Payment Terms browser, select Options > New > In New Window. You will see the Payment Terms profile.

You can also create a payment term from the following:

  • An existing Payment Terms profile by clicking Options > New.
  • Any profile/dialog box that contains the Payment Term field by selecting New from the right-click menu options.
