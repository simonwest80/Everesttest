# Custom Fields Dialog Box


The **Custom Fields** dialog box  comprises the following tabs:

- [Characters]({{site.sc_baseurl}}/options/miscellaneous-set-up/custom-fields/the-custom-field-profile/characters_tab.html)
- [Dates]({{site.sc_baseurl}}/options/miscellaneous-set-up/custom-fields/the-custom-field-profile/dates_tab.html)
- [Logicals]({{site.sc_baseurl}}/options/miscellaneous-set-up/custom-fields/the-custom-field-profile/logical_tab.html)
- [Numerics]({{site.sc_baseurl}}/options/miscellaneous-set-up/custom-fields/the-custom-field-profile/numeric_tab.html)
- [Memo]({{site.sc_baseurl}}/options/miscellaneous-set-up/custom-fields/the-custom-field-profile/custom_fields_memo.html)



Each tab in the profile displays the custom field name, the value stored  by the field, whether the custom field is web-enabled in order to make  it available for the online store, and if the field is mandatory.


{:.note}
To specify a value,  you can click the **Value** column  and enter the data for the custom field.
