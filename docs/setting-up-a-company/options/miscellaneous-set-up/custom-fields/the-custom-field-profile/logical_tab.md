# Custom Fields - Logicals


Use the **Logicals** tab of the  **Custom Fields** dialog box to specify  if the information is true or false.


{:.example}
The field name in a **User** profile  may be 'Willing to work nights' and the field itself would be true.


You can define any number of logical fields in this tab. By default,  only four logical fields are displayed.
