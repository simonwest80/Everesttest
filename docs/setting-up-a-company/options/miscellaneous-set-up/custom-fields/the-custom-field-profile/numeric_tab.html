Custom Fields - Numerics

Use the Numerics tab of the Custom Fields dialog box to specify numeric values for the record.

The field name in a Customer profile may be ‘Number of employees’ and the field data may be ‘200’.

You can also store information about the size of the customer’s company and its yearly revenues.

You can define any number of numeric fields in this tab. By default, only four numeric fields are displayed.
