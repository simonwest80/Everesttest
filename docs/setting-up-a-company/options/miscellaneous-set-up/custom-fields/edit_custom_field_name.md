# Edit Custom Field Name


You can edit the names of custom fields to suit your requirements.


{:.steps}
To edit the name of a custom field, do the following:

- From the **Custom Fields** dialog box, select the  required tab whose details you want to edit.
- Right-click  the grid and select **Properties**.  You will see the [**Custom 
 Field Definition** profile]({{site.sc_baseurl}}/options/miscellaneous-set-up/custom-fields/custom-field-definition-profile/custom_field_definition_profile.html).



You cannot modify the following fields:

- Custom Field
- Field Type
