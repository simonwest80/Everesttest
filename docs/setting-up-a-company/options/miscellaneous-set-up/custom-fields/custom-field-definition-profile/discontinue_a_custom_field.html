Discontinue a Custom Field

To discontinue a custom field, do the following:

  • From the Custom Fields dialog box, select Options > Custom Fields > List. You will see the Custom Fields browser.
  • From the Custom Fields browser, highlight the record.
  • Select Options > Discontinue. You will be prompted for confirmation.
  • Click OK. The selected record is no longer visible in the Custom Fields browser.

You can also discontinue a custom field from the Custom Field Definition profile by selecting Options > Discontinue.
