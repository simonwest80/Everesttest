# Access Custom Fields Dialog Box


You can add custom fields to the item, user, serial number, company,  customer and vendor profiles and contact manager tasks.


{:.note}
Custom fields are pre-fixed with a ‘c’  (path: Right-click the column header in the **Customer/Vendor/User/Serial number** browser and select **Field****C****hooser**.  You will see the **Customize** window).


{:.steps}
To access custom fields information, do the following:

- For customers,  vendors, items, tasks, documents, users, and serial numbers, select the  **Custom Fields** option from the  respective browsers. You will see the **Custom Fields** dialog box.
