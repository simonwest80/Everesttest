# The Stock Bin Profile


Use the **Stock Bin** profile (path:  **Inventory &gt; Stock Areas** >  **Options** > **New**  > **Stock Bin** > **In 
 New Window**) to set up the information required to create a new  stock bin.


**Code**
: Specify the unique code that identifies the stock  bin throughout **Everest**. This can  also be a bin number.


**Description**
: Describe the name of the stock bin.


Bin Type


This is of the following types:

- Stock
- Out



The type **Stock** signifies that  you own the inventory in the stock bin. The **Out**  type denotes that someone else owns the inventory.


![]({{site.sc_baseurl}}/img/lens.gif) [Stock Bin Types]({{site.sc_baseurl}}/options/miscellaneous-set-up/stock-areas/stock_area_types.html)


**Stock Type**
: Helps you to identify the different types of stock,  and thus, facilitates handling. This field is mandatory.


{:.example}
You can store items for exchange in a separate stock bin for easy identification.
: Click [here]({{site.sc_baseurl}}/misc/stock_type_options.html)  for Stock Type options.


**Parent Stock Area**
: Specifies the parent to which the stock bin belongs.


**Location Code**
: Specifies the location to which a stock bin belongs.


{:.note}
Based on the location code that you have defined  in the **Stock Area** profile this  field gets filled when you select Parent Stock Area.


{:.example}
When you create a stock bin that belongs to  a particular region, say California, select California as the location  code.


**Warehouse Position**
: Specifies the position of the stock bin within the  parent stock area. Items are picked based on the position specified. **** This option is enabled only if specify  the **Location Code****.**


{:.note}
Adequate security right is required to add,  modify, activate, and deactivate the bin details.
