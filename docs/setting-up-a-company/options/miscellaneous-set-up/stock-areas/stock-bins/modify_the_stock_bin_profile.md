# Modify the Stock Bin Profile


You can modify the **Stock Bin** profile  from any of the following locations:

- The **Stock 
 Areas** browser - Highlight a record where the Stock Level is "Bin"  and select **Options &gt; Edit &gt; In 
 New Window**.
- The **Stock 
 Bin** profile - Specify the code of the required stock bin.



You cannot modify the following fields in the **Stock****Bin** profile:

- Code
- Bin Type
