# Create a Stock Area


{:.steps}
To create a stock area, do the following:

- From the main  menu bar, select **Inventory &gt; Stock 
 Areas**. You will see the **Stock 
 Areas** browser.
- From the **Stock Areas** browser, select **Options**  > **New &gt; Stock Area &gt; In New 
 Window**. You will see the **Stock 
 Area** profile.



{:.note}
You can also create Stock Areas from the following:

- An existing  **Stock Area** profile by clicking  **Options &gt; New**.
- Any profile/dialog  box that contains the **Stock Area** field  by selecting **New** from the right-click  menu options.
