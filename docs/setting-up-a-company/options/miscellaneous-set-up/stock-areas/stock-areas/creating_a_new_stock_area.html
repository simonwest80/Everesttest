Create a Stock Area

To create a stock area, do the following:

  • From the main menu bar, select Inventory > Stock Areas. You will see the Stock Areas browser.
  • From the Stock Areas browser, select Options > New > Stock Area > In New Window. You will see the Stock Area profile.

You can also create Stock Areas from the following:

  • An existing Stock Area profile by clicking Options > New.
  • Any profile/dialog box that contains the Stock Area field by selecting New from the right-click menu options.
