# Attach a Multimedia File


{:.steps}
To attach a multimedia file, do the following:

- From the [**Multimedia 
 Setup** browser]({{site.sc_baseurl}}/misc/multimedia_setup_browser.html), select **Options**  > **New**. A new record will be  added to the browser.
- Specify the  required file by clicking the **Multimedia 
 File** column and then the lookup button ![]({{site.sc_baseurl}}/img/setup_browse_icon.gif). The other  columns in the browser are automatically updated.
- Save the details.



{:.note}
You can drag and drop multimedia files on  to the multimedia browsers in the Customer, Vendor, User, Item, Task,  and Appointment profiles.
