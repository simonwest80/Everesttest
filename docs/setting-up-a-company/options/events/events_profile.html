The Event Profile

Use the Event profile to view the events defined in Everest.

Event ID
This field denotes the unique system generated code by which the event is recognized.
Description
This field displays the description of the selected event.

Events are grouped under different categories, actions or event groups in Everest. The Category, Action and Event Group fields determine the respective category, action and group to which a specific event belongs. These fields cannot be modified.
