# The Events Browser


The **Events** browser (path: **CRM &gt; Business Activity Monitoring &gt; 
 Setup &gt; Events**) lists all the events that are defined in the  database.


The browser options are as follows:


**Event History**


The **Event History** option in  the **Events** browser displays the  history of events that have occurred in **Everest**.  Selecting this option displays the **Events 
 History** browser.


{:.example}
If you select the **Event****History** option for the ‘Customer  created’ event,  the **Events History** browser is  displayed. This browser displays details of the customers who are created  in the database.


{:.note}
Events are displayed from the date the company was created in **Everest**.


Options common to all browsers in **Everest**  are also available in this browser.
