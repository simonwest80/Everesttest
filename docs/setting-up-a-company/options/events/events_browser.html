The Events Browser

The Events browser (path: CRM > Business Activity Monitoring > Setup > Events) lists all the events that are defined in the database.

The browser options are as follows:

Event History

The Event History option in the Events browser displays the history of events that have occurred in Everest. Selecting this option displays the Events History browser.

If you select the EventHistory option for the ‘Customer created’ event, the Events History browser is displayed. This browser displays details of the customers who are created in the database.

Events are displayed from the date the company was created in Everest.

Options common to all browsers in Everest are also available in this browser.
