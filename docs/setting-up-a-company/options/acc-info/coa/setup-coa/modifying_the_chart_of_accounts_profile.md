# Modify the Chart of Accounts Profile


The **Chart of Accounts** profile  can be modified from any one of the following locations:

- The **Chart 
 of Accounts** browser - Highlight the record and select **Options 
 &gt; Edit &gt; In New Window**.
- The **Chart 
 of Accounts** profile - Specify the code of the required jurisdiction  in the **Code** field.



All fields in the **Currency** profile  can be modified except the code.
