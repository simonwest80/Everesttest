# The Chart of Accounts Profile - Custom Fields Tab


Use the **Custom Fields** tab to  define the additional information for any record as per your requirement.  Custom fields can be defined as character strings, date, logical, memo,  or numeric fields. This **** tab is  displayed only when you save the **Chart 
 of Accounts** profile.
