# The Chart of Accounts Profile - Accounting Tab


The **Accounting** tab displays  the balance in the selected account in the base currency. In case of accounts  that are set up in a foreign currency, you can also view the balance in  terms of that currency. This is merely a display tab and none of the fields  can be edited.


![]({{site.sc_baseurl}}/img/lens.gif) [Balance  and Reconciliation Information]({{site.sc_baseurl}}/options/acc-info/coa/chart-of-accounts-details/balance_and_reconciliation_information.html)


The **Accounting** tab comprises  the following sections:

- [Opening  Balance]({{site.sc_baseurl}}/misc/opening_balance_coa.html)
- [Posted  Balance]({{site.sc_baseurl}}/misc/posted_balance_coa.html)
- [Unposted  Balance]({{site.sc_baseurl}}/misc/unposted_balance_coa.html)
- [Current  Balance]({{site.sc_baseurl}}/misc/current_balance_coa.html)
- [Reconciliation  Details]({{site.sc_baseurl}}/misc/reconciliation_details_coa.html)
