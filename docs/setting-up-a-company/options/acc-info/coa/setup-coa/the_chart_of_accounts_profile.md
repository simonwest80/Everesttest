# The Chart of Accounts Profile


The **Chart of Accounts** profile  contains the following tabs:

- [General]({{site.sc_baseurl}}/options/acc-info/coa/setup-coa/chart_of_accounts_profile_general_tab.html)
- [Accounting]({{site.sc_baseurl}}/options/acc-info/coa/setup-coa/chart_of_accounts_profile_accounting_tab.html)
- [Multicurrency]({{site.sc_baseurl}}/options/acc-info/coa/setup-coa/chart_of_accounts_profile_multi_currency_tab.html)
- [Others]({{site.sc_baseurl}}/options/acc-info/coa/setup-coa/the_chart_of_accounts_profile_others_tab_acc.html)  (Only for Account Type ‘Bank’)
- [Custom  Fields]({{site.sc_baseurl}}/options/acc-info/coa/setup-coa/the_chart_of_accounts_profile_custom_fields.html)



{:.note}
The **Custom 
 Fields** tab does not appear for a new profile and is displayed only  for a saved profile.
