# <font color="#000000" class="hcp1">Bank Account</font>


<font color="#000000" class="hcp1">Use this option to set up an account as a bank account. 
 </font>When payment or receipt journals are created, accounts marked as  bank accounts are displayed in the **Bank**  field.


Setting up an account as a bank account also facilitates reconciliation  of this account with the statement received from the bank. Bank reconciliation  is available from the **Accounting**  menu.


Set up the following accounts as bank accounts:

- All bank accounts
- All cash accounts
- All credit  card clearing accounts



**Last Check Number**
: The number of the last check issued in sequence  from this bank is entered in this field. **Everest**  uses the next number when printing the computerized checks from this account.  This field is enabled only if the **Bank 
 Account** check box has been selected.
: The last check number issued from this bank can  be verified from this field. The check number may be modified when required  for instance, when you obtain a new check book.
