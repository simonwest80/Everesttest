Discontinue a Journal Group

When you discontinue a journal group, the record is not deleted from the database; it is deactivated. A discontinued record can be made available for selection by activating it again.

To discontinue a journal group, do the following:

  • From the main menu bar, select File > Setup > Accounting > Journal Groups. You will see the Journal Groups browser.
  • From the Journal Groups browser, highlight the record.
  • Select Options > Discontinue. You will be prompted for confirmation.
  • Click OK. The selected record is no longer visible in the Journal Groups browser.
