# View and Activate a Discontinued Journal Group


{:.steps}
To view and activate a discontinued journal group, do the  following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Accounting &gt; Journal Groups**. You will see the **Journal 
 Groups** browser.
- From the **Journal Groups** browser, select **Options** > **View Discontinued**. You will see the discontinued record in the  **Discontinued Journal Groups** browser.
- From the **Discontinued** **Journal 
 Groups** browser, highlight the record.
- Select **Options &gt; Activate**. You will be prompted  for confirmation.
- Click **OK**. The activated record will be displayed  in the **Journal Groups** browser.
