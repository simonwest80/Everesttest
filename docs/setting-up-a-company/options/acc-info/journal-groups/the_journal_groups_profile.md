# The Journal Groups Profile


**Code**
: Enter a unique alphanumeric code, using 1 to 15  characters to identify the journal group throughout **Everest**.  This is a mandatory field. Once saved, the code cannot be modified.


**Description**
: Enter an appropriate alphanumeric description for  the code, spanning a maximum of 40 characters. This is a mandatory field.
