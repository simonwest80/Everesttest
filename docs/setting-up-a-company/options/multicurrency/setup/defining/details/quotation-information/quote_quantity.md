# Quote Quantity


Use this option to specify the number of units in which the currency  is generally quoted.


{:.example}
**If the base currency is American Dollars (USD)  and the foreign currency is Indian Rupees (INR) and 100 INR = 2.15 USD,  the quantity for which the rate is being quoted for is 100 units.**
