Quote Type

The quote type indicates the manner in which the conversion rate of a foreign currency is quoted.

Base to Foreign Currency
Select this option, if the conversion rate for the currency is specified in terms of the base currency.

If the base currency is American Dollars (USD) and the foreign currency is British Pounds (GBP), and 1 USD = 0.68 GBP, select Base to Foreign Currency.

Foreign to Base Currency
Select this option, if the conversion rate for the currency is specified in terms of the foreign currency.

If the base currency is American Dollars (USD) and the foreign currency is British Pounds (GBP), and 1 GBP = 1.457 USD, select Foreign to Base Currency.
