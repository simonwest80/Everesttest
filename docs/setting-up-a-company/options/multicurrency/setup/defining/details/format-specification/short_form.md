# Short Form


Enter a shortened name for the currency. You can set up **Everest**  to print or display the short form entered here for all amounts in this  currency.


{:.example}
<font style="color: #000000;" color="#000000">American 
 dollars can be set up with the short form as <font face="Verdana" class="hcp2">'</font>USD<font face="Verdana" class="hcp2">'</font>.</font>
