# Decimals to Display


This field is used to enter the number of decimals (if any) to be included  when amounts in this currency are displayed.


{:.example}
In the case of **Japanese 
 Yen**, the number of decimals displayed will be zero, since the currency  is a whole unit.


{:.note}
You cannot modify this field if the currency  has been used in a transaction.
