# The Currency Profile 


Use the **Currency** profile to  specify the basic details for the currency such as the code and description.  You can also specify the format details, display, and printing and quotation  information.


**Code**
: Enter a unique code for the currency using 1 to  15 characters. This code will be used to identify this currency throughout  **Everest**. This is a mandatory field.  Once saved, this field cannot be modified.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Code  and Description]({{site.sc_baseurl}}/options/multicurrency/setup/defining/details/general-information/code_and_description_currency_profile.html)<font color="#000000" class="hcp3"> </font>


**Description**
: Enter a suitable alphanumeric description for the  currency using 1 to 40 characters. This is a mandatory field.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Code  and Description]({{site.sc_baseurl}}/options/multicurrency/setup/defining/details/general-information/code_and_description_currency_profile.html)<font color="#000000" class="hcp3"> </font>


**Short Form**
: Enter an abbreviation using up to 8 characters,  for the currency name.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Short form]({{site.sc_baseurl}}/options/multicurrency/setup/defining/details/format-specification/short_form.html)


**Symbol**
: Enter a symbol that denotes the currency.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Symbol]({{site.sc_baseurl}}/options/multicurrency/setup/defining/details/format-specification/symbol.html)


**Decimals to Display**
: Enter the number of decimal places to be displayed  for the currency. You can specify a maximum value of 4 in this field.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Decimals  to display]({{site.sc_baseurl}}/options/multicurrency/setup/defining/details/format-specification/decimals_to_display.html)


**Locale to be used to set display format of the  currency**
: Specify the locale or the regional settings to be  used for formatting numbers, currencies, time and date.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Locale]({{site.sc_baseurl}}/options/multicurrency/setup/defining/details/format-specification/locale.html)


**Quote Type**
: Specify whether the currency is quoted in terms  of the base currency or the foreign currency.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Quote Type]({{site.sc_baseurl}}/options/multicurrency/setup/defining/details/quotation-information/quote_type.html)


**Quote Quantity**
: Specify the quantity in which the foreign currency  is quoted. This is a mandatory field. You can enter a maximum of <font color="#000000" class="hcp3">4 numerals </font>in this field.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Quote  quantity]({{site.sc_baseurl}}/options/multicurrency/setup/defining/details/quotation-information/quote_quantity.html)


**Sub Unit**
: Specify the number of subunits that comprise one  unit of the currency.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Sub Unit]({{site.sc_baseurl}}/options/multicurrency/setup/defining/details/quotation-information/sub_unit.html)


**Display Preference**
: Specify the display settings for amounts in this  currency<font color="#000000" class="hcp3">.</font> The drop-down list provides you  with the options **Symbol**, **Short Form** and **None**.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Display  Preferences]({{site.sc_baseurl}}/options/multicurrency/setup/defining/details/print-info/display_preference.html)


**Document Printing Preference**
: Specify the print settings for amounts in this <font color="#000000" class="hcp3">currency.</font> The drop-down menu provides you with  the options **Symbol** and **None**.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Printing  Preferences]({{site.sc_baseurl}}/options/multicurrency/setup/defining/details/print-info/printing_preference.html)
