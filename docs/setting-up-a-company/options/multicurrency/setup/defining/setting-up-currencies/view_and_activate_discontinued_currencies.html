View and Activate Discontinued Currencies

To view and activate a discontinued currency, do the following:

  • From the main menu bar, select File > Setup > Accounting > Multicurrency > Currency. You will see the Currencies browser.
  • From the Currencies browser, select Options > View Discontinued. You will see the discontinued record in the Discontinued Currencies browser.
  • From the Discontinued Currencies browser, highlight the record.
  • Select

    Options > Activate. You will be prompted for confirmation.

  • Click

    OK. The activated record will be displayed in the Currencies browser.
