# <font style="color: #000000;" color="#000000">Set up a Currency</font>


{:.steps}
To define a currency**,**  do the following**:**

- From the main  menu bar, select **File &gt; Setup &gt; 
 Accounting &gt; Multicurrency &gt; Currency**. You will see the **Currencies** browser.
- From the **Currencies** browser, select **Options**  > **New &gt; In New Window**. You  will see the [**Currency**  profile]({{site.sc_baseurl}}/options/multicurrency/setup/defining/setting-up-currencies/profile_currency_general_tab.html).



{:.note}
Currencies may also be set up from any profile/dialog  box that contains the **Currency**  field by selecting **New** from the  right-click menu options.
