The Exchange Rates Browser

The Exchange Rates browser (path: File > Setup > Accounting > Multi-currency > Exchange Rates) displays the exchange rates that are defined for all currencies.

The following option is available in the Exchange Rates browser:

Multiple Rate Entry
The MultipleRate Entry option enables you to enter the exchange rates for a number of currencies at one stretch. This option brings up the Multiple ExchangeRates dialog box

Double-click the **Buy/Sell column of the required currency and enter the rate.**

You can also mention the date on which the exchange rates are quoted.

Options common to all browsers in Everest are also available in this browser.
