# Set up Exchange Rates


Exchange rates can be set up from the **Exchange 
 Rates** browser.


{:.steps}
To set up the exchange rates for a currency, do the following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Accounting &gt; Multi-currency &gt; Exchange Rates**. You will see  the **Exchange Rates** browser.
- From the **Exchange Rates** browser, select **Options** > **New 
 &gt; In New Window/In Current Window**. You will see the [**Exchange Rates** profile]({{site.sc_baseurl}}/options/multicurrency/exchange-rates/set-up-exchange-rates/exchange_rates_profile.html).
