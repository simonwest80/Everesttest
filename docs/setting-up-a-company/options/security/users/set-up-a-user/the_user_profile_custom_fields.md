# The User Profile - Custom Fields


Use the **Custom Fields** tab to  define additional information for any record as per your requirement.  Custom fields can be defined as character strings, or date, logical, memo  or numeric fields.


{:.example}
**In a **User**  profile, you might want to maintain a record of the user's emergency contact  number, or any personal information in the **Memo**  field.**
