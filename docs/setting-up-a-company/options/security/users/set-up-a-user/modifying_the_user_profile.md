# Modify the User Profile


**<font color="#000000" class="hcp2">You can modify the 
 <span class="hcp3">User</span> profile from any one of the 
 following locations:</font>**

- The **Users** browser - Highlight the record and select **Options 
 &gt; Edit &gt; In New Window**.
- The **User** profile - Specify the code of the user in the **Code** field.



**<font color="#000000" class="hcp2">You can modify all 
 fields in the <span class="hcp3">User</span> profile except 
 the <span class="hcp3">Code</span>.</font>**
