# Create a User


{:.steps}
To create a user, do the following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Users &gt; Users**. You will see the **Users** browser.
- From the **Users** browser, select **Options****&gt;** **New 
 &gt; In New Window**. You will see the [**User** profile]({{site.sc_baseurl}}/options/security/users/set-up-a-user/the_user_profile.html).



{:.note}
You can also create a User from the following:

- An existing  **User** profile by clicking **Options &gt; New**.
- Any profile/dialog  box that contains the **User** field  by selecting **New** from the right-click  menu options.
