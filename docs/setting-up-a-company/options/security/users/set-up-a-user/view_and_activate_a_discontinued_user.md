# View and Activate a Discontinued User


{:.steps}
To view and activate a discontinued user, do the following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Users &gt; Users**. You will see the **Users** browser.
- From the **Users** browser, select **Options** > **View Discontinued**.  You will see the discontinued user in the **Discontinued 
 Users** browser.
- F<font color="#000000" class="hcp2">rom the </font>**Discontinued**<font color="#000000" class="hcp2"> </font>**Users** <font color="#000000" class="hcp2">browser, highlight the user.</font>
- <font color="#000000" class="hcp2">Select </font>**Options &gt; Activate**<font color="#000000" class="hcp2">. 
 You will be prompted for confirmation.</font>
- <font color="#000000" class="hcp2">Click </font>**OK**<font color="#000000" class="hcp2">. The activated user 
 will be displayed in the </font>**Users** <font color="#000000" class="hcp2">browser. </font>
