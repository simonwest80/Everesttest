# View and Activate a Discontinued Group


{:.steps}
To view and activate a discontinued group, do the following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Users &gt; Groups**. You will see the **Groups**  browser.
- From the **Groups** browser, select **Options** > **View Discontinued**.  You will see the discontinued group in the **Discontinued 
 Groups** browser.
- F<font color="#000000" class="hcp2">rom the </font>**Discontinued**<font color="#000000" class="hcp2"> </font>**Groups**<font color="#000000" class="hcp2">, highlight 
 the group.</font>
- <font color="#000000" class="hcp2">Select </font>**Options &gt; Activate**<font color="#000000" class="hcp2">. 
 You will be prompted for confirmation.</font>
- <font color="#000000" class="hcp2">Click </font>**OK**<font color="#000000" class="hcp2">. The activated group 
 will be displayed in the </font>**Groups**<font color="#000000" class="hcp2"> browser. </font>
