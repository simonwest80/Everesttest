Create a User Group

You can create a new user group from any of the following locations:

  • The Groups browser by clicking Options > New > In New Window.
  • An existing Group profile by clicking Options > New.

To create a group, do the following:

  • From the main menu bar, select File > Setup > Users > Groups. You will see the Groups browser.
  • From the Groups browser select Options > New > In New Window. You will see the Group profile.

You can also create a user group from an existing Group profile by clicking Options > New.
