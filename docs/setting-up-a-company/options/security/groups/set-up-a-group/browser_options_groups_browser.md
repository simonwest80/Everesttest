# The Groups Browser


Access the **Groups** browser from  **File &gt; Setup &gt; Users &gt; Groups**.  The options available in the **Groups** browser  are detailed below:

- [List  of users]({{site.sc_baseurl}}/misc/list_of_users.html)
- [Rights]({{site.sc_baseurl}}/misc/rights_groups.html)
- [Events  History]({{site.sc_baseurl}}/misc/events_history.html)



Options common to all browsers in **Everest**  are also available in this browser.
