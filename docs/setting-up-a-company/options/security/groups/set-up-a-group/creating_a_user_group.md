# Create a User Group


You can create a new user group from any of the following locations:

- The **Groups** browser by clicking **Options &gt; 
 New &gt; In New Window**.
- An existing  **Group** profile by clicking **Options &gt; New**.



{:.steps}
To create a group, do the following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Users &gt; Groups**. You will see the **Groups**  browser.
- From the **Groups** browser select **Options**  > **New &gt; In New Window**. You  will see the [**Group** profile]({{site.sc_baseurl}}/options/security/groups/set-up-a-group/group_profile_general_tab.html).



{:.note}
You can also create a user group from an existing **Group**  profile by clicking **Options &gt; New**.
