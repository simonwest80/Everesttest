View and Activate a Discontinued Group

To view and activate a discontinued group, do the following:

  • From the main menu bar, select File > Setup > Users > Groups. You will see the Groups browser.
  • From the Groups browser, select Options > View Discontinued. You will see the discontinued group in the Discontinued Groups browser.
  • From the Discontinued Groups, highlight the group.
  • Select

    Options > Activate. You will be prompted for confirmation.

  • Click

    OK. The activated group will be displayed in the Groups browser.
