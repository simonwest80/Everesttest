# Assigning Rights to Groups - Security Rights Browser


Use the **Security Rights** browser  to specify the groups that must have a particular right or view the groups  assigned to an individual security right. You can assign a right to a  number of groups at the same time. You can also assign security rights  to users and view the users assigned to a specific security right from  this browser.
