Assigning Rights to Users from the Users Browser

You can only assign rights to one user at a time from the Users browser.

To grant security rights to two users 'Andy Kater' and 'Bill Sneider',

select 'Andy Kater' in the Users browser and assign rights by selecting Options> Rights > User Rights.

Save the details and return to the Users browser.

Similarly, assign rights to 'Bill Sneider'.

You can use this option to review the rights granted to a user.
