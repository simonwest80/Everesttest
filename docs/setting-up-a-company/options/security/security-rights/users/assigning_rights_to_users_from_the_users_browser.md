# Assigning Rights to Users from the Users Browser


You can only assign rights to one user at a time from the **Users**  browser.


{:.example}
To grant security rights to two users <font face="Verdana" class="hcp2">'</font>Andy  Kater<font face="Verdana" class="hcp2">'</font> and <font face="Verdana" class="hcp2">'</font>Bill Sneider<font face="Verdana" class="hcp2">',</font>


select <font face="Verdana" class="hcp2">'</font>Andy Kater<font face="Verdana" class="hcp2">'</font> in the **Users**  browser and assign rights by selecting **Options****&gt; Rights &gt; User Rights**.


Save the details and return to the **Users**  browser.


Similarly, assign rights to <font face="Verdana" class="hcp2">'</font>Bill Sneider<font face="Verdana" class="hcp2">'</font>.


You can use this option to review the rights granted to a user.
