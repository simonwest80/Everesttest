# The Job Costing Profile - Status


Use the **Status** tab to specify  the estimated income and revenue from the job. You can also specify whether  the job is open for transactions or is closed.


**Status**
: Specify whether the job has an 'Open' or 'Close'  status.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Status  Information]({{site.sc_baseurl}}/options/job-costing/job-costing-details/status_information.html)


**Estimated Revenue**
: Enter the approximate revenue expected from the  project being defined.


{:.note}
The number of decimals displayed in the **Estimated 
 Revenue** and **Estimated Cost** fields  depends on the following specifications:

- **Currency** profile - **File &gt; Setup &gt; 
 Accounting &gt; Multi-Currency &gt; Currency**.
- **Regional 
 Settings Properties** dialog box â€“ **Operating 
 Systems &gt; Regional Options &gt; Currency** tab.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Revenues  and due date information]({{site.sc_baseurl}}/options/job-costing/job-costing-details/revenues_and_due_date_information.html)


**Estimated Cost**
: Enter the expected cost for this project. You can  enter a maximum of 15 integers and 4 decimals in the **Estimated 
 Cost** field.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Revenues  and due date information]({{site.sc_baseurl}}/options/job-costing/job-costing-details/revenues_and_due_date_information.html)


**Start Date**
: Enter the estimated start date of the project.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Revenues  and due date information]({{site.sc_baseurl}}/options/job-costing/job-costing-details/revenues_and_due_date_information.html)


**End Date**
: Enter the estimated end date of the project.
