# Create a Job Code


{:.steps}
To set up a new job code, do the following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Miscellaneous &gt; Job Explorer**. You will see the **Job 
 Explorer**.
- From the **Job Explorer** select **Options**  > **New &gt; In New Window**. You  will see the [**Job 
 Costing** profile]({{site.sc_baseurl}}/options/job-costing/set-up-a-job-code/the_job_costing_profile.html).



{:.note}
You can also create a job code from any of the following:

- An existing  **Job Costing** profile by clicking  **Options &gt; New**.
- The Job Code  column of any purchase or sales document. Select **New**  from the right-click menu options.
