Create a Job Code

To set up a new job code, do the following:

  • From the main menu bar, select File > Setup > Miscellaneous > Job Explorer. You will see the Job Explorer.
  • From the Job Explorer select Options > New > In New Window. You will see the Job Costing profile.

You can also create a job code from any of the following:

  • An existing Job Costing profile by clicking Options > New.
  • The Job Code column of any purchase or sales document. Select New from the right-click menu options.
