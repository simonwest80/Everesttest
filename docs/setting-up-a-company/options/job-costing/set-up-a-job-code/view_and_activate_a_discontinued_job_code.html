View and Activate a Discontinued Job Code

To view and activate a discontinued job code, do the following:

  • From the main menu bar, select File > Setup > Miscellaneous > Job Explorer. You will see the Job Explorer.
  • From the Job Explorer, select Options > View Discontinued. You will see the discontinued record in the Discontinued Job Explorer browser.
  • From the Discontinued Job Explorer browser, highlight the record.
  • Select

    Options > Activate. You will be prompted for confirmation.

  • Click

    OK. The activated record will be displayed in the Job Explorer.
