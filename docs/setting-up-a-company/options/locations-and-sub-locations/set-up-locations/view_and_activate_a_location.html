View and Activate a Location/Sub-Location

To view and activate a discontinued location/sub-location, do the following:

  • From the main menu bar, select File > Setup > Accounting > Location/Sub-Location. You will see the Location/sub-locations browser.
  • From the Location/Sub-Locations browser, select Options > View Discontinued. You will see the discontinued record in the Discontinued Locations/Sub-Locations browser.
  • From the Discontinued Locations/Sub-Locations browser, highlight the record.
  • Select Options > Activate. You will be prompted for confirmation.
  • Click OK. The activated record will be displayed in the Locations/Sub-Locations browser.
