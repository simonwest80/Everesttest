The Location Profile - Stock Transfer

If the preference option Allow users to access stock bins of – Logged in location is selected in the Other Flow Control dialog box (path: File > Setup > Preferences > Flow control > Other Flow control) and the default stock bin is of non-logged in location security right will be checked at the time of attaching or assigning the quantity in the documents. Also, you have to select only those stock bins that belong to the login location/sub-location in the following sub-tabs:

  • Purchase Document
  • Sales Document
  • Assembly 1
  • Assembly 2
  • Inventory Adjustment

You can specify a stock area’s location in the Location Code field of the Stock Area profile (path: Inventory > Stock Areas > Options > Edit > In New Window, if the Stock Level is “Stock Area”) or in Location Code field of the Stock Bin profile (path: Inventory > Stock Areas > Options > Edit > In New Window, if the Stock Level is “Bin”.

If you do not have security right, message for changing the default stock bin will appear.

The Stock Transfer tab has the following additional option:

  • Miscellaneous
