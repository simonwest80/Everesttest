# Modify the Location Profile


You can modify a **Location** profile  from any one of the following locations:

- The **Locations/Sub-Locations** browser **-** Highlight the  record in the browser and select **Options 
 &gt; Edit &gt; In New Window**.
- The **Location** profile - Specify the code of the required location in the **Code** field.



{:.note}
**All fields in the **Location**  profile can be modified except the code.**
