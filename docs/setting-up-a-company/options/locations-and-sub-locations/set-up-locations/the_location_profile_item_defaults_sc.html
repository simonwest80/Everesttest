The Location profile - Item Defaults

Use the Location Profile – Item Defaults tab to configure the frequently used default values that will be displayed by default, when you create a new item.  This enables you to avoid configuring all the fields in Item, profiles every time, when you create a new item. The settings configured here will replicate in the Item profile and are editable.

Create a Location

Modify the Location Profile

: The Default settings configured for one location are not applicable to another.

Company default location
If your company’s default location is, for example ‘Virginia’, then, the Company default location radio button will display its label as Company default location: Virginia. If you select this option, the Item profile will display the Item defaults configured in the default location of your company. In Location Profile, if you select the default location of your company, then this radio button will be disabled and This location button will be selected and enabled.
This location
Select this option to configure Item Defaults for the selected location in Location Profile. If you select this option, then the Item Profile will display the Item Defaults configured in the selected location. If you select This location radio button, you will see the following six sub-tabs:
  • General
  •  Inventory
  • Sales
  • Purchases
  • Posting Group
  • E-Commerce
