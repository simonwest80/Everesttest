# The Location Profile - General


Use the **General** tab to specify  the basic information about a location such as the name address and telephone  number.


**Code**
: Enter a unique code for the location using 1 to  15 characters. This is a mandatory field. Once saved, this field cannot  be modified.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Code]({{site.sc_baseurl}}/options/locations-and-sub-locations/location-details/general-information/code_location_profile.html)<font color="#000000" class="hcp3"> </font>


**Description**
: Enter an appropriate alphanumeric description for  the location using 1 to 40 characters. This is a mandatory field.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Description]({{site.sc_baseurl}}/options/locations-and-sub-locations/location-details/general-information/description_location_profile.html)<font color="#000000" class="hcp3"> </font>


Apply mask


Use this option to apply the telephone, fax and mobile number formats  that are specified for the country.


**Telephone**
: Select this check box if the location’s telephone  number format must correspond to the format as defined in the **Country** profile.


**Mobile/Cell**
: Select this check box if the location’s mobile/cell  number format must correspond to the format as defined in the **Country**  profile.


**Zip/Postal Code**
: Select this check box if the location’s zip/postal  code format must correspond to the format defined in the [Country]({{site.sc_baseurl}}/options/international-set-up/country/profile/the_country_profile.html)  profile.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Apply mask]({{site.sc_baseurl}}/options/locations-and-sub-locations/location-details/general-information/apply_mask.html)


**Company**
: Enter the name of the company.


**Address**
: Enter the address of the location.


**City**
: Enter the city where the company is located.


{:.note}
This field is automatically populated with  the name of the city, if the country selected is United States and the  zip/postal code is also specified.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Address,  City, State, Zip and Country]({{site.sc_baseurl}}/options/locations-and-sub-locations/location-details/general-information/address_city_state_zip_and_country.html)


**State/Province**
: Enter the location’s state/province. This field  is mandatory if the country selected is the United States.


{:.note}
If the country selected is United States,  selecting a zip/postal code will automatically populate this field.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Address,  City, State, Zip and Country]({{site.sc_baseurl}}/options/locations-and-sub-locations/location-details/general-information/address_city_state_zip_and_country.html)


**Zip/Postal Code**
: Specify the location's zip/postal code. This field  is mandatory if the country selected is the United States.


{:.note}
If the country specified is United States:

- You can specify  (create/edit, select) a zip code.
- The US Fips  button is displayed.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Address,  City, State, Zip and Country]({{site.sc_baseurl}}/options/locations-and-sub-locations/location-details/general-information/address_city_state_zip_and_country.html)


**Country**
: Specify the country where you want to set up the  location.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Address,  City, State, Zip and Country]({{site.sc_baseurl}}/options/locations-and-sub-locations/location-details/general-information/address_city_state_zip_and_country.html)


**Telephone - 1, Extension**
: Enter the telephone number and extension.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Telephone]({{site.sc_baseurl}}/options/locations-and-sub-locations/location-details/general-information/telephone_.html)


**Telephone - 2, Extension**
: Enter the additional telephone number and extension.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Telephone]({{site.sc_baseurl}}/options/locations-and-sub-locations/location-details/general-information/telephone_.html)


**Fax, Extension**
: Enter the fax number and extension.


**Mobile/Cell**
: Enter the mobile or cell phone number for the location.


**Web URL**
: Enter the Web URL of the location’s home page.


**Telephone Code**
: This field is automatically updated with the international  dialing code for the country selected. This is a display field and cannot  be edited.


**Abbreviation**
: The International Organization for Standards has  published a standard for abbreviations of countries. This field is automatically  updated with the abbreviation of the country selected as per the ISO 3166  standard. This is a display field and cannot be edited.
