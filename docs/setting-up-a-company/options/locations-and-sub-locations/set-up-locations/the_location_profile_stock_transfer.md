# The Location Profile - Stock Transfer


If the preference option **Allow users 
 to access stock bins of – Logged in location** is selected in the  **Other 
 Flow Control** dialog box (path: **File**  > **Setup** > **Preferences**  > **Flow control** > **Other 
 Flow control**) and the default stock bin is of non-logged in location  security right will be checked at the time of attaching or assigning the  quantity in the documents. Also, you have to select only those stock bins  that belong to the login location/sub-location in the following sub-tabs:

- [Purchase  Document]({{site.sc_baseurl}}/misc/stock_transfer_purchase_document.html)
- [Sales  Document]({{site.sc_baseurl}}/misc/stock_transfer_sales_document.html)
- [Assembly  1]({{site.sc_baseurl}}/misc/stock_transfer_assembly_1.html)
- [Assembly  2]({{site.sc_baseurl}}/misc/stock_transfer_assembly_2.html)
- [Inventory  Adjustment]({{site.sc_baseurl}}/misc/stock_transfer_inventory_adjustment.html)



{:.hint}
You can specify a stock area’s location in  the **Location Code** field of the  **Stock 
 Area** profile (path: **Inventory 
 &gt; Stock Areas &gt; Options &gt; Edit &gt; In New Window**, if  the Stock Level is "Stock Area") or in **Location 
 Code** field of the **Stock 
 Bin** profile (path: **Inventory 
 &gt; Stock Areas &gt; Options &gt; Edit &gt; In New Window**, if  the Stock Level is "Bin".


{:.note}
If you do not have security right, message  for changing the default stock bin will appear.


The **Stock** **Transfer**  tab has the following additional option:

- [Miscellaneous]({{site.sc_baseurl}}/misc/stock_transfer_miscellaneous.html)
