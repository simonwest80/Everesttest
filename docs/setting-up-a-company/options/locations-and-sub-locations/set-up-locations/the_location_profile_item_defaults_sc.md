# The Location profile - Item Defaults


Use the **Location Profile – Item Defaults**  tab to configure the frequently used default values that will be displayed  by default, when you create a new item.  This  enables you to avoid configuring all the fields in Item, profiles every  time, when you create a new item. The settings configured here will replicate  in the **Item profile** and are editable.


![]({{site.sc_baseurl}}/img/lens.gif)[Create a Location]({{site.sc_baseurl}}/options/locations-and-sub-locations/set-up-locations/creating_a_location.html)


![]({{site.sc_baseurl}}/img/lens.gif)[Modify the Location  Profile]({{site.sc_baseurl}}/options/locations-and-sub-locations/set-up-locations/modifying_a_department.html)


{:.note}
: The Default settings configured for one location  are not applicable to another.


**Company default location**
: If your company’s default location is, for example  ‘Virginia’, then, the **Company default 
 location** radio button will display its label as **Company 
 default location: Virginia**. If you select this option, the **Item profile** will display the **Item defaults** configured in the default  location of your company. In **Location 
 Profile**, if you select the default location of your company, then  this radio button will be disabled and **This 
 location** button will be selected and enabled.


**This location**
: Select this option to configure **Item 
 Defaults** for the selected location in **Location**  Profile. If you select this option, then the **Item 
 Profile** will display the **Item 
 Defaults** configured in the selected location. If you select **This location** radio button, you will  see the following six sub-tabs:

- [General]({{site.sc_baseurl}}/misc/item_defaults_general_tab_sc.html)
- [ Inventory]({{site.sc_baseurl}}/misc/item_defaults_inventory_sc.html)
- [Sales]({{site.sc_baseurl}}/misc/item_defaults_sales_sc.html)
- [Purchases]({{site.sc_baseurl}}/misc/item_defaults_purchases_tab_sc.html)
- [Posting  Group]({{site.sc_baseurl}}/misc/item_defaults_posting_group_sc.html)
- [E-Commerce]({{site.sc_baseurl}}/misc/item_defaults_-_e-commerce.html)
