Create Sub-Locations

A sub-location can be created from the Locations/Sub-Locations browser.

To create a sub-location, do the following:

  • From the main menu bar, select File > Setup > Accounting > Location/Sub-Location. You will see the Locations/Sub-Locations browser.
  • From the Locations/Sub-Locations browser, select Options > New > Sub-Location > In New Window. You will see the Location/Sub-Location profile.
