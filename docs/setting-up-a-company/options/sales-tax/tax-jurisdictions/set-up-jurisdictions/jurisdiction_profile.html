The Jurisdiction Profile

Use this profile to set up the information required to define a jurisdiction.

Code
Enter a unique code using 1 to 15 characters for the jurisdiction being defined. This is a mandatory field. Once saved, this field cannot be modified.
Code and Name
Type
This field is used to indicate whether you are creating an 'Out' or 'In' type of jurisdiction. This is a mandatory field. Once saved, this field cannot be modified.
Type of Jurisdiction
Name
Enter a suitable name for the tax jurisdiction. This is a mandatory field.
Code and Name
Country
Specify the country for which the jurisdiction is being created. By default this field displays the country specified in the Location profile (path: File > Setup > Accounting > Location/Sub-Location > Options > Edit > In New Window) of the login location or the parent location of the login sub-location.
Location Information
State/Province
Enter the name of the state or province in which the jurisdiction being defined is located. This is a mandatory field if the country selected in the previous field is the United States.
Location Information
Tax
Specify the tax code to be associated with the particular jurisdiction. This is a mandatory field.
Tax code
Starting FIPS
Specify the starting FIPS codes if the jurisdiction is being set up for a state/province in the United States.
The Starting and Ending FIPS fields are mandatory if United States is selected in the Country field and In State/ Country is selected in the Type field. These fields are not enabled for an Out type of jurisdiction, or when a country other than the United States is selected in the Country field.
FIPS codes
Ending FIPS
Specify the ending FIPS codes if the jurisdiction is being set up for a state/province in the United States.
FIPS codes
