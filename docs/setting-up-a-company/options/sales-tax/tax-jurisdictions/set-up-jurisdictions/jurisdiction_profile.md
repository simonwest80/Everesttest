# The Jurisdiction Profile


Use this profile to set up the information required to define a jurisdiction.


**Code**
: Enter a unique code using 1 to 15 characters for  the jurisdiction being defined. This is a mandatory field. Once saved,  this field cannot be modified.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Code  and Name]({{site.sc_baseurl}}/options/sales-tax/tax-jurisdictions/jurisdiction-details/code_and_name.html)


**Type**
: This field is used to indicate whether you are creating  an <font face="Verdana" class="hcp2">'</font>Out<font face="Verdana" class="hcp2">'</font> or <font face="Verdana" class="hcp2">'</font>In<font face="Verdana" class="hcp2">'</font> type of jurisdiction.  This is a mandatory field. Once saved, this field cannot be modified.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Type  of Jurisdiction]({{site.sc_baseurl}}/options/sales-tax/tax-jurisdictions/jurisdiction-details/type_of_jurisdiction.html)


**Name**
: Enter a suitable name for the tax jurisdiction.  This is a mandatory field.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Code  and Name]({{site.sc_baseurl}}/options/sales-tax/tax-jurisdictions/jurisdiction-details/code_and_name.html)


**Country**
: Specify the country for which the jurisdiction is  being created. By default this field displays the country specified in  the **Location** profile (path: **File** > **Setup**  > **Accounting** > **Location**/**Sub-Location** **&gt; 
 Options &gt; Edit &gt; In New Window**) of the login location or  the parent location of the login sub-location.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Location  Information]({{site.sc_baseurl}}/options/sales-tax/tax-jurisdictions/jurisdiction-details/location_information.html)


**State/Province**
: Enter the name of the state or province in which  the jurisdiction being defined is located. This is a mandatory field if  the country selected in the previous field is the United States.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Location  Information]({{site.sc_baseurl}}/options/sales-tax/tax-jurisdictions/jurisdiction-details/location_information.html)


**Tax**
: Specify the tax code to be associated with the particular  jurisdiction. This is a mandatory field.
: ![]({{site.sc_baseurl}}/img/lens.gif) [Tax code]({{site.sc_baseurl}}/options/sales-tax/tax-jurisdictions/jurisdiction-details/tax_code.html)


**Starting FIPS**
: Specify the starting FIPS codes if the jurisdiction  is being set up for a state/province in the United States.


{:.note}
The **Starting** and **Ending FIPS**  fields are mandatory if United States is selected in the **Country**  field and **In State/ Country** is  selected in the **Type** field. These  fields are not enabled for an **Out**  type of jurisdiction, or when a country other than the United States is  selected in the **Country** field.
: ![]({{site.sc_baseurl}}/img/lens.gif) [FIPS codes]({{site.sc_baseurl}}/options/sales-tax/tax-jurisdictions/jurisdiction-details/fips_codes.html)


**Ending FIPS**
: Specify the ending FIPS codes if the jurisdiction  is being set up for a state/province in the United States.
: ![]({{site.sc_baseurl}}/img/lens.gif) [FIPS codes]({{site.sc_baseurl}}/options/sales-tax/tax-jurisdictions/jurisdiction-details/fips_codes.html)
