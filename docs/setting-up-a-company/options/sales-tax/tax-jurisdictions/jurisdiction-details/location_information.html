Location Information

Country
The Country field displays the country for which you are creating a tax jurisdiction. If the country selected is the United States, the State/Province field is automatically populated when you specify the starting FIPS code.
By default, the country specified in the Location profile (path: File > Setup> Accounting> Location/Sub-Location >Options > Edit > In New window) of the login location/sub-location is displayed in this field.
State/Province
For the United States, this field is used to enter the two-character state abbreviation for a state in the United States. If the jurisdiction is located outside the United States, specify the name of the state or province in this field.
