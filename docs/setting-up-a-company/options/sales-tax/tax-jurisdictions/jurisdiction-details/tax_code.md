# Tax Code


The tax code to be associated with the jurisdiction is specified in  this field.


{:.note}
**Any changes made to this tax code will affect  all documents pertaining to the particular jurisdiction. When you change  the tax code, **Everest** asks you  for confirmation of the same.**
