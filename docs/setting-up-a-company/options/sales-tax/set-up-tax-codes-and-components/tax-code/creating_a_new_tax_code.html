Create a Tax Code

Before you create tax codes, ensure that you have created all required tax components to be associated with it. This section describes how to create a tax code.

To create a tax code, do the following:

  • From the main menu bar, select File > Setup > Accounting > Taxes > Tax Codes. You will see the Tax Codes browser.
  • From the Tax Codes browser, select Options > New. You will see the Tax Code profile.

You can also create a Tax Code from the following:

  • An existing Tax Code profile by clicking Options > New.
  • Any profile/dialog box that contains the Tax Code field by selecting New from the right-click menu options.
