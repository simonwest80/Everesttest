# The Tax Codes Browser 


The **Tax Codes** browser (path:  **File &gt; Setup &gt; Accounting &gt; 
 Taxes &gt; Tax Codes**) displays the tax codes that are created.


Options common to all browsers in **Everest**  are available in this browser.
