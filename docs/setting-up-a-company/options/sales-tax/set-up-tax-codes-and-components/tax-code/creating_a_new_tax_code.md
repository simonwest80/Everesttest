# Create a Tax Code


Before you create tax codes, ensure that you have created all required  tax components to be associated with it. This section describes how to  create a tax code.


{:.steps}
To create a tax code, do the following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Accounting &gt; Taxes &gt; Tax Codes**. You will see the **Tax 
 Codes** browser.
- From the **Tax Codes** browser, select **Options 
 &gt; New**. You will see the [**Tax Code** profile]({{site.sc_baseurl}}/options/sales-tax/set-up-tax-codes-and-components/tax-code/tax_code_profile.html).



{:.note}
You can also create a Tax Code from the following:

- An existing  **Tax Code** profile by clicking **Options &gt; New**.
- Any profile/dialog  box that contains the **Tax Code**  field by selecting **New** from the  right-click menu options.
