Assigning Tax Components to a Tax Code

Tax Components can be assigned/unassigned to the tax code by clicking the Add Component and Remove Component icons. You can view the Tax Components profile by selecting Options > Tax Component from the menu.

The **Tax Code profile comprises two sections:**

  • Assigned
  • Unassigned

Depending upon the component selected, a new or an existing profile is displayed. You can also right-click the sections and choose the Tax Component option to display the profile.
