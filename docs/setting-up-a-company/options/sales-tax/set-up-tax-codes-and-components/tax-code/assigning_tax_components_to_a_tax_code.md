# Assigning Tax Components to a Tax Code


Tax Components can be assigned/unassigned to the tax code by clicking  the **Add Component** ![]({{site.sc_baseurl}}/img/setup_move_left.gif) and **Remove Component**![]({{site.sc_baseurl}}/img/setup_move_right.gif) icons. You can view the **Tax 
 Components** profile by selecting **Options 
 &gt; Tax Component** from the menu.


**The **Tax 
 Code** profile comprises two sections:**

- [Assigned]({{site.sc_baseurl}}/misc/assigned_section_tax_code_profile.html)
- [Unassigned]({{site.sc_baseurl}}/misc/unassigned_section_tax_code_profile.html)



{:.note}
Depending upon the component selected, a new  or an existing profile is displayed. You can also right-click the sections  and choose the **Tax Component** option  to display the profile.
