# Modify the Customer Class Profile


The **Customer** **Class**  profile can be modified from any of the following locations:

- The **Customer****Classes** browser - Highlight the  record and select **Options &gt; Edit &gt; 
 In New Window**.
- The **Customer****Class** profile - Specify the code  of the customer class in the **Code**  field.



{:.note}
Only the **Description** field can be modified in the profile.
