Create a vendor class

To create a vendor class, do the following:

  • From the main menu bar, select File > Setup > Accounting > Class > Vendor Class. You will see the Vendor Classes browser.
  • From the Vendor Classes browser, select Options > New > In New Window. You will see the Vendor Class profile.

You can also create a vendor class from an existing VendorClass profile by clicking Options > New.
