Modify the Zip Code Profile

You can modify the Zip Code profile from any of the following locations:

  • The Zip Codes browser - Highlight the record and select Options > Edit > In New Window.
  • The Zip Code profile - Specify the code of the zip code in the Code field.

All fields in the Zip Code profile can be modified except the Country and FIPS Code/Jurisdiction Locator.

Since the FIPS code is a combination of the state, county and city codes, it gets automatically modified when you change any of these.

You can modify zip codes only in an **Extended Mode setting. (path:File > Setup > Preferences > Workstation Settings > Company Settings tab)**
