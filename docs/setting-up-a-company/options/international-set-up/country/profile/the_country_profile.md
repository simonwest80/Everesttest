# The Country Profile


The **Country** profile comprises  the following tabs:

- [General]({{site.sc_baseurl}}/options/international-set-up/country/profile/the_country_profile_general.html)
- [Format  Address]({{site.sc_baseurl}}/options/international-set-up/country/profile/the_country_profile_format_address.html)



You can restore the address, zip/postal code, telephone or mobile formats  to system defaults by using the option **Restore 
 Default Formats** from the **Country** profile. When you select this option, you will see the **Restore Default Formats**  dialog box.
