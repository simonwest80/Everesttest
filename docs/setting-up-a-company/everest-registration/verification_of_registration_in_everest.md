# Verification of Registration in Everest


Each time the user logs into **Everest**,  the following checks are done:

1. Validation  and registration codes are verified for validity.
- Expiry  date, in case of evaluation, and rental copies are validated with the  log in date.



{:.note}
If the login  date comes after the date of expiration, **Everest**  does not allow the user to log in.

1. If the number  of users logged in (including the current user) is greater than the user  license, **Everest** does not allow  the user to log in.
- Activation  keys for the add-on options are verified and enabled only if the valid  key codes are provided.
