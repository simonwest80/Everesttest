# Company Setup Profile - Custom Fields Tab


Use the **Custom** **Fields**  tab to define additional information for any record. Custom fields can  be defined as character strings, dates, logicals,  memo or numerics fields. <font style="color: #000000;" color="#000000">This<span class="hcp1"> 
 </span>tab is displayed only when you save the profile.</font>


{:.example}
In the **Characters** tab, you can  specify the nature of the business or the type of incorporation.


In the **Dates** tab, you can specify  the date of incorporation or the annual day.


In the **Logicals**tab, you can specify whether the company is listed or not.


In the **Numerics** tab, you can  specify the number of employees or the annual revenue.
