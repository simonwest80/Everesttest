# Company Setup - Activation Tab


In this tab, you can execute the following  functions:

- Register a  company.



![]({{site.sc_baseurl}}/img/lens.gif) [Everest  Registration Details]({{site.sc_baseurl}}/everest-registration/everest_registration_dialog_box.html)

- Modify the  registration details.



{:.note}
**The country code and company name fields cannot  be modified.**


The **Activation** tab contains  the following subtabs:

- [Registration]({{site.sc_baseurl}}/misc/everest_registration_registration_tab.html)
- [Options]({{site.sc_baseurl}}/misc/everest_registration_option_activation_tab.html)
