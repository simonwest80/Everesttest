# Company Setup


The **Company** **Setup**  profile displays the details specified in the **Company****Creation** **Wizard**.


{:.steps}
To access the Company Setup profile, do the following:

- From the main  menu bar, select **File &gt; Setup &gt; 
 Company**. You will see the [**Company Setup** profile]({{site.sc_baseurl}}/company-set-up/the_company_setup_profile.html).
