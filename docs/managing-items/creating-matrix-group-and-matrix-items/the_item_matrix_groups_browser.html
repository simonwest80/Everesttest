The Item Matrix Groups Browser

The Item Matrix Groups browser (path: Inventory > Item Matrix > Matrix Groups) lists all the matrix groups that have been defined.

The options available in this browser (path: Item Matrix Groups browser > Options) are:

  • View Items
  • View Matrix

Standard options common to all browsers are also available in this browser.

Standard Browser Options

To create an item matrix group from the Matrix Groups browser, do the following:

  • From the main menu bar, select Inventory > Item Matrix > Matrix Groups. You will see the ItemMatrix Groups browser.
  • From the Item Matrix Groups browser, select Options > New. You will see the Item Matrix Wizard.
