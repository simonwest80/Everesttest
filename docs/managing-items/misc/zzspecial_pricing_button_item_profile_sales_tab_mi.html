Click this button to specify the different pricing schemes for the item in the Special Pricing dialog box. This button is enabled when you save the profile. You can set up the following pricing schemes for the item from this dialog box:

  • Promotional Pricing – On the Promotional Pricing tab, select Options > New > In New Window to create a pricing scheme in the Promotional Pricing dialog box.
  • Special Pricing – On the Special Pricing tab, select Options > New > In New Window to create a pricing scheme in the Special Pricing dialog box.
  • Quantity Discount – On the Quantity Discount tab, select Options > New > In New Window to create a pricing scheme in the Quantity Discount dialog box.
