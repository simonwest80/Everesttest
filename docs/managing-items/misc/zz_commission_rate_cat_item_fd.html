Enter the rate, if any, at which the sales representative has to be paid commission when an item under this category is sold. If a commission rate is specified in the Commission Rate % field of the Item profile (path: Inventory > Items > double-click on required item from the Items browser > Sales tab) for an item belonging to the category, the rate specified in the Item profile is utilized. If the rate is not specified in the Item profile, commission is payable to all items belonging to the category being defined, at the rate specified here.
