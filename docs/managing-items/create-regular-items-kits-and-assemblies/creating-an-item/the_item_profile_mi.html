The Item Profile

The Item profile consists of the following tabs:

  • General
  • Details (if “Item Type” is “Kit”)
  • Inventory
  • Sales
  • Purchases
  • Replenishment
  • Accounts
  • Notes
  • Assembly (if “Item Type” is “Assembly”)
  • Add-ons
  • Related Items
  • E-Commerce
  • Multimedia
  • Custom Fields
