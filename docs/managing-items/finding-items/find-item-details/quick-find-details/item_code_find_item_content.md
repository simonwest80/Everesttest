# Item Code


Search for an item with a specific item code. You can enter partial  values here, if you do not know the entire code. The search displays the  list of items whose codes begin with the partial value(s)  specified.


{:.example}
If you know that the item code begins with  ‘1’, type 1 in this field. When you click the **Find** button, all items whose codes begin with ‘1’ are displayed in the  search result.


Click the drop-down button to view the history of prior search parameter  values.


{:.example}
If you had selected DIA10GCHAIN in your prior  search, **Everest** would have maintained  a history of that search. In your next search, the drop-down list for  the field **Item Code** displays DIA10GCHAIN.


You can also select multiple items in this field. If you select multiple  item codes the word 'multiple' is displayed in this field in a URL link  format.


{:.steps}
To select multiple item codes, do the following:

- Click the browse  icon ![mi_browse_icon.gif]({{site.mi_baseurl}}/img/mi_browse_icon.gif) beside the **Item 
 Code** field. You will see the **Items** browser.
- From the **Items** browser, select the required items  and click **OK**.
- The **Selection**  dialog box is displayed. This dialog box displays the items that have  been selected.
- Click **OK**.
