Category

Search for an item belonging to a specific item category. Click the browse button to view the Categories browser, from where you can select a single category or multiple categories.

The categories you select from this browser are displayed along with their names in the Selection dialog box. If you select multiple categories, the word ‘Multiple’ is displayed in the field in the form of a URL link.

If you select multiple categories, items that belong to each of these categories are displayed in the search result.
