Find Items Details

The Find Items dialog box (path: Inventory > Find Items) allows you to specify the search criteria to generate the required items list.

The Find Items dialog box comprises the field Filter Name and the following tabs:

  • Quick Find
  • More C[hoices](/versata-everest-docs/managing-items/finding-items/find-item-details/inventory-details/the_find_items_dialog_box_stock_info_mi.html)
  • Inventory
  • Purchase Info
  • Sales Info
  • Custom Field
  • Advanced Find
  • Query
