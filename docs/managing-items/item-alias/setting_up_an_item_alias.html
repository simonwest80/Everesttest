Create an Item Alias

To create an item alias, do the following:

  • From the main menu bar, select Inventory > Aliases. You will see the Items Aliases browser.
  • From the Items Aliases browser, select Options > New > In New Window. You will see the Item Alias profile.

You can create item aliases from any of the following locations:

  • The Items Aliases browser - Select Options > New > In New Window.
  • An existing Item Alias profile - Select Options > New.
  • From the Items browser - Select Options > Aliases.
