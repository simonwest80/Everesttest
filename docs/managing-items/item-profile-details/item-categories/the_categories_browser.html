The Categories Browser

The Categories browser (path: Inventory > Categories) displays a list of the item categories created in Everest. The options (path: Categories browser > Options) available in the Categories browser are:

List of Items
Use this option to view a list of the items that belong to the selected category in the Items browser
The Items Browser
E-Commerce
Use this option to access the E-Commerce Setup - Category dialog box wherein you can web-enable the items. You can:
  • Attach audio and video files.
  • Specify the images and banners to be displayed.
  • Enter a detailed description for the category.
  • Specify hyperlinks for banners.

: Web enable Category

Custom Fields
The Custom Fields feature allows you to define additional information such as character strings, date fields, logical fields or numeric fields.
