Add-ons

Use the Add-ons option (Items browser > Options) to specify accessories for the item being defined. Accessories refer to those items that can be bought or sold along with a particular item.

  • This option is disabled for Kits.
  • The Add-on Items browser is automatically displayed while adding items (for which add-on items exist) into documents. You will have to set the preference under File > Setup > Preferences > Flow Control > Documents > Prompts > Sales Document > Sales Document 2 tab > Display list of add-ons when adding items to sales documents.
  • Add-on items are retrieved into sales documents with the default sales measure and decimal accuracy of that add-on item. The quantity defaults to 1. The measure and the quantity can be changed after the add-on item is retrieved into the sales document. The price for that add-on item is based on the measure selected in the document.

To set up Add-ons, do the following:

  • From the main menu bar, select Inventory > Items. You will see the Items browser.
  • From the Items browser, highlight the item for which you want to specify add-on(s) and select Options > Add-ons. You will see the Add-ons dialog box.

You can also set up add-ons items from the Item profile > Add-ons tab.
