Sales Journal Profile - General

The General tab of the Sales Journal profile displays details of the sales made. This profile comprises the following sections:

  • Journals - This section comprises the header information for the journal that is common to all transactions within the journal.
  • Transactions - This section displays details regarding the transactions.
  • Transaction Entries - This section displays details of the accounting entries created.
