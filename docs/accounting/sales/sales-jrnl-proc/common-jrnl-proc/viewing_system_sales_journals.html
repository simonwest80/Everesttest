View System Sales Journals

To view sales journals, do the following:

  • From the main menu, select Accounting > Account Receivable > Sales Journal. You will see the Unposted Sales Journals browser. This browser lists all sales journals that have not yet been posted.
  • From the Unposted Sales Journals browser, highlight the required journal and select Options > List Transaction. You will see the Sales Journal Transactions browser.
  • From the Sales Journal Transactions browser, highlight the required transaction and select Options > Edit > In New Window. You will see the Sales Journal profile.

Transactions in the journal can be viewed either by:

  • Selecting the transaction number in the Transaction # field or
  • Navigating to a transaction using the Next ** or **Previous ** buttons.

 To view the system and manual sales journals separately in the Sales Journals browser, group the browser by the View only field. System journals will be grouped under View Only : T(True) while manual journals will be grouped under View Only : F (False).
