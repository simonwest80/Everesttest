Purge Transactions - Sales Journal Transactions Browser

To purge a manual sales transaction from sales journals browser, do the following:

  • From the main menu bar, select Accounting > Account Receivable > Sales Journal. You will see the Unposted Sales Journals browser.
  • From the Unposted Sales Journals browser, highlight the required journal and select Options > List Transaction. You will see the Sales Journal Transactions browser.
  • Highlight the require transaction and select Options > Purge transaction. You will see the Confirm window.
  • Click OK. Everest informs you that the transaction has been purged.
  • Click OK to return to the Sales Journal Transactions browser.
  • Only manual sales transactions can be purged.
  • Once the transaction is purged it will not appear in this browser.
