Delete Transactions - Sales Journal Profile

To delete a manual sales transaction from the journal profile, do the following:

  • From the main menu bar, select Accounting > Account Receivable > Sales Journal. You will see the Unposted Sales Journals browser.
  • From the Unposted Sales Journals browser, highlight the required journal and select Options > List Transaction. You will see the Sales Journal Transactions browser.
  • From the Sales Journal Transactions browser, highlight the required transaction and select Options > Edit > In New Window. You will see the Sales Journal profile.
  • Select the required transaction and select Options > Delete transaction. You will see the Confirm window.
  • Click OK. Everest informs you that the transaction has been deleted.
  • Click OK to return to the Sales Journal profile.

Only manual sales transactions can be deleted.
