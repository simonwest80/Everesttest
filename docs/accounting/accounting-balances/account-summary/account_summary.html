Account Summary

Use the Account Summary option to view a summary of the amounts debited/credited to an account during a selected period.

 Only posted amounts are displayed.

To view the summary of an account, do the following:

  • From the main menu bar, select Accounting > Account Balances > Account Summary. You will see the Posted Summary dialog box.
  • Specify the code of the required account in the General Ledger Account field. You will see the summary of the account in the Summary section.

The Posted Summary dialog box has the standard browser options.
