# A/R Detail Browser


The **A/R Detail Browser** displays:

- Sales and sales  return documents that have outstanding balances. System and manual documents  are included in this browser.
- The total amount  of the document and the balance that is receivable /payable on each document.
- The customers telephone  number as defined in the **Customer** profile.
- Whether you have  put the customer on credit hold.
- The number  of days in time blocks (such as 0-30 days and 30-60 days) by which the  documents are due.



This browser is displayed when the **A/R 
 Detail** option is selected (path: **Accounting 
 &gt; Account Receivable &gt; A/R Detail**). The **A/R 
 Detail Browser** has the following options:

- [Document  History]({{site.acc_baseurl}}/misc/document_history_ap_detail_browser_option.html)
