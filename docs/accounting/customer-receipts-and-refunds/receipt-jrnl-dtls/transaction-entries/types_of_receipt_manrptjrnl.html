Types of Receipt

Entry Type
The entry type identifies
  • The type of receipt/refund
  • The reason for the receipt/refund
  • Whether the amount against the entry can be a debit or credit
  • The type of document against which the receipt/refund can be made
: The first entry in this section is the entry for the bank account. The first transaction entry type is automatically displayed based on the payment method selected in the **Payment
Type** field.
 If you receive payment by cash, the entry type for the first transaction entry will be ‘Payment by cash’.
Similarly if the receipt is through credit card, the first transaction entry type will be ‘Payment by credit card’.

The various open customer documents and the corresponding entry types that are automatically displayed in this section are:

Document Entry Type
Sales invoice that has an outstanding due Payment on invoice
Sales order that has an outstanding deposit Apply deposit
Credit order that has an outstanding deposit Apply/refund credit order
Credit memo that has an outstanding balance Refund credit memo
