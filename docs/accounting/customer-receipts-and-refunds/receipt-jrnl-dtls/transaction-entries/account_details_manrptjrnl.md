# Account Details


**Account**
: The account that has been debited or credited is  displayed in this column. The account that is debited or credited depends  on the **Entry Type** selected.
: ![]({{site.acc_baseurl}}/img/lens.gif) [Entry  types - Details]({{site.acc_baseurl}}/customer-receipts-and-refunds/receipt-jrnl-dtls/transaction-entries/entry_types_details_receipt_jrnl.html)
: The first entry automatically displays the bank  account specified for the journal. This account cannot be changed. The  account to be debited or credited can be selected for further entries.
: ![]({{site.acc_baseurl}}/img/lens.gif) [Accounting  defaults]({{site.acc_baseurl}}/accounting-flow-control-and-defaults/accounting-defaults/accounting_defaults.html)


**Description**
: The name of the account debited or credited is displayed  in this column. The account description cannot be modified.


**Debit and Credit**
: The amounts to be debited or credited must be entered  in this column. When a document is selected, the entire document amount  is displayed in the appropriate column; you may change the amount if required.  The entry type determines whether the entry is a debit or credit.
: ![]({{site.acc_baseurl}}/img/example.gif) Â If  the entry type is 'Refund Credit Memo',  the Accounts Receivable Control account can only be credited. You will  not be allowed to enter any amount in the debit column.
: The amount for the bank entry displays the net amount  of debits and credits. It is the difference between the total debit and  credit for the subsequent entries. This is automatically calculated and  cannot be entered.
: Amounts entered in these columns automatically update  the amount received from the customer, as displayed in the check section.
