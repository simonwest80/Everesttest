Delete Transactions - Receipt Journal Profile

To delete a receipt transaction from the receipt journal profile, do the following:

  • From the main menu bar, select Accounting > Account Receivable > Receipt Journal. You will see the Unposted Receipt Journals browser.
  • From the Unposted Receipt Journals browser, highlight the required journal and select Options > List Transaction. You will see the Receipt Journal Transactions browser.
  • From the Receipt Journal Transactions browser, highlight the required transaction and select Options > Edit > In New Window. You will see the Receipt Journal profile.
  • Select the required transaction and select Options > Delete transaction. You will see the Confirm window.
  • Click OK. Everest informs you that the transaction has been deleted.
  • Click OK to return to the Receipt Journal profile.
