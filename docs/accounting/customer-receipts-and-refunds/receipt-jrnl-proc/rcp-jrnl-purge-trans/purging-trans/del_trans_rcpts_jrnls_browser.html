Purge Transactions - Receipts Journal Transactions Browser

To purge a receipt transaction from receipts journals browser, do the following:

  • From the main menu bar, select Accounting > Account Receivable > Receipt Journal. You will see the Unposted Receipt Journals browser.
  • From the Unposted Receipt Journals browser, highlight the required journal and select Options > List Transaction. You will see the Receipt Journal Transactions browser.
  • From the Receipt Journal Transactions browser, highlight the require transaction and select Options > Purge transaction. You will see the Confirmation window.
  • Click OK. Everest informs you that the transaction has been purged.
  • Click OK to return to the Receipt Journal Transactions browser.

Once the transaction is purged it will not appear in this browser.
