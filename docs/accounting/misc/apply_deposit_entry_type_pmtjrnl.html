Apply Deposit

Select this option if the deposit paid to the vendor on a purchase order has to be applied to a purchase invoice or another purchase order. When you select this option the Vendor Deposit will be credited and the Accounts Payable control account will be debited.

When this entry type is selected:

  • The Purchase Order number should be specified in the Document # column of the Transaction Entries section.

When you select Apply Deposit option, you will have to follow it with a transaction entry of type Payment on Purchase Invoice or Deposit on Purchase Order.

  • The Vendordeposit account specified in the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting) can only be credited.
  • The Credit column displays the amount of deposit (received on the selected purchase order) that has not been applied to other documents.
