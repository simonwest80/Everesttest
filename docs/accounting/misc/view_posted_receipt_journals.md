# View Posted Receipt Journals


{:.steps}
To view the posted receipt journals, do the  following:

- From the main  menu, select **Accounting &gt; Account 
 Receivable &gt; Receipt Journal**. You will see the **Unposted 
 Receipt Journals** browser.
- From the **Unposted Receipt Journals** browser, select  **Options &gt; Posted Journals**.  You will see the [**Posted Receipt Journals** browser]({{site.acc_baseurl}}/misc/the_posted_receipt_journals_browser.html).
