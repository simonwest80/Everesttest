View Posted Payment Journals

To view the posted payment journals, do the following:

  • From the main menu, select Accounting > Account Payable > Payment Journal. You will see the Unposted Payment Journals browser.
  • From the Unposted Payment Journals browser, select Options > Posted Journals. You will see the Posted Payment Journals browser.
