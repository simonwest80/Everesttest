# Refund Of Purchase Invoice


Select this option to reverse a payment that was made on a purchase  invoice.


When this entry type is selected:

- The **Purchase 
 Invoice** number should be specified in the **Document 
 #** column of the **Transaction Entries**  section.
- The **Accounts 
 payable** account specified in the **Accounting**  dialog box (path: **File** > **Setup** > **Preferences**  > **Defaults** > **Accounting**)  can only be credited.
- The **Credit** column automatically displays the amount paid on the selected purchase  invoice.
