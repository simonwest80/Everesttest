---
title: Customer deposit
---


Specify the account that should be debited when deposits are received  from customers. The **Customer Deposit** account is normally a liability account.


All deposits from customers on sales documents are credited to this  account.


{:.note}
You can only select an account that has been  set up in the base currency. You can modify this account only before it  is used in a manual/system accounting entry.
