# The Posted Purchase Journal Transactions Browser


This browser is displayed when **Options &gt; Transactions &gt; General Ledger (Detail)** is selected from the **Posted Purchase Journals** browser. The **Posted Purchase Journal Transactions** browser lists all the transactions in the selected journal with details such as the document number, reference data, payment terms and amounts debited or credited.


Options common to all browsers are available in this browser.
