View Posted Receipt Journals

To view the posted receipt journals, do the following:

  • From the main menu, select Accounting > Account Receivable > Receipt Journal. You will see the Unposted Receipt Journals browser.
  • From the Unposted Receipt Journals browser, select Options > Posted Journals. You will see the Posted Receipt Journals browser.
