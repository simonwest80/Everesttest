# The Posted Receipt Journal Summary Browser


This browser is displayed when **Options** > **General** **Ledger** (**Summary**) is selected in the **Posted** **Receipt** **Journals** browser. The **Posted Receipt Journal Summary** browser lists all the accounts debited or credited in the journal, and the net amount of debit and credit for the accounts.


Options common to all browsers are available in this browser.
