# The Posted Receipt Journals Browser


When the **Posted Journals** option is selected, the **Posted Receipts Journals** browser is displayed. The **Transactions** menu options (path: **Posted** **Receipts Journals** browser **&gt; Options &gt; Transactions**) in the **Posted Receipts Journals** browser allows you to view details of transactions in the journal or a summary of the accounts debited or credited in the journal.


The **Transactions** menu has the following sub menu options:

- [General Ledger (Detail)]({{site.acc_baseurl}}/misc/general_ledger_detail_posted_receipts_journals_options.html)
- [General Ledger (Summary)]({{site.acc_baseurl}}/misc/general_ledger_summary_posted_receipt_journals_browser_options.html)
- [Payment Method (Summary)]({{site.acc_baseurl}}/misc/payment_method_summary_posted_receipt_journals_browser_options.html)



Options common to all browsers are available in this browser.
