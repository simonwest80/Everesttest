The Posted Receipt Journals Browser

When the Posted Journals option is selected, the Posted Receipts Journals browser is displayed. The Transactions menu options (path: Posted Receipts Journals browser > Options > Transactions) in the Posted Receipts Journals browser allows you to view details of transactions in the journal or a summary of the accounts debited or credited in the journal.

The Transactions menu has the following sub menu options:

  • General Ledger (Detail)
  • General Ledger (Summary)
  • Payment Method (Summary)

Options common to all browsers are available in this browser.
