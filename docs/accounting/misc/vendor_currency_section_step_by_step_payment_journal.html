Vendor Currency Section

Vendor currency
The vendor currency (which is also the document currency) as defined in the Vendor profile (path: Purchasing > Vendors > Options > Edit > In New Window > Multicurrencytab) is displayed. This field cannot be edited.
Source currency
The source currency is displayed in this field. This field cannot be edited.
Exchange rate
The current exchange rate between the vendor currency and the base currency is displayed. You can change this rate if required.
Destination currency
The destination currency is displayed in this field. This field cannot be edited.
Foreign amount
The amount in vendor’s currency, entered in the General tab of the Payment Journal profile, is automatically displayed in this field. This amount can be edited as required.
