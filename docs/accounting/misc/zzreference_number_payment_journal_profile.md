---
title: Reference Number
---


The type of check whether computer or manual or other types is displayed in this section. You can specify any kind of reference information by clicking in this section and entering the details. Use 1 to 40 characters of any type to specify the reference number. In the case of manual checks and other payments, the reference number cannot be duplicated for the same payment type.
