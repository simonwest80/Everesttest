The Receipt Journal Transactions Browser

This browser is displayed when the List Transaction option (path: Unposted Receipt Journals browser > Options > List Transaction) is selected. The Receipt Journal Transactions browser lists all the transactions in the journal with details such as the customer code, document type and amounts debited or credited.

If payment is received through a credit card, the card number is displayed in the Reference column. Credit card numbers are masked using the format XXXXXXXXXXXX when the **Enable Extended Mode** check box is cleared (path: **File > Setup > Preferences > Workstation Settings > Company Settings** tab).

The Receipts Journals Transactions has the following additional options:

  • Select All / Unselect All - Use this option to select or de-select all the records in the browser. This is a toggle option.
  • Print Check - You can print the check for each refund transaction separately from this browser. You can also print multiple checks by highlighting the required transactions and selecting the Print Check option.

Printing Checks from Receipt Journal Transactions Browser

  • Split Journal - If a journal that is not posted contains too many transactions, you may split the journal. Splitting the journal moves the selected transaction(s) into a new journal.

Split Receipts Journals

 The split journal option is not available for Posted Journals.

  • Delete Transaction - You can delete a transaction from the Receipt Journal Transactions browser.

Deleting Transactions - Receipts Journals Browser

 The delete transaction option is not available for Posted Journals.

  • List Entry – This option allows you to view the accounting entries for a selected transaction.

Options common to all browsers are also available in this browser.
