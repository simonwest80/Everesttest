Specify the date of the transaction. In case of system entries made for document payments or refunds, the date displays the login date when the payment/refund was entered. For manual entries, you can select any date that falls within the fiscal year and period for the journal as the date of the transaction.

The default date displayed in this field depends on whether an active period has been defined.

Active Period Defined? Date Displayed:
YES - If login date belongs to active period - Login Date
- If login date does NOT belong to active period - First date of the active period.
NO Login Date
