Accounting - Other Flow Control

You can set up Everest such that unless you first print the journals, they cannot be posted.

To make the printing of journals compulsory before posting, do the following:

  • From the main menu bar, select File > Setup > Preferences > Flow Control > Other Flow Control. You will see the Other Flow Control dialog box.
  • Ensure that the Allow user to post a journal even if it is not printed check box is not selected.
  • Click OK to save the changes.
