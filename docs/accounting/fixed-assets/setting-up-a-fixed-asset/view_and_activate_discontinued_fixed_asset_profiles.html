View and Activate Discontinued Fixed Asset Profiles

To view and activate a discontinued fixed asset profile, do the following:

  • From the main menu bar, select File > Setup > Accounting > Fixed Assets. You will see the Fixed Assets browser.
  • From the Fixed Assets browser, select Options > View Discontinued. You will see the discontinued record in the Discontinued Fixed Assets browser.
  • From the Discontinued Fixed Assets browser, highlight the record.
  • Select Options > Activate. You will be asked to confirm the activation.
  • Click OK. The activated record will be displayed in the Fixed Assets browser.
