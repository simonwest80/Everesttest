Purge Transactions - Payment Journal Transactions Browser

To purge a payment transaction from the payment journals browser, do the following:

  • From the main menu bar, select Accounting > Account Payable > Payment Journal. You will see the Unposted Payment Journals browser.
  • From the Unposted Payment Journals browser, highlight the required journal and select Options > List Transaction. You will see the Payment Journal Transactions browser.
  • From the Payment Journal Transactions browser, highlight the require transaction and select Options > Purge transaction. You will see the Confirm window.
  • Click OK. Everest informs you that the transaction has been deleted.
  • Click OK to return to the Payment Journal Transactions browser.

Once the transaction is purged it will not appear in this browser.
