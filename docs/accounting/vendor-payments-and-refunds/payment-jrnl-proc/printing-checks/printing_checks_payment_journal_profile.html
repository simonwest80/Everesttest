Print Checks - Payment Journal Profile

To print computerized checks from the payment journal profile, do the following:

  • From the main menu bar, select Accounting > Account Payable > Payment Journal. You will see the Unposted Payment Journals browser.
  • From the Unposted Payment Journals browser, highlight the required journal and select Options > List Transaction. You will see the Payment Journal Transactions browser.
  • From the Payment Journal Transactions browser, highlight the required transaction and select Options > Edit > In New Window. You will see the Payment Journal profile.

 The Check Printed column in the Payment Journal Transactions browser displays information on whether the check for a particular transaction has been printed or not.

  • Select Options > Print Checks from the Payment Journal profile. You will see the Check Printing dialog box.
  • After specifying the details, click Preview to view the check report from where you can print the checks by clicking the ** icon. Click **Print to directly print the checks without previewing them. The Printed Checks Browser ** is displayed.
  • Select the checks that have printed correctly and click OK.

 Show Me

The transactions for which the checks have been printed are marked ‘printed’.
