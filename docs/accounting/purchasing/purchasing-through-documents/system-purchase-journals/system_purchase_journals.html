System Purchase Journals

System journals relating to purchase and purchase returns are created when:

  • Purchase receipts are created.
  • Purchase invoices are created.
  • Debit memos are created.

You can neither append a manual transaction to a system purchase journal, nor can you modify it. You can only view the transactions.

To view the purchase journals, do the following:

  • From the main menu bar, select Accounting > Account Payable > Purchase Journal. You will see the Unposted Purchase Journals browser listing all purchase journals that are not yet posted.
  • To view the details of a particular journal, highlight the journal and select Options > List Transaction. You will see the Purchase Journal Transactions browser.
  • From the Purchase Journal Transactions browser, select Options > Edit > In New Window. You will see the Purchase Journal profile.
