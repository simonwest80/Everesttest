Posting Journals to the Accounts

This section details how to post a journal in Everest.

To post a journal, do the following:

  • From the main menu bar, select the required journal type from the Accounting menu. The Unposted <journal name> Journals browser appears listing all of the unposted journals of that type.
  • Highlight the journal you wish to post. You may select multiple journals by holding down the Ctrl key while you mouse click them.
  • Choose Options > Post Journal(s) or right-click the journal and select Post Journal(s).
  • When the posting process completes, the Journal Posting Results dialog box displays information about the post.

: Journal Posting Results

  • Click OK to close the dialog box. The successfully posted journals move to the Posted <journal name> Journals browser.
