Purge Transactions - Purchase Journal Transactions Browser

To purge a manual purchase transaction from purchase journal transactions browser, do the following:

  • From the main menu bar, select Accounting > Account Payable > Purchase Journal. You will see the Unposted Purchase Journals browser.
  • From the Unposted Purchase Journals browser, highlight the required journal and select Options > List Transaction. You will see the Purchase Journal Transactions browser.
  • From the Purchase Journal Transactions browser, highlight the required transaction and select Options > Purge transaction. You will see the Confirm window.
  • Click OK. Everest informs you that the transaction has been deleted.
  • Click OK to return to the Purchase Journal Transactions browser.
  • Only manual purchase transactions can be deleted.
  • Once the transaction is purged it will not appear in this browser.
