Add Transactions to a Manual Purchase Journal

Transactions can be added to an existing manual purchase journal as required.

To add entries to an existing manual purchase journal, do the following:

  • From the main menu bar, select Accounting > Account Payable > Purchase Journal. You will see the Unposted Purchase Journals browser.
  • From the Unposted Purchase Journals browser, highlight the required journal and select Options > List Transaction. You will see the Purchase Journal Transactions browser.
  • From the Purchase Journal Transactions browser, select Options > New > In New Window. You will see the Purchase Journal profile.

A new transaction is automatically displayed for entry when you select this option. The next transaction number is automatically generated.
