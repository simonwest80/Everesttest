Unposted Purchase Journals Browser

The Unposted Purchase Journals browser (path: Accounting > Account Payable > Purchase Journal) lists all the purchase journals that have not been posted.

The options available in this browser are:

Information Available

The options that provide information in the Unposted Purchase Journals browser are:

  • List Transaction
  • Transactions
  • Posted Journals

Other Options

The other options available in the Unposted Purchase Journals browser are:

  • Select / Unselect All
  • Purge Journal
  • Open Journal(s)
  • Lock Journal(s)
  • Print Journal(s)
  • Post Journal(s)

Even if the journal is purged, it will still appear in the journal browser with description as ‘Purged Journal’.

Options common to all browsers are also available in this browser.
