Journal Type

This field is used to populate the information in the Journal Name field. Choose the journal type to specify your find criteria.

 If you want to find activity in inventory adjustment accounts, select inventory adjustment in this field. The find will display the account activity of all accounts, which were used to carry out inventory adjustments.

The types of journal you can specify for search criteria are:

  • General - searches for account activity in General journals.
  • Purchase - searches for account activity in Purchase journals.
  • Payment - searches for account activity in Payment journals.
  • Sales - searches for account activity in Sales journals.
  • Receipt - searches for account activity in Receipt journals.
  • Inventory Adjustment - searches for account activity in IA journals.
