More Choices

The More Choices tab provides you additional search parameters such as department, currency and transaction amount.

The More Choices tab contains the following fields:

  • Transaction Amount
  • Foreign Currency Amount
  • Location/Sub-Location
  • Currency
  • Journal Group - Specify the journal group to base your search on.
  • Smart Filter
  • Account [Type](/versata-everest-docs/reports/misc/smart_filter.html)
