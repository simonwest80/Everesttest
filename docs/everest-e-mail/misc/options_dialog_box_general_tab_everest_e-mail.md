When you add a signature, the format selected  in this field is associated with the signature.


**Send/Receive message every**
: From the drop-down list select the time interval  (in mins.)  for **Everest E-mail** to check your  mail server for new mails and to send outgoing mails.


**Include Attachments in Reply**
: Select this option to include the attachment that  you received with the original mail, in your reply.


**Include Attachments in Reply All**
: Select this option to include the attachment that  you received with the original mail, in your reply to all the recipients  of the original message.


**Keep copy of sent messages in Sent Folder**
: Select this option to store a copy of the mails  you send in the **Sent Items** folder.


**Send E-mails from browsers and profiles using  Everest E-mail**
: Select this option to enable users to send e-mails  using **Everest E-Mail** client to  customers or vendors, from the respective profiles or browsers in **Everest**.


{:.note}
If this option is not checked, when you select  **E-mail** > **New**  from **Customers**/**Vendors**/**Users** browser, **Everest** will open up the e-mail client configured on your system for sending  mails, instead of **Everest E-Mail**.


**When new mail arrives**
: Select the type of mail notification in this section.  You can select any or all of the following check boxes:

- **Display 
 icon on the status bar** - An icon is displayed on the status bar  when you receive a new mail.
- **Play 
 sound** **-** A  sound file is played when you receive a new mail.
- **Display 
 notification message** **-** You  will see the **New Mail** dialog box,  when you receive a new mail in your inbox.
