# Reply to a Message


{:.steps}
To reply to a message, do the following:

- In the **Everest E-Mail** window, select the message  you want to reply to, in the **Folder View 
 Pane**.
- From **** the **Everest E-Mail** menu,  select **Messages** > **Reply**.  A new message window will open with the recipient's name or e-mail address  in the **To** field.
- Enter the contents  of the message.
- Click **Send**.



{:.note}
If you want to reply to all the recipients  of the message in the **To**, **Cc** and **Bcc**  boxes, select **Messages** > **Reply All**.
