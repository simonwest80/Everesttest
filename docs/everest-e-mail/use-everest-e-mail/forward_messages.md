# Forward Messages


{:.steps}
To forward a message, do the following:

- Open the message  you want to forward.
- From the **Everest E-Mail** menu, select **Messages**  > **Forward**.
- Enter the names  of the recipients in the **To**, **Cc** and **Bcc**boxes.
- Click **Send**. The message is stored in your  **Outbox**,  ready to be sent.



OR

- From the **Folder List** in the **Everest 
 E-Mail** main window, select **Inbox.**
- Select the  messages you want to forward.
- From the right-click  menu, select **Forward**.
- Enter the names  of the recipients in the **To**, **Cc** and **Bcc**  boxes.
- Click **Send**. The message is stored in your  **Outbox**,  ready to be sent.
