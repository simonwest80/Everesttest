# Copy Messages


{:.steps}
To copy messages to a folder, do the following:

- From the **Folder List** in the **Everest 
 E-Mail** main window, select a folder from which you want to  copy messages.
- Select the  messages you want to copy.
- From the **Everest E-Mail** menu, select **Edit**  > **Copy to Folder**. You will  see the **Copy Messages** dialog box.
- Select the  location where you want to copy the messages.
- Click **OK**. The messages are copied to the selected  folder.



{:.note}
If you want to create a new folder and copy  the messages into that folder, click **New 
 Folder**. You will see the **New Folder**  dialog box. Enter name of the new folder, select the location where you  want to create the folder and click **OK**.  The new folder is created.
