The Custom Tracking - Vendor Browser

The Custom Tracking - Vendor browser (path: Purchasing > Vendors > Options > Vendor Tracking) displays all the custom tracking groups and choices created. The options (path: Custom Tracking - Vendor browser > Options) available in this browser are:

  • Tree View
  • Notes Preview
  • Switch Pane
  • Prompt for Save
  • Custom Fields
