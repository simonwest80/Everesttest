Tree View

This option allows you to toggle between the ‘normal view’ and ‘tree view’ of the browser.

To get the tree view of the Custom Tracking browser, select Options > Tree View . To revert to the normal browser view, select Options > Tree View.

The tree view of the Custom Tracking - Inventory browser is divided into the following sections:

  • Tracking Category
  • Tracking Groups
  • Selection Section
