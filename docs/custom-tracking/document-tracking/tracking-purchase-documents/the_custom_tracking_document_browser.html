The Custom Tracking - Document Browser

The Custom Tracking - Document browser (path: Purchasing/Invoicing > selected document > Document browser > Options > Document Tracking) displays all the custom tracking groups created for the documents. The options (path: Custom Tracking – Document browser > Options) available in this browser are:

  • Tree view
  • Notes Preview
  • Switch Pane
  • Prompt for Save
  • Custom Fields
  • Standard Browser Options
