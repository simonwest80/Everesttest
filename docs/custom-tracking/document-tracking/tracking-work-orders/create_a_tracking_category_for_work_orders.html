Create a Tracking Category for Work Orders

To create a tracking category for work orders, do the following:

  • From the main menu bar, select Inventory > Custom Tracking > Documents. You will see the Custom Tracking Setup - Document browser.
  • From the Custom Tracking Setup - Document browser, select Options > New > Category > In New Window. You will see the Tracking Category profile.

 You can also create a tracking category for work orders from an existing Tracking Category profile - Select Options > New.
