The Custom Tracking - Document Browser

The Custom Tracking - Document browser (path: Invoicing > Select the required document > Document browser > Options > Document Tracking) displays all the custom tracking groups and choices created for documents. The options (path: Custom Tracking - Document browser > Options) available in this browser are:

  • Tree view
  • Notes Preview
  • Switch Pane
  • Prompt for Save
  • Custom Fields
