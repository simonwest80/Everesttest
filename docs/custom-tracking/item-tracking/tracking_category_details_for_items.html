Tracking Category Details for Items

The Tracking Category (Item) profile (path: Inventory > Custom Tracking > Inventory > Options > New > Category > In New Window) allows you to create categories and sub-categories for classifying custom tracking groups.

Code
Code is the unique number assigned to the tracking category. It is automatically generated when you save a new tracking category.
Description
The description identifies the name of the tracking category.
Parent Category
Parent category identifies the category to which a sub-category belongs.

 You could create a sub-category called ‘Bangles’ with ‘Jewelry’ as the parent category.

 Everest creates a default tracking category called ‘Default-Items’. All tracking groups and choices, by default, are organized under this category, unless you create a new one to reorganize the tracking groups.
