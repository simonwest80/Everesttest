Tracking Group Details for Items

The Custom Tracking Group - Items profile (path: **Custom Tracking Setup

  • Inventory** browser > Options > New > Group > In New Window) allows you to create a custom tracking group for items.

General Information

This section gives you information related to the General tab of the Custom Tracking Group profile.

Group Code
Group code is the unique number assigned to the custom tracking group. It is automatically generated when you save a new custom tracking group.
Description
The description identifies the tracking group.
 Descriptions can be self-explanatory like ‘Warranty period of items’
Allow multiple selections to be tagged
Using this option, a particular record can be tagged with more than one selection for the group.
 If multiple selections are allowed for the tracking group ‘Featured Items’ with choices ‘Exclusive Items’, ‘New Arrivals’ and ‘Season Special’ then an item can be tagged with more than one choice for the group.
Force tag at least one selection
This option makes tagging with at least one choice compulsory. This ensures that all records
have relevant data.
 If the tracking group for items ‘Featured items’ compels you to select at least one choice when a new item is created or when the profile of an existing item is accessed after creating the tracking group, the item profile cannot be closed without tagging the item with one of the choices for the group ‘Featured items’.
Refresh tag date if memo is updated
You can also give additional information for the choice as notes. When an item is tagged with a choice, Everest records the date of tagging. This option ensures that whenever you modify the notes to the choice, the tag date gets updated with the date of modification.
 If an item is tagged with the choice ‘Exclusive Items’ for the group ‘Featured Items’ you can add a note to the choice that the preference is the items featured.
Security Description
This field denotes the security right for access to the custom tracking group created. Only those users who are assigned with this security right can tag any selection belonging to this group.

E-Commerce Information

This section gives you information related to the E-Commerce tab of the Custom Tracking Group profile.

The tracking groups that you create can also be enabled to appear in different sections of your EverestE-Commerce store. The layout of the Web page, on which the tracking groups appear, depends on the Everest E-Commerce template for your online store.

 You can create a tracking group for featured items with choices for ‘Best Selling Items’, ‘Bargains’, and ‘Items on Sale’. Items tagged with any of these choiceare displayed on your Everest E-Commerce store.

Web-enabled item tracking groups are normally created to showcase featured items

From the E-Commerce tab you can:

  • Attach audio and video files.
  • Specify the images and banners to be displayed.
  • Specify other sales information for items on the online store.
  • Enter a detailed description for the item.
  • Specify items for download.
  • Specify hyperlinks for banners

Custom Fields Information

This section gives you details regarding the Custom fields tab of the Custom Tracking Group profile.

The Custom Fields tab allows you to define additional information for any record as per your requirement.

Custom fields can be defined as character strings, or date, logical, memo or numeric fields

You can give ‘Primary Vendor’ or ‘Additional URL’ in the “Characters” custom field.
