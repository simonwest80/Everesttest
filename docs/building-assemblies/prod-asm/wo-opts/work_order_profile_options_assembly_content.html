Work Order Profile Options

The following information can be accessed from the Options menu (path: Work Order profile > Options) of a work order:

  • Process
  • Print
  • Inline Notes
  • Preview Off/On
  • Line Items
  • Item Information
  • Serial Number History
  • Show all Serial #
  • Link Serial Number
  • Variance Analysis
  • Production Details
  • Contact Manager
  • Document Information
  • Transaction History
  • Attach/Retrieve Serial #
