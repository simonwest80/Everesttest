# Line Items


The **Line** **Items**  option (path: **Work** **Order**  profile > **Options** > **Line** **Items**)  allows you to do the following:


| **Option Name** | Description |
| **Insert**  - | Inserts a blank line in the Item Details Grid. |
| **Line Profile** - | Displays information about the line item highlighted in the Item Details  Grid. This option displays [**Work Order Lines** dialog box]({{site.ba_baseurl}}/prod-asm/wo-opts/line-items/work_order_lines_dialog_box_work_order_profile_options_line_items.html). |
| **Move up** - | Moves the selected line item one line up. |
| **Move down** - | Moves the selected line item one line down. |
