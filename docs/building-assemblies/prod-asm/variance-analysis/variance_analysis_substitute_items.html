Variance Analysis - Substitute Items

Variance for items that are substituted is computed by adding the values and quantities of the standard item and its substitute.

You use Item A in a work order. The input details are:


   Standard Actual
Quantity 10 5
Rate ($) 2 2.10



You also use 8 units of a substitute for A - Item B at $1.8.


 Rate Variance = (Standard rate - Actual rate) Actual Quantity


= (2-2.10)*5


= (0.5) (negative variance)
Quantity Variance = (Standard quantity - Actual quantity) * Standard rate


= (10-5)*2


= (10) (negative variance)
Cost Variance = Rate variance of A + Quantity variance for A - Total cost of B


= {10 - 0.5 -(8*1.8)}


= (4.9) (negative variance)		  

Variances are NOT displayed separately for substitute items. The variance computation for the item that is substituted also considers the substitute item.
