The Substitute Item Dialog Box

This dialog box allows you to specify the details of the substitute for the selected item.

Item
This field displays the Item Code of the item (in the work order) to be substituted. You cannot edit this field.
Substitute With
Select the substitute item. All add-on and related items specified in the Item profile for the selected item (path: Inventory > Items > Options > Edit > In New Window > Add-Ons/Related Items tab) are displayed for selection.

An item can be substituted with another item only in the Not Started stage of the Work Order.
