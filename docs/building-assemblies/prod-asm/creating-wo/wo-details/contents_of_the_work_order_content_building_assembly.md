# Work Order Details - Introduction


A Work Order is the document that identifies the item to be assembled  or the item assembled, and the item quantity (to be manufactured). The  work order serves as the authority to issue materials and assign labor  to production. The work order can only be created in the stock measure.


The information available on a **Work 
 Order** profile (path: **Inventory** > **Bill** **of****Material** > **Work****Order** > **Options**  > **Edit** > **In 
 New Window**) have been grouped into the following sections:

- [Document  Information]({{site.ba_baseurl}}/prod-asm/creating-wo/wo-details/document_information_work_order_building_assembly_content.html)
- [Assembly  Information]({{site.ba_baseurl}}/prod-asm/creating-wo/wo-details/assembly_information_work_order_profile_assembly_contents.html)
- [Item  Details]({{site.ba_baseurl}}/prod-asm/creating-wo/wo-details/item-dtls/item_details_grid_work_order_profile_assembly_contents.html)
- [Document  Tab View Information]({{site.ba_baseurl}}/prod-asm/creating-wo/wo-details/document_tab_view_information.html)
- [Status  Bar]({{site.ba_baseurl}}/prod-asm/creating-wo/wo-details/status_bar_work_order_profile.html)
