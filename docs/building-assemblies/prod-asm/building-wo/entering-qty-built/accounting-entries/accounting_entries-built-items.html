Accounting Entries - Built Items

The entries created to account for the assembled/built items and the cost of production are:

Account Path Debit/


Credit		 Amount
Inventory Control account The account selected in the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting > Accounts tab > Inventory
control field).		 Debit The total value of actual output and scrap.
Asset or Expense account The account selected in the Work Order profile (path: Inventory > Bill of Material > Work Order > Options > Edit > In
New Window > Item Details Grid > Expense Account column).		 Credit The actual value of non-inventoried components
Abnormal Loss account The account selected in the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting > Accounts tab > Abnormal
Loss field).		 Debit The value of abnormal loss.
Work in Progress account The account selected in the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting > Accounts tab > Work
In Progress Account field).		 Credit The value of work in progress
Abnormal Gain account The account selected in the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting > Accounts tab > Abnormal
Gain field).		 Credit The value of abnormal gain
