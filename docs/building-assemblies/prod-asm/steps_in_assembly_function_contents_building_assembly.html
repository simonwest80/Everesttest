Steps in Production

Everest manages the entire production function through Work Orders. All transfers of stock from raw materials to work-in-process and transfer of stock from work-in-process to finished goods is handled through the Work Order. The steps in production are:

  • Create a work order for the assembly.
  • Issue the work order to production.
  • Process the work order.
  • Build the work order.
  • Analyze variances.
