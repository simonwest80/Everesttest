# Details


**Created By**
: This field displays the name of the logged in user  by default.


**Assigned By**
: Specify the name of the user who assigned the work  order. By default this field displays the name of the logged in user.


**Assigned To**
: Specify the name of the person to whom the work  order is assigned.


**Main Order #**
: The work order number based on which the current  work order is being processed is displayed in this field. You cannot edit  this field.


**Back Order**
: The work order that is created based on the current  work order is displayed in this field. You cannot edit this field.


**Loc/Sub-Loc**
: Select the location or the sub-location for which  the work order is created. By default this field displays the logged in  location or sub-location. You can select another location/sub-location,  if you are executing a work order on behalf of another location/sub-location.


**Description**
: The **Description**  field displays the location/sub-location's description as entered in the  **Location/Sub-Location** profile.  You cannot edit this field.


**Assigned Date**
: This field displays the date on which the work order  was issued. The date displayed in this field by default. You cannot edit  this field.


**Build Date**
: This field displays the date on which the details  of the quantities built are entered. The date displayed in this field  is the login date (when the work order was built). You cannot edit this  field.
: ![]({{site.ba_baseurl}}/img/lens.gif) [The  Build Dialog Box]({{site.ba_baseurl}}/prod-asm/building-wo/entering-qty-built/the_build_profile.html)


**Build By**
: This field displays the name of the user who is  responsible for the production of the batch. This field is updated with  the name of the user specified in the **Build 
 by** field of the [**Build** dialog box]({{site.ba_baseurl}}/prod-asm/building-wo/entering-qty-built/the_build_profile.html). You cannot edit  this field.


**Actual Output**
: This field displays the actual quantity built or  manufactured as against the required output. This column is updated by  the quantity entered as built when the **Build**  option (path: **Work Order** profile  > **Options** > **Process**  > **Build**) is selected.
: ![]({{site.ba_baseurl}}/img/lens.gif) [The  Build Dialog Box]({{site.ba_baseurl}}/prod-asm/building-wo/entering-qty-built/the_build_profile.html)
