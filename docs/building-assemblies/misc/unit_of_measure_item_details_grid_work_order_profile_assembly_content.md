# Unit of Measure


This column displays the stock unit of measure for the item specified  in the **Item** (Assembly) profile  (path: **Inventory &gt; Items &gt; Options 
 &gt; Edit &gt; In New Window &gt; Inventory** tab **&gt; Stock Measure** field). You cannot edit fields in this column.
