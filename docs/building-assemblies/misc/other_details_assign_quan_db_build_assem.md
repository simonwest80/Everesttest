# Other details


**Assigned**
: This column displays all the serial numbers attached  to the purchase order with reference to the stock area to which they are  currently assigned.
: Use the buttons to assign the serial numbers to  the required stock bins.


{:.note}
To un-assign a serial number from a stock  bin use the > and >> buttons. The items are un-assigned from  the document.


**Available on Order**
: This column displays the list of serial numbers  attached to the document along with the stock bin that the serial number  has been assigned to.
: Use the buttons to assign the serial numbers to  the required stock bins. When the serial numbers are assigned to other  the stock bins, the quantities are automatically reduced from the default  stock bin.
