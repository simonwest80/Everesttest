Purchase Documents Browser (System)

When you select the All System Documents option (path: Vendors browser > Options > Documents), you will see the Purchase Documents browser. This browser displays all system documents that you created for the selected vendor. By default, this browser is grouped by the document type.

You can create documents of all the types that are listed in the browser. The document created depends on the record you highlight before selecting Options > New > In New Window.

 If the selected record was a purchase receipt, a new purchase receipt profile is displayed.

The options available depend on the selected document type.
