Apply Masks

You can set up a mask, i.e., a predefined format, for the telephone numbers and zip codes for each country in the Country profile (path: File > Setup > Miscellaneous > Country).

Country Profile

Use the Apply Masks option to indicate whether the vendor’s telephone numbers and zip code must be displayed in the defined format. If you do not select the ApplyMasks option, the telephone numbers and zip codes display as entered in the Vendor profile.

Click the Apply Mask button to view the Country profile where you can make the required changes.

These check boxes are selected by default.
