# Account Code


The vendor's account code uniquely identifies a vendor's Â account.  This code is automatically generated, if left blank. The vendor code assigned  in this case is the next number in sequence specified for the login location/sub-location.


The sequence numbers are specified in the **Vendor** field of the **Location** profile  (path: **File** > **Setup**  > **Accounting** > **Location/Sub-Location**>  **Options** > **Edit**  > **In New Window** > **Purchasing** tab).


If a sequence number is not assigned to the login location/sub-location,  the sequence number of the default location (path: **File**  > **Setup** > **Preferences**  > **Defaults** > **Accounting**  > **Miscellaneous** tab > **Default Location** field) is used.
