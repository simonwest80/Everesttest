# Apply Masks


You can set up a mask, i.e., a predefined format, for the telephone  numbers and zip<font color="#0000FF" class="hcp1"> </font>codes for each country in  the **Country** profile (path: **File** > **Setup**  > **Miscellaneous** > **Country**).


![]({{site.mv_baseurl}}/img/lens.gif) [Country  Profile]({{site.sc_chm}}/options/international-set-up/country/profile/the_country_profile.html)


Use the **Apply Masks** option to  indicate whether the vendorâ€™s telephone numbers and zip<font color="#0000FF" class="hcp1"> </font>code must be displayed in the defined format.  If you do not select the **Apply****Masks** option, the telephone numbers  and zip<font color="#0000FF" class="hcp1"> </font>codes display as entered in the **Vendor** profile.


Click the **Apply Mask** button  ![]({{site.mv_baseurl}}/img/managing_vendors_apply_masks.gif) to view the **Country**  profile where you can make the required changes.


These check boxes are selected by default.
