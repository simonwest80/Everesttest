# Telephone Code


The **Telephone Code** field is  automatically updated with the international dialing code for the country  selected. This field is non-editable. By default, this field displays  the code you define for the country (selected in the **Vendor** profile > **General** tab  >**Country** field) in the [**Country** profile]({{site.sc_chm}}/options/international-set-up/country/profile/the_country_profile_general.html).
