# Account Summary


The **Account Summary** option (path:  **Vendors** browser > **Options**  > **History** > **Account Summary**) displays a summary of your transactions with the  selected vendor. The vendor summary is updated whenever a purchase invoice  or a debit memo (manual or system) is created for the vendor or whenever  a payment is made to a vendor.


By default all amounts are displayed in the base currency. Click on  the **Account Summary** option to  view the **Vendors Summary** dialog  box which comprises the following tabs:

- [Vendor  Summary tab]({{site.mv_baseurl}}/profile-options/info/history/account-summary/vendor_summary_account_summary_history_additional_browser_options_vendor_profile.html)
- [Periodic  Summary tab]({{site.mv_baseurl}}/profile-options/info/history/account-summary/periodic_summary_account_summary_history_additional_browser_options_vendor_profile.html)
