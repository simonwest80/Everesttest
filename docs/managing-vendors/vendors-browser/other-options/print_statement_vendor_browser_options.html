Print Statement

Use this option to print a Vendor Statement report. This report may be sent to vendors as a confirmation of the balances due to them.

To print the vendor statements report, do the following:

  • From the main menu bar, select Purchasing > Vendors. You will see the Vendors browser.
  • Highlight the vendor for whom a statement has to be printed.
  • Select Options > Print Statement. You will see the Vendor Statement Printing Preferences dialog box.
  • Specify the report parameters and click Preview/Print.
