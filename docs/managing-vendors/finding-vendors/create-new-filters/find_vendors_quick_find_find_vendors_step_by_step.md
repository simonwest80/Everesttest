Specify the vendor's account name as per the **Vendor**  profile.


**You can enter a partial value. The search  result displays a list of vendors whose account names begin with the partial  value specified.**


![]({{site.mv_baseurl}}/img/example.gif)  Enter  'Ra' in this field and click the **Find**  button. The search results display a list of all vendors whose first names  begin with 'Ra'.


If you select multiple account names the word 'multiple' is displayed  in this field in a URL link format.

Enter the vendor's first name as per the **Vendor**  profile.


**You can enter a partial value. The search  result displays a list of vendors whose first name begins with the partial  value specified.**


![]({{site.mv_baseurl}}/img/example.gif)  Enter  'Ar' in the  field and click the **Find** button.  The search result displays a list of all vendors whose first names begin  with 'Ar'.


**Multiple selections are not allowed in  this search field. Click the drop down button to view the history of prior  search parameter values.**

Specify the vendor's last name as per the **Vendor**  profile.


You can enter a partial value. The search result displays a list of  vendors whose last name begins with the partial value that you specify.


![]({{site.mv_baseurl}}/img/example.gif)  Enter  'Ba' in the field and click the **Find**  button. The search results displays a list of all vendors whose last names  begin with 'Ba'.


**Multiple selections are not allowed in  this search field. Click the drop down button to view the history of prior  search parameter values.**

Enter the telephone number as per the **Vendor**  profile. You can enter a partial value for your search.


![]({{site.mv_baseurl}}/img/example.gif)  Enter  '5' in the field and click the **Find**  button. The search result displays a list of all vendors whose telephone  numbers begin with 5.


**Multiple selections are not allowed in  this search field. Click the drop down button to view the history of prior  search parameter values.**

Specify the vendor's e-mail  id as per the **Vendor** profile.


You must enter the full e-mail  ID of the vendor. If you make a partial entry in this field, **Everest**  informs you that the email ID is invalid. Click **OK**  to continue.


![]({{site.mv_baseurl}}/img/example.gif) If  the e-mail  id of the vendor is vendor@insight.com,  you have to enter the complete e-mail  id (vendor@insight.com)  to retrieve data.


**Multiple selections are not allowed for  this search field. Click the drop down button to view the history of prior  search parameter values.**

Specify the purchase representative's name as assigned in the[  Vendor Profile]({{site.mv_baseurl}}/creating/the_vendor_profile_steps_by_steps.html).


**You can search for all vendors attached  to a single or multiple purchase representatives. If you select multiple  purchase representatives the word 'multiple' is displayed in this field  in a URL link format.**


**Click the drop down button to view the  history of prior search parameter values.**

Specify the vendor's department as per the **Vendor**  profile.


| ![]({{site.mv_baseurl}}/img/lens.gif) | [Vendor Profile -  Miscellaneous Tab]({{site.mv_baseurl}}/creating/the-vendor-profile-miscellaneous/the_vendor_profile_miscellaneous_tab.html)<br/><br/><br/>[Location  and Departments]({{site.sc_chm}}/options/locations-and-sub-locations/locations_and_departments.html) |



**You can enter a partial value. The search  result displays a list of vendors whose department name begins with the  value specified.**


![]({{site.mv_baseurl}}/img/example.gif)  If  you want a list of all vendors whose department begins with the alphabets  ‘Ba’, type ‘Ba’ in the field.


Click the drop down button to view the history of prior search parameter  values.


You can search for all vendors attached to a single or multiple departments.  If you select multiple departments the word 'multiple' is displayed in  this field in a URL link format.

Specify the address type as per the **Vendor**  profile. **Select Billing, Shipping or 
 Contact address, to display a list of vendors with the specified address 
 type.**


**Click the drop down button to view the  history of prior search parameter values.**


**You can search for vendors with a single  address or multiple address types. If you select multiple address types  the word 'multiple' is displayed in this field in a URL link format.**

Specify the number of records to be retrieved. **The 
 options in this field are detailed below:**


| ****Option**** | ****Description**** |
| **All** | **Displays all the records.** |
| **Custom** | **Displays only the specified number of records.** |



![]({{site.mv_baseurl}}/img/example.gif)  If  you choose 10, the search retrieves all the records that match the set  parameter, but displays only 10 records at a time.


If you select **All,** the display  field (beside the field **Number****of** **records****to** **retrieve**)  is disabled. If you select the option **Custom**  you can specify the number of records to be retrieved.

# Find Vendors - Quick Find


The **Quick Find** tab lists the  most common search parameters that you will use to find vendors.


{:.note}
**The parameters in the **Quick 
 Find** tab will be populated automatically when you select an existing  filter (that contains the relevant **Quick 
 Find** tab parameters).**


**You can use the search options individually  or in a combination.**


**The options selected in the **Quick 
 Find** tab also appear in the **Advanced 
 Find** tab. You can also change the search parameters from the **Advanced Find** tab.**
