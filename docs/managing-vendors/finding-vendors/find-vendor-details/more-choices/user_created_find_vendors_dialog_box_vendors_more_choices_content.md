# User Created


**Use the **User 
 Created** field to search for vendors on the basis of the user who  created the **Vendor** profile.**


{:.steps}
To select  the user(s), who created the vendor, do the following:

- Click the browse  ![]({{site.mv_baseurl}}/img/managing_vendors_selection_button.gif) icon next to the **User 
 Created** field. You will see the **Selection**  dialog box.
- From the **Selection** dialog box, select the user(s)  of your choice.
- Click **OK**. **The** **User Created** **field is populated with the relevant values.**



Click the drop down button to view a history of prior search parameter  values. **If you select multiple users 
 the word 'multiple' is displayed in this field in a URL link format.**
