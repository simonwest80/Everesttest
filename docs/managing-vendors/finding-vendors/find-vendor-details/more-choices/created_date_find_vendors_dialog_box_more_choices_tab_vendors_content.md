# Created Date


Use the **Created 
 Date** field to search for vendors by their 'date of creation'. The  search displays all the vendors created on the specified date.


{:.note}
Enter the start date and end date to specify  a search for a date range.


All date options (such as today and this week) are with respect to the  login date.
