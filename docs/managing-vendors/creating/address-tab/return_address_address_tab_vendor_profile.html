Return Address

To add/edit/set as default a return address, do the following:

  • Click the New icon to add a new return address.
  • Click the Edit icon to modify the selected return address.

To edit the vendor’s return address:

  • Click the Edit icon from the Address tab of the Vendor profile. You will see the Address profile.
  • Edit the address details. However, Everest does not allow you to directly edit a return address, if it has been used while creating any sales/purchase document. In this case:
  • Select Options > Modify Address. You will see the Modify Address dialog box.
  • Select the required option and click OK.

  • Specify  the new address information for the vendor.

  • Click the Default icon to set the selected return address as the default return address of the vendor.
  • Click the Return Address Browser icon to view the Addresses browser that displays all the return addresses defined for the vendor.
