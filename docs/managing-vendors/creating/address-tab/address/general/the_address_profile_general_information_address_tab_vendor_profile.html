The Address Profile - General (General Information)

Residential
Select this check box if the vendor is located within a residential area as defined by freight carriers. This check box is selected by default.
Title
Enter the vendor’s title using 1 to 40 characters of any type.
You can enter a title that describes the vendor’s position or job, such as Managing Director, Director-General or Assistant Manager.
First Name
Enter the vendor’s first name using 1 to 25 characters of any type.
First Name and Last Name
Middle Name
Enter the vendor’s middle name using 1 to 25 characters of any type.
Last Name
Enter the vendor’s last name using 1 to 25 characters of any type.
First Name and Last Name
Company
Enter the name of the company that the vendor represents using 1 to 40 characters of any type. This is a mandatory field. By default this field displays the information specified in the Account name field (path: Vendor profile > General tab). If this field is left blank, it is automatically updated with the information from the First Name and Last Name fields.

It is mandatory to enter either the company name or the first name or the last name.
