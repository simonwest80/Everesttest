Contact Address

To add/edit/set as default a contact address, do the following:

  • Click the New icon to add a new contact address.
  • Click the Edit icon to modify the selected contact address.

To edit the vendor’s contact address:

  • Click the Edit icon from the Address tab of the Vendor profile. You will see the Address profile.
  • Edit the address details. However, Everest does not allow you to directly edit a contact address, if it has been used while creating any sales/purchase document. In this case:
  • Select Options > Modify Address. You will see the Modify Address dialog box.
  • Select the required option and click OK.

  • Specify  the new address information for the vendor.

  • Click the Default icon to set the highlighted contact address as the default return address of the vendor.
  • Click the Contact Address Browser icon to view the Addresses browser that displays all the contact addresses defined for the vendor.
