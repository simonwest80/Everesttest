Setting up Everest MailBridge

Setting up Everest MailBridge involves configuring Everest as well as Outlook .

To set up Everest MailBridge, do the following:

  • From the Start menu, select Programs > Everest Advanced Edition > Utilities > MailBridge. You will see the Settings dialog box.

The first time you log in to Everest MailBridge, you will be informed that the settings are not defined, and so e-mail cannot be recorded in Everest. You will be prompted to enter the settings. Click Yes to confirm your settings.

The Settings dialog box contains the following tabs:

  • Everest Server
  • E-mail Server
  • Preferences
