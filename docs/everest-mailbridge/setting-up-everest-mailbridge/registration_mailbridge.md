# Registration


**Everest MailBridge** is a regular  feature of **Everest Advanced Edition**  and the number of **MailBridge** connections  are not limited to the number of **Everest**  users. **MailBridge** can be connected  to the **Everest** database an unlimited  number of times.


If you buy a 5-user **Everest**  seat license, you can install more than 5 **Mailbridge**  connections in different machines and log the mails/tasks to the **Everest** server.


**MailBridge** requires at least  5MB of free space. You also need to have Outlook for sending and receiving  e-mails.
