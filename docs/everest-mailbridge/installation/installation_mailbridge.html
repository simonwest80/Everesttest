Installation

You may choose to install Everest MailBridgein the following instances:

  • Install Everest MailBridge during an Everest Advanced Client Edition installation using Custom Setup

: Client Installation

  • Install Everest MailBridge after a Typical Everest Advanced Client Edition installation which only installs the Everest Advanced Edition Client and Help files

: Add Components after a Typical Client Installation

For additional information, see the Everest Advanced Edition Installation Guide.
