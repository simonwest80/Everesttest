# About Everest


This dialog box displays version information about the **Everest**  software you are using. You can visit the **Everest 
 Software** website by clicking on the [http://www.everstsoftwareinc.com](http://www.everestsoftwareinc.com)  link. You can also view information about your workstation by clicking  the [**System Info**]({{site.wwe_baseurl}}/misc/system_information_button_about_everest_dialog_box.html) button. Click the [**Tech Support**]({{site.wwe_baseurl}}/misc/technical_support.html)  button to view information about **Everest 
 Software** Technical Support services.
