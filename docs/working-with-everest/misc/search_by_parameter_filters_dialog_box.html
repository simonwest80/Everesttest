Search by

This field enables you to search for a filter related to the selected browser, by selecting a parameter. You can search for filters by Filter ID, Created By or Filter Name. Click the respective column header in the Top (Display) Panel to search for filters by that parameter. The parameter name displayed in the Search By <parameter> field changes when you click a column header in the Top (Display) Panel.
