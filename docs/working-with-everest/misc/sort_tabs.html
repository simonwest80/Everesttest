Sort Tabs

These tabs enable you to sort the records in a browser based on the field specified in the sort tab.

To sort the Items browser on the basis of ‘Category’ click on the ‘Category’ sort tab.

When a sort tab is selected, you must specify the search criteria based on the sort tab.

If ‘Category’ is selected as the sort tab, you must enter a category code in the search field.
