Filter Records Using the Classic Style Filter

This filter allows you to filter the selected browser, using conditions similar to Advanced Find conditions.

Advanced Find

To filter a browser using the classic style filter, do the following:

  • From the required browser’s toolbar menu, click the Expert Style Filter icon. You will see the Filters dialog box.
  • Click the drop-down arrow beside the Add button. Select Classic Style from the drop-down list.
  • You will see the Specify Filter dialog box.
  • After specifying the search condition, click OK. You will see the Filter Name dialog box.
  • Specify a name for the newly created Classic Style filter, and click OK.
  • You will see the Filters dialog box. Click Apply to run the filter.
  • The browser lists records that meet the selection criteria specified. To further filter the records in the browser, or modify the filter, double-click the filter details displayed above the browser’s status bar, and define additional conditions.

When records are filtered using the Classic Style filter, Everest displays the details of the filter above the browser’s status bar.
