# Discontinue a Record


When you discontinue a record, it is temporarily unavailable for selection.  The record is not deleted from the database; it is merely deactivated.


{:.steps}
To discontinue a record, do the following:

- From the main menu  bar, open the browser from which you want to discontinue records.
- Highlight the record  you want to discontinue and select **Options 
 &gt; Discontinue**.
- Click **OK**.  The record becomes unavailable for selection.



{:.note}
You can view the discontinued records from  the **Discontinued &lt;Records&gt;**  browser by selecting **Options &gt; View 
 Discontinued** from any browser .
