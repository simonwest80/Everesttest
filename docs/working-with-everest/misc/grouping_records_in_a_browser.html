Grouping Records in a Browser

You can group records in a browser in any of the following ways:

  • Drag-and-drop - Open the browser you want to group and point the cursor to the required column header. Then drag and drop the column header onto the Group panel.

Dragging and dropping column headers to group the records is possible only if the Group Panel is displayed. You should have the Show Group Panel check box selected in the Browser Properties to view this section of the browser.

  • Using the Group by this column option - To group the records of a browser with this option, right-click on the required column header and select Group by this column.

To review sales performance over time, you can view the sales generated by all sales representatives per year and further group the sales quarter-wise. Additionally, you can view the total sales per quarter and year by specifying the Summary Type as “sum” for the Amount column. Click Show Me to view a multimedia presentation that displays how this can be done.
