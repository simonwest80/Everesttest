# Label Design


The label section displays the label that has been designed. You can  edit the various fields in the following ways:

- Resize the  label by dragging the borders to the required dimensions.
- Resize the  fields within the label.



**![]({{site.wwe_baseurl}}/img/lens.gif)  Properties**

- Modify the  label.



**![]({{site.wwe_baseurl}}/img/lens.gif)  Modifying  labels**

- Add or remove  text fields.
- Change the label  caption in a text box.
- Change the fonts  and colors of the fields.
