# Utility of Period Columns


These columns are useful in making a period comparison of the totals  of sales, purchases or number of customers, vendors, or users created.


{:.example}
To obtain a quarterly comparison of sales:

- From  the **Sales Invoices** browser, select  the summary type as **Sum** in the  **Browser Property Editor** (path:  From the browser select **Options &gt; 
 Properties &gt; Columns** tab **&gt; 
 Summary Type**).
- Group  the **Sales Invoices** browser first  by **Fiscal Year** and then by **Quarter**.

: ![]({{site.wwe_baseurl}}/img/lens.gif) [Group  by Box]({{site.wwe_baseurl}}/misc/group_by_box.html)

- Select  the **Chart** option to view the sales  in each quarter in each year.


{:.example}
To obtain a monthly comparison of number of customers created,

- From  the **Customers** browser, select  the summary type as **Count** in the  **Browser Property Editor** (path:  From the browser select **Options &gt; 
 Properties &gt; Columns** tab **&gt; 
 Summary Type**).
- Group  the Customers browser first by Year and then by Month.

: ![]({{site.wwe_baseurl}}/img/lens.gif) [Group  by Box]({{site.wwe_baseurl}}/misc/group_by_box.html)

- Select  the **Chart Setup** option from the  browser and specify **Record Count** in  Y-axis, **Year** in X-axis and **Month** in Z-axis to see a monthly comparison  of how many customers were added each year.
