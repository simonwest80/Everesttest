# Filters Drop-Down Menu Options


The **Filters** drop-down menu contains  options that enable you to run and create/modify Find and Advanced filters.


![]({{site.wwe_baseurl}}/img/lens.gif)  [Find  Function In Everest]({{site.wwe_baseurl}}/advanced-options/find-function/find_function_in_everest.html)


[Advanced  Find Filter]({{site.utl_chm}}/other-utilities/the-advanced-query-builder/the_query_builder_dialog_box.html)


The options available are:


**Filter List**
: The **Filters** drop-down  menu contains a list of filter types available in **Everest**.  You can select and run filters from this list. The types of filters available  are:

- Items
- Serial Numbers
- Customers
- Sales Documents
- Vendors
- Purchase Documents
- Account Activity
- Tasks/Appointments

: Select a filter type to view a list of filters under  that particular filter type. You can run filters, by clicking on the required  filter name.


{:.note}
The filter type will appear disabled in the  **Filters** drop-down menu, if there  are no filters set up under a filter type. Discontinued filters will not  be displayed in this menu.


**All Filters**
: Select this option to view the **Filters** browser, containing a list of all **Find** and **Advanced** filters created  in **Everest**. You can modify and  run existing filters.
: ![]({{site.wwe_baseurl}}/img/lens.gif)  [The Filters  Browser]({{site.wwe_baseurl}}/misc/the_filters_browser.html)


**Advanced Filters**
: Select this option to view a list of **Advanced** filters set up in **Everest**.  The following option is available under **Advanced** Filters:


**New**
: Select this option to bring up the **Advanced****Query Builder** dialog box which  allows you to build your own queries for retrieving data.
: ![]({{site.wwe_baseurl}}/img/lens.gif)  [Advanced  Query Builder]({{site.utl_chm}}/other-utilities/the-advanced-query-builder/the_query_builder_dialog_box.html)
