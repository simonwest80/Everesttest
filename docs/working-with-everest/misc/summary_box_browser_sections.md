# Summary Box


The summary for the column is displayed in this box. The summary box  is displayed only for those columns that have a summary type selected  in the [**Summary Type** field]({{site.wwe_baseurl}}/everest-client/ui/browsers/options/property-editor/browser_property_editor_columns_tab.html#summary_type_browser_properties_editor) (path: from the  browser **Options &gt; Properties &gt; 
 Columns** tab).
