# Rotate


Select **Rotate** from the **Edit** menu to display four sub-menu options  namely, **180**, **90 
 CW**, **90 CCW**, and **Custom**.  The selected objects rotate according to the sub-menu option selected.  The **Custom** option allows you to  rotate the object to any angle. The **Custom**  option is disabled if a barcode object is selected.
