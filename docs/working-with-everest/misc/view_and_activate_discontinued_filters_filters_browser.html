View and Activate Discontinued Filters

To view and activate a discontinued filter, do the following:

  • From the main menu bar, select Filters > All Filters. You will see The Filters Browser.
  • From the Filters browser, select Options > View Discontinued. You will see the discontinued filter in the Discontinued Filters browser.
  • From the Discontinued Filters browser, highlight the filter.
  • Select Options > Activate. You will be asked to confirm the activation.
  • Click OK. The activated filter will be displayed in the Filters browser.
