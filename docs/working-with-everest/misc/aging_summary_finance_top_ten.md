The **Accounts Receivable Aging** section  lists balances owed by your customers and displays the accounts receivable  based on the time period.


An aging analysis is displayed for the outstanding amounts based on  the total days outstanding. The number of days for which a document is  outstanding is calculated from the document's due date.


The periods are as follows:


| Total | Displays the total balance in the A/R account |
| Current | Displays the balance for the current system date |
| 30+ days | Displays the outstanding amounts exceeding 30 days |
| 60+ days | Displays the outstanding amounts exceeding 60 days |
| 90+ days | Displays the outstanding amounts exceeding 90 days |
| 120+ days | Displays the outstanding amounts exceeding 120 days |

The **Accounts Payable Aging** section  lists balances owed to vendors, and displays the amounts payable, based  on the time period.


An aging analysis is displayed for the outstanding amounts based on  the total days outstanding. The number of days for which a document is  outstanding is calculated from the document's due date.


The periods are as follows:


| Total | Displays the total balance in the A/P account |
| Current | Displays the balance for the current system date |
| 30+ days | Displays the outstanding amounts exceeding 30 days |
| 60+ days | Displays the outstanding amounts exceeding 60 days |
| 90+ days | Displays the outstanding amounts exceeding 90 days |
| 120+ days | Displays the outstanding amounts exceeding 120 days |
