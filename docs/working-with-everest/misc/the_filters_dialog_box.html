The Filters Dialog Box

This dialog box allows you to create new filters and modify existing filters for the selected browser. You can create and modify Classic Style and Expert Style filters. The Filters dialog box consists of the:

  • Top (Display) Panel
  • Search by <parameter> field
  • Filter Details Panel
  • Expression Panel

The corresponding SQL query that is displayed in the Expression Panel when a Classic Style filter is selected cannot be edited.

The options available in the Filters dialog box are:

Apply
Select this option to execute the selected filter.
Cancel
Select this option to close the Filters dialog box.
