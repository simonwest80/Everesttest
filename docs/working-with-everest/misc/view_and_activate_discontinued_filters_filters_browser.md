# View and Activate Discontinued Filters 


{:.steps}
To view and activate a discontinued filter,  do the following:

- From the main  menu bar, select **Filters &gt; All Filters**.  You will see [**The 
 Filters Browser**]({{site.wwe_baseurl}}/misc/the_filters_browser.html).
- From the **Filters** browser, select **Options 
 &gt; View Discontinued**. You will see the discontinued filter in  the **Discontinued Filters** browser.
- From the **Discontinued Filters** browser, highlight  the filter.
- Select **Options &gt; Activate**. You will be asked  to confirm the activation.
- Click **OK**. The activated filter will be displayed  in the **Filters** browser.
