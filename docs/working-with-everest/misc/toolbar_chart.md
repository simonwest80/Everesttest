# 'Title' Dialog Box - Toolbar Options


The options contained in the chart toolbar are detailed below:


| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_open_chart.gif) | Open Chart | Select this option to open an existing chart. |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_save_chart.gif) | Save Chart | Select this option to save the chart. |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_copy_to_clipboard.gif) | Copy to clipboard | Select this option to copy the chart to the clipboard. |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_gallery.gif) | Gallery | Select a chart type for the selected series or the entire chart. If  a particular series is selected, a multi-type chart will be displayed  in the chart area.<br/><br/><br/>![]({{site.wwe_baseurl}}/img/note.gif) You cannot combine all chart types. |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_vertical_color.gif) | Color | Click the drop-down button and select the required color. Drag the ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_color_icon.gif)icon and click the required area to change its color. |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_vertical_grid.gif) | Vertical grid | Click this button to show the vertical grid on the current graph. This  option is not available if you are currently viewing the 'Pie', 'Doughnut'  and the 'Pyramid' chart. To switch the vertical grid off, click the button  once again. |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_horizontal_grid.gif) | Horizontal grid | Click this button to show the horizontal grid on the current graph.  This option is not available if you are currently viewing the 'Pie', 'Doughnut'  and the 'Pyramid' chart. To switch the horizontal grid off, click the  button once again. |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_vertical_legend_box.gif) | Legend Box | Click this button to show  or hide the legend. The legend is only printed if you have chosen to display  the same on the window.<br/><br/><br/>![]({{site.wwe_baseurl}}/img/note.gif) A legend explains what the colors/ designs used in the chart indicate. |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_data_editor.gif) | Data Editor | Click this button to show or hide the data based on which the chart  is generated. |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_properties.gif) | Properties | Click this button to view and specify the chart properties. When you  select this option you will see the [**Chart FX <br/> Properties** dialog box]({{site.wwe_baseurl}}/everest-client/ui/browsers/options/chart/fx-profile/chart_fx_properties_profile.html). |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_2d_3d.gif) | 3D/2D | Click this button to view the chart in 3D or 2D mode. |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_rotate.gif) | Rotate | This option allows you to activate the 3-D rotated view for a 3-D chart.  When you select this option you will see the [**Chart FX <br/> Properties** dialog box]({{site.wwe_baseurl}}/everest-client/ui/browsers/options/chart/fx-profile/chart_fx_properties_profile.html). |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_z_clustered.gif) | Z-clustered | This option lets you display a series in different rows along the Z-axis.  In this case, you will view a 3-series chart. Each data series will occupy  one row of data and there will be 3 rows (Z-axis clusters) in the chart.  AREA charts are always clustered, as there is no way to paint different  series side by side. On the other hand, if you have a BAR chart, it is  possible to paint bars side-by-side and not clustered in the Z-axis. |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_zoom.gif) | Zoom | This option allows you to magnify a section of the chart. Click this  button and then select the area to be magnified. |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_print_preview.gif) | Print preview | Click this button to view the chart. You will see the [**Page Setup** dialog box]({{site.wwe_baseurl}}/misc/page_setup_chart_options.html). You can specify  additional print settings here. |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_print.gif) | Print | Prints the current graph. Click on this option to view the **Print**  dialog box where the printer and the print settings have to be defined.  The graph title, the graph and the legend (if you have the **Series** or **Values Legend** option  on) are printed. |
| ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_tools.gif) | Tools | Click this button to access the 'Tools' available with the Graph editor.<br/><br/><br/>**Values Legend** - Choose this  option to display the legend for the X-axis.<br/><br/><br/>**Series Legend** - Choose this  option to display the legend for the Z-axis.<br/><br/><br/>**Data Editor** - Choose this option  to view the data editor. The data editor displays the data based on which  the chart has been generated.<br/><br/><br/>**Palette Bar** - Choose this option  to change the colors in the graph.<br/><br/><br/>**Pattern Bar** - Choose this option  to change the patterns of the bars in the graph. |
