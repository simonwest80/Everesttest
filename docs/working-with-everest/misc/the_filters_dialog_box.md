# The Filters Dialog Box


This dialog box allows you to create new filters and modify existing  filters for the selected browser. You can create and modify **Classic 
 Style** and **Expert Style**  filters. The **Filters** dialog box  consists of the:

- [Top  (Display) Panel]({{site.wwe_baseurl}}/misc/filters_-_top_(display)_panel_filters_dialog_box.html)
- **Search by &lt;parameter&gt;**  field
- [Filter  Details Panel]({{site.wwe_baseurl}}/misc/filter_details_panel_filters_dialog_box.html)
- [Expression  Panel]({{site.wwe_baseurl}}/misc/expression_panel_filters_dialog_box.html)



{:.note}
The corresponding SQL query that is displayed in the **Expression 
 Panel** when a **Classic Style**  filter is selected cannot be edited.


The options available in the **Filters**  dialog box are:


**Apply**
: Select this option to execute the selected filter.


**Cancel**
: Select this option to close the **Filters**  dialog box.
