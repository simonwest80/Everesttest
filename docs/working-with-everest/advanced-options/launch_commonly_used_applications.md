# Launch Commonly Used Applications


You can launch commonly used applications such as MS Excel or Word from  within **Everest**.


{:.steps}
To set up an application to be launched from  Everest, do the following:

- From the main  menu bar, select **Utility &gt; Other Programs 
 &gt; Configure Other Programs**. You will see the [Configure  Utilities dialog box]({{site.wwe_baseurl}}/misc/configure_utilities_dialog_box.html).
- Specify the  details and click **OK** to save the  details.



{:.steps}
To launch the application from Everest, do the following:

- From the main  menu bar, select **Utility &gt; Other Programs**.  You will see a submenu listing all the applications that can be launched  from **Everest**.
- Select the  required application.
