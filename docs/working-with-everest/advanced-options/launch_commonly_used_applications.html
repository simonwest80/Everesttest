Launch Commonly Used Applications

You can launch commonly used applications such as MS Excel or Word from within Everest.

To set up an application to be launched from Everest, do the following:

  • From the main menu bar, select Utility > Other Programs > Configure Other Programs. You will see the Configure Utilities dialog box.
  • Specify the details and click OK to save the details.

To launch the application from Everest, do the following:

  • From the main menu bar, select Utility > Other Programs. You will see a submenu listing all the applications that can be launched from Everest.
  • Select the required application.
