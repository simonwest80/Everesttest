# Number of Records to Retrieve


This option is used to specify the number of records to be displayed  when you carry out a search.

- **Custom**  - If you choose this option, you can specify a numeric value for the number  of records to retrieve.



{:.example}
If you specify 10, the **Find**  function will retrieve all the records matching the parameters set, but  will display only 10 records at a time.
: However, in the resulting browser, you can choose  to get **More** or **All**  records.
: ![]({{site.wwe_baseurl}}/img/lens.gif) Â [Standard  Browser Options]({{site.wwe_baseurl}}/everest-client/ui/browsers/standard_browser_options.html)

- **All**  - If you choose this option, all records matching the filter parameters  will be retrieved.
