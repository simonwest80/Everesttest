# Finance


The **Finance** option displays  all the finance-related details, such as the aging summary and the top  five aging accounts.


{:.steps}
To view the finance information, do the following:

- Click on the  **Finance** option from the **Everest** desktop. You will see the **Finance** window.

: The **Finance**  window displays two sections:

- [Aging  Summary]({{site.wwe_baseurl}}/misc/aging_summary_finance_top_ten.html)
- [Top  Five Aging Accounts]({{site.wwe_baseurl}}/misc/top_five_aging_accounts_finance_top_ten.html)
