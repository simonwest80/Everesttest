# Resize Icons


The size of icons on the toolbars can be increased by selecting the  **Large icons** check box in the **Options** tab of the [**Customize** dialog box]({{site.wwe_baseurl}}/misc/customize_dialog_box.html).


To reduce the size of icons, this option must be unchecked.
