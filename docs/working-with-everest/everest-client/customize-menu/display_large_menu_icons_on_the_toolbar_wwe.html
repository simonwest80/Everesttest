Display Large Menu Icons on the Toolbar

You can display large menu icons on the toolbar for easy viewing.

To display large menu icons on the toolbar, do the following:

  • Select Customize from the right-click menu on the Main Menu or toolbar. You will see the Customize dialog box.
  • Select the Options tab.
  • Select the Large icons check box.
  • Click the Close button.

To return the icons back to the original size, deselect the Large icons check box again from the Options tab of the Customize dialog box.
