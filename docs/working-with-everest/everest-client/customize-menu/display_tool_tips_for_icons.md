# Display ToolTips  for Icons


ToolTips  display the name of the option or command associated with a button. To  display the ToolTips  for icons select the **Show ToolTips 
 on toolbars** check box in the **Options** tab of the [Customize dialog  box]({{site.wwe_baseurl}}/misc/customize_dialog_box.html).
