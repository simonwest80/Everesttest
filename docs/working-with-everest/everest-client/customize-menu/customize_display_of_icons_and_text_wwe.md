# Customize Display of Icons and Text 


You can customize the display of the image and text of the  menus and sub-menus.


{:.steps}
For a display of both image and text for  a menu, do the following:

- Select **Customize**  from the right-click menu on the Main Menu or Toolbar. You will see the  **Customize** dialog box.
- With the **Customize** dialog box open, select the  option **Image and Text** from the  right-click menu on the icon for which you wish to add text. You will  see the text of the menu appear next to the icon.
- To remove the text  from the icon again, select **Default style**  from the right-click menu on the icon.
- To align the image  and text, select the **Image on left**,  **Image on right**, **Image 
 on top** or **Image on Bottom**  option from the right-click menu on the icon.

- Click **Close**  to close the **Customize** dialog  box.



{:.note}
The **Text 
 Only (Always)** and **Text Only (in 
 Menus)** options displayed in the right-click menu for an icon are  obsolete and should not be used.
