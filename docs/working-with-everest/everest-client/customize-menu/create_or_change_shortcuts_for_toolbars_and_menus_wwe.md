# Create or Change Shortcuts for Toolbars and Menus


You can create or change the shortcut assigned to a toolbar or menu.


{:.example}
Shortcuts are useful for quickly opening frequently  used features. For example, you can press Shift+F2 to open the Purchase  Orders browser from the Everest desktop.


{:.steps}
To create or change the shortcut of the toolbar  or menu, do the following:

- Right-click on  the Toolbar and select **Customize**.  The **Customize** dialog box displays.
- With the **Customize** dialog box open, right-click  the menu icon for which you intend to create or modify the shortcut.
- Type the Shortcut  key combination in the **ShortCut**  field. You can use any of the special keys (Ctrl, Alt, Shift) in combination  with numeric, character, or function keys by inserting the + sign. For  example, you can change the Shortcut for Purchase Orders to Shift+F4 as  shown in the following figure.
