Display only Recently Used Commands

Submenus can be customized to display only those options that are frequently used by you.

To set up Everest to display only recently used commands, do the following:

  • Ensure that the Enhanced or Flat option has been selected as the Bar Style in the Workstation Settings dialog box (path: File > Setup > Preferences > Current Environment Settings > General tab).
  • Select the Options tab in the Customize dialog box.
  • Select the Menus show recently used commands first check box.
  • Click Close to save the settings.
