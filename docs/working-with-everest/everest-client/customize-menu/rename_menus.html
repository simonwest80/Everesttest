Rename Menus

You can change the display name of the menus in the Main Menu Bar.

To rename a menu in the main menu bar, do the following:

  • Right-click on any of the toolbars and select Customize from the drop-down menu. You will see the Customize dialog box.
  • Right-click on the required menu in the main menu bar and specify the new name in the Name field.
  • Press the Enter key to save the new name and return to the Customize dialog box.
