View Outlook Bar

The Outlook Bar appears in the left panel of the Everest main window. This panel contains shortcuts to frequently used options in Everest. The panel is divided into tabs based on the functions in the main menu. The tabs are: Accounting, Inventory, Purchasing, Invoicing and Contact Manager. Each tab contains shortcuts that help you perform activities in Everest.

Under the Inventory tab, you have a shortcut for creating a new item.

To activate the Outlook Bar, do the following:

  • From the main menu bar, select Windows > View Outlook Bar. You will see the Outlook Bar on the left panel of the Everest main window.

When the Outlook Bar is activated, a check mark appears beside the View Outlook Bar option under the Windows menu option.

To deactivate the Outlook Bar, do the following:

  • From the main menu bar, select Windows > View Outlook Bar. The Outlook Bar will disappear from the left panel of Everest main window.
