# Cascade


This option allows you to arrange all the windows that you have opened  in an orderly fashion. The windows are placed slightly offset from each  other. This option helps you identify all the windows that are open.


{:.steps}
To activate the Cascade option, do the following:

- From the main  menu bar, select **Windows &gt; Cascade**.  All the open windows will be displayed slightly offset from each other.
