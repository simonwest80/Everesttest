# Print Barcode Labels for Items


You can print barcode labels for your items from the Items  browser.


{:.steps}
To print barcode labels for items, do the  following:

- Select **Inventory**  > **Items**.
- From the **Items** browser, select the items that  need a printed label.
- Select **Label**  > **Label Preview** > <Label  Name>.



![]({{site.wwe_baseurl}}/img/preview_barcode_label_wwe.gif)


**Figure: Items Browser with Items for Barcode  Labels Selected**


![]({{site.wwe_baseurl}}/img/print_barcode_labels_wwe.gif)


**Figure: Barcode Labels - Preview Group of Labels**

- When you are satisfied  with the preview and label layout, click **Label**  > **Label Print** > <Label  Name>. The system prints the selected barcodes.
