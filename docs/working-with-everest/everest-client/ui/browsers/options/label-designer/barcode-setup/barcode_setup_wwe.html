Barcode Setup

Barcode labels allow quick data entry into Everest through the use of a scanner and help you to manage your inventory. You can create and print barcode labels from Everest and quickly scan those labels into Everest. To set up barcodes in Everest, you need to complete the following tasks:

  1. Configure your Barcode Labels

  2. Print Barcode Labels for Items

  3. Test Barcode Scanning

  4. Set Barcode Preferences (Optional)
