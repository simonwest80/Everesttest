# Barcode Setup


Barcode labels allow quick data entry into **Everest**  through the use of a scanner and help you to manage your inventory. You  can create and print barcode labels from Everest and quickly scan those  labels into **Everest**. To set up  barcodes in **Everest**, you need  to complete the following tasks:


1. [Configure  your Barcode Labels]({{site.wwe_baseurl}}/everest-client/ui/browsers/options/label-designer/barcode-setup/configure_your_barcode_labels_wwe.html)


2. [Print  Barcode Labels for Items]({{site.wwe_baseurl}}/everest-client/ui/browsers/options/label-designer/barcode-setup/print_barcode_labels_for_items_wwe.html)


3. [Test Barcode Scanning]({{site.wwe_baseurl}}/everest-client/ui/browsers/options/label-designer/barcode-setup/test_barcode_scanning_wwe.html)


4. [Set  Barcode Preferences (Optional)]({{site.wwe_baseurl}}/everest-client/ui/browsers/options/label-designer/barcode-setup/set_barcode_preferences_optional_wwe.html)
