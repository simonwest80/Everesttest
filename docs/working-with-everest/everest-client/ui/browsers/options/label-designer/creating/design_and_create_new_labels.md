# Design and Create New Labels


You can design and create a new label by using the fields in the selected  browser for populating the fields in the label. You can also add text  fields where you can input text.


{:.steps}
To create a label, do the following:

- From the required  browser, click on ![]({{site.wwe_baseurl}}/img/wwe_label_menu_icon.gif) and select **Design**.  You will see the[ Label Designer dialog  box]({{site.wwe_baseurl}}/everest-client/ui/browsers/options/label-designer/creating/designer/the_label_designer.html).
