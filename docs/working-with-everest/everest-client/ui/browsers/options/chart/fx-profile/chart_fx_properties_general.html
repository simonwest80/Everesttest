The Chart FX Properties Dialog Box - General

Effects

3D
Select this check box to display the chart in 3D mode.
Cluster
Select this check box to display each series in a different row along the Z-axis.
Stacked
Select whether the chart should be displayed in stacked, stacked 100% or side-by-side mode.
Axes Style
Specify the painting style for the axes. The options available are 3D Frame, Math and Flat Frame.
Grid lines
Specify whether horizontal or vertical gridlines or both have to be displayed at the major interval of the numerical (Y) or categorical (X) axis, respectively.

Colors

Palette
Select a specific palette to change all elements in the chart area.
Scheme
Specify the scheme to be applied to the elements in the chart area. The default is solid colors. You can also choose B&W or colored patterns.
Background
Select the color you want in the area surrounding the chart.
Chart Box
Select the color you want for the plot area.
Title
Enter the title of the chart.
