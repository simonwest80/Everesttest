# The Chart FX  Properties Dialog Box


The **Chart FX 
 Properties** dialog box is displayed when the **Properties** icon ![]({{site.wwe_baseurl}}/img/wwe_title_toolbar_options_properties.gif) is clicked. This dialog box consists of the  following tabs:

- [General]({{site.wwe_baseurl}}/everest-client/ui/browsers/options/chart/fx-profile/chart_fx_properties_general.html)
- [Series]({{site.wwe_baseurl}}/everest-client/ui/browsers/options/chart/fx-profile/chart_fx_properties_series.html)
- [Axes]({{site.wwe_baseurl}}/everest-client/ui/browsers/options/chart/fx-profile/chart_fx_properties_axes.html)
- [3D]({{site.wwe_baseurl}}/everest-client/ui/browsers/options/chart/fx-profile/chart_fx_properties_3d.html)
