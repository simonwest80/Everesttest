# Change the Browser View


The browser view can be changed to suit your requirements. You can hide  the grid lines in the browser, change the font of the column headers,  change the color scheme of the browsers, hide columns and specify the  number of records to be retrieved by default. All these settings are specified  in the **Browser Property Editor**  dialog box.


You can access the Browser Properties Editor in any  of the following ways:

- Selecting **Options &gt; Properties** in the required  browser.

: OR

- Clicking on  the **Properties** icon ![]({{site.wwe_baseurl}}/img/wwe_browser_ property_ icon.gif) in the browser toolbar.

: OR

- Select **Properties** from the right-click menu.  You will see the [**Browser 
 Property Editor** dialog box]({{site.wwe_baseurl}}/everest-client/ui/browsers/options/property-editor/the_browser_property_editor.html).
