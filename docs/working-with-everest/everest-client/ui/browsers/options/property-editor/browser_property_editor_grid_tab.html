Browser Property Editor - Grid

The Grid tab allows you to specify the font, font size and color scheme of the browser.

Grid Attributes

Font
Click the Font button to view the Font Setup dialog box. This screen enables you to specify the font settings such as size and color for the records contained in the browser.
Color
Specify the color for the borders of each record in the browser. This color will be displayed only when the browser is grouped and the grid lines are not displayed i.e., the Show grid lines option is not selected.
Show grid lines
Select this check box to view the grid lines in the browser.
Grid Line Color
Use this option to select the color of the grid lines from the drop-down list. This option is enabled only if the Show grid lines check box is selected.
Show row indicator
Select this check box to display an additional column in the browser that indicates the currently selected row.
Show group panel
Select this check box to view the group panel that displays the fields based on which the records in the browser have been grouped.
 Group by Box
Alternating colors
Select this check box to specify different colors for every other row in the browser. You can specify even and odd row colors only if this box is checked.
Even Row Color
Specify the color of even rows in the browser.
Even Row Text Color
Specify the color of the text in even rows of the browser.
Odd Row Color
Specify the color of the odd rows in the browser.
Odd Row Text Color
Specify the color of the text in odd rows of the browser.

The even- and odd-row color and text color options are enabled only if the Alternating colors check box is selected.

Column Header

Font
Click the Font button to view the Font Setup dialog box. This screen enables you to specify the font settings such as size and color for the column headers and group by box.
 Group by Boxes
Color
Specify the color of the background for column headers in this field.

Summary Colors

This section allows you to specify the color scheme of the summary box for each browser grouping.

Summary Level
In browsers, each grouping is assigned a level.
If the **Sales
Invoices** browser has been grouped by sales representatives, currency and then by customer, the customer grouping is assigned group level 1, currency grouping is assigned group level 2 and the sales representatives grouping is assigned group level 3.
For summarizing data each grouping is assigned a summary level. This option allows you to specify the level for which you will be defining the color of the summary box in the following field. The Grid Footer option allows you to specify the level for the summary box that is displayed at the bottom of the browser and at the end of each grouping.
 Summary Box
Sections of a Browser
Color
Specify the color of the summary box for each summary level. Select the summary level in the previous field and specify the color for each summary level in this field.
