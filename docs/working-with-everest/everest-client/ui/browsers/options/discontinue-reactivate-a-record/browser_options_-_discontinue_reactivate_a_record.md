# Browser Options - Discontinue/Reactivate a Record


**Everest** allows you to temporarily  [discontinue records]({{site.wwe_baseurl}}/misc/discontinue_a_record.html). When a record  is discontinued, it is not deleted from the database; it is merely deactivated.  You can view all discontinued records from the **Discontinued 
 &lt;Records&gt;** browser.


A discontinued record can be made available for selection by [activating]({{site.wwe_baseurl}}/misc/activate_a_discontinued_record.html)  it again.
