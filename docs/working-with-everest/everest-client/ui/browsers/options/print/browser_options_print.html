Browser Options - Print

The icon allows you to print details of the records contained in the active browser. You can print all the records displayed in the browser. You can even select the records that have to be included. You can also customize the output based on the columns that you want to include in the report.

To print records from a browser, do the following:

  • From the browser toolbar, click on the Preview Browser icon. You will see the Print Setup dialog box.
