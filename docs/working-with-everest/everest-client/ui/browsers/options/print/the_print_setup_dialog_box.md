# The Print Setup Dialog Box


The **Print Setup** dialog box enables  you to specify the records that have to be included in the print report.


**Columns to be printed**
: Select the columns to be included in the report.  A check mark is displayed beside a column that is selected.


**Selection  criteria**


**Selected records**
: Select this option to include only the selected  records in the report.


**All records**
: Select this option to print all the records in the  report.


**Expand selected groups**
: Select this option to print the records under each  group. This option is enabled only if the browser has been grouped. If  this option is not selected, only the number of records in each grouping  is displayed in the report.


{:.note}
This option is displayed when the **Selected 
 Records** option is chosen from **Selection 
 Criteria**.

**Expand Groups**
: Select this option to print all the records under  each group.


{:.note}
This option is displayed when the **All 
 Records** option is chosen from **Selection 
 Criteria**.


**Page Setup**
: Click this button to set margins, paper source,  paper size and page orientation for the report.


**Preview**
: Click this button to view how the report will look  when you print it.


**Print**
: Click this button to directly print the report.  The **Print** dialog box will be displayed  wherein you have to specify the print settings and the printer.


**Cancel**
: Click this button to exit the **Print 
 Setup** dialog box without printing.
