# Standard Menu Options in Profiles


Common options available in a profile and their functionality are briefly  explained below:


| **Button** | **Name** | **Description** |
| ![]({{site.wwe_baseurl}}/img/wwe_profile_options_new.gif) | New | Use this option to create a new profile. |
| ![]({{site.wwe_baseurl}}/img/wwe_profile_options_save.gif) | Save | Use this option to save the changes made to the profile and continue  working with it. |
| ![]({{site.wwe_baseurl}}/img/wwe_profile_options_save_and_close.gif) | Save and close | Use this option to save the changes made and close the profile. |
| ![]({{site.wwe_baseurl}}/img/wwe_profile_options_cancel.gif) | Cancel | Use this option to cancel the changes made and revert to the originally  saved profile. |
| ![]({{site.wwe_baseurl}}/img/wwe_standard_browser_profile_options_help.gif) | Help | Use this option to get context-sensitive help for the specific profile. |
