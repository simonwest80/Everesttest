# Login Location/Sub-Location


Specify the location/sub-location in which the transactions have to  be accounted for. You can change the location/sub-location in the [Work Station Settings]({{site.sc_chm}}/options/miscellaneous-set-up/workstation-settings/work_station_settings.html) dialog box.


{:.note}
A security right is required to change the  Location/Sub-location in the Login screen to any other Location/Sub-location.
