The Login Information Dialog Box

Use the Login Information dialog box to change the logged-in user or to change your password. You can also view the currently logged in user.

To change the Logged in user, do the following:

  • Click Change User. The Change User dialog box is displayed. See The Change User Dialog Box for more information.

To change your password, do the following:

  • Click Change Password. The Change Password dialog box is displayed. See The Change Password Dialog Box for more information.
