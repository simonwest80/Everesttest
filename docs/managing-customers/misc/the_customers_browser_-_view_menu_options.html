The Customers Browser - View Menu Options

My Customers
Select the My Customers option to view the records of customers assigned to you. This option is selected by default if you have set the preference Restrict sales reps from viewing all customers (path: File > Setup > Preferences > Flow Control > All > Miscellaneous > Others > Others 2).
All Customers
You can view the records of all other customers by selecting the All Customers option, provided you have adequate security rights.
