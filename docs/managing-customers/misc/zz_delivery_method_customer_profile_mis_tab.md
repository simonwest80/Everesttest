---
title: Delivery Method
---


Specify the delivery method to be used for this customer. When you create a new record, this field defaults to the delivery method specified in the **Customers** dialog box (path: **File 
&gt; Setup &gt; Preferences &gt; Defaults &gt; Customers** > **Delivery Method** field).
