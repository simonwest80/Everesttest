# Modify Address Dialog Box


This dialog box enables you to specify whether address details previously  used on documents should be updated or preserved.


**Update address information on existing documents**
: Select this option to update the old address details  of the customer on existing documents.


**Preserve address information on existing documents**
: Select this option to preserve the old address details  of the customer on existing documents.
