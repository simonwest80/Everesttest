Specify the currency to be used for invoicing the customer. Once saved, this field cannot be modified. This is a mandatory field. By default, this field displays the currency code specified in the Base Currency field of the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting > Multi Currency > Accounting tab).
