The Customer Profile - General

This tab allows you to enter basic details for the customer. The address entered in this tab is displayed as the billing address for the customer.

General Information **

  • Account code
  • Residential
  • Account name
  • Title
  • First name and Last name
  • Middle name

Address **

  • Address, City, State/Province
  • Zip/Postal code
  • Country

Contact Information **

  • Telephone code
  • Abbreviation
  • Telephone - 1, Extension
  • Telephone - 2, Extension
  • Fax, Extension
  • Mobile Cell
  • Pager
  • E-mail ID
  • Web URL
  • Destination zone
  • Apply Masks
