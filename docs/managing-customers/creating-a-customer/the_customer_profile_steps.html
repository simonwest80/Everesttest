The Customer Profile

The Customer profile contains the following tabs:

  • General
  • Addresses
  • Payment information
  • Accounting
  • Finance Charges
  • Document Policy
  • Miscellaneous
  • Security
  • E-Commerce
  • Logo
  • Multimedia
  • Custom Fields

The Customer profile contains an Options menu, with options common to the Customers browser.

 The Multimedia and Custom Fields tabs are displayed only for a saved profile.
