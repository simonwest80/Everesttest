Country

This field allows you to specify the country in which the customer’s address is located.

Everest automatically populates the City and State/Province fields when you select a zip/postal code for the United States from the drop-down list.

By default, the country specified in the Location profile (path: File > Setup > Accounting > Location/Sub-Location > Options > Edit > In New Window > Location profile > General tab) of the login location/sub-location is displayed in this field.
