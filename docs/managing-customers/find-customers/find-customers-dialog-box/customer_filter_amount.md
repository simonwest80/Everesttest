# Find Customers - Amount


The **Amount** tab facilitates a  search based on the balances due from or to the customer.


**Balance in Base Currency**
: Select the relevant option and enter the balance  amount payable (by the customer) in the base currency.


**![]({{site.mc_baseurl}}/img/lens.gif)[Balance  in Base Currency]({{site.mc_baseurl}}/misc/balance_in_base_currency.html)**


**Balance in Customers Currency**
: Select the relevant option and enter the balance  payable (by the customer) in the customer's currency.


**![]({{site.mc_baseurl}}/img/lens.gif)[Balance  in Customers Currency]({{site.mc_baseurl}}/misc/balance_in_customer_currency.html)**
