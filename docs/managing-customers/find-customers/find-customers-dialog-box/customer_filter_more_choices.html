Find Customers - More Choices

The More Choices tab provides parameters in addition to the parameters defined in the Quick Find tab.

 The parameters in the More Choices tab will be populated automatically when you select an existing filter (that contains the relevant More Choices tab parameters).

City
Enter the name of the customer’s city.
City Details
State
Enter the name of the customer’s state.
State Details
Zip Code
Enter the zip code of the customer’s address.
Zip Code Details
Country
Enter the customer’s country.
Country Details
Created Date
Specify the created date.
Created Date
Payment Terms
Specify the payment terms.

Payment Terms

User Created
Specify the user who has created the customer in the Customer profile.

User Created

On Credit Hold
Specify whether customers on credit hold should be included.
Credit Hold Details
Account Lockout
Specify the account lockout status of a customer to be used as a search filter.
