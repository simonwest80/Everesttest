# Contact Manager


The **Contact Manager** option (path:  **Customers browser &gt; Options &gt; Contact 
 Manager**) allows you to create tasks and appointments for users,  and associate them to the selected customer. The options available in  the **Contact Manager** menu are listed  below:


**New Task**


This option allows you to create a task for the selected customer. The  customer’s account code is automatically updated in the **Task**  profile.


![]({{site.mc_baseurl}}/img/lens.gif) [Create  a Task]({{site.cm_chm}}/tasks/create-a-task/creating_a_task.html)


**New Incoming Call**


This option allows you to record and track details of telephone calls  from the selected customer.


![]({{site.mc_baseurl}}/img/lens.gif) [Incoming  Call]({{site.cm_chm}}/tasks/incoming-call/incoming_call.html)


**New Outgoing Call**


This option allows you to record and track details of telephone calls  to the selected customer.


![]({{site.mc_baseurl}}/img/lens.gif) [Outgoing  Call]({{site.cm_chm}}/tasks/outgoing-call/outgoing_call.html)


**New Note**


This option allows you to record any relevant details of the selected  customer, and make it available for other users as well.


![]({{site.mc_baseurl}}/img/lens.gif) [General  Notes]({{site.cm_chm}}/tasks/general-notes/general_notes.html)


**Task History**


This option allows you to view details of tasks for the selected customer.


![]({{site.mc_baseurl}}/img/lens.gif) [Task]({{site.cm_chm}}/tasks/task-details/task_details.html)


**My Task History**


This option displays details of tasks defined for the current logged  in user, with respect to the selected customer.


![]({{site.mc_baseurl}}/img/lens.gif) [Task]({{site.cm_chm}}/tasks/task-details/task_details.html)


**New Appointment**


This option allows you to create and assign appointments for all the  users in Everest, with respect to the selected customer.


![]({{site.mc_baseurl}}/img/lens.gif) [Appointment]({{site.cm_chm}}/appointments/appointment.html)


**Appointment History**


This option allows you to view details of appointments for the selected  customer.


![]({{site.mc_baseurl}}/img/lens.gif) [Appointment]({{site.cm_chm}}/appointments/appointment.html)


**Customer’s Calendar**


This option displays the Contact Manager profile, which allows you to  view and manage tasks and appointments for the selected customer.


![]({{site.mc_baseurl}}/img/lens.gif) [The  Everest Contact Manager]({{site.cm_chm}}/the_everest_contact_manager.html)
