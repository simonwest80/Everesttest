# Print Label


This option displays the **Select a Report** dialog box, containing options to print the billing address of  the customer onto labels.


{:.steps}
To print labels for a customer, do the following:

- From the **Customer** profile, select **Options** **&gt;Print Label**. You will  see the [**Select****a** **report**]({{site.mc_baseurl}}/misc/select_a_report_dialog_box_managing_cust.html)  dialog box.
- Click the **Print** option to print the label or click  the **Preview** option to view the  report.
