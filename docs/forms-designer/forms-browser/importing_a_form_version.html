Importing a Form Version

Use the Import Forms option to import customized form versions that were created using ExportForms option from Everest database.

: You can use the imported custom form version to update the existing custom form version.

Updating Existing Forms

To import customized form versions, do the following:

  • From the Forms Browser, select Options > Import Forms. Click Import Forms. You will see the standard windows file Open dialog box.
  • Specify the file name to be imported and click Open. You will see the Import Dialog – Forms Browser.
