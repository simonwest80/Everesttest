{:.steps}
To change an object's position, do the following:

- Right click any object and select **Alignment**  to change the object's horizontal and vertical position on the target  form. The **Alignment** dialog box  appears.
- Click a horizontal alignment choice to select  it.
- Click a vertical alignment choice to select it.
- Click **OK**  after you have made your selections.

{:.steps}
To change an object's size, do the following:

- Right click any object and select **Size**  to change the object's height and width. The **Size**  dialog box appears.
- In the **Width**  section, select the appropriate width option.
- If you selected **Width**,  enter the width in the Width field.
- In the **Height**  section, select the appropriate height option.
- If you selected **Height**,  enter the height in the Height field.
- Click **OK**  after you have made your selections.

{:.steps}
To change the order of the tabs, do the following:

- Right click any object and select Tab Order to  change the order of the tabs. The **Tab 
 Order** dialog box appears.
- Use the **Up**  and **Down** arrows to move the position  of the tabs.
- When you have arranged the tabs in the appropriate  order, click **OK**.

# Right Click Options
