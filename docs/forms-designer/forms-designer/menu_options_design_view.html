This option opens the Component Palette dialog box. Use the Component Palette to add window controls to the target form.

This option opens the Field Chooser dialog box. Use the Field Chooser to add fields to the target form.

This option opens the Object Inspector dialog box. Use the Object Inspector to change the properties of controls and objects, such as caption name, position, font, color, and size.

This time and effort saving feature allows you to choose a previously created version as the template for your target form.

Use the Save option to save your current work to a new form. If the Standard Version of the form is edited and saved, it is saved as a different version. The Everest system checks the new form being saved for errors. A new form cannot be created until the errors are resolved.

  • Form Name – Enter a unique name for your new form in this field.
  • Document Alias – Specify the document alias that the new form will apply to.

Use the Save As option to give a version of the standard form a name or to save your current work as a previously saved version, replacing the earlier version.

  • Form Name – Enter a unique name for your new form in this field or enter the name of an existing form version to overwrite that version.
  • Document Alias – Specify the document alias that the new form will apply to.

This option adds all mandatory fields to a highlighted location on a target form, making it easier to create a form with all the mandatory fields on one tab.

When you try to add mandatory fields in Sales and Credit documents, the controls will overlap.

This option removes all edits completed before the last save. If you were creating a new version of a profile, the target form reverts to the form it was created from.

This function exits the Design view and returns you to the standard view. If changes were made to the current form, you will receive a dialog box concerning saving the changes.

Menu Options

In Design View, a Designer menu item is visible to use when customizing forms.

To access the Menu Options, do the following:

From the Profile (Design) menu, select Designer.

The following menu options are available from design view:

  • Component Palette Menu Option
  • Field Chooser Menu Option
  • Object Inspector Menu Option
  • Load
  • Save
  • Save As
  • Add Mandatory
  • Restore to Last Save
  • Exit Design Mode
