- On the **Toolbars**  tab, click the **New** button. The  **Custom** dialog box appears.
- Enter the new toolbar name and click **OK**.  The **Rename** and **Delete**  buttons are now activated for the new toolbar created.
- Click **Close**.  The new toolbar is added to the profile.

- Right click the **Profile**  menu bar and click **Customize**.  The **Customize** dialog box appears.
- Click the **Commands**  tab.
- Highlight a **Category**  and drag and drop **Commands** onto  the new toolbar as necessary.

- Click the **Options**  tab. Select any additional menu or toolbar options.

 - Large Icons
 - Show Tooltips on Toolbars
 - Show Shortcut Keys in Tooltips
 - Menu Animations

- Click **Close**.

# Make any Necessary Changes to the Menu and Toolbar


{:.steps}
To make changes to the Menu and Toolbar, do the following:

- Right click the Profile menu bar and select **Customize**. The **Customize**  dialog box appears. Use the **Toolbars**  tab to create new toolbars, rename, and delete them.



{:.note}
: When the Forms Designer is enabled, the customizable  forms do not allow menu and toolbar customization in normal mode.
