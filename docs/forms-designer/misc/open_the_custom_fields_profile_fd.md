# Open the Custom Fields Profile


{:.steps}
To open the Custom Fields profile, do the  following:

- Using the **Sales Opportunity** browser as an example,  from the **Sales Opportunity** browser,  select **Options** > **Custom 
 Fields** or press **Ctrl + F**.  The finished custom form will be accessible from all of the sales document  browsers. You will see the **Custom Fields**  profile that contains the custom fields applicable to the selected browser.
