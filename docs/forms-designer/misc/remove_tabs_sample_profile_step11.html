Remove Tabs

To remove all tabs except the Items tab, do the following:

  • Get a tab to be deleted in focus. (Select the tab control of the tab to be deleted.)
Tab Control
Select a Tab Control
  • Press the Delete key. The tab is removed from the profile.
  • Repeat the above steps to delete all of the remaining tabs except the “Items” tab.
