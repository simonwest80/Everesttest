Design the Target Form

Design the target form by completing any combination of the following tasks, as necessary to complete your design.

  • Add Fields to the Target Form
  • Add Additional Controls Using the Component Palette
  • Modify a Control or Object
  • Tab and Page Controls
  • Select the Page Control
  • Select a Tab Control
  • Add a New Tab
  • Rename a Tab
  • Reorder Tabs
  • Delete a Tab
  • Change an Object’s Position
  • Change an Object’s Size
  • Change the Tab Stop Order
  • Make any Necessary Changes to the Menu and Toolbar

Use Save or Save As to save your changes.
