# Assign Groups to a Form Version


The **Assign Forms** dialog box  for groups displays the groups that are assigned to the specific version  of a form selected.


The **Assign Forms** dialog box  for groups (path: **Forms Browser**  > **Options** > **Assign****Forms** > **Groups**)  contains two sections: [Assigned](javascript:kadovTextPopup(this)){:id="a1"}<script type="text/javascript" language="JavaScript1.2"><!--

//--></script>.


{:.steps}
To  grant groups a version of a form, do the following:

- Move the group or groups to be assigned from the  **Unassigned** section to the **Assigned** section using the arrow keys.



| Arrow Key | Function |
| ![]({{site.fd_baseurl}}/img/arrow to left.bmp)  **Single Left Arrow** | Moves a selected group to the section on the left |
| ![]({{site.fd_baseurl}}/img/double arrow to left.bmp)  **Double Left Arrow** | Moves all groups to the section on the left |
| ![]({{site.fd_baseurl}}/img/arrow to right.bmp)  **Single Right Arrow** | Moves a selected group to the section on the right |
| ![]({{site.fd_baseurl}}/img/double arrow to right.bmp)  **Double Right Arrow** | Moves all groups to the section on the right |


- After the appropriate groups have been assigned  to the customized form versions, click **Save**.



{:.note}
You can also double-click an item to move  it from the **Unassigned** section  to the **Assigned** section and back  without using the arrows.
