View a Point of Sale (Credit Memo) Profile

You can view the Point of Sale(credit memo) profile from one of the following locations:

  • The Credit Memos browser - Highlight the record and select Options > Edit > In Current Window/In New Window.

You can view only Point of Sale (credit memo) documents from the Credit Memos browser by selecting View > Pointof Sale. All the other options under the View menu should be deselected.

  • The Credit Memo profile - Specify the document number of the required memo in the Document # field.
