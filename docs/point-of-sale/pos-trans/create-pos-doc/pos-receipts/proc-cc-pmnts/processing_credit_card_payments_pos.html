Processing Credit Card Payments

You ca process credit card payments from customers from Everest or outside Everest. If you do not want to use Everest to process certain credit card payments, clear the Use Payment Processor check box in the Credit Card Processing tab (path: File > Setup > Accounting > Payment Methods > Options > Edit > In New Window > Credit Card Processing tab).

Credit card payments may be processed using any of the following payment services that Everest supports:

  • ICVerify
  • Payflow Pro

A credit card payment is automatically processed when the payment method for the credit card has the UsePayment Processor check box selected in the Credit Card Processing tab.
