# POS Cash Registers Browser Options


The **POS Cash Registers** browser  (path: **Invoicing &gt; Point of Sale**)  contains the following options:

- [Log  In]({{site.pos_baseurl}}/using-cash-registers/log_in_pos_cash_registers_browser_option.html)
- [Declare]({{site.pos_baseurl}}/using-cash-registers/declare_the_cash_register.html)
- [Lock/Unlock]({{site.pos_baseurl}}/using-cash-registers/locking_and_unlocking_the_cash_drawer.html)
- [Show  Sales]({{site.pos_baseurl}}/misc/show_sales_pos_cash_register_browser_option.html)
- [Show  History]({{site.pos_baseurl}}/misc/show_history_pos_cash_register_browser_option.html)
- [Cash  Register Closing Report]({{site.pos_baseurl}}/misc/cashier_closing_report_pos_cash_register_browser_option.html)
- [Store  Closing Report]({{site.pos_baseurl}}/misc/store_closing_report_pos_cash_register_browser_option.html)
- [Documents]({{site.pos_baseurl}}/misc/documents_pos_cash_register_browser_option.html)



Options common to all browsers are also available in this browser.
