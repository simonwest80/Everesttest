Item Information

Items in a Point of Sale document are listed in the Item Details Grid. The Item Details Grid displays items in a tabular format. Each row displays information about the item specified in that row.

You must select at least one item in the Point of Sale invoice to save it.

Item information for the items (in the Item Details Grid) has been grouped under the following sections:

  • Item Details
  • Discounts
  • Other Information
