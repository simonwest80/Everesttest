The Point of Sale (Invoice) Profile

The Point of Sale (Sales Invoice) profile (path: Invoicing > Point ofSale > From the POSCash Registers browser, double-click on the required record) contains the following information:

  • Customer Information
  • Cash Register Details
  • Sales Rep and Cashier
  • Document Information
  • Item Information
  • Document Tab Views
  • The Point of Sale Outlook Bar
  • Status Bar Information
