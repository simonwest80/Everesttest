Print

Point of Sale documents can be printed from within a document.

You can select the details that are to be printed on a Point of Sale document from the Reporting Preferences dialog box (path: File > Setup > Preferences > Defaults > Reporting Preferences). These can be changed for the current printing.

You can set up a preference to automatically pop up the Document Printing Preferences dialog box by selecting the option Prompt user with Document Printing Preferences Profile before printing a document (path: File > Setup > Preferences > Flow Control > Other Flow Control > General tab).

To select details to be printed on a Point of Sales document, do the following:

  • From the main menu bar, select Invoicing > Sales Invoices. You will see the Sales Invoice browser.
  • From the Sales Invoice browser, select View > Point of Sale. Select a Point of Sale document and choose Options > Print Documents.  You will see the Document Printing Preferences dialog box.

The Document Printing Preferences dialog box consists of the following tabs:

  • Printing Preferences
  • Fax/Cover
