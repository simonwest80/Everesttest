# Line Items


The **Line Items** option (path: **Point of Sale (sales invoice)** profile  > **Options &gt; Line Items**)  allows you to do the following:


| Options | Description |
| Insert | Inserts a blank line in the Item Details Grid. |
| Line Profile | Displays information about the line highlighted in the  Item Details Grid. When you select this option the [**Point of Sale Line Profile**]({{site.pos_baseurl}}/misc/the_point_of_sale_line_profile.html) is displayed. |
| Discount Notes | Allows you to enter notes with regard to discounts given  to customers. When you select this option the **Discount <br/> Notes** dialog box is displayed. Enter the details and select **File** > **Save <br/> and Close**.<br/><br/><br/>![]({{site.pos_baseurl}}/img/example.gif)  You  can enter ‘Discount offered during the Christmas season’. |
| Move up | Moves the selected line item one line up. |
| Move down | Moves the selected line item one line down. |
