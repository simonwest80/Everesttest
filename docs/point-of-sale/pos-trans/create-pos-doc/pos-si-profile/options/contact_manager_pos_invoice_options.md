# Contact Manager


The **Contact Manager** option (path:  **Point of Sale (sales invoice)**  profile > **Options &gt; Contact Manager**)  allows you to create tasks related to the current document. This option  has the following submenu options:


My Tasks - Current Document
: The **My Tasks - 
 Current Document** option allows you to view the tasks that are created  for the current document. It is enabled only for saved Point of Sale (sales  invoice) documents.


Pending Tasks - Current User
: The **Pending Tasks 
 - Current User** option allows you to view all the pending tasks  for the logged in user (The user name is displayed in the **Everest**  status bar).
