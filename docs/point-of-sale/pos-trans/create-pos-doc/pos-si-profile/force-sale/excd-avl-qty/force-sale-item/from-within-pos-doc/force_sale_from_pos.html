Force Sale - From within the Point of Sale Document

When the quantity of non-serialized items entered/scanned into the Point of Sale (sales invoice) document exceeds the quantity available, you can manually change the item type for a force sale. To do this, click in the Type column for the line item and press the Shift + Spacebar keys. The icon appears in the Type column, indicating that it is a force sale.
