Create a Point of Sale (Sales Invoice) Profile

You can create a Point of Sale (sales invoice) profile by logging in to an available cash register.

To create a Point of Sale document, do the following:

  • From the main menu bar, select Invoicing > Point of Sale. You will see the POS Cash Registers browser.
  • From the POS Cash Registers browser, highlight the required cash register and select Options > Log In. You will see the Point of Sale (Invoice) profile.
  • When you save the Point of Sale (sales invoice) document, the Payment Details dialog box is automatically displayed.
