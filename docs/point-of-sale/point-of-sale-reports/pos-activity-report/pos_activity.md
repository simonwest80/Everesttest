# POS Activity


The **POS** **Activity** report allows you to view the following:

- A detailed statement  of each cash register for any given period.
- A breakup of the  total collection and refunds based on the payment method. The collection  summary has two sections:


- **POS 
 Collection Section** – Displays details of collections or refunds  made on POS documents, and deposits applied from Sales Orders and Credit  Memos.
- **Non-POS 
 Collection Section** – Displays details of collections or refunds  made on regular sales documents.


- A breakup of the  total sales, returns, tax and the exempt amount.
- Details of each  item sold from the particular cash register, if you select the **Include 
 Item Details** check box in the Report Parameters dialog box.
- Information on  when the cash drawer was opened during the period of no sale/return.



{:.steps}
To generate the POS Activity report, do the  following:

- From the main  menu bar, select **Reports &gt; Invoicing 
 &gt; POS Activity**. You will see the [**POS Activity Report** dialog box]({{site.pos_baseurl}}/point-of-sale-reports/pos-activity-report/pos-activity-report-dialog-box/the_pos_activity_report_dialog_box.html).
- Specify the  report parameters and click the buttons at the bottom of the dialog box  to perform different functions:
- **Preview**,  to preview the report in the **Report Preview**  dialog box.
- **Print**,  to print the report.
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file format.
