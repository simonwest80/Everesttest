Retrieve Item Matrices

You can retrieve items that belong to a matrix either by selecting the master item for the matrix from the Items browser (path: Inventory > Items), or by selecting the Item Matrix option (path: Point of Sale (sales invoice) profile > Options > Retrieve > Item Matrix).

Item Matrix

When the item matrix is retrieved by selecting the master item for the matrix, the quantity specified for the item of all combinations for the matrix is retrieved into the document.
