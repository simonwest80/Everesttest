# Discount Notes


This option (path: **Point of Sale**  profile > **Line** **Items**  > **Discount** **Notes**)  allows you to enter notes with regard to discounts given to customers.


{:.example}
You can enter ‘Discount offered during the Christmas  season’ in  the **Discount Notes** dialog box.


When you select this option, the **Discount 
 Notes** dialog box is displayed. Enter the details and select **Options** > **Save 
 and Close**.
