Select a form (such as the Credit Memo - Dot Matrix form) that you would like to associate with the document type in a Point of Sale environment.

You can set Credit Memo - Dot Matrix form as the default form for all credit memo documents. Whenever a Point of Sale (credit memo) is printed, the Credit Memo-Dot Matrix document will be used.
