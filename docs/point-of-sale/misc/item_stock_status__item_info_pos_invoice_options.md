# Item Stock Status


The **Stock Status** (path: **Point of Sale (sales invoice)** profile  > **Options &gt; Item &gt; Item Stock 
 Status**) option displays the **Item 
 Stock Status** browser for all inventoried items in the Point of  Sale document. This option allows you to view the stock status for the  selected item and the stock area to which it is assigned. The quantities  can be viewed either by purchase measure or sales measure of the item.
