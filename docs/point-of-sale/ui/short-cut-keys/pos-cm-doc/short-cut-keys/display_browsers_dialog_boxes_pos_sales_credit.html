Display Browsers/Dialog Boxes

Press Key(s) To
Ctrl+U Display the Customer profile.
Ctrl+G Display the Custom
Tracking browser for the customer.
Ctrl+A Display the Customer
Aging Information browser.
Ctrl+W Display the Multimedia
Setup browser for the document.
Ctrl+I Display the Items browser for item selection.
Shift+F8 Display the Items browser for kit selection
Ctrl+O Display the Item profile.
Shift+F2 Display the Multimedia
Setup browser for the line item.
Ctrl+L Display the Item Stock Status browser for all items.
` Display the Retrieve
Serial Numbers dialog box.
Ctrl+Q Display the Retrieve
Serial Numbers dialog box (range).
Shift+` Display the Retrieve
Mfg. Serial
Number dialog box.
Ctrl+H Display the AttachQuantity For dialog box.
Ctrl+K Display the Retrieve
Kits dialog box.
Ctrl+D Display the PricingAnalysis window.
F11 Display the TaxBreakdown window.
Ctrl+1 Display the Payment
Details dialog box.
Ctrl+2 Display the Payment
Details dialog box.
Ctrl+3 Display the Payment
Details dialog box.
Ctrl+4 Display the Payment
Details dialog box.
Ctrl+5 Display the Confirm dialog box.
Alt+F8 Display the Inline
Notes window.
Shift+F6 Display the Notes window.
Ctrl+T Display the Transaction
History browser.
Ctrl+Alt+D Display the Discount
Notes window.
Ctrl+Alt+L Display the Item Stock
Status browser for the line item.
Ctrl+Alt+E Display the Extended
Notes window.
Ctrl+Alt+W Display the serial numbers that belong to the selected


line item.
Ctrl+Right Display the Point of
Sale Line Profile.
Ctrl+Alt+V Display the Serial
# for all items browser.
Shift+F7 Display the Customers browser.
Shift+Ctrl+W Display the Document
Discount dialog box.
Ctrl+F12 Display the Find Line
Items browser.
Ctrl+BkSp Display the Return
Processing browser for a Return.
Shift+BkSp Display the Return
Processing browser for a Return
with Gift Receipt.
