# The Object Inspector Dialog Box


The **Object Inspector** dialog **** box displays properties and values  for each property of the **Parameter** dialog  box.


**Properties and Events**


The **Properties** and **Events**  are different for different fields that are selected in the **Report Designer** dialog  box.


Properties common to all the fields are grouped under the following  categories:


**Appearance**
: This category displays properties of how the control  appears when the report is run. The following properties are available  for this category:

- **Caption**  - Enter the title of the **Parameter** dialog  box.



{:.note}
This option is available only when you select  the title field at the top of the **Report 
 Parameter** dialog box.

- **ParentColor**  - Select the same color for the **Parameter** dialog box as that of the control. If you select this property  for all controls, the **Parameter**  dialog box has a uniform color.

: ![]({{site.rmgr_baseurl}}/img/example.gif)  Changing  the background color property to gray causes all the controls on the dialog  box to have a gray background.


**Behavior**
: Specify the tab order of the control.


**Font**
: Specify the font, height and parent font of the  control. The following properties are available for this category:

- **Font**  - Specify the font and font height of the control.
- **ParentFont**  - Select the same font for the **Parameter** dialog **** box as of the control.

: ![]({{site.rmgr_baseurl}}/img/example.gif)  Changing  the Font property to 12-point **Courier**  causes all the controls on the dialog box to use that font.


**Misc**
: This field displays the name of the control.


**Other**

- **ParameterName**  - A drop-down of all the parameters defined while creating the .rpt file  are displayed. Select the parameter name for each of the parameter fields,  depending on the data field type.



**Position**

- **Height**  - Specify the height of the control.
- **Left**  - Specify the distance between the internal left edge of the control and  the left edge of the **Parameter** dialog  box.
- **Top**  - Specify the distance between the internal top edge of the control and  the top edge of the **Parameter** dialog  box.
- **Width**  - Specify the width of the control.
