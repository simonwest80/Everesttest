The Data tab displays the following icons:

Pointer

Use this tool to select, resize and move the controls when you are editing the Report Parameter dialog box.

DB lookup combo

The DB lookup combo shows the following additional options in the Object Inspector dialog box properties.

  • ComboField
  • ComboSelected
  • ComboSQL
  • KeyField

DB lookup tree

The DB lookup tree shows the following additional options in the Object Inspector dialog box properties.

  • DisplayField
  • KeyField
  • ListField
  • ParameterName
  • ParentField
  • Text
  • TreeSQL

If you want to create a report for account activity, you will choose account from, account to, date and include un- posted check box as the parameters.

You can choose the Label control - in “Text” Data Type and place it on the Parameter dialog box, to create a label - “Account From” for the parameter field Account From.

In the Report Designer dialog box, you can choose the control icons Date control Check box control - and place them into the Parameter dialog ** box.
