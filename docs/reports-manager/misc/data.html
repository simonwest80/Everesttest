Data

The Data tab displays the following icons:

Pointer
Use this tool to select, resize and move the controls.
DB Lookup Computer
Use this control to select the DB lookup computer which shows the following additional options in the Object Inspector dialog box properties.
  • ComboField
  • ComboSelected
  • ComboSQL
  • KeyField
DB Lookup Tree
The DB lookup tree shows the following additional options in the Object Inspector dialog box properties.
  • DisplayField
  • KeyField
  • ListField
  • ParameterName
  • ParentField
  • Text
  • TreeSQL

 To create a report for account activity, choose account from, account to, date and include un-posted check box as the parameters.

You can choose the “Label” component in “Text” Data Type and place it on the Parameter Dialog box, to create a label - “Account From” for the parameter field Account From.

In the Report Designer dialog box, you can choose the control icons and place them into the Parameter Dialog box.
