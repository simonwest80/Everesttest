Modify Dialog

Use Modify Dialog option to modify the Reports Parameter dialog box. You need both the Extract and Update Reports options to change a report. Once you modify a report you must update it (.czr file) before you can run it.

To modify a report, do the following:

  • From the main menu bar, select Reports > Reports Manager. You will see the Reports Manager window.
  • From the Reports Manager window, select a .czr file and choose File > Modify Dialog. You will see the Report Parameter Interface.

: Report Parameter Interface

  • Modify the Report Parameter dialog box and save the changes.
  • Update the .czr file with the modified .rpt report using the Update option.
  • Register the report.
