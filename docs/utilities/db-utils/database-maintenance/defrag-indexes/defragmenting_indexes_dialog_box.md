# Defragmenting  Indexes Dialog Box


**Log File Path**
: By default, the log file path is displayed in this  field. You cannot edit the file path. After defragmenting, the log file  is stored in a temporary folder. It displays the tables that are defragmented  in your database, and the start and completion time of the process.


Options


**Run in Foreground**
: Select this option to display and view the progress  of defragmentation.


**Run in Background**
: Select this option to hide the progress of defragmentation.
