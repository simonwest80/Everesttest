# Shipment Tracking Import


The **Shipment Tracking** utility  allows you to import shipment tracking information given by vendors for  drop ship orders in **Everest**.


{:.steps}
To import shipment tracking information,  do the following:

- From the main  menu bar, select **Utility** >  **Import** > **Shipment****Tracking** **Import**.  You will see the **Import** **Shipment** **Tracking****Wizard**.



The **Import Shipment Tracking Wizard**  comprises the following sections:

- [Welcome]({{site.utl_baseurl}}/db-utils/shipment-tracking-import/import_shipment_tracking_wizard_welcome.html)
- [Select  Excel Source File Section]({{site.utl_baseurl}}/db-utils/shipment-tracking-import/select_excel_source_file_section.html)
- [**Shipment 
 Tracking Information to be imported**]({{site.utl_baseurl}}/db-utils/shipment-tracking-import/shipment_tracking_information_to_be_imported.html)
- [**Map 
 the Source Fields**]({{site.utl_baseurl}}/db-utils/shipment-tracking-import/map_the_source_fields_shipment_tracking.html)
- [**Importing 
 Shipment Tracking Information**]({{site.utl_baseurl}}/db-utils/shipment-tracking-import/import_shipment_tracking_information.html)
