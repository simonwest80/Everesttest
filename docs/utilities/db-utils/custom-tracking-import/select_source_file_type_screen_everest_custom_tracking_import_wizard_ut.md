# Select Source File Type Screen - Everest Custom Tracking Import Wizard

- Select the Excel  spreadsheet as the **Import File Type**.
- Click the ![]({{site.utl_baseurl}}/img/importmatrix_select_file_button_ut.gif) Â button  and attach your Excel source file in the **Select 
 the Excel spreadsheet** field.
- Select the sheet  you wish to import from the **Table to 
 Import** drop-down.
- Select whether  the custom tracking details are for a customer, vendor or item from the  **Custom Tracking Type** drop-down.
- Click the **Next** button. You will see the **Map 
 the Source Fields** screen of the Wizard.
