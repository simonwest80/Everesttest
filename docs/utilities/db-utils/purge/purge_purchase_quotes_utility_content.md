# Purge Purchase Quotes


All purchase quotes that can be purged are displayed in the **Purge**  [**Purchase** **Quotes**]  browser. Only purchase quotes that are created and not yet converted to  purchase orders can be purged.


{:.steps}
To purge purchase quotes, do the following:

- From the main  menu bar, select **Utility** >  **Purge** > **Purchase 
 Quotes**. You will see the **Purge 
 [Purchase Quotes]** browser.
- From the Purge **[Purchase Quotes]** browser, select the  record you want to purge.
- Select **Options** > **Purge**.  You will see the **Confirmation**  window.
- Click **OK** to purge the record.



{:.note}
To purge all the records displayed in the  **Purge [Purchase Quotes]** browser,  select **Options** > **Select 
 All** > **Purge**. You will  see the **Confirmation** window. Click  **OK** to purge all the records.
