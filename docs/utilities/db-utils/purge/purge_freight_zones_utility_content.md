# Purge Freight Zones


All freight zones that can be purged are displayed in the **Zones**  browser.


{:.steps}
To purge a freight zone, do the following:

- From the main  menu bar, select **Utility** >  **Purge** > **Freight****Zones**. You will see the **Zones** browser.
- From the **Zones** browser, select the freight zone  you want to purge.
- Select **Options** > **Purge**.  You will see the **Confirmation**  window.
- Click **OK** to purge the freight zone.

: ![]({{site.utl_baseurl}}/img/warning.gif) When a Freight Zone is purged, all  the associated Freight Zone Details and Freight Rates are also purged.
