# Purge Debit Quotes


All debit quotes that can be purged are displayed in the **Purge**  [**Debit** **Quotes**]  browser. Only debit quotes that are created and not yet converted to debit  orders can be purged.


{:.steps}
To purge debit quotes, do the following:

- From the main  menu bar, select **Utility** >  **Purge** > **Debit 
 Quotes**. You will see the **Purge 
 [Debit Quotes]** browser.
- From the **Purge [Debit Quotes]** browser, select  the record you want to purge.
- Select **Options** > **Purge**.  You will see the **Confirmation**  window.
- Click **OK** to purge the record.



{:.note}
If you want to purge all the records displayed  in the **Purge** **[Debit 
 Quotes]** browser, select **Options**  > **Select All** > **Purge**.  You will see the **Confirmation**  window. Click **OK** to purge all  the records.
