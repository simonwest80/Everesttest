# Purge Browser Options


The **Purge** **&lt;Record&gt;**  browser has the following options:


![]({{site.utl_baseurl}}/img/example.gif) **Purge** [**Sales** **Quote**]  browser (path: **Utility** > **Purge** > **Quote**).


**Select All**
: The **Select All**  option selects all the records that are displayed in the browser.


**Unselect  All**
: The **Unselect 
 All** option deselects  all the records that have been selected in the browser.


**Expired Quotes**
: Quotes are assigned an expiry date in the quote  profile. This option displays all the expired quotes.


{:.note}
: The expiry date  indicates:

- The date until  which the price quoted is valid, in case of a Purchase Quote.
- The date (specified  by the vendor) until which the return price of the item is valid, in the  case of a Debit Quote.
- The date until  which the quote is valid, in case of a Sales Quote.
- The date (you  specify to the customer) until which the return price of the item is valid,  in the case of Credit Quote.


**Purge**
: The **Purge**  option purges the selected records.


Options common to all browsers are also available in this browser.
