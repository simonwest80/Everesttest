Purge Credit Quotes

All credit quotes that can be purged are displayed in the Purge [Credit Quotes] browser. Only credit quotes that are created and not yet converted to credit orders can be purged.

To purge credit quotes, do the following:

  • From the main menu bar, select Utility > Purge > Credit Quotes. You will see the Purge [Credit Quotes] browser.
  • From the Purge [Credit Quotes] browser, select the record you want to purge.
  • Select Options > Purge. You will see the Confirmation window.
  • Click OK to purge the record.

To purge all the records displayed in the Purge [Credit Quotes] browser, select Options > Select All > Purge. You will see the Confirmation window. Click OK to purge all the records.
