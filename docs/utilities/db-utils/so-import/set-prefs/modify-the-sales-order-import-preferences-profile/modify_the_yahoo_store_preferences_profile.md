# Modify the Sales Order Import Preferences Profile


A **Sales Order Import Preferences** profile can be modified from any one of the following locations:

- The **Sales Order Import Preferences** browser - Highlight the record and select **Options** > **Edit** > **In New Window**.
- The **Sales Order Import Preferences** profile - Specify the Account code of the online store you want to modify.



All fields in the **Sales Order Import Preferences** profile can be modified, except the **Store Code** field.
