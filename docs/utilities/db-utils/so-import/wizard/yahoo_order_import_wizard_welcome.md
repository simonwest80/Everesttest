# Sales Order Import Wizard - Welcome


When you access the **Sales Order Import 
 Wizard**, the **Welcome** window  is displayed. This section provides you general information about the  steps to import sales orders from a Yahoo store or any other source, into  **Everest**.


Click **Next**. You will see the  [**Order 
 Details**]({{site.utl_baseurl}}/db-utils/so-import/wizard/yahoo_order_import_wizard_order_details.html) section of **Sales Order 
 Import Wizard**.
