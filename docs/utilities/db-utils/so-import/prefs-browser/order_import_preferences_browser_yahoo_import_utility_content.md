# Sales Order Import Preferences Browser


The **Sales Order Import** **Preferences** browser displays the list  of all the stores that have preferences defined. Browser columns can be  re-sized or moved as needed. The option available in this browser is:


**View Orders Created**
: This option allows you to view the **Sales 
 Documents** browser which displays the orders created through importing  from your online store.


Options common to all browsers in **Everest**  are available in this browser.
