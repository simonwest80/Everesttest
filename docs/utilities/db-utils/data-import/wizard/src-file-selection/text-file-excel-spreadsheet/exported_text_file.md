# Text File/Excel Spreadsheet


If you select Text File or Excel Spreadsheet file options in the **Import File Type** field, specify the  name of the Text/Excel file that has to be imported. Click the browse  ![]({{site.utl_baseurl}}/img/utility_browse_button.gif) button to specify the path and the file name.


{:.hint}
Clean up your Excel source files before importing  them. Remove rows that contain only empty cells by selecting the entire  row and deleting it.


If you are importing an Excel Spreadsheet, it is important that you  remove any rows that contain only empty cells to prevent data import errors.


| ![]({{site.utl_baseurl}}/img/note.gif) | The **Update <br/> Information for existing items** check box is displayed only while  importing item data into **Everest**.  Select this check box to update the relevant fields when importing items  that already exist in your database. When you select this option, the  **Field Mapping** section in the **Data Import Wizard** displays the **Update** column. You can update the existing  item details in your database by selecting the required fields.<br/><br/><br/>If you receive the error 'External file not  in the expected format' when importing data from an Excel sheet, you must  save the file in the latest Excel format.<br/><br/><br/>{:.steps}<br/>To save the file in the latest Excel format, do the following:<br/><br/>- Open the file with  Microsoft Excel.<br/>- Select **File <br/> &gt; Save As**. You will see the **Save <br/> As** dialog box.<br/>- Select Microsoft  Excel 4.0 Worksheet in the **Save as type**  field and click **Save**.<br/><br/><br/><br/>You can now import the data into **Everest**. |



Click **Next**. If the text file  selected is of type **Text - Fixed Width**,  you will see the [ASCII  Fixed Text Files]({{site.utl_baseurl}}/db-utils/data-import/wizard/src-file-selection/text-file-excel-spreadsheet/ascii_fixed_text_files.html) section.


If the text file selected is of type **ASCII 
 Delimited Files**, you will see the [ASCII  Delimited File Import]({{site.utl_baseurl}}/db-utils/data-import/wizard/src-file-selection/text-file-excel-spreadsheet/ascii_delimited_text_files.html) section.
