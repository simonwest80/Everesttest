Text File/Excel Spreadsheet

If you select Text File or Excel Spreadsheet file options in the Import File Type field, specify the name of the Text/Excel file that has to be imported. Click the browse button to specify the path and the file name.

Clean up your Excel source files before importing them. Remove rows that contain only empty cells by selecting the entire row and deleting it.

If you are importing an Excel Spreadsheet, it is important that you remove any rows that contain only empty cells to prevent data import errors.

The Update
Information for existing items check box is displayed only while importing item data into Everest. Select this check box to update the relevant fields when importing items that already exist in your database. When you select this option, the Field Mapping section in the Data Import Wizard displays the Update column. You can update the existing item details in your database by selecting the required fields.


If you receive the error ‘External file not in the expected format’ when importing data from an Excel sheet, you must save the file in the latest Excel format.



To save the file in the latest Excel format, do the following:

- Open the file with Microsoft Excel.
- Select File
> Save As. You will see the Save
As dialog box.
- Select Microsoft Excel 4.0 Worksheet in the Save as type field and click Save.



You can now import the data into Everest.

Click Next. If the text file selected is of type Text - Fixed Width, you will see the ASCII Fixed Text Files section.

If the text file selected is of type ASCII Delimited Files, you will see the ASCII Delimited File Import section.
