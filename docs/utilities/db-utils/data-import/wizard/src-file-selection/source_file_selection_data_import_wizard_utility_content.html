Source File Selection

The source text file could belong to any of the file types listed in this section. This section allows you to choose the file type of the source data.

Import File Type
Select the file type of the source data file. Data can be imported into Everest from the following file types:
  • Excel Spreadsheet
  • Text - Fixed Width
  • Text - Comma Delimited
  • Text - Tab Delimited
  • Text - Semicolon Delimited
  • Text - Space Delimited
  • Use ODBC DSN

If you select the Use ODBC DSN (Data Source Name) file option, select the ODBC Data Source Name to use and then select the table to import.

Refer to the Text File/Excel Spreadsheet section for additional information on the remaining selections.

Options

The selections available in the Options section depend upon the type of data selected. Be sure to review the selections in the Options section before continuing with the import process.

Load Settings
You can load a .Map file that you saved previously to save time. Click the Load Settings button and select your map file. The file type and path information loads along with the field map settings. The field map settings will appear on the Field Mapping screen.

When you imported item categories previously, on the Field Mapping screen, you saved the field mappings in a map file. To save time importing other categories into Everest, load the map file and the fields will be mapped automatically.

  • Click Next. You will see the Field Mapping section.
