Add-on Items Import

Everest allows you to import add-on items using the Everest Add-on Items Import Wizard. This feature is especially useful to import a large number of add-on items at a time by importing an excel (.xls/.csv) file that lists all the add-on items.

You have 5 add-on items for each item and you have such data for 150 items. You can easily use the Everest Add-on Items Import Wizard to import all the data at a time.
