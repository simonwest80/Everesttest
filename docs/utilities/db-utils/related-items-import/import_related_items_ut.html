Import Related Items

Use the Everest Related Items Import Wizard to select your source excel file in which you have stored data for related items.

It is mandatory that the original or parent item code should exist as an item in Everest.

To import related items, do the following:

  • From the main menu bar, select Utility > Import > Related Items. You will see the Welcome screen of the Everest Related Items Import Wizard.
  • Click the Next button. You will see the Select Source File Type screen of the Wizard.
