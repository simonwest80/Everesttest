Related Items Import

Everest allows you to import related items using the Everest Related Items Import Wizard. This feature is especially useful to import a large number of related items at a time by importing an excel (.xls/.csv) file that lists all the related items.

You have 5 related items for each item and you have such data for 150 items. You can easily use the Everest Related Items Import Wizard to import all the data.
