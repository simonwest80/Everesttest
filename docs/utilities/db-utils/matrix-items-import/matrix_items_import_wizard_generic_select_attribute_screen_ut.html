Matrix Items Import Wizard - Generic - Select Attribute

To select matrix attributes, do the following:

  • Click the first blank row. You will see the arrow for the drop-down menu.
  • Select an attribute for the matrix group from the drop-down menu.
  • To add a new row, click the Insert button.
  • To create new attributes in Everest, right-click and select New from the Attribute Name dialog box.

The Matrix Attributes Browser

  • Select the checkbox Include element details in extended sales description to include the element details in the Sales documents.
  • Select the check box Automatically Create New Elements only if you want the system to create the matrix elements that are defined in the Import file but not available in Everest.
  • Click Next. You will see the **Generic
  • Field map (ITEMS)** screen of the Wizard.
