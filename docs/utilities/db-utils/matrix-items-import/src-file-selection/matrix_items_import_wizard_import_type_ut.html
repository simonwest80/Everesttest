Matrix Items Import Wizard - Import Type

Use the Matrix Items Import Wizard - Import Type screen to select your source file.

To select the source file, do the following:

  • From the main menu bar, select Utility > Import > Matrix Items. You will see the Welcome screen of the Everest Matrix Items Import Wizard.
  • Click Next. You will see the Import Type screen of the Wizard.

Figure: Matrix Items Import Wizard - Import Type

  • Click the  button to attach the import file in the Select the Excel Spreadsheet field.
  • Select the sheet you wish to import from the Table to Import drop-down menu.
  • Click Next. You will see the **Generic
  • Select Attributes** screen of the Wizard.
