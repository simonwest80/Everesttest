Options

The Options section allows you to specify the location in which the backup data will be stored.

File
Select this option to take the backup on a file.
File Name
Enter the name of the file in the File name field or click the icon to view the Backup Device Location dialog box. Specify the path and file name and click OK.
Device
Select this option to take backup on a device.
Device Name
This field refers the alias for the file name and path specified in the File Name field. Specify a suitable device name in the Device Name field. The path and the file name are then assigned to the device.
