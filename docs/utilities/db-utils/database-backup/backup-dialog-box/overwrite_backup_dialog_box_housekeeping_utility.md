# Overwrite


The **Overwrite** section allows you to append data to the tape or the disk, in cases where the tape/disk already contains data.


**Append to media**
: Select this option to append data to the tape/disk. Data is just added to the media already containing backup.


**Overwrite existing media**
: Select this option to overwrite the existing data on the tape/disk with the new data.
