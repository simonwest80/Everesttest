# The Change Item Code Dialog Box


The **Change** **Item 
 Code** dialog box, allows you to replace the existing item code with  a new item code. The new item code replaces all instances of the old item  code in the database.


{:.note}
If the item code already exists in the database,  you cannot replace it with another item having the same item code.


**Current Item Code**
: Select the existing code of the item from the **Items** browser. To select an item, right-click  and choose **Select Item** or press  the **F2** key.


**New Item Code**
: click this column and enter the new code for the  item.


{:.note}
You can sort the item codes by clicking **Current Item Code** or **New 
 Item Code** columns.


**Description**
: This field displays the description of the selected  item. This is a display field cannot be edited.


Click the **Start** button to begin  the process. The item code is changed throughout the **Everest**  database.
