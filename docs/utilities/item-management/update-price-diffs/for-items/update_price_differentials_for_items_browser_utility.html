The Update Price Differentials for Items Browser

The Update PriceDifferentials forItems browser (path: Utility > Update Price Differentials > Items) displays the list of all the upgrades that have been created in Everest.

The options available in the Update Price Differentials for Items browser are listed below:

Select/Unselect All
Selects/deselects all the records displayed in the browser.
Process
Select Process to update the item price. The item prices are updated as specified in the Related Items profile (path: Inventory > Items > Options > Related Items).

Options common to all browsers are also available in this browser.
