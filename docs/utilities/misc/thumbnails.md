# Thumbnail


Select the image that is to be associated as a thumbnail. Click the  ![]({{site.utl_baseurl}}/img/utility_lens_button.gif) button beside the field, to browse for the image file.  The thumbnail images are displayed in the **List**  page of your online store.


{:.note}
The thumbnail image should be of dimension  100x100 pixels.
