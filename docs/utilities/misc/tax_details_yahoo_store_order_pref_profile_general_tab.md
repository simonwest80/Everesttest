# Tax Details


**Jurisdiction to be used for tax calculation**
: Specify the jurisdiction that must be used to compute  tax.
: ![]({{site.utl_baseurl}}/img/lens.gif) [Login  Jurisdiction for Tax Calculations]({{site.utl_baseurl}}/misc/login_jurisdiction_for_tax_calculations_default_sales_order_info_order_import_preferences_details_utility_main_content.html)
: [The  Jurisdiction Profile]({{site.sc_chm}}/options/sales-tax/tax-jurisdictions/set-up-jurisdictions/jurisdiction_profile.html)


**Item to be used for adjusting Positive Tax Difference**
: Specify the item code for adjusting the positive  tax difference, if any, between the online store and **Everest**.  The item selected here should be a non-inventoried item. This field is  mandatory.
: ![]({{site.utl_baseurl}}/img/lens.gif) [Positive  Tax Difference]({{site.utl_baseurl}}/misc/positive_tax_difference_utility_yahoo_import_utility_content.html)


**Item to be used for adjusting Negative Tax Difference**
: Specify the item code for adjusting the negative  tax difference, if any, between the online store and **Everest**.  The item selected here should be a non-inventoried item. This field is  mandatory.
: ![]({{site.utl_baseurl}}/img/lens.gif) [Negative  Tax Difference]({{site.utl_baseurl}}/misc/negative_tax_difference_yahoo_import_utility_content.html)


**Exempt tax code to be used for creating Special  Tax Rules**
: Specify the "exempt" tax code that must  be used to create special tax rules. The items specified for the positive  and negative tax differences are not taxable. If special tax rules do  not exist in your database, exempting these items, **Everest**  automatically creates special tax rules for every jurisdiction. This field  is mandatory.
