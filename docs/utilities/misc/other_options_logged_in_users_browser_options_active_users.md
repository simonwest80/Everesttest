# Other Options


**List Groups**


The **List** **Groups** option (path: **Logged** **in** **Users** browser > **Options** > **List** **Groups**) displays the **Groups** browser. This browser lists all the groups that the<font color='blue'> </font>user selected, in logged in user browser belongs to.


**List Semaphore Locks**
: The **List** **Semaphore** **Locks** option (path: **Logged** **in** **Users** browser > **Options** > **List** **Semaphore** **Locks**) displays the **Semaphore** **Locks** browser.


![]({{site.utl_baseurl}}/img/lens.gif)[Semaphore Locks Browser]({{site.wwe_chm}}/misc/semaphore_locks_browser.html)


**Clear all Inactive Users**
: The **Clear** **all** **Inactive** **Users** option (path: **Logged** **in** **Users** browser > **Options** > **Clear all Inactive Users**) clears all the locks set by users who are logged into **Everest** but are inactive.


**Clear Inactive User**
: The **Clear** **Inactive** **User** option (path: **Logged** **in** **Users** browser > **Options** > **Clear all Inactive User**) clears the locks set by the selected user, if the is logged into **Everest** but is inactive.


**Edit Clear Lock Job Schedule**
: The **Edit** **Clear** **Lock** **Job** **Schedule** option (path: **Logged** **in** **Users** browser > **Options** > **Edit** **Clear** **Lock** **Job** **Schedule**) displays the [**Edit** **Clear** **Lock** **Job** dialog box]({{site.utl_baseurl}}/misc/edit_clear_lock_job_dialog_box_utility_content.html).
