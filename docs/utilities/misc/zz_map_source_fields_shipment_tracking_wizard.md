---
title: Map Source fields
---


The **Map the Source Fields** section  consists of:

- **Destination 
 Panel** - The **Destination** panel  displays the column name, description, type and size of the **Everest**  database table.
- **Source 
 Panel** - The **Source** panel  displays the column name and a sample view of the data in the source file.



{:.note}
All mandatory fields in the destination file  have to be mapped to the relevant fields in the source file.
