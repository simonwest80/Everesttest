# Store Information


**Store Code**
: Specify the code for the store that is to be imported  into **Everest**. This is a mandatory  field.
: ![]({{site.utl_baseurl}}/img/example.gif) If you own a store 'ABC', you must create a store with the  code 'ABC' in the **Sales Order Import 
 Preferences** profile.
: ![]({{site.utl_baseurl}}/img/lens.gif) [Store  Code]({{site.utl_baseurl}}/misc/store_code_orders_import_to_everest_dialog_box.html)


**Description**
: Enter a description of the store you have created.  This is a mandatory field.


**Document Alias for Sales Orders**
: Select the document alias to be used for orders  created on your online store. You can select aliases only for sales orders.  Â This is  a mandatory field.
: ![]({{site.utl_baseurl}}/img/lens.gif) [Document  Alias Profile]({{site.bp_chm}}/doc-aliases/info/the_doc_alias_profile_contents_businesss_process_in_everest_content.html)


**Location/Sub-Location\***
: Specify the location/sub-location for which the  sales orders must be created in **Everest**.


**Store currency \***
: Specify the currency in which transactions are carried  out on your online store.
