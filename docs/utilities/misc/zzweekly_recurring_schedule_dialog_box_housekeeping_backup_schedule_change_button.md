---
title: Weekly
---


Select the **Weekly** option to set a weekly schedule to backup data. When **Weekly** option is selected the **Every** <**number** **of**> **Week**(s) on field and the **week** **day** check boxes are enabled. Select the required fields based on your schedule.


![]({{site.utl_baseurl}}/img/example.gif)  If you want to backup data every alternate Friday, enter 2 in the **Every** <**number** **of**> **Week**(s) on field and select the **Friday** check box.
