---
title: Exempt tax code
---


Specify the tax code that is to be attached to the special tax rules  for the item defined for positive and negative tax differences. This item  must have a special tax rule for the order's jurisdiction with an exempt  tax code. If it does not, a special tax rule will be automatically  added for the item with the tax code selected here.
