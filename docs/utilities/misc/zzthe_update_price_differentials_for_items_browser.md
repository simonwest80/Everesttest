# The Update Price Differentials for Items Browser


The **Update** **Price****Differentials** **for****Items** browser (path: **Utility**  > **Update Price Differentials for Items**)  displays the list of all the upgrades that have been created in **Everest**.


The options (path: **Update Price Differentials 
 for Items** browsers **** >  **Options**) available in the **Update Price Differentials for Items**  browser are listed below:


**Select/Unselect  All**
: Selects/unselects  all the records displayed in the browser.


**Process**
: Click the Process button to update the item price.


**Load All**
: Loads all the records into the browser.


{:.note}
Options common to all browsers are also available  in this browser.
