# Daily Frequency


This section allows you to set the frequency at which the data backup  must be performed.


![]({{site.utl_baseurl}}/img/lens.gif) [Occurs]({{site.utl_baseurl}}/misc/occurs_section_edit_recurring_job_schedule_dialog_box_database_backup_utility.html)


**Occurs once at**
: Use this option to backup data only once  on the selected day depending on the daily, weekly or monthly basis. Specify  the time at which the data backup must be performed.


**Occurs every**
: Use this option to backup data once every  minute/hour on the selected day depending on the daily, weekly or monthly  basis. Specify the start and end times during which the data must be backed  up.
: ![]({{site.utl_baseurl}}/img/example.gif) If you want to backup data every two hours, from 11.00 a.m.  to 10 p.m. on the selected day, you must enter 2 in the **Occurs 
 every &lt; &gt;** and select **minutes**  or **hours**. Then specify 11.00 a.m.  in the **Starting at** field and 10  p.m. in the **Ending at** field.
