The Update Price Differentials for Kits Browser

The Update Price Differentials for Kits browser (path: Utility > Update Price Differentials for Kits) displays the list of all kit updates created in Everest.

The options (path: Update Price Differentials for Kits browsers > Options) available in the Update Price Differentials for Kits browser are listed below:

Process
Click the Process button to update the item kit price.
Load All
Loads all the records into the browser.

Options common to all browsers are also available in this browser.
