# Daily Frequency


Use this section to set the frequency at which the activity must be  performed.


**Occurs once at**
: Use this option to execute the activity only once  on the selected day depending on the daily, weekly or monthly basis. Specify  the time at which the activity must be performed.


**Occurs every**
: Use this option to execute the activity once every  minute/hour on the selected day depending on the daily, weekly or monthly  basis. Specify the start and end times during which the activity must  be performed.
