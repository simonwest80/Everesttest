# Default Payment Method


The **Default Payment Method** field  is used if the payment methods in **Everest**  do not match with the payment method on the online sales orders.


![]({{site.utl_baseurl}}/img/example.gif) Cash,  Company Check, Visa, Master Card and Wire Transfer are types of payment  methods.
