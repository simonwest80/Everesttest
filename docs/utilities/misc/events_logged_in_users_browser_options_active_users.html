Events

Events are a pre-defined list of all significant activities in Everest. Events are triggered as the result of functions accessed by users and user groups.

History
The History option (path: Logged inUsers browser > Options > Events > History) displays a list of all events logged for the current logged in user.

The list of events can comprise; creation of new records or transactions, modification of a record, modification of important fields in records, printing reports, logging in and logging out.
