# Server Error Log


The **Server Error** **Log**  browser displays a list of errors that have occurred in the server.


{:.note}
This browser can be viewed only from the system  where there is a server installation of **Everest**.


{:.steps}
To view the server error log browser, do  the following:

- From the main  menu bar, select **Utility** >  **Error Log** > **Server 
 Errors**. You will see the **Server 
 Error Log** browser.



The **Server Error Log** browser  displays the following information:


Server Error Log Browser columns:


| **Column** | **Displays** |
| Level | The levels at which the error occurred. |
| Date/Time | The date and time when the error occurred |
| User ID | The logged in user when the error occurred |
| Source Desc | The method in which the error occurred or exception raised. |
| User Code | The user code of the person who logged into the session. |
| Error No. | The error number generated. |
| Error Description | The description for the error generated. |
| Server Name | The name of the server the user logged into. |
| Object ID | The MTS Object ID also known as Application ID. |
| Error Type | Errors can be of the following types:<br/><br/>- Database Errors<br/>- Other System Related  Errors<br/>- Object Generated  Errors |
| Source Description | The method in which the error occurred or exception raised. |
| Object Description | Shows if the error is system generated. |
| Object Type | The kind of MTS object in which the error occurred. |
| Severity | Indicates the severity of the error occurred. |
