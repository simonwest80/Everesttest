Advanced Query Builder

The Advanced Query Builder utility in Everest allows you to build your own queries for retrieving data. It helps you to directly interact with the database with the help of SQL commands, thus giving you the option of creating your own custom reports. The queries can be saved in the Everest database, so that it can be used as and when required, without having to rewrite the query.

The power of queries lies in being able to bring together or perform an action on data from more than one table or query.

You might want to view a customer’s information with the orders the customer placed. To see this information, you need data from the Customers and Orders tables.

When you want to retrieve data from more than one table, ensure that there is a link or join between the field lists of the two tables. A query thus constructed will provide the output from both the tables based on the specified criteria.

A link is an association between a field in one table and a field of the same data type in another table.

Accessing the Advanced Query Builder

The Advanced Query Builder contains the following sections:

  • Filter Name
  • Everest Schema Information (Tables)
  • Custom Query Builder section
  • Query Details section (Tabs)
