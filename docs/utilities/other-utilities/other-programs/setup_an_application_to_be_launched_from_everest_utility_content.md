# Setup an Application to be Launched from Everest


{:.steps}
To setup  an application to be launched from Everest, do the following:

- From the main  menu bar, select **Utility** >  **Other** **Programs**  > **Configure** **Other****Programs**. You will see the [**Configure 
 Utilities**]({{site.utl_baseurl}}/other-utilities/other-programs/the_configure_utilities_dialog_box_utility_content.html) dialog box.
