The Configure Utilities Dialog Box

Utilities
This column displays a list of the applications that have been set up so that they can be launched from Everest.
Add
The Add option allows you to set up a new application to be launched from Everest. When you click this option, you must specify the location of the executable file.
Delete
The Delete option allows you to delete an application’s connection to Everest. To remove the link of the application to Everest, highlight the application in the Utilities section and click this option.
and
These buttons allow you to change the order in which the applications appear in the Other Programs submenu. Highlight an application in the Utilities section and click the Up or Down button to move the application accordingly.
Title
Enter a name for the application. This is the name that will be displayed in the Utility > Other Programs submenu.
Program
Specify the location of the executable file. You can either enter it specify the location by clicking the look up button.
Working Directory
Specify the location of the working directory of the application. This field automatically gets populated when you specify the Program location.
Parameters
You can additionally specify a parameter for the application, that is specify a particular function to be carried out.
