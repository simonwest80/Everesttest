# View Registered Add-Ins


Installed and validated add-ins register upon **Everest**  startup. You can view the list of registered add-ins on a workstation.


{:.steps}
To view all registered add-ins on a workstation,  do the following:

- From the main menu  bar of the appropriate workstation, select **Utility**  > **Add-In** **Manager**.  You will see the **Add-In 
 Manager** dialog box.
- Review the list  of add-ins and add-in statuses.
- Click the **Close** button to exit the **Add-In 
 Manager** utility without making any changes.
