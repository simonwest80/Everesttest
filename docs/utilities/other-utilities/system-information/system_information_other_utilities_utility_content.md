# System Information


The **System** **Information** utility (path: **Utility** > **System Info**) allows you to view your workstation                                                                     information such as memory, environment, directories, screen, network, accessibility and keyboard.


The **System** **Information** window consists of two panels:

- <font face='Symbol'> </font>[The left panel]({{site.utl_baseurl}}/other-utilities/system-information/the_left_panel_system_information_other_utilities_utility_content.html)
- <font face='Symbol'> </font>[The right panel]({{site.utl_baseurl}}/other-utilities/system-information/the_right_panel_system_information_other_utilities_utility_content.html)
