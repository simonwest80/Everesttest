# The Right Panel


The right panel is divided into two columns; the Item column and the  Value column.


Item Column


The **Item** column displays the  list of components for the selected icon.


Value Column


The **Value** column displays the  value of the component for the selected icon.


{:.note}
The right panel displays the information for  an icon selected from the left panel.
