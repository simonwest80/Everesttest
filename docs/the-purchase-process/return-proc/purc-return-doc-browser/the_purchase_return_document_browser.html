The Purchase Return Document Browser

The Purchase Return document browser lists all documents for a selected purchase return document type. (Select Debit Quotes, Debit Orders or Debit Memos in the Purchasing menu to bring up the browser for each of these document types). The browser will list all documents for that type irrespective of their document aliases.

To view the debit quote/order/memo browser, do the following:

  • From the main menu, select Purchasing > Debit Quote/Order/Memo.
  • You will see the relevant document browser.

To view documents with a specific alias, do the following:

  • From the main menu, select Purchasing > Other Documents. You will see a submenu that lists the purchase documents.
  • From the submenu, select the required document type. You will see another submenu that lists the aliases for the document.
  • Select the alias to view its document browser.

The Other Documents option displays debit quotes, debit orders and debit memos along with other document aliases. These are the system aliases for debit quotes, debit orders and debit memos. Selecting any of these options brings up a browser of debit quotes, orders or memos that do not have a user-defined document alias.

If you create a debit quote without selecting an alias for it, the document will be displayed the Debit Quotes browser from the OtherDocuments option.
