Common Purchase Return Document Browser Options

**View**
This option allows you to view a purchase return document. In cases where the user does not have access to modify a document, the user can select this option to view the document.
The View menu has the following submenu options:
In Current Window Select this option when you want to close the existing open document and open the new document in the same window.
In New Window Select this option to open the document in a new window without closing the document that is currently open.
**Transaction
History**
Select this option to view the transaction history of a selected document.

Purchase Documents - Transaction History

**DocumentTracking**
Select this option to view and edit the tracking group and selections of a selected document.

Purchase Documents - Document Tracking

**Multimedia**
Select this option to view and add multimedia files for a selected document.

Purchase Documents - Multimedia

**PrintDocument**
Select this option to print the selected documents. Multiple documents can be printed at the same time.

Printing Purchase Documents

Print Shipping Label
Select this option to print shipping labels for the selected documents. Multiple shipping labels can be printed at the same time.
 Shipping Label Formats
Contact Manager
Select this option to view tasks, appointments and calls associated with the selected documents.

Purchase Documents - Contact Manager

**Search
by Vendor**
This option allows you to search for documents by the vendor name. Highlight a document from the browser and select this option. You will see a submenu that lists the document types. Select the relevant document type. All the documents of that type are listed for the vendor.
Print Serial Numbers
The PrintSerial Numbers option allows you to print serial numbers attached to items in a purchase return document.
Print Serial Numbers
E-mail Documents
The E-mail Documents option allows you to send multiple documents by e-mail to more than one vendor. When you select multiple documents, they are attached to a single e-mail, and Everest warns you that the documents belong to different vendors.
Document E-mail Preferences Dialog Box
Pre-requisites for Emailing Documents
