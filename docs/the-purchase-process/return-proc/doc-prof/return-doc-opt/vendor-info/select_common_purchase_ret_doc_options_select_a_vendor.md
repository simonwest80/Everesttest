# Select a Vendor


The **Select** option (path: Purchase  Return Document profile > **Options**  > **Vendor** > **Select**)  brings up the **Vendors** browser  and allows you to change the vendor on the purchase return document.


{:.note}
You cannot change the vendor:

- On  a debit order, if it has a payment history.
- On  a debit memo, once it has been saved.
