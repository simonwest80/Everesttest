# Stock Status


The **Stock** **Status**  option (path: Purchase/Purchase Return Document profile > **Options**  > **Item** > **Stock****Status**) displays the [**Item Stock Status** dialog box]({{site.mi_chm}}/the-items-browser/information-available/item_stock_status_item_browser_option.html) for  the selected inventoried/serialized/auto-serialized item in the purchase  document.


![]({{site.pp_baseurl}}/img/lens.gif) [Inventoried  Items]({{site.mi_chm}}/item-profile-details/inventory-type/inventoried-item/inventoried_items_items.html)


**Stock Status for All Items**
: This option (path: purchase/purchase return document  profile > **Options &gt; Item &gt; Stock 
 Status for All Items**) displays the [**Item Stock Status** dialog box]({{site.mi_chm}}/the-items-browser/information-available/item_stock_status_item_browser_option.html) for  all the inventoried/serialized/auto-serialized items contained in the  purchase/purchase return document.
