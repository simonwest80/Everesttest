# Hide


The **Hide** column (check box)  allows you to hide lines in the Item Details Grid.


![]({{site.pp_baseurl}}/img/steps.gif)To hide a line in the item details grid, do  the following:

- From the Purchase  Return Document profile highlight the item.
- Scroll to the **Hide** column.
- Check the box.
- Click on the **Hide** button ![]({{site.pp_baseurl}}/img/pur_hide.gif) displayed  in the purchase return document status bar. The selected line is hidden  and will not be displayed in the Item Details Grid.
- Click on the **Show** button ![]({{site.pp_baseurl}}/img/pur_show.gif) displayed  in the purchase return document status bar to view the hidden line. The  selected line is displayed in the Item Details Grid.
