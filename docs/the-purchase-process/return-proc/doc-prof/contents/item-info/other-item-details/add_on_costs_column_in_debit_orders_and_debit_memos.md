# Add-on Costs Column in Debit Orders and Debit Memos


The add-on costs distribution utility is available only when you enter  negative quantities on a debit order or memo.


This column is automatically updated with the amount calculated using  the **Add**-**on****Costs** **Distribution**  utility. This is a display column and cannot be edited.
