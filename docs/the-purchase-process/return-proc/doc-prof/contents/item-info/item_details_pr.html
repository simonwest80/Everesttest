Item Information

Items in a document are listed in the ‘Item Details Grid’. The item details grid displays the item(s) attached to a document in a tabular format. Each row in the grid displays information about the item specified in the Item Code column.

A debit quote or a debit order may be created without specifying item details; the item details can be entered later by editing the document. You must select at least one item in the debit memo.

Item information displayed for a line item the items (in the Item Details Grid) has been grouped under the following sections:

  • Item Details
  • Discount Details
  • Tax Details
  • Account Details
  • Other Item Details
