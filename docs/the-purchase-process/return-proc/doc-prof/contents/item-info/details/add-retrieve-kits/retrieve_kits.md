# Retrieve Kits Dialog Box


**Kit Code**
: Specify the kit code, the components of which you  want to retrieve into the document.


**Quantity**
: Enter the number of kits that you want to include  in the document.


**Description**
: The description entered for the selected kit in  the **Description** field (path: **Inventory** > **Items**  > **Options &gt; Edit &gt; In New Window**  > **General** tab) is displayed  in this field. You cannot edit this field.
