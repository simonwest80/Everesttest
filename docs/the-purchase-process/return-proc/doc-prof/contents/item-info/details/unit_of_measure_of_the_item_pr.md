# Unit of Measure


The **UM** (Unit of Measure) of  an item is automatically populated with the default purchase measure specified  in the **Measure** field (path: **Inventory** > **Items**  > **Options** > **Edit**  > **In New Window** > **Purchases** tab).


This is a display column and cannot be edited.
