Description/Note

The Description/Note column displays the description of an item as entered in the Description field (path: Inventory > Items > Options > Edit > In New Window > General tab).

You can edit the item description/notes, by using the InlineNotes option (path: Purchase Return Document profile > Options > Inline Notes).

You may want to enter the item’s description as it appears in the vendor’s catalog.
