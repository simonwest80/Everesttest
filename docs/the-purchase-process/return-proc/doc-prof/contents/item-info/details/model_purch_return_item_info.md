# Model


You may have two different models of an item manufactured by the same  company.


{:.example}
Printers HP Color LaserJet  5M and HP Color LaserJet  5/ 5M PS are 2 models manufactured by HP in the LaserJet  range.


If you have specified the model of the item in the **Model**  field of the **Item** profile (path:  **Inventory &gt; Items &gt; Options &gt; 
 Edit &gt; In New Window &gt; General** tab), the model of the item  is displayed in the **Model** column.  You cannot edit this column.
