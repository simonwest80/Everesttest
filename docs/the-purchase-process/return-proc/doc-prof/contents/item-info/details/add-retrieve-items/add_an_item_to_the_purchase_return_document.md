# Add an Item 


You can add an item to a purchase return document in two ways:

- From the **Options** menu
- By using the smart  search option



![]({{site.pp_baseurl}}/img/steps.gif)To add an item to the purchase return document,  do the following:

- From the **Purchase Return Document** profile, select  **Options** > **Item** > **Select Items**.  You will see the **Items** browser.
- From the **Items** browser, highlight the item and  double-click. The item is added to the purchase return document



![]({{site.pp_baseurl}}/img/note.gif) The item selected in  the Item Details Grid can be an item of a special type such as an item  kit or an item matrix.


Add a Discontinued Item


You can add a discontinued item to a purchase return document.


![]({{site.pp_baseurl}}/img/steps.gif)To add a discontinued item to a purchase return  document, do the following:

- From the Purchase  Return Document profile, select **Options 
 &gt; Item &gt; Select Items**. You will see the **Items**  browser.
- From the **Items** browser, select **View &gt; Discontinued**. The discontinued items are also displayed  in the **Items** browser.
- From the **Items** browser, highlight the discontinued  item and click **OK**.



**Everest** displays a message stating  that the item is a discontinued one. You will be asked whether you want  to use the discontinued item. If you select **Yes**,  the discontinued item will be added to the purchase return document. If  you select **No**, the item is not  added to the purchase return document.


Add Multiple Items


If you add multiple items to a purchase return document of which one  is a discontinued item, you will receive a message stating that discontinued  items exist in the selection. You can choose to:

- **Retrieve 
 active items** - If you select this option, the item/s that have  not been discontinued are added to the purchase return document.
- **Retrieve 
 active and discontinued items** - If you select this option, all  the items you have selected (including the discontinued items you have  selected) are added to the purchase return document.
