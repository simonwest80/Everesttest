Status Bar Information - Purchase Return Documents

The Status Bar is present at the bottom of all document profiles in Everest. The options available in the status bar of purchase return document profiles are:

  • Preview On/Off
  • Show/Hide
  • Zoom In/Out
  • Batched Payments Pending
  • Document Policy
  • Posted/Unposted
  • Item Notes

The Stock Area field displays in the status bar when you need to select a stock area. Select a stock area and press the Enter key or ESC key to exit this field and continue.
