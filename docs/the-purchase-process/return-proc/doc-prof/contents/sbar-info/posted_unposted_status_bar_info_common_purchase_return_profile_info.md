# Posted/Unposted


The posting status of the purchase return journal, in which the accounting  entry for the debit memo is made, is displayed in this section. This status  is displayed only for debit memos.


| Status Displayed | Description |
| ![]({{site.pp_baseurl}}/img/pur_posted.gif) | The purchase return journal has been posted. |
| ![]({{site.pp_baseurl}}/img/pur_not_posted.gif) | The purchase return journal has not been posted. |
