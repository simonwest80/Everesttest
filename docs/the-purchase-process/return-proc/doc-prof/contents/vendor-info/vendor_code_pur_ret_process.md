# Vendor Code


The vendor code identifies the vendor who  must be debited for the items returned.


You cannot save a purchase return document unless you select a vendor  code.


If you create a vendor from the **Vendor**  field (path: click the drop-down button > right-click in the drop-down  list > select **New** from the  right-click menu options > enter details and save), it displays the  code of the new vendor.


When you [**specify 
 a vendor code**]({{site.pp_baseurl}}/misc/specifying_a_vendor.html), the purchase return document is populated with  the following information:

- [Vendor  Billing Address]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/vendor-info/vendor_s_billing_address.html)
- [Vendor  Contact Address]({{site.pp_baseurl}}/return-proc/doc-prof/contents/vendor-info/vendor_s_contact_address_pr.html)
- [Vendor  Shipping Address]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/vendor-info/return_to_address_pr.html)
- [Vendor  Notes]({{site.pp_baseurl}}/return-proc/doc-prof/contents/vendor-info/vendor_notes_purchase_return_contents.html)
- [Payment  Information]({{site.pp_baseurl}}/return-proc/doc-prof/contents/tabs/details/payment-information/payment_information_pr.html)
- [Delivery  Method]({{site.pp_baseurl}}/return-proc/doc-prof/contents/tabs/details/delivery-information/delivery_information_pr.html)
- [Currency  Information]({{site.pp_baseurl}}/return-proc/doc-prof/contents/tabs/details/for-cur-info/currency_pr.html)



{:.note}
When you select a vendor code, vendor information is  populated in the purchase return document if you have defined it in the  **Vendor** profile.


The format of all addresses for a vendor depends on  the country your vendor belongs to. You can change the format in the **Country** profile (path: **File** > **Setup** > **Miscellaneous** > **Country** > **Options &gt; Edit &gt; In New Window**  > **Format Address** tab).
