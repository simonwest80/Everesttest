# Document Totals


The following totals are summed, automatically updated and displayed  in all purchase return documents:


| **Field** | **Description** |
| Taxable | Displays the total value of items that are subject  to tax. This amount excludes discounts allowed, if any.<br/><br/><br/>The line items that are subject to tax have a Tax  Code associated in the Item Details Grid.<br/><br/><br/>![]({{site.pp_baseurl}}/img/lens.gif) [Tax  Code]({{site.sc_chm}}/options/sales-tax/tax-jurisdictions/jurisdiction-details/tax_code.html)<br/><br/><br/>![]({{site.pp_baseurl}}/img/note.gif)  <font style="color: #0000ff;" color="#0000FF">&nbsp;</font>Tax  inclusive items are included in this total if they have a purchase tax  associated in the item details grid but the tax is stripped out.<br/><br/><br/>Manual Debit Memos<br/><br/><br/>This field displays  the tax-exclusive total amount of all the “Taxable Purchase” entry types. |
| Tax | The **Tax**  field displays the total purchase tax on items (the sum of all the values  in the **Tax** **Amount**  column of the item details grid). This is a display field and cannot be  edited.<br/><br/><br/>Manual Debit Memo<br/><br/><br/>This field displays the total purchase tax of all  recoverable and non-recoverable entries automatically generated by the  system. The Tax Breakdown is not available on manual documents. |
| Exempt | The **Exempt**  field displays the total amount (after reducing the discount, if any)  for the line items that have a non-taxable tax code or that do not have  a tax code associated with them. This is a display field and cannot be  edited.<br/><br/><br/>Manual Debit Memo<br/><br/><br/>This field displays the total amount of all the  “Exempt Purchase” and “None” entry types. |
| Payment discount | <font style="color: #000000;" color="#000000">This field is disabled for purchase return documents.</font> |
| Total | The **Total**  field displays the sum total of the **Taxable**,  **Tax** and **Exempt**  amounts. It is a display field and cannot be edited.<br/><br/><br/>Manual Debit Memo<br/><br/><br/>This total matches the amount specified in the Accounts  Payable entry. |
| Paid | Displays the total amount paid on the document.  Click on the ![]({{site.pp_baseurl}}/img/pur_document_alias.gif) button beside this field to view the **Payment** **History**  for payments made against the document. |
| Balance | This field displays the balance amount (total amount  - paid amount) to be paid on the document by the vendor for refunds. |
