Due Date

The Due date in purchase return documents allows you to specify the date when refunds are expected from the vendor, and unlike purchase documents, does not depend on the Vendor’s RMA No. and Date.

Select the Override checkbox to enter a due date for the document. Enter the due date in the field below the Override checkbox.

Override
Select this check box to ignore the payment term’s due date specified for the document and use the Due Date field to specify the last date of payment. If this check box is not selected the Due Date option is disabled. Enter the due date in the field below the Due Date check box.
