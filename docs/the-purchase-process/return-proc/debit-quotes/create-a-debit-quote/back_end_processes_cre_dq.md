# Back End Processes - Create a Debit Quote


When a debit quote is created the following back end processes occur:

- The next sequence  number assigned to the debit quotes in the logged in location, or the  location of the logged in location/sub-location Â is  assigned to the debit quote.

- If sequence numbers  are not assigned for the logged in location, or the location of the logged  in location/sub-location, the sequence number assigned for the **Default
