Create a Debit Quote

To create a debit quote from the debit quote browser, do the following:

  • From the main menu bar, select Purchasing > Debit Quotes. You will see the Debit Quotes browser.
  • From the Debit Quotes browser, select Options > New > In New Window. You will see the Debit Quote profile.

You can also create a debit quote from an existing Debit Quote profile by selecting Options > New > In New Window.
