# Purge a Debit Quote


You cannot cancel a debit quote. You can however, purge it. There are  two ways to purge debit quotes:

- From the **Debit Quotes** browser (path: **Options 
 &gt; Purge**).
- From the **Utility** menu (path: **Utility 
 &gt; Purge &gt; Debit Quotes**).



![]({{site.pp_baseurl}}/img/steps.gif)To purge multiple quotes, do the following:

- From the main  menu, select **Utility** > **Purge** > **Debit****Quotes**. You will see the **Purge** **[Debit 
 Quotes]** browser.
- Select **Options &gt; Expired Quotes** to view  quotes past their expiry date.
- Using the **Ctrl**  key, select multiple quotes.
- Select **Options &gt; Purge**.
