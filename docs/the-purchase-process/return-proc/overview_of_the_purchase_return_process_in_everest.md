# Overview of the Purchase Return Process in Everest


The purchase return process in **Everest**  is managed through the following purchase return documents:

- Debit Quotes
- Debit Orders
- Debit Memos



The [Purchase  Return Process Flowchart]({{site.pp_baseurl}}/return-proc/flowchart_of_the_purchase_return_process.html) gives you an overview of the purchase return  process in **Everest**. It depicts  each step in the process, the documents created and the system journals  in which the accounting entries are made.
