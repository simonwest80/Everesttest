# Convert an Existing Debit Quote to a Debit Order


A debit quote can be converted to an order by selecting the **Reserve**  option in the debit quote.


![]({{site.pp_baseurl}}/img/lens.gif) [Reserve in Debit Quote]({{site.pp_baseurl}}/purc-proc/pqs/pq-processes/reserve.html)


When you select the **Reserve**  option, all details in the debit quote are displayed in a new **Debit****Order** profile. The **Requested**  column displays the quantity quoted in the debit quote. You can change  any information (including vendor information) in the order.


The debit quote can no longer be accessed through the **Debit****Quotes** browser; however, the debit  quote number is displayed as a reference number in the debit order that  has been created.
