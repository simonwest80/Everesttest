# Accounting Entries - Payment on Debit Order


The accounting entry created when a payment is made on a debit order  is detailed below:


| Account | Entry Type | Path | Debit/<br/><br/><br/>Credit | Amount |
| Bank Account | Payment by cash/check/credit card/others | The account selected in the **Payment <br/> Method** dialog box (path: **File**  > **Setup** > **Accounting**  > **Payment Methods** > **Options &gt; Edit &gt; In New Window**  > **General** tab). | Credit | Amount Paid |
| Vendor Deposit | Apply/Refund debit order | The account selected for **Vendor Deposit** in the **Accounting** dialog  box (path: **File** > **Setup**  > **Preferences** > **Defaults**  > **Accounting** > **Control <br/> Accounts** tab). | Debit | Amount Paid |
