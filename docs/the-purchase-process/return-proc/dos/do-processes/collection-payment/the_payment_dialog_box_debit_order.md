# The Payment Dialog Box


Use the **Payment** dialog box to  specify the payment details for debit orders. The **Payment**  dialog box consists of the following tabs:

- **General**  -  The fields  in this tab are categorized under the following heads:
- [Payment  Details]({{site.pp_baseurl}}/misc/payment_details_payment_db_do_dm.html)
- [Payment  Info]({{site.pp_baseurl}}/misc/payment_dialog_box_payment_details.html)
- [Exchange  Info]({{site.pp_baseurl}}/misc/payment_dialog_box_exchange_rate_details.html)
- [Prior  Payments]({{site.pp_baseurl}}/misc/payment_dialog_box_prior_payments.html) - This tab is NOT displayed when payments are made  from  a debit memo and is applicable to foreign currency vendors.
