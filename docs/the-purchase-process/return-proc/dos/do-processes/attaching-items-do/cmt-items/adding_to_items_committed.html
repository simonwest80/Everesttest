Adding to Items Committed

You can commit additional items to a debit order in any of the following ways:

  • Click the Committed column to display the Retrieve SerialNumber dialog box  (for serialized items) or the Attach Quantity dialog box (for non-serialized items). The Attach Quantity dialog box is displayed only if you select the Use default stock area to assign quantity for non-serialized items in debit documents option (path: File > Setup > Accounting > Location/Sub-Location > Options > Edit > In New Window > Stock Areas > Miscellaneous tab).
  • Follow the same procedure for entering committed quantities.
