Mark Complete

A debit order is closed either when all items on the order are returned or when you mark it complete. You can mark an order complete even if the order contains back ordered items or line items with committed quantities.

Use the Mark Close/Open toggle option (path Debit Order profile > Options > Process) to manually mark the debit order as complete. When you select this option, Everest prompts you to confirm that the order must be marked complete. Click Yes or No as per your requirement.

When you mark an ‘open’ order complete:

  • The requested quantity is reduced to zero
  • The committed quantity, if any, is detached

When you mark a ‘backorder’ complete:

  • The requested quantity is reduced to the quantity debited (on partially received line items)
  • The committed quantity, if any, is detached
