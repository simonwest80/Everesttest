Process to a Debit Memo

Items received against a debit order are processed through the Memo option (path: DebitOrder profile > Options > Process > Memo). When this option is selected, the details of the debit order are copied to a Debit Memo profile.

When you save the debit memo, the Debited column of the converted debit order is updated with the quantity returned (as entered in the Debited column of the debit memo). If quantities committed on the debit order are returned on the debit memo, the quantity in the Committed column of the debit order reduces. You can view fully shipped debit orders in the Shipped Debit Orders browser (path: Purchasing > Debit Orders > Options > Shipped Debit Orders).

Requested, Committed and Debit Columns in Debit Orders

If items are returned in stages, you can create a debit memo for each return against the debit order.
