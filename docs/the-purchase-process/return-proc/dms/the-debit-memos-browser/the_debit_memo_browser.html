The Debit Memos Browser

The Debit Memos browser (path: Purchasing > Debit Memos) displays all the debit memos that have been created. It also contains the following options:

  • Print Serial Numbers
  • Other Options
  • Payment Information
  • Journal Transactions
  • Show Attached Serials (Barcodes)
  • Serial Number History
  • View Options
  • Filter Documents Based on Dates
  • Options Common to Purchase Return Documents
  • Standard Browser Options
