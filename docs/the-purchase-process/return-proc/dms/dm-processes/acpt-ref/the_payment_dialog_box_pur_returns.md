# The Collection Dialog Box


You can account for refunds and apply debit memos to orders/invoices  from this dialog box.


The **Collection** dialog box consists  of the following tabs:

- [General]({{site.pp_baseurl}}/misc/collection_dialog_box_-_general.html)
- [Apply  Deposits/Refunds]({{site.pp_baseurl}}/misc/apply_deposits_refunds_pur_return_docs.html)
