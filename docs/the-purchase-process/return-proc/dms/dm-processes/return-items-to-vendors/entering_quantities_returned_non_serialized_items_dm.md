# Enter Quantities Returned - Non-Serialized Items


You can enter the quantity of non-serialized items that are returned  on a debit memo by selecting the **Show/Attach 
 Area-wise Stock** option (path: **Debit 
 Memo** profile > **Options &gt; 
 Attach/Retrieve Serial Number**). The [**Attach Quantity** dialog box]({{site.pp_baseurl}}/return-proc/dos/do-processes/attaching-items-do/non-ser-items/the_attach_quantity_profile_attach_qty_to_a_do_and_dm_pr_content.html) is displayed  which enables you to enter the quantity returned to each stock area.


You can also directly enter the returned quantity in the **Debited**  column, if you select the **Use default 
 stock area to assign quantity for non-serialized items in debit documents**  option in the **Location** profile  (path: **File &gt; Setup &gt; Accounting 
 &gt; Locations/Sub-Location &gt; Stock Transfer - Miscellaneous**  tab). Otherwise, when you try to enter a quantity in the **Debited**  column, the **Attach Quantity** dialog  box is displayed where you can specify the stock area to which the items  are assigned and the item quantity.


You can also enter negative quantities (i.e. purchase items on the debit  memo).

- Enter the quantity  prefixed with a negative sign in the **Debited**  column, if you select the **Use default 
 stock area to assign quantity for non-serialized items in debit documents**  option (path: **File &gt; Setup &gt; Accounting 
 &gt; Locations/Sub-Location &gt; Stock Transfer - Miscellaneous**  tab).
- Use the **Transfer/Assign 
 new stock** option (path: **Debit 
 Memo** profile > **Options &gt; 
 Attach/Retrieve Serial Number**), if you do not select the **Use default stock area to assign quantity for 
 non-serialized items in debit documents** option. **Everest**  displays the [**Assign Quantity** dialog box]({{site.pp_baseurl}}/misc/the_assign_quantity_dialog_box_do_pr_contents_inv_items.html) when  you select this option.
