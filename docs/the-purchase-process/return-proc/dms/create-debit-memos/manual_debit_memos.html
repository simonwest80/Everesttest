Manual Debit Memos

When you manually create an entry in a purchase journal debiting the vendor, a manual debit memo is automatically created. The document number is prefixed with ‘M’ to denote it is a manual debit memo.

If you do not specify a document number, the sequence numbers for manual debit memos are automatically generated based on the settings in the **Location profile (path: File > Setup > Accounting > Location/Sub-Location > Options > Edit > In New Window > Purchase tab > Manual Memo # field) of the logged in location/sub-location. If a sequence number is not assigned for the login location/sub-location, the sequence number assigned for the Default Location (path: File > Setup > Preferences > Defaults > Accounting > Miscellaneous tab) is used.**

You can make modifications or accept refunds from within the manual document profile.

View Manual Debit Memos
You can view manual debit memos from the Debit Memos browser by selecting Accounting from the View menu (path: Debit Memos browser > View > Accounting). Manual debit memos have a DM# that is prefixed with “M”.  Manual documents also display “Accounting” in the Type column.
Modify Manual Debit Memos
You can modify manual debit memos from the Debit Memo profile just as you would with system documents. This is the preferred method for making manual debit memo modifications since only limited modifications are possible from the purchase journal.

To modify manual debit memos, do the following:

  • From the Debit Memos browser, select View > Accounting.
  • Select the manual debit memo (The DM# begins with “M” and the Type column displays “Accounting”) and press the Enter key or double-click it. You will see the manual debit memo profile.
  • Make the necessary modifications. You can update any of the following items:
  1. Add or edit Document Notes (Click the icon next to the Document #, press **Shift
    • F6, or select **Options > Document > Document Notes.)
    • Update custom fields (Select Options > Document > Custom fields or press Ctl + Alt + C.)
    • Change document alias (Click the icon next to the Document # or press **Ctl
    • Alt + D**.
    • Update fields in the manual debit memo, including Contact, Ship To, Purchase Rep, Terms, Due date, Override, Tax Invoice Number and Inv. date, Vendor’s Invoice Number and Date, Shipment Tracking #, Reference, and Loc/Sub-loc (Location/Sub-Location).
    • Click  to save your changes.
