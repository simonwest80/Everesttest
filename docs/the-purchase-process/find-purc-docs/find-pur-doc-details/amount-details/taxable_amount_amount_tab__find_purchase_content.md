# Taxable Amount


Document Amount


Specify the taxable amount based on which search has to be performed.  This amount appears in the **Taxable**  field of purchase documents.


Received on Invoice Amount


Specify the taxable amount based on which search has to be performed.  This amount appears in the **Taxable**  field of a purchase invoice.
