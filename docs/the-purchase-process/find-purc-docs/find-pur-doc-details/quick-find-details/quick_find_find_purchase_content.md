# Quick Find


The **Quick Find** tab lists the  most common search parameters used to find purchase documents. The **Quick Find** tab consists of the following  fields:

- [Document  Number]({{site.pp_baseurl}}/find-purc-docs/find-pur-doc-details/quick-find-details/document_number_quick_find_find_purchase_content.html)
- [Order  Status]({{site.pp_baseurl}}/find-purc-docs/find-pur-doc-details/quick-find-details/document_type_quick_find_find_purchase_content.html)
- [Currency]({{site.pp_baseurl}}/find-purc-docs/find-pur-doc-details/quick-find-details/currency_more_choices_find_purchase_content.html)
- [Account  Name]({{site.pp_baseurl}}/find-purc-docs/find-pur-doc-details/quick-find-details/account_name_quick_find_find_purchase_content.html)
- [Document  date]({{site.pp_baseurl}}/find-purc-docs/find-pur-doc-details/quick-find-details/document_date_quick_find_find_purchase_content.html)
- [Number  of records to retrieve]({{site.pp_baseurl}}/find-purc-docs/find-pur-doc-details/quick-find-details/number_of_records_to_retrieve_quick_find_find_purchase_content.html)



{:.example}
Document Number; Location/Sub-Location; Document Date


You can use these parameters individually or in combination with other  parameters to narrow down your search. The options selected in this tab  also appear in the **Advanced Find**  tab.
