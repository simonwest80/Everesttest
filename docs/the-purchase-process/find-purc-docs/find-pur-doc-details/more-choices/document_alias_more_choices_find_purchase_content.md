# Document Alias Group and Document Alias


**Document Alias Group**
: You can refine your search criteria by selecting  an alias group for documents.


{:.example}
All quotes, orders and invoices for service contracts may be grouped  under ‘Service Documents’. To find all documents created for service contracts,  enter 'Service Documents' in this field.


**Document Alias**
: You can find purchasing documents using the Document  Alias of the documents.


{:.example}
To find all Vendor RMA order, specify the  alias "Vendor RMA order" in this field.
