# Find Purchase Document - Quick Find


**Document number**
: Specify the document number as per the purchase  document.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Document  Number]({{site.pp_baseurl}}/find-purc-docs/find-pur-doc-details/quick-find-details/document_number_quick_find_find_purchase_content.html)


**Document Type**
: Specify the type of document that you want to find.


Processed
: Specify the details for searching purchasing documents  based on the Processed check box in the documents. The options available  are:

- **Yes**  - This option allows you to search for processed documents.
- **No**  - This option allows you to search for unprocessed documents.
- All  - **This option allows you to****search for both type of documents**.



Paid
: Specify the details for searching documents based  on the payment status. The options available are:

- **Yes**  - Searches for fully paid documents only
- **No**  - Searches for partly paid and unpaid documents only.
- **All**  – Searches for both paid and unpaid documents.



**Order Status**
: Specify the status of purchase and debit orders  that must be displayed in the search results.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Order  Status]({{site.pp_baseurl}}/find-purc-docs/find-pur-doc-details/quick-find-details/document_type_quick_find_find_purchase_content.html)


**Currency**
: Specify the document currency to base your search  on.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Currency]({{site.pp_baseurl}}/find-purc-docs/find-pur-doc-details/quick-find-details/currency_more_choices_find_purchase_content.html)


**Location/Sub-Location**
: Specify the location/sub-location based on which  search has to be performed.


**Purchase Representative**
: Specify the purchase representatives whose documents  must be listed in the search results. If the document is re-assigned to  another purchase representative, the search considers only the latter  one.


**Account Name**
: Specify the vendor’s account name as per the **Vendor** profile.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Account  Name]({{site.pp_baseurl}}/find-purc-docs/find-pur-doc-details/quick-find-details/account_name_quick_find_find_purchase_content.html)


**Document date**
: Specify the document creation date based on which  the search must be performed. When you select any option from the drop-down  list, the dates in the ‘From’ and ‘To’ fields get populated automatically.
: ![]({{site.pp_baseurl}}/img/lens.gif)  [Date  Filter]({{site.wwe_chm}}/misc/date_filter.html)


**Number of records to retrieve**
: Specify the number of records to retrieve.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Number  of records to retrieve]({{site.pp_baseurl}}/find-purc-docs/find-pur-doc-details/quick-find-details/number_of_records_to_retrieve_quick_find_find_purchase_content.html)
