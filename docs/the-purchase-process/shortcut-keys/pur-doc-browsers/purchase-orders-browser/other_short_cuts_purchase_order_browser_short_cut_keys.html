Other Shortcut Keys

Press Key(s) To
Alt+G Get all the available purchase order documents into the PurchaseOrders browser.
Ctrl+F4 Close the Purchase Orders browser.
Ctrl+Ins Create a new Purchase Order profile in the current window.
Enter Open and modify the selected PurchaseOrder profile.
F4 Toggle between Received Purchase Orders and Open Purchase Orders.
F5 Update the Purchase Orders browser with all current changes made.
Ins Create a new Purchase Order profile by opening a new window without closing the existing open window.
Shift+Ctrl+C Collapse grouping. Hides the records under each group and displays only the main groups.
Shift+Ctrl+E Expand grouping. Displays all the records under each group.
