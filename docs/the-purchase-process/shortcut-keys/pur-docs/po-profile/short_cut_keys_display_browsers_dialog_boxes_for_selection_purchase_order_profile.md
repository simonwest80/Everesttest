# Display Browsers/Dialog Boxes


| **Press Key(s)** | **To** |
| ~ | Display the **Retrieve** **Serial****Numbers** dialog box. |
| Alt+F8 | Display the **Description** dialog  box for the selected line item. |
| Ctrl+A | Display the **Vendor** **Aging****Information** browser. |
| Ctrl+Alt+C | Display the **Custom** **Fields**  dialog box from a saved purchase order.<br/><br/><br/>Display **Find** **Customers**  dialog box on a purchase order that has not been saved. |
| Ctrl+Alt+F8 | Display the **Detach** **Serial****#** dialog box. |
| Ctrl+Alt+L | Display the **Item** **Stock****Status** dialog box for the selected  line item. |
| Ctrl+Alt+S | Display the **Quantity** **Stock****Area** **Wise**  browser for the selected line item. This option is enabled in saved purchase  orders only for non-serialized items with committed quantity. |
| Ctrl+Alt+V | Display the **Serial** **#****for** **all****items** browser. This browser displays  all serial numbers that belong to all serialized items in the document. |
| Ctrl+Alt+W | Display the **Serial** **#** browser for the selected serialized item. |
| Ctrl+G | Display the **Vendor** **Tracking**  browser. |
| Ctrl+H | Display the **Attach** **Quantity**  dialog box only in cases when negative quantities of inventoried, non  - serialized items are attached to the purchase order. |
| Ctrl+I | Display **Items** browser for items  selection. |
| Ctrl+K | Display the **Retrieve** **Kits**  dialog box. |
| Ctrl+L | Display the **Item** **Stock****Status** dialog box for all items. |
| Ctrl+M | Display the **Matrix** **Items <br/> View** dialog box if a matrix item is selected. |
| Ctrl+O | Display the **Item** profile for  the selected line item. |
| Ctrl+P | Display the **Document** **Printing****Preferences** dialog box on a saved  order. |
| Ctrl+Q | Display the **Retrieve** **Serial****Numbers** (**Range**)  dialog box. |
| Ctrl+R | Display the **Payment** dialog box  for the purchase order profile. |
| Ctrl+Right Arrow | Display the **Purchase** **Document****Lines** dialog box for the selected  line item. |
| Ctrl+T | Display the **Transaction** **History** browser from a saved purchase  order. |
| Ctrl+U | Display the **Vendor** profile. |
| Ctrl+W | Display the **Multimedia** **Setup** browser. |
| F11 | Display the **Tax** **Breakdown**  dialog box. |
| F8 | Display the **Payment** **History**  browser for selected purchase order. |
| Shift+~ | Display the **Retrieve** **Mfg****Serial** **Numbers**  dialog box. |
| Shift+F5 | Display the **Custom** **Tracking**  browser. |
| Shift+F6 | Display the **Document** **Notes**  dialog box. |
| Shift+F7 | Display the **Vendors** browser  for vendor selection. |
| Shift+F8 | Display **Items** (Kits) browser  for kit selection. |
| Ctrl+E | Display the **Document E-mail Preferences**  dialog box. |
| Shift+Ctrl+K | Display the **Task** profile. |
| Shift+Ctrl+I | Display the **Incoming Call** profile. |
| Shift+Ctrl+O | Display the **Outgoing Call** profile. |
| Shift+Ctrl+N | Display the **General Notes** profile. |
| Shift+Ctrl+H | Display the **Task History** browser. |
| Shift+Ctrl+M | Display the **My Tasks** browser. |
| Shift+Ctrl+A | Display the **Appointment** profile. |
| Shift+Ctrl+L | Display the **Appointment History**  profile. |
| Shift+Ctrl+C | Display the **Calendar View** profile. |
| Ctrl+F12 | Display the **Find Line Items**  browser. |
| Ctrl+Y | Display the **Item Purchase History**  browser. |
