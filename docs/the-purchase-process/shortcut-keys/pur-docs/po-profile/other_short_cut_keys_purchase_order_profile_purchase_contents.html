Other Shortcut Keys

Press Key(s) To
Ctrl+Alt+H Import items into the document from third party catalog.
Ctrl+Alt+Space Preview on/off of the items.
Ctrl+DOWN Arrow Move selected Line Item one line down.
Ctrl+F4 Close the Purchase Order profile.
Ctrl+N Create a new purchase order from an order that has not been saved.
Ctrl+S Save the purchase order.
Ctrl+Space Zoom in/out of the item grid.
Ctrl+UP Arrow Move the selected Line Item one line up.
F4 Process the purchase order to a purchase invoice.
F5 Process the purchase order to a purchase receipt.
F6 Toggle between document currency and the base document.
Shift+Ctrl+C Collapse grouping. Hides the records under each group and displays only the main groups.
Shift+Ctrl+E Expand grouping. Displays all the records under each group.
Shift+Ctrl+Z Cancel the changes made to the PurchaseOrder profile.
Shift+F10 Display the group panel where you can drag and drop any column header to group the data by that column.
