Other Shortcut Keys

Press Key(s) To
Crtl+UP Arrow Move the selected line item one line up in the items details grid.
Ctrl+Alt+H Import items from the third party catalog.
Ctrl+Alt+Space Preview on/off of the item details grid.
Ctrl+DOWN Arrow Move the selected line item one line down in the items details grid.
Ctrl+F4 Close the Purchase Quote profile.
Ctrl+N Create a new Purchase Quote profile from a quote that has been saved.
Ctrl+S Save the Purchase Quote profile.
Ctrl+Space Zoom in/out of the item details grid.
F3 Process the purchase quote to a purchase order.
F6 Toggle between document currency and the base currency.
Shift+Ctrl+Z Cancel the changes made to the PurchaseQuote profile.
Shift+F10 Display the group panel where you can drag and drop any column header to group the data by that column.
