# Other Shortcut Keys


| **Press Key(s)** | **To** |
| Ctrl+Alt+H | Import Items from the third party catalog. |
| Ctrl+Alt+Space | Preview on/off of the items. |
| Ctrl+DOWN Arrow | Move the selected Line Item one line down. |
| Ctrl+F4 | Close the **Debit** **Order**  profile. |
| Ctrl+N | Create a new **Debit** **Order**  profile from a saved debit order. |
| Ctrl+S | Save the debit order. |
| Ctrl+Space | Zoom in/out of the item grid. |
| Ctrl+UP Arrow | Move the selected Line Item one line up. |
| F4 | Process the debit order to a debit memo. |
| F6 | Toggle between document currency and base currency. |
| Shift+Ctrl+Z | Cancel the changes made to the **Debit****Order** profile. |
| Shift+F10 | Display the group panel where you can drag and drop any column header  to group the data by that column. |
