Other Shortcut Keys

Press Keys To
Ctrl+Alt+H Item Import form third party catalog.
Ctrl+Alt+Space Preview on/off of the items.
Ctrl+DOWN Arrow Move the selected Line Item one line down.
Ctrl+F4 Close the Debit Quote profile.
Ctrl+N Create a new Debit Quote profile.
Ctrl+S Save the Debit Quote profile.
Ctrl+Space Zoom in/out of the item grid.
Ctrl+UP Arrow Move the selected Line Item one line up.
F3 Process the debit quote to a debit order.
F6 Toggle between document currency and the base currency.
Shift+Ctrl+Z Cancel the changes made to the DebitQuote profile.
Shift+F10 Display the group panel where you can drag and drop any column header to group the data by that column.
