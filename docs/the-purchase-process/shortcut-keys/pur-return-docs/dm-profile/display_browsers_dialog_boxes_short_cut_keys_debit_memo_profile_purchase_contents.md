# Display Browsers/Dialog Boxes


| Press Key(s) | **To** |
| ~ | Display the **Retrieve** **Serial****Numbers** dialog box on a debit  memo that has not been saved. |
| Alt+F8 | Display the **Description** dialog  box for the selected line item. |
| Ctrl+A | Display the **Vendor** **Aging****Information** browser. |
| Ctrl+Alt+C | Display the **Custom** **Fields**  dialog box on debit memo that has been saved.<br/><br/><br/>Display **Find** **Customers**  dialog box on a debit memo that has not been saved. |
| Ctrl+Alt+F8 | Display **Detach** **serial <br/> #** dialog box on a debit memo that has not been saved. |
| Ctrl+Alt+I | Displays the **Assign Quantity**  dialog box. This option will be available only if the debited quantity  is zero. |
| Ctrl+Alt+L | Display the **Item** **Stock****Status** dialog box for the selected  line item. |
| Ctrl+Alt+S | Display **Quantity** **stock****area** **wise**  dialog box. In a saved debit memo this option is enabled only for non-serialized  items if the Debited quantity is greater than 0. |
| Ctrl+Alt+V | Display the **Show Serial # for all items**  browser. |
| Ctrl+G | Display the **Custom** **Tracking**  browser for the vendor. |
| Ctrl+H | Display the **Attach** **Quantity**  dialog box only in cases when negative quantities of inventoried, non  - serialized items are attached to the purchase order. |
| Ctrl+I | Display the **Items** browser for  item selection. |
| Ctrl+K | Display the **Retrieve** **Kits**  dialog box. |
| Ctrl+L | Display the **Item** **Stock****Status** dialog box for all items. |
| Ctrl+M | Display the **Matrix** **Items****View** dialog box for a debit memo  that has not been saved. |
| Ctrl+O | Display the **Item** profile for  the selected line item. |
| Ctrl+Q | Display the **Retrieve****Serial** **Numbers**  (Range) dialog box. |
| Ctrl+R | Display the **Payment** dialog box  for a debit memo that has been saved and has a balance. |
| Ctrl+Right Arrow | Display the **Item** profile for  the selected line item. |
| Ctrl+T | Display the **Transaction** **History** browser for a debit memo that  has been saved. |
| Ctrl+U | Display the **Vendor’s** profile. |
| Ctrl+W | Display the **Multimedia** **Setup** browser for a saved debit memo. |
| F11 | Display the **Tax** **Breakdown**  dialog box. |
| F8 | Display the **Payment** **History**  browser for the debit memo on which payment has been made. |
| Shift+~ | Display the **Retrieve** **Mfg****Serial** **Numbers**  dialog box for a debit memo that has not been saved. |
| Shift+F5 | Display the **Document** **Tracking**  dialog box for a debit memo that has been saved. |
| Shift+F6 | Display the **Document** **Notes**  dialog box. |
| Shift+F7 | Display the **Vendors’** browser  for vendor selection for a debit memo that has not been saved. |
| Shift+F8 | Display the **Item** (Kits) browser  for kits selection. |
| Ctrl+E | Display the **Document E-mail Preferences**  dialog box. |
| Shift+Ctrl+K | Display the **Task** profile. |
| Shift+Ctrl+I | Display the **Incoming Call** profile. |
| Shift+Ctrl+O | Display the **Outgoing Call** profile. |
| Shift+Ctrl+N | Display the **General Notes** profile. |
| Shift+Ctrl+H | Display the **Task History** browser. |
| Shift+Ctrl+M | Display the **My Tasks** browser. |
| Shift+Ctrl+A | Display the **Appointment** profile. |
| Shift+Ctrl+L | Display the **Appointment History**  profile. |
| Shift+Ctrl+C | Display the **Calendar View** profile. |
| Ctrl+F12 | Display the **Find Line Items**  browser. |
| Ctrl+Y | Display the **Item Purchase History**  browser. |
