# Other Shortcut Keys


| Press Key(s) | **To** |
| Ctrl+Alt+H | Import items form third party catalog. |
| Ctrl+Alt+Space | Preview on/off of the items. |
| Ctrl+DOWN | Move the selected Line Item one line down. |
| Ctrl+F4 | Close the **Debit** **Memo**  profile. |
| Ctrl+N | Create a new **Debit** **Memo**  profile from a saved debit memo. |
| Ctrl+S | Save the **Debit** **Memo**  profile. |
| Ctrl+Space | Zoom in/out of the item grid. |
| Ctrl+UP | Move the selected Line Item one line up. |
| F6 | View the **Debit** **Memo**  profile in the document currency. |
| Shift+Ctrl+Z | Cancel the changes made to the **Debit****Memo** profile. |
| Shift+F10 | Display the group panel where you can drag and drop any column header  to group the data by that column. |
