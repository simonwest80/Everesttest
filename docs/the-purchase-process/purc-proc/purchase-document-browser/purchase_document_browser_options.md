# Common Purchase Document Browser Options


The options common to all purchase document browsers are:


**View**
: This option allows you to view a purchase document  even if you do not have access to modify a document.
: The **View**  menu has the following submenu options:


| In Current Window | Select this option when you want to close the existing open document  and open the new document in the same window. |
| In New Window | Select this option to open the document in a new window without closing  the document that is currently open. |



****Transaction****History****
: Select this option to view the transaction history  of a selected document.


![]({{site.pp_baseurl}}/img/lens.gif) [Purchase  Documents - Transaction history]({{site.pp_baseurl}}/purc-proc/doc-profile/doc-options/transaction-history/transaction_history.html)


****Document****Tracking****
: Select this option to view and edit the tracking  group and selections of a selected document. .


![]({{site.pp_baseurl}}/img/lens.gif) [Purchase  Documents - Document Tracking]({{site.pp_baseurl}}/purc-proc/doc-profile/doc-options/ex-doc-details/document_tracking_add_info.html)


**Multimedia**
: Select this option to view and add multimedia files  for a selected document.


![]({{site.pp_baseurl}}/img/lens.gif) [Purchase  Documents - Multimedia]({{site.pp_baseurl}}/purc-proc/doc-profile/doc-options/ex-doc-details/multimedia_purchases.html)


**Print Document**
: Select this option to print the selected documents.  You can print multiple documents at the same time.


![]({{site.pp_baseurl}}/img/lens.gif) [Printing  Purchase Documents]({{site.pp_baseurl}}/purc-proc/doc-profile/doc-options/prnt-purc-docs/printing_purchase_documents.html)


**Print Shipping Label**
: Select this option to print shipping labels for  the selected documents. You can print multiple shipping labels at the  same time. **Everest** provides you  with 5 standard formats for shipping labels.
: ![]({{site.pp_baseurl}}/img/lens.gif)  [Shipping  Label Formats]({{site.pp_baseurl}}/misc/shipping_label_formats.html)


**Contact Manager**
: Select this option to view tasks, appointments and  incoming/outgoing calls associated with the selected documents.


![]({{site.pp_baseurl}}/img/lens.gif) [Purchase  Documents - Contact Manager]({{site.pp_baseurl}}/purc-proc/doc-profile/doc-options/contact-manager/contact_manager_purchase.html)


**Search by Vendor**
: This option allows you to search for documents by  the vendor name. Highlight a vendor’s record from the browser and select  this option. You will see a submenu that lists the document types. Select  the relevant document type. All the documents of that type are listed  for the vendor.


**Print Serial Numbers**
: This option allows you to print the serial numbers  of items attached to a purchase document.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Print  Serial Numbers]({{site.pp_baseurl}}/misc/print_serial_numbers_pur_browser_options_pur_con.html)


**E-mail Documents**
: This option allows you to send multiple documents  by e-mail to more than one vendor. Select the required documents before  selecting this option. When you select multiple documents, they are attached  to a single e-mail, and **Everest**  warns you that the documents belong to different vendors.
: The contact addresses of the vendors in the selected  purchase documents are displayed in the **To**  field, with the document numbers in the **Subject**  field. The **Cc** field displays the  billing addresses of all the vendors, if the billing address is not already  used for the **To** field. If the  Default contact address and the billing address have the same e-mail ID,  then the e-mail address is displayed only in the **To**  field. You can double click an account name and view the e-mail address  specified for the selected vendor. Additional e-mail addresses for users,  customers and vendors can be added from the **Address 
 Book**, which can be accessed by clicking the **To**  or **Cc** buttons.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Document  E-mail Preferences Dialog Box]({{site.pp_baseurl}}/purc-proc/doc-profile/doc-options/email-purc-docs/document_e_mail_preferences_dialog_box_email_common_pur_doc_options.html)
: [Pre-requisites  for Emailing Documents]({{site.pp_baseurl}}/misc/pre_requisites_for_e_mailing_documents.html)


Options common to all browsers are also available in this browser.
