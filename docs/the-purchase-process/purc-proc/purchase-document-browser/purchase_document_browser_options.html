Common Purchase Document Browser Options

The options common to all purchase document browsers are:

View
This option allows you to view a purchase document even if you do not have access to modify a document.
The View menu has the following submenu options:
In Current Window Select this option when you want to close the existing open document and open the new document in the same window.
In New Window Select this option to open the document in a new window without closing the document that is currently open.
**TransactionHistory**
Select this option to view the transaction history of a selected document.

Purchase Documents - Transaction history

**DocumentTracking**
Select this option to view and edit the tracking group and selections of a selected document. .

Purchase Documents - Document Tracking

Multimedia
Select this option to view and add multimedia files for a selected document.

Purchase Documents - Multimedia

Print Document
Select this option to print the selected documents. You can print multiple documents at the same time.

Printing Purchase Documents

Print Shipping Label
Select this option to print shipping labels for the selected documents. You can print multiple shipping labels at the same time. Everest provides you with 5 standard formats for shipping labels.
 Shipping Label Formats
Contact Manager
Select this option to view tasks, appointments and incoming/outgoing calls associated with the selected documents.

Purchase Documents - Contact Manager

Search by Vendor
This option allows you to search for documents by the vendor name. Highlight a vendor’s record from the browser and select this option. You will see a submenu that lists the document types. Select the relevant document type. All the documents of that type are listed for the vendor.
Print Serial Numbers
This option allows you to print the serial numbers of items attached to a purchase document.
Print Serial Numbers
E-mail Documents
This option allows you to send multiple documents by e-mail to more than one vendor. Select the required documents before selecting this option. When you select multiple documents, they are attached to a single e-mail, and Everest warns you that the documents belong to different vendors.
The contact addresses of the vendors in the selected purchase documents are displayed in the To field, with the document numbers in the Subject field. The Cc field displays the billing addresses of all the vendors, if the billing address is not already used for the To field. If the Default contact address and the billing address have the same e-mail ID, then the e-mail address is displayed only in the To field. You can double click an account name and view the e-mail address specified for the selected vendor. Additional e-mail addresses for users, customers and vendors can be added from the Address Book, which can be accessed by clicking the To or Cc buttons.
