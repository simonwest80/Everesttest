# Purchase Receipt Profile Options


**Payment History**
: The [**Payment 
 History** browser]({{site.pp_baseurl}}/purc-proc/pos/po-processes/payments/payment-history/payment_history_pur.html) (path: **Options 
 &gt; Payment &gt; Payment History**) displays payment/refund details  for the document. This option is enabled only if a payment is made on  the purchase receipt.


**Show Serial Number**
: The **Show Serial 
 Number** option (path: **Options &gt; 
 Item &gt; Show Serial #**) displays all the serial numbers that are  attached to a selected item.


**Serial Number History**
: This option (path: **Options 
 &gt; Serial Number History**) displays a history of movement of a  serial number between different stock areas and the documents in which  the movement was recorded. It displays the date on which a serial number  was generated, the date it entered stock, the purchase order or purchase  receipt on which it was generated, the stock area to which it was received,  the sales order to which it is attached and the area it was moved to when  it was attached etc.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Serial  Number History Browser]({{site.pp_baseurl}}/purc-proc/pos/po-processes/cmt-items-po/enter-cmt-qty/serial-number-history-browser/serial_number_history_browser_pur_con.html)


**Price Summary**
: This option brings up the **Purchase Receipt 
 Doc #** window, which displays the original values that were  saved after creating the purchase receipt as well as the values that were  saved after the last modification. This option will be enabled only when  the columns like Item Cost, Discount Value, Discount %, Tax Code, Job  Code, and Add-on Cost of Purchase receipt are modified.


**Show All Serial #**
: This option brings up the [Serial  # for All Items browser]({{site.pp_baseurl}}/misc/serial_for_all_items_browser_commit_items_pur_con.html).  All serial numbers attached to the document are displayed in this browser.


**Show/Attach area-wise stock**
: This option (path: **Purchase 
 Receipt** profile > **Attach**/**Retrieve** **Serial****number &gt; Show/Attach area-wise stock**)  is available only for non-serialized items. Select this option to view  the stock areas to which received quantities of the selected item are  assigned.


**Transfer/Assign new stock**
: Select this option to assign quantities in the purchase  document. **Everest** displays the  **Assign 
 Quantity** dialog box.


**Detach Serial #**
: This option allows you to detach the serial number  attached to an item in the purchase receipt before it is saved. Enter  the serial number in the **Detach serial 
 #** field.


The other options available in the **Purchase 
 Receipt** profile are:

- [View  Tax Breakdown]({{site.pp_baseurl}}/purc-proc/pos/po-processes/tax-breakdown/tax_breakdown_pur_options_payment_tab_breakdown.html)
- [Fill  Sales Order]({{site.pp_baseurl}}/purc-proc/prs/fill-sales-order/fill_sales_order.html)
