# Back End Processes - Cancel a Purchase Order


The following back end processes occur when a purchase order is canceled:

- The purchase  order is converted to a purchase quote.
- The quote bears  the purchase quote number assigned to the cancelled purchase order.
- The sequence  number of the cancelled purchase order is not available for further use.
