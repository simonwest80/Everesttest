# Serialized Inventory


In case of serialized inventory, you can detach serial numbers from  the purchase order by selecting:

- The **Detach****Serial #** option from the **Purchase** **Order**  profile.



{:.note}
Only serial numbers attached to negative quantities  can be detached using this option.

- The **De-serialize**  option from the **Serial** **#****for all items**/**Serial 
 #** browser.



![]({{site.pp_baseurl}}/img/steps.gif)To detach a committed serial number from the  purchase order profile, do the following:

- From the **Purchase** **Order**  profile, select **Options** > **Detach** **Serial****#**. You will see the **Retrieve 
 Serial** **Numbers** dialog box.
- Enter the serial  number of the item to be detached and click **Apply**.



![]({{site.pp_baseurl}}/img/steps.gif)To detach committed serial numbers from the  serial # for all items browser, do the following:

- From the **Purchase** **Order**  profile, select **Options** > **Show** **All****Serial** **#**.  You will see the **Serial #** **for all items** browser.
- From the **Serial # for all items** browser, highlight  the required serial number and select **De-serialize**  from the right-click menu.
