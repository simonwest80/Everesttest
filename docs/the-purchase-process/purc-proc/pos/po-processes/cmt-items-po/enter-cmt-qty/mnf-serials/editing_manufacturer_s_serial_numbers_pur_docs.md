# Edit Manufacturer's Serial Numbers


You can edit the manufacturer’s serial number from the **Serial 
 Number for all Items** browser (path: purchase document profile >  **Options** > **Show****All** **Serial**  #) **OR** the **Serial 
 #** browser (path: purchase document profile > **Options**  > **Items** > **Show****Serial** **#**).  You can also edit the serial number from the [**Assign Manufacturer's Serial Number**  dialog box]({{site.pp_baseurl}}/purc-proc/pos/po-processes/cmt-items-po/enter-cmt-qty/mnf-serials/assign_manuf_serial_no.html).


![]({{site.pp_baseurl}}/img/steps.gif)To edit the manufacturer’s serial number,  do the following:

- From the **Serial** **#** browser or **Serial Number for all 
 Items** browser, click the manufacturer’s serial number column for  the required item.
- Enter the new  serial number.
- Click **OK**.
