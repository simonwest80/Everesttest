# View Serial Numbers


![]({{site.pp_baseurl}}/img/steps.gif)To view serial numbers committed to a serialized/auto-serialized  item, do the following:

- From the **Purchase Order** profile, select the line  item.
- Select **Options** > **Item** > **Show Serial #**.  You will see the **Serial 
 #** browser.



![]({{site.pp_baseurl}}/img/steps.gif)To view serial numbers committed to all items  in the document, do the following:

- From the **Purchase Order** profile, select **Options** > **Show Serial #.** You will see the [**Serial # for All Items** browser]({{site.pp_baseurl}}/misc/serial_for_all_items_browser_commit_items_pur_con.html).
