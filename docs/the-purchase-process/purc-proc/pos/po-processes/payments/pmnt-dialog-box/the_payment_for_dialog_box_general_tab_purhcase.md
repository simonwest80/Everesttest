# The Payment Dialog Box - General


The **General** tab of the **Payment** dialog box allows you to enter  the details of payment such as the payment method and amount.


The fields in this tab is grouped under the following sections:

- [Payment  Amount Details]({{site.pp_baseurl}}/purc-proc/pos/po-processes/payments/pmnt-dialog-box/payment_amount_details.html)
- [Payment  Reference Details]({{site.pp_baseurl}}/purc-proc/pos/po-processes/payments/pmnt-dialog-box/payment_reference_details.html)
