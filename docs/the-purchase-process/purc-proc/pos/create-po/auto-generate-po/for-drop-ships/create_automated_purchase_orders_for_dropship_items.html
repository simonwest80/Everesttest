Create Automated Purchase Orders for Dropship Items

To create purchase orders for dropship items on sales orders, do the following:

  • From the main menu bar, select Inventory > Automated Purchases > For Dropships. You will see the **Automated Purchases
  • Drop Ship Items** browser.
  • Deselect any item if you do not want to create a purchase order for it.
  • Select Options > Generate POs. You will be asked to confirm your selection.
  • Click Yes. You will see the Purchase Order Status dialog box, which displays a list of purchase orders created.
  • Double-click a document to view it.
