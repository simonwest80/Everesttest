# Create a Purchase Order


![]({{site.pp_baseurl}}/img/steps.gif)To create a purchase order from the purchase  orders browser, do the following:

- From the main  menu bar, select **Purchasing** > **Purchase Order**. You will see the **Purchase Orders** browser.
- From the **Purchase Orders** browser, select **Options** > **New** > **In New Window**.  You will see the [**Purchase 
 Order** profile]({{site.pp_baseurl}}/purc-proc/pos/create-po/create-a-new-purchase-order/the-purchase-order-profile/the_purchase_order_profile.html).



{:.note}
You can also create a purchase order from  an existing **Purchase Order** profile  by selecting **Options &gt; New**.
