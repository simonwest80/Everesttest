Purchase Order Profile Options

The options available in a purchase order (path: Purchase Order profile > Options) are:

Payment History
The Payment History browser ** (path: **Options > Payment > PaymentHistory) displays payment/refund details for a purchase document.

This option is enabled only if a payment is made on the purchase order.

**Show all Serial
#**
The Showall serial# option (path: Options > Show allserial #) displays all serial numbers attached to the document.

The Serial # for all Items browser displays serial numbers only if they are attached to the document.

Show/Attach Area-wise Stock
The Show/AttachArea-wise stock option (path: Options > Attach/RetrieveSerial # > Show/Attach Area-wisestock) allows you to allows you to attach negative quantities of non-serialized items.
This option is available only for inventoried items that are not serialized.
 Enter Negative Quantities
Show serial number
The ShowSerial Number option (path: Options > Item > ShowSerial #) displays all the serial numbers that are attached to a selected item.
Serial number history
The Serialnumber history option (path: Options > Serial History History) displays a history of movement of a serial number between different stock areas and the documents in which the movement was recorded. It displays the date on which a serial number was generated, the date it entered stock, the purchase order or purchase receipt on which it was generated, the stock area to which it was received, the sales order to which it is attached and the area it was moved to when it was attached etc.
Serial Number History Browser
Detach Serial #
This option allows you to detach the serial number attached to an item in the purchase order. Enter the serial number of the item in the Detach serial # field.

The other options available on the Purchase Order profile are:

  • Mark Order Open/Close
  • Cancel an Order
  • Process the Order to a Purchase Receipt
  • Process the Order to a Purchase Invoice
  • Recurrence Pattern
  • Fill Sales Order
