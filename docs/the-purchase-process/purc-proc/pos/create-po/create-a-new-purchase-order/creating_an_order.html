Create a Purchase Order

To create a purchase order from the purchase orders browser, do the following:

  • From the main menu bar, select Purchasing > Purchase Order. You will see the Purchase Orders browser.
  • From the Purchase Orders browser, select Options > New > In New Window. You will see the Purchase Order profile.

You can also create a purchase order from an existing Purchase Order profile by selecting Options > New.
