# Sales/Purchase Analysis Browser


The **Sales/Purchase Analysis**  browser is displayed when you specify details in the **Sales/Purchase 
 Analysis** dialog box (path: **Inventory**  > **Sales/Purchase Analysis**)  and click **OK**.


This browser lists details of all purchase and sales of all items during  the specified period.


**View Measure**


You can view the Sales/Purchase Analysis in different units of measure:


**Stock in:**


**Stock Measure**


Select this option to view the Sales/Purchase Analysis in the stock  measure. This option is selected by default.


**Default Purchase Measure**


Select this option to display the Sales/Purchase Analysis in the default  purchase measure.


**Default Sales Measure**


Select this option to display the Sales/Purchase Analysis in the default  sales measure.


{:.note}
You can only view the **Sales/Purchase 
 Analysis** browser in different units of measure if the **Show 
 Details** box in the **Sales/Purchase 
 Analysis** dialog box is unchecked.


The additional options available in this browser are standard options  common to all browsers in **Everest**.
