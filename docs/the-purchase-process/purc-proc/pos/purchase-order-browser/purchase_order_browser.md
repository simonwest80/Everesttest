# The Purchase Orders Browser 


The **Purchase Orders** browser  (path: **Purchasing** > **Purchase****Orders**) displays all open (not  fully received and invoiced) purchase orders. It contains the following  options:

- [Print  Options]({{site.pp_baseurl}}/misc/print_options_po_browser_options.html)


- Print Serial Numbers
- Print Document


- [Serial  Number History]({{site.pp_baseurl}}/misc/serial_number_history_po_browser_options.html)
- [Payment  Information]({{site.pp_baseurl}}/purc-proc/pos/po-processes/payments/payment-history/payment_history_pur.html)
- [Other  Options]({{site.pp_baseurl}}/misc/other_options_po_browser_options.html)


- Search by Vendor
- Process > Unreceived  Orders
- Process > Uninvoiced  Orders
- Fully Received and  Invoiced/Open Purchase Orders
- Include/Exclude Manual  Documents


- [Standard  Browser Options]({{site.wwe_chm}}/everest-client/ui/browsers/standard_browser_options.html)
- [Purchase  Document Browser Options]({{site.pp_baseurl}}/purc-proc/purchase-document-browser/purchase_document_browser_options.html)



Filter records based on date


The **Dates** field in the **Purchase** **Orders** browser enables you to filter purchase orders based on their creation  date. By default, the browser displays all documents created during the  period specified in the **Select the default 
 date option based on which the records will be displayed in the purchase 
 browser** option (path: **File &gt; 
 Setup &gt; Preferences &gt; Flow Control &gt; Documents &gt; Prompts &gt; 
 Purchase Document** tab).
