The Purchase Orders Browser

The Purchase Orders browser (path: Purchasing > PurchaseOrders) displays all open (not fully received and invoiced) purchase orders. It contains the following options:

  • Print Options

  • Print Serial Numbers

  • Print Document

  • Serial Number History
  • Payment Information

  • Other Options

  • Search by Vendor
  • Process > Unreceived Orders
  • Process > Uninvoiced Orders
  • Fully Received and Invoiced/Open Purchase Orders

  • Include/Exclude Manual Documents

  • Standard Browser Options
  • Purchase Document Browser Options

Filter records based on date

The Dates field in the Purchase Orders browser enables you to filter purchase orders based on their creation date. By default, the browser displays all documents created during the period specified in the Select the default date option based on which the records will be displayed in the purchase browser option (path: File > Setup > Preferences > Flow Control > Documents > Prompts > Purchase Document tab).
