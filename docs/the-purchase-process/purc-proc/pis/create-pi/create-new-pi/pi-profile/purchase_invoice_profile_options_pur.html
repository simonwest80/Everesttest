Purchase Invoice Profile Options

Payment History
The Payment History browser (path: Options > Payment > Payment History) displays payment/refund details for the document. This option is enabled only if a payment is made on the purchase invoice.
Freight Estimation
Everest allows you to computer freight using the Everest Freight Estimator.
Show Serial Number
The Show Serial Number option (path: Options > Item > Show Serial #) displays all the serial numbers that are attached to a selected item.
Serial Number History
This option (path: Options > Serial Number History) displays a history of movement of a serial number between different stock areas and the documents in which the movement was recorded. It displays the date on which a serial number was generated, the date it entered stock, the purchase order/receipt/invoice on which it was generated, the stock area to which it was received, the sales order to which it is attached and the area it was moved to when it was attached etc.
Serial Number History Browser
Price Summary
This option brings up the Purchase Invoice Doc # Price Summary which displays the original values that were saved after creating the purchase invoice as well as the values that were saved after the last modification. This option will be enabled only when the columns like Item Cost, Discount Value, Discount %, Tax Code, and Add-on Cost of Purchase invoice are modified.
Show All Serial #
This option brings up the Serial # for All Items browser. All serial numbers attached to the document are displayed in this browser.
Show/Attach area-wise stock
Use this option (path: Options > Attach/Retrieve Serial number > Show/Attach area wise stock) to attach negative quantities to the purchase invoice. This option is enabled if you do not select the Use default stock area to receive items when creating a purchase receipt/invoice option in the Location profile (path: File > Setup > Accounting > Locations/Sub-Location > Stock Transfer - Miscellaneous tab).
This option is available only for inventoried items that are not serialized.
Transfer/Assign new stock
Select this option (path: Options > Attach/Retrieve Serial # > Transfer/Assign New Stock) to assign quantities in the purchase invoice. Everest displays the Assign Quantity dialog box.
Select this option to assign quantities in the purchase document. Everest displays The Assign Quantity Dialog Box.
Detach Serial #
Use this option to detach the serial number attached to an item in the purchase invoice before it is saved. Enter the serial number in the Detach serial # field.

Other Options

The other options available in the Purchase Invoice profile are:

  • Make Payments on a Purchase Invoice
  • Receive Refund
  • View Tax Breakdown
  • View Received Amounts
  • Fill Sales Order
