# Fill Sales Order


The **Fill** **Sales****Order** option (path: **Purchase****Invoice** profile > **Options**  > **Fill** **Sales****Order**) gives you information on  the 'Open' sales orders that can be fulfilled by the current invoice.


{:.note}
This option is enabled only if the purchase  invoice is saved.


![]({{site.pp_baseurl}}/img/steps.gif)To view the quantity that can be fulfilled  by the current purchase invoice, do the following:

- From the **Purchase** **Invoice** profile, select **Options**  > **Fill** **Sales****Order**. You will see the [**Parts** **Fulfillment**  dialog box]({{site.pp_baseurl}}/misc/parts_fulfillment_dialog_box_fill_sales_order_purchase_recipt_options.html).
- Select the  required options and click **OK**.  You will see the [Fill Sales  Order Browser]({{site.pp_baseurl}}/purc-proc/pis/fill-sales-order/fill_sales_order_browser_pi_pur.html).
