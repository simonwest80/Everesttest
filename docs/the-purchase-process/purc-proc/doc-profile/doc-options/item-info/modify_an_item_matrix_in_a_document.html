Modify an Item Matrix

Matrix item details can be modified in the Item Details Grid individually or by bringing up the Matrix Items View dialog box for the matrix items in the matrix group.

To modify an item matrix, do the following:

  • From the purchase document profile highlight the matrix item.
  • Select Options > Items > Edit Matrix Group. You will see the MatrixItems View dialog box.
  • Make the required changes and click OK.
