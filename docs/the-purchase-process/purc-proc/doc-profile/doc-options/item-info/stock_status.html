Stock Status

The Stock Status option (path: Purchase/Purchase Return Document profile > Options > Item > StockStatus) displays the Item Stock Status dialog box for the selected inventoried/serialized/auto-serialized item in the purchase document.

Inventoried Items

Stock Status for All Items
This option (path: purchase/purchase return document profile > Options > Item > Stock Status for All Items) displays the Item Stock Status dialog box for all the inventoried/serialized/auto-serialized items contained in the purchase/purchase return document.
