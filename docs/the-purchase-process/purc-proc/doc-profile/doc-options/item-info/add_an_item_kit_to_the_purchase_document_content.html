Add an Item Kit

To add an item kit to the purchase document, do the following:

  • From the purchase document profile, select Options > Item > Select Kits. You will see the Items browser.
  • From the Items browser (which will now display only a list of kits), highlight the kit and click OK. You will see the RetrieveKits dialog box.
  • You can select any number of kits from the Items browser. The Retrieve Kits dialog box is displayed once for every kit that you select.
  • The measure of kit components in documents can be changed.
