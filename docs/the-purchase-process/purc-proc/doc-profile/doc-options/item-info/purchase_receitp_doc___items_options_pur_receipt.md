# Purchase Receipt Doc # window - Price Summary


The **Purchase Receipt Doc #** window  displays the Line Items for which there has been any change in the values  of the fields mentioned below:

- Item Cost
- Discount Value
- Discount %
- Tax Code
- Add on Cost



This window displays the summary of the original values that were saved  after creating the purchase receipt as well as the values that were saved  after the last modification. This option will be enabled only when the  purchase receipt is changed for the fields mentioned above.


{:.note}
Display of values will be as per the Decimal  Accuracy defined and will be based on the Purchase UOM as defined in the  Line Item of the purchase receipt and not the default UOM.
