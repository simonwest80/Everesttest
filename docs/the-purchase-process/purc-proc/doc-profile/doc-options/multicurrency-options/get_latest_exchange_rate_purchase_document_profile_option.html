Get Latest Exchange Rate

The Get Latest Exchange Rate option (path: Purchase Document profile > Options > Get Latest Exchange Rate) allows you to retrieve the latest exchange rate on a purchase document, as on the login date. You can edit the exchange rate.

Login Time and Date

The exchange rate is retrieved from the Buy Rate/Sell Rate fields in the Exchange Rates profile (path: File > Setup > Accounting > Multi-Currency > Exchange Rates > Options > Edit > In New Window).

Exchange rate profile

Whether the Buy Rate/Sell Rate is displayed, in turn depends on the option selected in the Rates for purchase document field in the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting > Multi Currency > Exchange Rates tab).
