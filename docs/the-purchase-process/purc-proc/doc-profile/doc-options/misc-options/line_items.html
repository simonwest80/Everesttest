Line Items

The Line Items option (path: Purchase Document/Purchase Return Document > Options > Line Items) allows you to:

Option Description
Insert Inserts a blank line in the Item Details Grid.
Line Profile Displays information about the line highlighted in the Item Details Grid. This option brings up the Purchase DocumentLines dialog box.
Move up Moves the selected line item one line up.
Move down Moves the selected line item one line down.
Find Locates a line item in the purchase/purchase return document. When you have a number of items in a purchase/purchase return document use this option to find a line item you want to edit. You will see the Find Line Items browser.
