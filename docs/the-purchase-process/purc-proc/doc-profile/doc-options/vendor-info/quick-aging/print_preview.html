Print (Vendor Aging Information)

The Print option (path: Vendor Aging Information browser ** > **Options > Print) allows you to preview and print the documents listed in the browser.

To view the print setup dialog box, do the following:

  • From the Purchase document profile, select Vendor > Quick Aging. You will see Vendor AgingInformation browser.
  • Select Options > Print. You will see the Print Setup dialog box.
