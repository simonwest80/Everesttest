# Browser Column Filters


You can filter for documents that match specified criteria for each  column listed in the **Vendor** **Aging** **Information**  browser.


{:.example}
If you click on the column **Document****#**, you will see a list of the  document numbers of all the documents in the browser.


![]({{site.pp_baseurl}}/img/steps.gif)To filter a column, do the following:

- From any column  title, click the down arrow ![]({{site.pp_baseurl}}/img/pur_drop_down_button.gif). You will see a list with  the options **All**, **Custom**  and a list data in that column.
- Click the option  (**All**, **Custom**  or a document number) that you want the column to be filtered on.



![]({{site.pp_baseurl}}/img/steps.gif)To customize a filter for a column, do the  following:

- From any column  title, click the down arrow ![]({{site.pp_baseurl}}/img/pur_drop_down_button.gif). You will see a list of options.
- From the list  of options, select **Custom**. You  will see the **Custom AutoFilter**dialog box.
- Define the  settings in the dialog box and click **OK**.
