# Print (Vendor Aging Information)


The **Print** option (path: **Vendor Aging Information** browser **** > **Options**  > **Print**) allows you to preview  and print the documents listed in the browser.


![]({{site.pp_baseurl}}/img/steps.gif)To view the print setup dialog box, do the  following:

- From the Purchase  document profile, select **Vendor**  > **Quick** **Aging**.  You will see **Vendor** **Aging****Information** browser.
- Select **Options** > **Print**.  You will see the [**Print Setup** dialog box]({{site.pp_baseurl}}/misc/the_print_setup_dialog_box_aging_options_pur_pur_ret.html).
