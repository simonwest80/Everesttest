# The Document Printing Preferences Dialog Box


**Report Type**
: Select the type of document report to be printed.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Report  Type]({{site.pp_baseurl}}/misc/report_type_pur.html)


**Company Information**
: Select the type of information that is to be printed  on the document.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Printing  Company Information]({{site.pp_baseurl}}/misc/printing_company_information.html)


**Billing Address**
: Select the Billing address format.


![]({{site.pp_baseurl}}/img/lens.gif) [Billing  Address]({{site.pp_baseurl}}/misc/billing_address_purchase.html)


**Printer**
: Select the printer to which this print job must  be directed. All printers set up on your workstation are listed for selection.


**Copies**
: Enter the number of copies that are to be printed.


Document Details
: Specify the [Document  Details]({{site.pp_baseurl}}/misc/document_details_printing_perferences_purchasing_content.html).
