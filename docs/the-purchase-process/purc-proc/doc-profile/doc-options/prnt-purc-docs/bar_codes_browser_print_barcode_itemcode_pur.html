Bar Codes Browser

This browser displays all the item codes attached to the document. You can print the item codes included in the document as barcodes from here.

To print an item code as a barcode, do the following:

  • From the Bar Codes browser, select the item codes that have to be printed as barcodes.
  • Click the button > LabelPrint > Item Code - 11x33.

You can also create a new label using the Label Designer option.

For Matrix Items, you can add the Attribute 1 – 5 fields to your barcode labels using the Field Chooser in the Bar Codes browser. These fields are relevant only to Matrix Items. The Attribute fields that you add to the Bar Codes browser appear in the Label Designer dialog box and they are available to add to your barcode labels.
