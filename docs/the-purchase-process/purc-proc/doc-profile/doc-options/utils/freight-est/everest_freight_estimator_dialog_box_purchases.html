Everest Freight Estimator Dialog Box

The Everest Freight Estimator dialog box allows you to compute freight on the document.

The fields in this dialog box have been grouped under the following sections:

  • Shipping Details
  • Shipping Origin Details
  • Shipment Destination Details
  • Summary of Charges
