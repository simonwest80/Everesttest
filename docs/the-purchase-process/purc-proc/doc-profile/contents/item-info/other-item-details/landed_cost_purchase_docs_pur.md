# Landed Cost


This non-editable column displays the total cost per unit incurred in  bringing each inventoried item into a saleable condition. This includes  the cost of the item, any add-on costs, and any non-recoverable tax, and  omits discounts. The calculation is the same for documents set to FOB  Origin and to FOB Destination.


The system calculates Landed Cost using the following purchase document  columns:


Landed Cost = Cost after Discount + Add-on Costs + (Non-recoverable  Tax / Quantity)
