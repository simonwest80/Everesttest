# Landed Cost Amount


This non-editable column displays the total monetary amount of landed  cost for each line in the purchase document. This is the amount paid for  the entire quantity of each line item including any add-on costs and the  non-recoverable tax. The system calculates the Landed Cost Amount using  the purchase document columns:


Landed Cost Amount = Amount + Add-on Amount + Non-recoverable Tax
