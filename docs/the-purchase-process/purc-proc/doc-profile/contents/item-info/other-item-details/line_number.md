# Line Number


The **Line Number** column displays  the line number of the item selected in the Item Details Grid. All line  items are printed in order of their line numbers.


Use the **Move up**/**Move 
 down** option (path: **Options**  > **Line Items** > **Move 
 up**/**Move down**) to rearrange  the order of the items in the Item Details Grid.
