# Add-on Amount


The Add-on Amount column displays the total of the add-on cost amounts  for all the add-on costs elements. You can drill down in any line item  to see the distribution of different add-on costs for that line item by  double clicking the value in this column. An Add-on Costs Distribution  dialog box opens displaying each Add-on Cost ID with its description,  distribution method, and monetary amount.


This column does not display in purchase and debit quotes.
