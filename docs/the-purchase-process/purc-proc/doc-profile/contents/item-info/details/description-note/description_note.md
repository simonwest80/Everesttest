# Description/Note


The **Description**/**Note**  column displays the description of an item as entered in the **Description**  field (path: **Inventory** > **Items** > **Options 
 &gt; Edit &gt; In New Window** > **General**  tab).


You can edit the item description/notes, by using the **Inline****Notes** option (path: **Purchase 
 Document** profile > **Options**  > **Inline** **Notes**).


{:.example}
You can update the item description to match  it with the vendor's catalog.
