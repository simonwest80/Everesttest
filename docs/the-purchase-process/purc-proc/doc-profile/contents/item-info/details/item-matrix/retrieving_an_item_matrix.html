Retrieve an Item Matrix

Items that belong to a matrix group can be retrieved either by selecting the master item for the matrix group from the Items browser (path: Purchase Document profile > Options > Item > Select Items), or by selecting the Item Matrix option (path: Purchase Document profile > Options > Retrieve > Item Matrix).

To retrieve an item matrix, do the following:

  • From the purchase document, select Options > Retrieve > Item Matrix. You will see the Items browse displaying all the item matrix groups.
  • Select an item matrix and click OK. The Matrix ItemsView dialog box or the Matrix Items browser is displayed depending on whether you select Matrix or Items in the Matrix Group Display Preference Type field (path: Inventory > Items > Options > Edit > In New Window > Item profile - General tab).

You can select any number of matrix items from the browser. The Matrix Items View dialog box is displayed once for every matrix that is selected.

If there is a time lag before the Matrix Items View dialog box is displayed, you can view the progress in the Retrieving Matrix window. Click View List in this window to view the Matrix Items browser
