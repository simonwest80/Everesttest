# Model


You may have two different models of an item manufactured by the same  company.


{:.example}
HP Color LaserJet  5M and HP Color LaserJet  5/ 5M PS Printers


The code of the item model specified in the **Model**  field of the **Item** profile (path:  **Inventory &gt; Items &gt; Options &gt; 
 Edit &gt; In New Window &gt; General** tab) is displayed in the **Model** column. You cannot edit this column.
