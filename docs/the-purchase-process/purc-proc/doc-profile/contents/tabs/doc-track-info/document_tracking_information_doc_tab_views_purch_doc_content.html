Document Tracking Information

The Document Tracking tab allows you to tag the document with the appropriate tracking selections for the groups available.

This tab is displayed only when you save the purchase document.

You cannot modify the selections if the tracking group is in “view only” mode for that document type.

Track a Purchase Document

You can set document tracking preferences to ensure that the tracking groups automatically pop up either when you create, edit or save a document, or only when you create a document (path: File > Setup > Preferences > Flow Control > Document Tracking).
