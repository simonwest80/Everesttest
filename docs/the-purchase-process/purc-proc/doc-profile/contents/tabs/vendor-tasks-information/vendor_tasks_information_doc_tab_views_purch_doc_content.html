Vendor Tasks Information

The Vendor Tasks tab allows you to:

  • View existing tasks, incoming and outgoing calls, and general notes associated to the vendor selected in the purchase document.
  • Associate new tasks, incoming and outgoing calls, and general notes to the vendor selected in the purchase document.

Contact Manager

To associate a new task to the vendor, do the following:

  • From the browser in the Vendor Tasks tab of the purchase document, right-click and select New > Task. You will see the Task profile.
  • Enter and save the required details.

Similarly, you can associate an incoming or outgoing call and general notes to the vendor selected in a purchase document.
