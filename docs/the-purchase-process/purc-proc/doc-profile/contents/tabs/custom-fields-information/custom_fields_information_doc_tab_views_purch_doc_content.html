Custom Fields Information

Custom fields defined on the Custom Fields tab of the purchase document apply to the overall purchase document. They do not apply to individual line items. Line item custom fields are defined separately. Use the Purchase Document Lines dialog box – Custom Fields tab (path: purchase/purchase return document profile > Options > Line Items > Line Profile or press Ctrl + Right Arrow in any document) to create line item custom fields for purchase documents.

If purchase documents are processed by different users, create a Processed By character custom field to track this information.
