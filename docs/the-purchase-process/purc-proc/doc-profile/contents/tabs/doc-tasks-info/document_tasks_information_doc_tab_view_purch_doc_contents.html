Document Tasks Information

The Document Tasks tab allows you to:

  • View existing tasks, incoming and outgoing calls, and general notes that are associated to a purchase document.
  • Associate new tasks, incoming and outgoing calls, and general notes to a purchase document.

This tab is displayed only when you save the purchase document.

Contact Manager

To associate a new task to a purchase document, do the following:

  • From the browser in the Document Tasks tab of the purchase document, right-click and select New > Task. You will see the Task profile.
  • Enter and save the required details.

Similarly, you can associate an incoming or outgoing call and general notes to a purchase document.
