# Status Bar Information - Purchase Documents


The Status Bar is present at the bottom of all document profiles in  **Everest**. The options available  in the status bar of purchase document profiles are:

- [Preview  On/Off]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/sbar-info/preview_off_on_status_bar_information_contents_of_a_purchase_document.html)
- [Show/Hide]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/sbar-info/hide_show_all_status_bar_information_common_purchase_profile_information.html)
- [Zoom  Out/In]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/sbar-info/zoom_in_out_status_bar_info_common_purdchase_document_profile_information.html)
- [Batched  Payments Pending]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/sbar-info/batched_payments_pending_purchase_doc_status_bar.html)
- [Document  Policy]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/sbar-info/document_policy_status_bar_information_common_purchase_documnet_profile_contents.html)
- [Unposted/Posted]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/sbar-info/posted_unposted_status_bar_info_common_purchase_profile_info.html)
- [Item  Notes]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/sbar-info/item_notes_status_bar_options_purchase_doc_profile.html)



{:.note}
The **Stock 
 Area** field displays in the status bar when you need to select a  stock area. Select a stock area and press the **Enter**  key or **ESC** key to exit this field  and continue.
