# Batched  Payments Pending


If a batch payment is pending for a document, you will see the ![]({{site.pp_baseurl}}/img/pur_batched_payment_pending.gif) icon displayed in the document status bar. Click the icon  to view the **Batch 
 Payment Process** browser.


**![]({{site.pp_baseurl}}/img/note.gif) Â The  indicator is displayed on purchase order and invoices, and debit orders  and memos.**


**You can process the payment  from this browser.**


**Convert an Order to a Final Document**


If you convert an order that has a pending batch payment to an invoice/memo,  the final document created also displays this icon.
