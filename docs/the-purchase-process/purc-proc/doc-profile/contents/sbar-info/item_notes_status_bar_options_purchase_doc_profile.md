# Item Notes


This option brings up the item notes for the selected line item, as  specified in the **Item** profile  (path: **Inventory** > **Items**  > **Options** > **Edit** > **In New Window** >  **Notes** Tab).


| Status Displayed | Notes Available |
| ![]({{site.pp_baseurl}}/img/pur_upsell_notes.gif) | Upsell information, and notes and specifications |
| ![]({{site.pp_baseurl}}/img/pur_upsell.gif) | Only upsell information |
| ![]({{site.pp_baseurl}}/img/pur_notes.gif) | Only notes and specifications |
