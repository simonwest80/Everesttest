# The Recurrence Pattern Dialog Box (Purchase Order)


Use the **Recurrence** **Pattern**  dialog box to define the schedule for recurring documents.


Document aliases must be the same for all documents that are part of  a recurrence pattern. You can set a document alias in the initial document  used to create a recurrence pattern, but the **Document 
 Alias** button ![]({{site.pp_baseurl}}/img/document_alias_button_disabled_recurrence_docs_pur.gif) is disabled in the recurring documents  created since it cannot be changed.


The information in this dialog box is classified under the following  sections:

- [Schedule  Information]({{site.pp_baseurl}}/misc/schedule_information_recurrence_pattern_dialog_box_purchase_order_content.html)
- [Credit  Card Information]({{site.pp_baseurl}}/misc/credit_card_payment_recurrence_pattern_dilaog_box_purchase_orders.html)
- [Other  Information]({{site.pp_baseurl}}/misc/other_information_recurrence_pattern_dialog_box_purchase_orders.html)
