Back End Processes - Create a Purchase Quote

When a purchase quote is created, the following back end processes occur:

  • The next sequence number for purchase quotes for the logged in location or location of the logged in location/sub-location is assigned. The document number assigned is specified in the Location profile (path: File > Setup > Accounting > Location/Sub-Location> Options > Edit > In New Window > Purchasing tab).

  • If a sequence number has not been assigned here the sequence number assigned to the Default Location field (path: File > Setup > Preferences > Defaults > Accounting > Miscellaneous tab) is used.
