Document Information

The following information are displayed about the document:

Document #
This field displays the number assigned to the purchase receipt and is auto-generated. The number is displayed for a new document, only after it is saved.

Document Number

Document Alias
The default document alias specified for purchase receipts is displayed in this field. You can select a different document alias, if required.

Document Alias

Document Notes
Click the icon beside the Document # field to specify document notes.

Document Notes

Document Date
This field , beside the Document# field, displays the login date. You cannot edit this field.

Document Dates

Document Policy
Click the icon to view the document policy. You can change/edit the document policy.

Document Policy

Document Status
This section displays the status of the document.

Document Status
