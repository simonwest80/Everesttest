Entry for Non-inventoried Items

The accounting entry created when a debit memo for inventoried items is raised is:

Account Path Debit/


Credit		 Amount
Accounts Payable Account The account specified in the AccountsPayable field of the Accounting dialog box (path: File > Setup > Preferences > Defaults > Accounting > Posting Group tab).


Accounting dialog box - Posting Group tab		 Debit Total value of the returned items less the discount.
Tax Component Account (asset account
defined for Value Added Tax recoverable in tax component in purchase tax
code)		 The account selected for TaxAccount Code in the Tax Components profile (path: File > Setup > Accounting > Taxes > TaxComponents > Options > Edit > InNew Window > Purchase tab). Credit The value of tax paid on the items.
Asset/Expense Account The account defined for assets/expenses in the Posting Group profile (path: File
> Setup > Accounting > Posting Group profile).


Items - Accounts		 Credit The cost of the items - discount.
