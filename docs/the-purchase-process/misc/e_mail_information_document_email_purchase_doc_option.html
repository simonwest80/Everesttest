E-Mail Information

To
Enter the recipient’s e-mail ID. This field displays the e-mail ID specified for the vendor’s default contact address in the Vendor profile (path: Purchasing > Vendors > Options > Edit > In New Window > Addresses tab). If a default contact address is not specified, this field defaults to the e-mail ID specified in the billing address.
Click the button and you will see the Address Book dialog box. You can specify additional addresses from this dialog box.
CC
Enter the email ID of a recipient who should receive a copy of this mail. This field displays the default e-mail ID entered in the vendor’s default billing address (path: Purchasing > Vendors > Options > Edit > In New Window > General tab).
If you specify the e-mail ID as per the vendor’s billing address in the To field, this field remains blank.
Click the button and you will see the Address Book dialog box. You can specify additional addresses from this dialog box.
Subject
Enter an appropriate subject for the mail. This field, by default, displays the document alias defined for the document(s) in the document profile. The document number(s) is/are also displayed in this field.
