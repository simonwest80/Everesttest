# Document Information


**Document #**
: This field displays the number assigned automatically  to the purchase order. The number is displayed, for a new document, only  after it is saved.


**![]({{site.pp_baseurl}}/img/lens.gif) [Document  Number]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/document-information/document_number.html)**


**Document Alias**
: The default document alias specified for purchase  orders is displayed in this field. You can change the document alias.


**![]({{site.pp_baseurl}}/img/lens.gif) [Document  Alias]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/document-information/document_alias.html)**


**Document Notes**
: Click the ![]({{site.pp_baseurl}}/img/pur_document_notes.gif) icon beside the **Document #** field to specify document  notes.


**![]({{site.pp_baseurl}}/img/lens.gif) [Document  Notes]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/item-info/details/description-note/description_note.html)**


**Document Date**
: This field ![]({{site.pp_baseurl}}/img/pur_document_date.gif) beside the **Document****#** field displays the login date.  You cannot edit this field.


**![]({{site.pp_baseurl}}/img/lens.gif) [Document  Dates]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/document-information/document_date.html)**


**Document Status**
: This section displays the status of the document.


**![]({{site.pp_baseurl}}/img/lens.gif) [Document  Status]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/document-information/document_status.html)**
