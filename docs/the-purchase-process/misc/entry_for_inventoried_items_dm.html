The accounting entry created when a debit memo for inventoried items is raised is:

Credit		 Amount

Gain/Loss on Return The account specified in the Gain/Loss
on Return fields (path: File > Setup > Preferences > Defaults > Accounting > Posting Group tab).		 Credit/ Debit The profile/loss on the return
Inventory Control Account The account specified in the InventoryControl field (path: File > Setup > Preferences > Defaults > Accounting > Posting Group tab). Credit The average cost of the item in case of inventoried items and the actual cost  in case of serialized items.
