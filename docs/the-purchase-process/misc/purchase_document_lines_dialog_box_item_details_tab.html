The Purchase Document Lines Dialog Box - Item Details Tab

Line No
This field displays the positioning of the line item in the item detail grid.
Hide
This field displays if the line item is hidden.
Hide
Service
This check box is selected if the line item is a “Service” type. These are items sent for service or repair.
Item Code
This field displays the code of the selected item. The description of the item is displayed beside this field.
Item Code
Location/Sub-Location
This field indicates the location/sub-location to which expenses for that line item are to be allocated.
Loc/Sub-Loc
UM
This field displays the unit of purchase measure for the item.
Unit of Measure
Ordered
This field displays the number of items ordered. In case of purchase return documents, it indicates the quantity that has to be returned.
Quantity
Received on Invoice
This field displays the number of items received on a purchase invoice. In case of purchase return documents, it indicates the quantity that is returned. The amount, discount value, and tax amount fields refer to the Received on Invoice quantity.
Invoiced
This field displays the number of items invoiced (Invoiced Quantity). The amount, discount value, and tax amount fields refer to the Invoiced Quantity.
Committed
Committed quantities on purchase orders can represent either:
  • Items that you physically receive but do not take into stock until approval, or
  • Items in transit.
: The committed items on a purchase order are assigned to an “Out” type of area. The stocks committed are not available for sale or use until you receive them on a purchase receipt or invoice.
For debit orders, this displays the quantity of the inventoried items requested that you have committed for shipment. The committed quantity could represent the following:
  • Items that you physically send but do not take out of stock until approval
  • Items in transit
  • Items sent for exchange or repair

: The committed items on a debit order are assigned to the “Stock” type of stock area. They continue to remain in your ownership.

Cost
This field displays the cost of the item, in base currency, for one unit of purchase measure.
 Item Cost
Discount Value
This field displays the discount available on the item cost.
Discount %
This field displays the discount percentage available on the item cost.

Discount %

Amount
This field displays the net purchase amount (in base currency) for the item purchased.
Purchase Documents
Amount = (Item Quantity x Item Cost (per unit) - Item Quantity x Discount (per unit)) + Amount calculated in the Add-on Costs column.
Purchase Return Documents
Amount = (Item Quantity x Item Cost (per unit) - Item Quantity x Discount (per unit))
Tax Code
This field displays the tax code assigned to the item.
Tax Code
Tax Amount
This field displays the total amount of tax applied to the item.
Add-on Amount
This field displays the additional costs associated with the line item, such as freight. The Add-on amount is required to arrive at an accurate purchase cost.
Job Code
This field displays the job code to which the item is associated
Job Code
Delivery Date
On a purchase quote or purchase order, this field displays the expected delivery date of the line item from the vendor. On a purchase receipt, this field identifies the date that you actually received the line item. For debit quotes and debit orders, it is the date that you expect to return the line item to the vendor. For debit memos, it is the date that you actually returned the line item.
Stock Bin
This field displays the stock bin where the inventoried line item is currently located.
Weight
This field displays the weight of the selected line item
Weight
