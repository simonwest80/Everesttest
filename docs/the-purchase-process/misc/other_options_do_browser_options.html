Shipped/Open Debit Orders
Select this toggle option to view either shipped (closed) debit orders or open debit orders.
Search By Vendor
Select this ** option ** to view a list of all orders created for the vendors of the selected documents.

If you highlight documents with “ABC” and “EFG” as vendors; and select Search by Vendor > Debit Orders, the browser displays all debit orders created for these vendors.
