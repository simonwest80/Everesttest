# Purchase Document Browser (System)


This browser displays all system documents that have been created for  the selected vendor. By default, this browser is grouped by the document  type.


You can create documents of all the types that are listed in the browser.  The document created depends on the record you highlight before selecting  **Options** > **New** > **In New Window**.
