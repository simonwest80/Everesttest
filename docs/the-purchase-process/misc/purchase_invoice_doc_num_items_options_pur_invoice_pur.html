Purchase Invoice Doc # window - Price Summary

The Purchase Invoice Doc # window displays the Line Items for which there has been any change in the values of the fields mentioned below:

  • Item Cost
  • Discount Value
  • Discount %
  • Tax Code
  • Job Code
  • Add-on Cost

This window displays the summary of the original values that were saved after creating the purchase invoice as well as the values that were saved after the last modification. This option will be enabled only when the purchase invoice is changed for the fields mentioned above.

The summary includes a toggle, Invoiced summary/Received summary, so you can switch the price summary details from the invoiced quantity to the quantity received on the invoice.

Display of values will be as per the Decimal Accuracy defined and will be based on the Purchase UOM as defined in the Line Item of the purchase invoice and not the default UOM.
