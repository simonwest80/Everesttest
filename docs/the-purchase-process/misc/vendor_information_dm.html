Vendor Information

The following information are displayed for the vendor:

Vendor
Specify the vendor code to whom the items are being returned. The vendor’s billing address is automatically populated from the Vendor profile. You can select a different address.

Vendor Code

Contact
Specify the address of the person who is to be contacted for information on the debit memo. This field automatically displays the default Contact address specified in the Vendor profile. You can select a different address.

Vendor’s Contact Address

**Ship To**
Specify the address to which the returns must be shipped. This field automatically displays the default Return address specified in the Vendor profile. You can select a different address.

Vendor’s Shipping Address

When the shipping address is not defined for a purchase or purchase return document, the system prints the document’s location address in place of the shipping address.
