# The Purchase Document Lines Dialog Box


The **Purchase** **Document****Lines** dialog box displays a brief  description of the item highlighted in the item details grid. This dialog  box is displayed when you select the **Line 
 Profile** option (path: Purchase/Purchase Return Document profile  > **Options** > **Line 
 Items** > **Line Profile**).  You cannot edit the data in this dialog box. Click the ![]({{site.pp_baseurl}}/img/pur_previous.gif) button  to view details of the previous line item on the purchase/purchase return  document and the ![]({{site.pp_baseurl}}/img/pur_next.gif) button to view details of the next line  item.


The **Purchase Document Lines**  dialog box contains the following tabs:

- [Item  Details]({{site.pp_baseurl}}/misc/purchase_document_lines_dialog_box_item_details_tab.html)
- [Custom  Fields]({{site.pp_baseurl}}/misc/the_purchase_document_lines_dialog_box_custom_fields_tab.html)
