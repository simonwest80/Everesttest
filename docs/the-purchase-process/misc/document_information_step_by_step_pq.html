Document Information

The following information are displayed about the document:

Document #
The document number is automatically assigned based on the sequence number specified for purchase quotes in the logged in location or the location of the logged in location/sub-location.
If sequence numbers are not assigned to the location, the sequence numbers specified for the Default Location (path: File > Setup > Preferences > Defaults > Accounting > Miscellaneous tab) are used.
The document number is displayed for a new purchase quote only after the document details are saved.

Document Number

Document Alias
The default document alias specified for purchase quotes is displayed in this field. You can specify another alias, if required.

Document Alias

Document Notes
Click the icon beside the Document # field to specify document notes.

Document Notes

Document Date
This field beside the Document# field displays the login date. This is a display field and cannot be edited.

Document Dates
