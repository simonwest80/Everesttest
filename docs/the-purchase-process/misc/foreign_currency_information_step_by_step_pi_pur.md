# Foreign Currency Information


**Currency**
: This field displays the currency defined for the  vendor in the **Vendor's** profile.  This is a display field and cannot be edited.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Currency]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/tabs/details/for-cur-info/currency.html)


**Exchange Rate**
: This field will be enabled only for foreign currency  vendors. By default it displays the latest exchange rate. You cannot modify  this rate on a saved purchase invoice.
