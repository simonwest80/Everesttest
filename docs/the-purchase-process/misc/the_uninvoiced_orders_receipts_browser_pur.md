# The Uninvoiced Orders/Receipts Browser


The Uninvoiced Orders/Receipts browser contains much of the information  available in purchase invoices. It provides the following standard browser  options:

- Field Chooser
- Properties
- Export
- Close



Line  items are grouped by their invoiced status.
