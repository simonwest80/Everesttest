Add an Item - Smart Search Option

This option allows you to search for an item and add it to the purchase return document.

You can search for an item based on the item code, model, item alias, serial # and manufacturer’s serial #.

A search dialog box is displayed as the user enters a character in the Item Code column in the item details grid of the purchase return document. This dialog box displays the different types of matches for the item code you have typed in the Search for field. The search sequence is based on the method specified in flow control preferences for items in the Items dialog box (path: File > Setup > Preferences > Flow Control > Items).

If you have selected serial #, model and item alias as the search sequence in the Items dialog box, the search will display matching codes, if any, in the same order. It will first list all serial numbers with the matching code, then all the models and finally all the item aliases.

To search for an item and add it to the purchase return document, do the following:

  • Type in any character in the Item Code column in the item details grid of the purchase return document. You will see the Smart Search dialog box.
  • Select the required item and click OK. The item is added to the purchase return document.
