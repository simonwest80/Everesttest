# Other Options


**Search By Vendor**
: Select the **Search****by** **Vendor**  to view the list of all purchase/purchase return and manual documents  created for the vendors of the selected documents.


{:.example}
If vendors of the highlighted documents are  "ABC" and "EFG";  and you select debit memos, then the browser will come up with all debit  memos created for these two vendors.


**Shipment Tracking Information**
: Select this option (path: **Debit****Memo** browser > **Options**  > **Shipment** **Tracking****Information**) to view the [**Shipment** **Tracking****Information** browser]({{site.pp_baseurl}}/misc/shipment_tracking_information_browser_debit_memo_browser_options.html).


{:.note}
This option can also be accessed from the **Debit****Memo** profile (path: **Debit****Memo** profile > **Options**  > **Shipment** **Tracking****Information**).
