# Add-on Costs Amount


Enter the total amount of add-on costs to be distributed on the items  in the vendor currency. You can modify the Add-on Cost for a saved purchase  invoice in **Add-on Cost Distribution**  dialog box. Once it is modified, the system will automatically update  the value of Add-on cost for each line item in the purchase invoice as  applicable.
