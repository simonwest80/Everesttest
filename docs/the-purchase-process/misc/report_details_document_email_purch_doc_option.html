Report Details

Report Type
Specify the type of document report to be sent by e-mail. The options are listed below
Option Select this option to:
Purchase/Purchase return Document (quote, order, receipt, or invoice) Generate the report for the purchase/purchase return document in A4 size paper.
Purchase/Purchase return Document (with Model #) Include the item’s model number along with the purchase/purchase return document details.
Purchase/Purchase return  Document Double Line Display model and brand information in addition  to the item code and description. The first line displays the item code, model and brand. The second line displays the item description.
Vendor Document (Dot Matrix) Generate the report on a Dot Matrix Printer.

: This field, by default, displays the report type specified for purchase/purchase return documents in the Documents dialog box (path: File > Setup > Preferences > Defaults > Documents > Internal Reports > Documents tab).

Company Information
Specify the format in which the address and logo must be displayed on the document. The options are:
Option Select this option to:
Logo Bitmap Display both the company’s logo and address on the document that has to be sent by e-mail.
Text Fields Display the company’s name and address as text.
None Hide the company’s name and address are not printed.
: The address and logo of the document’s location or location of the document’s location/sub-location as specified in the Location profile (path: **File
> Setup > Accounting > Location/ Sub-Location > Options** > Edit > **In
New Window** > Image tab) is used. If this is not assigned, the address/logo of the **Default
Location** (path: **File > Setup
> Preferences > Defaults > Accounting > Miscellaneous** tab) is used.
By default, this field displays the preferences set in the Reporting Preferences dialog box (path: File > Setup > Preferences > Defaults > Reporting Preferences).
Billing Address
Specify how the billing address must appear on the document. You can select one of the following options.
Option Name Select this option
to:
Full (with Telephone/ Fax number) Display the vendor’s billing address with telephone and fax numbers.
Mail/Envelope Purposes Display the vendor’s billing address without telephone number.

: By default, this field displays the preferences set in the Reporting Preferences dialog box (path: File > Setup > Preferences > Defaults > Reporting Preferences).

Driver and Copies
These fields are not relevant while e-mailing documents.
