# Billing Address


The **Billing** **Address**  field allows you to select the type of billing address that must be printed.  The options are:


| **Option** | **Description** |
| **Full (with Telephone/Fax number)** | Select this option to print the vendor's billing address with telephone  and fax numbers. |
| **Mail envelope purposes** | Select this option to print the vendor's billing address without telephone  numbers. |
