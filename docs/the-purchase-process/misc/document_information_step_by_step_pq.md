# Document Information


The following information are displayed about the document:


**Document #**
: The document number is automatically assigned based  on the sequence number specified for purchase quotes in the logged in  location or the location of the logged in location/sub-location.
: If sequence numbers are not assigned to the location,  the sequence numbers specified for the **Default 
 Location** (path: **File** >  **Setup** > **Preferences**  > **Defaults** > **Accounting**  > **Miscellaneous** tab) are used.
: The document number is displayed for a new purchase  quote only after the document details are saved.


**![]({{site.pp_baseurl}}/img/lens.gif) [Document  Number]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/document-information/document_number.html)**


**Document Alias**
: The default document alias specified for purchase  quotes is displayed in this field. You can specify another alias, if required.


**![]({{site.pp_baseurl}}/img/lens.gif) [Document  Alias]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/document-information/document_alias.html)**


**Document Notes**
: Click the ![]({{site.pp_baseurl}}/img/pur_document_notes.gif) icon beside the **Document #** field to specify document  notes.


**![]({{site.pp_baseurl}}/img/lens.gif) [Document  Notes]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/item-info/details/description-note/description_note.html)**


**Document Date**
: This field ![]({{site.pp_baseurl}}/img/pur_document_date.gif) beside the **Document****#** field displays the login date.  This is a display field and cannot be edited.


**![]({{site.pp_baseurl}}/img/lens.gif) [Document  Dates]({{site.pp_baseurl}}/purc-proc/doc-profile/contents/document-information/document_date.html)**
