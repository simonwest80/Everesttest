# Amount


Specify the amount of each add-on cost elements. The system uses the  currency of the vendor selling the items. Enter numeric characters up  to 12 digits (including decimal places). The amount you enter automatically  distributes across the inventoried items shown in the bottom grid based  on the chosen distribution method.


If you edit this amount, the system automatically recalculates the add-on  costs distributed to the line items.


On a purchase invoice, the Amount displays for both the Received on  Invoice and Invoiced columns Depending on the whether the document is  displayed in base currency or the document currency, the amount in this  column is also displayed in the same currency Â .
