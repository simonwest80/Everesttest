# Serial No. Transfer for All Items browser


This browser displays details of the serialized items attached to the  document like serial number, item code, manufacturer's serial number,  stock type and stock area code. You can enter or edit the manufacturer's  serial number by clicking the relevant column.


This browser has the following right-click options:


**Assign Manufacturer's Serial No.**
: Select this option to assign a range of similar  manufacturer's serial numbers to the items displayed in the browser. This  option brings up the **Assign Manufacturer's 
 Serial No.** dialog box. This dialog box allows you to enter a prefix,  suffix and starting number for a range of manufacturer's serial numbers.  Specify the number of items to which you want to assign serial numbers  in the **Quantity** field.


**Custom Fields**
: Select this option to specify custom information  for the highlighted serial number. This option brings up the **Custom 
 Fields** browser.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Custom  Fields]({{site.sc_chm}}/options/miscellaneous-set-up/custom-fields/creating_user_defined_custom_fields.html)


**Serial History Browser**
: Select this option to view the history of the highlighted  serial number. This option brings up the **Serial 
 Number History** browser.
: ![]({{site.pp_baseurl}}/img/lens.gif)<font style="color: #ff0000;" color="#FF0000"> </font>[Serial  Number History browser]({{site.pp_baseurl}}/misc/serial_number_history_browser_pur.html)
