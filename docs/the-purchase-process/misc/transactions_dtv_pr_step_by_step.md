# Transactions


The **Transactions** tab displays  the accounting entry that was created in the system when the purchase  receipt was saved. The tab displays details such as the accounts that  have been debited or credited, amount of debit/credit, the purchase journal  and transaction number in which the entry was recorded, the document number  and date, and the source of the transaction.


{:.note}
This tab is displayed only on a saved purchase  receipt.
