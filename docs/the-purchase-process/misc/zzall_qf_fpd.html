Specify the type of purchase and debit orders that must be included in the search results.

This option is enabled only when you select Purchase Orders or Debit Orders or both in the Document Type field.

The following options are available:

Options Search includes:
All All purchase and debit orders
Back Order Only back orders.
Open & Back Order Both open and back orders. Closed orders are excluded in this search.
Open Only open orders.
Closed Only closed orders.
