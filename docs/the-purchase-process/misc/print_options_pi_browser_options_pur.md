# Print Options


**Print Serial Numbers**
: Select the **Print****Serial** **Numbers**  option to print serial numbers attached to a receipt.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Print  Serial Numbers]({{site.pp_baseurl}}/misc/print_serial_numbers_pur_browser_options_pur_con.html)


**Print Put Away Report**
: Select this option to print a report to help you  stock received items.
: ![]({{site.pp_baseurl}}/img/lens.gif)[Print  Put Away Report]({{site.pp_baseurl}}/misc/print_put_away_report_pr_pi_browser_option_pur.html)


**Print Document**
: Select the **Print****Document** option to print the selected  document.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Printing  Purchase Documents]({{site.pp_baseurl}}/purc-proc/doc-profile/doc-options/prnt-purc-docs/printing_purchase_documents.html)
