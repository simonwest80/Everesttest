The Purchase Document Lines Dialog Box

The Purchase DocumentLines dialog box displays a brief description of the item highlighted in the item details grid. This dialog box is displayed when you select the Line Profile option (path: Purchase/Purchase Return Document profile > Options > Line Items > Line Profile). You cannot edit the data in this dialog box. Click the button to view details of the previous line item on the purchase/purchase return document and the button to view details of the next line item.

The Purchase Document Lines dialog box contains the following tabs:

  • Item Details
  • Custom Fields
