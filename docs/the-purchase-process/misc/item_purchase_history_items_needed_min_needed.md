# Item Purchase History


Use the **Item Purchase History**  option to view the purchase documents that contain the selected item.  This option brings up the **Item Purchase 
 History** browser, which is grouped based on the vendors from whom  the items have been purchased.


This data enables the purchase department to negotiate a better purchase  price.
