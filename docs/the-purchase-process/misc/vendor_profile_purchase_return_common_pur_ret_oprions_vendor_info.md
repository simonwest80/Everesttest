# Vendor Profile


The **Profile** option (path: Purchase  Return Document profile > **Options**  > **Vendor** > **Profile**)  allows you to access the profile of the vendor. You can make any modifications  required to the vendor profile.


![]({{site.pp_baseurl}}/img/lens.gif) [Vendor  Profile]({{site.mv_chm}}/creating/the_vendor_profile_steps_by_steps.html)


{:.example}
The **Vendor**  profile can be used to view the maximum credit that can be availed form  the vendor.
