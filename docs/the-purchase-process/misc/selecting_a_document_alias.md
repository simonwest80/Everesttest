# Select a Document Alias


![]({{site.pp_baseurl}}/img/steps.gif)To change the alias for a document, do the  following:

- Click the **Document Alias** button ![]({{site.pp_baseurl}}/img/pur_document_alias.gif) displayed beside the document name. You will see the **Document Aliases** browser.
- From the **Document** **Aliases**  browser, double-click a suitable alias. The document is renamed.
