# Print Serial Numbers


![]({{site.pp_baseurl}}/img/steps.gif)To print serial numbers attached to a purchase  document, do the following:

- From the Purchase  Document browser, highlight the required record.
- Select **Options** > **Print****Serial** **Numbers**.  You will see the **Serial** **Numbers** browser.
- From the **Serial** **Numbers**  browser, select the required serial numbers and click the ![]({{site.pp_baseurl}}/img/pur_print.gif) icon. You will see the [**Print** **Setup**  - **Serial** **Numbers**  dialog box]({{site.pp_baseurl}}/misc/print_setup_serial_numbers_dialog_box_pur_browser_option_pur_con.html).
- Click **Print**.
