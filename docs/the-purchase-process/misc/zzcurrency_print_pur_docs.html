Specify the currency in which to print the document. It has the following options:

  • Print in Base Currency - Prints the document in the base currency, irrespective of the currency defined for the vendor.
  • Print in Document Currency - Prints the document in the currency defined for the selected vendor.
  • Print in both Base and Document Currency - Prints the document in the currency defined for the selected vendor. However, the document and the tax totals are displayed additionally in the base currency at the bottom right hand side of the document.
