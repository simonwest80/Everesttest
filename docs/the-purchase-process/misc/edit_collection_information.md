# Edit Collection Information


Collections from vendors are accounted in a [payment  journal]({{site.pp_baseurl}}/misc/payment_journal_for_collection_on_debit_order.html). You can edit vendor collections in any of the following ways:

- Voiding the journal  from the payment journal.
- Voiding the transaction  from within the payment journal.
- Voiding, or voiding  and recreating the transaction from the payment register.
- Voiding the payment  from the **Payment 
 History** browser.
