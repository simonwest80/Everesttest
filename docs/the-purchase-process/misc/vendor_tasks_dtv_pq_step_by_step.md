# Vendor Tasks


The **Vendor Tasks** tab allows  you to execute the following functions:

- View existing  tasks, incoming and outgoing calls, and general notes associated to the  vendor selected in a purchase quote.
- Associate new  tasks, incoming and outgoing calls, and general notes to the vendor selected  in a purchase quote.



{:.note}
This tab is displayed only when you select a vendor.
