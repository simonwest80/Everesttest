# The Tax Breakdown Window


The **Tax Breakdown** window **** displays the following buttons:


**![]({{site.pp_baseurl}}/img/pur_itm_tax_brkup.gif) Group by Item**


Select this button to view a detailed breakup of the taxes charged on  each item. All the tax components that are applicable are displayed separately  for each item.


**![]({{site.pp_baseurl}}/img/pur_tax_brkup.gif) Group by Tax**


Select this button to view a detailed breakup of the taxes charged on  items grouped by the tax components.


The amounts in this window are displayed in the currency in which the  document is viewed.
