# Document Information


The following information are displayed about the document:


**Document #**
: The document number is automatically assigned based  on the sequence number specified for debit memos in the logged in location  or sub-location.
: If sequence numbers are not assigned to the location,  the sequence numbers specified for the **Default 
 Location** (path: **File** >  **Setup** > **Preferences**  > **Defaults** > **Accounting**  > **Miscellaneous** tab) is used.
: The document number is displayed for a new debit  memo after the document details are saved.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Document  Number]({{site.pp_baseurl}}/return-proc/doc-prof/contents/document-information/document_number_pr.html)


**Document Alias**
: The default document alias specified for the debit  memos is displayed in this field. You can change this alias, if required.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Document  Alias]({{site.pp_baseurl}}/return-proc/doc-prof/contents/document-information/document_alias_pr.html)


**Document Notes**
: Click the ![]({{site.pp_baseurl}}/img/pur_document_notes.gif) icon beside the **Document #** field to specify document  notes.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Document  Notes]({{site.pp_baseurl}}/return-proc/doc-prof/contents/document-information/document_notes_pr.html)


**Document Date**
: This field ![]({{site.pp_baseurl}}/img/pur_document_date.gif) beside the **Document****#** field, displays the login date.  You cannot edit this field.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Document  Dates]({{site.pp_baseurl}}/return-proc/doc-prof/contents/document-information/document_date_pr.html)


**Document Policy**
: Click the ![]({{site.pp_baseurl}}/img/pur_document_policy.gif) icon on the status bar  to view the document policy. You can modify/edit this policy, if required.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Document  Policy]({{site.pp_baseurl}}/return-proc/doc-prof/contents/document-information/document_policy_pr.html)


**Document Status**
: This section displays the status information for  the document.
: ![]({{site.pp_baseurl}}/img/lens.gif) [Document  Status]({{site.pp_baseurl}}/return-proc/doc-prof/contents/document-information/document_status_pr.html)
