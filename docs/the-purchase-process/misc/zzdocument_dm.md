---
title: Document Number
---


The number assigned to the document is displayed in this field. The  document number is automatically assigned based on the sequence number  specified for debit memos in the login location or the parent location  of the login sub-location.


If sequence numbers have not been assigned to the location the sequence  numbers specified for the default location is used.


The document number is displayed for a new debit memo after the document  details are saved.
