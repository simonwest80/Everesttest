Other Options

Search By Vendor
Select the Searchby Vendor to view the list of all purchase/purchase return and manual documents created for the vendors of the selected documents.

If vendors of the highlighted documents are “ABC” and “EFG”; and you select debit memos, then the browser will come up with all debit memos created for these two vendors.

Shipment Tracking Information
Select this option (path: DebitMemo browser > Options > Shipment TrackingInformation) to view the Shipment TrackingInformation browser.

This option can also be accessed from the DebitMemo profile (path: DebitMemo profile > Options > Shipment TrackingInformation).
