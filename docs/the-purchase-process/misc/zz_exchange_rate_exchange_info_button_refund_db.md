---
title: Exchange Rate
---


The **Exchange** **Rate**  field displays the current rate of exchange defined in the **Exchange 
 Rate** profile (path: **File**  > **Setup** > **Accounting**  > **Multic****urrency** > **Exchange 
 Rates**), between document currency and the base currency.


You can change this rate if required. Correspondingly the **Amount**  fields of the Payment currency, Bank currency and Base currency sections  also change.
