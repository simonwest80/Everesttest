# Lot Number Details


Items that are mass-produced are manufactured in lots or batches. You  can assign unique lot numbers or batch numbers and affix them on all received  quantities of mass manufactured items.


If items are tracked for lots, the **Lot 
 # Details** section is displayed.


**Lot Number**
: Click the drop-down ![]({{site.pp_baseurl}}/img/pur_drop_down_button.gif) icon on the **Lot Number** field. Right-click and select  **New**. You will see the [**Lot** profile]({{site.wm_chm}}/lot-number-tracking/assigning-a-lot-number/the_lot_number_profile_wm.html).


**Lot Expiration**
: The expiry date that you specify for the lot in  the **Lot** profile is displayed here.


**Quantity**
: Select a stock in the Stock Bin column and enter  the quantity received (in that stock bin).


Set up Lot Tracking


You can define that items be tracked for lots in the **Item**  profile (path: **Inventory** >  **Items** > **Options 
 &gt; Edit &gt; In New Window** > **Purchases**  tab > **Use** **Lot****Tracking** option).


![]({{site.pp_baseurl}}/img/lens.gif) [Set  up Lot Tracking for Items]({{site.pp_baseurl}}/purc-proc/prs/pr-processes/qty-rcvd/lot-trk/enable_lot_tracking_for_items_purchase_receipt_add_quantities_purchase_contents.html)


{:.note}
If you select this option, you will not be  able to save the purchase receipt unless you specify lot numbers for the  quantity received.


Auto generated Lot Numbers


You can also set a preference for automatically generating lot numbers  for items in the **Item** profile  (path: **Inventory &gt; Items &gt; Options 
 &gt; Edit &gt; In New Window &gt; Purchases** tab > **Auto 
 generate lot numbers** option).


{:.note}
The **Auto 
 generate lot numbers** option is enabled only if you select the **Use lot tracking option** in the **Item** profile.
