Document Information

The following information are displayed about the document:

Document #
This field displays the number assigned to the purchase invoice and is auto-generated. The number is displayed for a new document, only after it is saved.

Document Number

Document Alias
The default document alias specified for purchase invoice is displayed in this field. You can select a different document alias, if required.

Document Alias

Document Notes
Click the icon beside the Document # field to specify document notes.

Document Notes

Document Date
Normally, the Document Date field is view only and you cannot edit this field. You can, however, edit the Document Date field before saving when creating a new Purchase Invoice.
This field beside the Document # field, displays the login date. You cannot edit this field unless you are creating a new Purchase Invoice.
With the Purchase Invoice Date: Modify security right, you can edit the new Purchase Invoice date before saving. The Document Date cannot be less than any Purchase Orders or Purchase Receipts linked to the Purchase Invoice and it must fall into a valid (Open) accounting period.

Document Dates

Document Policy
Click the icon to view the document policy. You can change/edit the document policy.

Document Policy

Document Status
This section displays the status of the document.

Document Status
