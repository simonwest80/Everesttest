# Serial Number History


Select this option to view the serial number details of serial numbers  attached to a purchase receipt. The **Serial****Number** browser displays the serial  numbers grouped by Item Code, the manufacturer's serial number, the vendor,  the document number, the cost, the lot number, the area code and the stock  area type.
