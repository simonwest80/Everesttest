# Debit Orders on Back Order


Debit orders with a "back order" status display a detailed  break-up of the document value for shipped items, and also those on back  order.


**Order Total**
: The total value of all the items on the Debit Order  profile is displayed in the **Order Total**  field. This field cannot be edited.


**Debited**
: The total value of items that<font style="color: #ff0000;" color="#FF0000"> 
 </font>have been returned is displayed in the **Debited**  field. This field cannot be edited.


Back Ordered


**Taxable**
: The taxable value of items on back order is displayed  in the **Taxable** field. This field  cannot be edited.


**Tax**
: The amount of tax on the taxable value of items  is displayed in this field. This field cannot be edited.


**Exempt**
: The value of items that are exempt from tax is displayed  in this field. This field cannot be edited.


**Total**
: The total value of items including tax if applicable,  is displayed in this field. This field cannot be edited. The total value  is computed using the following formula:
: Total Value = Taxable amount + Tax Amount + Exempt  Amount


**Deposit**
: The total amount that has been refunded by the vendor  till date is displayed in this field. The deposit amount includes amount  of refunds and purchase invoices applied. This field cannot be edited.  Click the ![]({{site.pp_baseurl}}/img/pur_document_alias.gif) button beside this field to view details of the  payment(s) made.


**Balance**
: The amount refundable by the vendor on the document  is displayed in this field.
