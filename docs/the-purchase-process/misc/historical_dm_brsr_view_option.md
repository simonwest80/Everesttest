# Historical


Select this option to view a list of all the vendor documents that are  imported using the **Data Import Wizard** (path: **Utility &gt; Import &gt; 
 Data** > **Historical Vendor Documents**).


**Everest** prefixes the document  number you enter with 'H', that denotes a historical document.
