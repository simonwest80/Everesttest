# Serial # Browser


The **Serial #** browser displays  a list of the serial numbers committed to the document for the selected  item. The browser provides details such as the item code, the manufacturer's  serial number (if assigned), the purchase document reference, the stock  area, the lot number, vendor, the cost and the sales document reference.


**Serial # Browser Options**


The options in the **Serial #**  browser can be accessed by right-clicking anywhere in the browser. The  following options are available:

- **Assign 
 Manuf. Serial No.** - Select this option to specify the manufacturer's  serial number for the item.
- **Deserialize**  - Select this option to detach a serial number that has been attached  to the purchase/debit order.
- **Open 
 Document** - This option is enabled only when you select the document  in the **PO#, PR#** or **Doc #** columns. Select this option to view the document.
- **Custom 
 Fields** - Use this option to define additional information for the  serial number. Custom fields can be defined as character strings, date  fields, logical fields or numeric fields.
- **Serial 
 History Browser** - Select this option to view the **Serial 
 Number History** browser.
