Lot Number Details

Items that are mass-produced are manufactured in lots or batches. You can assign unique lot numbers or batch numbers and affix them on all received quantities of mass manufactured items.

If items are tracked for lots, the Lot # Details section is displayed.

Lot Number
Click the drop-down icon on the Lot Number field. Right-click and select New. You will see the Lot profile.
Lot Expiration
The expiry date that you specify for the lot in the Lot profile is displayed here.
Quantity
Select a stock in the Stock Bin column and enter the quantity received (in that stock bin).

Set up Lot Tracking

You can define that items be tracked for lots in the Item profile (path: Inventory > Items > Options > Edit > In New Window > Purchases tab > Use LotTracking option).

Set up Lot Tracking for Items

If you select this option, you will not be able to save the purchase receipt unless you specify lot numbers for the quantity received.

Auto generated Lot Numbers

You can also set a preference for automatically generating lot numbers for items in the Item profile (path: Inventory > Items > Options > Edit > In New Window > Purchases tab > Auto generate lot numbers option).

The Auto generate lot numbers option is enabled only if you select the Use lot tracking option in the Item profile.
