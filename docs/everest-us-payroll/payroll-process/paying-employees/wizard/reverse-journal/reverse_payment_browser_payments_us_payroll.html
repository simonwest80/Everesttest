Reverse Payments Browser

The Reverse Payments browser (path: Accounting > US Payroll > Payments > Reverse Payment) lists all original payment payroll transactions for which a payment journal has been created. When you reverse a transaction in this browser, Everest reverses it but you will not be able to perform further reversing, purging, or voiding on this transaction. If you reverse and recreate a transaction in this browser, Everest replaces the reversed transaction with the new recreated one.

Only those transactions that have check number assigned and printed can be reversed.

Information in the Reverse Payments browser is grouped into the following sections:

  • View Options
  • Transactions Grid
