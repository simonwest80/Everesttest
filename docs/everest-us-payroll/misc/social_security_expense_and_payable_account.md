# Social Security Expense and Payable Account


**Social Security Expense Account and Location/Sub-Location**
: This is the expense account to which the social  security taxes payable as an employer are debited. Location/Sub-Location  is the location or sub-location to which this expense is allocated.
: ![Lens.gif]({{site.prl_baseurl}}/img/lens.gif) [Locations  and Sub-Locations]({{site.sc_chm}}/setting_up_a_company.html)


**Social Security Payable Account**
: This is the liability account to which the social  security taxes payable as an employer have to be credited.


**Social Security Payable Vendor**
: This is the vendor to whom the employer's social  security taxes have to be paid.
