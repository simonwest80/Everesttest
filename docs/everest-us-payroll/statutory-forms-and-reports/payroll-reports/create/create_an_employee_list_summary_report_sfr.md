# Employee List (Summary) Report


Unlike the **Employee List** (**Detail**) report, the summary report gives  only the essential details of the employees who are on the payroll of  the company. It displays the following:

- Employee code/Employee  ID
- Name and address
- Date of joining



{:.steps}
To create an employee list (summary) report,  do the following:

- From the main  menu bar, select **Reports &gt; Payroll 
 &gt; Employee List** (**Summary**).  You will see the [**Employee List (Summary) Report** dialog  box]({{site.prl_baseurl}}/misc/employee_list_summary_report_dialog_box_sfr.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview**  dialog box, before printing
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats.
