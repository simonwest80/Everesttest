# Deductions Report


This **** report summarizes deductions  made from all the employees' paychecks under each payroll item of type  deduction in the specified period. This report displays the following:

- Employee Name
- Payroll Item Description
- Pay Period
- Start Date
- Amount
- Total Deductions



{:.steps}
To create a deductions report, do the following:

- From the main  menu bar, select **Reports &gt; Payroll 
 &gt; Deductions**. You will see the [**Deduction Report** dialog box]({{site.prl_baseurl}}/misc/deduction_report_dialog_box_sfr.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview**  dialog box, before printing
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats.
