# Employee Journals Report


This report shows details of all payroll transactions created in the  specified period. The employee journals report displays the following:

- Employee code/Employee  ID
- Transaction number
- Transaction type
- Standard earnings
- Additions to pay  check
- Company contributions
- Breakup of the  total withholding amount
- Breakup of the  employer associated taxes
- Total employer  taxes
- Total tax withholding  taxes



{:.steps}
To create an employee journals report, do  the following:

- From the main  menu bar, select **Reports &gt; Payroll 
 &gt; Employee Journals**. You will see the [**Employee Journals Report** dialog box]({{site.prl_baseurl}}/misc/employee_journals_report_dialog_box_sfr.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview**  dialog box, before printing
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats.
