# Employee Earnings (Detail) Report


This report shows details of the earnings of all the employees for the  specified period. The earnings of the employee are categorized into standard  earnings, additions to paycheck, company-paid benefits, third party payments,  company contributions, tax withholdings, deductions, gross pay, net pay  and take home pay.


This report gives the payroll item-wise breakup of the employeesâ€™ earnings  under each category. The employee earnings (summary) report displays the  following:

- Standard earnings
- Additions to paycheck
- Company-paid benefits
- Third party payments
- Company contributions
- Tax withholdings
- Deductions
- Gross pay
- Net pay
- Take home pay



{:.steps}
To create an employee earning (detail) report,  do the following:

- From the main  menu bar, select **Reports &gt; Payroll 
 &gt; Employee Earnings** (**Detail**).  You will see the [**Employee Earning Details** dialog box]({{site.prl_baseurl}}/misc/employee_earning_details_dialog_box_sfr.html).
- Specify the  report parameters and click:


- **Preview**,  to preview the report in the **Report Preview**  dialog box, before printing
- **Print**,  to print the report
- **Export**,  to save the report in text, Excel, HTML or Crystal Report file formats.
