Modify State Details

Everest US Payroll allows you to modify all the information except the State Code in the State Setup profile.

You can access the State Setup profile you want to modify from the following locations:

  • State Setup browser - Highlight the record you want to modify and select Options > Edit >  In New Window.
  • The State Setup profile - Specify the State Code of the record you want to edit.
