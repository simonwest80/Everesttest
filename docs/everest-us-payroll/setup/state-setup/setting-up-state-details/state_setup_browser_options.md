# State Setup Browser 


The **State Setup** browser (path: **Payroll &gt; Setup &gt; State**) displays a list of all states you have set up for **Everest US Payroll**. Each row is specific to a state and provides relevant information of that particular state. Options available with this browser are standard browser options common to all browsers in **Everest**.
