Payroll Item Accounts

Accounts for payroll items can be specified in the Employee profile (path: Payroll > Employees > Employee Data > from the Employees browser, double-click the required record). If no accounts are specified for the payroll item of an employee, the accounts defined in the Payroll Items dialog box are considered.

Details of the payroll item accounts have been grouped into the following sections:

  • Debit and Credit Accounts
  • Vendor Code
