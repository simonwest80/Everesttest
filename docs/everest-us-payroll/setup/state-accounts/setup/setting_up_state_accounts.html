Set up State Accounts

You can access the State Accounts dialog box from the following locations:

  • The State Accounts browser - Select Options > New > In New Window.
  • An existing State Accounts dialog box - Select Options > New.

To set up state accounts, do the following:

  • From the main menu bar, select Payroll > State Taxes > Accounts. ** You will see the **State Accounts browser.
  • From the State Accounts browser, select Options > New > In New Window. You will see the State Accounts dialog box.
