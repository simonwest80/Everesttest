Payroll Item Browser Options

The Payroll Items browser (path: Payroll > Payroll Items) displays a list of all the payroll items defined by you. Each row is specific to a payroll item and provides basic information of that particular payroll item.

Except Form W2 Tracking, all the other options (path: Payroll Items browser > Options) available from this browser are standard options common to all browsers in Everest.

Form W2 Tracking

This option allows you to set up the W2 tracking of payroll items.
