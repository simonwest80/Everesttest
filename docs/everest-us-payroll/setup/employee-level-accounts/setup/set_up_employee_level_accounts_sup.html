Set up Employee Level Accounts

You can set up employee level accounts from the following locations:

  • The Employees browser - Highlight the employee and select Options > Employee Accounts.
  • An existing Employee profile - Select Options > Accounts.

To set up employee level accounts, do the following:

  • From the main menu bar, select Payroll > Employees > Employee Data. You will see the Employees browser.
  • From the Employees browser, highlight the employee for whom you want to set up accounts and then select Options > Employee Accounts. You will see the Employee Account Setup dialog box.
