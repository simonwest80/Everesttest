Associating Payroll Items

You are required to associate payroll items for Boxes 7, 8, 10, 11 and 12.

To associate payroll items for the boxes 7, 8, 10, 11 and 12, do the following:

  • From the main menu bar, select Payroll > Setup > W2 Tracking. You will see the Employee Form W2 Tracking.
  • Click the box for which you want to set up details. You will see the Form W2 Tracking Details dialog box ** with the respective box number. This box lists all the payroll items that can be tracked in the **Unassigned column.

Verify the association of payroll items in the W2 Tracking dialog box as per the guidelines provided in the IRS document. If the wage base amounts are incorrect, you may have erred in setting up W2 tracking for payroll items. You can rectify the settings in the W2 Tracking dialog box at any point of time.
