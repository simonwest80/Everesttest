# Add Users to a Sub-Site


Add end users to a sub-site to give them permission to use/view the  sub-site.


To give the end users access to restricted content, create a sub-site,  configure it with restricted content, and add users to the restricted  sub-site. For more information, see the [Additional  Information on Microsoft Windows速  SharePoint速  Services]({{site.db_baseurl}}/share-point-services-basics-based-on-windows-network-users-/additional_information_on_microsoft_windows_sharepoint_services_ead.html) section.


![]({{site.db_baseurl}}/img/lens.gif)[Additional  Information on Microsoft Windows速  SharePoint速  Services]({{site.db_baseurl}}/share-point-services-basics-based-on-windows-network-users-/additional_information_on_microsoft_windows_sharepoint_services_ead.html)


{:.steps}
To add users to a Sub-site, do the following:

- From the **Sub-Site Everest Dashboard** menu bar,  click **Site Settings**.
- In the **Administration**  section, select **Manage Users**.  The **Manage Users** page is displayed.
- Click **Add 
 Users**. The first **Add Users**  page for the sub-site is displayed.

- In the **Users**  field, enter one or more users who are allowed to access the sub-site.
- Select the **Site Groups** for the user or users entered,  such as **Reader** for read-only access.
- Click **Next**.  The second **Add Users** page is displayed  for the sub-site.
- Click **Finish**  to add the user to the sub-site.
