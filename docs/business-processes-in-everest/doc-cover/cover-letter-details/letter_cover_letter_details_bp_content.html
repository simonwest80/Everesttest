Letter

Use the Letter tab to enter comments/message for the fax/regular letter to be printed in the cover letter. This text is displayed by default when you print the cover letter for a document. You can modify this text from the Document Printing Preferences profile before you print the document.

 The Printing Preferences Dialog Box

 You can change the letter type to regular or fax at any time, even if the cover letter is used in a document.
