# The Cover Letter Browser


The **Cover Letter**  browser (path: **File &gt; Setup &gt; Miscellaneous 
 &gt; Cover Letter**) displays a list of all the cover letter templates  that have been created, along with the cover letter code, description,  document type, letter type, document number, last printed date and time  and last modified date and time. Options common to all browsers are also  available in this browser.
