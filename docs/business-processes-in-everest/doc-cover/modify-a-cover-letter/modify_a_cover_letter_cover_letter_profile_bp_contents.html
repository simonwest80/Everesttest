Modify a Cover Letter

You can modify the CoverLetter profile from the following locations:

  • The Cover Letter browser - Highlight the record you want to modify. Select Options > Edit > In NewWindow.
  • The Cover Letter profile - Specify the Cover Letter code in the Code field.

You can modify the following fields in the Cover Letter profile:

  • Description (General tab)
  • Letter Type (General tab)
  • Cover Letter Memo (Letter tab)
