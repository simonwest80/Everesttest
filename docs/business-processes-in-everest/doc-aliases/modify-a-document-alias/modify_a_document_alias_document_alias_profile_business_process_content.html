Modify a Document Alias

You can modify a Document Alias from the following locations:

  • The Document Aliases browser - Highlight the record you want to modify. Select Options > Edit > In New Window.
  • The Document Alias profile - Specify the document alias code.

You can modify only the following fields:

  • Document Policy
  • Sales Cycle days
  • Document Alias Group
  • Document Alias used when creating (if enabled)
