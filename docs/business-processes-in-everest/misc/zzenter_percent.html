Enter a default percentage by which if the current last cost varies from the previous last cost, the user is prompted whether to update the Item profile (path: Inventory > Items > Options > Edit > In New Window) with the current last cost.

You can enter a value in this field only if the Prompt if % difference is option is selected in the When the last cost of an item changes at the time of receiving, how do you wish to prompt the user for updating the last cost? field.
