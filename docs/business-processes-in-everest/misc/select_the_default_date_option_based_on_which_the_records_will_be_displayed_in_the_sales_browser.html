Select the default date option based on which the records will be displayed in the sales browser

Use this field to filter sales documents in sales browsers based on their creation date.

 All date options (such as today and this week) are with respect to the login date.

The options available in this field are detailed below:

Option Displays a list of:
All All documents (irrespective of their creation date).
Today Documents created on the login date.


Login Time and Date
This Week Documents created during the week in which you have logged in.
This Week-to-date Documents created from the beginning of the week to the login date.
This Month Documents created during the month in which you have logged in.
This Month-to-date Documents created from the beginning of the month to the login date.
This Year Documents created during the year in which you have logged in.
This Year-to-date Documents created from the beginning of the year to the login date.
Yesterday Documents created on the day prior to the login date.
Last Week Documents created during the week prior to the one in which you have logged in.
Last Week-to-date Documents created from the beginning of the week prior to the week in which you have logged in to the login date.
Last Month Documents created during the month prior to the one in which you have logged in.
Last Month-to-date Documents created from the beginning of the month prior to the one in which you have logged in to the login date.
