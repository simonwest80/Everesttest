Define Activities

To define activities, do the following:

  • From the main menu bar, select CRM > Business Activity Monitoring > Processes/Activities. You will see the Process/Activities browser.
  • From the browser, select Options > New > Activity > In New Window. You will see the Activity profile.

You cannot access a new Activity profile without a process. To enable the new profile, a process should already exist under which the activity is to be defined.
