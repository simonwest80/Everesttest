# Discontinue a Template


When you discontinue a template, it is temporarily unavailable for selection.  The template is not deleted from the database and is merely deactivated.  A discontinued template can be made available for selection by activating  it.


{:.steps}
To discontinue a template, do the following:

- From the main  menu bar, select **CRM 
 &gt; Business Activity Monitoring &gt; Templates**. You will see  the **Templates** browser.
- From the browser,  highlight the template.
- Select **Options &gt; Discontinue**. You will be  prompted for confirmation.
- Click **OK**. The record that was selected is  no longer visible in the **Templates** browser.
