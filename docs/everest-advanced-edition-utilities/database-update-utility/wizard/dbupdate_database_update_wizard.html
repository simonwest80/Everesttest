Database Update Wizard

The Database Update Wizard guides you through the process of updating the database. You can navigate within the wizard by clicking the Next button.

The Database Update wizard contains the following sections:

  • Database Server and Company Details
  • Security Group Creation and Assignment (If the new version contains new security rights)
  • Database Backup
  • Comparing System Database with Everest Defaults
  • System Database Update
  • Company Database Update
  • Update Completed
