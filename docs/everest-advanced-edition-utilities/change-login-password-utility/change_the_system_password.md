# Change Everest System Password


The system password is relevant when you create new companies in **Everest**. If a new password has been  assigned to the system, you can activate the new password in **Everest**  with the help of the **Change** **Login** **Password**  dialog box.


**Server Name**
: Select the server name from the drop-down list.  This is a mandatory field.
: ![Lens.gif]({{site.advutl_baseurl}}/img/lens.gif) [Change  Everest Database Username  and Password]({{site.advutl_baseurl}}/change-login-password-utility/change_the_database_username_and_password.html)


Options


**Change Everest System Password**
: Select this option to change the password that has  been specified for the system. This is a mandatory field.


**System Login**
: By default the system login is displayed in this  field.


**Old Password**
: Enter the current login password. This is a mandatory  field.


**New Password**
: Enter the new password that has been assigned to  the system login. You can enter up to 255 alphanumeric characters in this  field. This is a mandatory field.


{:.note}
You can activate the new password only if  a new password has been assigned for logging into the system.


**Retype the New Password**
: Re-enter the new password. This is a mandatory field.


Click **OK**.
