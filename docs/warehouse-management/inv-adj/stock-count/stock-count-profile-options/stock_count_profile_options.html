Stock Count Profile Options

The various options available in Stock Count profile options are as follows:

  • Process - Retrieve Items
  • Process - Block for Counting
  • Process - Take Snapshot and Count
  • Process - Cancel
  • Process - Mark Counted/Recounted
  • Process - Clear Counts
  • Process - Finalize and Adjust Inventory
  • Process - Mark all Rows as Verified
  • View - System Quantities
  • View - Cost Details
  • Refresh - Refresh System
  • Print - Count Sheet
  • Print - Discrepancy
