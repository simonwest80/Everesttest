# The Stock Count Browser


The **Stock Count** browser (path:  **Inventory** > **Inventory 
 Adjustment** > **Stock Count**)  lists all the documents created for the purpose of counting and reconciling  the item quantities in **Everest**.


The options (path: **Inventory**  > **Inventory Adjustment** >  **Stock Count** > **From 
 the Stock Count** browser > **Options**)  available in the **Stock Count** browser  are explained below:

- [Purge]({{site.wm_baseurl}}/misc/purge.html)
- [Print - Count  Sheet]({{site.wm_baseurl}}/misc/print_-_count_sheet.html)
- [Print - Discrepancy  Report]({{site.wm_baseurl}}/misc/print_-_discrepancy.html)
