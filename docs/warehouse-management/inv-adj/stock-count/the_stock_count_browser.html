The Stock Count Browser

The Stock Count browser (path: Inventory > Inventory Adjustment > Stock Count) lists all the documents created for the purpose of counting and reconciling the item quantities in Everest.

The options (path: Inventory > Inventory Adjustment > Stock Count > From the Stock Count browser > Options) available in the Stock Count browser are explained below:

  • Purge
  • Print - Count Sheet
  • Print - Discrepancy Report
