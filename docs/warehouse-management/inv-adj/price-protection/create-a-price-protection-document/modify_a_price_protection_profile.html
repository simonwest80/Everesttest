Modify a Price Protection Profile

Modification to the Price Protection document can be made only if the journal in which the entry has been created has not been posted.

You can modify the following fields:

  • Location/Sub-Location
  • Inventory Adjustment account
  • Document Notes

You can access the Price Protection profile for modification from the following location:

Inventory > Inventory Adjustment > Price Protection > Options > Edit > In New Window.
