# Inventory Adjustment Document Details


Inventory adjustments in **Everest**  are recorded in **Inventory Adjustment** documents **** for each type  of adjustment.


**Everest** allows you to make inventory  adjustments to multiple items in a single inventory adjustment document.


Details common to all types of inventory adjustment documents are grouped  as under:

- [Document  information]({{site.wm_baseurl}}/misc/document_information_inv_adj_details.html)
- [Item  information]({{site.wm_baseurl}}/misc/item_information_inv_adj_details.html)
- [Document  tab view Information]({{site.wm_baseurl}}/misc/document_tab_view_information_inv_adj_doc_det.html)
- [Status  bar information]({{site.wm_baseurl}}/misc/status_bar_information_inv_adj_det.html)
