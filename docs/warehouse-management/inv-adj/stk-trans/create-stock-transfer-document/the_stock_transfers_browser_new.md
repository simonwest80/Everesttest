# The Stock Transfer Browser


The **Stock Transfer** browser (path:  **Inventory &gt; Inventory Adjustment &gt; 
 Stock Transfer**) displays all stock transfer documents created in  **Everest**. Options available in  this browser are standard options common to all browsers in **Everest**.

- To transfer stock  for a company within a single location, select **Options**  > **New** > **In 
 New Window**.
- To transfer stock  across locations, select **Options**  > **New Request** > **In 
 New Window**. The **Status**  column tracks the state of the stock transfer document and displays the  state - Request, In-Transit and Completed.
- To delete a request,  select **Options** > **Purge 
 Request**.



**C**ustom Fields


This option brings up the **Custom Fields**  dialog box that allows you to define additional information for any record  as per your requirement. Custom fields can be defined as character strings,  date fields, logical fields or numeric fields.


Print Document


You can print a stock transfer document from the **Stock 
 Transfer** profile or from the **Stock 
 Transfers** browser by selecting **Options 
 &gt; Print**. The **Stock 
 Transfer Report** dialog box is displayed wherein you can specify  the printing details.
