Create a Stock Transfer Document

To transfer a stock of items, do the following:

  • From the main menu bar, select Inventory > Inventory Adjustment > Stock Transfer. You will see the Stock Transfers browser.
  • From the Stock Transfer browser, select Options > New > In New Window if you are a company with a single location. You will see the Stock Transfer profile for a single location transfer.
  • From the Stock Transfers browser, select Options > New Request > In New Window if you wish to transfer stock across locations. You will see the Stock Transfer profile for stock transfer across locations.

Stock Transfer can also be made from an existing Stock Transfer profile by selecting Options > New.
