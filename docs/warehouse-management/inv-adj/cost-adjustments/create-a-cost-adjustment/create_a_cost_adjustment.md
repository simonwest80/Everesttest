# Create a Cost Adjustment


{:.steps}
To make cost adjustments, do the following:

- From the main  menu bar, select **Inventory &gt; Inventory 
 Adjustment &gt; Cost Adjustment**. You will see the **Cost 
 Adjustment** browser.
- From the **Cost Adjustment** browser, select **Options &gt; New &gt; In New Window**.  You will see the [**Cost 
 Adjustment** profile]({{site.wm_baseurl}}/inv-adj/cost-adjustments/create-a-cost-adjustment/the_cost_adjustment_profile.html).



{:.note}
Cost adjustments can also be made from an existing **Cost Adjustment** profile by selecting  **Options &gt; New**.
