# Modify a Cost Adjustment Profile


After saving the **Cost Adjustment**  document, if the journal in which the entry has been created is not posted,  you can modify the following fields:

- Document Notes
- Loc/Sub-Loc  (Location/Sub-Location)



After posting, you can only modify the following field:

- Document Notes



You can access the **Cost Adjustment**  document for modification from the following location:


**Inventory &gt; Inventory Adjustment 
 &gt; Cost Adjustment &gt; Options &gt; Edit &gt; In New Window**.
