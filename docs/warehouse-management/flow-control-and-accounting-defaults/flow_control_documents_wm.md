# Flow Control - Documents


You may set the preference **Use lot 
 tracking** to use lot tracking, in the **Item**  profile (path: **** **Inventory 
 &gt; Items &gt; Options &gt; Edit &gt; In New Window** > **Item** profile > **Purchases**  tab). If this option is not selected, you are not required to assign items  to lots.


Default Stock Area to Transfer Inventory Written off


The option **Select stock area to transfer 
 inventory written off** (path: **File 
 &gt; Setup &gt; Accounting &gt; Location/Sub-Location &gt; Stock Transfer 
 &gt; Inventory Adjustment** tab) is the default stock area to which  inventoried items written off through the **Write 
 off Quantity** profile are moved. This is stock area of type **Out**.
