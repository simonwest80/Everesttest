# Lot Tracking in Everest


**Everest** allows you to specify  lot numbers of inventoried items received. When the item is sold or returned,  you can enter the lot number to which the item belongs.


To enable lot tracking, select the option **Use lot tracking** from the **Item**  profile **** (path: **Inventory 
 &gt; Items &gt; Items** browser > **Item**  profile > **Inventory** tab).
