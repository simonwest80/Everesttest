Modify a Lot Number

You can modify lot numbers from the following locations:

  • The Lots browser - Highlight the record and select Options > Edit > In New Window.
  • An existing Lot Number profile - Specify the Lot ID of the record you want to modify.

You cannot modify the following fields:

  • Lot ID
  • Lot number
  • Item code
  • Vendor code

None of the fields can be modified in a Lot Number profile if the lot number is used in a purchase document.
