Assign a Lot Number

You can assign lot numbers to items from the following locations:

  • The Lots browser - Select Options > New > In New Window .

  • An existing Lot Number profile - Select Options > New.

  • From the Purchase Receipts profile - Select Options > Attach Retrieve Serial # > Transfer/Attach Quantity. In the Lot Number field, select New from the right-click menu.
  • From the Purchase Invoices profile - Select Options > Attach Retrieve Serial # > Transfer/Attach Quantity. In the Lot Number field, select New from the right-click menu.

To assign a lot number, do the following:

  • From the main menu bar, select Inventory > Lots. You will see the Lots browser.
  • From the Lots browser, select Options > New > In New Window. You will see the Lot Number profile.
