General Information

Serial Number
This is the unique serial number assigned to the item when it is received on purchase order, receipt, or invoice.
Item Code
The item code identifies the item to which the serial number belongs to.
Model Number
The model number for the item as defined in the Item profile is displayed.
Description
Description identifies the item description to which the serial number belongs.
Manufacturer’s Serial Number
The manufacturer’s serial number for the unit can be assigned here.
