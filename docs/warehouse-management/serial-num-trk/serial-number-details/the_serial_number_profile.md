# Serial Number Details


The **Serial Number** profile provides  information of the purchase details, manufacturer's serial number, current  stock area and the document against which the serial number is attached.


Information on this profile has been grouped into the following sections:

- [Purchase  Details]({{site.wm_baseurl}}/serial-num-trk/serial-number-details/purchase_details.html)
