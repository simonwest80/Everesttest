# Custom Fields for Serial Numbers


The **Custom Fields** feature allows  you to define additional information for any record as per your requirement.  Custom fields can be defined as character strings, date fields, logical  fields or numeric fields. Custom fields can be set up for serial numbers.


![example.gif]({{site.wm_baseurl}}/img/example.gif) You  can set up custom fields for Manufacturing Date, Packing Date, Date of  Expiry, Warranty  Date, Warranty Number etc.


To set up custom fields, access the **Custom 
 Fields - Serial Number** profile by selecting **Options 
 &gt; Custom Fields** from the **Serial 
 Number** profile. Then, specify the information required for the  serial number in custom fields.


{:.steps}
To access the serial number profile, do the  following:

- From the main  menu bar, select **Inventory &gt; Serial 
 Numbers**. You will see the **Serial 
 Numbers** browser.
- From the **Serial Numbers** browser, highlight the  record you want to view or modify and select **Options 
 &gt; Edit &gt; In New Window**. You will see the [**Serial Number** profile]({{site.wm_baseurl}}/serial-num-trk/serial-number-details/the-serial-number-profile/the_serial_number_profile_1.html) of the record  you selected.
