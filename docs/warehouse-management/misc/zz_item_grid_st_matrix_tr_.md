**Item 
 Grid**


**Item Code**
: This column displays the code of the matrix item  you have selected for transfer in the **Matrix 
 Item View** dialog box.


**Description**
: The description of the matrix item is displayed  in this column.


**Average cost**
: The average cost of a matrix item is displayed in  this column.


**Quantity Available**
: This column displays the total quantity available  for transfer in that lot, if any, from a particular stock area. This quantity  cannot be edited.


**Quantity transferred**
: Enter the quantity of matrix items to be transferred  in this **** column.


**Amount**
: The total value of matrix items to be transferred  is displayed in this column.
