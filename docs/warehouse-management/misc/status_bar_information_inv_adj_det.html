Status Bar Information

The Status Bar is displayed at the bottom of all document profiles in Everest. The options available in the status bar of inventory adjustment document profiles are elucidated below:

Zoom Out/In

click the Zoom Out button to view only the Document Information, Item Grid information and Status Bar Information. Click the Zoom In button to see all the details for a document.

Item Notes

This option brings up the item notes for the selected line item, as specified in the Item profile (path: Inventory > Items > Options > Edit > In New Window > Notes Tab). You cannot edit the item notes from here.

Notes Available Status Displayed
Upsell information, and notes and specifications
Only upsell information
Only notes and specifications

Item Profile - Notes

Stock Status

This option displays the stock status for the items in the inventory adjustment documents. The stock status indicator denotes that the available stock for the item is greater than zero. click this icon to view the Item Stock Status dialog box that displays detailed information about the stock status of the items in the inventory adjustment documents

Item Stock Status

Posted/Not Posted

The posting status of the transaction journal, in which the accounting entry is made, is displayed in this section. When you click in this section, the Journal Transactions browser, which contains the accounting entry pertaining to the inventory adjustment, is displayed.

Status Displayed Description
The journal has been posted to the accounts
The journal has not been posted
