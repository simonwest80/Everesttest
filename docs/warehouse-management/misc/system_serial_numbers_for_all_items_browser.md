# System Serial Numbers for all Items Browser


The **System Serial Numbers for All Items**  browser displays details of the system quantities of serialized items.  You can view the serial numbers of the items available in the system stock,  the item code, the stock area type and the bin where the system quantity  is present. This browser comes up when you double click in the **System 
 Quantity** column of the **Stock Count**  profile.
