# Process - Clear Counts


The **Clear Counts** option is enabled  after either of the options – **Block for 
 Counting** or **Take Snapshot to Count**  – is selected and the status changes to **Counting 
 in Process**.


Click this option to clear the counts, in case of discrepancy, and reset  the physical quantities to zero. The status of the document remains the  same, that is, **Counting in Process.**


**Differences between Recount and Clear Count**


| Clear Count | Recount |
| This option is enabled when the document is in the **Counting <br/> in Process** stage. | This option is enabled when the document is in the **Counted**  stage. |
| If you want to change the quantity in the **Physical <br/> Quantity** column at this stage, you can use the **Clear <br/> Counts** option to set the physical quantity to zero. | If you want to change the quantity in the **Physical <br/> Quantity** column when the document has been marked as counted, you  can use the **Recount** option to  edit the physical quantity. |
| The document continues to remain in the **Counting <br/> in Process** stage. | Choosing this option reverts the document to the **Counting <br/> in Process** stage. |
