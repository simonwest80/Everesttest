---
title: Purchase Price
---


Select the parameter based on which to perform the serial number search  (by purchase cost).


If the parameter selected requires a range to be specified, both the  fields beside the **Purchase Cost** field  are enabled. Enter the starting value in the first field and the ending  value in the second field.


If the parameter selected requires one value to be specified, enter  the value in the first field. The second field is disabled.
