Print a Stock Transfer Document

You can print a stock transfer document from the Stock Transfer profile or from the Stock Transfer browser.

Select Options > Print from the profile or the browser.

Print Serial Number Details
Select this option to print serial numbers attached to the stock transfer document.
Print Lot Details
Select this option to print details of the lot numbers attached to the stock transfer document.
Number of Copies
Specify the number of copies you want to print.
