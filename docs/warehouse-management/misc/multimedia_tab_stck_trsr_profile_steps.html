Multimedia Tab

The Multimedia tab allows you to attach images, sound, video, documents, spreadsheets, and other files to a stock transfer document. You can store these files in the multimedia repository.

This tab is displayed only when you save the stock transfer document.

Multimedia

To attach a multimedia file to a stock transfer document, do the following:

  • From the browser in the Multimedia tab of the stock transfer document ** profile, select **Options > New. A new record will be added to the browser.
  • Click the Multimedia File column of the new record to enable the lookup icon .
  • Select the required multimedia file by clicking the lookup icon. The other columns in the browser are automatically updated.
  • Save the details.
