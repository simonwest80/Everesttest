# Multimedia Tab


The **Multimedia** tab allows you  to attach images, sound, video, documents, spreadsheets, and other files  to a quantity adjustment document. You can store these files in the multimedia  repository.


{:.note}
This tab is displayed only when you save the quantity  adjustment document.
